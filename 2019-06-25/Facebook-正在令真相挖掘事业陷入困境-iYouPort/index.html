    Facebook 正在令真相挖掘事业陷入困境

    McCaffrey Newsletters , OSINT , 关于 Facebook , 关于透明度革命 , 行动派- 社会反抗的思考 2019年6月26日 2019年6月25日 0 Minutes
    • 噩梦成真了

    IYP 在去年的答读者问中解释过为什么OSINT是“退一步的想法”，而不是进一步，在这里看到《 情报不是简单的信息 》。

    我们的基本原则应该是：不要依赖自己无法掌控的东西。这就是为什么民间独立的OSINT注定处于弱势地位 —— 因为它所依赖的是工具掌握在技术寡头手里。

    这是对民间独立OSINT长期以来的担忧，现在噩梦成真了。

    寡头以“隐私”为掩护剥夺正义人士分享数据的能力 —— 这招稳准狠， 隐私的绝对性解释是OSINT的唯一弱穴。

    这不会是偶然现象，或许只是一个开始。

    请注意这里的关键字是：“民间独立- 调查”；开源情报也是政府间谍部门很常用的方法，但是他们不会被打击。权力再次失衡。

    如果民间输掉了这场对战 Facebook 的战役，真想挖掘的事业将就此面临迅速加深的危机。（当权者对民间独立调查的压制去年已经开始了，见这里《 “谷歌地图”是比詹姆斯邦德更棒的间谍 》）。

    Facebook 悄然改变了一系列高级功能，这些功能以前允许用户以强大的方式搜索社交网络 —— 也就是 OSINT 的主要工具之一。

    例如，通过关键字在特定日期范围内查找 Facebook 上的所有帖子、所有点赞某个 Facebook 页面并居住在特定城市或两个特定用户访问过的地方的人等等。

    这些搜索功能收集的所有信息都来自用户的公开个人资料 —— 也就是开源情报的概念 。

    但是，相关新闻突出了 Facebook 最近对所谓的“隐私”的重视 —— 简单说是在隐私的旗号下。

    这些功能可能会被滥用于查找 Facebook 用户的信息，但是，它们对于开源调查人员来说也有很大的帮助。 比如查找网络水军、追踪群体事件揭露官方谎言等等 。相见《 OSINT 》操作。

    社交媒体研究专家兼在线调查组织 Bellingcat 的成员 Henk Van Ess 在 Twitter 的直接消息中表示，“大多数工具都已关闭。”

    该功能称为图谱搜索，基本上需要在浏览器中输入正确的 URL，其中包含一些参数，例如嵌套在该 URL 中的关键字。

    开源情报（OSINT）社区的一些成员构建了工具来简化这些 URL 的生成并使搜索过程更容易。

    Van Ess 在其网站上发布了公共工具，用于搜索包含特定日期、月份或年份的关键字的帖子以及日期范围。

    OSINT 培训师 Michael Bazzell 在他的网站上发布了大量的 Facebook 搜索工具，包括查找 Facebook 用户评论过的所有照片的工具。

    另一项名为 StalkScan 的服务由道德黑客 Inti De Ceukelaire 运营，允许用户简化类似的搜索。

    “正常的 Facebook 搜索功能根本做不到这点：它返回的结果要少得多，而你可以搜索的最小时间范围是一个月，” Bellingcat 的成员尼克·沃特斯（Nick Waters）在一封写给 Facebook 员工的信件中写道。

    但据多家社交媒体研究人员和工具维护人员都称，图谱搜索已经停止工作。

    “由于 Facebook 的变化，StalkScan 目前无法使用，” StalkScan 网站 上的一则公告写道。

    Bazzell 还面临着其他麻烦，⚠️虚拟主机 威胁要关闭他的网站 ，除非他移除那些搜索工具。

    “每个人都开始报道 Facebook Graph 被破坏的同时，我的网站完全崩溃了，我被暂停了。我不是在责怪 Facebook。我只是说：在同一时间发生所有事的可能性是多少”，他说。

    使用这些工具来挖掘真相的团体已经感觉到了压力。周六， Waters 只能在推特上寻求帮助 ，以寻找有助于研究也门空袭的调查的材料 。

    “很明显这一工具被一些非常重要的社会部门使用，从人权调查人员和需要真相的公民，从调查人口贩运和性奴役的警察，到紧急救援人员……”

    Van Ess 还与 Motherboard 分享了一些消息，他说这些消息来自寻找替代工具或更新的人，包括记者和追踪腐败的公司。

    “仅仅通过阅读 DM [直接消息]，我就能看到杀死 Graph 的影响有多深远： 涉及侵犯人权，战争罪，恐怖主义，极端主义，白领犯罪（欺诈，洗钱，逃税），腐败，虚假宣传活动，环境犯罪，网络犯罪的调查，都受到了挫折 —— 该名单还可以更长 ，“他说。

    然而，这些工具的确被滥用了。 Bazzell 在他的播客中说，虽然他的网站面临很多恶意攻击，但工具部分是被攻击最多的目标；

    “我的在线工具一直在被滥用，”Bazzell 在播客中说道。 “有许多公司试图使用我的 PHP 工具来集中收集信息。”

    “我的 Facebook PHP 查找工具经常被那些每分钟提交数千个请求以将 Facebook 用户名翻译成ID号码的人滥用，然后将其用作数据库创建过程的一部分，”他补充说。

    这一变化似乎也开始了 OSINT 社区与 Facebook 之间的猫捉老鼠游戏，⚠️社区不断更新工具，而 Facebook 不断阻拦。

    “我修补了我的工具5次，每次，大约2小时后，工具就会被FB [Facebook]瘫痪。其他工具开发者也经历了同样的事，” Van Ess 说。

    他说他已经找到了许多仍然允许相同搜索功能的解决方法，现在密码保护他的网站，所以只有经过审核的人才可以使用这些工具。

    Van Ess 授予主板访问权限以验证工具；主板确认其中一些确实有效，但其他的在撰写本文时并没有效果。

    在他的播客中，Bazzell 说他的 Facebook 工具在更新后再次运行。

    Bazzell 表示他们仍有一些问题，有些人使用该网站的移动版本，“但我们正在恢复该功能。”这些工具仅次于 Bazzell 自己的会员专区，而以前是公开的。

    Bellingcat 计划向 Facebook 发布一封公开信。在撰写本文时，Bellingcat 仍在起草这封信。

    今年3月，Facebook 首席执行官马克扎克伯格表示，该公司正朝着“用户隐私的新方向”发展。这是剑桥分析公司丑闻之后的插曲。但是，不久之后，Facebook 被发现以明文存储了数亿个密码…… 所以， 信他是愚蠢的 。

    ⚠️对工具的封查完全不能保证不会再出现剑桥分析公司那样的事，他们在暗地里，而公民调查人员在明处。这就是 Facebook 以“隐私”为大旗的荒谬。

    ⚠️更重要的是，如果此次对战 Facebook 导致的协商结果是要求使用OSINT工具的人遵循实名制，那将出现另一个危机—— 尤其是对于居住在不自由国家的民间调查人员来说，如您所知。
