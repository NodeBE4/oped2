    如何穿越付费墙：12个省钱大法

    McCaffrey Knowledge Node , 技术防身/自我保护方法 , 给行动者的护身符和武器 2020年3月24日 2019年11月7日 0 Minutes
    • 注意！别太狠！

    付费墙是有多讨厌，如今已经尽人皆知，《 地狱般的付费墙：免费、开放和民主化的 Web 时代已成往事了吗？ 》。

    但是没办法，内容提供者需要吃饭，需要活下去才能继续为读者服务 —— 像 IYP 这样自掏腰包的穷光蛋已经快撑不住了。

    但是IYP坚持不搞付费墙，因为我们自己就经常被其他资源的付费墙惹恼，己所不欲。

    尤其是那些大牌企业媒体搞的付费墙。它们原本就有着流油的资金背景和广告收入，它们还想让你掏腰包，对于老百姓来说，这不公平。

    绕过付费墙是可能的吗？是的。

    某些时候，甚至一个简单的浏览器技巧就足以帮您绕过付费墙以阅读文章、观看视频等。

    付费墙有两种，您最有可能解锁的是“软”的那种。软付费墙是一种让您能看到一小部分内容的拦截方法，而硬的付费墙则是，如果不付钱，你一个字都看不到。

    请注意 ：我们不建议您使用付费墙拦截器，因为它们会使一家本来有潜在收入来源的公司饿死。与广告拦截器类似，您应该明智地使用它们，并考​​虑它们可能会对您喜欢的网站产生的影响。

    12种解锁付费墙的方法

    您可以尝试多种方法来突破收费壁垒。有些方法对于有些墙来说注定无法使用，但肯定有一个或两个选项适用于您尝试访问的网站。

    1、如果付费墙的Firefox扩展

    使用 此 Firefox 扩展程序 绕过付费专区。它适用于彭博社、丹佛邮报、巴尔的摩太阳报、先驱报以及其他许多网站。

    只需使用该页面顶部的下载链接即可获取文件，然后将其拖到 Firefox 窗口中进行安装。

    从那里开始，您所要做的就是访问该站点以查看该扩展是否删除了付费墙。您也可以进入设置以在任何受支持的站点上禁用它（允许特定的付费墙）。

    2、寻找其他地方的文章

    复制文章标题，并将其粘贴到搜索引擎中以查找被转载的内容。

    这是绕过付费墙的最简单方法之一，因为很多时候，基于订阅的新闻机构的热门文章都会被完全免费复制和发布到其他地方。

    像这类有多个空格的搜索，要记住的重要一点是：将所有单词括在引号中 ，就像上面图中那样。这将限制并排除不相关的搜索结果，并保证您找到的内容与您所关注的文章相关。

    3、试用 Unpaywall Chrome扩展程序

    Unpaywall 是 Chrome 浏览器的扩展程序，可解除对学术文章的付费墙的限制。它会在互联网上搜索该产品的免费PDF版本，然后为您提供查找免费版本的链接。

    4、重置您的浏览器 Cookie

    删除您的浏览器 cookie 或使用 “隐身模式” 绕过付费墙。

    如果该网站允许您查看少量文章，然后使用付费墙阻止访问，那么显然，他们是通过存储 cookie 来跟踪您浏览了多少页面的。所以，对症下药。

    删除 Cookie 或以隐身模式打开网站以绕过 Cookie，就好像您是新访客一样出现在该网站上，从而重置了您可以打开的免费文章的数量。

    5、使用 Outline.com

    在 Outline 中输入目标页面的 URL。Outline 是用于避免干扰的，但是由于其工作原理，通常有助于避免支付墙。

    6、手动删除付费墙

    如果可能的话，请删除付费墙弹出窗口。

    一些网站使用超级基本的付费墙系统，其中唯一阻止您查看页面的是弹出窗口。虽然没有退出按钮，但您可以查看页面源码并将其手动删除，以显示其背后的内容。

    例如，在 Chrome 中，右键单击付费墙消息，然后选择 “Inspect”，使用Chrome工具左上方的选项来查找付费专区的源代码。

    找到它后，右键单击该数据，然后选择 “ Delete element ”。您可能需要删除多个东西，直到找到正确的方式来摆脱整个支付壁垒。

    如果付费墙使用 JavaScript，那么可能会起作用的类似选项是使用比如 Quick JavaScript Switcher for Chrome 的工具阻止脚本。

    如果您不使用Chrome ，或这些方法不起作用，请尝试使用可以删除脚本的Web代理，例如 HMA 。

    7、中断页面完全加载

    在付费墙弹出之前快速暂停页面。如果运气不好无法删除或阻止 JavaScript，则可以中断页面的完全加载，以阻止显示付费专区弹出窗口。

    但是你必须足够快；如果页面加载速度太快或付费墙在文章出现之前就启动了，那么该功能就无法使用。要尝试此操作，只需刷新页面并在加载内容后立即按 Esc 键几次。

    8、挖掘存档站点

    诸如 Wayback Machine 和 Archive.is 之类的存档站点会保存网页以用于存档。您可以利用这些方法查找由于付费专区封锁而无法访问的文章。

    9、使用“稍后阅读”工具

    将文章发送到书签或类似 Pocket 的稍后阅读服务，以绕过文章限制。

    但只有在您访问的前几次拥有免费访问该网站的权限时，此方法才有效。

    其原理与上面的 cookie 方法类似，因为该页面的内容将发送到其他地方并且不会加载到您的计算机上。

    10、将页面转换为PDF

    有很多把页面转换为PDF的转换器，其中一个就是 Webpagetopdf.com 。

    只需将付费墙页面的URL放入文本框中，进行转换，然后下载PDF，以使您可以脱机访问该文章，并且始终可以免费访问。

    11、查找登录详细信息

    如果您要处理的是 “硬” 付款墙，并且需要用户帐户，请尝试使用共享登录服务。

    诸如 BugMeNot 之类的网站可能具有登录详细信息，您可以用来登录该网站，并假装是真实用户来绕过付费墙。

    12、注册免费试用

    如果可以申请试用，以便您可以绕开付费墙，那么就先使用它，然后在付费前取消试用。这绝对是一件非常容易的事。

    但是，如果您这样做， 请记住务必取消试用 ，否则必须付费！您可以使用诸如 “ Privacy ” 之类的虚拟支付服务来确保您用于支付的卡仅涵盖试用，不会向您收取该服务的费用。

    最后

    就是这样！

    以上方法会针对不同的墙有效，而不是所有方法对每一面墙都有效。所以，本文是一个手册，您可以在不同情况下尝试其中不同的方法。

    ⚠️ 请注意：不要太狠！ 有些内容提供者 —— 尤其是 独立的替代媒体 ，就如IYP这样 —— 他们活得非常艰难，他们为了能和撒谎的企业媒体战斗，几乎付出了一切。他们需要你的支持才能坚持下去。所以，绕过付费墙的方法请针对那些本来就很肥的企业媒体，他们不缺这点钱。⚪️
