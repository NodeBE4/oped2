    如何撤销 Facebook、Twitter 和 Google 上的第三方访问权限

    McCaffrey Knowledge Node , 技术防身/自我保护方法 2020年5月6日 2019年11月29日 0 Minutes
    • 基础知识：如果您离不开这些寡头服务的话，需要做到尽可能保护自己 —— 这里是您需要定期检查的东西

    重申：这不是一次性工作，是需要您定期检查的。

    如果您懒得使用电子邮件地址注册服务，则可以使用社交媒体或Google帐户注册这些服务。那些按钮就摆在那里，基本所有人都知道它们。

    然后，这些网站与 Facebook、Google、Twitter 等寡头服务联系，授予您身份验证令牌，同时提供你的基本详细信息，例如你的姓名、照片和电子邮件地址等等。

    问题在于，一旦您通过门户将社交媒体帐户授予第三方网站就可能带来风险。如果该网站有不良意图怎么办？如果他们尝试获取更多的用户信息（例如密码）怎么办？

    这就是为什么您必须定期检查哪些网站有权访问您的社交媒体帐户或Google帐户的原因。

    如果您不再使用该网站，请立即撤消其访问权限。以下是具体方法。

    Facebook

    登录 Facebook 后， 转到此链接 。在这里，您将看到使用 Facebook 凭据注册并登录的所有各个站点。

    您可能需要先判断，然后再撤销，以查看是否可以解决问题。例如，如果该应用在您的 Facebook 墙上留下了不需要的状态更新，则您可以更改隐私设置，以便没人会看到它。

    如果您决定撤销该访问权限是唯一的答案，那就选中在其旁边是一个小复选框。

    现在，“删除”按钮将亮起。单击它会要求您确认您确实要删除该应用程序。您还可以告诉 Facebook 删除该应用可能已发布在 Facebook 上的所有帖子、照片或视频。

    再次单击“删除”将撤消该应用访问您帐户的权利，您将得到确认。

    Twitter

    登录您的 Twitter 帐户，然后在“设置”中 转到此处 。 “Apps and Sessions” 标签是所有操作的地方 —— 用于查看谁有权访问您的 Twitter 详细信息。

    自几个月前的清理以来，我的 Twitter 帐户中又积累了多少。也是为什么说您应该定期做这项工作。

    只需单击要撤消的站点，然后就像 Facebook 那样，您将看到该网站的详细信息。

    点击底部的“撤消访问权限”即可。

    如果您突然改变主意，Twitter 不会像 Facebook 那样要求您确认，而是将撤消链接转换为 “撤消撤消访问权限” 链接。只需单击该按钮，即可将一切恢复原状。

    返回您的“应用程序”页面，重复这套操作以删除您要删除的其他第三方。

    Google

    Google 要求您登录帐户，您可以 在此处 或通过 Gmail 设置进行访问。

    向下滚动直到看到下面这个框：

    点击底部的“管理第三方访问权限”以正确打开它。

    自上次清理以来，我这里又多了30多个应用程序，因此，马上清理。

    然后：

    点击蓝色的“撤消访问权限”按钮，系统会要求您确认一下。

    单击确定后，该应用程序将消失。继续执行相同的过程删除所有第三方。

    显然，您可以使用更多服务登录网站。以 WordPress 为例，苹果公司最近宣布他们也将很快提供“使用您的 Apple ID 登录”功能。也就是说，您还要注意苹果。

    但目前，您需要不断关注 Facebook、Twitter 和 Google，这仨是最厉害的，几乎无处不在的。⚪️
