    这里是亚马逊的 big brother 技术，你应该能感受到它的可怕

    McCaffrey Knowledge Node , 关于谷歌和亚马逊 , 监视与操纵 , 监视资本主义 2018年5月31日 2019年4月28日 0 Minutes

    位于华盛顿州西雅图的电子商务公司 Amazon.com 是美国零售业巨头。2016 年，亚马逊的电子商务平台占当年全国在线零售总额的 43％。根据在线统计门户 Statista 的数据，2017 年，亚马逊在美国的电子商务净销售额达到 1780 亿美元。四月中旬，亚马逊首席执行官贝佐斯在年度致股东信中宣布，该公司的 Prime 会员服务已超过1亿全球用户。

    是全球，请注意，不仅仅是美国。这些跨国巨头的长臂祸害的可不止一两个国家。

    它已经发育得太大了，难怪这家科技巨头能够获得大量不考虑自己私人数据安全的用户。

    尽管大多数实体零售商只能获得信用卡信息，或者通过消费者忠诚度计划获取一些个人信息，但对亚马逊隐私政策的分析表明，该公司收集的数据包括：收货人、客户 IP 地址、用户个人资料中的个人信息甚至客户的朋友的电子邮件地址等等。亚马逊以各种方式利用这些数据来增加平台的价值，包括产品推荐以及供应链和产品价格的优化。但只有这些吗？绝对不是。

    剑桥分析公司丑闻后，数据隐私问题一直处于最前沿位置，看起来亚马逊在对其平台上的消费者数据安全性方面并不完全清晰。最近的报告显示，亚马逊的 Mechanical Turk 在线市场是由 Facebook 丑闻背后的剑桥分析公司应用程序开发商 Aleksandr Kogan 开发的数据收集检测应用程序的另一个使用平台。数据隐私问题也出现在亚马逊网络云服务平台周围，包括私人公司和五角大楼在内的老大哥们在亚马逊云服务上存储网络监控数据。

    亚马逊服务器还使用其 Alexa 数字个人助理服务收集消费者的语音录音，除非消费者手动删除录音，否则这些录音很有可能落入坏人手中。

    使用 IP 地址查明用户的地理位置

    亚马逊开发了一种技术手段来跟踪消费者的实际位置，是 美国专利号 7937336 所公开的：标题为 Predicting Geographic Location Associated with Network Address。2011 年5月发布。

    它涵盖一个计算机系统，该系统包含一个数据存储器，该存储器包含一个决策树，该决策树被构建为产生与用户相关的 IP 地址的预测地理位置以及计算装置，其获得包括 IP 地址和用户简档信息的第一组样本数据；用第一组样本数据训练决策树；获得包括与 IP 地址相关的地理位置信息的第二组样本数据；用第二组数据修剪训练过的决策树；然后用第三组样本数据来限定决策树，以向每个叶节点分配一个置信度分数指示预测的地理位置的准确度。

    根据该专利所释，这项发明中的决策树是通过机器学习构建的，以确定预测属性，允许网络服务针对一组潜在用户进行各种活动。但是，消费者应该知道，该技术使亚马逊能够确定“身份未被网络服务识别的潜在用户的用户属性”，包括与用户设备的地理位置相关的属性、网络服务提供商（ISP）信息、甚至与地理位置相关的家庭收入状况。

    如果这些数据落在警方手里，你知道会发生什么。亚马逊等这些巨头一直在和情报机构合作，这是 Snowden 所揭露的所有关键信息之一。帮执法者和税务官员确认比特币用户的位置

    比特币等加密货币的卖点之一就是数字货币提供的匿名性，但是由于 美国专利号 9947033 所描述的名为 Streaming Data Marketplace 的技术，在亚马逊的平台上，你完全无法保证匿名性！

    该技术在今年 4 月份发布，它声称一种被配置为便于组织流式传输数据的计算设备，该设备具有一个处理器；与处理器电子通信的存储器；存储在存储器中的指令可执行以经由流传输数据服务的第一来源请求的第一数据流；确定从第一数据流接收的数据量；接收来自数据流的数据；通过流传输数据服务识别来自包括不同类型数据的第二来源流传输数据；关联和组合数据流以增加第一数据流的效用；并将组合数据作为组合数据流发布到流数据服务。

    该发明旨在使流式分析服务能够开发用于处理每小时数百 TB 数据的业务应用，包括网站点击流、社交媒体和操作日志。然而，该专利还描述了这种技术的示例性使用，涉及运输地址与比特币交易的相关性，政府机构可以使用它来“关联税收交易数据以帮助识别交易参与者”。它还考虑到了使用该技术的执法机构能将全球比特币交易与送货地址相关联！

    再次提醒：是全球。它毁掉了比特币最大的价值。使用移动设备数据预测消费者的未来位置

    你在到达某一地点之前亚马逊就知道你要去哪里了，因为它有个专利：名为 “移动用户的预测位置”， 美国专利号 9123014 ，发布于2015年9月。

    它涵盖体现可在计算设备中执行的程序的非暂时性计算机可读介质，包括从移动设备用户接收包裹递送服务注册的代码；通知注册的包裹递送服务的代码；接收移动用户的最近位置的更新代码；接收更新到移动用户最近位置的订阅请求的代码；通知用户最近位置的代码；部分基于最近位置生成的用户未来位置预测的代码；以及基于未来移动用户的预测位置来启动递送的代码。

    该发明旨在“改进包裹递送并减少当包裹被发送到达某个位置但预期的接收者不在那里时可能发生的延迟”。但你必须知道，在你达到某地之前亚马逊就能知道你要去那里，这件事特别令人感觉不安。

    如果你在逃亡路上，或者未来准备逃亡，请千万不要使用包括亚马逊在内的任何一种网络电商或服务。否则基本可以确定，你是逃不掉的。基于生命里程碑的用户购买行为预测

    亚马逊利用你的重要生活事件数据来销售产品，正如 美国专利号 9369536 中所声称的技术所证明的那样，标题为 Event-Based User Behavior Timeline, Predictions, and Recommendations。

    它涵盖由处理器可执行的数据存储中的模块系统，包括确定用户重要生活事件的时间表模块，如生日、假期、住宿变更、孩子出生、教育里程碑或周年纪念；预测模块，基于来自所述时间线模块的关于用户行为预测标准的结构分析，来确定对用户行为的预测，所述用户行为预测标准具有跟随重要生活事件发生的趋势；推荐模块，基于用户行为的预测向用户提供项目推荐。

    该项发明利用社交网络活动、用户通信和人口统计数据的情感分析来改进产品推荐，例如当人们改变居住地时向他们提供购买主要电器的建议。然而，大多数亚马逊客户可能不知道该公司为其客户维护活动时间线。

    你的生活重要事件就是你的情绪规律，也就是你的弱点。这些数据如果被你的敌人所掌握，他们就可以利用它来操纵你追踪卡路里摄取量和订餐历史，以预测膳食支付

    亚马逊还从消费者那里收集外卖和递送订单信息，该技术 美国专利号 9898788 ，标题为 Predictive Restaurant Ordering。

    它涵盖包括存储可执行指令的非暂时性计算机可读介质的自动化食品服务订购系统，以获得与一个或多个客户相关联的过去状态上下文数据指示；获得历史菜单项目订购数据；从个人电子设备获得指示用户的当前、最近或既定状态的情境数据，包括卡路里摄取或锻炼活动；基于所获得的数据确定在未来什么时候传送菜单项；从多个食品服务中选择未来最有可能被选中的食品服务；提前确定订单；至少在交付时间内启动未来食品服务订单。

    亚马逊甚至在专利中解释为“人们在饥饿的时候才想起下单，但食品会在一段时间后才能到达”……听起来对你有用是吧？是为你着想？并不是，你应该感到惊讶，亚马逊追踪的消费者数据甚至已经延伸到外卖食品订单了，而不仅仅是 Amazon.com 平台上销售的书籍和电子设备。

    再次重申本网的一贯警告：如果您在乎自己的隐私，请不要让这些陌生人太过了解你 —— 不要参与任何一种声称的网络测试游戏、不要上传自己的健康监测数据运动数据在互联网上、出门用现金购买所需的东西。否则将导致您在任何时候都有可能被具有预期目的的人所利用。
