    6 个最好的开源桌面电子邮件客户端

    McCaffrey Knowledge Node , 技术防身/自我保护方法 2018年6月13日 2019年8月10日 0 Minutes

    对本机应用程序的偏好、易于离线使用、大量插件、跨平台的便利，以及足够的安全性需求，开源桌面电子邮件客户端都能更好的满足您。这里推荐一些相比下较好的

    Thunderbird

    Thunderbird 最初是由 Mozilla 开发的开源电子邮件客户端，它是跨平台的，具有为用户提供高速、保密性访问电子邮件服务的最新技术。

    在 2017 年中期，该项目宣布将取消 Mozilla 的基础设施，但将 Mozilla 基金会作为其法律和财政之家。为了推进该项目，已经招聘了几名新员工，并计划引入新的开发人员，并将代码库转换为基于 Web 技术。

    Thunderbird 功能齐全，有许多很好的插件，可添加从日历支持到高级地址簿集成等所有内容，以及许多专业功能，包括主题和大文件管理。

    主要功能 如下：

    • 使用户能够拥有个性化的电子邮件地址
    • 一个单击通讯录
    • 附件提醒
    • 多频道聊天
    • 标签和搜索
    • 启用搜索网络
    • 快速过滤器工具栏
    • 消息存档
    • 大文件管理
    • 安全功能，如网络钓鱼保护，无跟踪
    • 自动更新等等

    Claws Mail

    Claws 邮件是一个基于 GTK + 的用户友好、轻量级和快速的电子邮件客户端，它还包括新闻阅读器功能。 有一个优雅和复杂的用户界面，也支持类似于其他电子邮件客户端的面向键盘的操作。

    主要功能 包括：

    • 高可插拔
    • 支持多个电子邮件帐户
    • 支持消息过滤
    • 颜色标签
    • 高度可扩展
    • 外部编辑器
    • 用户定义的标题
    • Mime附件
    • 管理 MH 格式的消息，提供快速访问和数据安全性
    • 导入和导出来自其他电子邮件客户端的内容

    Evolution

    它最初是为企业开发的。Evolution 支持 Exchange Server 以及您可能在企业环境中找到的许多其他设置。除了处理您的电子邮件之外，它还是一个完整的个人信息管理器（PIM），包括日历、任务列表、联系人管理器和笔记应用程序。

    如果您对垃圾邮件过滤、GNU Privacy Guard（GPG）支持，与 LibreOffice 的集成或一系列其他功能感兴趣，您可以考虑使用它。

    一些基本的电子邮件管理功能以及高级功能如下：

    • 帐户管理
    • 更改邮件窗口布局
    • 删除和取消删除邮件
    • 排序和组织邮件
    • 快捷键用于阅读邮件的功能
    • 邮件加密和证书
    • 通过邮件发送邀请
    • 自动完成电子邮件地址
    • 消息转发
    • 拼写检查
    • 使用电子邮件签名
    • 离线工作以及许多其他功能

    Geary

    Geary 是最初由 Yorba Foundation 开发的项目，它创建了许多不同的 GNOME 软件工具。 如果你正在寻找一个简单而高效的电子邮件客户端提供基本的功能，那么 Geary 可以是一个不错的选择。

    有以下特点：

    • 支持常见的电子邮件服务提供商，如 Gmail，Yahoo！Mail，以及许多流行的 IMAP 服务器
    • 简单，现代和直接的界面
    • 快速帐户设置
    • 快速关键字搜索
    • 全功能的 HTML 邮件编辑器
    • 桌面通知支持

    KMail

    KMail 是 Kontact 的邮件组件，Kontact 是 KDE 附带的个人信息管理器。KMail 支持各种电子邮件协议，包括 IMAP，SMTP 和 POP3，通过与其他 Kontact 组件集成，它可以被视为一个完整的组件套件。尽管有 Linux 路由，但也可以使用 Windows 版本。

    KMail 主要功能包括：

    • 支持标准邮​​件协议，如 SMTP，IMAP 和 POP3
    • 支持纯文本和安全登录
    • 读取和写入 HTML 邮件
    • 与垃圾邮件检查器（如 Bogofilter，SpamAssassin 等）的集成
    • 支持接收和接受邀请
    • 强大的搜索和过滤功能
    • 加密的密码保存在 KWallet 中
    • 备份支持
    • 与其他 Kontact 组件完全集成，还有更多

    Sylpheed

    Sylpheed 是一个简单，轻便，易于使用，跨平台的电子邮件客户端，是有特色，它可以在 Linux，Windows，Mac OS X 和其他类 Unix 操作系统上运行。

    它提供了一个直观的用户界面，面向键盘使用。 适用于新用户和高级用户。

    具有以下功能：

    • 简单，美观，易于使用的界面
    • 轻量级操作
    • 可插拔
    • 组织良好，易于理解的配置
    • 垃圾邮件控制
    • 支持各种协议
    • 强大的搜索和过滤功能
    • 与外部命令的灵活合作
    • 安全功能，如 GnuPG，SSL / TLSv
    • 高级日语处理，以及更多
