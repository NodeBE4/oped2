    Facebook 全球游说反对数据隐私法

    McCaffrey Newsletters , 关于 Facebook , 监视资本主义 2019年5月28日 2019年4月27日 0 Minutes
    • 为了反对隐私立法，一掷千金地全球游说、收买政客，连女权主义著作都被用来作为游说工具了，只有这一家公司这么不要脸吗？

    Facebook 已经针对世界各地的政界人士 — 包括前英国财政大臣乔治奥斯本 — 承诺投资和激励措施，主要目的是寻求迫使他们代表 Facebook 反对数据隐私立法，相关内部 Facebook 文件的最新泄漏揭露了这一惊人的故事。

    “观察家报”和“计算机周刊”发现的 这些文件揭示了一项秘密的全球游说活动，该游说针对数百名立法者和监管机构，试图在全球范围内产生影响，包括在英国，美国，加拿大，印度，越南，阿根廷，巴西，马来西亚和欧盟的所有28个国家 。这些文件包括的重点信息是：

    – 欧洲各地的政界人士在战略行动中应努力阻止“过度限制”的 GDPR 立法。其中包括爱尔兰总理表示他的国家可以发挥重大影响力推行特殊主张，以促进 Facebook 的利益，尽管技术上它应该保持中立。

    – 如果各国不能支持或通过对 Facebook 友好的法律，Facebook 将拒绝他们的投资。

    这些文件似乎来自加利福尼亚州应用程序开发商 Six4Three 针对 Facebook 的法庭案件，并透露桑德伯格认为欧洲的数据保护立法对公司构成“严重”威胁。2013年达沃斯经济峰会后发表的一份备忘录引用桑德伯格的话描述了该公司在“数据和隐私方面”面临的来自欧洲的“艰难战斗”及其“关键”努力，以阻止“过度规范的新法律”。

    最具启发性的是，它包含了该公司与当时爱尔兰总理恩达·肯尼（ Enda Kenny ）的“良好关系”的详细信息，此人被称为“Facebook 的朋友”。

    爱尔兰在监管欧洲科技公司方面发挥着关键作用，因为其数据保护专员代表所有28个成员国。这份备忘录激起了数据保护倡导者的兴趣，他们一直抱怨该公司与爱尔兰政府的“亲密”关系。

    该备忘录指出 Kenny 对 Facebook 决定将其总部设在都柏林的“赞赏”，并指出新提议的欧洲数据保护立法是“对欧洲就业、创新和经济增长的威胁”。然后继续说爱尔兰准备担任欧盟轮值主席国，因此“有机会影响欧洲数据指令的决定”。 该备忘录特别声称 Kenny 提出利用欧盟轮值主席国的“重大影响力”作为影响其他欧盟成员国的手段 ，“尽管技术上爱尔兰应该在这个角色中保持中立”。

    备忘录说：“总理承诺利用他们的欧盟轮值主席国权力在指令上取得积极成果。”Kenny 在2017年辞职，没有回应观察家饱的评论请求。

    研究数字技术民主影响的剑桥学术和观察家报作家 John Naughton 表示，泄露事件对于揭露爱尔兰国家作为大型科技公司的附庸的方式是“爆炸性的”。他指出，爱尔兰曾对这些公司表示欢迎；“ 该国主要政客们显然认为自己是数据监视资本主义的隐蔽说客 。”

    Facebook 发言人表示，这些文件仍然在加利福尼亚法院封闭，并且无法对其进行任何详细回复，并称：“与去年违反法院命令的其他文件一样，这只是故事的一面，省略了重要的背景。“

    2013年备忘录由 Marne Levine 撰写，他现已成为 Facebook 的高级管理人员，Facebook 当时的政策和全球通信负责人是 Elliot Schrage，现在由 Nick Clegg 担任这一职务。

    除了 Kenny 之外，其他几十位政治家，包括美国参议员和欧洲委员也都被提及，其中还包括印度13任总统普拉纳布·慕克吉、现任欧盟脱欧谈判代表的米歇尔·巴尼耶和奥斯本 。

    备忘录声称，当时的财政大臣利用与桑德伯格的会面要求 Facebook 投资政府的“科技城”，而桑德伯格表示她将“审查”任何提案。作为交换，她要求他“在欧洲数据指令辩论中更加积极主动，并真正帮助制定[有利于 Facebook 的]提案”。

    该备忘录显示奥斯本要求进行详细的通报，并说他将“弄清楚如何更多地参与”。他提议在唐宁街为 Sandberg 的书举办一场推广活动，这一活动于2013年春季开始。

    奥斯本告诉 Observer：“我认为这位英国财政大臣与全球最大公司之一的首席运营官见面并不令人感到意外……Facebook 和其他美国科技公司已经私下表达了对拟议的欧洲数据指令的担忧。对于您的具体询问，我没有跟进这些问题，也没有游说欧盟，因为我不同意这些。“

    他指出，他曾帮助在唐宁街11号推广桑德伯格的书“并非秘密”，并补充道：“这本书关于女性赋权的信息受到广泛赞誉，尤其是”卫报“和”观察家报“。

    事实上， 该备忘录显示，桑德伯格的女权主义回忆录被 Facebook 团队用作了一种游说工具，并且是一种赢得女性立法者支持 Facebook 的更广泛议程的手段 。

    该备忘录特别透露了与在欧洲有影响力的欧洲司法、基本权利和公民身份委员 Viviane Reding 的会面，并指出她作为“欧洲数据指令的架构师”的关键角色，并描述了该公司与她的“困难”关系。由于她的存在，该备忘录称，“不是美国公司的粉丝”。

    “她参加了 Sheryl 的晚宴，之后我们就与她会面了，”这份备忘录说，但她觉得这是一场“非常’美国式的’讨论”，该团队认为这是一个挫折。

    就 Facebook 赢得影响力的全球努力而言，达沃斯会议只是冰山一角 。这些文件还揭示了加拿大和马来西亚如何使用新的数据中心选择的承诺，声称创造就业机会以赢得立法保障。当加拿大人对获得 Facebook 想要的让步犹豫不决时，备忘录指出：“Sheryl 采取了坚定的方法并概述了数据中心的决定迫在眉睫。她强调，如果我们无法从加拿大政府那里获得关于管辖权问题的安慰，我们还有其他选择。“ 该部长最终在当天结束时提供了 Facebook 所要求的协议。
