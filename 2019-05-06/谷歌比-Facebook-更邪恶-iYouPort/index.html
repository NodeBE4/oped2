    谷歌比 Facebook 更邪恶

    McCaffrey Newsletters , 关于谷歌和亚马逊 , 监视资本主义 2019年5月7日 2019年5月7日 0 Minutes
    • 你喜欢圆形监狱吗？欢迎加入啊～ 这就是为什么谷歌比 Facebook 更邪恶，因为它可以让你”愿意”变成无知的小白鼠。他们是怎么做的？

    想象一下：一家巨头科技公司正在跟踪您在手机上的一切操作，即使您没有使用任何服务，他们也可以跟踪您看到的特定图像；它还会跟踪您的所有网络流量，因为您安装了一个专门设计的路由器；即使某些流量已加密，但由于DNS解析的工作原理，它仍然可以知道您访问的网站都是什么；哦，它还可以录制放置在电视附近的自定义麦克风的音频，并分析它听到的所有内容。

    不，我们不是在谈论 Facebook，我们已经详细分析了很多 Facebook 的狡猾、隐私侵犯和破坏安全的所谓“用户研究”计划的可怕后果。这里说的是谷歌的 “ScreenWise Meter”，另一个伪装的“研究计划”，就像 Facebook 一样，近期被曝光时引发了舆论爆炸。

    为了监视 iOS 用户，Facebook 绕过了 Apple 严格的应用程序分发规则；而谷歌在此做的事完全一样。苹果公司迅速 撤销了 这两个巨头的企业证书，关闭了所有 Facebook 和谷歌的内部iOS应用程序和工具。

    我们绝不是 Apple 围墙花园的粉丝，它严格控制谁能在其平台上玩谁不能；然而，这场大戏引发了对用户的欺骗性信息以及围绕两个全景监视器的数据收集实践的严重关注。

    尽管谷歌退出了iOS应用程序，但其 Screenwise Meter 监控程序的所有其他部分仍处于运行状态 — 在某些情况下，他们收集的关于“被研究用户”的数据甚至比 Facebook 弄到的更多。

    在某些方面，谷歌的监视项目并不像 Facebook 那样糟糕，而在其他方面，谷歌更糟糕。“不太糟糕”的部分在于：它不是直接针对青少年的，没有偷偷地隐藏自己的目的，它没有要求用户安装自定义根证书，谷歌反乌托邦盈利模式本身使得它很清楚公司正在做什么；

    “更糟糕”的部分：它要求你进入一个圆形监狱。虽然一直以来，谷歌在所有人的线上和线下生活中都有很强大的跟踪监视能力，但是，Screenwise 项目依旧能在此之上迈出了一大步。

    Screenwise Meter 移动应用程序和 Web 扩展程序基本上允许 Google 查看您在手机屏幕和Web浏览器窗口中看到的内容。该应用程序可以通过在智能手机上侧载“自定义”应用程序来监控您的所有应用使用情况和网络流量。由于谷歌没有要求你安装像 Facebook 这样的根证书，他们无法解密 HTTPS 流量，但是，应用程序可以在你的屏幕上看到任何内容，详见其隐私政策的 “屏幕上内容”部分 。

    假设您打开 Snapchat 应用程序，谷歌可以看到这一点；假设您需要输入密码，谷歌也可以看到这一点；假设您发送了一条消息给朋友，是的，谷歌也可以看到。

    其网络扩展甚至超出了 Facebook 愿意做的跟踪水平。Facebook 显然能够跟踪80％的互联网流量，但谷歌认为这是不够的：即使是通过 HTTPS，扩展程序也会将您的所有网络浏览报告回 Google。

    它还可以收集您在任何网站上的每一个动作（从撰写私人消息到浏览购物网站），以及存储或保存在浏览器中的任何信息。⚠️ 谷歌 甚至承认 通过这个计划收集社会安全号码和信用卡号码，尽管它声称这些“不是监控的焦点”。

    此外，Screenwise 通过 自定义路由器 侵入您的私人生活空间。它无法拦截HTTPS流量，但由于DNS查找目前尚未加密，因此 Google 可以记录任何人在连接到 WiFi 时访问的每个网站。当然，它也可以在家庭WiFi上看到任何未加密的应用和网络流量。

    最重要的是，有一个 “TV Meter” ，它是一个永远在线的麦克风，可以收集和发送来自电视的Google音频并且可以拾取任何附近发出的对话 — — 窃听你的起居室。

    虽然谷歌对其计划的解释比 Facebook 更为明确，但对于许多人来说，如果他们没有阅读所有冗长的隐私政策的耐心，他们就不会知道自己正在如何被谷歌监视。

    这些设备不仅仅是监视一个人 — 它们在监整个家庭，这可能涉及不太可能知道监视存在的客人和13岁以下的儿童。是的，谷歌“禁止”13岁以下的儿童参与这种侵入式数字跟踪，提供在孩子参与或客人来访时暂停跟踪的选项，并要求用户通知任何住宿客人有关监控的信息。但 实际上，这是在为公司提供保障，而不是保护您的孩子或客人 — — Screenwise 只是将责任转移到受监视的用户身上 — — 这完全是在宽松的隐私法律下允许的行为，并且需要在新的隐私法律下进行更改的部分，所以它目前是灰色地带，这就是谷歌的狡猾 。

    最后，Google 的所有声明都没有明确说明它与谁共享了所有这些数据。在其隐私政策的最后，谷歌提到它可以与“受信任的企业”共享所有这些收集到的信息，而不会暗示这些信息的获取者可能是谁、或他们可能对我们的数据做些什么。

    Screenwise 不是唯一的问题。 一项新的研究详细介绍 了 Google 一直以来如何欺骗普通用户“选择”接受跟踪。

    随着时间的推移，越来越清楚的是，我们不能依赖公司的“道德”和“价值体系”来判断自己的“同意”。它们从来没有道德。充满行话的对话框、精美的打印页面和隐藏的隐私政策都是骗局。当通过收集和销售我们的数据来推动利润时，这些公司眼里只有钱。
