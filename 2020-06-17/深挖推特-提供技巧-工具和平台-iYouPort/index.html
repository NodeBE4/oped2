    它们可以组合使用。别忘了这里介绍的技巧：《 从推特中挖掘真相不需要太复杂的工具：一个常用工具的全面指南 》。

    cybersec 带有 “cybersec” 一词的推文。

    security -information 带有 “security” 这个词的推文，但没有 “information” 这个词。

    HongKong lang:en “HongKong” 相关推文，仅限英文。您可以换任何语种，使用 fr，nl，ar，bg，cs 等。

    filter:verified elections 经过验证的帐户发布有关选举的信息。

    from:username 你想要的目标用户发送的推文。

    to:username -filter:links 发送给目标用户的推文，没有链接。

    hongkong filter:retweets 旨在显示 hongkong 相关内容的转推。

    hongkong exclude:retweets 过滤掉转推的 hongkong 推文。

    filter:replies cybersecurity 回复给 cybersecurity 推文的文本。

    exclude:replies cybersecurity 没有人回复的 Cybersecurity 推文。

    security list:TIME/time-staff security lists 中的推特账户。

    security min_retweets:10 Security 推文被转发了至少10次或更多。

    security min_faves:10 Security 推文被 点赞/收藏 10次或更多。

    conference near:cityname 列出您输入的城市附近举行的会议/研讨会。

    security near:LA within:10mi 来自以 Los Angelos 为中心10英里半径内的任何人的有关 Security 的推文。可以更改城市名（美国，2个大写字母）和英里半径数字。

    security reviews since:2019–01–01 until:2020–05–01 从2019年1月1日到2020年5月01日内发布的有关 Security 的推文。

    hacking filter:links hacking 主题的内容带有关于该主题的链接。

    hacking filter:images 有关 hacking 主题的图片。可添加标签以限定搜索。

    hacking filter:news 带有新闻来源的 Hacking 主题推文。

    videos filter:safe 标记为 safe 的视频内容。

    images filter:safe 标记为 safe 的图片内容。

    cybersec filter:media 关于 cybersec 的媒体推文。
