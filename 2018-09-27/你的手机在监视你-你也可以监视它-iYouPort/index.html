    你的手机在监视你，你也可以监视它

    McCaffrey Knowledge Node , 技术防身/自我保护方法 , 给行动者的护身符和武器 2018年9月27日 2019年4月27日 0 Minutes
    • 只要你开通了智能手机，不管你有没有让它这么做，它都可以和各种各样的公司进行沟通，其中许多公司你都不会知道他们是谁，以及会把你怎么样……要解决这个问题的方法是首先我们应该了解自己都被拿走了什么

    你的手机和其他个人电子设备非常了解你。它们知道并且经常分享这些信息，例如你在哪里、你要去哪里、你是谁的朋友、你使用什么应用程序、你访问哪些网站、你多久访问一次、你给谁发电子邮件、你打电话给谁，等等等等。

    所以要问一下，你知道自己的应用程序和设备正在分享哪些关于你的信息吗？和谁分享？我曾经以为我知道，但现在我不太确定了。

    随着 被揭露的信息越来越多地出现 ，我对自己所知道的——主要是关于我的哪些数据正在被记录下来，以及我的应用程序和设备正在分享哪些信息——的信心在缓慢但毫无疑问地下降。

    随着越来越怀疑，我决定要确定我的设备实际上向外分享了什么。不只是觉得这是个不错的主意，而是真的去了解。所以我想出了一个计划来寻找答案，并且希望能获得大家的帮助。

    计划是这样的

    在大约一周的时间里，我将截取并记录从我最私人的两个与互联网连接的设备发送的每一点数据，这两个设备分别是我的手机和笔记本电脑。它将包括发送给世界各地公司的所有形式的个人信息。

    如果你对这事感兴趣，你可以阅读以下关于具体如何操作的技术细节。

    最后，我会得到类似这样的东西。

    记录了数据之后就是仔细研究，看看我能找到什么。你也将有机会看到其中的一些数据（抱歉，不包括我的银行密码）。

    为什么我们要关心这些？

    大多数人永远都看不到他们手机数据被拿走了什么。要查起来并不容易，因为需要专业软件和一些技术知识。

    于是，很多人只能假装信任科技公司不会做什么手脚。

    而如今，这种信任越来越多地遭到了被背叛。最近几年，大大小小的公司一再背叛我们对它们的信任，比如， 即使你关掉了谷歌的定位系统，谷歌也一直在对你跟踪定位 。

    • 延伸阅读：《 可以采取哪些措施来阻止 Google 和其他人跟踪您的手机？ 》

    你想知道什么？

    在我已经看到的所有设置的数据中，已经有了一些发现。例如，你知道吗， 每次你在 iPhone 上打开 Safari 时，书签中的任何网站都可以跟踪它？

    尽管这样做是为了评估我们在智能手机、社交媒体和物联网时代的个人隐私，我还是真的很想与你交流一下。

    你想知道什么？你对你的设备分享关于你的信息有什么担心吗？ 我想我会有以下发现：

    • 大量询问 ：我想会有大量数据，我认为只有少数的询问是由我明确提出的。
    • 大公司为主 ：我认为许多这些询问的信息将会与 Facebook、Google、Twitter、Apple 和 Microsoft 等大公司分享数据。
    • 无法跟踪的询问 ：也将有大量数据发送给身份更加模糊的公司，其中许多公司可能难以识别。我预计这甚至对我来说也很困难，因为我是与这些数据相关的人。
    • 一些公司会对我的 网页浏览习惯有惊人的完整了解 。

    技术细节 如果你是技术控，以下参见如何操作的简短版本。事实上非常简单：

    • 创建一个VPN，在整个数据收集期间用它来访问互联网。
    • 这个 VPN 将通过运行 MITM 代理的独立虚拟机路由所有数据，MITM 代理是用于拦截、测试和分析 HTTP 请求的软件。
    • 几乎每个请求(包括 HTTPS )的内容和其他请求数据都会被记录下来。

    具体的设置方式 在 GitHub 上有更多的细节 。如果你真的感兴趣，可以自己试试。◾️
