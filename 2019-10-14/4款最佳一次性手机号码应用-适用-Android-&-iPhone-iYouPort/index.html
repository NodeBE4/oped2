    4款最佳一次性手机号码应用：适用 Android & iPhone

    McCaffrey Knowledge Node , 技术防身/自我保护方法 , 给行动者的护身符和武器 2019年10月15日 2019年9月23日 1 Minute
    • 真实电话号码是非常危险的，能挖出一大堆有关您的私密信息。我们曾经演示过如何做这件事。随着越来越多的应用和订阅需要您的电话号码，隐私被剥夺的危机已经非常深刻。这种情况下怎么办？

    本文提供一些解决方案，您可以选择适合自己的。尤其是，⚠️ 如果您的手机屏幕被偷窥了、或者更糟糕的是手机被警察抢走了，您也能隐藏自己的数据、以及您正在隐藏的事实。

    一次性手机号码不仅仅适合犯罪分子或者婚内出轨，也适合 您这样的希望避免被专制政权和犯罪分子骚扰的任何人 。

    在这个数字世界中，我们的许多互动都是在线进行的，一次性电话号码就像一个缓冲区 —— 它可以让您在不必提供个人信息（如真实姓名或电话号码）的情况下享受网络。

    在这里看到电话号码有多可怕《 如果只知道一个电话号码，你能挖出多少有效信息 ？》

    使用一次性手机号码应用程序，您可以创建临时号码，这些号码不易追溯到您本人。

    与您的真实号码不同，一次性电话应用程序不会在来电显示上显示出您的姓名，并且可以随时删除。

    在两个应用程序商店中测试了几个应用程序之后，发现有四个应用程序是最好的，可以提供基本要素和工具来增强体验。

    关键比较点

    Play 商店和 App Store 上有许多应用程序可以让您创建第二个号码，但大多数都不能很好地模拟一次性电话。

    真实世界中的一次性电话通常是一次性预付费电话，人们可以暂时使用并最终在被追踪号码时处理掉。

    具有同样功能的应用程序应该在其他方案中保持您的通信私密性，例如管理多个身份角色（见这里《 角色扮演技巧 》）。

    比较一下：

    功能列表：

    最便宜的预付计划：您可以选择最低价格购买，功能包括电话和短信。

    无限计划：呼叫或短信不受限制。

    高级功能价格：激活应用中其他功能的成本，例如移除广告或激活语音邮件。其他费用标准无法提供的功能。

    免费试用：为新用户提供的免费试用期。

    图片消息：应用程序支持MMS，允许用户从其一次性号码发送图片消息。

    提醒：能够为之前收到的通知创建新提醒，以提醒您稍后处理该消息。

    请勿打扰：该应用程序包含自己的“请勿打扰”模式，允许您禁用涉及一次性号码的通知和呼叫。

    呼叫时长：默认情况下，应用程序的呼叫将使用您的运营商号码的分钟数。

    通过运营商号码路由呼叫：当在数据接收较差的区域（LTE或Wi-Fi信号较差）时，可以通过您的运营商号码路由呼叫。不过请注意。 这将导致您的电话帐单上显示您的一次性手机号 。

    语音邮件：该应用程序支持单独的语音邮件，用于涉及第二个号码的呼叫。

    第三方应用程序集成：允许用户将其他帐户链接到该号码，以添加新功能并控制涉及一次性号码的通信。集成的一些示例包括将语音邮件保存到 Dropbox 并使用 Slack 发送和接收文本消息。

    语音控制：使用Siri等语音助手管理一次性号码的功能。

    视频通话：您可以使用这个号码进行视频通话。只有 TextMe Up 包含此功能，但它需要通话双方都使用该应用程序。

    呼叫转移：将呼叫重定向到您选择的其他号码的能力。这对于 多个一次性号码 来说非常有用，因为您可以将每个号码的呼叫定向到一个号码，从而使其更 易于管理 。

    接受比特币：可以使用比特币进行应用内购买 ，例如预付费计划。虽然不是完全匿名的，但多数情况下不容易被追踪。

    匿名注册：可以在不使用您的电话号码或电子邮件地址的情况下创建帐户。

    加密消息：该应用程序支持其用户之间的端到端加密消息。

    自毁消息：在此应用程序的用户之间进行传递的消息可以设置定时器。届时消息将从所有设备中删除。

    💡 能够伪装应用程序：该应用程序具有内置功能，允许它掩盖其在您的设备上的外观。 CoverMe 是本列表中唯一具有此功能的应用程序，它允许用户将应用程序的外观更改为新闻阅读器 。

    内置应用程序锁定 ：应用程序中包含创建PIN或密码以锁定未经授权的用户的功能。此外，所有应用程序都支持Touch ID以方便使用。

    方法

    虽然许多应用程序可以提供第二个电话号码，但本文只选择提供 难以追溯到您的临时号码的应用程序 。

    传统的一次性手机可以抛弃，于是追求应用程序能够像创建它们一样轻松彻底的抛弃。

    由于一次性电话的多功能性，您可能需要多个临时号码用于不同的活动。因此，每个应用程序必须能够同时支持多个号码。

    由于操作系统之间的差异很小，我们只查找支持Android和iOS的应用程序。

    每个操作系统的限制对应用程序的性能影响不大，因此本列表仅包含支持两种主要操作系统的应用程序。

    也许很多人都知道 Google Voice；虽然该软件的确有此功能，但它 并不是 用于此目的的最佳应用程序。

    具体来说，它一次仅限于两个号码，并且运营成本比较高（30美元），与本文列表中的其他应用相比，这是一个糟糕的选择。

    1、Hushed

    如果您正在寻找能够提供完整一次性号码体验的应用程序，那么 Hushed 就是您的最佳选择。

    虽然它没有本列表中其他应用程序那么多的功能，但它 确实提供了隐私、低廉价格和干净界面的最佳组合 。

    • Install Hushed: Android | iOS

    Hushed 不会 在您的电话账单上留下任何通话记录，包括提及您的 Hushed 号码和发送的任何短信。
    Hushed 通过使用 VoIP 来实现这一点，VoIP 通过网络管理通信。对于那些试图隐藏联系关系的人来说，这很重要。
    Hushed 是本列表中唯一 接受比特币 的应用程序，可以匿名使用。虽然 仍然可以追踪 ，但是相比信用卡来说，完全可以加一层厚厚的保护。
    其中一个更令人印象深刻的功能是 Hushed Messenger。
    Hushed Messenger 是一个消息传递应用程序，允许 Hushed 用户之间的免费安全消息传递。
    使用 Virgin Security 的 Perfect Forward Secrecy 对消息进行 端对端加密 。
    但是，与许多其他消息传递应用程序不同，您的帐户不依赖于电子邮件地址或电话号码。
    相反，用户创建一个 唯一的私人身份号码（PIN） ，然后用于开启对话。
    您可以根据需要创建任意数量的帐户。一旦您不再需要该帐户就可以轻松放弃它。
    来自非 Hushed 用户的消息以及 Hushed 用户之间的私人消息传递支持图片消息。但是，只有后者支持 自毁消息 。

    定时消息（如 Hushed 所称）会在计时器到期后擦除消息的所有痕迹。

    消息计时器在您发送后立即开始， 允许在被查看之前删除已发送的消息。

    为了获得额外的隐私，iOS用户可以启用“本机呼叫应答”。这将更改来自您的一次性号码的来电外观，以匹配传统来电。这样， 如果有人看到您的手机，他们也不会意识到您有第二个号码 。

    Hushed 在本列表中虽然不是功能最多或最安全的，但它确实提供了最佳的功能组合。

    计划起价为1.99美元，无限制计划每月只需3.99美元，并且能够不在电话账单上留下任何痕迹的情况下拨打电话。

    2、Burner

    Burner 可能是本列表中第二个最受欢迎的应用程序，它提供了许多附加功能来增强体验。

    虽然与其他应用程序（例如Hushed）相比，它的隐私功能有限，但还是相比下很出色的一款。

    • Install Burner: Android | iOS

    Burner 提供第三方集成，虽然 Hushed 也提供，但它的名单远没有 Burner 那么广泛。

    这些集成包括连接您的 Slack 帐户以查看和回复涉及您的一次性号码的文本消息，将语音邮件保存到 Dropbox 或 SoundCloud，以及使用 Evernote 创建SMS自动回复机器人。

    有八种不同的服务可以集成到 Burner 中，每种服务都提供更多独特的功能。

    Burner是唯一一款包含提醒功能的应用，可让您为之前查看的通知设置提醒。这样，您可以在以后有机会处理它们。

    此外，Burner还允许您专门为涉及其号码的呼叫设置“请勿打扰”模式，以防止任何响铃或通知出现在您的设备上。

    对于iOS，通过隐藏来自 Spotlight 搜索的 Burner 联系人和消息，可以更进一步提升体验。

    对于 Android，可以使用VoIP呼叫。此功能仍处于iOS版的测试阶段，因此只有少数用户可以访问。

    您的活动不会显示在手机账单上。

    但是，如果没有启用此功能所有通话都会显示在您的电话帐单上。

    无限制计划起价为每月4.99美元，是第二低的价格。为了帮助您做出决定，Burner 提供 最长七天的免费试用 。

    虽然它缺乏一些安全的消息传递功能，但其长长的应用程序集成列表、Siri 管理的功能以及其他独特的功能，值得选择。

    3、CoverMe

    CoverMe 是那些想要最安全功能的人的最佳选择。

    CoverMe 包含一些工具来隐藏周围人的通信， 包括伪装应用程序的能力 。

    • Install CoverMe: Android | iOS

    CoverMe 可以将自己伪装成新闻阅读器。启用后，应用程序会将其图标更改为 CMN News（Android）或 MNN Pro（iOS）。

    并且，这种隐藏比简单的图标更改要聪明得多。

    当应用程序打开时，它将显示为新闻阅读器，列出您可以阅读的文章。即使是消息通知也会显示为突发新闻，掩盖其真实身份。

    默认情况下，应用程序会要求您创建PIN码以阻止未经授权的用户。可以伪装此锁定屏幕以令其显示为照片查看器。

    与新闻阅读器伪装一样，激活只需在应用程序状态下摇动您的手机，这就 能在您发觉身后有人偷窥时快速隐藏应用程序 。

    还有一项称为魔法通知的功能。

    也就是说， 您可以将所有传入的消息修改为适当的替代文本，而不是实际的消息，从而隐藏对话的内容 。

    例如，您可以将来自朋友的所有邮件显示为“早上好！”

    CoverMe 还包括一个 完全加密和隐藏的私人Vault 。只有当您使用真实密码登录时，您才能看到其内容。

    💡 您还可以创建一个诱饵密码来欺骗想要获取访问权限的个人 。

    诱饵密码会将闯入者导航到另一个 Vault，您可以随意填充任何内容（包括什么都不填），同时保证真实数据的安全。

    ⚠️这对于被警察强行抢走手机的危险来说非常有用。

    您的 vault 可以管理许多不同的项目，包括其他形式的媒体，但存储空间有限（请参阅下面的屏幕截图）。但是，高级版提供无限存储空间。

    对于5.99美元的一次性费用，您不仅可以获得无限存储空间，而且还可以访问所有前面提到的功能（以及其他一些功能，例如呼叫筛选和单独的语音邮件）。

    还有一个“无限制”的计划，但它仅限于短信，因为通话时间仍然限制在3,000分钟之内。

    CoverMe 还包含一个 防盗功能 ，可以拍摄未经授权的用户尝试登录应用程序的情况。如果当有人试图未经请求访问您的帐户，它会向您的电子邮件发送一张图片以提醒您。

    与其他 CoverMe 用户通信时还可以使用安全的端到端加密消息传递，并包含自毁消息等功能。

    收件人打开邮件后10秒钟开始计时。虽然仍然可以截屏，但 CoverMe 可以防止重新发送或转发消息。

    CoverMe 最大的弱点是它的价格。定价从7.99美元起，比本列表中的其他应用程序都贵。

    无限制的计划也更贵，每年99.99美元。此外，高级功能的额外成本使首次使用非常昂贵。

    但是，无需说它的隐秘效果真的很棒。对于那些需要隐藏的人来说它是物有所值的。

    尤其是你如果感觉自己的手机有可能会被警方查抄的话 。

    4、TextMe Up

    TextMe Up 是基本必需品级别的。

    它缺少本列表中其他应用程序拥有的功能。但是不耽误办事儿。

    • Install TextMe Up: Android | iOS

    TextMe Up 是本列表中唯一一个允许您 免费获得一个号码 的应用程序。其他号码需要订阅。

    订阅是电话号码到期之前可用的时间长度。您可以购买一周到一年的订阅。

    甚至可以选择无限制使用，还包括其他高级功能，如美国和加拿大的免费通话，以及呼叫转移。

    订阅不包括拨打电话和发送短信所需的积分。

    通过观看视频和观看广告可以赚取积分。对 TextMe Up 用户（或 TextMe 管理的其他应用程序）的任何呼叫都是免费的。

    使用范围比较大，为200多个国家/地区提供呼叫和短信服务。

    TextMe Up 是本列表中唯一包含应用内广告的。但是，广告可以以每月4.99美元的价格删除，这个费用同时可以访问高级版本。

    高级版还包括美国和加拿大的无限通话和短信，以及其他一些独家功能。

    TextMe Up 是唯一提供视频通话的应用，但它要求通话双方都是 TextMe Up 用户 。

    总之，如果您愿意接受免费获得积分所需的步骤，它是个好的选择。

    结论

    由于互联网上存在的各种危险，通过限制在网络上共享的个人信息来保护自己的数据至关重要。

    随着太多应用想要你的电话号码，投资一次性电话变得越来越值得。

    使用上述应用程序您可以快速创建一次性号码，同时保护您的运营商号码。 这不仅可以保护您的数据，还可以提供受保护的通信线路，可以随时抛弃。

    虽然本列表中的每个应用程序都能很好地管理临时号码，但其中 Hushed 是最佳选择。

    它价格低廉，功能齐全且安全，让您可以谨慎地开展业务。

    无论您是在线销售、管理多家公司、参与敏感的社运动员、还是与多人约炮，您都可以相信 Hushed 拥有维护每一条沟通渠道的所有工具。⚪️
