    想做自己的电子游戏吗？6种工具帮您实现这点

    McCaffrey Knowledge Node , 给行动者的护身符和武器 2020年8月7日 2020年8月6日 0 Minutes
    • 你不需要很多钱，也不需要花十几年的时间学编码。只要您拥有创造力，就能实现理想或财富，更好的是两者兼备 ……

    作为一种娱乐方式，视频游戏很可能将取代电影和音乐，成为世界上最受欢迎的娱乐方式。

    几乎每个人都是玩家。无论是只玩奇怪的休闲手机游戏，还是花费数百小时的时间在AAA硬核游戏中冲刺。

    并且更重要的是， 人们可以从游戏中学到很多知识 —— 那是任何文字论述和视频指导都无法实现的培训目标 。还记得我们曾经推荐过的反抗技巧游戏吗？在下面回顾：

    • 《 7个以抵抗为主题的棋盘游戏，升级你的战斗技巧 》

    很多当权者会责令封锁游戏，因为他们知道游戏的能力。

    但是如果更多人学会自己制作游戏呢？在您所在的社区内实现更迷人的技术技巧培训？会不会更好 ……

    人们也许对游戏开发有所误解，认为似乎只有充足预算和才华横溢的人才可以进入游戏开发领域。事实上并非如此。

    对于独立的小型游戏开发者而言，如今不仅是历史上最好的时机，而且，实现您的视频游戏愿景也能是很简单的事。

    已经有大量的游戏创建工具就摆在那里，非专业和没有技术经验的开发人员也可以使用它们来实现自己的愿景。

    本文将介绍六个工具的介绍，旨在从成本和技术技能两方面减轻游戏开发的辛苦。其中有些甚至可以被用来创造那种能红极一时的游戏，帮你赚钱。

    1、Stencyl

    Stencyl 是基于订阅的游戏创建工具，可让您轻松地为多个平台创建2D游戏。游戏准备就绪后，您可以将其作为应用程序发布到 Android 和 iOS，但也支持所有桌面平台和浏览器。

    如果您喜欢手机游戏，那么您很有可能实际玩过 Stencyl 制作的游戏。

    使用该工具编写的许多游戏已跻身各种应用商店的前十名。 其中 Wrassling 是一个特别有趣的游戏，在发行时得到了很多好评。

    Stencyl 引以为豪的是，制作者可以完全不必为任何代码而揪头发，至少不是原始形式。在游戏逻辑方面，他们明智地使用拼合系统，将不同的逻辑参数汇总在一起，然后填写变量。

    因此，实际上，有很多“编码”，但是坦率地说，它就像使用乐高积木一样简单。

    2、RPG大师（各种版本）

    JRPG 仍然代表着蓬勃发展的游戏类型，现在您可以使用 RPG Maker 轻松制作经典的《最终幻想》那类游戏。有许多版本的 RPG Maker 可选。

    最好仔细检查下哪个版本的工具最适合您的需求，因为特定版本具有某些特定的功能。例如，虽然 RPG Maker XP 最便宜很诱人，但有些技巧您无法使用。但另一方面，由于某些开发者对其地图系统的偏爱，XP仍然具有强大的追随者。

    如果您想制作特定风格的游戏（或进行一些类似的修改），那么 RPG Maker 应该是您的第一站。

    RPG Maker 一直是 Windows 游戏的创造者，但现在可以发布到其他平台（例如 Android），尽管需要一点调整才能使其正常工作。

    3、Twine 2.0

    Twine 突破了应该算作游戏创作工具的极限。它最初被用来建立用户自己选择的冒险型体验.

    Twine 是一种开源工具，基本上可以让您构建一个庞大的故事节点网络，然后将其发布为可运行的HTML应用。

    能将 Twine 从交互式的叙事工具提升到可以实际制作游戏的东西是在于它的脚本编写能力。用户能够构建简单的地牢爬虫并将游戏玩法添加到自己的故事中。

    Twine 2.0无疑是此列表中最基本的工具之一，但是如果使用得当，它可以用于获得令人难以置信的强大效果。还记得 《黑镜》“潘达斯奈基” 那集 吗？它里面就使用了 Twine 2.0。

    4、Game Maker Studio 2

    Game Maker Studio 2是游戏创作软件包中的最新版本，可追溯到1999年，当时它被称为 Animo。

    这是一款用于制作2D游戏的工具，几乎包含制作2D风格的游戏所需的一切。一些3D的东西也是可能的，但没有2D创建系统那种通常简单直观的环境。

    通过 Game Maker Studio，您可以为2D对象设置动画、绘制原始2D图元并获得所需的外观。它具有许多直观的工具可以使用。

    在 Game Maker Studio 的帮助下有许多商业上成功的游戏，但是最著名的游戏无疑是 Undertale。

    该产品有许多平台支持。您可以发布到 Windows，macO，Ubuntu，HTML5，Android，iOS，PS4，Xbox One 甚至是 Switch。

    Game Maker Studio 不是免费的，但是有一个负担得起的 “ Creator Edition”，它限制了用户在 Windows 和 Macintosh 上进行发布。但您始终可以升级到 “Desktop” 许可，以扩展您的游戏的购买范围。

    5、ClickTeam Fusion 2.5

    由法国开发者 Clickteam 创建的 Fusion 2.5 对于想要制作2D游戏但又不想花几年时间学习编码的人来说是一个好的选择。

    它使用了定制的编码语言，非常简单，以至于普通用户可以在不到一小时的时间内学会它。

    有大量使用 Fusion 2.5 制作的商业上成功的独立游戏，其中最著名的可能是Freddy 的《五夜狂欢》。开发人员使用 Fusion 2.5 制作游戏，并使用 3ds Max 预先渲染3D图形。

    这个例子只是说明，只要您有创造力就可以做到什么样。

    6、Construct 3

    Construct 3 是高度流行游戏开发工具的最新版本。与 Stencyl 一样，您需要每月（或每年）支付访问开发环境的费用。您制作的任何游戏都是您的财产，您可以自由出售它们以获取利润。

    Construct 3 仍是相当新的东西，但是 Construct 2 已经产生了一些值得注意的游戏。

    Construct 可以导出到 JavaScript、HTML5 和其他一些与PC和Web兼容的标准，也有原生的 Android 和 iOS 支持。

    可视化脚本系统可能是 Construct 的最大卖点，并且被认为是对游戏逻辑设计的一种特别直观的理解。定价也相当合理，特别是如果您考虑成果可能是一款热门独立游戏能为您带来多少财富的话。

    打造自己的大片

    游戏世界充斥着各种小型独立开发者的故事，他们凭借一款没人能放下的游戏而大受欢迎。

    游戏好还是不好，并不在于投了多少钱；好的游戏都必需具有创新的设计，完善和引人注目的中心机制的特定组合。

    虽然掌握一些编码的技术技能总是一个好主意，但是这些直观的游戏创作工具意味着您无需费心就能制作出您一直想要的游戏。它所要做的就是创造力，和一个好点子。
