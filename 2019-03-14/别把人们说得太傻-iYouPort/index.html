    别把人们说得太傻

    McCaffrey Newsletters , 关于 Facebook 2019年3月14日 2019年4月26日 0 Minutes

    皮尤研究所（Pew Institute）的 一项新研究表明 ，有74％的 Facebook 用户不知道 Facebook 会跟踪他们并分析他们的兴趣爱好和私密的个人特性。但，这是真的吗？

    该研究的参与者首先指出 Facebook 的广告偏好页面，该页面列出了一系列个人兴趣。 近60％的参与者承认 Facebook 归总的兴趣列表对他们的实际兴趣来说非常或比较准确，51％的人表示他们对 Facebook 创建这样的深度窥探的列表感到不舒服 。

    数据的覆盖范围一般强调消费者的无知。“大多数 Facebook 用户仍然对其令人毛骨悚然的监视行为一无所知”，TechCrunch 的报道说；“Facebook 的行为对大多数用户来说还是一个谜”，纽约时报说；“大多数用户仍然不知道 Facebook 广告如何运作”，Wired 说。

    一堆主流媒体异口同声，但他们说的究竟是不是准确的？

    我更倾向于更乐观地看待这些数据。就在2017年，大多数美国人还都不能说出受第一修正案保护的单一权利。如果人们依然无法赶上监视资本主义的变迁，应该并不惊人。

    但是，让我们再看一下数字。根据皮尤的说法，26％的美国人都知道 Facebook 会记录他们的兴趣列表并使用它来定位广告；大约 有2.14亿美国人 拥有 Facebook 个人资料。如果是这种情况，那么在过去的十年中，应该有 5560 万人已经了解了社交网络跟踪的工作原理。

    再回看 九月发布的调查 ：Pew 在5月和6月对超过 3,400 位美国 Facebook 用户进行了调查，结果发现，在18到29岁的人中，有44％的人表示他们去年已经从手机中删除了该应用程序。其中一些人可能会在以后重新安装它。但总体而言，26％的受访者表示他们删除了该应用程序，而42％的受访者表示他们已经停用了几周或更长时间，54％的受访者调整了他们的隐私设置。

    该调查没有试图将这些删除归因于任何特定原因，尽管作者推测近一年内的数据隐私丑闻已经造成了损失。但至少可以看出，数千万美国人中的一些人是知道 Facebook 的监视资本主义商业模式是如何运作的，这也是该公司增长放缓的原因之一（总的来说，北美市场饱和可能是一个更大的原因）

    值得注意的是，大多数受访者表示，Facebook 已经准确地盯住了他们的兴趣，并且绝大多数受访者表示这份兴趣分析列表让他们感到不舒服。

    但是那些既知道 Facebook 如何运作又对此感到不舒服的人群，虽然比你想象的要小，却足以对 Facebook 的未来发挥重要作用。5560万美国人已经代表了一个健康的选区 — 从 Facebook 用量的下降来看，他们显然已经投票了。

    多年来，人们将记住生日的任务外包给了 Facebook。其间虽然有些人退出了，但其他人依旧依赖该应用程序提醒才能想起给朋友留言。Joanna Stern 说：30岁的 Kaveri Chandrashekar 是孟买的一名作家，估计她曾经在 Facebook 上收到了75个生日祝福。由于隐私问题，她于2017年6月停用了自己的帐户。当她最近的生日到来时，她通过短信和电话收到了一小部分祝福。朋友和家人直到当天晚些时候都没有记住她的生日。而她的祖母却反问：“为什么 Facebook 没有提醒我？”

    这大概就是问题：Facebook 已经深度侵入了人们的生活，变成了习惯。它被依赖了，而任何一种依赖不仅是绑架，而且极难根除。（更多详见：《 完美的绑架 》）

    社交媒体的利弊几乎均分，如何扬长避短是门学问。比如 ASPCA 在9月份进行的一项研究发现，76％ 被调查的庇护所和救援组织表示，他们的社交媒体关注者数量在去年有所增加，Facebook 被认为是更方便找到收养的最有效平台；66％的人表示社交媒体提高了筹款水平，56％的人表示，社交媒体有助于他们在安置高级动物或有特殊需求的人群方面取得了更大的成功。（关于《 如何扬长避短 》）

    最近，Facebook 的“十年挑战”显然是一项收集生物识别数据的阴谋，整个“挑战”包括将十年前和今天发布的两张个人照片并排放置，详见这里的分析：《 “十年”的阴谋 》

    戒断 Facebook 或者谷歌并不容易，即便人们能广泛认识到监视资本主义对人权的侵害，虚无和习惯性依赖也会阻止很多人的行为改变。我们应该追求的不是停留在“让多少人认识到这个问题的严重性”层面上，而是，如何提供给人们更好的且同样便捷的替代方案。

    但是必须说，主流媒体引述调查数据原本应该给予研究人士以精准清晰的洞察窗口 — 他们应该是帮助解决问题的人，而不是带来更多的误导。
