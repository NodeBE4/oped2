    15个免费的卫星图像数据源：甚至可以看到车牌……

    McCaffrey Knowledge Node , OSINT , 给行动者的护身符和武器 2020年8月11日 2020年3月23日 1 Minute
    • 卫星图像是开源情报调查的硬件

    调查界的小秘密是：有数十种权威来源的高质量卫星图像。

    但是，还有更多 ……

    您不仅可以下载最新，最出色的卫星图像……而且可以完全免费。

    您只需要知道在哪里可以找到它们。以下就是免费卫星图像资源的清单。

    1、USGS Earth Explorer

    无论您身在何处， USGS Earth Explorer 都可以提供大量的卫星图像资源：

    LANDSAT： 想象一下，在任何给定地点都能回顾40多年；在任何给定时间段内，您都可以看到发生了什么变化。那有多值钱？

    这是 Landsat 卫星影像的骄傲。它具有近40年的历史，是任何其他卫星图像所无法比拟的。通过 USGS Earth Explorer 上的 Landsat 您可以免费下载这个星球的历史。

    高程： 想要高程吗？ASTER 和 SRTM 是世界著名的数字高程模型。通过 USGS Explorer，在地球上的任何地方找到你想要的东西。

    高光谱 ：探索来自 Hyperion 的高光谱图像。只有通过这个平台您才能拥有如此难得的机会。

    USGS Earth Explorer 是免费卫星图像的世界一流来源。无论您身在何处，都可以随时使用它。

    这里有一个有关如何从 USGS Earth Explorer 下载免费 Landsat 影像的 入门指南 。

    2、Sentinel Open Access Hub

    Sentinel-2 是开放式卫星图像新的令人兴奋的时代的开始。

    不仅仅是因为其覆盖范围遍及全球；而且，Sentinel-2 是当今可下载的最清晰的免费卫星图像工具。

    Sentinels Open Access Hub 是下载 Sentinel 图像的官方总站。 这里有一份指南 ，可以帮您了解如何下载 Sentinel-2 图像。

    SENTINEL-2 ：两颗卫星组成 Sentinel-2。由于其受欢迎程度，未来几年将再增加2颗卫星。

    Sentinel-2 具有10米的分辨率 (红绿蓝、近红外)。它的12个光谱带包括SWIR。SWIR 波段具备更强的大气穿透能力，有助于对地物景观进行较以往更深入的分析判读。

    SENTINEL-1 ：如果需要合成孔径雷达（SAR）图像，则 Sentinel-1 为C波段。 SAR的潜力是无限的。但是您不会得到像 Sentinel-2 这样的漂亮图片。

    下载 Sentinel 图像后，如何使用它？下载ESA的开源 Sentinel-2 Toolbox 。所有工具都为您提供了包含丰富的分析、可视化和处理工具集的框架。

    3、NASA Earthdata Search

    NASA 的 Earthdata Search 有很多让人喜欢的地方。特别是在进行了新的改版之后。它具有全新的外观和设计。首先，卫星数据的选择令人难以置信。

    DERIVED DATA: NASA的大多数地球数据是用于地球科学研究的分析产品。大多数产品来自 DAAC。

    LAND COVER: NASA Earthdata 还是全球土地使用和覆盖的绝佳来源。它具有专业的卫星数据，如永久冻土和湿地类型。

    建议从一个简单的搜索开始。然后，更改时间范围标准以缩小搜索范围。最后下载免费的卫星图像。经过几次尝试您就能掌握其窍门。👇这里有一份使用指南将有所帮助。

    4、NOAA Data Access Viewer

    您可以在这里发现沿海地区的权威数据集。但这不仅适用于卫星图像。

    数据来自美国国防部（DOD）极地轨道运行环境卫星（POES）、NOAA的地球静止运行环境卫星（GOES）和其他衍生数据。

    航空/卫星影像 ：NOAA 数据访问查看器可保存卫星，航空和 LiDAR 图像。首先，输入您感兴趣的区域。所有可用的数据集将显示在右侧窗格中。从这里开始，只需下载。

    5、DigitalGlobe Open Data Program

    如果您想要世界上最清晰的卫星图像，请尝试利用 DigitalGlobe 的功能。准备惊讶吧，它的清晰度以至于几乎可以看到车牌。

    您可以通过两种方式从 DigitalGlobe 下载免费的卫星图像：

    开放数据计划 ：对于任何自然灾害，DigitalGlobe 的 开放数据计划 都会提供卫星图像以供救援。例如，火灾、洪水、飓风、台风和地震就是该列表的一部分。他们根据知识共享4.0许可将数据发布到公共领域。

    样本数据 ：如果您只是想尝试，可以从他们的 样本图像 中下载。例如，它具有建筑物占地面积、立面和真彩色图像。

    这些是高分辨率卫星图像的示例。如果您在这里找不到想要的东西，那么下一个选择就是可以尝试的。

    6、Geo-Airbus Defense

    就像 DigitalGlobe 一样，Geo-Airbus Defense 是一家商业供应商。Geo-Airbus Defense 是 SPOT、昴宿星、RapidEye 和 Terrasar 数据的绝佳来源。

    来自这些卫星的及时和准确的数据正被用于各种领域，如安全、油气、采矿和能源、农业、环境和灾害测绘。

    它还收集了一些样本图像。但不幸的是，他们的免费卫星图像选择非常有限。

    示例图像 ：Geo-Airbus Defense 提供了140多个用于实验的 示例图像 。对于光学成像，使用 SPOT 可以将其精确到1.5米。 TerraSAR-X 的雷达图像清晰度至3米。

    最后，您可以得到无与伦比的12米的 WorldDEM。这比 ASTER 和 SRTM 全局DEM更精确。

    7、NASA Worldview

    NASA Worldview 对于下载原始数据来说不是很好。例如，您甚至不能在这里下载著名的 Blue Marble。但是你可以在 NASA NEO 上做到。

    科学产品 ：NASA Worldview 的目标是展示跨学科的科学产品。例如关于灾害和灾难的部分。此外您可以对卫星衍生产品进行圆角处理。

    如果要下载免费的卫星图像，则必须在地图上添加图层。接下来在下面的滑块中选择日期。最后，单击图例中的数据选项卡，单击 “下载数据” 按钮。

    8、NOAA CLASS

    从陆地到天空到海洋，NOAA 的卫星图像库是大型阵列数据综合管理系统（ CLASS ）。

    要访问它，您必须注册一个帐户。然后，您可以登录并开始下载卫星图像。NOAA CLASS 乍看起来有点混乱。但是您可以将它分为3个部分：

    OCEANIC IMAGERY : NOAA 通过 CoastWatch 管理和保护海岸线。CoastWatch 是风、温度和颜色等测量值的集合。所有这些都来自 POES，JASON 和 GOES 等卫星。

    ATMOSPHERIC IMAGERY : GOES-R 是迄今为止最先进的气象卫星。因此，这些卫星图像场景捕获了从飓风到龙卷风的所有信息。

    ENVIRONMENT AND CLIMATE : 这里涵盖从臭氧到气溶胶的所有产品。

    目前 NOAA 国家数据中心支持 PoE、DMSP、GoS、Metop、Jason-2数据和选定的模型再分析数据。它将把核电站、JPS（以前的 NPOES）、GOS-R、Jason-3 的数据收集归档，计划中的地球观测系统包括 nexrad 产品。

    9、National Institute for Space Research (INPE)

    国家空间研究所（INPE）是巴西的太空研究部门。他们开发了 INPE图像目录 ，这是一个可以下载免费卫星图像的库。该目录的很大一部分是来自CBERS的卫星图像。

    CBERS ：该目录包括来自他们自己的中国-巴西地球资源卫星2和2b（CBERS-2、CBERS-2b）的卫星图像。它包括来自美国、英国和印度的卫星，Aqua、CBERS、Landsat、ResourceSat、S-NPP、Terra 和 UK-DMC 2。

    一个警告是该数据特定于南美和非洲。如果可以的话，您需要将网站从葡萄牙语翻译成英语（直接转译中文的话读起来很难受）。

    完成此操作后，您将必须创建一个帐户。因为对于您的每个请求，INPE 都会将下载详细信息发送到您的电子邮件中。

    接下来，选择您的卫星和传感器。然后，选择一个位于南美和非洲的国家。最后，将其添加到购物车，单击 “Go Ahead”。从这里，您可以从 INPE FTP 站点下载免费的卫星图像。

    10、Bhuvan Indian Geo-Platform of ISRO

    印度在卫星遥感技术上取得了巨大成功。这可以追溯到1998年印度遥感卫星（IRS1A）的发射。从那时起，在 Bhuvan Indian Geo-Platform 上就可以获取数据。

    该平台构建良好。但是大多数数据仅适用于印度。这包括 IMS-1（高光谱），Cartosat，OceanSat 和 ResourceSat。都是印度的卫星。以下产品也可以在印度境外下载：

    • NDVI（标准化差异植被指数）全球覆盖率
    • CARTODEM：30米高程模型使用来自重叠的 Cartosat-1 图像的立体图像。

    11、JAXA’s Global ALOS 3D World

    ALOS World 3d 是一个30米空间分辨率的数字表面模型（DSM）。 JAXA构建了它，并于近期向公众公开。

    这是目前最精确的全球尺度高程数据。

    使用先进的陆地观测卫星 “Daichi”（ALOS） — — Palsar 的L 波段， JAXA Global ALOS portal 是一个令人兴奋的发展。

    为了获得高精度的DSM，您必须通过 JAXA Global ALOS portal 下载。接下来，您必须注册一个帐户。批准后，您可以就下载它了。

    12、VITO Vision

    VITO Vision 网站提供 PROBA-V、SPOT-Vegetation 和 METOP 免费卫星图像。这些粗分辨率的卫星可以雕刻出地球表面的植被格局。

    创建 VITO Vision 帐户需要花费一些时间和精力。该界面直观易用，可随时提供免费的低分辨率卫星数据。这种数据类型对于不需要更详细信息的大型应用程序非常有用。

    13、NOAA Digital Coast

    当您在 NOAA’s Digital Coast 上探索数据时，这一切都与美丽的海滨有关。您将在此图像平台上获得沿海数据。

    要下载数据，请选择您感兴趣的区域。定义要下载的数据集。并筛选结果。这里有底栖生物、海拔、图像、土地覆盖和社会经济数据。

    您可以从一系列免费的卫星图像中进行选择，例如雷达，红外和真彩色复合图像。

    14、Satellite Land Cover

    Landcover.org 列出了从陆地卫星、MODIS 和 AVHRR 获得的全球土地覆盖数据。

    利用这个工具，研究工作可以量化全球土地覆盖和土地覆盖变化。这包括地表的植被、地质、农业、水文和城市地区。

    15、UNAVCO

    如果你不知道 UNAVCO 是啥，这里有一个快速而有用的3分钟视频可以了解它。

    他们的主要目标是使用大地测量技术来支持科学研究。洪水、板块构造、地震 — — 这个世界是相通的。UNAVCO 通过提供对大地测量数据的访问来促进研究。他们用自己的 UNAVCO SAR Archive Search User Interface 。

    该网站充斥着来自卫星的合成孔径雷达。但是首先，您需要一个用户名和密码。然后就可以使用数据了。

    最后

    卫星数据为我们如何了解地球开辟了新的渠道。它是调查研究的刚需。

    你已经知道，并不需要花几天的时间去寻找一个好的地图，你可以利用这个免费的卫星图像数据列表。

    现在您可以从全球各地获得免费的卫星图像了 —— 找到您想要的东西，不再被宣传的谎言所误导。
