    Facebook为俄罗斯互联网巨头创造了一个特殊的数据扩展……究竟为什么？

    McCaffrey Newsletters , 关于 Facebook 2018年1月12日 2019年4月26日 0 Minutes
    • 你知道 Facebook 和 Twitter 是俄罗斯亿万富翁资助的吗？到底谁是“干预美国选举”的罪魁祸首？

    自3月份以来，当有消息指责剑桥分析公司使用 Facebook 应用程序在未经用户同意的情况下收集了多达 8700 万人的数据这一事实时，该社交网络巨头不得不反复回答它是如何出卖用户的数据以及把数据给了谁。在丑闻发生后不久，Facebook 就在一篇博客文章中匆匆自卫，称在 2014 年，它改变了 API 的一个元素，“以防止第三方收集用户朋友的数据”，就像剑桥那种第三方。Facebook 称虽然它在 2014 年宣布了这一变化，但原本已经可以访问数据的应用程序在 2015 年 5 月之前仍然可以访问。

    然后，在上个月末向众议院能源和商务委员会提交的 700 多页书面答复中，Facebook 承认，一些应用程序可以拥有长达六个月的访问权限，以允许它们“遵守”新规则。 名单上有数十家公司 ，包括像 Hinge 这样的约会应用程序、也有像 Spotify 这样的音乐流媒体服务， 但是，还有一个颇为引人注目的，那就是：俄罗斯互联网巨头 Mail.ru 。

    根据 Facebook 的说法， Mail.ru 获得了为期两周的数据扩展，以放缓两个消息传递 APP 上的功能，该功能使用户能够与那些也拥有 Mail.ru 应用程序的人一起查看他们的 Facebook 好友列表和消息。在扩展期间，至少应用程序只能访问人们的朋友列表，而不能访问有关这些朋友的喜好或兴趣的任何信息。然而， 早在该扩展到位之前，Facebook 就称 Mail.ru 在该平台上运行了数百个应用程序，所有这些都在 Facebook 的旧规则下运作，也就是说确实允许应用开发者收集用户的朋友数据 。其中一些应用程序最早于 2009 年就开始运行了。

    “一些应用程序是在 2015 年平台更新之前构建的，因此它们确实可以访问我们平台的早期版本，”Facebook 发言人说， “这使得用户可以选择同意分享关于他们自己以及他们的朋友的信息。”

    Facebook 表示， Mail.ru 的大多数应用程序都是保密的测试版应用程序，只有少数实际上是公开发布的。 但是它没说究竟有多少用户在未经同意的情况下就将隐私信息暴露给了 Mail.ru 。该公司补充说，自 2015 年5月 Facebook 更改其 API 以来， Mail.ru 的应用程序集合无法访问人们的朋友数据。尽管如此，Facebook 现在正在调查 Mail.ru ，以及在更改之前可以访问大量用户数据的所有其他应用程序。

    但是，该发言人表示，调查本身并不是一种谴责： “我们没有发现任何与 Mail.ru 滥用有关的迹象。如果我们发现任何可疑活动或可能的误用，那就是我们开启正式审计的时候。”

    ⚠️ Facebook 在 2015 年之前为数千家其他公司提供了与 Mail.ru 相同的数据访问权限。然而，最近对俄罗斯在 2016 年大选之前操纵社交网络的担忧可能会使两家公司之间的关系成为新的亮点……

    Mail.ru 的一位发言人在给在线媒体 WIRED 的一份声明中写道：“我们假设在改变 API 的同时改变了那些尚未更新到最新版本的热门应用程序的客户条款[…]我们绝对严格控制我们与 Facebook 的合作以满足我们产品的业务需求，严格按照 Facebook 的规定。“

    对于那些熟悉 Facebook 首席执行官马克扎克伯格与Yuri Milner. 的关系 的人来说，Facebook 通过 Mail.ru 促成数据扩展的事实可能并不令人惊讶。

    这个俄罗斯亿万富翁兼 Mail.ru 的创始人也是 Facebook 的主要投资者 。（Milner.的一位发言人在一份声明中表示，“Yuri Milner.自 2003 年以来一直没有担任 Mail.ru 的首席执行官。在 2010 年的 Mail.ru 首次公开募股后不久，他卖掉了公司的所有股份。2012 年，他从董事会辞职，从那时起就没有参与其中“）

    不过， 去年，有关 Milner. 与克里姆林宫的关系的报道也浮出水面了 。

    2017 年 11 月，在 “巴拿马文件” 泄漏了 1340 万份与离岸支付相关的机密资料之后，“纽约时报”报道说， Milner.已经获得了数亿美元的俄罗斯国家资金，他通过他的国际投资公司 DST Global 部分投资给 Facebook 和 Twitter。

    虽然报告中没有任何内容表明这些投资是俄罗斯影响力行动的一部分，但是，在美国对俄罗斯干预选举进行联邦调查后，对这一消息的关注爆发了。去年秋天，Milner.在一封公开信中为自己的声誉辩护，称他试图渗透美国科技公司以帮助俄罗斯政府的说法是“牵强附会”和“谎言”。

    在 Facebook 披露之后的一份声明中，众议院委员会排名成员、新泽西州民主党人 Frank Pallone Jr. 表示，Facebook 对国会的回答“比他们提出的问题更多”。虽然他没有回应 WIRED 关于 Mail.ru 事件的评论请求，但 Pallone Jr. 在声明中表示，“令人不安的是， 在丑闻曝光四个月后，Facebook 仍然不知道有多少人拥有其用户的数据，这些数据是否今天仍在被使用，以及怎么用的 。“

    一直在调查俄罗斯操纵社交媒体平台事件的参议院情报委员会副主席、民主党参议员 Mark Warner 在一份声明中表示，“我们需要确定与 Mail.ru 共享的用户信息都是些什么，以及有可能利用这些数据干什么“。

    Warner 特别关注 Mail.ru 的主要参与者，包括主要投资人 Alisher Usmanov，“此人与弗拉基米尔·普京 Vladimir Putin 有着密切关系 。”

    至少， 对俄罗斯操纵 Facebook 事件的调查已经持续了近一年，而 Facebook 直到现在才提供这些信息，这至少表明 Facebook 显然缺乏透明度 。

    在众议院委员会的数千份提问中，Facebook 一再被问及俄罗斯国家机构对 Facebook 用户数据的访问权限。然而 Facebook 回应说，它在 2013 年至 2017 年期间他们收到了来自俄罗斯政府提出的 34 项数据请求，但没有提供任何数据。

    但是专家表示 ， Mail.ru 相关的交易与 Facebook 向包括华为等中国公司在内的设备制造商提供数据的消息，如果放在一起观察的话，足够反映 Facebook 在处理国际政权问题方面的天真 。

    “如果你是一个旗下公司具有一定规模的俄罗斯商人，你就无法逃避俄罗斯情报部门需要你提供信息的要求，”美国外交官 Brett Bruen 说，他曾担任奥巴马的全球性行动总监，现在经营着一个咨询公司叫 Global Situation Room。 他指出，“这就是今天人们在俄罗斯开展业务时不得不面对的现实。”

    这也不是俄罗斯独有的状况。Bruen 指出， 美国国家安全局早已找到了自己的方法来收集美国科技公司的数据，正如 Edward Snowden 所揭示的那样。与国家资助的情报机构对所有这些隐私数据所做的事相比，剑桥分析公司丑闻几乎微不足道 。 “剑桥分析公司是一家相对较小的公司，曾经在边缘折腾，现在他们把这些信息转交给了一个大型情报机构。”
