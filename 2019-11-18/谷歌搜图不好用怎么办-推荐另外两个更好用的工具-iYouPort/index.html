    谷歌搜图不好用怎么办：推荐另外两个更好用的工具

    McCaffrey Knowledge Node , OSINT , 给行动者的护身符和武器 2019年11月19日 2019年7月30日 0 Minutes
    • 亲测好用的工具推荐给大家

    反向图像搜索是任何深度调查和信息验证的重要部分。

    有数十种专业搜索引擎可供选择，每种搜索引擎都有自己的优点和缺点，但是使用多个独立的搜索引擎逐一检查图像会非常耗时。

    • 搜索引擎大全《 如果你擅长搜索，你能找到一切：*没有谷歌*的强大搜索世界 》

    令人沮丧的是，许多 RI 工具只允许您选取一两个搜索引擎，如 Google（有明显限制）和 Tineye（撞大运机器……）。

    但是， 还有另外两个很棒的工具，可以让您通过大量的自定义搜索囊括很多个搜索引擎 。

    Reverse Image Search 和 Search By Image 是我最喜欢的两个。

    Reverse Image Search

    Andreas Bielawski 的反向图像搜索工具 适用于 Chrome 和 Firefox，是一个很简单但非常有效的工具。

    安装完成后，只需右键单击图像，RIS 即可让您选择要用于对所选图像进行反向搜索的搜索引擎。

    最常见的搜索引擎几乎全包括在内，不仅有通常的谷歌和 Yandex，还有更多的利基搜索引擎，如动漫专家工具 IQDB。

    我最喜欢 RIS 的功能是它允许你添加任意数量的自定义搜索引擎，这样你就可以创建一个真正的瑞士军刀，用于快速反向图像搜索。

    该工具仍在积极开发中，目前的测试效果是完全可靠的。

    Firefox 版的 在这里 获取；Chrome 的请按照 此处 的说明操作。

    Search By Image

    Search By Image 是荷兰的开源调查专家推荐的。到目前为止，它一直令我印象深刻，它正在慢慢成为我的首选工具。

    与RIS一样，它可以为一键式查询添加自定义范围的多种不同搜索引擎，并且它还具有一些额外的功能，可以给它带来很多的优势。

    它不允许你添加任何新的搜索引擎，但实际上已经没有其他更好的搜索引擎没有包括在内了。

    通过右键单击图片然后选择要使用的搜索引擎（或者如果您愿意，可以选择所有搜索引擎）来查询图像：

    “按图搜索”允许您从计算机上传图像（或通过URL从Web上传）并对该特定照片进行反向图像搜索。

    并且，我最喜欢的功能是，它能够捕获图像的一小块区域，然后对该特定区域进行反向图像搜索。

    当您想要关注特定细节时，这是一种很常用的RI技术，但通常涉及手动剪切图像然后将特定片段上传到搜索引擎。

    但使用 Search By Image 您可以非常简单地做到这点，而无需其他工具协助。这是一个非常有用的功能。

    不得不说我对“Search By Image”印象非常深刻，虽然它仍在开发中。长期测试显示，Search By Image 的有效频率最高。

    下载它的 Chrome 版在 这里 ，火狐 在这里 ，Opera 在这里 。
