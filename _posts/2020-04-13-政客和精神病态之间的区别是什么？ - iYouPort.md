  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    政治人物比普通民众更容易成为反社会行为者
   </strong>
   。所有精神病/反社会型人格障碍领域的专家都不会质疑这点 ……这对我们的社会来说是极为痛苦的，但它确实解释了很多现象，无耻的欺骗性政治行为就是其中之一。” —— 哈佛大学医学院临床心理学家和前任讲师
   <a class="markup--anchor markup--p-anchor" data-href="http://www.huffingtonpost.com/david-freeman/are-politicians-psychopaths_b_1818648.html" href="http://www.huffingtonpost.com/david-freeman/are-politicians-psychopaths_b_1818648.html" rel="noopener noreferrer" target="_blank">
    Martha Stout 博士
   </a>
   。
  </p>
  <p class="graf graf--p">
   二十年前，一个报纸头条提出了
   <a class="markup--anchor markup--p-anchor" data-href="http://www.independent.co.uk/news/q-whats-the-difference-between-a-politician-and-a-psychopath-a-none-1361687.html" href="https://www.independent.co.uk/news/q-whats-the-difference-between-a-politician-and-a-psychopath-a-none-1361687.html" rel="noopener noreferrer" target="_blank">
    这样一个问题
   </a>
   ：“政客和精神病态之间的区别是什么？”
  </p>
  <p class="graf graf--p">
   曾经和现在的答案都是相同的：没有区别。
  </p>
  <p class="graf graf--p">
   漠不关心、冷漠无情、自私、不负责任、寄生的罪犯，与
   <a class="markup--anchor markup--p-anchor" data-href="http://www.thedailybeast.com/articles/2013/06/07/congressmen-come-and-go-but-corruption-is-here-to-stay.html" href="http://www.thedailybeast.com/articles/2013/06/07/congressmen-come-and-go-but-corruption-is-here-to-stay.html" rel="noopener noreferrer" target="_blank">
    对选民撒谎的官员
   </a>
   之间没有区别，他们对选民的意愿视而不见、欺骗纳税人辛苦赚来的钱、拥护企业精英、巩固军
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%86%9b%e4%ba%8b%e5%b7%a5%e4%b8%9a%e7%bb%bc%e5%90%88%e4%bd%93/" href="https://www.iyouport.org/category/%e5%86%9b%e4%ba%8b%e5%b7%a5%e4%b8%9a%e7%bb%bc%e5%90%88%e4%bd%93/" rel="noopener noreferrer" target="_blank">
    事工业综合体
   </a>
   、并丝毫不去考虑他们的轻率行动和匆忙通过的立法可能对无力防守的公民造成的威胁。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    精神病态和政治人物都有趋向于自私、冷酷、无情地利用他人、不负责任、病态的欺骗、吹嘘、花言巧语、缺乏同理心和肤浅
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.independent.co.uk/news/q-whats-the-difference-between-a-politician-and-a-psychopath-a-none-1361687.html" href="https://www.independent.co.uk/news/q-whats-the-difference-between-a-politician-and-a-psychopath-a-none-1361687.html" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     的倾向
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   具有超凡魅力的政治家，就如犯罪心理变态者那样，表现出
   <a class="markup--anchor markup--p-anchor" data-href="http://www.independent.co.uk/news/q-whats-the-difference-between-a-politician-and-a-psychopath-a-none-1361687.html" href="https://www.independent.co.uk/news/q-whats-the-difference-between-a-politician-and-a-psychopath-a-none-1361687.html" rel="noopener noreferrer" target="_blank">
    无法为自己的行为承担责任
   </a>
   ，具有高度的自我优越感，长期处于不稳定状态，具有社交偏离性和寄生性的生活方式，需要不断的刺激，并且拥有不切实际的目标。
  </p>
  <p class="graf graf--p">
   不论您是谈论哪个党派的政客，都是如此。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    政治上的精神病态患者在很大程度上都是用相同的方式
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.theatlantic.com/health/archive/2012/07" href="https://www.theatlantic.com/health/archive/2012/07" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     掩盖自己
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，令他们充满了看似轻松的魅力，并且拥有精明的头脑。正是这些领导人最终制造了病态：极权社会专心于权力和控制，并破坏正常的自由和行使自由权的任何人。
   </strong>
  </p>
  <p class="graf graf--p">
   一旦精神病态者获得了权力，结果通常是某种形式的极权政府或病态统治。
  </p>
  <p class="graf graf--p">
   作家 James G. Long
   <a class="markup--anchor markup--p-anchor" data-href="https://www.americanthinker.com/articles/2014/08/the_road_to_pathocracy.html" href="https://www.americanthinker.com/articles/2014/08/the_road_to_pathocracy.html" rel="noopener noreferrer" target="_blank">
    指出
   </a>
   ：“政府的行为损害了大部分本国人民的利益，只偏爱某些群体。我们目前正在目睹社会公民的两极分化，非法行为、以及大规模的和不必要的债务。
   <strong class="markup--strong markup--p-strong">
    这是系统性精神病态的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.americanthinker.com/2014/08/the_road_to_pathocracy.html" href="http://www.americanthinker.com/2014/08/the_road_to_pathocracy.html" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     典型特征
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，在苏联过度扩张和崩溃时，发生了非常相似的事
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   它标志着民主政府的灭亡，并为军国主义、僵化、不宽容和不人道的极权主义政权奠定了基础。
  </p>
  <p class="graf graf--p">
   令人难以置信的是，
   <strong class="markup--strong markup--p-strong">
    尽管有明确证据表明精神病态政府已经对国家及其公民造成了损害，但选民仍在继续选择精神病态者担任权力和影响力职位。
   </strong>
  </p>
  <p class="graf graf--p">
   调查记者
   <a class="markup--anchor markup--p-anchor" data-href="http://thinkprogress.org/justice/2014/01/09/3140081/bridge-sociopathy/" href="https://thinkprogress.org/justice/2014/01/09/3140081/bridge-sociopathy/" rel="noopener noreferrer" target="_blank">
    Zack Beauchamp 表示
   </a>
   ：“ 2012年，一群心理学家根据历史上每位总统的人格数据得出的心理疾病特征估值，评估了从华盛顿到小布什的所有总统。他们发现，总统往往具有精神病态者特有的无畏和低焦虑水平，这些特征似乎可以帮助这一职务，但是，也可能使他们做出
   <a class="markup--anchor markup--p-anchor" data-href="http://healthland.time.com/2012/09/11/psychopathic-traits-what-successful-presidents-have-in-common/" href="http://healthland.time.com/2012/09/11/psychopathic-traits-what-successful-presidents-have-in-common/" rel="noopener noreferrer" target="_blank">
    鲁莽的决定
   </a>
   ，从而伤害他人的生活。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    优先考虑权力的意愿高于一切，包括高于其同胞的福利、冷酷无情、缺乏同理心和
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://newrepublic.com/article/111087/wisdom-of-psychopaths-kevin-dutton" href="https://newrepublic.com/article/111087/wisdom-of-psychopaths-kevin-dutton" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     完全缺乏良心
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，这是反社会人格的主要特征
   </strong>
   。
  </p>
  <p class="graf graf--p">
   当我们自己的政府不再将我们视为具有尊严和价值的人，而是视为要操纵的对象、挖掘数据的目标、和由警察控制的事物时，人们已经不得不真正开始思考这个关于每个人最大利益的问题。
  </p>
  <p class="graf graf--p">
   如果我们不敢走出困境，就会继续承受被虐待、被入狱、被不公正惩罚的境遇 —— 我们将不再会能在宪政共和下运作。取而代之的是一种病态统治：精神病态政府的暴政。
  </p>
  <p class="graf graf--p">
   更糟糕的是，
   <strong class="markup--strong markup--p-strong">
    心理病理学分析不仅限于担任政府高级职务的人。它可以像病毒一样在人群中传播。
   </strong>
  </p>
  <p class="graf graf--p">
   一项关于病态统治的
   <a class="markup--anchor markup--p-anchor" data-href="http://motherboard.vice.com/blog/you-are-as-evil-as-your-social-network-alexander-haslam-on-what-the-prison-experiment-got-wrong" href="https://motherboard.vice.com/blog/you-are-as-evil-as-your-social-network-alexander-haslam-on-what-the-prison-experiment-got-wrong" rel="noopener noreferrer" target="_blank">
    学术研究
   </a>
   得出结论：
   <strong class="markup--strong markup--p-strong">
    “暴政之所以盛行，是因为人们积极地将那些促进恶行的人视为道德的”。
   </strong>
  </p>
  <p class="graf graf--p">
   人们不仅会排队并向他们致敬，有些人甚至还会成为恶的代理人。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://motherboard.vice.com/blog/you-are-as-evil-as-your-social-network-alexander-haslam-on-what-the-prison-experiment-got-wrong" href="https://motherboard.vice.com/blog/you-are-as-evil-as-your-social-network-alexander-haslam-on-what-the-prison-experiment-got-wrong" rel="noopener noreferrer" target="_blank">
    Alex Haslam 教授说
   </a>
   ，很大程度上取决于领导者如何 “培养追随者的认同感”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “一个很明显的事情是，当权者最喜欢谈论的是 ‘我们’ 而不是 ‘我’，而领导力实际上是在培养这种关于 ‘我们’ 的共同认同感，然后，就能使人们以当权者所希望的方式行事 ……”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    就如当权者最喜欢宣称 “国家安全”，事实上这个词只是 “政权稳固” 的委婉说法；暗示 “国家” 是为了暗示其中有 “人民”，事实上到处都没有人民，一切都是为了当权者自己的利益。
   </strong>
  </p>
  <p class="graf graf--p">
   现代公司制国家的目标是显而易见的：在其公民之间促进、培养和嵌入一种共同认同感。为此，“我们人民” 就等同于 “我们这个警察国家”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “我们人民” 正在迅速沦为奴隶，以屈服于一种面目全非的、官僚主义的专制政府机构，该机构通过无数的法律法规和禁令无情地侵蚀着人民的自由。
   </strong>
  </p>
  <p class="graf graf--p">
   对这种政权的任何抵抗都取决于选择反击的人心中舆论的力量。这意味着公民必须非常小心，以避免与压迫性政权同步。
  </p>
