  <p class="graf graf--p">
   萨勒曼[MBS]吸引的美国名人和他打通的那些门都不足为奇。硅谷在过去的十年中从沙特王子这样的人手里拿了很多钱，MBS的名字在这里尽人皆知。
  </p>
  <p class="graf graf--p">
   Altman 和Andreessen 将加入一个顾问委员会，帮助沙特建设价值 5000 亿美元的城市项目。Thiel 和 Khosla 将报名参加利雅得-沙漠达沃斯的炫目会议。欢笑环绕四周。
  </p>
  <p class="graf graf--p">
   但是，六个月后，MBS 下令在沙特领事馆野蛮杀害了华盛顿邮报专栏作家 Jamal Khashoggi。一些愤怒出现了，至少在表面上看起来是愤怒，访问被取消；硅谷是否正在重新考虑从这个独裁统治中获取金钱的主意是否真的很好？
  </p>

  <p class="graf graf--p">
   然而，
   <strong class="markup--strong markup--p-strong">
    愤怒，仅仅是一个小的、暂时的、稍纵即逝的时刻
   </strong>
   。
  </p>
  <p class="graf graf--p">
   因为事实证明，这里并没有产生一个新的道德指南针，不会为外国资金对硅谷的渗透提供护栏。不过一个局外人不同 —— 美国联邦政府 —— 其目标不是穆罕默德·本·萨勒曼。而是，习近平。
  </p>
  <p class="graf graf--p">
   在过去的一年里，中国而不是沙特阿拉伯，已经发现自己被冻结在硅谷之外，具有讽刺意味的是，谁能真正关心通过科技行业肆无忌惮地注入美国的外国资金问题？总的来说不是投资者或创业公司。应该是联邦政府。
  </p>
  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    硅谷的新规则
   </strong>
  </h4>
  <p class="graf graf--p">
   从外面看，硅谷只是在印钱，通过创业公司为你提供便宜的Uber设施。但是，如果你追踪通过风险投资公司进入创业公司的资金 —— 并追踪那些风险资本从他们所谓的合伙人那里获得的资金 —— 你就可以看到一场壮观的淘金热气候，全部都是渴望向饥饿的初创公司提供资金的海外投资者。
  </p>
  <p class="graf graf--p">
   看看 Y Combinator Demo Day。十年来对于寻找下一家伟大公司的科技投资者们来说，这是个标志性的时刻；为最优秀的精英专业人士提供一个充满活力的机会去参看十几个或更多的初创公司，这些人是亲密到能彼此共享一个重要密码的朋友关系。场面足够小，可以
   <a class="markup--anchor markup--p-anchor" data-href="https://techcrunch.com/2009/03/18/y-combinator-demo-day-spring-2009/" href="https://techcrunch.com/2009/03/18/y-combinator-demo-day-spring-2009/" rel="noopener noreferrer" target="_blank">
    放进
   </a>
   YC硅谷办公室的一个房间里。
  </p>
  <p class="graf graf--p">
   而现在，Demo Day 更像是一个世界博览会，充斥着外来的和对外的数百个简介。上个月在旧金山湾旁举办的该会议上挤满了外国出生的企业家在这里大嚼M＆M饼干，
   <strong class="markup--strong markup--p-strong">
    向富裕的投资者和来自俄罗斯、西班牙、德国、尼日利亚、日本等地的政府官员推销创业公司。当然 —— 还有中国。
   </strong>
  </p>
  <p class="graf graf--p">
   这是2019年的硅谷 —— 一个渴望实现其宏伟战略的外国游乐场。在某种程度上，这是可以接受的：如果美国在科技公司方面有绝对优势、如果资本主义是全球性的 —— 那么应该欢迎硅谷的转型。
  </p>
  <p class="graf graf--p">
   美国欢迎外国资金进入纽约证券交易所和 Nasdaq；所以也应该欢迎外国资金进入美国私营公司，特别是来自新加坡等密切伙伴的资金。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    外国资金的增加使硅谷成为了风险资本家和创业公司的地缘政治雷区，要求美国创业公司做出判断，并对几十年来业内奇怪的迥异做出反应。
   </strong>
  </p>
  <p class="graf graf--p">
   沙特阿拉伯的谁应该对谋杀 Jamal Khashoggi 负有直接责任？
  </p>
  <p class="graf graf--p">
   华为是对美国国家安全的威胁吗？
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “地缘政治的世界正在冒险”，活跃于网络安全领域的风险投资家 Rob Ackerman 说，“并不是非黑即白的，它处于灰色的中间地带，我们都在试图弄清这一点”。
  </p>
  <p class="graf graf--p">
   现在居住在以色列的美国投资者 Mike Eisenberg 回忆起最近对一位企业家说的话：
   <strong class="markup--strong markup--p-strong">
    “你以为你在做生意。实际上你在做政治”。
   </strong>
  </p>
  <p class="graf graf--p">
   甚至在过去两年重塑了每个美国行业的力量 即 唐纳德特朗普，加剧这一现实之前，情况就是如此。特朗普行政当局对源源不断输入硅谷的外国资金很不满意。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但是长期以来，硅谷的大多数人对待外国资金的态度都不过是耸耸肩，把钱视为钱，而不是真正考虑从某些国家引入资金的道德和监管挑战
   </strong>
   ，这是来自 Recode 采访的50多个风险投资家、创业公司、律师和其他参与跨境投资的人士透露的。
  </p>
  <p class="graf graf--p">
   而现在，硅谷正在争先恐后地评估自己在这个新世界秩序中的曝光率。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    特别是来自两个国家的资金引发了硅谷关于创业公司及其投资者的争论：即 中国和沙特阿拉伯
   </strong>
   。
  </p>
  <p class="graf graf--p">
   关于从中国接受资金的争论引发了两个矛盾的关于中国的基本政治信仰（这是美国舆论中典型的两极化观点之一）：一方面，中国被认为是一个高度复杂的美国对手，共产党的人悄悄渗透到硅谷，他们来到这里是为了窃取知识产权，或者就像其他寻求经济利益的外国玩家一样；另一方面这被认为是刻板印象，不公平的定位。
  </p>
  <p class="graf graf--p">
   Recode 已经了解到，在过去一年中，由于美国外国投资委员会采取了更为激进的监管，有多达十几家与中国挂钩的公司缩减了在美国的投资计划（CFIUS）。
  </p>

  <p class="graf graf--p">
   另一边，围绕从沙特接受资金的争论更为激烈：
   <strong class="markup--strong markup--p-strong">
    一些初创公司和投资者呼吁硅谷在 Khashoggi 被谋杀之后寻找道德支柱。但是，这些少数人的声音正在逐渐变得微弱。在信誉危机面前沙特人已经成功摆平，用他们的钱给硅谷戴上了手铐。
   </strong>
  </p>
  <p class="graf graf--p">
   沙特人仍然间接地将数十亿美元投入热门创业公司，几乎没有受到任何公开的谴责。
  </p>
  <p class="graf graf--p">
   事实证明，在沙特这边，高尚的态度并不是阻止外国资金的办法。
  </p>
  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    剥开硅谷资金的迷雾
   </strong>
  </h4>
  <p class="graf graf--p">
   当美国公司与外国投资者合作时，事情会变得相当混乱。
  </p>
  <p class="graf graf--p">
   在优步，日本软银的外国投资者由于监管方面的担忧，可能永远不会占据他们的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bloomberg.com/news/articles/2019-04-10/two-uber-board-seats-imperiled-by-u-s-national-security-review" href="https://www.bloomberg.com/news/articles/2019-04-10/two-uber-board-seats-imperiled-by-u-s-national-security-review" rel="noopener noreferrer" target="_blank">
    两个董事会席位
   </a>
   ，尽管他们在一年多前向该公司投了高达100亿美元之后一直在进行着艰苦的谈判。
  </p>
  <p class="graf graf--p">
   在 Grindr，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.reuters.com/article/us-grindr-m-a-exclusive/exclusive-us-pushes-chinese-owner-of-grindr-to-divest-the-dating-app-sources-idUSKCN1R809L" href="https://www.reuters.com/article/us-grindr-m-a-exclusive/exclusive-us-pushes-chinese-owner-of-grindr-to-divest-the-dating-app-sources-idUSKCN1R809L" rel="noopener noreferrer" target="_blank">
    美国监管机构要求其中国
   </a>
   所有者出售该同性恋约会应用程序，因为担心北京可能会将消费者数据用作勒索。
  </p>
  <p class="graf graf--p">
   但要了解外国资金在硅谷周围存在的数量 —— 以及企业家在涉及国际水域的暗流时如何被扫地出门的 —— 只需要听听一家名为 Lucid Motors 的公司的故事。
  </p>
  <p class="graf graf--p">
   Lucid 的故事展示了美国创业公司从非美国人那里拿钱时可能遇到的奇怪挑战。
  </p>
  <p>
   <img alt="" class="alignleft wp-image-3793 jetpack-lazy-image" data-lazy-sizes="(max-width: 400px) 100vw, 400px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/沙特-4.png?resize=400%2C418&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/沙特-4.png?w=896&amp;ssl=1 896w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/沙特-4.png?resize=287%2C300&amp;ssl=1 287w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/沙特-4.png?resize=768%2C802&amp;ssl=1 768w" data-recalc-dims="1" height="418" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/沙特-4.png?resize=400%2C418&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="400"/>
   <noscript>
    <img alt="" class="alignleft wp-image-3793" data-recalc-dims="1" height="418" sizes="(max-width: 400px) 100vw, 400px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/沙特-4.png?resize=400%2C418&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/沙特-4.png?w=896&amp;ssl=1 896w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/沙特-4.png?resize=287%2C300&amp;ssl=1 287w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/沙特-4.png?resize=768%2C802&amp;ssl=1 768w" width="400"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Lucid Motors 是最雄心勃勃的初创公司之一，与 Elon Musk 的特斯拉作战，过去几年一直在努力设计豪华电动车。
  </p>
  <p class="graf graf--p">
   为了开发电池然后从零开始制造一流的汽车，Lucid 不得不筹集大量资金。这可不总是那么容易的。
  </p>
  <p class="graf graf--p">
   就像许多怀揣着巨大梦想而囊中羞涩的创业公司一样，Lucid Motors 已经将期待转向了海外。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们是一家美国公司，”该公司的首席执行官 Peter Rawlinson
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2019/2/21/18235019/lucid-motors-electric-suv-ev-saudi-arabia" href="https://www.theverge.com/2019/2/21/18235019/lucid-motors-electric-suv-ev-saudi-arabia" rel="noopener noreferrer" target="_blank">
    告诉 The Verge
   </a>
   ，“我们将在美国制造世界上最好的汽车。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Lucid 可能非常自豪地成为一家美国公司，但其投资基础几乎全部来自外国 —— 最近，这意味着来自沙特，而此前，意味着中国。
   </strong>
  </p>
  <p class="graf graf--p">
   尽管该公司总部设在加利福尼亚州的纽瓦克，但几乎所有的融资都至少部分来自亚洲的投资者。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们是亚洲公司还是加州公司？”一位知情人士这样说到。“整个品牌都在加州附近。但是从内部看，它在亚洲具有很大的影响力。”
  </p>
  <p class="graf graf--p">
   该公司的一半是由亚洲投资者拥有的。
   <strong class="markup--strong markup--p-strong">
    亚洲人的参与将导致一系列极不寻常的财务举措，显示了创业公司从外国投资者那里获取资金的风险
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这个故事集中于一位名叫贾跃亭的中国亿万富翁身上，他是中国企业集团 LeEco 的创始人，并于2011年率先投资了 Lucid 的前身公司。但回想起来，让贾进入该公司是一个错误。
  </p>
  <p class="graf graf--p">
   慢慢地，秘密地，贾积累到了 Lucid 越来越多的所有权股份，一位知情人士表示，在2016年左右的所有权高峰期，贾拥有 Lucid 近30％。
  </p>
  <p class="graf graf--p">
   而且，事实证明，贾也是 Lucid 最接近的竞争对手之一的完整所有者，这是一家名为 Faraday Future 的电动汽车创业公司。
  </p>
  <p class="graf graf--p">
   贾于2017年
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2017/12/20/16801846/faraday-future-funding-investment-payroll-jia-yueting-ceo" href="https://www.theverge.com/2017/12/20/16801846/faraday-future-funding-investment-payroll-jia-yueting-ceo" rel="noopener noreferrer" target="_blank">
    成为 Faraday 的首席执行官
   </a>
   —— 与 Lucid 发生明显且具有破坏性的利益冲突，当时 Lucid 正在亏损并探索可能的销售途径。
  </p>
  <p class="graf graf--p">
   在美国的创业生态系统中，这种情况极为罕见 —— 风险投资公司拥有管理利益冲突的规则。
  </p>
  <p class="graf graf--p">
   贾拒绝出售他的股票，引发了关于如何处理中国强盗男爵问题的无休止的争论。他没有总体规划，只是寻求控制。
  </p>
  <p>
   <img alt="" class="aligncenter size-large wp-image-3794 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/强盗男爵.png?resize=1024%2C580&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/强盗男爵.png?resize=1024%2C580&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/强盗男爵.png?resize=300%2C170&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/强盗男爵.png?resize=768%2C435&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/强盗男爵.png?resize=1100%2C624&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/强盗男爵.png?w=1256&amp;ssl=1 1256w" data-recalc-dims="1" height="580" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/强盗男爵.png?resize=1024%2C580&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>

   </noscript>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “然后你就会不断地想，’这到底是怎么回事？’”一位知情人士回忆道， “只是一个不断地分心状态”。
  </p>
  <p class="graf graf--p">
   一位消息人士说，最终，经过两年的磨合，贾最终屈服并出售了几乎所有的股份。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他承认如果我们不让投资者进入，公司就会失败 —— 我们正在亏钱，失去员工，继续下去我们将失去一切，“这位消息人士说。 “他[贾]可能很疯狂，但他在经济上并不擅长。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    故事的结局是什么？没有结局。仅仅是一个有争议的外国资金来源被换成了另一个有争议的外国资金来源。
   </strong>
  </p>
  <p class="graf graf--p">
   这次是沙特。
  </p>
  <p class="graf graf--p">
   准确说是沙特主权财富基金，在主权财富基金升级其在 Lucid 竞争对手特斯拉
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ft.com/content/42ca6c42-a79e-11e8-926a-7342fe5e173f" href="https://www.ft.com/content/42ca6c42-a79e-11e8-926a-7342fe5e173f" rel="noopener noreferrer" target="_blank">
    的股份
   </a>
   时，他们在该公司投了10亿美元。一位知情人士表示，沙特人的参与 —— 而不是中国人 —— 使事情变得“好一些”。
  </p>
  <p class="graf graf--p">
   据消息人士称，沙特协议受到的影响一直微乎其微，直到 Khashoggi 被杀 。但是，凭借多数股权，Lucid 现在是沙特在硅谷足迹的皇冠上宝石之一，为创业公司引入了一个全新的伦理和地缘政治问题。
  </p>
  <p class="graf graf--p">
   这个传奇还没有结束。沙特人对 Lucid 的投资必须得到美国政府监管机构 CFIUS 的批准。该部门一直拖着 —— 但五月份终于点亮了绿灯。
  </p>
  <h4 class="graf graf--h4 graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--h4-strong">
    “America First” 如何袭击硅谷
   </strong>
  </h4>
  <p class="graf graf--p">
   事情就是这样：尽管沙特阿拉伯采取了令人毛骨悚然的杀戮行动、并长期拥有令人头疼的人权记录，但该国仍然是美国的盟友，并且从未试图做出引起美国“国家安全状态愤怒”的任何事：比如窃取美国的知识产权。
  </p>
  <p class="graf graf--p">
   干这种事的主要是中国。美国政府的审查就是这么说的。
  </p>
  <p class="graf graf--p">
   四年前，中国领导层宣布了一项新的十年愿景，即 将工业制造业经济转向技术和科学方面的专业知识：中国制造2025。美国的鹰派人士在2016年特朗普大选中将这一中国雄心视为一种威胁。
  </p>
  <p class="graf graf--p">
   他们找到了一件武器：美国外国投资委员会（CFIUS），这是一个模糊不清的、并且在硅谷长期以来一直是陷入困境状态的无实权机构 —— 由财政部长担任主席。去年夏天，共和党人通过了一项法案，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.reuters.com/article/us-usa-cfius-congress/senate-house-panels-ok-bills-to-tighten-foreign-investment-oversight-idUSKCN1IN2TC" href="https://www.reuters.com/article/us-usa-cfius-congress/senate-house-panels-ok-bills-to-tighten-foreign-investment-oversight-idUSKCN1IN2TC" rel="noopener noreferrer" target="_blank">
    大大加强了
   </a>
   CFIUS 的调查资源，并授权该机构的能力范围扩展到少数股权投资领域。
  </p>
  <p class="graf graf--p">
   硅谷想要继续忽视华盛顿是不行的。
  </p>
  <p>
   <img alt="" class="alignright wp-image-3795 jetpack-lazy-image" data-lazy-sizes="(max-width: 400px) 100vw, 400px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/特朗普。中国.png?resize=400%2C510&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/特朗普。中国.png?w=728&amp;ssl=1 728w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/特朗普。中国.png?resize=235%2C300&amp;ssl=1 235w" data-recalc-dims="1" height="510" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/特朗普。中国.png?resize=400%2C510&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="400"/>
   <noscript>
    <img alt="" class="alignright wp-image-3795" data-recalc-dims="1" height="510" sizes="(max-width: 400px) 100vw, 400px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/特朗普。中国.png?resize=400%2C510&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/特朗普。中国.png?w=728&amp;ssl=1 728w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/特朗普。中国.png?resize=235%2C300&amp;ssl=1 235w" width="400"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   去年修订的 CFIUS 立法背后的想法 —— 编纂成 2018年“外国投资风险评估现代化法案”（FIRRMA） —— 试图严格规范来自海外的所有新资金流入所谓的“关键技术”。但你没有必要挖掘潜台词就能弄清楚在法案的发起人看来谁是最大的坏蛋。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我认为一直以来对中国的重视是不够的，”特朗普去年8月该法案通过后在白宫的罗斯福厅
   <a class="markup--anchor markup--p-anchor" data-href="https://www.whitehouse.gov/briefings-statements/remarks-president-trump-roundtable-foreign-investment-risk-review-modernization-act-firrma/" href="https://www.whitehouse.gov/briefings-statements/remarks-president-trump-roundtable-foreign-investment-risk-review-modernization-act-firrma/" rel="noopener noreferrer" target="_blank">
    说道
   </a>
   ，“[中国的存在]已经有很长一段时间了。”
  </p>
  <p class="graf graf--p">
   如果特朗普的目标是对中国设立障碍以保护硅谷，正如他后来所说的，其结果已经成功超越了他最大的预期。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这是’美国第一’的象征”，Vertex Ventures 的投资者 Jonathan Heiliger 表示，该公司主要由新加坡主权财富基金 Temasek 支持。
  </p>
  <p class="graf graf--p">
   如果这就是战略，显然它起效了 。
  </p>
  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    技术部门如何变得如此偏执
   </strong>
  </h4>
  <p class="graf graf--p">
   让我们回顾一下，解释资金如何流经硅谷的风险投资生态系统。
  </p>
  <p class="graf graf--p">
   风险投资公司利用合作伙伴的关系、经验和业绩记录，每隔两三年在一个“基金”中从所谓的有限合伙人那里筹集数百万美元 ——这些合作伙伴应该在公司中具有“有限的”运营影响力（实际的运营者是普通合伙人 — 或风险投资公司。）
  </p>
  <p class="graf graf--p">
   然后风险投资公司出去投资于特定战略或特定类型的公司。虽然有限合伙人偶尔会直接投资创业公司，但这些机构很少与企业家建立关系，因此他们通常需要风险投资公司为他们做交易。
  </p>
  <p class="graf graf--p">
   作为回报，风险投资公司向这些有限合伙人收取一小部分资金用于管理费，然后他们将其余部分投资于创业公司 —— 这些资金可以催生像 Instagram、Uber 或 Airbnb 这样的公司并创造巨额意外之财。当然，大多数投资都失败了，所以风险很高。
  </p>
  <p class="graf graf--p">
   在过去的几年里，美国风险投资公司对来自中国的投资无法抗拒的态度可以理解为一种看不见的盲目或迟到的觉醒。但一夜之间，一个新监管环境的诞生引发了突然到来的偏执。
  </p>
  <p class="graf graf--p">
   这种偏执是否值得，取决于你在美国更广泛的关于中国愿景的政治辩论中的立场。两个极端立场同时存在。
  </p>
  <p class="graf graf--p">
   一些支持中国直接投资的倡导者将新监管规定视为普通民族主义，对所有中国人一视同仁 —— 无论是来自中国大陆的、香港还是台湾的 —— 并且认为联邦政府分不清习近平的专制政府和只是想投资于财务回报的富人商人。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我真的无法做出这种区分，因为中国境内的每个人都对共产党及其目标感到满意，”赛门铁克前首席执行官 Michael Brown 说道，他是硅谷中国鹰派的主要代言人。
  </p>
  <p class="graf graf--p">
   但是 ，⚠️
   <strong class="markup--strong markup--p-strong">
    请注意，硅谷重量级人物 —— 甚至包括 Brown 本人—— 在提出遏制中国的论点时都没有提到人权问题或文化冲突。
   </strong>
  </p>
  <p class="graf graf--p">
   对中国资金的担忧主要源于 CFIUS —— 而且，对于一些人来说，首先政府审查的根本原因是：“知识产权盗窃”。而不是人权纪录。
  </p>
  <p class="graf graf--p">
   这就解释了为什么联邦调查局官员正在硅谷涌现：最近几个月来，特工们一直在与投资者会面，以警告他们创业公司从中国拿钱的危险，据两位知情人士透露。
  </p>
  <p class="graf graf--p">
   美国联邦调查局在被问及这件事时在一份声明中告诉 Recode，“联邦调查局与这些公司合作，提高他们对潜在的经济间谍活动和盗窃商业机密之威胁的认识”；“我们的目标是让公司意识到风险，并为他们提供必要的工具，以保护他们免受可能对其知识产权构成威胁的任何人的侵害。”
  </p>
  <p class="graf graf--p">
   是 FIRRMA —— 虽然不是FBI —— 才是华盛顿的长矛之角。
  </p>
  <p class="graf graf--p">
   在 FIRRMA 的帮助下，CFIUS 审查现在不仅可以在外国投资者拥有公司大部分股权的情况下启动​，而且，当投资者在公司中拥有一些决策权时也可以启动调查，即使该投资者拥有的公司股票很少。
  </p>
  <p class="graf graf--p">
   尤其是，如果该创业公司使用“关键技术”，将触发公司和外国投资者作为交易的一部分提交的审查，并且可能需要数周甚至数月才能获得任何类型的答案。
  </p>
  <p class="graf graf--p">
   与在政府中被解释为“关键技术”相关的任何事都是不可取的。（“关键技术”是该法律中的一个标志性名词）而且，由于法律写得如此模糊 —— 有27个类别涵盖从“生物技术”到“电池制造”的所有内容都属于“关键技术” —— 风险投资公司变得非常诚惶诚恐。
  </p>
  <p class="graf graf--p">
   如果创业公司处于前沿技术领域（比如 机器人或无人机），那就更糟了。有外国关系的投资者告诉 Recode 他们现在基本上认为大部分技术部门都属于*有效的禁区*。
  </p>
  <p class="graf graf--p">
   而且，这种寒蝉效应是真实的。投资者建议他们的初创公司对外国资金 —— 特别是中国出资者 —— 持超谨慎态度。
  </p>
  <p class="graf graf--p">
   硅谷的一些人甚至开展了一些项目，以确定所有主要由中国政府支持的风险投资公司 —— 制定自己的
   <strong class="markup--strong markup--p-strong">
    私人投资者黑名单
   </strong>
   。
  </p>
  <p class="graf graf--p">
   美国联邦政府还
   <a class="markup--anchor markup--p-anchor" data-href="https://ustr.gov/sites/default/files/enforcement/301Investigations/301%20Report%20Update.pdf" href="https://ustr.gov/sites/default/files/enforcement/301Investigations/301%20Report%20Update.pdf" rel="noopener noreferrer" target="_blank">
    赞助了两份报告
   </a>
   ，试图跟踪中国在硅谷的足迹 —— Brown 撰写的其中一篇报道，这对于鼓励 FIRRMA 的通过是至关重要的，不论是支持者还是批判者都同意这点。
  </p>
  <p>
   <img alt="" class="alignleft wp-image-3796 jetpack-lazy-image" data-lazy-sizes="(max-width: 400px) 100vw, 400px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中国钱.png?resize=400%2C509&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中国钱.png?w=724&amp;ssl=1 724w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中国钱.png?resize=236%2C300&amp;ssl=1 236w" data-recalc-dims="1" height="509" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/中国钱.png?resize=400%2C509&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="400"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   Ilya Golubovich 说，他的公司决定放弃其在航空航天领域的一些投资组合：“我本来希望留着那些公司的，但鉴于 CFIUS 的敏感性，我们做出了一项决定，即航空航天业不是我们想要触及的领域。” Ilya Golubovich 运营着一个三分之二来自俄罗斯和前苏联的有限合伙人支持的基金。
  </p>
  <p class="graf graf--p">
   这种感觉是有普遍性的。一位活跃于边境技术公司融资领域的风险资本家表示，他现在认为他的投资组合公司也许永远无法从外国投资者那里筹集资金了。
  </p>
  <p class="graf graf--p">
   这种状况也让创始人感到紧张，他们甚至在被抛给全球政治狼群之前就已经有了足够的担忧。
  </p>
  <p class="graf graf--p">
   想象你自己置身于创业公司首席执行官的位子上，比如，与一位联合创始人争吵，处理违背合同的客户，或准备接纳更多的外部投资者。接受中国投资者的资金——这可能意味着等待 CFIUS 审查数个月，以及可能最终被禁止 —— 即便最终被通过也很可能没那么令人兴奋了。
  </p>
  <p class="graf graf--p">
   一位风险资本家表示，她最近有两三次被创始人问到她是否有中国的有限合伙人（她确实有）。另一位风险资本家告诉 Recode，一些公司现在要求投资者签署附函，确认某些信息不会与其有限合伙人共享。
  </p>
  <p class="graf graf--p">
   可以说，在这个美中关系的新时代，这种偏执是值得的 —— 当时特斯拉
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnn.com/2019/03/22/tech/tesla-xiaopeng-motors-lawsuit/index.html" href="https://www.cnn.com/2019/03/22/tech/tesla-xiaopeng-motors-lawsuit/index.html" rel="noopener noreferrer" target="_blank">
    起诉一名前雇员
   </a>
   涉嫌窃取知识产权并将其传递给中国竞争对手；一名中国籍女子走近特朗普的佛罗里达房产，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.justice.gov/sites/default/files/testimonies/witnesses/attachments/2018/12/18/12-05-2018_john_c._demers_testimony_re_china_non-traditional_espionage_against_the_united_states_the_threat_and_potential_policy_responses.pdf" href="https://www.justice.gov/sites/default/files/testimonies/witnesses/attachments/2018/12/18/12-05-2018_john_c._demers_testimony_re_china_non-traditional_espionage_against_the_united_states_the_threat_and_potential_policy_responses.pdf" rel="noopener noreferrer" target="_blank">
    携带着
   </a>
   包含恶意软件的USB驱动器 …… 过去七年来，美国政府
   <a class="markup--anchor markup--p-anchor" data-href="https://www.justice.gov/sites/default/files/testimonies/witnesses/attachments/2018/12/18/12-05-2018_john_c._demers_testimony_re_china_non-traditional_espionage_against_the_united_states_the_threat_and_potential_policy_responses.pdf" href="https://www.justice.gov/sites/default/files/testimonies/witnesses/attachments/2018/12/18/12-05-2018_john_c._demers_testimony_re_china_non-traditional_espionage_against_the_united_states_the_threat_and_potential_policy_responses.pdf" rel="noopener noreferrer" target="_blank">
    提出的90％以上
   </a>
   的经济间谍案都集中在中国。
  </p>
  <p class="graf graf--p">
   可以肯定的是，一些创始人也报告了奇怪的经历。
  </p>
  <p class="graf graf--p">
   一位备受瞩目的创业公司创始人告诉 Recode，他在2016年与一家由国家支持的中国实体就战略投资进行谈判。
   <strong class="markup--strong markup--p-strong">
    但在最后一刻，中国投资者要求确保他们能够获取客户数据和其他知识产权。然后这笔交易就破裂了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   第二位创始人告诉 Recode，由于担心该公司正试图吸收他们的知识产权，他放弃了中国科技巨头3500万美元的投资。“他们显然正在记录并试图内化他们所能做的一切，”他说。
  </p>
  <p class="graf graf--p">
   并且，一位从中国互联网巨头那里拿钱的创始人表示，他的外国潜在客户有时会质疑他的公司是否是中国政府企业间谍活动的前线。他不知道该说些什么。
  </p>
  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    中国僵尸资金的崛起
   </strong>
  </h4>
  <p class="graf graf--p">
   在这个新的世界秩序中，拥有中国或其他广泛的外国关系的投资者只有两种选择：离开美国、或冻结非美国人。
  </p>
  <p class="graf graf--p">
   因此，随着硅谷 CEO 离开中国公司，大多数中国投资者并没有愚蠢地追求它们。银行家和风险资本家们表示，实际上很少有交易真正落空。双方越来越多地完全避免彼此。
  </p>
  <p class="graf graf--p">
   问题在于，中国投资者已经在逆向选择中挣扎，或者以不那么优雅的方式表现出他们认为自己是底部供应商的看法。文化脱节和缺乏联系通常意味着一个拿中国钱的首席执行官是在向市场表明她的公司一直在努力筹集资金（是实在没路了的意思），而现在在 FIRRMA 时代尤其如此。
  </p>
  <p class="graf graf--p">
   一位中国的投资者表示，自11月新的 FIRRMA 规定实施以来，他就没有在美国发布任何条款清单（引领投资回合的提议）。但他更积极地在海外寻找交易 —— 具有讽刺意味的是，经常转向回到中国投资。
  </p>
  <p class="graf graf--p">
   Recode 已经了解到，有多达十几家与中国有关联的公司已经缩减了在美国的投资计划，其中一些公司是大幅缩减。
  </p>
  <p class="graf graf--p">
   中国从硅谷的退出步伐一直是安静的、微妙的和难以追踪的。受到心理作用的驱动，一些中国投资者告诉 Recode，他们不喜欢在他们“被认为是敌人”的国家进行投资。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这已经催生了越来越多可以称之为僵尸基金的东西：中国支持的实体，即使仍在技术上运作，但实际上已经死亡。
   </strong>
  </p>
  <p class="graf graf--p">
   以 WestSummit Capital 为例，该公司十年前从中国主权财富基金筹集了3亿美元，投资于像游戏平台 Unity 这样的创业公司。
  </p>
  <p class="graf graf--p">
   消息人士称，该公司已经减少了支持美国公司的计划，现在转而专注于世界其他地区，一位消息人士表示，它必须努力从中国首都以外的投资者那里筹集资金。
  </p>
  <p class="graf graf--p">
   这家耗资8000亿美元的国有中国投资公司（CIC）曾希望在硅谷开设办事处。但是，现在一切都取消了。几乎肯定会在可预见的未来放弃对硅谷创业公司的任何直接投资。
  </p>
  <p class="graf graf--p">
   据一位了解 CIC 活动的人士透露，CIC 仍持有少数基金中有限合伙人的股权，例如克里姆林宫支持的 DST Global，私募股权巨头 Silver Lake Partners，以及跨境投资者 DCM Ventures。但现在没有合理的投资者希望得到与中国国家直接有关的资金。
  </p>
  <p class="graf graf--p">
   消息人士称，最近重新命名为 DHVC 的 Danhua Capital 等公司已经成为硅谷投资者中最不受欢迎的人，担心他们参与交易会危及交易。
  </p>

  <p class="graf graf--p">
   投资者和其他工作人员已经逐渐离开阿里巴巴的旧金山办事处，从三年前的大约25人下降到今天的大约15人。
  </p>
  <p class="graf graf--p">
   据知情人士透露，阿里巴巴的美国投资团队也出现了类似的下滑，因为它大幅缩减了美国的投资计划，一些投资者称只是无聊。
  </p>
  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    沙特人如何避免自己经历中国人的故事？
   </strong>
  </h4>
  <p class="graf graf--p">
   中东投资者正在处理与新加坡人、中国人和其他外国人相同的 CFIUS 问题。但他们还有一个基本的政治挑战：硅谷的一些人从根本上认为沙特人是邪恶的人。
  </p>
  <p class="graf graf--p">
   在过去十年左右的时间里，中东寡头们已经悄悄将他们的现金刘流注入了硅谷。
  </p>
  <p class="graf graf--p">
   原因在于：像沙特和阿联酋这样的国家的经济围绕着一种宝贵的资源 — 石油，年轻的领导层通过收购 Uber 和 Magic Leap 等美国公司的股票而开拓未来。这种想法比赌油碳的风险要小得多。
  </p>
  <p class="graf graf--p">
   在硅谷这种糟糕的初创公司投资环境下，沙特人的钱特别受欢迎，被认为是“人傻钱多”。
  </p>
  <p class="graf graf--p">
   根据 CB Insights 的数据，2016年沙特投资者的交易额高达53亿美元，涉及19项不同的交易。
  </p>
  <p class="graf graf--p">
   沙特计划跟随阿布扎比的主权财富基金穆巴达拉，在硅谷中心开设高端办事处。中东投资者已成功地与红杉资本、Thrive Global 和 Data Collective 这样的蓝筹风险投资公司合作，后者为他们提供了他们无法独立进军的创业公司。
  </p>
  <p class="graf graf--p">
   一个双赢。
  </p>
  <p class="graf graf--p">
   因此，穆罕默德·本·萨勒曼（Mohammed bin Salman）在2018年4月访问硅谷时吸引了一大群观众，这并不奇怪。这是技术圈与沙特关系的最高点。
  </p>
  <p class="graf graf--p">
   但是，应沙特王室的要求，Jamal Khashoggi 在土耳其遭到残酷杀害。硅谷首次开始更加关注那些揣着数十亿美元支票的人。
  </p>
  <p class="graf graf--p">
   如果中国人入侵硅谷会引发知识产权盗窃问题、保护主义监管和地缘政治竞争等问题，沙特的消费热潮就会引发道德、平等和基本人权等问题。
  </p>
  <p>
   <img alt="" class="aligncenter wp-image-3798 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/推特.png?resize=800%2C803&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/推特.png?resize=1020%2C1024&amp;ssl=1 1020w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/推特.png?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/推特.png?resize=300%2C300&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/推特.png?resize=768%2C771&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/推特.png?resize=1100%2C1104&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/推特.png?w=1152&amp;ssl=1 1152w" data-recalc-dims="1" height="803" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/推特.png?resize=800%2C803&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>
   <noscript>
    <img alt="" class="aligncenter wp-image-3798" data-recalc-dims="1" height="803" sizes="(max-width: 800px) 100vw, 800px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/推特.png?resize=800%2C803&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/推特.png?resize=1020%2C1024&amp;ssl=1 1020w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/推特.png?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/推特.png?resize=300%2C300&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/推特.png?resize=768%2C771&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/推特.png?resize=1100%2C1104&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/推特.png?w=1152&amp;ssl=1 1152w" width="800"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   沙特仍然有
   <a class="markup--anchor markup--p-anchor" data-href="https://www.recode.net/2018/10/24/18020282/saudi-arabia-silicon-valley-investors-venture-capital-money" href="https://www.recode.net/2018/10/24/18020282/saudi-arabia-silicon-valley-investors-venture-capital-money" rel="noopener noreferrer" target="_blank">
    六家国有企业
   </a>
   在科技生态系统中投入了大量资金。它们的范围从看似无害的阿卜杜拉国王科技大学到更明显与国家相关的公共投资基金 —— 所有这些都与同一个王室有关。
  </p>
  <p class="graf graf--p">
   但由于日本软银的450亿美元投资，
   <strong class="markup--strong markup--p-strong">
    沙特已经成为硅谷最强大的风险投资公司背后最强大的力量。
   </strong>
  </p>
  <p class="graf graf--p">
   与其他国际投资者一样，SoftBank
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theinformation.com/articles/under-scrutiny-softbank-tries-new-deal-structures?utm_source=newsletter&amp;utm_medium=email&amp;utm_campaign=newsletter_axiosprorata&amp;stream=top#_ga=2.229373343.2071010782.1556163075-2036980734.1520901066" href="https://www.theinformation.com/articles/under-scrutiny-softbank-tries-new-deal-structures?utm_source=newsletter&amp;utm_medium=email&amp;utm_campaign=newsletter_axiosprorata&amp;stream=top#_ga=2.229373343.2071010782.1556163075-2036980734.1520901066" rel="noopener noreferrer" target="_blank">
    也有 CFIUS 问题
   </a>
   ，但这个愿景基金是后期风险投资的主导力量。它提供了一种基本上没人在硅谷提供的产品：友好的检查、对创业公司创始人的高估值，这样他们就可以永远保持私有化。
  </p>
  <p class="graf graf--p">
   在 Khashoggi 被谋杀之后，Vision Fund 只遭受了极少的反弹。
   <strong class="markup--strong markup--p-strong">
    它现有的投资组合公司对 Vision Fund 或其沙特支持几乎没说过任何话。
   </strong>
  </p>
  <p class="graf graf--p">
   该基金与该国一些最有前途的初创公司达成了数十亿美元的交易 —— 即使是在 Khashoggi 去世后 —— 首席执行官偶尔会告诉记者，他们看到沙特与软银之间“存在一定距离”，
  </p>
  <p class="graf graf--p">
   是的，一些风险资本家，比如 Felicis Ventures 的 Wesley Chan 和 Founders Fund 的 Keith Rabois 都认为沙特的钱是血钱，而硅谷应该拒绝它。然而事实是，他们正在输掉辩论。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “多数人的态度是不评论，普遍的态度是所有钱都是钱，也有一种更尖锐的态度，’我不会接受坏人的钱’”，联合创业公司 Knotel 的首席执行官 Amol Sarva 说，“
   <strong class="markup--strong markup--p-strong">
    尖锐的态度也是道德的态度，但它的问题在于你很难言行合一”。
   </strong>
  </p>
  <p class="graf graf--p">
   这是因为在有道德意识的投资者看来，那些主张正义的人你们在2019年拿到的钱真的都那么干净吗？
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    世界是如此复杂。当你处理巨额美元数字时，你处理的实际上是地缘政
   </strong>
   治”，Sarva 说。
   <a class="markup--anchor markup--p-anchor" data-href="https://therealdeal.com/2019/04/24/knotel-in-talks-for-roughly-200m-series-c-sources/" href="https://therealdeal.com/2019/04/24/knotel-in-talks-for-roughly-200m-series-c-sources/" rel="noopener noreferrer" target="_blank">
    据报道
   </a>
   ，Sarva的创业公司正在从科威特的主权财富基金筹集资金。
  </p>
  <p class="graf graf--p">
   如果你不从沙特阿拉伯拿钱，最好也不过是从新加坡的 Temasek 拿钱 —— 一般认为后者不是硅谷外国货币问题的一部分 —— 但是有人会说新加坡代表了一个不容忍言论自由的压抑社会；据知情人士透露，来自马来西亚主权财富基金的 Khazannah 的资金被创业公司和创始人质疑其与1MDB丑闻之间存在关系；或者拿卡塔尔的投资是否意味着一个拥有糟糕的人权记录并支持恐怖主义的国家的支持？
  </p>
  <p class="graf graf--p">
   或者风险投资公司是否不应该从涉嫌爆炸性贿赂丑闻的精英大学的捐赠中获取资金？可是任何一家风险投资公司都不会拒绝硅谷最受欢迎的有限合伙人 Yale。那不会发生。
  </p>
  <figure aria-describedby="caption-attachment-3799" class="wp-caption aligncenter" id="attachment_3799" style="width: 1024px">
   <img alt="" class="size-large wp-image-3799 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/硅谷.png?resize=1024%2C671&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/硅谷.png?resize=1024%2C671&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/硅谷.png?resize=300%2C197&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/硅谷.png?resize=768%2C503&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/硅谷.png?resize=1100%2C721&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/硅谷.png?w=1456&amp;ssl=1 1456w" data-recalc-dims="1" height="671" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/硅谷.png?resize=1024%2C671&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>
   <noscript>
    <img alt="" class="size-large wp-image-3799" data-recalc-dims="1" height="671" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/硅谷.png?resize=1024%2C671&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/硅谷.png?resize=1024%2C671&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/硅谷.png?resize=300%2C197&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/硅谷.png?resize=768%2C503&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/硅谷.png?resize=1100%2C721&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/硅谷.png?w=1456&amp;ssl=1 1456w" width="1024"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-3799">
    Will foreign money ever leave Silicon Valley? Justin Sullivan/Getty Images
   </figcaption>
  </figure>
  <p class="graf graf--p">
   中国和沙特阿拉伯的经验表明，外国资金将继续留在硅谷—— 直到政府决定真正关心和干涉。风险投资行业不会自我监管。初创公司几乎没有表现出对VC进行监管的任何意愿。
  </p>
  <p class="graf graf--p">
   但是华盛顿呢？如果特朗普政府真的想遏制沙特的影响力，那么联邦政府就可以像对待中国人一样对待沙特人。显然华盛顿不会这样做。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果资金与我们的道德规范能保持一致当然很好，但事实上并不总是这样，” 网络安全投资者 Rob Ackerman 说， “这只是人性的现实。”
  </p>
  <p>
   “IF YOU’RE AN ENTREPRENEUR AND YOU HAVE CHOICES, WHY WOULD YOU TAKE MONEY FROM THEM? UNLESS YOU’RE A MASOCHIST.”
   <a href="https://www.vox.com/recode/2019/5/1/18511540/silicon-valley-foreign-money-china-saudi-arabia-cfius-firrma-geopolitics-venture-capital" rel="noopener noreferrer" target="_blank">
    Silicon Valley is awash in Chinese and Saudi cash — and no one is paying attention
   </a>
   (except Trump)
  </p>
