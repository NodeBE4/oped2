   <li>
    <span style="color: #00ccff;">
     <em>
      中国读者也许一提到 Facebook 只是想到扎克伯格一个人，不，扎克伯格可不是一般的“创业者”或者什么“青年才俊”，Facebook 与政权之间有着深层且紧密的结构。本文尝试通过开源数据来挖掘这些政治旋转门。再次提醒，谷歌也一样。并且，这些巨头之间很可能并非竞争关系
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p graf-after--p" id="ffa6">
   本文将尝试绘制 Facebook 内部的社会结构和公司的权力关系。您将能看到，为什么 Facebook 是无法被扳倒的、为什么出现如此严重的隐私出卖之后国会的问询依旧隔靴搔痒，为什么除非打破数字极权整体架构否则无法实现变革，以及，请重读我们早前的建议《
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/steemit/@iyouport/facebook-distributed-social-network" href="https://steemit.com/steemit/@iyouport/facebook-distributed-social-network" rel="noopener nofollow noreferrer" target="_blank">
    如何才能挑战 Facebook 并取得胜利？ — — Distributed Social Network
   </a>
   》。
  </p>
  <p class="graf graf--p graf-after--p" id="3926">
   <strong class="markup--strong markup--p-strong">
    本文是对当代政治机构与全球商业之间密切关系的揭露和批评，即对政治旋转门问题提出的警告
   </strong>
   。简而言之，我们希望能探讨如何处理监视资本主义这个利维坦。
  </p>
  <p class="graf graf--p graf-after--p" id="8b8b">
   为了掌握 Facebook 的就业结构，我们使用了 1000 人的公开 LinkedIn 个人资料，表明 Facebook 是他们的雇主，以及整个 Facebook 管理层的传记。通过将他们的社会背景、教育、地位、和现在的职位，映射到公司的层次结构中，我们深入了解了
   <strong class="markup--strong markup--p-strong">
    “Facebook 政府”
   </strong>
   整体的各种社会关系。
   <strong class="markup--strong markup--p-strong">
    这些见解可用于解释公司和网络相关参与者的一些行为以及商业模式的演变，并有助于尝试预测未来的发展
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="e3dd">
   Facebook 被理解为“超级集体”，涉及一系列复杂的
   <strong class="markup--strong markup--p-strong">
    不透明决策
   </strong>
   。这种分析应该有助于我们认识到为什么它是像 Facebook 这样的公司可以存在的唯一方式。
  </p>
  <p class="graf graf--p graf-after--p" id="4d68">
   本调查采用植根于网络理论的方法、和新闻学研究中的网络分析，重点关注不同社会行为者的关系。此外，在
   <a class="markup--anchor markup--p-anchor" data-href="http://www.lombardinetworks.net/" href="http://www.lombardinetworks.net/" rel="noopener nofollow noreferrer" target="_blank">
    Mark Lombardi 的概念艺术绘画
   </a>
   中发现了数据可视化的灵感，在 Josh On 和 Littlesis 等媒体艺术作品中发现了
   <a class="markup--anchor markup--p-anchor" data-href="http://www.theyrule.net/" href="http://www.theyrule.net/" rel="noopener nofollow noreferrer" target="_blank">
    他们的规则
   </a>
   ……所有提到的作品都为我们提供了一种方法，将话语与设施/设备分析和艺术活动的工具结合起来，
   <strong class="markup--strong markup--p-strong">
    用于（重新）概念化和可视化这些结果
   </strong>
   。
  </p>

   <h3 class="graf graf--h4 graf-after--figure" id="cac0">
    <strong>
     <span style="color: #ff6600;">
      — — 知识劳工贵族 — —
     </span>
    </strong>
   </h3>
   <p class="graf graf--p graf-after--h4" id="d888">
    本文想要更多地了解 Facebook 公司封闭社会内外的特点，他们的阶层嵌入性以及他们的人际关系网络; 我们想了解这些阶层的具体结构，更多地了解这些阶层所包含的人，他们是如何在世界各地游走或迁移的。
   </p>
   <p class="graf graf--p graf-after--p" id="3ad0">
    我们也有兴趣了解马克·扎克伯格（Mark Zuckerberg）统治的该公司的所有人的角色，构建
    <strong class="markup--strong markup--p-strong">
     Facebook 金字塔
    </strong>
    。我们考虑到了阶级决定论的概念，以及这些阶级自我复制并创造其成员以令其构成最佳运作环境的想法。Bourdieu 关于社会空间、习惯、文化、教育/信息和社会资本的观念（Bourdieu 1982，1997年）在本研究之前和期间的反思中都存在。
   </p>

   <p class="graf graf--p graf-after--figure" id="e9d2">
    我们几乎可以将任何一类 Facebook 员工视为高薪知识劳工贵族的完美典范。这种模式与 IT 行业背后隐藏的其他形式的劳动力
    <strong class="markup--strong markup--p-strong">
     完全相反
    </strong>
    ，“如采矿的奴隶型劳动力、军事控制和高度开发的硬件装配工人、不稳定的数字服务工作者、受到帝国剥削的知识工作者、从事电子废物工业回收和管理的工人、或其他劳动力“（Fuchs 2016：61）。
   </p>
   <p class="graf graf--p graf-after--p" id="3eca">
    在 Facebook 这个知识劳工金字塔的顶端，我们看到了一个
    <strong class="markup--strong markup--p-strong">
     “硅苏丹”
    </strong>
    的小圈子，它控制并影响该领域最大公司的个人网络，这些人坐在董事会席位上拥有他们的部分股份。

     美国的最初积累显示出与硅谷寡头集团有一些有趣的相似之处。
    </p>
    <p class="graf graf--p graf-after--p" id="141e">
     19 世纪下半叶的特点是为美国单一市场创建了基础设施（铁路），并通过向工业社会的过渡，将创新产品带到了家门口。19 世纪的美国“强盗贵族们”
     <strong class="markup--strong markup--p-strong">
      通过挤压竞争来积累巨额财富，就像今天的“硅苏丹”一样
     </strong>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="5e3a">
     一个说明性的比较将告诉我们：“洛克菲勒曾控制过世界 80％ 的石油供应; 今天谷歌拥有欧洲 90％ 的搜索市场和 67％ 的美国搜索市场，而 Facebook 拥有 42％ 的社交媒体市场份额“（Economist 2014）。这两组基础设施先驱之间的相似性导致了
     <strong class="markup--strong markup--p-strong">
      一个警告性结论
     </strong>
     ：在全球范围内带头将工业时代转变为信息社会的人物“被指为贪婪的商界人士，他们眷顾政治家、雇用血汗工厂的劳动、挤压其他股东，尤其是垄断市场”（Economist 2014）。曾经被视为“创造性破坏者”的硅谷企业现在不顾一切地“解决人类问题”，从衰老到太空旅行，
     <strong class="markup--strong markup--p-strong">
      将他们的权利置于整个世界变得愈加依赖的数据行业的顶端
     </strong>
     。
    </p>

    <p class="graf graf--p graf-after--p" id="5eac">
     <strong class="markup--strong markup--p-strong">
      Marc Andreessen：权力的纽带
     </strong>
     — — 在右下角，你能找到 Marc Andreessen，他是扎克伯格在 Facebook 最亲密的董事会成员之一。正如我们所看到的，他曾经在伊利诺伊大学接受教育，这是他的传记的一部分，并没有将他与他的任何其他 FB 同事联系起来。然而，他也是硅谷“Andreessen Horowitz”最具影响力的风险投资公司之一的合伙人。（参考：1、
     <a class="markup--anchor markup--p-anchor" data-href="http://a16z.com/" href="https://a16z.com/" rel="noreferrer noopener nofollow" target="_blank">
      http://a16z.com/
     </a>
     ；2、
     <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Andreessen_Horowitz" href="https://en.wikipedia.org/wiki/Andreessen_Horowitz" rel="noreferrer noopener nofollow" target="_blank">
      https://en.wikipedia.org/wiki/Andreessen_Horowitz
     </a>
     ）
    </p>
    <p class="graf graf--p graf-after--p" id="9641">
     通过这家公司，
     <strong class="markup--strong markup--p-strong">
      Andreessen 与一些非常重要的公司有联系，如 Foursquare，Groupon，Skype，Twitter，eBay，AOL 和 GitHub❗️
     </strong>
     。他是一个非常强大的权力者案例，他在市场上具有相当综合的功能，因为他的业务关系连接了该领域的主要参与者。（拥有股票的百分比并不意味着一个人掌握技术。然而，Andreessen 通常是投资者或上述公司的董事会成员，这当然使他有可能通过技术和内幕知识从内部获得任何类型的信息）
    </p>
    <p class="graf graf--p graf-after--p" id="868c">
     如上图所示，我们可以看到 Andreessen 对通信和金融服务感兴趣：通过他的风险投资公司，
     <strong class="markup--strong markup--p-strong">
      他曾经是 Skype 的投资者，而且他仍然是 eBay 的董事会成员，两家公司的前任副总裁是 Dan Neary，他目前是 Facebook 的二级管理人员，曾在 Sheryl Sandberg 的亚太市场工作
     </strong>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="6ff5">
     <strong class="markup--strong markup--p-strong">
      Peter Thiel：代理和分析
     </strong>
     — — Peter Thiel 在左下角，此人是硅谷最有影响力的人物之一，也是 FB 管理层成员，还是 LinkedIn 的早期投资者。他更是世界知名品牌 PayPal，Clarium Capital（全球宏观对冲基金），Founders Fund（风险投资公司），Valar Ventures，Mithril Capital 的联合创始人，并曾担任 Y Combinator 的合伙人。无疑是风险资本领域最有影响力的人物，
     <strong class="markup--strong markup--p-strong">
      将他的影响力扩大到数百家甚至数千家创业公司
     </strong>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="4f35">
     Thiel 成立的公司之一正在引起特别的关注：Palantir Technologies，这是一家分析软件公司。泄露给 TechCrunch 的
     <a class="markup--anchor markup--p-anchor" data-href="https://techcrunch.com/2015/01/11/leaked-palantir-doc-reveals-uses-specific-functions-and-key-clients/" href="https://techcrunch.com/2015/01/11/leaked-palantir-doc-reveals-uses-specific-functions-and-key-clients/" rel="noopener nofollow noreferrer" target="_blank">
      文件显示
     </a>
     ，
     <strong class="markup--strong markup--p-strong">
      截至 2013 年，Palantir 的客户包括至少 12 个美国政府机构，包括中央情报局，国土安全部，国家安全局，联邦调查局，疾控中心，海军陆战队，空军，特种作战司令部，西点军校等❗️
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="2094">
     <strong class="markup--strong markup--p-strong">
      该公司最初由中央情报局的非营利性风险投资部门 In-Q-Tel 资助，并被不同的政府机构使用
     </strong>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="2293">
     尽管此人在硅谷的一些亲密同事并不热衷于此，但
     <strong class="markup--strong markup--p-strong">
      他自己还是美国总统唐纳德特朗普的顾问
     </strong>
     ，也是特朗普与科技界的桥梁。除了 Erskine Bowles（将在下面进一步提及）之外，这是
     <strong class="markup--strong markup--p-strong">
      Facebook 管理委员会与政治和政党的第二个重要联系
     </strong>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="e2cc">
     <strong class="markup--strong markup--p-strong">
      Sheryl Sandberg：政府、金融部门和公司
     </strong>
     — — Sheryl Sandberg 在 Facebook 的职位是首席运营官和董事。她控制与小企业、广告和全球运营、全球营销、游戏销售等相关的业务。
    </p>
    <p class="graf graf--p graf-after--p" id="e914">
     在“Facebook之前的生活”这本书中，她写道在哈佛商学院的学习使她成为教授的助手，随后成为
     <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/United_States_Secretary_of_the_Treasury" href="https://en.wikipedia.org/wiki/United_States_Secretary_of_the_Treasury" rel="noopener nofollow noreferrer" target="_blank">
      <strong class="markup--strong markup--p-strong">
       美国财政部参谋长
      </strong>
     </a>
     。除了参加从企业和政府中收集有影响力的女性的一系列基金会和非营利组织之外，她还是全球发展中心或布鲁金斯学院等智囊团的董事会成员（其中包括其他任务）处理美国的防务政策，但一个有趣的事实是，她还是星巴克咖啡的董事会成员。
    </p>
    <p class="graf graf--p graf-after--p" id="52ff">
     这显示了一种相当有趣的方式，所有提到的参与者绝不是将他们的商业利益限制在数字或在线领域，相反，他们也对其他收入来源感兴趣。
     <strong class="markup--strong markup--p-strong">
      Sheryl Sandberg 的例子清楚地表明了国家机构和私人资本之间在个人层面上的紧密联系。她代表的是一位在商界非常有影响力的公务员
     </strong>
     。
    </p>
    <h3 class="graf graf--p graf-after--p" id="3af7">
     <span style="color: #ff6600;">
      <strong class="markup--strong markup--p-strong">
       与政治和政党的关系密切
      </strong>
     </span>
    </h3>
    <p class="graf graf--p graf-after--p" id="35ca">
     董事会成员 Erskine Bowles（在左上角看到）来自
     <strong class="markup--strong markup--p-strong">
      马克扎克伯格最亲密的内圈，他也是白宫的参谋长，并且是国家财政责任与改革委员会的
     </strong>
     <a class="markup--anchor markup--p-anchor" data-href="https://www.fiscalcommission.gov/" href="https://www.fiscalcommission.gov/" rel="noopener nofollow noreferrer" target="_blank">
      <strong class="markup--strong markup--p-strong">
       联合主席
      </strong>
     </a>
     。他本人在 2010 年
     <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/National%20_Commission_on_Fiscal_Responsibility%20_and_Reform" href="https://en.wikipedia.org/wiki/National%20_Commission_on_Fiscal_Responsibility%20_and_Reform" rel="noopener nofollow noreferrer" target="_blank">
      帮助建立的机构
     </a>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="3830">
     他通过在摩根士丹利的
     <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Morgan_Stanley" href="https://en.wikipedia.org/wiki/Morgan_Stanley" rel="noopener nofollow noreferrer" target="_blank">
      工作经验
     </a>
     、和通用汽车的技术行业与金融业
     <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/General_Motors" href="https://en.wikipedia.org/wiki/General_Motors" rel="noopener nofollow noreferrer" target="_blank">
      的联系
     </a>
     ，展示了 Facebook 可以通过其董事会成员所依赖的那种系统性支持。
    </p>
    <p class="graf graf--p graf-after--p" id="81ba">
     <strong class="markup--strong markup--p-strong">
      Erskine Bowles 代表金融、技术和 IT 产业与政治和民主党的联系
     </strong>
     。这些联系确实表明了政治行政部门的行为者与私营企业的共同利益。
    </p>
    <p class="graf graf--p graf-after--p" id="eb44">
     <strong class="markup--strong markup--p-strong">
      从欧洲公共和政治传统的角度来看，在所提及的有利可图的业务背景下的这种政治参与是有问题的。从民权和隐私政策的角度来看，政府机构、秘密情报部门和社交网络（即通信基础设施）之间强有力的制度联系只能是批判性的
     </strong>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="df11">
     这些例子表明精英是如何与政治机构合并以集中力量的。但是，我们提醒读者，这种现象通常不会被视为错误，尤其是在美国，因为所有这些人公开谈论这些“成就”是非常积极的 — — 是的，
     <strong class="markup--strong markup--p-strong">
      中国也一样，这些“精英”以攀权附势为荣而不是为耻
     </strong>
     。与此同时，在其他工业部门和其他国家机构之间也可以找到类似的联系。
     <strong class="markup--strong markup--p-strong">
      如果没有这些攀权附势的能力，像 Facebook 这样的公司实现如今的全球霸权是不可能的
     </strong>
     。
    </p>

       </p>
       <p class="graf graf--p graf-after--p" id="d0b7">

       </p>
       <p class="graf graf--p graf-after--p" id="c4d2">
        拥有大数量 Facebook 员工的
        <strong class="markup--strong markup--p-strong">
         第二名国家是英国
        </strong>

       </p>
       <p class="graf graf--p graf-after--p" id="9447">
        当我们谈论城市中心时，为 Facebook 工作的人数最多的是在旧金山地区。然而，对 Facebook 而言，
        <strong class="markup--strong markup--p-strong">
         第二大城市并非在美国，而是在英国
        </strong>
        — 伦敦。我们看到了诸如生活地点等细节个人信息的数据不一致和存在巨大差异的原因。确认数据集问题的一个事实是，在城市关键字上我们也发现了新加坡或爱尔兰等国家的城市，出于同样的原因 — 人员太少，我们认为美国其他一些城市因此没有必要列入此列表。
       </p>
       <p class="graf graf--p graf-after--p" id="6fba">
        对于专业活动领域，绝大多数人在他们的帐户中阐述了他们是软件工程师。以下的其他领域分别为互联网、信息技术和服务、人员配置和招聘、市场营销和广告。这些都是值得注意的，并且可以重新组合以构建更大的字段，因为它们中至少有一些相同的活动。
       </p>
       <p class="graf graf--p graf-after--p" id="7e20">
        在此图的底部，我们可以看到 LinkedIn 个人资料中列出的职位列表。最大的一组预期是软件工程师。最后，陈述了一些较小的专业（例如工程经理、研究科学家、产品经理等），这些专业也可以在更大的领域被重新组合，或者在某种程度上被添加到最大的组中。
       </p>
       <p class="graf graf--p graf-after--p" id="e3bb">

         </p>
         <p class="graf graf--p graf-after--p" id="7afd">
          即使世界正处于全球后发展的阶段，公司在经济、政治方面的深刻内涵和社会精英/建立一个社会/国家是使公司足够强大以实现全球化的原因 — — 而不是像通常认为的那样，通过世界各地精英的合作。
         </p>
         <p class="graf graf--p graf-after--p" id="7d29">
          用金字塔或
          <a class="markup--anchor markup--p-anchor" data-href="https://labs.rs/en/colonization-with-love/" href="https://labs.rs/en/colonization-with-love/" rel="noopener nofollow noreferrer" target="_blank">
           利维坦
          </a>
          来形容就很方便了，因为许多社会等级和面孔构成了一个中心人物，权力的中心。同时，正如本调查所显示的那样，网络的真正结构还包括公司更高层次中特定人员的个人社交网络。
          <strong class="markup--strong markup--p-strong">
           如果除去利润不做考虑的话，这就是将整个结构保持在一起并且不受任何政治制度变化所影响的原因
          </strong>
          。
         </p>
         <figure aria-describedby="caption-attachment-964" class="wp-caption aligncenter" id="attachment_964" style="width: 1920px">
          <img alt="" class="size-full wp-image-964 jetpack-lazy-image" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/04/9.gif?resize=1100%2C866&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" height="866" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/04/9.gif?resize=1100%2C866&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
          <noscript>
           <img alt="" class="size-full wp-image-964" data-recalc-dims="1" height="866" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/04/9.gif?resize=1100%2C866&amp;ssl=1" width="1100"/>
          </noscript>
          <figcaption class="wp-caption-text" id="caption-attachment-964">
           亚伯拉罕·博斯为托马斯·霍布斯（Thomas Hobbes）撰写的“利维坦”（Leviathan）一书所作的插图（1668）
          </figcaption>
         </figure>
         <p class="graf graf--p graf-after--figure" id="62bd">
          正如我们所看到的，它实际上是一个由许多结构组成的相对复杂的网络，其动力由特定参与者的利益所驱动。这种规模的公司一直在影响、组织和决定精英的生活、全球经济、“小人物”或“普通用户”的日常生活。
         </p>
         <p class="graf graf--p graf-after--p" id="c160">
          重要的是要精确地描述网络的哪些部分以及哪些部分与哪些参与者相关联。一旦我们理解了大公司与政治的深层交织，就更容易认识和阐明这些力量在政治舞台上的支持。
         </p>
         <p class="graf graf--p graf-after--p" id="b03a">
          <strong class="markup--strong markup--p-strong">
           本文仅仅以 Facebook 为例，仅仅基于开源数据，没有任何对内部举报人的依赖、没有任何黑客行为，已经足够揭示一个利维坦的内幕。希望这一开源调查思路演示能帮助中国的积极人士展开对权力关系的深度挖掘（我们对宫廷八卦和小道消息毫无兴趣）任何时候，拿出数据为证，将无人能辩驳
          </strong>
          。⚪️
         </p>
        </div>
