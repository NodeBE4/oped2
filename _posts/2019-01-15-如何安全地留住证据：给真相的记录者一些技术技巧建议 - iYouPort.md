  <p class="graf graf--p graf-after--figure" id="2747">
   如果你正在记录侵犯人权的行为、社会敏感问题、政府阴暗面，互联网可以帮助将真相传播到前所未有的广度让人们知晓，同时也可能会让你入狱。无论你是保护环境的活动家、记录系统性国家暴力的公民记者、还是驻扎冲突地区的独立调查人士，
   <strong class="markup--strong markup--p-strong">
    都要重点考虑数字记录和传输的安全问题，这不仅仅是为了你自己。如果您无法妥善保护视频、照片和通信记录，那么您与之通信或传输文档的每一个人都将面临风险
   </strong>
   。只有更好的安全性才能更好地工作。
  </p>
  <p class="graf graf--p graf-after--p" id="04da">
   在中国，普通人也面临着极其严重的政府监视和间谍行为，安全知识、威胁警报和技术援助对中国社会来说尤其重要。然而很多人并没有重视起证据保护的环节，抓不住重点的拍摄记录反而令当局掌握了辩解的借口；被查抄的资料导致记录者和相关人等被逮捕、真相被掩盖；最终只有极少的证词而没有可靠的资料，很难平冤。
  </p>
  <p class="graf graf--p graf-after--p" id="3fea">
   最初，数字威胁一直被认为是压倒性的，许多人认为需要特殊的技术专业知识才能理解和减轻所面临的安全威胁。中国社会似乎至今如此。这也是为什么隐私保护知识的传播非常困难。但这是一个错误的观念，虽然可以理解。
  </p>
  <p class="graf graf--p graf-after--p" id="d09c">
   事实上，提高您的数字安全性只是意味着需要弄清楚您自己所面临的风险是什么，并由此判断您愿意学习哪些工具和技术来保护自己和周围的人。
  </p>
  <p class="graf graf--p graf-after--p" id="3c88">
   安全培训师称此流程为风险评估或威胁建模。这听起来似乎有点复杂，但有如今许多组织都有在线创建资源，可以帮助您评估和满足您的数字安全需求。本文列出了一些类似的指南，以及它们可以教给您的内容。
  </p>
  <p class="graf graf--p graf-after--p" id="601e">
   <strong class="markup--strong markup--p-strong">
    评估您的风险
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="3d25">
   要得到更好的安全性的第一步是威胁建模或风险评估。这只需要问自己以下五个问题。考虑拿出纸笔，并可以和与您密切合作的人一起讨论这些问题，因为
   <strong class="markup--strong markup--p-strong">
    安全是一项集体的努力，一个人的安全可能无法解决整体问题
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li graf-after--p" id="1d8c">
    我需要保护什么？
   </li>
   <li class="graf graf--li graf-after--li" id="7a64">
    我需要防御的是谁？
   </li>
   <li class="graf graf--li graf-after--li" id="7c3e">
    他们想要多少信息，以及获取信息有多容易？
   </li>
   <li class="graf graf--li graf-after--li" id="42d7">
    如果他们得到我要保护的东西会发生什么？
   </li>
   <li class="graf graf--li graf-after--li" id="4207">
    我愿意做些什么来阻止这种情况发生？
   </li>
  </ul>
  <p class="graf graf--p graf-after--li" id="6464">
   就这么简单。让我们使用一个具体的例子以便更详细地看下这些问题。也许你拍摄了一名警察殴打抗议者的视频，这时候您就非常有必要保护您所创建的材料，因为当局会想尽一切办法销毁它，甚至将您逮捕。
  </p>
  <p class="graf graf--p graf-after--p" id="4e50">
   不仅需要保护收集的数据不落入坏人之手，您可能还需要保护抗议者的真实身份。为了您自己的安全，您可能需要隐藏您拍摄了事件这一事实。总之需要保护所有这一切不受警察的影响。但也许您还需要将更多信息隐藏起来，例如，您的工作地点、真实姓名等，因为您可能因为“激进”而被解雇。这只是需要保护的大量案例之一，你必须仔细考虑自己的情况。
  </p>
  <p class="graf graf--p graf-after--p" id="bf7c">
   一旦你弄清了需要保护什么以及防御什么人，接下来就会想到实际的风险是什么。也许您已经知道，警察经常会抢走人们的手机和电脑，他们对拍摄记录非常生气。而且你不知道他们有什么技术，但你需要怀疑他们可以通过你的手机了解到关键信息。
  </p>

  <p class="graf graf--p graf-after--p" id="cc19">
   然后，如果您的安全性受到损害，您必须考虑后果，以及您为避免这些后果而愿意采取的措施。例如，警察抢走你的拍摄资料的后果可能只是你无法再公开这些真相了，但如果他们对被拍摄感到非常愤怒，可能会导致你最终受到警察的骚扰和威胁。也许这名遭到严重殴打的抗议者被反咬为“打警察”（这种事经常发生），并且由于你丢失了自己拍摄到的视频真相，这些指控就不会被取消。在这些情况下，您自己和您涉及的人的生命都可能处于危险之中。
  </p>
  <p class="graf graf--p graf-after--p" id="561e">
   最后，使用这些信息，您可以决定愿意采用哪些工具以及愿意改变哪些习惯性做法，以避免这种后果。例如，在手机上启用加密是一个简单的步骤，可能会使警方更难获取您的数据。也许你非常关心你拍摄的人并且不希望他们的身份被公开 — 他们所面临的报复风险可能非常大。在这种情况下，您可以考虑下载和
   <strong class="markup--strong markup--p-strong">
    使用
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://securityinabox.org//en/guide/obscuracam/android" href="https://securityinabox.org//en/guide/obscuracam/android" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     ObscuraCam
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    或

    </strong>

   <strong class="markup--strong markup--p-strong">
    等技术来删除视频的可识别部分
   </strong>
   。这可能比开启加密要稍微复杂一些，但它可以挽救一个人的生命。
  </p>
  <p class="graf graf--p graf-after--p" id="0924">
   <strong class="markup--strong markup--p-strong">
    学习如何保护自己和他人
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="ac60">
   我们都知道，任何时候证据都很重要。政府的恶行最怕被揭发和公之于众，中国当局称其为“负面消息”而被禁止传播，因为那是他们的罪证。
  </p>
  <ul class="postList">
   <li class="graf graf--li graf-after--p" id="4bc2">
    肯尼亚公民组织拍摄了几个小时的公民证词和现场录像，关于他们的房屋被政府强行拆除的证据，最终非洲最高地区法院作出判决，将土地还给人民。如果没有这些证据，就无法获得这样的裁决；
   </li>
   <li class="graf graf--li graf-after--li" id="af9e">
    一名美国公民拍摄了警察打人的视频，然而她被警察诬告“袭警”，在她被强行居留的时候，警方删除了她设备上储存的视频，也就是警察打人的证据。但警察没想到她首先将视频上传到“云端”了，最终她拿出证据证明了自己的清白；
   </li>
   <li class="graf graf--li graf-after--li" id="4a53">
    危地马拉的活动家拍摄了政府暴行的证据，记录了当地人民抗争的过程，作为人权纪录的铁证……
   </li>
  </ul>

     和所有当权者一样，缅甸军方一直在试图狡辩，而活动家制作的数据日志清楚地显示了清除行动是如何在8月25日之前开始的，当时据报道发生了反叛袭击，有一些被军方挖出的残缺不全的乱坟坑，当局辩称这些尸体是由罗兴亚叛乱分子屠杀的印度教徒。照片被迅速传播到社交媒体上作为对罗兴亚人的污蔑。
    </p>
    <p class="graf graf--p graf-after--p" id="d2dc">
     但是没有用，因为证据已经被记录了，现在所有人都知道被杀的人是谁。Hanan 与孟加拉国难民营的一位记者密切合作，帮助在线研究和发布信息，有助于即时揭示政权的谎言。
    </p>
    <p class="graf graf--p graf-after--p" id="51aa">
     准确识别和对付推特水军也是一项重要工作，要防止他们试图污染和掩盖消息。由于账户创建时间可见，并且他们均发布几乎一样的垃圾信息，于是识别并不难，同时注意及时揭露。
    </p>
    <p class="graf graf--p graf-after--p" id="cb4c">
     <strong class="markup--strong markup--p-strong">
      <em class="markup--em markup--p-em">
       延伸阅读：《
      </em>
     </strong>
     <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%B7%A8%E9%AD%94%E5%B4%9B%E8%B5%B7-%E6%94%BF%E5%BA%9C%E8%B5%84%E5%8A%A9%E7%9A%84%E7%BD%91%E7%BB%9C%E6%B0%B4%E5%86%9B%E6%AD%A3%E9%81%8D%E5%B8%83%E5%85%A8%E7%90%83-a45710e8e431" href="https://www.iyouport.org/%E5%B7%A8%E9%AD%94%E5%B4%9B%E8%B5%B7%EF%BC%9A%E6%94%BF%E5%BA%9C%E8%B5%84%E5%8A%A9%E7%9A%84%E7%BD%91%E7%BB%9C%E6%B0%B4%E5%86%9B%E6%AD%A3%E9%81%8D%E5%B8%83%E5%85%A8%E7%90%83/" rel="noopener noreferrer" target="_blank">

      </em>
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="3296">
     如果您是目击者，在发生重要事件的场合，大多时候需要手机来记录信息。使用手机记录的要点如下：
    </p>
    <ul class="postList">
     <li class="graf graf--li graf-after--p" id="f498">
      <strong class="markup--strong markup--li-strong">
       规划您的拍摄
      </strong>
      ：手机的存储容量有限，根据你需要录制的内容注意手机上可用的空间量，尽可能选择重点细节；
     </li>
     <li class="graf graf--li graf-after--li" id="6a66">
      <strong class="markup--strong markup--li-strong">
       捕获细节和视角很重要
      </strong>
      ：您需要迅速准确地抓取重点。手机具有广角镜头，具有有限的变焦能力。先从物理上靠近记录细节，然后向后移动以显示更广泛的事件视角；
     </li>
     <li class="graf graf--li graf-after--li" id="73c2">
      <strong class="markup--strong markup--li-strong">
       保持你的拍摄稳定
      </strong>
      ：不要来回挥动手机试图一次拍摄所有内容，请选准镜头，并保持稳定至少10秒，避免缩放，将肘部靠在身体上以增加支撑力和稳定性；
     </li>
     <li class="graf graf--li graf-after--li" id="5d3b">
      <strong class="markup--strong markup--li-strong">
       平稳地握住你的手机
      </strong>
      ：水平握住手机以捕捉更宽的角度，以便在电视或计算机显示器上播放时图像看起来更好；
     </li>
     <li class="graf graf--li graf-after--li" id="7651">
      <strong class="markup--strong markup--li-strong">
       注意音效
      </strong>
      ：现场证词很重要，您需要靠近受访者，以便麦克风可以获得更好的音频。要注意可能会淹没采访的环境吵闹声；
     </li>
     <li class="graf graf--li graf-after--li" id="5836">
      <strong class="markup--strong markup--li-strong">
       注意光线
      </strong>
      ：移动电话在低光照条件下录制效果不佳，请尽可能在光线充足的地方录制采访记录。为了获得最佳图像，请始终让光线保持在背后；
     </li>
     <li class="graf graf--li graf-after--li" id="40cd">
      <strong class="markup--strong markup--li-strong">
       人身安全
      </strong>
      ：请禁用 GPS，并在不使用后立即取下电池并关闭手机；
     </li>
     <li class="graf graf--li graf-after--li" id="cf7e">
      <strong class="markup--strong markup--li-strong">
       保护您的朋友和联系人
      </strong>
      ；如果您的手机被警方抢走，他们可以访问联系人姓名、通话记录、短信、照片、视频和音频文件。如果存在手机被没收的风险，请删除或加密所有敏感数据；
     </li>
     <li class="graf graf--li graf-after--li" id="114f">
      <strong class="markup--strong markup--li-strong">
       保护你的载体
      </strong>
      ：评估保护您的卡可以免受当局干扰的最佳方法。考虑使用空白卡更换旧卡，隐藏使用过的卡或将其交给朋友安全存储。如果您可以安全地连接到互联网，请考虑立即将媒体文件上传到稳妥的网站，或通过电子邮件或文本与受信任的朋友分享，以扩大备份量。
     </li>
    </ul>
    <p class="graf graf--p graf-after--li" id="57b2">
     以上是作为公民社会参与者的一些必备知识。技术、技巧、技能不仅需要学习，而且需要练习，在平日多多练习相关技术，熟悉操作方法，以便您能在突发事件来临时准确地采取这些方法。我们知道中国有很多有志之士，我们希望能在“装备”方面助大家一臂之力。
    </p>
