  <div class="section-inner sectionLayout--insetColumn">
   <p class="graf graf--p graf-after--figure" id="c892">
    Privacy International 的
    <a class="markup--anchor markup--p-anchor" data-href="https://www.privacyinternational.org/campaigns/state-sponsors-surveillance-governments-helping-others-spy" href="https://www.privacyinternational.org/campaigns/state-sponsors-surveillance-governments-helping-others-spy" rel="noopener nofollow noreferrer" target="_blank">
     新报告
    </a>
    显示，拥有强大安全机构的国家一直在为其他国家培训、装备监视设备，甚至直接为外国监视机构提供资金。
    <strong class="markup--strong markup--p-strong">
     在技​​术进步的推动下，加强监视既有助于全球威权主义的增强，也有助于增强提供国的权威主义本身，这是对民主、人民权利和法治构成严重攻击的行为。
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="75c4">
    该报告探讨了一系列“监视援助”计划的例子：
   </p>
   <ul class="postList">
    <li class="graf graf--li graf-after--p" id="67ce">
     提供了
     <strong class="markup--strong markup--li-strong">
      美国
     </strong>
     国务院、国防部和司法部如何促进外国监视能力的实例，以及大型武器公司如何融入这些计划的概述，包括美国的监视培训基地；
    </li>
    <li class="graf graf--li graf-after--li" id="7cb0">
     <strong class="markup--strong markup--li-strong">
      欧盟和欧洲各国
     </strong>
     正在全球范围内提供监视技术。欧盟已经花费数十亿美元在国外开发边境控制和监视能力，以阻止向欧洲的移民；
    </li>
    <li class="graf graf--li graf-after--li" id="2c90">
     中国政府的监视技术能力在所谓“一带一路”倡议和其他扩展国际市场的努力中也得到了支持。
    </li>
   </ul>
   <p class="graf graf--p graf-after--li" id="609e">
    这些计划不是将资源投入到长期发展解决方案中，而是进一步巩固了威权主义，并在世界范围内引发了权力滥用 — — 首先就是导致不安全的因素。隐私国际组织表示，如果这些“援助”国家真正想要帮助其他国家保持安全和稳定，他们应该建立学校、医院和其他基础设施，并促进民主和人权，而不是推广监视技术。
   </p>
   <p class="graf graf--p graf-after--p" id="cfac">
    本文是一个指南，介绍一些可用于研究和调查的开源数据资料，即
    <a class="markup--anchor markup--p-anchor" data-href="https://www.wix.com/dashboard/6fc0666c-c125-4701-93b5-303f8db9aa33/blog/5b7e2bed1445ca016550b0e1/edit/search/posts?query=%23OSINT" href="https://www.wix.com/dashboard/6fc0666c-c125-4701-93b5-303f8db9aa33/blog/5b7e2bed1445ca016550b0e1/edit/search/posts?query=%23OSINT" rel="nofollow noopener noreferrer" target="_blank">
     #OSINT
    </a>
    开源情报资源，欢迎中国读者加入调查。
   </p>
   <p class="graf graf--p graf-after--p" id="8862">
    <strong class="markup--strong markup--p-strong">
     培训和资金
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="2aa8">
    外国安全部队的培训和资金由国际机构、执法部门、军事和情报机构、以及司法和涉外部门完成。因此，获取数据取决于上述这些相关机构。其中许多提供了一些这些程序的一般信息，这些程序也可用于进一步的研究。
   </p>
   <p class="graf graf--p graf-after--p" id="a2a2">
    下面来演示这一调查。
   </p>
   <p class="graf graf--p graf-after--p" id="deb0">
    <strong class="markup--strong markup--p-strong">
     — — 美国 — —
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="1717">

    <a class="markup--anchor markup--p-anchor" data-href="https://www.state.gov/t/pm/rls/rpt/fmtrpt/index.htm" href="https://www.state.gov/t/pm/rls/rpt/fmtrpt/index.htm" rel="noopener nofollow noreferrer" target="_blank">

     <p class="graf graf--p graf-after--p" id="60a0">

      </strong>
      ，因此只能依赖于我们介绍给过的诸多调查研究技术、信息自由法案下的请求、内部文件泄漏和政府举报人。但后者如今变得越来越困难了，比如在美国，奥巴马政府八年内逮捕的举报人是他所有前任的总合还多，并且如今的特朗普政府看起来仍在延续这一逮捕判刑的措施，就在上周，第一个将内部文件提供给媒体以警告俄罗斯预计在大选前攻击选举系统的举报人，
      <a class="markup--anchor markup--p-anchor" data-href="https://arstechnica.com/tech-policy/2018/08/nsa-leaker-who-mailed-doc-outlining-russian-hacking-gets-5-years-in-prison/" href="https://arstechnica.com/tech-policy/2018/08/nsa-leaker-who-mailed-doc-outlining-russian-hacking-gets-5-years-in-prison/" rel="noopener nofollow noreferrer" target="_blank">
       被判刑
      </a>
      。
     </p>

      </strong>
      。
     </p>
     <p class="graf graf--p graf-after--p" id="084a">
      由于政治透明度越来越低，目前此处的开源调查只能使用 SII，它旨在通过提供公司信息帮助研究人员获取数据; 使用这种方法也可以进行进一步的研究，例如依照信息自由法案提交请求，虽然有难度，而且耗时。有一些一般信息可以在政府报告中找到。
     </p>
     <p class="graf graf--p graf-after--p" id="c0c7">
      <strong class="markup--strong markup--p-strong">
       信息自由法案听起来不错，但也不是万能的
      </strong>
      。许多二战后美国进行心理战的详细文件一直以来尘封于世，属于绝密文件受特殊法律保护，无法通过信息自由法案或任何正常渠道获得解密。直到国会对水门事件的调查才让一些实施心理战的神秘机构首次露出冰山一角。
     </p>
     <p class="graf graf--p graf-after--p" id="2442">
      一些国家，例如印度，公布了有关进口和出口的运输数据。在此数据中搜索监控公司名称（在 SII 中可用），或者使用监控行业常用的行业术语，可以找到进口监控设备。

       但是，
       <strong class="markup--strong markup--p-strong">
        通过此类请求获取信息很困难，并且取决于有关机构的限制性政策
       </strong>
       。直接装备国外同行的情报机构的行为，比如在英国等一些国家，会完全置身于任何信息自由立法案之外。
      </p>
      <p class="graf graf--p graf-after--p" id="68a4">
       世界各地的隐私权国际组织和其他非政府组织一直呼吁欧盟通过具有约束力的立法，而不是依赖此类披露，使欧盟强制要求监视技术出口的透明度报告 — 希望能在 2019 年的下一次欧洲议会选举之前通过。根据欧盟立法者准备采取的措施，最终要追求监视出口商、产品和最终用户的信息定期公开发布。
      </p>
      <p class="graf graf--p graf-after--p" id="86c8">
       这些改革的通过是很重要的; 透明度不仅对于提供有关监视技术行业和贸易的信息非常重要，而且对于解决腐败问题也至关重要，
       <strong class="markup--strong markup--p-strong">
        这也是民主的必要条件
       </strong>
       ：这些决定是在政府政策的基础上作出的。如果监督机构 — 议会和公众不知道这些决定是什么，他们就无法知道政策是否得到了遵守，更不用说政策本身是否是人民的民主意愿。
      </p>
      <blockquote class="graf graf--blockquote graf-after--p" id="ea95">
       <p>
        <em class="markup--em markup--blockquote-em">
         虽然威权政府继续保密，并且随着更民主的政府继续使用过于宽泛的所谓“国家安全”定义并优先考虑公司保密，世界各地的研究人员也都应该寻找和揭露监控技术的转移。否则只能看到监视技术被用来剥夺人民的权利，进一步推动威权主义以破坏民主。

          <img alt="" class="size-full wp-image-4709" data-recalc-dims="1" height="519" sizes="(max-width: 1067px) 100vw, 1067px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-109.png?resize=1067%2C519&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-109.png?w=1067&amp;ssl=1 1067w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-109.png?resize=300%2C146&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-109.png?resize=768%2C374&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-109.png?resize=1024%2C498&amp;ssl=1 1024w" width="1067"/>
         </noscript>
         <figcaption class="wp-caption-text" id="caption-attachment-4709">
          BAE Systems 向美国陆军出售信号监测设备
         </figcaption>
        </figure>
        <p class="graf graf--p graf-after--figure" id="ad3c">
         <a class="markup--anchor markup--p-anchor" data-href="https://plus.google.com/b/104314277918209223775/104314277918209223775/posts/MwTZGz9Az3Y" href="https://plus.google.com/b/104314277918209223775/104314277918209223775/posts/MwTZGz9Az3Y" rel="noopener nofollow noreferrer" target="_blank">
          此前我们编译过中国的监视技术向俄罗斯出口的消息
         </a>
         ：俄罗斯调查委员从美亚柏科采购取证魔方去检测和解码消息应用 Telegram、Skype 和 Viber 等的存档。根据政府采购文件，调查委员会上个月在采购取证魔方上花了 500 万卢布。（我们三年多前介绍过“美亚柏科”，这个中国互联网监视技术生产商，你可以
         <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/435" href="https://pao-pao.net/article/435" rel="noopener nofollow noreferrer" target="_blank">
          在这里看到
         </a>
         ）
        </p>
        <p class="graf graf--p graf-after--p" id="9001">
         取证魔方能破解 Android OS 和 iOS，自动提取设备 ID，包括用户名，电子邮件和电话号码。它还能获取社交网络 Facebook 和 Twitter 的数据。取证魔方解码的是 Telegram 的普通消息，而非启用端对端加密的加密消息。
        </p>
        <p class="graf graf--p graf-after--p" id="7a1e">
         关于中国国内，
         <a class="markup--anchor markup--p-anchor" data-href="https://plus.google.com/b/104314277918209223775/104314277918209223775/posts/MguQ68xNVFC" href="https://plus.google.com/b/104314277918209223775/104314277918209223775/posts/MguQ68xNVFC" rel="noopener nofollow noreferrer" target="_blank">
          已经有数据显示
         </a>
         ，自2016年初以来几乎每个省份的警察局都试图购买智能手机数据提取技术，同时内部安全支出大幅增加，全力以赴为打击异议。这些文件提供了一个罕见的一瞥，当政府压制不同意见时，中国努力推动使用高科技监控工具武装安全部队。
        </p>
        <p class="graf graf--p graf-after--p" id="5b19">
         扫描仪是手持式或桌面式设备，可以入侵智能手机，提取和分析联系人列表，照片，视频，社交媒体帖子和电子邮件等。手持设备还允许警察快速窃听街道上的电话内容。被揭露的数据包括来自33个省份，地区和直辖市中的32个、171个警察局的监视请求，似乎只显示了总支出的一部分。
        </p>
        <p class="graf graf--p graf-after--p" id="2ffb">
         该数据显示，自2016年初以来，这种监视设备的预算或支出超过1.29亿元（1900万美元），数量在2017年和2018年再次加速。上海的监视技术预算约为60万元人民币，用于购买手机扫描仪和数据抓取工具。
        </p>
        <p class="graf graf--p graf-after--p" id="5bb5">
         据称全国各地的警察局都可以查询和提取电子设备扫描数据的集中存储库，“几乎每个派出所都有这种入侵设备……根据今年早些时候在北京举行的警察技术展览会上看到的价格，中国制造的设备成本仅为1万元人民币，而其他设备成本则高达数十万元人民币。
        </p>
        <p class="graf graf--p graf-after--p" id="d5a1">
         需要提醒注意的是：
         <strong class="markup--strong markup--p-strong">
          采购文件显示，一些警察局要求提供工具，可以从Twitter，Facebook 及其WhatsApp聊天服务，Alphabet Inc（GOOGL.O）谷歌 Chrome浏览器上的用户帐户中提取数据，还有日本的 Line 消息平台
         </strong>
         。
        </p>
        <p class="graf graf--p graf-after--p" id="2907">
         该文件称，“基本数据收集功能”必须包括“手机密码，通讯录，通话记录，短信记录，彩信，图片，音频和视频数据，日历，备忘录和移动应用数据”。其他被列出的工具可能会破坏知名智能手机品牌，如三星电子，黑莓，中国自己的小米和华为，以及对苹果 iPhone 的暴力破解。三星，黑莓，小米和华为没有回应置评请求。苹果拒绝发表评论。
        </p>
        <p class="graf graf--p graf-after--p" id="7111">
         <strong class="markup--strong markup--p-strong">
          2017年，中国在国内维稳方面的支出约为1.24万亿元，占政府总支出的6.1％，超过军费支出。在北京新疆等地区，内部安全预算（其中监控技术的部分）已经翻了一番
         </strong>
         。在北京科技区的一个狭窄的培训中心里，工程师展示了一台这样的机器：灰色的鞋盒大小，它可以从三星智能手机中提取数据。
        </p>
        <p class="graf graf--p graf-after--p" id="1689">
         该扫描仪由以色列公司 Cellebrite 制造，但中国厦门美亚柏科、海信科技和 Pwnzen Infotech 等公司也在中国广泛使用。这种技术有望破解大多数智能手机，包括 iPhone。
        </p>
        <p class="graf graf--p graf-after--p" id="2b5b">
         没有理由相信中国的监视技术除了向俄罗斯出口之外不会再向其他国家出口，事实上我们
         <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E6%9C%80%E6%96%B0%E6%8A%A5%E5%91%8A-%E6%98%AF%E8%B0%81%E5%9C%A8%E6%8F%90%E4%BE%9B%E5%85%88%E8%BF%9B%E7%9A%84%E7%9B%91%E6%8E%A7%E6%8A%80%E6%9C%AF%E5%8D%8F%E5%8A%A9%E5%8E%8B%E8%BF%AB%E6%80%A7%E6%94%BF%E5%BA%9C%E5%AF%B9%E4%BA%BA%E6%9D%83%E7%9A%84%E4%BE%B5%E7%8A%AF-a78ed7d635fe" href="https://medium.com/@iyouport/%E6%9C%80%E6%96%B0%E6%8A%A5%E5%91%8A-%E6%98%AF%E8%B0%81%E5%9C%A8%E6%8F%90%E4%BE%9B%E5%85%88%E8%BF%9B%E7%9A%84%E7%9B%91%E6%8E%A7%E6%8A%80%E6%9C%AF%E5%8D%8F%E5%8A%A9%E5%8E%8B%E8%BF%AB%E6%80%A7%E6%94%BF%E5%BA%9C%E5%AF%B9%E4%BA%BA%E6%9D%83%E7%9A%84%E4%BE%B5%E7%8A%AF-a78ed7d635fe" rel="noopener noreferrer" target="_blank">
          在早前的报道中
         </a>
         已经介绍过：中国公司向玻利维亚、委内瑞拉和厄瓜多尔提供监控技术能力。在厄瓜多尔，中国电子公司向该国 24 个省提供了监视摄像头网络 — — 包括一些配备面部识别功能的摄像头、以及一个定位和识别手机的系统。
        </p>
        <p class="graf graf--p graf-after--p" id="82bc">
         尤其是被“一带一路”计划带动的有可能是大面积的输出，令人焦虑。
        </p>
        <p class="graf graf--p graf-after--p" id="9a02">
         并且需要重点提醒的是：
         <strong class="markup--strong markup--p-strong">
          中国是多家提供电话破解服务和取证软件的公司的所在地，
         </strong>
         而不仅仅是
         <a class="markup--anchor markup--p-anchor" data-href="https://www.techdirt.com/articles/20180818/11382940457/chinese-surveillance-state-is-basically-us-surveillance-apparatus-minus-constitutional-rights.shtml" href="https://www.techdirt.com/articles/20180818/11382940457/chinese-surveillance-state-is-basically-us-surveillance-apparatus-minus-constitutional-rights.shtml" rel="nofollow noopener noreferrer" target="_blank">
          <strong class="markup--strong markup--p-strong">
           Cellebrite 和 Grayshift
          </strong>
         </a>
         <strong class="markup--strong markup--p-strong">
          ，尽管这两者最出名的是向美国执法部门出售技术
         </strong>
         。电话破解在中国并非必不可少，大多数公民在被要求时会及时交出密码，以免于被拘押。
        </p>
        <p class="graf graf--p graf-after--p" id="faa6">
         值得注意的不只有技术。这种技术无处不在。除了全面的面部识别和其他生物识别跟踪外，美国执法部门还可以获得大部分内容，板式读卡器、取证设备、众多执法数据库、社交媒体跟踪软件……这些都已经在使用中。中国也一样。
        </p>
        <p class="graf graf--p graf-after--p" id="ac9c">
         中国部署的大部分内容都是以所谓的“安全”名义进行的。这与911事件后美国建立的大规模监视设备的理由相同。极权主义国家的框架已经到位。美国与中国区别的唯一因素是美国的宪法权利，但斯诺登提供的文件显示，大面积违宪行为是如此的普遍和深远。
        </p>
        <figure aria-describedby="caption-attachment-4710" class="wp-caption aligncenter" id="attachment_4710" style="width: 1748px">
         <img alt="" class="size-full wp-image-4710 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-22.jpeg?resize=1100%2C580&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-22.jpeg?w=1748&amp;ssl=1 1748w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-22.jpeg?resize=300%2C158&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-22.jpeg?resize=768%2C405&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-22.jpeg?resize=1024%2C540&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-22.jpeg?resize=1100%2C580&amp;ssl=1 1100w" data-recalc-dims="1" height="580" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/0-22.jpeg?resize=1100%2C580&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

         </noscript>
         <figcaption class="wp-caption-text" id="caption-attachment-4710">
          对政府和情报部门违宪的监控行为深感失望的美国网友们
         </figcaption>
        </figure>
        <p class="graf graf--p graf-after--figure" id="f770">
         目前为止没有关于中国的更多消息了。
         <strong class="markup--strong markup--p-strong">
          虽然“信息自由法案”不怎么好用，但中国彻底没有；虽然部分中国人已经明白，透明度革命是当下几乎唯一的选择，但是中国没有 Snowden 和 Wikileaks 的精神；虽然中国有很专业的黑客，但目前为止还没有具备这一层面上的正义感的行动者。于是当局可以保持封闭，一切都无从知晓
         </strong>
         （我们怀疑俄罗斯监视设备的购买是俄罗斯方面有意透露的，参见这里：
         <a class="markup--anchor markup--p-anchor" data-href="https://plus.google.com/b/104314277918209223775/104314277918209223775/posts/DpgKTQEF6qj" href="https://plus.google.com/b/104314277918209223775/104314277918209223775/posts/DpgKTQEF6qj" rel="noopener nofollow noreferrer" target="_blank">
          关于中俄军事关系的升级分析
         </a>
         ）
        </p>
        <p class="graf graf--p graf-after--p" id="9540">
         所以，关于“怎么办”的答案就在上述。⚪️
