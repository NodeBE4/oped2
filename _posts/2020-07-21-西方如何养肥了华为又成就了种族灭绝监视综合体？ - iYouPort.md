     IBM：播下了邪恶的种子
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   特朗普抨击华为对国家安全构成威胁，并谴责英国与该中国科技巨头联姻。
  </p>
  <p class="graf graf--p">
   👉然而，他没有透露IBM在华为快速扩张中的重要作用。
  </p>
  <p class="graf graf--p">
   2019年，IBM 首席执行官 Ginni Romety 加入了特朗普的美国劳动力政策咨询委员会。 2020年1月，她与伊万卡·特朗普（Ivanka Trump）联手发起了一场全国性的运动。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这反映了IBM在政治上与白宫的密切关系
   </strong>
   。
  </p>
  <p class="graf graf--p">
   自2008年以来，IBM总体上*仅*从臭名昭著的移民海关与执法局（ICE）和海关和边境保护局（CBP）处就获得了17亿美元的合同。特别是在特朗普的领导下，
   <strong class="markup--strong markup--p-strong">
    IBM 成为最大的政府技术供应商
   </strong>
   。
  </p>
  <p class="graf graf--p">
   👉但是，正是IBM为中国的监视国家噩梦设计了蓝图，在其被称为 “智能星球” 的概念下，它声称为实现更高效、更可持续的城市。在纸上听起来很棒。但是，这实际上只是反乌托邦的噩梦。
  </p>
  <p class="graf graf--p">
   IBM于
   <strong class="markup--strong markup--p-strong">
    2009年
   </strong>
   在中国首次提出了 “智能星球” 的概念，在全中国范围内召集了22个论坛，与200多个市的市长进行了接触。
  </p>
  <p class="graf graf--p">
   同年，IBM与沈阳市东北大学联合建立了 “生态城市” 研究实验室，地方政府出资2.75亿元人民币。
  </p>

   <a class="markup--anchor markup--p-anchor" data-href="https://www-03.ibm.com/press/us/en/pressrelease/28411.wss" href="https://www-03.ibm.com/press/us/en/pressrelease/28411.wss" rel="noopener noreferrer" target="_blank">

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     为中国第一个警察国家奠定基础
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   就在那年下半年，IBM 继续将克拉玛依
   <a class="markup--anchor markup--p-anchor" data-href="http://www.chinadaily.com.cn/business/2012-11/14/content_15935692.htm" href="http://www.chinadaily.com.cn/business/2012-11/14/content_15935692.htm" rel="noopener noreferrer" target="_blank">
    确立为
   </a>
   新疆地区的第一个智能城市试点项目。
  </p>
  <p class="graf graf--p">
   根据2012年名为 “智能城市” 的IBM投资者
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ibm.com/investor/att/pdf/investor0512/presentation/06_Smarter_Cities_Growth_Markets.pdf" href="https://www.ibm.com/investor/att/pdf/investor0512/presentation/06_Smarter_Cities_Growth_Markets.pdf" rel="noopener noreferrer" target="_blank">
    资料
   </a>
   ，该公司已在全国建立了82个IBM分支机构，包括研究、创新和软件开发，包括在新疆的 “智能城市” 克拉玛依。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12751 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/9-3.png?resize=1100%2C618&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/9-3.png?w=1367&amp;ssl=1 1367w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/9-3.png?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/9-3.png?resize=1024%2C575&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/9-3.png?resize=768%2C431&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/9-3.png?resize=1100%2C618&amp;ssl=1 1100w" data-recalc-dims="1" height="618" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/9-3.png?resize=1100%2C618&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12751" data-recalc-dims="1" height="618" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/9-3.png?resize=1100%2C618&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/9-3.png?w=1367&amp;ssl=1 1367w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/9-3.png?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/9-3.png?resize=1024%2C575&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/9-3.png?resize=768%2C431&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/02/9-3.png?resize=1100%2C618&amp;ssl=1 1100w" width="1100"/>
   </noscript>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    IBM与中国移动合作
   </strong>
   ，使克拉玛依市
   <a class="markup--anchor markup--p-anchor" data-href="https://cn.ambafrance.org/Karamay-ville-intelligente-du-Xinjiang" href="https://cn.ambafrance.org/Karamay-ville-intelligente-du-Xinjiang" rel="noopener noreferrer" target="_blank">
    成为
   </a>
   新疆第一座享有无线通信系统的城市，该系统涵盖了城市生活的方方面面。
  </p>
  <p class="graf graf--p">
   据官媒《中国日报》报道，“ IBM提出的概念是基于物联网和云计算等技术的、涵盖了运输、医疗保健和维稳。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    从一开始，无处不在的监视功能就内置在 IBM 的克拉玛依（Karamay）设计概念中。
   </strong>
   该系统非常复杂，比如 当失业率过高时，它将提醒政府官员。
  </p>
  <p class="graf graf--p">
   关于IBM在克拉玛依市干了什么的公开信息很少。
  </p>
  <p class="graf graf--p">
   但是，2015年一份报告对印度中央邦委托在智能城市方面取得的全球进展进行了研究，该报告对 “公共安全/维稳” 部分做了一些说明。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="http://mpurban.gov.in/SmartCity/pdf/ReportonInternationalCaseStudies.pdf" href="http://mpurban.gov.in/SmartCity/pdf/ReportonInternationalCaseStudies.pdf" rel="noopener noreferrer" target="_blank">
    该报告
   </a>
   由总部位于弗吉尼亚州的 ICF International（一家享负盛名的数十亿美元的全球咨询公司）进行。该报告指出，
   <strong class="markup--strong markup--p-strong">
    IBM 的项目建立了一个平台，“可以与警用地理信息系统（PGIS）互连，以实现各种各样的智能服务。包括视频的预防性维护、行为分析、面部识别、视频摘要、快速搜索、图像锐化和案例调查分析。”
   </strong>
  </p>
  <p class="graf graf--p">
   IBM开始在克拉玛依市工作的第二年，华为发布了 “安全城市解决方案” 手册，将该城市确定为成功的 “安全城市” 案例研究。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iotone.com/files/pdf/vendor/Huawei_Smart_City_Solution_2013.pdf" href="https://www.iotone.com/files/pdf/vendor/Huawei_Smart_City_Solution_2013.pdf" rel="noopener noreferrer" target="_blank">
    该手册介绍了
   </a>
   华为如何安装 “统一的安全城市平台”，该平台将
   <strong class="markup--strong markup--p-strong">
    为警察提供
   </strong>
   与 ICF International 有关IBM的报告中所述的监视功能。
  </p>
  <p class="graf graf--p">
   👉简而言之，在 IBM 为卡拉玛依的所谓 “智能” 维稳方法创建了初始设计之后，华为继续进行了构建。
  </p>
  <p class="graf graf--p">
   我问过IBM目前与华为的关系，以及它参与建立所谓的智能城市基础设施的基础，尤其是在维稳方面的基础，这些基础设施使得大规模监视在整个新疆地区得以迅速扩展。
  </p>
  <p class="graf graf--p">
   尽管有许多征求意见的请求，并且 IBM 发言人做出了一些保证，即将提供答案，但是，最终该公司
   <strong class="markup--strong markup--p-strong">
    没有做出任何回应
   </strong>
   。
  </p>
  <p class="graf graf--p">
   根据总部位于伦敦的全球隐私权倡导组织和监管机构 “隐私国际” 的智能城市专家 Eva Blum-Dumontet 所说：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     “西方国家开发了智能城市的模型 —— 值得记住的是，智能城市这个词源于IBM —— 通常是高度侵犯隐私的，并且涉及到系统性地收集公民数据。现在也难怪其他国家也在这样做。
    </em>
   </strong>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    不幸的是，这种模式在政府监视下经常可以看到：西方国家决定接受新制度或新法律，因为这一切都属于法治，然而。我们最终看到的是该法律或制度在国外被复制，有时后果更糟。”
   </em>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     与恶魔共舞
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   克拉玛依
   <a class="markup--anchor markup--p-anchor" data-href="http://info.upla.cn/html/2013/12-11/248141.shtml" href="http://info.upla.cn/html/2013/12-11/248141.shtml" rel="noopener noreferrer" target="_blank">
    只是
   </a>
   更大计划中的第一个项目而已，仅在2014年上半年就涉及在新疆16个主要城市和63个县建设 5,000 个4G移动基站。到那年年底，建造了12,000个4G基站。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    到2016年
   </strong>
   ，随着智能城市基础设施在新疆的扩展，IBM
   <strong class="markup--strong markup--p-strong">
    仍在
   </strong>
   克拉玛依开展业务，并根据该市地方政府网站开始
   <a class="markup--anchor markup--p-anchor" data-href="http://www.karamay.gov.cn/2016-10/11/content_27027825.htm" href="http://www.karamay.gov.cn/2016-10/11/content_27027825.htm" rel="noopener noreferrer" target="_blank">
    引入
   </a>
   “认知物联网”（IoT）。
  </p>
  <p class="graf graf--p">
   IBM 的技术围绕 Watson IoT 平台构建，Watson IoT 平台是一个AI系统，可以在大量不同类型的数据中找到模式和关系。
  </p>
  <p class="graf graf--p">
   沃森（Watson）在中国的其他应用之一是 CCTV.com，使用该软件通过移动设备测量 GPS、海拔、体温和心率，从而使粉丝能够跟踪 Vaude-Tibet 自行车挑战。
  </p>
  <p class="graf graf--p">
   根据 IBM 的 Kevin Larson 的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.avnet.com/wps/portal/us/resources/technical-articles/article/iot/cognitive-computing-and-ibm-watson-iot-unlocking-the-power-of-information/" href="https://www.avnet.com/wps/portal/us/resources/technical-articles/article/iot/cognitive-computing-and-ibm-watson-iot-unlocking-the-power-of-information/" rel="noopener noreferrer" target="_blank">
    说法
   </a>
   ，⚠️该平台的强大之处在于它能够提取和分析几乎任何格式任何形式的数据，以及搜索与其他类型数据的相关性。
  </p>
  <p class="graf graf--p">
   它可以跨气象数据、科学文章、车队、环境传感器或视听源执行此操作。这使得该平台非常适合广泛的监视应用，⚠️
   <strong class="markup--strong markup--p-strong">
    包括检查视频和照片以识别可以将人和事件联系起来的场景和样式，以及检查来自客户电话、博客、博客评论和推文的记录
   </strong>
   —— 再一次以关联信息和检测模式的方式。
  </p>
  <p class="graf graf--p">
   目前尚不清楚 Watson 平台在克拉玛依市中实际应用的确切程度，以及华为对 Watson 的访问权限 —— IBM和华为的发言人均未作任何澄清。
  </p>
  <p class="graf graf--p">
   👉但是，这座城市是许多维吾尔族 “再教育” 营的所在地，居民经常因穿着 “穆斯林服装” 或 “留胡须” 而被警察带走。
  </p>
  <p class="graf graf--p">
   该市广泛的监视网络（最初是在IBM的 “公共安全/维稳” 平台下建立的，后来演变为
   <a class="markup--anchor markup--p-anchor" data-href="https://e.huawei.com/en/publications/global/ict_insights/201509091408/focus/201509091608" href="https://e.huawei.com/en/publications/global/ict_insights/201509091408/focus/201509091608" rel="noopener noreferrer" target="_blank">
    华为的
   </a>
   “安全城市” 计划）使这种做法更加自动化。
  </p>
  <p class="graf graf--p">
   根据该计划，华为向克拉玛依当局提供了将视频监控与AI和大数据相结合的技术，提供了公司所谓的 “一种融合的、可视化的命令解决方案，
   <strong class="markup--strong markup--p-strong">
    使政府机构可以灵活地共享视频资源并协作跟踪任何他们想要的人。”
   </strong>
  </p>
  <p class="graf graf--p">
   去年12月，华为官方网站上
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/lifang072/status/1073197924027375616/photo/1?ref_src=twsrc%5Etfw%7Ctwcamp%5Etweetembed%7Ctwterm%5E1073197924027375616&amp;ref_url=http%3A%2F%2Fwww.epochtimes.com%2Fb5%2F18%2F12%2F14%2Fn10911475.htm" href="https://twitter.com/lifang072/status/1073197924027375616/photo/1?ref_src=twsrc%5Etfw%7Ctwcamp%5Etweetembed%7Ctwterm%5E1073197924027375616&amp;ref_url=http%3A%2F%2Fwww.epochtimes.com%2Fb5%2F18%2F12%2F14%2Fn10911475.htm" rel="noopener noreferrer" target="_blank">
    发布了
   </a>
   有关其参与大规模监视的屏幕截图。这些资料显然在华为网站上很难获得，活动家只有通过注册帐户并登录才能访问。但是，在其他地方，华为公司的代表似乎很乐意谈论这点。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “华为安全城市项目对克拉玛依建设世界一流的公共安全基础设施做出了重大贡献”，华为高管 Viktor Yu 在公司网站上
   <a class="markup--anchor markup--p-anchor" data-href="https://e.huawei.com/en/publications/global/ict_insights/201509091408/focus/201509091608" href="https://e.huawei.com/en/publications/global/ict_insights/201509091408/focus/201509091608" rel="noopener noreferrer" target="_blank">
    写道
   </a>
   。“该项目支持全市访问数百个摄像机提供的监视视频，
   <strong class="markup--strong markup--p-strong">
    使成千上万的警察能够有效监控近30万人
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   Yu 继续说：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “在过去几十年中，指挥平台使用了大量的累积犯罪数据，已经预先集成了各种预警方案。当检测到异常时，分析平台会自动生成优先警报。其影响是大大降低了犯罪率，并广泛维护了社会稳定。”
   </em>
   —— 他承认了协助维稳。
  </p>
  <p class="graf graf--p">
   在过去的十年中，华为在中国各地 —— 上海、江苏、广东和其他城市 —— 建立了
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20190410112804/https:/actfornet.com/HUAWEI_VIDEO_SURVEILLANCE_DOCS/Huawei%20IVS%20Intelligent%20Video%20Surveillance%20Case%20Study.pdf" href="https://web.archive.org/web/20190410112804/https:/actfornet.com/HUAWEI_VIDEO_SURVEILLANCE_DOCS/Huawei%20IVS%20Intelligent%20Video%20Surveillance%20Case%20Study.pdf" rel="noopener noreferrer" target="_blank">
    数十个
   </a>
   维稳城市计划。
  </p>
  <p class="graf graf--p">
   2018年5月，华为与新疆首府乌鲁木齐市公安局
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theglobeandmail.com/world/article-huaweis-partnership-with-china-on-surveillance-raises-concerns-for/" href="https://www.theglobeandmail.com/world/article-huaweis-partnership-with-china-on-surveillance-raises-concerns-for/" rel="noopener noreferrer" target="_blank">
    建立了
   </a>
   新的合作伙伴关系，继续在新疆开展工作，建立了所谓的 “智能安防行业” 创新实验室。
  </p>
  <p class="graf graf--p">
   华为董事陶景文在该公司的网站上说，该实验室将为该地区创造一系列专业知识，以 “开启智能警务的新时代，并帮助建立一个更安全，更智能的社会”。
  </p>
  <p class="graf graf--p">
   目前在新疆使用的技术中，有先进的面部识别技术，旨在根据其外观专门搜索维吾尔族人，从而有可能迎来纽约时报所说的 “

   ”。

   该实验室原本计划制造原型，以支持包括制造业、零售业、医疗保健业、和 “公共部门” 在内的各个行业的 “数字化转型”，👉
   <strong class="markup--strong markup--p-strong">
    这当然需要为政府机构直接提供的任何服务，也包括警察和维稳机构。
   </strong>
  </p>
  <p class="graf graf--p">
   微软在中国的许多智能城市项目 *表面上* 看起来无害。其中一项是与北京市政府和环保部合作的城市空气质量项目，该项目利用大数据监测和预测空气质量。在另一个项目中，微软正在与武汉市政府合作，使用聊天机器人来回答市民在城市服务方面的常规问题，例如，如何获得驾驶执照。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是多年联姻的高潮
   </strong>
   。 2014年，微软与云南、株洲、长春、扬州、和温州等中国多个省市签署了一系列的合作协议，这是该公司所谓智能城市的 CityNext 计划的一部分。
  </p>
  <p class="graf graf--p">
   没有证据表明，通过这些举措微软直接为中国政府专门开展了与监视有关的项目。他们隐蔽得非常好。
  </p>
  <p class="graf graf--p">
   但是在2016年，
   <strong class="markup--strong markup--p-strong">
    微软的中国子公司在北京为维稳部举办了一个研讨会
   </strong>
   ，展示了所谓的 “公共安全” 解决方案。
  </p>
  <p class="graf graf--p">
   微软公司负责公共安全和国家安全的总经理 Arthur Thomas 对该研讨会的坦率描述可以在微软的网站上找到。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://cloudblogs.microsoft.com/industry-blog/government/2016/01/04/china-subsidiary-showcases-solutions-with-the-ministry-of-public-safety/" href="https://cloudblogs.microsoft.com/industry-blog/government/2016/01/04/china-subsidiary-showcases-solutions-with-the-ministry-of-public-safety/" rel="noopener noreferrer" target="_blank">
    Thomas 说
   </a>
   ：“微软拥有正确的分析解决方案，可以评估潜在的紧急情况，确定其预期的影响，然后促进执行突发事件行动计划。”
  </p>
  <p class="graf graf--p">
   他说，“我们通过将来自不同来源（例如位置、跟踪、传感器、视频、交通报告、医院状况、天气报告、社交媒体和其他动态数据）的信息，与GIS数据（例如图像、海拔、街道、和关键基础架构等）结合起来以做到这一点。”
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12752 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/10-2.png?resize=1100%2C649&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/10-2.png?w=1217&amp;ssl=1 1217w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/10-2.png?resize=300%2C177&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/10-2.png?resize=1024%2C604&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/10-2.png?resize=768%2C453&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/10-2.png?resize=1100%2C649&amp;ssl=1 1100w" data-recalc-dims="1" height="649" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/10-2.png?resize=1100%2C649&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   我问微软，在计划向涉嫌压制异见人士和少数族裔的专制国家提供这种技术之前，该公司是否向中国政府强调了任何担忧。微软拒绝置评。
  </p>
  <p class="graf graf--p">
   在该研讨会上，微软的技术合作伙伴之一是澳大利亚公司 iOmniscient —— 该公司已为中国城市
   <a class="markup--anchor markup--p-anchor" data-href="https://www.securityinfowatch.com/video-surveillance/press-release/10481592/iomniscient-iomniscient-video-analytics-solution-to-protect-chengdu-metro" href="https://www.securityinfowatch.com/video-surveillance/press-release/10481592/iomniscient-iomniscient-video-analytics-solution-to-protect-chengdu-metro" rel="noopener noreferrer" target="_blank">
    提供了
   </a>
   视频监视和分析技术，包括为成都地铁服务和中国
   <a class="markup--anchor markup--p-anchor" data-href="http://www.safetysecurityasia.com.sg/network/pdf/Infocom&amp;SecuritySystems-FaceRecognitioninaCrowdSystemfromiOmniscient.pdf" href="http://www.safetysecurityasia.com.sg/network/pdf/Infocom&amp;SecuritySystems-FaceRecognitioninaCrowdSystemfromiOmniscient.pdf" rel="noopener noreferrer" target="_blank">
    军工厂
   </a>
   提供的视频监视和分析技术。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “中国最大的常备军在其军工厂中使用了 iOmniscient，” iOmiscient 公司产品展示台上就是这么写的。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12753 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/11-1.png?resize=1100%2C569&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/11-1.png?w=1104&amp;ssl=1 1104w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/11-1.png?resize=300%2C155&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/11-1.png?resize=1024%2C530&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/11-1.png?resize=768%2C397&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/11-1.png?resize=1100%2C569&amp;ssl=1 1100w" data-recalc-dims="1" height="569" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/02/11-1.png?resize=1100%2C569&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   👉没错，近年来，华为参与提供的众多压制性监视技术
   <strong class="markup--strong markup--p-strong">
    并没有
   </strong>
   阻止微软与这家中国公司紧密合作。
  </p>
  <p class="graf graf--p">
   2017年，微软与华为签署了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zdnet.com/article/huawei-announces-microsoft-cloud-apps-partnership/" href="https://www.zdnet.com/article/huawei-announces-microsoft-cloud-apps-partnership/" rel="noopener noreferrer" target="_blank">
    一项协议
   </a>
   ，声称建立 “开放、共赢的生态系统”，“在公共云上启动深度合作”。该协议将把微软企业服务带到华为云。次年，两家公司
   <a class="markup--anchor markup--p-anchor" data-href="https://www.huawei.com/en/press-events/news/2018/10/huawei-microsoft-all-flash-azure-stack-solution" href="https://www.huawei.com/en/press-events/news/2018/10/huawei-microsoft-all-flash-azure-stack-solution" rel="noopener noreferrer" target="_blank">
    又推出了
   </a>
   基于微软 Azure 堆栈软件的联合云解决方案。
  </p>
  <p class="graf graf--p">
   次年，华为与新疆乌鲁木齐市警察局就 “公共安全”
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theglobeandmail.com/world/article-huaweis-partnership-with-china-on-surveillance-raises-concerns-for/" href="https://www.theglobeandmail.com/world/article-huaweis-partnership-with-china-on-surveillance-raises-concerns-for/" rel="noopener noreferrer" target="_blank">
    继续加强
   </a>
   对新疆监视国家的支持。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这只是更广泛的同谋的又一个例子而已
   </strong>
   。去年年底，澳大利亚战略政策研究所
   <a class="markup--anchor markup--p-anchor" data-href="https://www.aspi.org.au/report/mapping-more-chinas-tech-giants" href="https://www.aspi.org.au/report/mapping-more-chinas-tech-giants" rel="noopener noreferrer" target="_blank">
    发现
   </a>
   ：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <em class="markup--em markup--p-em">
    华为在新疆的工作范围广泛，包括直接与中国政府在该地区的公安部门合作。在有关华为和5G技术的辩论中，应考虑华为在新疆的活动。”
   </em>
  </p>
  <p class="graf graf--p">
   当我询问微软是否考虑过与建立全球最大监视系统的中国公司合作时的道德问题时，微软不想发表评论。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     华为，华为，无处不在 ……
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   华为与英国政府的蓬勃发展只是最近的成功。该公司有关维稳城市计划的官方文献现在声称已将该模型输出到世界各地的230多个城市。
  </p>
  <p class="graf graf--p">
   华为安全城市解决方案的最早采用者包括很多具有威权主义和内部暴力历史的国家，包括俄罗斯、巴基斯坦、老挝、安哥拉、亚美尼亚、阿塞拜疆、塔吉克斯坦、乌克兰、乌兹别克斯坦、吉尔吉斯斯坦和哈萨克斯坦。
  </p>
  <p class="graf graf--p">
   在塞尔维亚，华为与贝尔格莱德首府签订了所谓的公共安全合同，其面部识别监视摄像头系统的安装因违反公民隐私法而受到数字权利监督机构 “Share Foundation” 的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zdnet.com/article/huaweis-surveillance-system-in-serbia-threatens-citizens-rights-watchdog-warns/" href="https://www.zdnet.com/article/huaweis-surveillance-system-in-serbia-threatens-citizens-rights-watchdog-warns/" rel="noopener noreferrer" target="_blank">
    严厉批评
   </a>
   。
  </p>
  <p class="graf graf--p">
   尽管 Boris Johnson 政府已经提出了很多想法，即 不允许华为访问 “安全关键” 基础架构，但是，这已经在发生了。
  </p>
  <p class="graf graf--p">
   GlobalData 技术研究团队的首席分析师 David Bicknell 说：“华为在英国关键的国家基础设施中的重要性已经不新鲜了。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “自2003年以来，它一直致力于政府项目，并于2011年聘请了该国顶级IT专家 John Suffolk ……
   <strong class="markup--strong markup--p-strong">
    在过去的二十年中，该公司已经深深地融入了英国的电信市场，以至于政府愿意冒险与主要盟友进行公开对抗，而不是将华为排除在英国的5G网络之外。
   </strong>
   ”
  </p>
  <p class="graf graf--p">
   👉 但是，由于华为技术已与可信赖的第三方供应商达成协议，因此已经被英国和欧洲的一些最敏感的基础设施所采用。根据公司的
   <a class="markup--anchor markup--p-anchor" data-href="https://tcca.info/huawei-and-frequentis-sign-mou-to-accelerate-new-opportunities-in-public-safety-industry/" href="https://tcca.info/huawei-and-frequentis-sign-mou-to-accelerate-new-opportunities-in-public-safety-industry/" rel="noopener noreferrer" target="_blank">
    新闻材料
   </a>
   ，其中包括 “空中交通管理（民用和军用空中交通管理和防空）、以及公共安全与运输（警察、消防和救援服务、紧急医疗服务、船只交通和铁路等）。”
  </p>
  <p class="graf graf--p">
   根据华为网站上的公告，与 Frequentis 的合作伙伴关系将推动 “共同开发公共安全解决方案的开放生态系统，以迎接更安全的城市”。两家公司将共同创建 “用于指挥和控制中心的下一代调度解决方案”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “通过支持不同机构之间的监视系统无缝集成、实时语音和数据传输服务等支持功能，
   </em>
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     该联合解决方案将帮助警察维稳机构提高效率，同时降低关键任务IT投资的总成本
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    。”
   </em>
  </p>
  <p class="graf graf--p">
   👉Frequentis 为包括美国、英国和整个欧洲在内的
   <strong class="markup--strong markup--p-strong">
    140多个国家/地区
   </strong>
   提供其技术，从而使其成为世界上最大的语音通信系统提供商。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.frequentis.com/en/about-us" href="https://www.frequentis.com/en/about-us" rel="noopener noreferrer" target="_blank">
    它的客户包括
   </a>
   英国国防部、伦敦市警察局、美国国家航空航天局、美国海军、澳大利亚皇家空军、瑞典海事局和欧洲管制局、后者正致力于欧洲委员会的 “欧洲单一天空” 项目，以创建第一个用于空中交通管制的泛欧洲语音通信系统。
  </p>
  <p class="graf graf--p">
   ⚠️该声明断言，
   <strong class="markup--strong markup--p-strong">
    Frequentis 与华为的合作伙伴关系将产生的技术市场已接近全球，覆盖欧洲、中东、非洲、亚太地区、和拉丁美洲
   </strong>
   。 Frequentis 没有回应置评请求。
  </p>
  <p class="graf graf--p">
   像所有其他公司一样，华为也最终未能回答记者的问题，尽管做出了最初的回应和反复的保证。
  </p>
  <p class="graf graf--p">
   👉他们中的任何一个都不愿意结束这种噩梦般的技术合作，因为他们在促进不仅已经成为历史上最大的大规模监视行动，而且，现在已成为种族文化灭绝行动的崛起中
   <strong class="markup--strong markup--p-strong">
    发挥了至关重要的作用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   随着华为继续稳步渗透全球各地的电信，安全、监控和其他关键技术基础架构；当政府吹嘘自己的技术实力时 —— 值得记住的是，他们如何磨练和完善自己的能力 ——
   <strong class="markup--strong markup--p-strong">
    他们一直踩在中国受压抑的维吾尔族人的鲜血和尸体之上。⚪️
   </strong>
