  <p class="graf graf--p">
   2016年的一个晚上，一位名叫 Alex Gutiérrez 的二十五岁社区大学生正在加利福尼亚圣华金河谷的高档餐厅 La Piazza Ristorante Italiano 等候餐桌。
  </p>
  <p class="graf graf--p">
   Gutiérrez 发现了 Yorai Benzeevi，他是一名经营当地医院的医生，正与医院董事会成员 Parmod Kumar 坐在一张桌子旁。
  </p>
  <p class="graf graf--p">
   他们似乎心情愉快，喝着昂贵的酒，有说有笑。
  </p>
  <p class="graf graf--p">
   这激怒了 Gutiérrez。
  </p>
  <p class="graf graf--p">
   Gutiérrez 最近加入了一个名为 Citizens for Hospital Accountability 的 Tulare 当地公民组织。
  </p>
  <p class="graf graf--p">
   该组织指责 Benzeevi 以牺牲资金紧张的医院为代价中饱私囊，该医院随后宣布破产。
  </p>
  <p class="graf graf--p">
   根据法庭文件，Benzeevi 的合同非常有利可图；在2014年给会计师的电子邮件中，他估计他的医院业务每年可以产生900万美元的收入（在 Tulare，家庭年收入中位数大约是每年四万五千美元。）
  </p>
  <p class="graf graf--p">
   公民团体制定了一项雄心勃勃的计划，通过把他的盟友踢出医院董事会来摆脱 Benzeevi。
  </p>
  <p class="graf graf--p">
   Gutiérrez 是一名政治学专业的学生，​​是红杉学院青年民主党俱乐部的领导者，在2016年的总统竞选期间，他参加了伯尼桑德斯的集会。
  </p>
  <p class="graf graf--p">
   Gutiérrez 的父亲是一个奶牛场主，每周工作六天，每天工作12小时；桑德斯关于企业贪婪、收入不平等以及美国营利性医疗保健系统的弊病的演讲，引起了 Gutiérrez 的共鸣。
  </p>
  <p class="graf graf--p">
   看到 Benzeevi 和 Kumar 在 La Piazza 享受高档食品的样子激怒了 Gutiérrez 的不公平感。
  </p>
  <p class="graf graf--p">
   他花了一周的时间在圣诞节和新年之间挨家挨户敲门，要求邻居签署请愿书进行召回投票，最终获得了超过1100个签名。
  </p>
  <p class="graf graf--p">
   Gutiérrez 后来问他的母亲 Senovia 是否会竞选 Kumar 的席位；公民团体认为 Senovia 作为一个移民和社会工作者的双重身份将成为一个吸引人的候选人，在这个约 60％西班牙裔的社区。
  </p>
  <p class="graf graf--p">
   Benzeevi 感觉到了明显的威胁，他曾在华盛顿特区的一家律师事务所咨询过如何开展拯救 Kumar 席位的活动。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一位顾问将他转介给以色列私人情报公司 Psy-Group
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Psy-Group 的口号是“塑造现实”，其技术包括使用精心设计的虚假身份来操纵其目标
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Psy-Group 是新一波私人情报公司的一部分，这些公司从以色列的秘密间谍队伍中招募 —— 自称为“私人摩萨德”。
  </p>
  <p class="graf graf--p">
   这些公司中最具侵略性的家伙似乎愿意为客户做任何事。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9115 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-2.jpg?resize=800%2C450&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-2.jpg?w=800&amp;ssl=1 800w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-2.jpg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-2.jpg?resize=768%2C432&amp;ssl=1 768w" data-recalc-dims="1" height="450" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/1-2.jpg?resize=800%2C450&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>

  <p class="graf graf--p">
   Psy-Group 从许多竞争对手中脱颖而出，因为
   <strong class="markup--strong markup--p-strong">
    它不只是收集情报；它还专门用于秘密传播信息，以操纵人们的信仰和行为方式
   </strong>
   。
  </p>
  <p class="graf graf--p">
   其工作人员利用技术创新和政府监管松懈赚钱。
  </p>
  <p class="graf graf--p">
   以色列前高级情报官员 Uzi Shaya 说：
   <strong class="markup--strong markup--p-strong">
    “社交媒体让你几乎可以接触任何人并控制他们的思想”；
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “你想做什么就可以做什么。你可以成为任何你想要扮演的人。这[社交媒体]是一个战场，选举在此获胜，恐怖活动在此升级。没有法规。这是一个无人区。“
  </p>
  <p class="graf graf--p">
   近年来，Psy-Group 已经构思出各种精心设计的秘密行动。
  </p>
  <p class="graf graf--p">
   在阿姆斯特丹，该公司编写了一份名为布伦斯塔德基督教会的宗教教派的报告，Psy-Group 声称自己撰写了一本“比新约更重要的书”；
  </p>
  <p class="graf graf--p">
   在加蓬，Psy-Group 投入了“Operation Bentley” —— 通过收集和传播有关其主要政治对手的情报，努力“保持”总统阿里·邦戈·翁丁巴掌握权力；
  </p>
  <p class="graf graf--p">
   在纽约，Psy-Group 代表富有的犹太裔美国捐助者开展了一场运动，
   <strong class="markup--strong markup--p-strong">
    以尴尬和恐吓美国大学校园里的学生活动家
   </strong>
   ，这些活动家支持一场向以色列施加经济压力的运动，因为以色列对待巴勒斯坦人的行为。
  </p>
  <p class="graf graf--p">
   不止于此。
  </p>
  <p class="graf graf--p">
   Psy-Group 的雄心壮志是打入美国大选市场。
  </p>

  <p class="graf graf--p">
   在2016年总统竞选期间，该公司向唐纳德特朗普的竞选团队成员展示了其
   <strong class="markup--strong markup--p-strong">
    影响选举结果的能力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Psy-Group 的老板 Joel Zamel 甚至要求前众议院议长 Newt Gingrich 向特朗普的女婿贾里德库什纳建议 Zamel 的服务。
  </p>
  <p class="graf graf--p">
   鼓励业务的努力甚至还包括对该公司的
   <strong class="markup--strong markup--p-strong">
    在线欺骗技巧
   </strong>
   的简单粗暴宣称。
  </p>
  <p class="graf graf--p">
   在2016年大选之后，Psy-Group 与 Benzeevi 的谈判促使该公司制定了在州和地方层面开展更多业务的计划。
   <strong class="markup--strong markup--p-strong">
    干涉任何选举不论太小
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一份公司文件报告称，Psy-Group 的影响力服务平均仅花费三十五万美元 —— 每小时仅为275美元。
  </p>
  <p class="graf graf--p">
   该公司出版了一本挑衅性小册子，上面印有一条金鱼的图像，背面绑着鲨鱼鳍，标语写着：“真实取决于感知”。
  </p>
  <p class="graf graf--p">
   另一本小册子展示了一只猫的倒影是一只狮子，并在该公司的服务中列出了
   <strong class="markup--strong markup--p-strong">
    “蜜罐诱捕” 项目
   </strong>
   。（在间谍世界中，蜜罐诱捕通常涉及部署性吸引力的操作，以诱导目标提供信息）
  </p>
  <p class="graf graf--p">
   Psy-Group 为 Benzeevi 提出了一项提案，承诺在 Tulare 进行“协调情报运作和影响力运动”，以保留 Kumar 在医院董事会的席位。
  </p>
  <p class="graf graf--p">
   这些宣传战操作人员将使用虚假身份向社区成员“揭示并传递可操作的情报”。该行动的代号为 Project Mockingjay，这是以小说“饥饿游戏”中虚构的鸟类为名的，这种鸟以其模仿人类声音的能力而闻名。
  </p>

    现代私人情报市场可以追溯到20世纪70年代

   <a class="markup--anchor markup--p-anchor" data-href="https://www.newyorker.com/magazine/2009/10/19/the-secret-keeper" href="https://www.newyorker.com/magazine/2009/10/19/the-secret-keeper" rel="noopener noreferrer" target="_blank">

   的前检察官开始雇用警察侦探、FBI 以及法务会计师，代表公司、会计师事务所以及其他客户进行侦探工作。

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这家公司也招募了少数前 CIA 官员
   </strong>
   ，但很少宣传这些雇员 —— Kroll 知道与CIA 关系过于密切可能会危及员工的安全（CIA名声太臭了）。
  </p>
  <p class="graf graf--p">
   如今，已经有成千上万的
   <strong class="markup--strong markup--p-strong">
    以色列版本
   </strong>
   的 Kroll 进入私人情报市场。
  </p>
  <p class="graf graf--p">
   这些公司具有独特的优势：
   <strong class="markup--strong markup--p-strong">
    很少有国家能比以色列生产更多训练有素且经过战争考验的情报专业人员
   </strong>
   —— 以色列间谍工作者占总人口的比例最大。
  </p>
  <p class="graf graf--p">
   以色列的征兵对大多数公民来说是强制性的，而高级情报部门通常会在高中时就确定有才能的新兵。
  </p>
  <p class="graf graf--p">
   这些士兵接受过各种语言和技术技能的强化训练。
  </p>
  <p class="graf graf--p">
   经过几年的政府服务后大多数人都被解职，此时许多人完成了自己的教育历程并进入了平民就业市场。
  </p>
  <p class="graf graf--p">
   Gadi Aviran 是以色列私人情报行业的先驱之一。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “每年都有大量的人才涌入军队”，创立情报公司 Terrogence 的 Aviran 表示，“像我这样的公司必须做的就是站在门口说’你看起来不错‘”。
  </p>
  <p class="graf graf--p">
   Aviran 以前是以色列军事情报研究小组的负责人，在那里他负责监督分析人员，这些分析师声称寻找恐怖主义威胁，
   <strong class="markup--strong markup--p-strong">
    监控和审查从电话通信和互联网数据流的几乎一切
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Aviran 说，这个过程就像是 “盯着一条流淌的河流，以找到其中是否有任何值得注意的东西。” 该系统在发生攻击后通常能够有效地分析攻击，但是在提供预警方面效果不佳。
  </p>
  <p class="graf graf--p">
   Aviran 开始考虑采用更有针对性的方法。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    间谍、私人调查员、犯罪分子、甚至一些记者，长期以来都使用虚假身份诱骗人们提供信息，这种做法被称为假托
   </strong>

   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    社交工程
   </a>
   常用方法）。
  </p>
  <p class="graf graf--p">
   互联网使假托的操作更加容易了。Aviran 认为虚假的在线角色（就是袜子木偶/假托），可以用来监视恐怖组织，并阻止计划中的袭击事件发生。
  </p>

  <p class="graf graf--p">
   Tulare 看起来并不像一个能成为影响力运动的目标。该镇的名字来源于一个湖泊，该湖泊于1773年由一位西班牙指挥官 Los Tules 命名；这座小镇后来被写入了 一首歌名为“Ghost of Bardsley Road”，这首歌讲述了一个骑着白色本田摩托车的无头幽灵。

  <p class="graf graf--p">
   今天，这个城市只有六万多人。该县在乳制品生产方面领先全国。不久前，野火在加利福尼亚肆虐时，风将烟雾推入 Tulare，在空气中留下刺鼻的气味。
  </p>
  <p class="graf graf--p">
   Citizens for Hospital Accountability 组织开始于一个简单的 Facebook 页面。起初，该组织的领导人希望 Alex Gutiérrez 能够竞选 Kumar 的席位，但他计划在市议会中担任一个职位。
  </p>
  <p class="graf graf--p">
   Senovia 是备用选择。出生于墨西哥中部的阿瓜斯卡连特斯州，是十二个孩子中最小的一个。她的父母是耕种玉米和豆类的贫困农民，直到干旱迫使他们放弃土地。
  </p>
  <p class="graf graf--p">
   她十六岁时开始全职工作；二十四岁时，她越过 Tijuana 的边境，与住在洛杉矶的男友 Miguel Gutiérrez 相聚。
  </p>
  <p class="graf graf--p">
   他们结婚了，两年后，他们搬到了 Tulare，在那里，Senovia 抚养了五个男孩，并通过兼职担任管家来补充家庭的收入。
  </p>
  <p class="graf graf--p">
   当她三十五岁时，她获得了高中毕业证书，然后进入了社区学院并继续获得了学士学位，在加利福尼亚州立大学弗雷斯诺分校。2015年，她成为美国公民，并完成了社会工作硕士学位。
  </p>
  <p class="graf graf--p">
   Alex 怀疑他的母亲是否会同意参加竞选。她从未表现出对政治的兴趣。

  <p class="graf graf--p graf--startsWithDoubleQuote">

  </p>
  <p class="graf graf--p">
   在 Senovia 的三居室住宅的家庭聚餐中，Alex 告诉她关于医院“腐败和管理不善”的故事。Senovia 告诉他，“我很乐意这么做，因为你很积极。”
  </p>
  <p class="graf graf--p">
   医院董事会竞选通常是小事。 董事会的一名前成员表示，她的竞选活动仅花费了一百五十美元。即便如此 Alex 还是决定担任他母亲的竞选经理。
  </p>
  <p class="graf graf--p">
   他觉得一万美元就足够了。为了帮助，Citizens for Hospital Accountability 组织在 Cinco de Mayo 举办了筹款活动。

  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   1991年，当苏联解体时，美国政府确信它摆脱了冷战期间面临的许多挑战，对信息战的兴趣逐渐消退。军方的特种部队进入了信息战空窗期。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">

  </p>
  <p class="graf graf--p">

  </p>

  </p>
  <p class="graf graf--p">
   这些网站向访客发送了很多文章，这些文章致力于在医院董事会的斗争中抹黑 Senovia Gutiérrez 及其盟友。
  </p>

   <strong class="markup--strong markup--p-strong">

   </strong>

  </p>

  </p>
  <p class="graf graf--p">
   该网站使用了 Senovia 的昵称 Martha。这些照片似乎是为了使它看起来好像是 Senovia 收受了贿赂。（信封里有一张30美元的 Tommy Hilfiger 礼券）

   <a class="markup--anchor markup--p-anchor" data-href="https://www.ourvalleyvoice.com/2017/07/06/tulare-politics-get-fishy-hospital-recall-nears/" href="https://www.ourvalleyvoice.com/2017/07/06/tulare-politics-get-fishy-hospital-recall-nears/" rel="noopener noreferrer" target="_blank">

  </p>
  <p class="graf graf--p">
   在选举前夕，Alex 的房子被烧毁了，他几乎失去了一切，包括他最后一批竞选传单。
  </p>
  <p class="graf graf--p">
   他怀疑大火可能与选举有关，但当地消防部门官员表示他们没有看到任何违规行为的证据。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9134 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/15-1.jpeg?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/15-1.jpeg?w=2048&amp;ssl=1 2048w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/15-1.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/15-1.jpeg?resize=768%2C432&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/15-1.jpeg?resize=1024%2C576&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/15-1.jpeg?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/15-1.jpeg?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9134" data-recalc-dims="1" height="619" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/15-1.jpeg?resize=1100%2C619&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/15-1.jpeg?w=2048&amp;ssl=1 2048w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/15-1.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/15-1.jpeg?resize=768%2C432&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/15-1.jpeg?resize=1024%2C576&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/09/15-1.jpeg?resize=1100%2C619&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   Burstien 希望 Psy-Group 在 Tulare 的工作能够帮助该公司落实其他小型活动的订单，但事实证明这一点过于乐观了。
  </p>
  <p class="graf graf--p">
   他告诉同事他已接近完成几项交易，但发展新客户的目标失败了，而且，在2018年2月，Burstien 发现他已经无法支付工资单。
  </p>
  <p class="graf graf--p">
   在 Tulare，对 Psy-Group 战略的考验发生在 2017年7月11日晚上。有超过一千人投票，只有一百九十五人投票支持 Kumar。Senovia Gutiérrez 赢得了75％的选票。
  </p>
  <p class="graf graf--p">
   最终，攻击 Senovia 的网站在社区中几乎无人关注。“就像他们组织了一场音乐会，结果没有人出现”，一位计算机安全专家在审查了这些网站的跟踪数据后表示。
  </p>
  <p class="graf graf--p">
   在 Senovia 胜利之后，Benzeevi 的合同被取消了。Larry Blitz 作为临时 CEO 介入，他发现该医院的财务记录完全混乱。
  </p>
  <p class="graf graf--p">
   该医院的债务超过三千六百万美元，不得不关闭近一年。一天早上，Blitz 的首席财务官发现警方正在拿走电脑和电话。作为欺诈调查的一部分，当地地方检察官已经发布了四十多份搜查令，这是当地历史上最大规模的此类调查之一。
  </p>
  <p class="graf graf--p">
   Benzeevi 和他的法律团队拒绝回答有关 Psy-Group 的问题。起初，Kumar 说，他不知道秘密信息战，然后他就不再回电话了。
  </p>
  <p class="graf graf--p">
   根据前公司官员的说法，Zamel 决定在2018年2月关闭 Psy-Group，
   <strong class="markup--strong markup--p-strong">
    但它的消亡并没有抑制它提供的许多服务的胃口。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Psy-Group 的一些前员工与臭名昭著的 Black Cube 公司会面，讨论就业机会
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Black Cube 广受批评，但这些批评似乎并没有伤害其生意；一位熟悉该公司运营的人说各种公司对他们的兴趣日益浓厚。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1998年至2002年间曾担任摩萨德主管的 Efraim Halevy 最近加入了 Black Cube 的顾问委员会
   </strong>
   。
  </p>
  <p class="graf graf--p">
   摩萨德的退伍军人、内塔尼亚胡的前国家安全顾问 Uzi Arad 说，看到他的一些前同事成为“雇佣兵”，他感到羞耻，并补充道，“这是非常不道德的，他们应该知道。”
  </p>
  <p class="graf graf--p">
   去年，Black Cube 搬到了特拉维夫最昂贵的街区之一，现在它在 Bank Discount Tower 占据了一个时尚的全楼层办公室。入口没有标记，漆成黑色；大门由指纹读取器控制。办公室的一个区域装饰着间谍纪念品，包括老式的加密机器。
  </p>
  <p class="graf graf--p">
   一些 Psy-Group 的资深人士对该公司的关闭表示遗憾。 “如果公司仍然开放，
   <strong class="markup--strong markup--p-strong">
    所有这些所谓的负面新闻都会给我们带来很多客户
   </strong>
   ，”一位人士说。
  </p>
  <p class="graf graf--p">
   以色列前高级情报官员表示全球对 “私人摩萨德” 的需求正在增长，并且所谓的影响力运营市场正在扩展到新的商业领域 。
  </p>
  <p class="graf graf--p">
   特别是，这位前官员引用了
   <strong class="markup--strong markup--p-strong">
    使用虚拟人物形象（假人）以影响房地产价格的潜在巨大市场
   </strong>
   —— 例如，通过创造投标人为房产提供更多资金的错觉，或者通过传播有关有毒化学品存在的谣言来制造恐慌……
  </p>
  <p class="graf graf--p">
   一位前 Psy-Group 官员说。假托（袜子木偶）的可能应用列表是“无穷无尽的” ……⚪️
  </p>
