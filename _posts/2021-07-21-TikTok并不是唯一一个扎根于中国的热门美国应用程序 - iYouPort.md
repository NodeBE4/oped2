  <p class="graf graf--p">
   它是世界上下载量最大的新闻应用 — — 而它所承载的内容完全是本地的。在美国，它所带来的流量超过了除三个巨头应用程序之外的所有应用程序。小型出版商对它带来的页面浏览量赞不绝口：2020年约7亿。它的下载量超过了《纽约时报》、BBC甚至谷歌新闻。它正在重新思考如何通过人工智能驱动的界面和其用户渴望的本地COVID-19更新来传递本地新闻。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “它” 就是News Break。2020年，News Break 通过App Store和Google Play拥有4500万月度活跃用户和2400万安装量，它自称是 “全国第一的智能本地新闻平台”。
  </p>
  <p class="graf graf--p">
   然而，
   <strong class="markup--strong markup--p-strong">
    它并没有扎根于它所服务的当地社区；事实上，它是由中国企业家和硅谷的计算机科学家创立的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   认识一下人工智能驱动的聚合器 News Break。
  </p>
  <p class="graf graf--p">
   News Break 是一个跨境公司的典型例子。它的总部设在加州山景城，并在西雅图、北京和上海设有办事处，由成立于2015年的母公司 Particle Media 拥有。News Break 由一个在美国和中国都有丰富经验的中国企业家和工程师组成的团队领导。
   <strong class="markup--strong markup--p-strong">
    首席执行官 Jeff Zheng 是前雅虎高管，他与前百度高管任旭阳于2015年共同创立了 News Break，
   </strong>
   后者于2020年离职。
  </p>
  <p class="graf graf--p">
   News Break 利用从中国引进的策略获得了成功，
   <strong class="markup--strong markup--p-strong">
    在中国，通过算法传递新闻 — — 由字节跳动的 “头条” 开创的做法 — — 已经蓬勃发展。一个由人工智能驱动的 “兴趣引擎” 根据过去的参与度选择读者可能喜欢的文章 — — 就像YouTube和TikTok选择观众可能喜欢的视频那样。即，《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bd%93-%e4%b8%8d%e4%bd%9c%e6%81%b6-%e6%88%90%e4%b8%ba%e7%a9%ba%e8%b0%88%ef%bc%8c%e5%ae%9a%e5%88%b6%e4%ba%ba%e5%b0%86%e7%bb%88%e7%bb%93%e6%b0%91%e4%b8%bb/" href="https://www.iyouport.org/%e5%bd%93-%e4%b8%8d%e4%bd%9c%e6%81%b6-%e6%88%90%e4%b8%ba%e7%a9%ba%e8%b0%88%ef%bc%8c%e5%ae%9a%e5%88%b6%e4%ba%ba%e5%b0%86%e7%bb%88%e7%bb%93%e6%b0%91%e4%b8%bb/" rel="noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     定制人
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》。
   </strong>
  </p>
  <p class="graf graf--p">
   像 News Break 这样的内容聚合器不仅仅是在美国市场上获胜。北京昆仑科技拥有的 Opera News 和深圳 Transsion Holdings 开发的 Scooper News 都已经在非洲和欧洲取得了重大进展。这些应用程序可以针对主流媒体倾向于忽视的受众，而且它们提供的不仅仅是新闻，还有来自互联网各个角落的信息。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国是世界上受控制最严格的新闻环境之一，但在中国发展起来的做法却在全世界获得了 “成功”  — — 由于普遍存在的审查制度
   </strong>
   。如果没有自由访问、分享和讨论某些重大新闻的能力，用户不太可能关注 “朋友” 的分享，这就为算法专制的介入创造了更大的空间。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     进入冠状病毒
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   该公司自己的大突破是在2020年。当它在2016年推出时，根据 Sensor Tower 的数据，它在全球下载量最大的新闻应用中达到了第39位  — — 一个有希望的开始，但并不占优势。然后，COVID-19袭来；根据出版分析公司 Parse.ly 的数据，2020年3月，News Break 的流量激增，比前一个月增长了516%，因为它成为COVID-19本地化更新的首选来源。News Break 现在被列为世界上下载量最大的新闻应用程序之一。
  </p>
  <figure class="graf graf--figure">
   <img class="graf-image aligncenter jetpack-lazy-image" data-height="416" data-image-id="0*kM8DAxNRq7VCs2Bx" data-lazy-src="https://cdn-images-1.medium.com/max/1067/0*kM8DAxNRq7VCs2Bx?is-pending-load=1" data-width="980" src="https://cdn-images-1.medium.com/max/1067/0*kM8DAxNRq7VCs2Bx" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7"/>
   <noscript>
    <img class="graf-image aligncenter" data-height="416" data-image-id="0*kM8DAxNRq7VCs2Bx" data-width="980" src="https://cdn-images-1.medium.com/max/1067/0*kM8DAxNRq7VCs2Bx"/>
   </noscript>
  </figure>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “头条” 的安装量可能要大得多。Sensor Tower 大概没有头条的安卓应用在中国的下载量，因为Google Play在中国无法使用。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “News Break 的特别之处在于，它在当地读者最需要的时候提供了他们所需要的东西”，Parse.ly 负责数据洞察的 Kelsey Arendt 说。Arendt 说，这在其他聚合器中是属于 “附加的或不错的东西”，但对 News Break 来说是 “核心重点”。
  </p>
  <p class="graf graf--p">
   投资者已经注意到了。在1月份完成了1.15亿美元的C轮融资后，News Break 声称它成为了一个新晋的硅谷独角兽。支持者包括著名的 Francisco Partners、IDG Capital 和 ZhenFund。
  </p>
  <p class="graf graf--p" style="padding-left: 40px;">
   <em>
    关于 Francisco Partners，见《
    <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%bf%99%e4%b8%aa%e5%af%8c%e6%9c%89%e7%9a%84%e7%9b%91%e8%a7%86%e6%8a%80%e6%9c%af%e7%bb%8f%e9%94%80%e5%95%86%e6%b5%ae%e5%87%ba%e6%b0%b4%e9%9d%a2-%e8%bf%98%e6%9c%89%e4%bb%96/" href="https://www.iyouport.org/%e8%bf%99%e4%b8%aa%e5%af%8c%e6%9c%89%e7%9a%84%e7%9b%91%e8%a7%86%e6%8a%80%e6%9c%af%e7%bb%8f%e9%94%80%e5%95%86%e6%b5%ae%e5%87%ba%e6%b0%b4%e9%9d%a2-%e8%bf%98%e6%9c%89%e4%bb%96/" rel="noopener" target="_blank">
     这个富有的监视技术经销商浮出水面 …… 还有他那个900万美元WhatsApp骇客工具
    </a>
    》；《
    <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a1%85%e8%b0%b7%e6%8a%95%e8%b5%84%e5%85%ac%e5%8f%b8%e4%bb%8e%e7%9b%91%e8%a7%86%e5%9b%bd%e5%ae%b6%e4%b8%ad%e8%8e%b7%e5%88%a9/" href="https://www.iyouport.org/%e7%a1%85%e8%b0%b7%e6%8a%95%e8%b5%84%e5%85%ac%e5%8f%b8%e4%bb%8e%e7%9b%91%e8%a7%86%e5%9b%bd%e5%ae%b6%e4%b8%ad%e8%8e%b7%e5%88%a9/" rel="noopener" target="_blank">
     硅谷投资公司从监视国家中获利
    </a>
    》
   </em>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     News Break 的特殊性
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   News Break 成功的关键是为每个用户*定制*的人工智能个性化和本地化的新闻源。News Break 称其核心技术为 “兴趣引擎”（interest-based engine）。“基于兴趣的引擎结合了搜索、推荐和智能互动，这需要一个高技术的堆栈”，CEO Zheng 在2017年告诉中国媒体说，“它鼓励用户与产品进行更丰富、更智能的互动。”
  </p>
  <p class="graf graf--p">
   为了让用户着迷，News Break 还利用了频繁的推送通知 — — 自从安装了iPhone版的该应用后，记者平均每天都会收到20条提醒 — — 它还付钱给当地的博主和内容制作者，让他们写文章、发表意见和分享当地的新闻事件。
  </p>
  <p class="graf graf--p">
   新闻只是一个切入点。News Break 主要投资于所谓的 “长尾”，这是一种流行的营销策略，迎合各种小众兴趣，而不是只有几个流行的垂直领域。一旦用户注册了他们感兴趣的垂直领域，News Break 就会推荐更多相关渠道。
  </p>
  <p class="graf graf--p">
   像中国大多数流行的应用程序一样，所有这些应用程序都努力成为一个 “万能的应用程序” ，News Break 并不满足于只做一个新闻聚合器。
   <strong class="markup--strong markup--p-strong">
    为了将用户高度锁定在其生态系统中，News Break 也是一个 Nextdoor 和 Craigslist，为特定社区提供分类信息，如本地工作列表、活动、餐厅、交易和服务。
   </strong>
  </p>
  <p class="graf graf--p" style="padding-left: 40px;">
   <em class="markup--em markup--p-em">
    如果您错过了《
   </em>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ae%a2%e6%88%b7%e4%b9%8b%e5%9f%8e/" href="https://www.iyouport.org/%e8%ae%a2%e6%88%b7%e4%b9%8b%e5%9f%8e/" rel="noopener" target="_blank">
    <em class="markup--em markup--p-em">
     订户之城
    </em>
   </a>
   <em class="markup--em markup--p-em">
    》
   </em>
  </p>
  <p class="graf graf--p">
   凭借对用户兴趣的深入了解，News Break 能够建立出版数据分析师所说的高 “需求”，即每篇文章的平均浏览量，这相当于受众覆盖的质量。
   <strong class="markup--strong markup--p-strong">
    2021年3月，News Break 作为新闻流量来源排名第4，仅次于谷歌、SmartNews 和 Flipboard。但根据 Parse.ly 的数据，News Break 每篇文章的平均浏览量要高于谷歌新闻。
   </strong>
  </p>
  <figure class="graf graf--figure">
   <img class="graf-image aligncenter jetpack-lazy-image" data-height="411" data-image-id="0*4kXg6CFVadH9io4C" data-lazy-src="https://cdn-images-1.medium.com/max/1067/0*4kXg6CFVadH9io4C?is-pending-load=1" data-width="980" src="https://cdn-images-1.medium.com/max/1067/0*4kXg6CFVadH9io4C" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7"/>
   <noscript>
    <img class="graf-image aligncenter" data-height="411" data-image-id="0*4kXg6CFVadH9io4C" data-width="980" src="https://cdn-images-1.medium.com/max/1067/0*4kXg6CFVadH9io4C"/>
   </noscript>
  </figure>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     News Break 不会拯救本地新闻
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   小型和本地出版商对 News Break 给他们网站带来的流量赞不绝口。根据 Parse.ly 的数据，在2020年，News Break 成为他们各自网络中的顶级推荐者之一，仅次于谷歌和 Flipboard。
  </p>
  <p class="graf graf--p">
   但像 News Break 这样的聚合器不会取代扎根于读者社区的本地报纸，Lenfest 新闻研究所的编辑和通信主管 Joseph Lichterman 说，该研究所是一家位于费城的以新闻业为重点的非营利组织，他说，“News Break 缺乏独立的和原创的声音以及社区感，而这对本地新闻来说是至关重要的”。
  </p>
  <p class="graf graf--p">
   该应用程序也不会对当地报纸的底线有多大帮助，这些报纸越来越依赖通过订阅或会员制获得的直接读者收入，而数字广告收入却在不断减少”，Lichterman 说。这些类型的新闻聚合器可能有助于增加原始流量，但它们不太可能形成将读者转化为付费支持者的那种习惯。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     TikTok不是
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   网络安全专家担心，在美中关系紧张的时候，News Break 在中国的存在 — — 微软和谷歌等跨国公司也是如此 — — 如果它继续增长，可能会吸引更多的审查。例如，鉴于其中国所有权，在美国受欢迎的应用程序TikTok去年成为美国关于与北京共享数据的争议中心，尽管最近公民实验室的审计发现该应用程序并没有对美国构成国家安全风险。
  </p>

    </em>

   <em class="markup--em markup--p-em">
    》
   </em>
  </p>
  <p class="graf graf--p">
   News Break 没有回应评论请求。但一位要求匿名的 News Break 员工告诉记者，该公司的所有数据都存储在亚马逊网络服务上，其服务器位于美国。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果数据存储在美国的AWS，我不知道中国政府如何能在AWS不知道的情况下获取这些数据。中国没有法律明确授予中国政府这种权力”，New America 的网络政策研究员萨克斯（Samm Sacks）说。
  </p>
  <p class="graf graf--p">
   虽然 News Break 在中国的存在会引起有关信息控制的潜在问题  — — 美国读者看到的新闻在多大程度上可能受到审查，以及它如何使人们暴露在虚假信息中。但专家说，如果不对该应用程序的数据基础设施和公司的架构进行研究，就很难评估人们应该对这些问题有多关注。
  </p>
  <p class="graf graf--p">
