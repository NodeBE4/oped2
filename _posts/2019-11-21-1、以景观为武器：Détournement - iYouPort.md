  <p class="graf graf--p">
   现代资本主义通过景观维护社会控制，利用大众传播将我们所有人变成了单纯的消费者，变成了自己生活、历史和权力的
   <strong class="markup--strong markup--p-strong">
    被动观众
   </strong>
   。
  </p>
  <p class="graf graf--p">
   人们由此失去了对自己的一切的掌控权。
  </p>
  <p class="graf graf--p">
   这就是法国哲学家兼活动家 Guy Debord 在1967指出的问题。这本著作直到1995年才被翻译成英文出版。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    在以现代生产条件为主导的社会中，生活被视为一种巨大的景观积累，直接生活的一切都已经消失了
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   这种与感受与体验分离的政治后果是理解我们如何体验世界以及如何改变世界的关键。
  </p>
  <p class="graf graf--p">
   Debord 的《景观社会》和 Raoul Vaneigem 的《日常生活的革命 The Revolution of Everyday Life》几乎同时出版，两本书都详尽阐述了景观的概念 —— 这个概念几乎在五十年代初就已经出现，并贯穿了情境主义国际后来的整个革命历史。
  </p>
  <p class="graf graf--p">
   正是这两本书，在理论逻辑方面将情境主义国际推向了学术巅峰。
  </p>
  <p class="graf graf--p">
   想想看，那些目击灾难性事件的人如何经常说这种体验“像电影一样”。
  </p>
  <p class="graf graf--p">
   同样，作为积极人士，往往更关注的是媒体对我们的行动产生的关注，而不是行动本身最终的结果。
  </p>
  <p class="graf graf--p">
   还记得我们对美国著名反对派作家 Don DeLillo 代表作《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%8e%b0%e4%bb%a3%e6%96%b0%e9%97%bb%e4%b9%8b%e6%84%9a%e8%a0%a2-2%e2%80%8a-%e2%80%8a%e5%9b%9b%e7%a7%8d%e6%9c%80%e8%87%b4%e5%91%bd%e7%9a%84%e5%80%be%e5%90%91%e6%80%a7-%e4%b8%ad%e5%9b%bd/" href="https://www.iyouport.org/%e7%8e%b0%e4%bb%a3%e6%96%b0%e9%97%bb%e4%b9%8b%e6%84%9a%e8%a0%a2-2%e2%80%8a-%e2%80%8a%e5%9b%9b%e7%a7%8d%e6%9c%80%e8%87%b4%e5%91%bd%e7%9a%84%e5%80%be%e5%90%91%e6%80%a7-%e4%b8%ad%e5%9b%bd/" rel="noopener noreferrer" target="_blank">
    白噪音
   </a>
   》的推荐吗？这部作品中就充分展示了情境主义国际的中心论点。
  </p>
  <p class="graf graf--p">
   这就是现实。持续了几十年的现实。
  </p>
  <p class="graf graf--p">
   我们的感受、我们的信念、我们如何表达愿望、我们所相信的可能性 —— 所有这一切都被我们消费和生产的媒体所过滤和约束。
  </p>
  <p class="graf graf--p">
   如今已经很多人认识到了，
   <strong class="markup--strong markup--p-strong">
    这样的世界是对所有人的绑架
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这就是法国情境主义运动领导人 Debord 所描述和谴责的景观社会。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这个集中化的互联网就是景观社会的全球性跃进
   </strong>
   。大众传播的变革展示了资本主义如何不仅渗透所有人生产和消费的东西，而且，渗透了我们如何沟通、如何表达、如何与社会关联……
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    景观 —— 如大众娱乐、新闻和广告所表现的 —— 使我们与自己和自己的欲望迅速疏远，所有这一切都是为了促进资本的积累
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    景观越来越成为当权者实施社会控制的主要机制
   </strong>
   。这是诱惑和分心的控制，而不是力量。
  </p>
  <p class="graf graf--p">
   就如 Debord 所言，所有人的生活已经退化，从掌控生活退化到仅仅“拥有”生活，继而又从“拥有”退化到旁观。
  </p>
  <p class="graf graf--p">
   ⚠️这种退化已经深入到每个人的意识里。
  </p>
  <p class="graf graf--p">
   想想看，日常中的“活动家”们有多少人只关心维持我们作为积极分子的自我形象？—— ⚠️我们不再首先考虑实战的打击力度，而是沉迷于自己的 poses “看起来怎么样”。
  </p>
  <p class="graf graf--p">
   为了提出可信的挑战，为了寻求使所有人摆脱景观的操控之力，情境主义者引入了一种解决方案：试图
   <strong class="markup--strong markup--p-strong">
    扭转景观的力量以对抗其自身
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这就是今天的主题：给行动者的建议。
  </p>
  <p class="graf graf--p">
   💡改变目标消息或品牌的含义；将关键信息打包为具有高度传染性的媒体病毒。
  </p>
  <p class="graf graf--p">
   每一种城市生活都包围在广告、企业艺术和大众媒介流行文化的日常冲击中。
  </p>
  <p class="graf graf--p">
   尽管这种景象可能具有压迫性和疏离性，但它的无处不在为*符号学柔术*和*创造性破坏*提供了充足的机会。
  </p>

   <li class="graf graf--li">
    回顾《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/3%e3%80%81%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%a1%8c%e5%8a%a8%e5%8f%96%e5%86%b3%e4%ba%8e%e4%bd%a0%e7%9a%84%e7%9b%ae%e6%a0%87%e7%9a%84%e5%8f%8d%e5%ba%94%ef%bc%9a%e6%88%98%e7%95%a5%e6%9f%94%e6%9c%af/" href="https://www.iyouport.org/3%e3%80%81%e7%9c%9f%e6%ad%a3%e7%9a%84%e8%a1%8c%e5%8a%a8%e5%8f%96%e5%86%b3%e4%ba%8e%e4%bd%a0%e7%9a%84%e7%9b%ae%e6%a0%87%e7%9a%84%e5%8f%8d%e5%ba%94%ef%bc%9a%e6%88%98%e7%95%a5%e6%9f%94%e6%9c%af/" rel="noopener noreferrer" target="_blank">

    </a>

   </li>

  <p class="graf graf--p">
   颠覆性和边缘化的观念可以通过
   <strong class="markup--strong markup--p-strong">
    重新利用
   </strong>
   从大众媒体中提取的元素并对其注入激进的内涵，以获得广泛的传播。
  </p>
  <p class="graf graf--p">
   这种技术被称为 détournement。由 Guy Debord 和情境主义者们广泛推动，这个词来自法语，大致翻译为“倾覆”或“反转”。
  </p>
  <p class="graf graf--p">
   Détournement 是当代媒体作品的变体，它是一种 “将旧有作品以颠倒的方式重新创作”的手法，而被选做重新创作的原作品（détourned）必须是一个被大众所熟悉的媒介，以便能够有效和迅速的传达
   <strong class="markup--strong markup--p-strong">
    与原作相反
   </strong>
   的意图和信息。
  </p>
  <p class="graf graf--p">
   类似一种有讽刺意味的模仿，不过它是采取直接引用或忠实模拟原始作品来制造这种强烈暗示的，而
   <strong class="markup--strong markup--p-strong">
    不是
   </strong>
   重新创作新的作品来达到同样的效果。
  </p>
  <p class="graf graf--p">
   以此来
   <strong class="markup--strong markup--p-strong">
    传达相反的意念、颠覆主流媒体、及传递自己的作品
   </strong>
   。
  </p>
  <p class="graf graf--p">
   IYP曾经介绍过的替代媒体运动就非常适合借助 Détournement 的手法。见：
  </p>
  <ul class="postList">

  </p>
  <p class="graf graf--p">
   1、冲突：要解决的问题或冲突是什么？如何构筑框架，该框架遗漏了什么？
  </p>
  <p class="graf graf--p">
   2、人物：这可能是一个深刻的组织问题，即 — 谁是“我们”？这个故事中的其他角色是谁？角色是自己说话还是代表他们希望表达的人？
  </p>
  <p class="graf graf--p">
   3、意向：什么样的强大图像可以帮助传达完整的意向？是否有可以描述问题的比喻或类比？
  </p>
  <p class="graf graf--p">

    展示，而不是告知
