  <p class="graf graf--p">
   ⚠️如今，人们使用不同的平台来共享有关自己的信息，却没有意识到这些信息会如何暴露出自己的习惯、兴趣、位置和明显的见解 ——
   <strong class="markup--strong markup--p-strong">
    这非常危险
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    无数种机构包括政府间谍一直在广泛地使用社交媒体监视以观看特定的关键字、主题标签、使用信息建立支持情报的联系，并跟踪他们想要的任何人。
   </strong>
  </p>
  <p class="graf graf--p">
   虽然这里面的动机并非都是邪恶的。
  </p>
  <p class="graf graf--p">

   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%bd%91%e7%bb%9c%e6%b0%b4%e5%86%9b%e5%a6%82%e4%bd%95%e8%af%95%e5%9b%be%e5%bd%b1%e5%93%8d%e9%80%89%e4%b8%be%ef%bc%9f-%e4%b8%80%e4%b8%aa%e8%b0%83%e6%9f%a5%e6%a1%88%e4%be%8b%ef%bc%8c%e8%a7%82%e5%af%9f/" href="https://www.iyouport.org/%e7%bd%91%e7%bb%9c%e6%b0%b4%e5%86%9b%e5%a6%82%e4%bd%95%e8%af%95%e5%9b%be%e5%bd%b1%e5%93%8d%e9%80%89%e4%b8%be%ef%bc%9f-%e4%b8%80%e4%b8%aa%e8%b0%83%e6%9f%a5%e6%a1%88%e4%be%8b%ef%bc%8c%e8%a7%82%e5%af%9f/" rel="noopener noreferrer" target="_blank">

   </a>

    照片地理位置定位
   </a>
   以识别虚假信息、或者查找并警告
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%8d%e8%a6%81%e5%bf%bd%e8%a7%86%e5%a8%81%e8%83%81%e6%83%85%e6%8a%a5%e4%b8%ad%e7%9a%84%e5%9c%b0%e7%bc%98%e6%94%bf%e6%b2%bb/" href="https://www.iyouport.org/%e4%b8%8d%e8%a6%81%e5%bf%bd%e8%a7%86%e5%a8%81%e8%83%81%e6%83%85%e6%8a%a5%e4%b8%ad%e7%9a%84%e5%9c%b0%e7%bc%98%e6%94%bf%e6%b2%bb/" rel="noopener noreferrer" target="_blank">
    威胁因素
   </a>
   的存在，等等。
  </p>
  <p class="graf graf--p">
   早在五年前IYP创立之初我们就开始警告：⚠️切勿在各种不同的网络平台上使用相同的用户名/ID/照片和简介，因为
   <strong class="markup--strong markup--p-strong">
    这样做非常危险
   </strong>
   —— 相当于给了监视跟踪一个极为便利的机会。
  </p>
  <p class="graf graf--p">
   但直到如今，依旧有很多人在这样做。
  </p>
  <p class="graf graf--p">
   如果您很受欢迎、是网络名人，并且希望仅通过自己的名字被他人认识，这可以说是很平常的事，但是，⚠️建议您不要将暗网或黑客论坛中的身份与任何社交媒体服务混在一起，否则…… 那简直太糟糕了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    缺乏基本的操作安全性（OPSEC）是一切灾难之源
   </strong>
   。
  </p>

  </p>

   ⚠️因为每个人的脑回路都是有固定模式的，你平日可能没有注意到这点。于是，如果两个账户的操作者为同一个人，那么几乎肯定会漏出马脚。
  </p>
  <p class="graf graf--p">
   语言学取证是非常专业的，并且只能由专业人员来得出结论，例如确认某人的原籍国、性情偏好、关联并揭示多重虚假身份等等。
  </p>
  <p class="graf graf--p">
   某些字词、句子或成语，在不同国家/地区具有独特的含义，不同领域（工作经历）也会导致人们产生趋同的语言习惯；根据这项研究，可以将两个身份的写作风格联系起来，并挖掘到背后人的真实身份。
  </p>

   <h3 class="graf graf--p">
    <span style="color: #800080;">
     <strong class="markup--strong markup--p-strong">
      SocialPath
     </strong>
    </span>
   </h3>
   <p class="graf graf--p">
    查看与个人相关的所有帖子、推文或照片是繁琐且耗时的工作，手动做会很辛苦 …… 幸运的是，相关自动化的便捷工具其实很多。
   </p>
   <p class="graf graf--p">
    SocialPath 是一款简单的应用程序，可在社交媒体（Facebook、Instagram、Twitter、Reddit、和 Stackoverflow）中查找帐户。
   </p>
   <p class="graf graf--p">
    它使用 Django 作为后端，并使用 D3js 绘制图表；并提供有关个人资料的基本信息，例如 简介、网址、关注者或朋友。
   </p>
   <p class="graf graf--p">
    详细信息页面包含基于用户活动的图表，即 热图日历、lollipop图表、滚动图和 wordcloud。它用于显示用户时间轴、主题标签、词频、提及、评论、以及更多能揭露帐户类型的信息。

   </p>

    <img alt="" class="aligncenter size-full wp-image-9848 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/21.jpeg?resize=1100%2C331&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/21.jpeg?w=1538&amp;ssl=1 1538w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/21.jpeg?resize=300%2C90&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/21.jpeg?resize=768%2C231&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/21.jpeg?resize=1024%2C308&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/21.jpeg?resize=1100%2C331&amp;ssl=1 1100w" data-recalc-dims="1" height="331" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/21.jpeg?resize=1100%2C331&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

     https://github.com/woj-ciech/SocialPath
    </a>
   </p>
   <h3 class="graf graf--p">
    <span style="color: #ff6600;">
     <strong class="markup--strong markup--p-strong">
      结论
     </strong>
    </span>
   </h3>
   <p class="graf graf--p">
    处于显而易见的原因，这里不会演示任何真实的调查过程。您可以自己实践。
   </p>
   <p class="graf graf--p">
    ⚠️
    <strong class="markup--strong markup--p-strong">
     如上所述，如果您使用互联网，就必须能做到
    </strong>
    ：
   </p>
   <ul class="postList">
    <li class="graf graf--li">
     <strong class="markup--strong markup--li-strong">
      切勿
     </strong>
     将不同平台上的身份信息发生关联 —— 不论是名字、照片、语言习惯、还是兴趣偏好或简介；
    </li>
    <li class="graf graf--li">
     暗网是相对安全的，但暗网也是警察间谍的重点监视目标（他们的逻辑很傻逼，他们认为“你如果不想鼓捣什么猫腻就不会想使用暗网”）；所以，
     <strong class="markup--strong markup--li-strong">
      操作安全
     </strong>
     是一切的基础！不要犯低端错误，见《
     <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bd%bf%e7%94%a8-tor-%e4%bf%9d%e6%8a%a4%e8%87%aa%e5%b7%b1%e6%97%b6%e5%8d%83%e4%b8%87%e4%b8%8d%e8%a6%81%e5%81%9a%e8%bf%99%e4%b9%9d%e4%bb%b6%e4%ba%8b%ef%bc%81/" href="https://www.iyouport.org/%e4%bd%bf%e7%94%a8-tor-%e4%bf%9d%e6%8a%a4%e8%87%aa%e5%b7%b1%e6%97%b6%e5%8d%83%e4%b8%87%e4%b8%8d%e8%a6%81%e5%81%9a%e8%bf%99%e4%b9%9d%e4%bb%b6%e4%ba%8b%ef%bc%81/" rel="noopener noreferrer" target="_blank">
      使用 Tor 保护自己时千万不要做这九件事
     </a>
     》；
    </li>
    <li class="graf graf--li">
     <strong class="markup--strong markup--li-strong">
      在线分身
     </strong>
     是非常必要的，其关键在于每一种分离角色之间都不可以有任何相关性！必须不能有其他人知道那是您的分身。
    </li>
   </ul>
   <p class="graf graf--p">
    SocialPath 表明，任何人都可以轻松创建此类挖掘社交媒体情报的应用程序，因此你必须假定，
    <strong class="markup--strong markup--p-strong">
     政府间谍可以拥有更为强大的情报挖掘工具、实时监控能力、以及更大的数据库 —— 包括来自先前被抢占市场的转储
    </strong>
    。
   </p>
   <p class="graf graf--p">
    Be careful! ⚪️
   </p>
