  <p class="graf graf--p graf-after--figure" id="3296">
   知情权即公民权，即民主的动力。因为公民只有在充分了解政权都干了什么的时候，才能做出正确的选择。撒谎是权力运作的基本模式，公民了解得越少、判断力越低下越容易被愚弄，权力被监督的程度就越低、异议就越少，政权就越能稳固。所以，是真相在决定我们能否享受民主，还是只能被威权。
  </p>
  <p class="graf graf--p graf-after--p" id="9a8e">
   如果您一直在阅读我们的文章会能有一个清晰的印象，即 我们一直在强调真相的强大力量，不论是透明度革命还是开源调查（OSINT）- 推广挖掘真相的技术能力，同时，也为中国社会的真相淡漠之现状深深惋惜。
  </p>
  <p class="graf graf--p graf-after--p" id="1450">
   但这种状况 — 即 大众甚至决策者无视事实的盲目和荒诞 — 并非中国所特有，在很多民主国家也是如此，并且后者所带来的灾难格外令人焦虑。
  </p>
  <p class="graf graf--p graf-after--p" id="0f86">
   威权社会对真相失去兴趣的结果是丧失变革的机会，“帮助”独裁政府稳固了权力；而民主社会无视真相的结果是直接导致民主的衰落，威权崛起。
  </p>
  <p class="graf graf--p graf-after--p" id="8124">
   为什么会这样？兰德公司最近提交的一份报告详细分析了在美国发生的这种状况，被称之为“真相衰变”。这是什么意思？是什么导致的真相衰变？这是新事物吗？它的后果是什么？然后呢？我们还可以怎么办。
  </p>
  <p class="graf graf--p graf-after--p" id="74d0">
   您可以
   <a class="markup--anchor markup--p-anchor" data-href="https://www.rand.org/content/dam/rand/pubs/research_reports/RR2300/RR2314/RAND_RR2314.pdf" href="https://www.rand.org/content/dam/rand/pubs/research_reports/RR2300/RR2314/RAND_RR2314.pdf" rel="noopener nofollow noreferrer" target="_blank">
    在这里看到
   </a>
   这份报告的原版（长达326页）。本文将提取其主体框架，并结合中国的现实，做出一些思考。
  </p>
  <p class="graf graf--p graf-after--p" id="10fe">
   在过去二十年中，美国的国家政治和民间话语的整体特点可以说就是“真相衰变”，这一概念被定义为一组四个相互关联的趋势：
  </p>
  <ol class="postList">
   <li class="graf graf--li graf-after--p" id="2cde">
    对事实和事实的数据分析性解释日益增加的分歧;
   </li>
   <li class="graf graf--li graf-after--li" id="6d38">
    意见与事实之间界限的模糊化;
   </li>
   <li class="graf graf--li graf-after--li" id="d299">
    意见和个人经验对事实的认识产生越来越多的影响;
   </li>
   <li class="graf graf--li graf-after--li" id="3d59">
    对曾经受尊重的事实信息来源的信任明显下降。
   </li>
  </ol>
  <p class="graf graf--p graf-after--li" id="7b32">
   对于趋势 1，我们曾经在《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%AA%92%E4%BD%93-%E4%BD%A0%E7%9C%9F%E7%9A%84%E7%9F%A5%E9%81%93%E4%BB%80%E4%B9%88%E6%98%AF%E4%BA%8B%E5%AE%9E%E5%90%97-f86286d2b040" href="https://medium.com/@iyouport/%E5%AA%92%E4%BD%93-%E4%BD%A0%E7%9C%9F%E7%9A%84%E7%9F%A5%E9%81%93%E4%BB%80%E4%B9%88%E6%98%AF%E4%BA%8B%E5%AE%9E%E5%90%97-f86286d2b040" rel="noopener noreferrer" target="_blank">
    媒体，你真的知道什么是事实吗
   </a>
   》这篇文章中引用过一段 Jon Stewart 在《夜线》节目中的经典对话，可以说明这种现象的存在（进入上面链接可见）。
  </p>
  <p class="graf graf--p graf-after--p" id="e020">
   相比下这还是有数据情况下的分歧（糟糕的是数据存在严重差异），而在中国，完全可以没有调查数据仅凭个人视角的一面之词就可以下结论 — — 其分歧显然高度基于个人经验差异。最近期出现的一个例子就如“消费降级”论。最初来自纽约时报一篇简短的专栏文章，作者是中国人，文章中没有给出足够的调查数据便得出了“消费降级”的结论，然而这一结论却迅速成为了流行语，引发了舆论热潮，卖热点的内容营销者们纷纷补刀。中国的微信指数显示，在8月22日到8月27日这不到一周的时间里，中国人对于消费降级的讨论达到了前所未有的巅峰，微信指数环比增长一度超过1000%。
  </p>

  <p class="graf graf--p graf-after--figure" id="1f37">
   但事实是什么呢？就算拿不到统计数据简单观察也能看到，中国电商平台上的高级商品正在呈现增多的趋势，与此同时“拼多多”等低价平台也十分火爆，整体上只能算是消费分层。但依旧没有真实的数据可以说明问题。可为什么如此快速地引发了舆论热潮？难道没有人意识到支持这一结论的证据的缺失吗？很可能的确没有。这正是上述趋势 2 中所指出的。
  </p>
  <p class="graf graf--p graf-after--p" id="afe1">
   趋势 2 是个极为典型的现象，即 “我才是对的” — 人们被个人意见所引导，将意见误认为是事实。我们曾经将其解释为社交媒体时代特有的自恋和偏执。详见“
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/761" href="https://pao-pao.net/article/761" rel="noopener nofollow noreferrer" target="_blank">
    自恋型人格与社交网络媒介的关系
   </a>
   “，以及”
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/778" href="https://pao-pao.net/article/778" rel="noopener nofollow noreferrer" target="_blank">
    自恋型偏执如何激发了政治极化和语言暴力
   </a>
   “。
  </p>
  <p class="graf graf--p graf-after--p" id="3e4f">
   在“消费降级”的例子中，这一“意见”分为两个方面，一方面来自中国大众，尤其是被高房价拖累、背负着沉重贷款的中产阶级，消费降级这一结论如果成真，将在感性层面上给这一阶层群体带来安慰，于是显然，他们倾向于认可这一结论，而不需要论据；另一方面来自纽约时报，这一结论显然具有吸睛度，并且符合媒体对长期经济数据造假的中国社会的推测。这篇文章就是这样被发布出来的。
  </p>
  <p class="graf graf--p graf-after--p" id="fc02">
   至于为什么忽然成了流行语，显然，这其中也包含了上述趋势 3 中的表述。
  </p>
  <p class="graf graf--p graf-after--p" id="f3c0">
   如果你想要，这类例子遍地都是。
  </p>
  <p class="graf graf--p graf-after--p" id="5e4f">
   对于趋势 4 来说，中国社会并不十分明显，也没有足够的调查能支持这点。但其中一种变化似乎存在，那就是对所谓的自媒体的信任正在下降，这是由于审查的广泛和不断的深入的结果，加之
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%B7%A8%E9%AD%94%E5%B4%9B%E8%B5%B7-%E6%94%BF%E5%BA%9C%E8%B5%84%E5%8A%A9%E7%9A%84%E7%BD%91%E7%BB%9C%E6%B0%B4%E5%86%9B%E6%AD%A3%E9%81%8D%E5%B8%83%E5%85%A8%E7%90%83-a45710e8e431" href="https://medium.com/@iyouport/%E5%B7%A8%E9%AD%94%E5%B4%9B%E8%B5%B7-%E6%94%BF%E5%BA%9C%E8%B5%84%E5%8A%A9%E7%9A%84%E7%BD%91%E7%BB%9C%E6%B0%B4%E5%86%9B%E6%AD%A3%E9%81%8D%E5%B8%83%E5%85%A8%E7%90%83-a45710e8e431" rel="noopener noreferrer" target="_blank">
    隐性外宣的无处不在
   </a>
   和难以分辨、更有隧道化的 Twitter 中文网络上假消息的盛行，一些人甚至患上了中文恐惧症，毫不夸张，他们开始厌倦一切中文内容，不论其是否有道理。
  </p>
  <p class="graf graf--p graf-after--p" id="a1f0">
   而趋势 4 在美国是极为明显的，即 主流媒体的衰落。您也可以在这
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%AA%92%E4%BD%93-%E4%BD%A0%E7%9C%9F%E7%9A%84%E7%9F%A5%E9%81%93%E4%BB%80%E4%B9%88%E6%98%AF%E4%BA%8B%E5%AE%9E%E5%90%97-f86286d2b040" href="https://medium.com/@iyouport/%E5%AA%92%E4%BD%93-%E4%BD%A0%E7%9C%9F%E7%9A%84%E7%9F%A5%E9%81%93%E4%BB%80%E4%B9%88%E6%98%AF%E4%BA%8B%E5%AE%9E%E5%90%97-f86286d2b040" rel="noopener noreferrer" target="_blank">
    篇文章
   </a>
   中读到对这一状况的描述。关于为什么会这样，说来话长，这里有一些近期的证据可供参考《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%BC%BA%E6%9D%83%E8%83%BD%E6%9D%80%E6%AD%BB%E8%AE%B0%E8%80%85-%E4%BD%86%E6%97%A0%E6%B3%95%E6%9D%80%E6%AD%BB%E6%96%B0%E9%97%BB-2018-%E6%96%B0%E6%8A%A5%E5%91%8A%E5%87%BA%E5%8F%B0-%E4%B8%BA%E4%BB%80%E4%B9%88%E5%85%A8%E7%90%83%E6%96%B0%E9%97%BB%E8%87%AA%E7%94%B1%E6%8C%87%E6%95%B0%E4%BB%8D%E5%9C%A8%E4%B8%8B%E9%99%8D-4d73bea6d5b5" href="https://www.iyouport.org/%E7%8E%B0%E4%BB%A3%E6%96%B0%E9%97%BB%E4%B9%8B%E6%84%9A%E8%A0%A2%E5%B9%B6%E9%9D%9E%E6%84%8F%E8%AF%86%E5%BD%A2%E6%80%81%E5%80%BE%E5%90%91%E6%80%A7%E9%97%AE%E9%A2%98-1-%E6%94%BF%E6%B2%BB%E7%9C%9F/" rel="noopener noreferrer" target="_blank">
    一
   </a>
   》《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E7%94%B5%E5%AD%90%E9%82%AE%E4%BB%B6%E8%A2%AB%E6%9B%9D%E5%85%89-%E6%8F%AD%E7%A4%BA%E8%AE%B0%E8%80%85%E4%B8%BB%E5%8A%A8%E6%8E%A5%E5%8F%97%E6%83%85%E6%8A%A5%E9%83%A8%E9%97%A8%E7%9A%84%E5%AE%A1%E6%9F%A5-%E8%BF%99%E7%A9%B6%E7%AB%9F%E6%98%AF%E4%B8%BA%E4%BB%80%E4%B9%88-d2745e575501" href="https://www.iyouport.org/%E7%8E%B0%E4%BB%A3%E6%96%B0%E9%97%BB%E4%B9%8B%E6%84%9A%E8%A0%A2-2%e2%80%8a-%e2%80%8a%E5%9B%9B%E7%A7%8D%E6%9C%80%E8%87%B4%E5%91%BD%E7%9A%84%E5%80%BE%E5%90%91%E6%80%A7-%E4%B8%AD%E5%9B%BD/" rel="noopener noreferrer" target="_blank">
    二
   </a>
   》《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E7%89%B9%E6%9C%97%E6%99%AE%E6%94%BF%E5%BA%9C%E5%AF%B9%E5%AA%92%E4%BD%93%E7%9A%84%E6%88%98%E4%BA%89%E4%B8%8D%E5%90%8C%E4%BA%8E%E5%A5%A5%E5%B7%B4%E9%A9%AC%E6%97%B6%E4%BB%A3-%E5%AE%83%E5%B7%B2%E5%8F%98%E5%BE%97%E6%9B%B4%E5%8A%A0%E9%98%B4%E6%9A%97-6a12cd3fd022" href="https://www.iyouport.org/%E7%8E%B0%E4%BB%A3%E6%96%B0%E9%97%BB%E4%B9%8B%E6%84%9A%E8%A0%A2-3%e2%80%8a-%e2%80%8a%E6%B7%B1%E5%BA%A6%E5%89%96%E6%9E%90%E5%9B%9B%E7%A7%8D%E8%87%B4%E5%91%BD%E5%80%BE%E5%90%91%E6%80%A7/" rel="noopener noreferrer" target="_blank">
    三
   </a>
   》。
  </p>
  <p class="graf graf--p graf-after--p" id="f9f0">
   当 Wikileaks 揭露民主党真相时，获得了共和党的高调称赞；当 Wikileaks
   <strong class="markup--strong markup--p-strong">
    继续
   </strong>
   揭露共和党的真相时，遭到了两党联手的抨击。当棱镜计划被曝光时，俄罗斯立刻引其为意识形态武器，而三年之后，
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E4%BF%84%E7%BD%97%E6%96%AF%E6%AD%A3%E6%A8%A1%E4%BB%BF%E7%BE%8E%E5%9B%BD%E6%A3%B1%E9%95%9C%E8%AE%A1%E5%88%92%E5%BB%BA%E7%AB%8B%E5%A4%A7%E8%A7%84%E6%A8%A1%E7%9B%91%E8%A7%86%E7%B3%BB%E7%BB%9F-wikileaks-%E6%8A%AB%E9%9C%B2-%E4%B8%AD%E6%96%87%E7%89%88-53580712e7db" href="https://medium.com/@iyouport/%E4%BF%84%E7%BD%97%E6%96%AF%E6%AD%A3%E6%A8%A1%E4%BB%BF%E7%BE%8E%E5%9B%BD%E6%A3%B1%E9%95%9C%E8%AE%A1%E5%88%92%E5%BB%BA%E7%AB%8B%E5%A4%A7%E8%A7%84%E6%A8%A1%E7%9B%91%E8%A7%86%E7%B3%BB%E7%BB%9F-wikileaks-%E6%8A%AB%E9%9C%B2-%E4%B8%AD%E6%96%87%E7%89%88-53580712e7db" rel="noopener noreferrer" target="_blank">
    俄罗斯的大规模监视系统被曝光
   </a>
   时……如你所知。这些看上去像是普通党争和地缘政治游戏的现象，有可能并没那么简单，它暗示了更为广泛的问题。
  </p>
  <p class="graf graf--p graf-after--p" id="9fa8">
   显然，这些趋势有很多原因，并且对于不同社会特征来说其侧重也会有所不同。但兰德的报告主要关注四个方面：
  </p>
  <ol class="postList">
   <li class="graf graf--li graf-after--p" id="f30c">
    人类认知加工的特征，如认知偏差等;
   </li>
   <li class="graf graf--li graf-after--li" id="f5de">
    全球信息系统的变化，包括社交媒体的流行和 24 小时新闻周期;
   </li>
   <li class="graf graf--li graf-after--li" id="e949">
    对教育系统的竞争要求减少了社会在媒体素养和批判性思维上花费的时间;
   </li>
   <li class="graf graf--li graf-after--li" id="d3da">
    极化问题，无论是政治还是社会人口。
   </li>
  </ol>
  <p class="graf graf--p graf-after--li" id="746f">
   <strong class="markup--strong markup--p-strong">
    这四个方面可以说是抓得非常准确
   </strong>
   ，并且对于中国来说同样适用。这些问题我们在曾经的文章中基本都分析过，比如“
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/782" href="https://pao-pao.net/article/782" rel="noopener nofollow noreferrer" target="_blank">
    互联网人为何如此容易被骗
   </a>
   ”、“
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/773" href="https://pao-pao.net/article/773" rel="noopener nofollow noreferrer" target="_blank">
    隧道生态和心理操控
   </a>
   ”，等等。
  </p>
  <p class="graf graf--p graf-after--p" id="f512">
   其中不得不提及的是“24小时新闻周期” — — 今日的新消息迅速覆盖昨日，不论昨日有多么的轰动，一切犹如过眼云烟。这是与碎片化的、冲动短视的互联网媒介效应直接相关的，即 人们已经来不及关注问题的解决，仅仅只能关注问题的存在，显然，对于政权来说这是绝佳的环境，只要短暂的敷衍、压制舆论，人们就会迅速遗忘，一切都无需改变。然后，同样的问题继续出现，继续这一流程，直到所有人习以为常懒得再次辩论。
  </p>
  <p class="graf graf--p graf-after--p" id="6e68">
   此外人类认知缺陷是固有的，它可以通过教育的深入和普及加以修正弥补，而媒体素养和批判性思维教育更有可能是媒介效应改变的结论，也包括复杂的极化问题。
  </p>
  <p class="graf graf--p graf-after--p" id="c418">
   <strong class="markup--strong markup--p-strong">
    真相衰变最具破坏性的后果包括对民间话语的侵蚀、政治瘫痪、个人与政治和民间机构的异化和脱离，以及国家政策的不确定性
   </strong>
   。
  </p>
