  <p class="graf graf--p">
   如今科技行业并没有像几年前那样使人们着迷了。跟上它的问题及其修补程序以及引起新问题的修补程序，这令人头晕。很难从噪声中分离出有意义的威胁。这种状况非常不利于反抗 —— 校准失败。
  </p>
  <p class="graf graf--p">
   于是，我们需要一个清晰的列表。
  </p>
  <p class="graf graf--p">
   Slate 联系了许多多年来一直对技术进行批判性思考的记者、学者、拥护者和其他人，要求他们指出他们认为最值得警惕的技术公司，然后让他们解释为什么这些公司值得警惕。
  </p>
  <p class="graf graf--p">
   每个受访者分别对10家公司进行了排名（子公司被视为母公司的一部分），然后将他们的答案加在一起，得到了这一列表。
  </p>
  <p class="graf graf--p">
   发现了什么？
  </p>
  <p class="graf graf--p">
   尽管美国主要的科技公司名列榜首，但受访者也深为关注涉足监控和AI的外国公司，如中国、以及为监视资本主义的数据经纪业务提供动力的国内推手。
  </p>
  <p class="graf graf--p">
   没有人认为 Twitter 是最烂的，但是，很多人对此有点担心 —— 有潜在危害的公司可能会和具有长期存危害的公司一样地令人痛苦。
  </p>
  <p class="graf graf--p">
   关心隐私的人很关心虚假信息问题，关心虚假信息的人却不怎么关心隐私问题 ……
  </p>
  <p class="graf graf--p">

   。这个曾经的在线书商已经发展成为零售、转售、送餐、视频流、云计算、高档产品、娱乐、廉价人工、智能家居技术、监控技术 …… 和间谍机构最大承包商的巨头。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该公司对全球所有人的了解之深度已经超乎想象。换句话说，它可以做到最有效地操纵你。
   </strong>
  </p>
  <p class="graf graf--p">
   此外，该公司的 “last mile” 运输操作导致了倦怠、受伤、甚至死亡，所有这些都与仓库操作有关，该公司像对待机器人那样对待人类工人。
  </p>
  <p class="graf graf--p">
   由于反垄断组织不断对亚马逊在网络零售领域的主导地位提出了更高的批评，该公司反驳称，“与沃尔玛相比，亚马逊在零售总额中所占的份额仍然较小” …… 但是，沃尔玛越来越像亚马逊。
   <strong class="markup--strong markup--p-strong">
    整个经济生态都是如此
   </strong>
   。
  </p>
  <p class="graf graf--p">
   邪恶案例：太多了，数不胜数。
  </p>
  <p class="graf graf--p">
   受访者指出：亚马逊已经导致世界各地的本地商店、服务、新闻、音乐、社区 …… 的死亡；据报道，该公司工作环境恶劣，很多危险的病态的劳工政策引起广泛关注；
   <strong class="markup--strong markup--p-strong">
    通过其Ring门铃帮助构建了广泛的警察监视网络，并使用 Alexa 设备进行更深入的私生活监视
   </strong>
   ；快速运输以及基于AWS的云计算带来了巨大的碳足迹；
   <strong class="markup--strong markup--p-strong">
    向拥有可疑人权记录的军事和情报机构提供技术
   </strong>
   ，包括美国海关和边境保护局的行动，他们在边境拆散家庭；未能审核其平台上的内容，导致大量危险的假货盛行，包括容易损坏的危险假货如儿童汽车安全座椅；有着著名的敌对工作场所文化，事实证明这种文化助长了对妇女和少数民族的骚扰；离岸避税 ……” —— 加州大学伯克利分校 Morgan G. Ames
  </p>
  <p class="graf graf--p">
