  <p class="graf graf--p">
   暗网的定义总是让人们将其与犯罪活动联系在一起。但是，这种刻板印象有时可能被过分简化了。
  </p>
  <p class="graf graf--p">
   尽管有一些客观明确的犯罪参数，但也有一个灰色地带 —— 由出于政治动机的特工组成，
   <strong class="markup--strong markup--p-strong">
    他们可能会或可能不会犯通常定义的罪行，但是，仍会采取有效的行动来影响和推进自己制定的议程
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些团体，包括民族国家行动者 —— 具有网络战任务的国家资助的黑客 —— 值得研究。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     为什么民族国家行为者转向暗网？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   暗网提供了任何人都可以在其中操作的匿名环境。对于民族国家行为者来说，在这种匿名程度的掩盖下可以实现许多关键目标。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    民族国家网络行为者将利用暗网进行情报收集和资源开发、针对政府和企业的间谍活动、漏洞获取和开发、地缘政治影响力的虚假信息战、基础设施破坏和财务收益。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、情报和间谍活动
   </strong>
   —— 基于网络的信息作战的早期工作是由美国政府的国家安全局（NSA）和中国人民解放军（PLA）进行的。
  </p>
  <p class="graf graf--p">
   美国国家安全局使用信息行动来收集外国对手的秘密情报，中国则以其广泛的间谍活动和知识产权盗窃活动而闻名，并取得了巨大成功。
  </p>
  <p class="graf graf--p">
   这包括监视其本国公民，人们使用暗网试图
   <a class="markup--anchor markup--p-anchor" data-href="https://www.technologyreview.com/s/415726/china-cracks-down-on-tor-anonymity-network/" href="https://www.technologyreview.com/s/415726/china-cracks-down-on-tor-anonymity-network/" rel="noopener noreferrer" target="_blank">
    绕开国家控制
   </a>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、基础设施破坏
   </strong>
   —— 由国家资助的针对其他国家的网络战行动已经广泛传播，主要针对包含敏感政府或公司信息以及战略计划的网络。
  </p>
  <p class="graf graf--p">

    展示了
   </a>
   对关键基础设施（例如 电信、公用事业等）和信息网点进行的攻击能力，这会破坏一个国家整体，并在克里米亚持续冲突期间对乌克兰进行黑客攻击。
  </p>

  <p class="graf graf--p">
   随着社会转向持续的数字通信，网络已成为此类信息战活动的首选媒介。
  </p>
  <p class="graf graf--p">
   不论大小，所有民族国家都利用网络活动来做所有事，从宣传议程到在暗网中的活动、以及在整个社交媒体平台上鼓吹代理国。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、漏洞获取和开发
   </strong>
   —— 在暗网论坛和加密聊天中你经常能找到很多讨论黑帽漏洞的东东。
  </p>
  <p class="graf graf--p">
   这些系统漏洞已针对所有类型的关键操作系统和UNIX发行版进行了详细说明和共享。暗网为匿名研究和测试源代码的需求提供了宝贵的资源。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、盈利能力
   </strong>
   —— 面临美国和联合国极端经济制裁的国家正在转向暗网谋求经济利益。
  </p>
  <p class="graf graf--p">
   近年来，朝鲜已经成功地在整个东亚地区发起了全国性的银行系统黑客攻击活动。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     在暗网中的民族国家行为者会怎么做？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在过去的几年中，DarkOwl 研究人员指出，民族国家越来越多地将暗网用作各种关键情报和网络军事战役的基于信息的战场。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在数字信息运营时代，有三个 *主要民族国家行为者*：美国、俄罗斯和中国。
   </strong>
  </p>
  <p class="graf graf--p">
   尽管美国、俄罗斯和中国确实在以网络为中心的金融资源和人力方面仍然明显领先，但由于近年来泄漏的高级漏洞以及进行的逆向工程，已经出现了鲜为人知的民族国家的大量崛起。
  </p>

   </strong>
  </p>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     全球网络战气候的背景
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   现代网络战的谱系远比人们所认为的来自一战期间的信息战和宣传战运动的谱系要古老得多。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e6%88%98%e5%92%8c%e4%bf%a1%e6%81%af%e6%88%98/" rel="noopener noreferrer" target="_blank">
    信息战和宣传战
   </a>
   是自第二次世界大战以来广泛使用的概念，美国人和英国人有效地利用宣传来影响世界各地的态度。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    不要错过我们的专栏 “心理战和信息战”，它收集在
    <a class="markup--anchor markup--li-anchor" data-href="https://start.me/p/nRBzO9/iyp-3" href="https://start.me/p/nRBzO9/iyp-3" rel="noopener noreferrer" target="_blank">
     列表-3 “斗智” 中
    </a>
   </li>
  </ul>
  <p class="graf graf--p">
   从那时起，宣传战就一直秘密地和公开地用于影响地缘政治事件和人口。
  </p>
  <p class="graf graf--p">
   不只俄罗斯的巨魔农场，美国政府也已经根据军事领域的手册和标准操作程序对数字领域的信息操作进行了精心设计和建立。

   研究将这些指纹视为既是与国家行为者使用匿名网络相关的迹象，也是其动机。
  </p>
  <p class="graf graf--p">
   暗网民族国家行为者有一些关键指纹，与他们对暗网的积极使用相关。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    a）民族国家行为者使用暗网购买和窃取网络漏洞
   </strong>
  </p>
  <p class="graf graf--p">
   民族国家行为者从地下市场获取被公开的网络漏洞，以进行逆向工程 —— 经常成功构建间谍软件以应对针对政府或关键基础设施网络的任何攻击。
  </p>
  <p class="graf graf--p">
   冒充漏洞买家的民族国家行为者的关键标志是，他们可获得大量预算和财政资源来购买所提供的商品。
  </p>
  <p class="graf graf--p">
   常规的暗网用户会定期讨论“tells”，以检测网络上的警察和/或政府间谍。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    b）民族国家行为者获得敌对政府和其他具有地缘政治或军事利益的实体的凭证。
   </strong>
  </p>
  <p class="graf graf--p">
   例如，暗网上充斥着 US* .gov 电子邮件地址，可利用这些电子邮件地址进行暴力网络入侵或针对性的网络钓鱼活动。
  </p>
  <p class="graf graf--p">
   目前为止 DarkOwl Vision 已经检测到超过 550000 个暗网网页，其凭据包括 .gov 电子邮件地址。
  </p>
  <p class="graf graf--p">
   伊朗在泄漏凭证和网络信息方面在民族国家行为者中占很大比重，但是无法轻易辨别此信息是否已经被另一民族国家行为者或警惕性黑客团队泄漏。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    c）精心设计的鱼叉式网络钓鱼攻击活动不仅被针对公司网络的犯罪分子所利用，而且民族国家行为者也将其用于政治和军事议程。
   </strong>
  </p>
  <p class="graf graf--p">
   最近的报道表明，朝鲜利用中国的模式通过数字间谍活动来促进技术发展，从而成功地利用网络钓鱼攻击来获取众多学术研究组织和美国关键智囊团的访问权限。
  </p>
  <p class="graf graf--p">
   在 STOLEN PENCIL 行动中，朝鲜针对斯坦福大学的核计划、核扩散和警察小组。攻击基础设施与朝鲜进行的其他活动重叠。此活动中使用的IP地址之一（157.7.184.15）还托管了 bigwnet [.] com 域，该域用作恶意软件 “ BabyShark” 的命令和控制基础结构。
  </p>

  </h3>
  <p class="graf graf--p">
   全球恐怖主义经常在某些国家的财政和政治推动下，在暗网上不断变化，对地缘政治事件和政策以及不断变化的技术具有反应性。
  </p>
  <p class="graf graf--p">
   ISIS，基地组织、黎巴嫩真主党等许多大型极端主义组织已宣布自己为 “民族国家”，并充斥着渴望完成其军事资源的项目‘，例如网络军队和战术黑客团队。
  </p>
  <p class="graf graf--p">
   在西方，关于这种准民族国家在暗网上的真实活动，与公开报道存在很多矛盾。
  </p>
  <p class="graf graf--p">
   几年前，ISIS被评估为广泛使用匿名网络来掩盖其成员和新兵的位置和身份。还用于访问许多隐藏服务，包括与 Daesh 相关的内容，即ISIS的阿拉伯语缩写，包括招募和恐怖主义宣传材料。
  </p>
  <p class="graf graf--p">
   虽然使用Tor之类的匿名网络将来在公开招募恐怖分子和进行宣传方面将会受到限制，但是，恐怖分子非常偏爱加密移动应用程序来进行组织协调和沟通。
  </p>
  <p class="graf graf--p">
   去年，威尔逊中心（Wilson Center）教授 Gabriel Weinmann 发表了一份详尽的报告，详细说明了恐怖分子继续使用暗网以及相关的加密通信协议和技术的原因。
  </p>
  <p class="graf graf--p">

    the report
   </a>
   below]
  </p>
  <p class="graf graf--p">
   1、恐怖分子使用暗网作为掩体 —— 社交媒体公司和安全官员对互联网的广泛监视，导致从社交媒体平台中删除极端主义内容的速度更快。与此相关的是，恐怖分子网络越来越多地使用暗网进行通信、激进化和计划攻击。
  </p>
  <p class="graf graf--p">
   2、恐怖分子使用暗网进行招募：虽然可以在互联网网络平台上进行初步联系，但通常在端对端加密应用程序（如Telegram）上提供有关如何访问暗网上圣战组织附属网站的说明。
  </p>
  <p class="graf graf--p">
   尽管如此，仍然观察到一些恐怖组织，例如Jaish-e-Mohammed在目睹ISIS成功地将 “圣战新娘” 用作伊拉克和叙利亚的战士后，利用暗网积极招募女性战士。
  </p>
  <p class="graf graf--p">
   3、恐怖分子使用暗网作为宣传手段：从表面网上清除极端主义和恐怖主义内容的措施增加了恐怖组织的材料可能丢失的风险。大部分内容随后在暗网上重新出现。
  </p>
  <p class="graf graf--p">
   4、恐怖分子使用加密货币来逃避检测和筹款：恐怖分子与罪犯一样，也使用加密货币，因为它可能提供一定程度上的匿名形式。
  </p>
  <p class="graf graf--p">
   根据一个暗网新闻媒体报道，在2017年底，研究人员目睹了ISIS筹款活动的激增，特别是致力于比特币捐赠的专用站点，证实ISIS已经意识到金融交易监控的风险。目前，没有迹象表明与ISIS有关的恐怖分子有意洗钱。

   在暗网上，只有非常有限的容易被发现是ISIS或将恐怖组织的隐藏服务。
  </p>
  <p class="graf graf--p">
   当ISIS在Tor匿名网络上更加活跃时，例如，“ CyberKahilafah”（伊斯兰国有效的黑客组织）在2016年非常活跃，在暗网发布了ISIS相关内容，例如视频和宣传教育资料。
   <strong class="markup--strong markup--p-strong">
    一些暗网论坛认为这是西方国家政府经营的蜜罐。
   </strong>
  </p>
  <p class="graf graf--p">
   由于国际联盟在 “反恐战争” 中做出了巨大努力，一些恐怖组织的基础设施和组织实力只能通过匿名网络进行广泛协调。
  </p>
  <p class="graf graf--p">
   2016年，国际黑客组织 Anonymous 在暗网上对可疑的ISIS成员进行了攻击，发布了其成员的联系信息（电子邮件地址、社交媒体帐户）及其支持者特别是 Nasher 伊斯兰国（@nashirislamicstateEN）的公开网站。
  </p>
  <p class="graf graf--p">
   直到2019年，针对ISIS的匿名攻击仍在继续，更多 Daesh / ISIS 成员的社交媒体和个人信息在多个深度Web粘贴服务之间被曝光。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     总结
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   民族国家背景的网络威胁参与者都非常聪明，他们利用开源和暗网资产的混合来完成其关键的信息运营任务。
  </p>
  <p class="graf graf--p">
   网络战斗人员、政府资助的代理人和雇佣军团队，利用暗网进行情报收集和资源开发、政府和企业间谍活动、网络漏洞利用开发和测试、针对地缘政治影响的虚假信息操作、基础设施破坏和财务收益等。
  </p>
  <p class="graf graf--p">
   尽管在某些暗网使用案例中可以辨认出独特的国家 “指纹”，但以前仅属于美国国安局和中央情报局的网络武器的公开发布使以前实力较弱的国家也有能力重新塑造自己的力量，获得他们以前无法达到的影响力和技术能力。
  </p>
  <p class="graf graf--p">
   全球恐怖主义经常由特定民族国家在财务和政治上支持，在暗网上具有不可预测的且往往是反应性的痕迹 —— 对地缘政治事件和政策以及不断变化的技术具有反应性。
  </p>
  <p class="graf graf--p">
   恐怖分子的适应能力使他们从暗网转移到了端到端的加密专有协议（如 Whatsapp 和 Telegram），他们可以在其中匿名招募、策划和传播宣传。
  </p>
  <p class="graf graf--p">
   随着民族国家背景的网络攻击者、网络代理和恐怖组织在使用暗网和匿名化技术方面能力的不断发展，网络安全社区必须保持警惕，以继续就其不断变化的策略、技术和手段进行识别和跟踪。
  </p>

  <p class="graf graf--p">
   根据 Threat Connect 发布的
   <a class="markup--anchor markup--p-anchor" data-href="http://cdn2.hubspot.net/hubfs/454298/Project_CAMERASHY_ThreatConnect_Copyright_2015.pdf?submissionGuid=e173e32e-af62-4148-b9a6-30748da43c59&amp;t=1443030820943" href="https://cdn2.hubspot.net/hubfs/454298/Project_CAMERASHY_ThreatConnect_Copyright_2015.pdf?submissionGuid=e173e32e-af62-4148-b9a6-30748da43c59&amp;t=1443030820943" rel="noopener noreferrer" target="_blank">
    深入情报
   </a>
   ，精心策划的间谍活动涉及一个复杂的资源域网络，其中包括位于科罗拉多州丹佛市的IP地址。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、俄罗斯 🇷🇺
   </strong>
  </p>
  <p class="graf graf--p">
   近年来，从众多媒体和FBI的行动中可以明显看出，俄罗斯政府和情报部门已经深入渗透了暗网，开展了针对世界各地目标的众多大规模国家攻击活动。
  </p>
  <p class="graf graf--p">
   攻击经常包括美国及其西方盟友，这可能被视为一场全面的网络战争，展示了各种各样的先进技术网络能力。
  </p>
  <p class="graf graf--p">
   美国国防部网络战略部的研究人员正在努力确定可用于俄罗斯网络战役的网络专家的确切人数，但是有
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zdnet.com/article/counting-up-the-worlds-cyber-armies/" href="https://www.zdnet.com/article/counting-up-the-worlds-cyber-armies/" rel="noopener noreferrer" target="_blank">
    报道称
   </a>
   ，有许多精英部门组织了黑客行动包括26165及其姊妹单位74455。
  </p>
  <p class="graf graf--p">
   俄罗斯擅长使用代理，它雇用了高级的非政府附属网络犯罪组织来代表他们进行APT攻击。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、以色列 🇮🇱
   </strong>
  </p>
  <p class="graf graf--p">
   以色列是一个高度机密和有影响力的民族国家行为者。
  </p>
  <p class="graf graf--p">
   以色列的精英网络间谍组织8200部队可以与NSA媲美，但其工作重点更加集中且计算得当。
  </p>
  <p class="graf graf--p">
   8200部队
   <a class="markup--anchor markup--p-anchor" data-href="https://www.jpost.com/Israel-News/IDF-decides-not-to-have-a-cyber-command-department-477169" href="https://www.jpost.com/Israel-News/IDF-decides-not-to-have-a-cyber-command-department-477169" rel="noopener noreferrer" target="_blank">
    补足了
   </a>
   以色列国防军（IDF）的其他许多高科技部队。
  </p>
  <p class="graf graf--p">
   由于部分攻击活动需要更多的法律自由和更少的国际审查，以色列网络公司也雇用了前8200部队人员，以实施以色列支持的在暗网中的秘密活动。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、德国 🇩🇪
   </strong>
  </p>
  <p class="graf graf--p">
   德国、英国和法国都具有复杂的网络能力。
  </p>
  <p class="graf graf--p">
   德国最近建立了自己的网络和信息空间司令部（CIR），分配了13000 多名人员来抵御网络入侵攻击和虚假信息宣传活动。
  </p>
  <p class="graf graf--p">
   德国执法部门还破坏了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bundesrat.de/SharedDocs/TO/975/erl/10.pdf?__blob=publicationFile&amp;v=1%2F%20https%3A%2F%2Fnews.ycombinator.com%2Fitem%3Fid%3D19399576" href="https://www.bundesrat.de/SharedDocs/TO/975/erl/10.pdf?__blob=publicationFile&amp;v=1%2F%20https%3A%2F%2Fnews.ycombinator.com%2Fitem%3Fid%3D19399576" rel="noopener noreferrer" target="_blank">
    多起
   </a>
   暗网中违法活动。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、英国 🇬🇧
   </strong>
  </p>
  <p class="graf graf--p">
   最近有报道称，来自英国的黑客入侵了俄罗斯的Turla组织，突显了英国强大的能力。
  </p>
  <p class="graf graf--p">
   自2014年以来，英国顶级间谍机构 GHCQ 的能力翻了一番，提供了从战术到高端反国家进攻网络作战的全光谱功能。
  </p>
  <p class="graf graf--p">
   由于英国的NHS在2017年成为 WannaCry 的主要受害者，因此英国不仅可以防御未来的袭击，而且可以在需要时进行反击。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    7、乌克兰 🇺🇦
   </strong>
  </p>
  <p class="graf graf--p">
   乌克兰最初未被认为是值得重视的国家黑客组织。
  </p>
  <p class="graf graf--p">
   过去，乌克兰的网络能力主要围绕有组织犯罪和暗网梳理为主。鉴于最近有关乌克兰政府和财团商人被媒体广泛报道以及他们在美国大选政治中的影响，乌克兰在国际舞台上的 “影响力” 是显而易见的。
  </p>
  <p class="graf graf--p">
   这种 “影响” 再加上乌克兰因吞并克里米亚而与俄罗斯进行的持续战争，包括防御俄罗斯对乌克兰电力基础设施的网络攻击，使乌克兰在网络领域名列前十位。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    8、法国 🇫🇷
   </strong>
  </p>
  <p class="graf graf--p">
   法国于2019年初发布了其新的军事网络战略，
   <a class="markup--anchor markup--p-anchor" data-href="https://warontherocks.com/2019/04/a-close-look-at-frances-new-military-cyber-strategy/" href="https://warontherocks.com/2019/04/a-close-look-at-frances-new-military-cyber-strategy/" rel="noopener noreferrer" target="_blank">
    该战略包括
   </a>
   两个单独的文件：防御性网络战的部长级政策和军事网络战学说的公开元素 。法国在欧盟和北约组织中具有重要影响力，弥补了该国缺乏人力资本的弱点。
  </p>
  <p class="graf graf--p">
   9、伊朗 🇮🇷
  </p>
  <p class="graf graf--p">
   伊朗是主要的民族国家网络战参与者，在中东国家（以色列除外）中处于领先地位。
  </p>
  <p class="graf graf--p">
   十多年来，针对各种西方国防和商业网络的伊朗网络军一直是一个巨大的威胁。在美国通过 Stuxnet （震网）病毒成功渗透并关闭了其核离心机系统之后，伊朗投入了巨资开发技能和资源，以保持自己在国际网络舞台上的地位。
  </p>
  <p class="graf graf--p">
   它们还以 “代理” 配置大量运行，与其他较小的国家合作共享技术和资源。
  </p>
  <p class="graf graf--p">
   据估计，伊朗的任何民族国家级网络攻击都可以在朝鲜、叙利亚和也门等国家的帮助下进行。
  </p>
  <p class="graf graf--p">
   众所周知，伊朗还与真主党和私人黑客组织勾结。通过培训私人黑客和恐怖分子，伊朗可能成为策划下一次全球性网络战的关键。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    10、朝鲜🇰🇵
   </strong>
  </p>
  <p class="graf graf--p">
   朝鲜声称对多起针对国际基础设施的大规模攻击负责，这是对国际经济制裁的回应，因为他们拒绝停止核计划。
  </p>
  <p class="graf graf--p">
   根据开源情报
   <a class="markup--anchor markup--p-anchor" data-href="https://thenextweb.com/security/2019/09/24/north-korean-hackers-are-targeting-atms-in-india-with-new-data-stealing-malware/" href="https://thenextweb.com/security/2019/09/24/north-korean-hackers-are-targeting-atms-in-india-with-new-data-stealing-malware/" rel="noopener noreferrer" target="_blank">
    报告
   </a>
   ，朝鲜黑客已成功部署了一种新的名为 ATMTrack 的 ATM 恶意软件，该恶意软件可轻易窃取印度易受攻击的 ATM。
  </p>
  <p class="graf graf--p">
   据估计，ATMDTrack 是 DTrack 恶意软件家族的组成部分，该家族不仅涉及命令和控制远程访问木马（RAT）软件，还涉及键盘记录、检索浏览器历史记录、收集主机IP地址、有关可用网络和活动连接的信息、列举所有正在运行的进程，并列出受害计算机的所有可用磁盘卷上的所有文件。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    11、印度 🇮🇳
   </strong>
  </p>
  <p class="graf graf--p">
   2018年，印度建立了国家技术研究组织， 作为保护国家关键基础设施并处理该国关键部门的所有网络安全事件的主要机构。
  </p>
  <p class="graf graf--p">
   除了来自巴基斯坦的网络攻击之外，印度还面临来自其他主要恶意民族国家行为者的攻击，正如前面提到的朝鲜对印度银行基础设施的攻击。
  </p>
  <p class="graf graf--p">
   12、加拿大 🇨🇦
  </p>
  <p class="graf graf--p">
   2018年，加拿大通过了全面的立法机关，以授权加拿大的通信安全机构（CSE）进行有效的进攻性网络运营。
  </p>
  <p class="graf graf--p">
   全面的 C-59 法案使 CSE（加拿大国家安全局）采取了更为 “主动的网络” 姿态，而不是之前的基本防御和被动反应立场。
  </p>
  <p class="graf graf--p">
   立法
   <a class="markup--anchor markup--p-anchor" data-href="https://www.lawfareblog.com/zero-deh-canada-takes-bold-step-towards-offensive-cyber-operations" href="https://www.lawfareblog.com/zero-deh-canada-takes-bold-step-towards-offensive-cyber-operations" rel="noopener noreferrer" target="_blank">
    要求
   </a>
   CSE “ 在全球信息基础架构之上或通过全球信息基础架构进行活动，以降低、破坏、影响、响应、干扰针对个人、国家、国际事务、国防或安全的组织或恐怖组织的能力、意图或活动。”
  </p>
  <p class="graf graf--p">
   加拿大不会在网络世界舞台上独居，而是拥有资源和议会支持来影响、保护和捍卫加拿大基础设施免受敌对国家的袭击。⚪️
  </p>
  <p class="graf graf--p">

  </p>
