  <h3 class="graf graf--h3 graf-after--li" id="fdc6">
   数字信息战的兴起：我们是怎么到了这种地步的？
  </h3>
  <p class="graf graf--p graf-after--h3" id="954d">
   信息战是心理战的一部分（虽然不是全部），如果您还记得我们在《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E7%8B%AC%E5%AE%B6-%E5%8C%97%E4%BA%AC%E7%9A%84%E6%94%BF%E6%B2%BB%E5%AE%A3%E4%BC%A0%E6%A8%A1%E5%BC%8F%E6%AD%A3%E5%9C%A8%E6%94%B9%E5%8F%98-%E6%88%96%E5%B7%B2%E8%B8%A9%E4%B8%AD%E4%B8%AD%E5%9B%BD%E5%BC%82%E8%AE%AE%E7%9A%84%E5%BC%B1%E7%82%B9-d1bf5ddfd412" href="https://www.iyouport.org/%e7%8b%ac%e5%ae%b6%ef%bc%9a%e5%8c%97%e4%ba%ac%e7%9a%84%e6%94%bf%e6%b2%bb%e5%ae%a3%e4%bc%a0%e6%a8%a1%e5%bc%8f%e5%8f%91%e7%94%9f%e6%94%b9%e5%8f%98/" rel="noopener noreferrer" target="_blank">
    心理战
   </a>
   》那篇文章中对其曾经用法的简要介绍，就能明白，这一战略的历史之深远、以及它是如何在技术的协助下得到了脱缰之马般的肆虐。
  </p>
  <p class="graf graf--p graf-after--p" id="bb84">
   这绝不是什么新奇的思路。它是自古以来各种战争中最为关键的部分：
  </p>
  <blockquote class="graf graf--blockquote graf-after--p" id="29c4">
   <p>
    <em class="markup--em markup--blockquote-em">

    </em>
   </p>
  </blockquote>
  <p class="graf graf--p graf-after--blockquote" id="d56c">

  </p>
  <p class="graf graf--p graf-after--p" id="0a96">
   俄罗斯
   <strong class="markup--strong markup--p-strong">
    绝不是
   </strong>
   唯一一个擅长歪曲互联网舆论和选民认知的国家。牛津大学的研究早已发现，社交媒体状态下，有组织的操纵行为
   <strong class="markup--strong markup--p-strong">
    自 2010 年以来至少在 28 个国家存在！
   </strong>

   </strong>

    <strong class="markup--strong markup--p-strong">

    什么是事实什么是虚构
   </a>
   。然而截止今天，在线传播虚假信息和欺骗手段背后所依赖的技术仍处于起步阶段，信息认证的重要性及其必要的技能，也只是刚刚开始形成。

  <h3 class="graf graf--h3 graf-after--p" id="65a8">
   数字信息战未来的关键要素
  </h3>
  <p class="graf graf--p graf-after--h3" id="187f">
   未来数字信息大战中的一个共同主题将是恐惧、不确定性和令人疑惑的故意传播（fear, uncertainty, doubt）也称为 FUD 在线。 显然，喜
   <strong class="markup--strong markup--p-strong">
    欢刺激性信息、漠视调查取证及其相关技术、封闭、虚无主义和偏执的用户群体，将成为宣传的主攻目标 — — 这些特征恰恰是社交网络用户群的最普遍特征
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="a028">
   在支持性技术的推动下，三项关键战术将在战争的未来发挥至关重要的作用，就如密歇根大学社交媒体责任中心首席技术官 Aviv Ovadya 所描述的那样：
  </p>
  <ul class="postList">
   <li class="graf graf--li graf-after--p" id="05fc">

     外交和声誉操纵
    </strong>

   </li>
   <li class="graf graf--li graf-after--li" id="52d2">
    <strong class="markup--strong markup--li-strong">
     间接操纵的网络钓鱼
    </strong>
    ：使用恶意 AI来模仿值得信赖的实体，针对的是特定的高价值目标，迫使目标以其他方式采取行动，即 社交工程技术；
   </li>
   <li class="graf graf--li graf-after--li" id="38fe">
    <strong class="markup--strong markup--li-strong">
     计算化宣传
    </strong>
    ：利用社交媒体、人类心理学、扭曲性的消息、八卦和不透明的算法来操纵公众舆论。
   </li>
  </ul>

   </strong>

  <p class="graf graf--p graf-after--p" id="fbb4">
   外交操纵是为了在地缘政治决策上影响对手，令其产生错误信念的行为。
  </p>
  <p class="graf graf--p graf-after--p" id="0be3">
   华盛顿大学（UW）的研究人员已经成功地使用人工智能创建了奥巴马的伪造视频，视频中说出了“他从未说过的话”。
  </p>
  <p class="graf graf--p graf-after--p" id="4c97">
   根据该大学
   <a class="markup--anchor markup--p-anchor" data-href="http://grail.cs.washington.edu/projects/AudioToObama/" href="https://grail.cs.washington.edu/projects/AudioToObama/" rel="noopener nofollow noreferrer" target="_blank">

  </p>
  <p class="graf graf--p graf-after--p" id="5b1c">
   伪装奥巴马成功意味着任何人都能被如此伪装。很容易想象这样一个被改变过的视频 — 如果足够好看，看起来很真实 — 很快就会造成严重的破坏性。
   <strong class="markup--strong markup--p-strong">
    如果当权者伪造反对派和活动家的视频……不论是抹黑还是定罪，都将非常容易实现。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="bf18">
   人工智能的进步正在开创一个假视频和音频的新时代，这将对国内政治和外交的未来产生深远的影响。虽然该技术在大学实验中仍处于发展阶段，但不久的将来它将变得非常容易操作。
  </p>
  <p class="graf graf--p graf-after--p" id="3e61">
   <strong class="markup--strong markup--p-strong">
    AI 创建假冒的数字内容
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="ec41">
   GAN（生成性对抗网络）是一种用于执行无监督机器学习的 AI 。在 GAN 中，相反的神经网络协同工作以制造越来越逼真的音频、图像和视频内容。 GAN 中的一个神经网络充当箔片，推动其他网络产生更高保真度的结果。神经网络还可以更容易伪造音频，可以将源音频的元素转换为统计属性，并且可以重新排列这些属性以制作高超的假音频剪辑。

   显然，人工智能对视频和音频干预的广泛采用意味着对“高价值目标”的声誉攻击也将相应增加。
   <strong class="markup--strong markup--p-strong">

   </strong>

  <p class="graf graf--p graf-after--p" id="4b17">
   已经有越来越多的人已经开始拍色情视频，被称为 deepfakes （“深度学习”和“假”的混成词），使用叠加名人的面孔到成人电影明星面部的软件。当一个名为“deepfakesapp”的匿名用户于2017年12月发布该技术的第一个版本时，那是 deepfake 一词首次出现在 Reddit 上。
  </p>
  <p class="graf graf--p graf-after--p" id="3377">
   另一位用户后来很快发布了一个名为 FakeApp 的改进版本。FakeApp 使用由 Google 开发的名为 TensorFlow 的深度学习程序，允许用户创建面部交换的逼真视频。
  </p>
  <p class="graf graf--p graf-after--p" id="8b3c">
   该应用程序及其基础技术正在获得强大的牵引力。
  </p>
  <p class="graf graf--p graf-after--p" id="c939">
   Deepfake Society 是一个策划使用 FakeApp 制作的深度造假视频的网站，自2月推出以来已有超过 100 万的观看次数。Deepfake Society 禁止色情内容。但
   <strong class="markup--strong markup--p-strong">
    类似的网站

     特征包括：对政党的忠诚、对枪支控制或移民等热门问题的立场
    </strong>
    ，即使一个人神经质，对他人持怀疑态度，或者相信占星术的迹象等等。如下图：
   </p>
   <p>
    <img alt="" class="aligncenter size-full wp-image-3680 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-309.png?resize=1024%2C924&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-309.png?w=1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-309.png?resize=300%2C271&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-309.png?resize=768%2C693&amp;ssl=1 768w" data-recalc-dims="1" height="924" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-309.png?resize=1024%2C924&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>
    <noscript>
     <img alt="" class="aligncenter size-full wp-image-3680" data-recalc-dims="1" height="924" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-309.png?resize=1024%2C924&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-309.png?w=1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-309.png?resize=300%2C271&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-309.png?resize=768%2C693&amp;ssl=1 768w" width="1024"/>
    </noscript>
   </p>
   <p class="graf graf--p graf-after--figure" id="6865">
    <strong class="markup--strong markup--p-strong">
     针对数字欺骗的新兴解决方案
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="089f">
    没时间等待解决方案了。包括埃及、巴西和墨西哥在内的国家都在 2018 年举行大选，美国 2018 年的中期选举即将举行。全球民主正在越来越多地被社交网络宣传和先进的数字欺骗所操纵。这是进行时，而且进行的速度非常快！
   </p>
   <p class="graf graf--p graf-after--p" id="4b8e">
    我们必须开发新的技术和应用来对抗信息战。首先，我们需要一种可扩展的方式来发现高精度的假视频。
   </p>
   <p class="graf graf--p graf-after--p" id="1cde">
    而制止社交网络宣传是一个更复杂的问题。这不仅取决于技术掌握在谁手里，更有公民是否能对真相保持足够的热情，对民主保持足够的信念。
   </p>
   <p class="graf graf--p graf-after--p" id="8ca3">
    <strong class="markup--strong markup--p-strong">
     发现隐藏的元数据以验证图像和视频
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="aa27">
    国际特赦组织是世界上最大的基层人力资源组织，在争夺用户提交的侵犯人权视频证据的斗争中处于第一线。 该组织的公民证据实验室专门揭示图像和视频的背景。该实验室正在构建专业知识和技术，以验证视频捕获的时间、地点和方式。
   </p>
   <p class="graf graf--p graf-after--p" id="0513">
    使用的就是我们在
    <a class="markup--anchor markup--p-anchor" data-href="https://www.wix.com/dashboard/6fc0666c-c125-4701-93b5-303f8db9aa33/blog/5ba062b92b3b020057dcb7f0/edit/search/posts?query=%23OSINT" href="https://www.iyouport.org/category/osint/" rel="nofollow noopener noreferrer" target="_blank">
     #OSINT
    </a>
    开源调查演示中多次介绍过的方法。
   </p>
   <p class="graf graf--p graf-after--p" id="2e0b">
    例如，该实验室使用谷歌地球和搜索引擎 Wolfram Alpha 交叉抓取视频中的环境和天气条件，以查看视频是否在其声称的条件下被捕获。
   </p>
   <figure aria-describedby="caption-attachment-3681" class="wp-caption aligncenter" id="attachment_3681" style="width: 1024px">
    <img alt="" class="size-full wp-image-3681 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-310.png?resize=1024%2C473&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-310.png?w=1024&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-310.png?resize=300%2C139&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-310.png?resize=768%2C355&amp;ssl=1 768w" data-recalc-dims="1" height="473" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-310.png?resize=1024%2C473&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>
    <noscript>
     <img alt="" class="size-full wp-image-3681" data-recalc-dims="1" height="473" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-310.png?resize=1024%2C473&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-310.png?w=1024&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-310.png?resize=300%2C139&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-310.png?resize=768%2C355&amp;ssl=1 768w" width="1024"/>

     Citizen Evidence Lab triangulates details in user-submitted video of a shooting in Papua New Guinea to authenticate the video’s origins. Source: Amnesty International
    </figcaption>
   </figure>
   <div class="section-inner sectionLayout--insetColumn">
    <p class="graf graf--p graf-after--figure" id="10d1">
     Citizen Evidence 实验室还有一个名为 YouTube 数据查看器的工具，可以从 YouTube 上托管的视频中
     <strong class="markup--strong markup--p-strong">
      提取隐藏的元数据
     </strong>
     。以判断用户传递的是否是旧视频或伪造视频。
    </p>
    <p class="graf graf--p graf-after--p" id="4ef8">
     <strong class="markup--strong markup--p-strong">
      用于将数字内容追溯到源的区块链
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="a4cc">
     支持区块链背后的加密技术还可以帮助确保数字内容来自可靠、负责任的来源。
    </p>
    <p class="graf graf--p graf-after--p" id="c5be">
     从本质上讲，媒体可以加盖一个唯一的加密标识符，当与区块链上的记录交叉引用时 — 可以毫无疑问地证明媒体的来源。没有标识符的媒体将被视为不太可靠。
    </p>
    <p class="graf graf--p graf-after--p" id="d15e">
     这种技术对于发现在上下文中试图欺骗的图像和视频也特别有用。然而，数字取证组和媒体密码学家仍然需要努力解决人工智能从头就开始制作的虚假视频的问题。似乎这种问题越来越多出现了，尤其是在大型的有计划的操纵运动中。
    </p>
    <p class="graf graf--p graf-after--p" id="d932">
     <strong class="markup--strong markup--p-strong">
      发现 AI 生成的人物
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="4cfa">
     麻省理工学院的研究人员展示了 Eulerian 视频放大技术，该技术可以帮助在视频中发现人工智能制作的人物影像。
    </p>
    <p class="graf graf--p graf-after--p" id="0421">
     这种视频放大技术可以通过观察由于血流引起的皮肤颜色的细微变化来检测诸如人的心率之类的微小细节，从而区分真人与 AI 生成的人：人工智能尚不足以在假视频中创建如皮下血液流动这种真实的水平。
    </p>
    <figure aria-describedby="caption-attachment-3682" class="wp-caption aligncenter" id="attachment_3682" style="width: 1024px">
     <img alt="" class="size-full wp-image-3682 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-311.png?resize=1024%2C419&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-311.png?w=1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-311.png?resize=300%2C123&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-311.png?resize=768%2C314&amp;ssl=1 768w" data-recalc-dims="1" height="419" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-311.png?resize=1024%2C419&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>

     </noscript>
     <figcaption class="wp-caption-text" id="caption-attachment-3682">
      Source: MIT
     </figcaption>
    </figure>
    <p class="graf graf--p graf-after--figure" id="d98b">
     交叉引用视频元数据、记录区块链上的合法内容、以及使用先进的视频放大技术是协同的步骤。然而，这些工具和技术的可扩展性不足以消除开源 AI 启用数字欺骗的迫在眉睫的威胁。我们需要一个可扩展的解决方案，因为如今几乎任何人都可以制作出高品质的假货。
    </p>
    <p class="graf graf--p graf-after--p" id="b950">
     <strong class="markup--strong markup--p-strong">
      大规模检测图像和视频操作
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="1a7f">
     国防高级研究计划局（DARPA）至少发起了两项研究呼吁，以建立可扩展的数字媒体认证系统。
    </p>
    <p class="graf graf--p graf-after--p" id="def5">
     该媒体取证（MediFor）项目是建立一个平台，使用算法检测图像和视频操作。MediFor 有朝一日可以创建一个众包平台，观众可以集体调查视频的真实性。
    </p>
    <p class="graf graf--p graf-after--p" id="93d3">
     DARPA 的 MEMEX 项目可以帮助构建一个庞大的在线搜索引擎，能够交叉引用整个互联网（包括暗网）的图像数据。哥伦比亚大学的
     <a class="markup--anchor markup--p-anchor" data-href="http://www.ee.columbia.edu/ln/dvmm/memex/index.html" href="http://www.ee.columbia.edu/ln/dvmm/memex/index.html" rel="noopener nofollow noreferrer" target="_blank">
      一个由 MEMEX 资助的项目
     </a>
     证明了能够在数 TB 的结构化和非结构化数据中找到人类贩运受害者的类似图像。这项工作可以帮助发现源自其他来源的 AI 生成的图像和视频的各个方面。
    </p>
    <p class="graf graf--p graf-after--p" id="a843">
     <strong class="markup--strong markup--p-strong">
      打击社交媒体宣传
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="f632">
     公民可以保护自己免受在社交媒体上的算法操纵。关键是及时发现数字宣传以采取行动。
    </p>
    <p class="graf graf--p graf-after--p" id="e112">
     目前，对抗社交媒体宣传的可扩展技术手段在很大程度上是理论性的，并且至多在发展的早期阶段。一些组织已经发表了关于如何打击社交媒体宣传的想法，例如利用破坏坏僵尸网络的好机器人，或者识别伪装成人类的坏机器人。
    </p>
    <p class="graf graf--p graf-after--p" id="4bee">
     同样，解决社交网络宣传的一次性项目正在世界范围内发展。
    </p>
    <p class="graf graf--p graf-after--p" id="e875">
     印第安纳大学是发布 OSoMe 项目测试版的机构之一：社交媒体观察站。IU 的 OSoMe 包括“ 可视化声称和事实检查的技术 ”以及“在新闻源中检测和阻止 Twitter 机器人”的工具。
    </p>
    <p class="graf graf--p graf-after--p" id="0568">
     乌克兰的 Kyiv-Mohyla 新闻学校以及 KMA 数字化新闻未来推出了
     <a class="markup--anchor markup--p-anchor" data-href="http://stopfake.org/" href="http://stopfake.org/" rel="noreferrer noopener nofollow" target="_blank">
      Stopfake.org
     </a>
     事实核查网站。该网站是一个信息中心，用户可以在其中查看和分析克里姆林宫生产的在线宣传的各个方面。
    </p>
    <p class="graf graf--p graf-after--p" id="0e03">
     <strong class="markup--strong markup--p-strong">
      最后的想法
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="53f2">
     打击信息战的未来依然不确定，但还是有希望的。DARPA，企业，创业公司，非营利组织和大学的强大队伍都在长期打击信息战的斗争中取得了进展。
     <strong class="markup--strong markup--p-strong">
      最为关键的是，调查研究和反制技术开发均应保持独立性，远离政府和商业巨头的利益操控，让真相真正成为真相
     </strong>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="4450">
     必须强调的是，数字保护的责任最终将取决于用户。在可预见的未来，我们将继续负责地评估我们所消费的信息的真实性。
    </p>
    <p class="graf graf--p graf-after--p" id="d075">
     这意味着要意识到我们所服务的叙事，并让自己习惯于去识别证实来自个人偏见的叙事与那些挑战我们信仰的叙事。并且，希望能动员更多技术爱好者加入到保卫民主的战役中来。⚪️
    </p>
