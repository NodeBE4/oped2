  <p class="graf graf--p">
   痛苦是个人的；欢笑才是大众的 —— 如果你想要人们加入你、与你共鸣，那就要以欢笑为武器。
  </p>
  <p class="graf graf--p">
   对一些人来说，隐秘反叛小丑军（CIRCA）可能看起来只是一群衣衫褴褛的活动家，他们戴着假鼻子、满身都是油漆涂料、迷彩裤和乱蓬蓬的假发。
  </p>
  <p class="graf graf--p">
   表面上看的确是。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    他们实际上是一支训练有素的行动者团队，一支真正的民兵队伍，一个专业的小丑营
   </strong>
   。
  </p>
  <p class="graf graf--p">
   艺术活动家 John Jordan 及其同事 L.M. Bogad、Jen Verson 和 Matt Trevelyan 于2003年底成立了 CIRCA，第一个目标是：捣乱乔治·W·布什访问伦敦。
  </p>
  <p class="graf graf--p">
   CIRCA 旨在成为一种
   <strong class="markup--strong markup--p-strong">
    新的公民不服从方法
   </strong>
   ，将古老的小丑艺术与非暴力直接行动的当代策略相结合。
  </p>
  <p class="graf graf--p">
   它很快成为了一种成功的 meme 和
   <strong class="markup--strong markup--p-strong">
    国际抗议现象
   </strong>
   ，自组织的团体在从哥伦比亚到新西兰等
   <strong class="markup--strong markup--p-strong">
    几十个国家
   </strong>
   的首脑会议和军事基地之外的街道上采取行动。
  </p>
  <p class="graf graf--p">
   CIRCA 与专业小丑们一起制定了一种方法论，将玩乐和游戏引入政治组织过程。
  </p>
  <p class="graf graf--p">
   倡导者开展了一系列培训，鼓励活动家重新考虑如何
   <strong class="markup--strong markup--p-strong">
    利用自己的身体，发展自己的直觉，并“找到自己的小丑”
   </strong>
   —— 一种孩子般的慷慨和自发的状态。
  </p>
  <p class="graf graf--p">
   反叛小丑训练试图卸下活动家的盔甲，并找到其中隐藏的那个脆弱的个人。
  </p>
  <p class="graf graf--p">
   这场运动强调了被太多其他运动所忽视的个人转型的内在工作；CIRCA 将灵魂和街道视为斗争的场所。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    寻找绝对自发的自我，小丑的深层工作可以对参与者产生深刻的心理影响。
   </strong>
  </p>
  <p class="graf graf--p">
   CIRCA 的战斗人员并不打算假装成小丑，他们应该是真正的小丑。💡小丑是一种存在，而非技术性状态。
  </p>
  <p class="graf graf--p">
   这就是 CIRCA 的核心前提：
   <strong class="markup--strong markup--p-strong">
    嘲弄和完全令敌人陷入困惑，可能比直接对抗更强大。
   </strong>
  </p>
  <p class="graf graf--p">
   在其中一个案例中，一群七十人的小丑直接走到一排全副武装的英国防暴警察队伍面前，奇怪的是，这些警察完全无法控制住小丑队伍。
  </p>
  <p class="graf graf--p">
   当回顾事件的监视视频时人们才发现，在坚硬的盾牌后面，警察们大多在止不住地笑，而无法集中注意力。
  </p>
  <p class="graf graf--p">
   其他小丑在口袋里装满了各种奇怪的垃圾，看起来鼓鼓囊囊的，就像是隐藏着武器，当小丑们被阻拦并被强行搜查时，警察需要数小时的时间才能搞清楚这堆垃圾里都有些什么。
  </p>
  <p class="graf graf--p">
   一场最有趣的战役是小丑军直接走进了官方军队招募机构，并以一种*小丑的方式*尝试应招，从而造成了非常多的混乱，以至于该机构不得不在当天关闭；继而，CIRCA 在外面建立起了自己的破旧滑稽的招募摊位。
  </p>
  <p class="graf graf--p">
   著名无政府主义活动家艾玛·戈德曼（Emma Goldman）提出过这样一个问题：
   <strong class="markup--strong markup--p-strong">
    “如何成就一个人的自我并与他人合一，深深地感受所有人，并保持自己的特质。”
   </strong>
  </p>
  <p class="graf graf--p">
   CIRCA 做到了这一点，令参与者发现他们自己内心的小丑，同时穿着统一的“制服”，使他们深刻感觉到自己是一个强烈联系的群体的一部分。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    为什么会有效
   </strong>
  </p>
  <p class="graf graf--p">
   反叛小丑是帮助许多人参与激进政治的门户，而这些政治一直因其严肃性而阻止了太多人的参与。
  </p>
  <p class="graf graf--p">
   对于许多新兵来说，这是他们第一次体验公民不服从，但是滑稽的和面具般的化妆使他们能够放飞自我，往往是来自意想不到的荒谬和创造性的方式。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
