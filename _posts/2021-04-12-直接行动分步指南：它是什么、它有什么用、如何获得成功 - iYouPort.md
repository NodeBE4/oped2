  <h3 class="graf graf--p">

    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   头脑风暴可以
   <strong class="markup--strong markup--p-strong">
    从您想要解决的问题开始
   </strong>
   ，或者从您想要做出的社会贡献开始；它可以来自您所拥有的资源、您所渴望的经验、或您想要合作的人提供的信息。
  </p>
  <p class="graf graf--p">
   您可以策划一次短期的冒险，也可以策划一次长期的活动。
  </p>
  <p class="graf graf--p">
   通常情况下，最好的头脑风暴发生在非正式谈话的过程中 — — 相信您最疯狂的想法可以变成现实，并希望尝试一下。
  </p>
  <p class="graf graf--p">
   同样的道理，即使是参加别人组织的活动，最好也要自己带一个计划，这样您才能以自己的方式做出贡献。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">

   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a1%ae%e5%ae%9a%e5%8f%98%e5%8c%96%e7%9a%84%e6%9d%a0%e6%9d%86%ef%bc%9a%e4%bf%a1%e6%81%af%e8%a1%8c%e5%8a%a8%e4%b8%bb%e4%b9%89%e6%8c%87%e5%8d%97%ef%bc%882%ef%bc%89-%e6%88%98%e7%95%a5%e6%95%88/" href="https://www.iyouport.org/%e7%a1%ae%e5%ae%9a%e5%8f%98%e5%8c%96%e7%9a%84%e6%9d%a0%e6%9d%86%ef%bc%9a%e4%bf%a1%e6%81%af%e8%a1%8c%e5%8a%a8%e4%b8%bb%e4%b9%89%e6%8c%87%e5%8d%97%ef%bc%882%ef%bc%89-%e6%88%98%e7%95%a5%e6%95%88/" rel="noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     战略和战术
    </strong>
   </a>
   。邀请朋友，或散发传单，或挨家挨户地宣布。
  </p>
  <p class="graf graf--p">
   提前想好自己的方案，以防别人没有方案。
  </p>
  <p class="graf graf--p">
   如果想采取更秘密的行动，可以和一两个值得信赖的朋友在安全的环境中进行头脑风暴。在你们商量好的时候，把你们的想法留给自己，这样当你们准备尝试的时候，它依旧是最新的。
  </p>

   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     2、确立目标  — — 确立行动目标并确定其优先次序
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    您的行动是为了谁？
   </strong>
   是针对现场受众、企业媒体观众、特定企业的老板、股东、警察和政府、其他社会成员、还是参与者本身？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    它要达到什么目的？
   </strong>
   是为了沟通思想、呼吁人们关注不公、激励人们、确保资源、设定特定的基调、造成显著的物质损失、提供一种威慑、展示一种他人可以应用的反抗模式、还是作为参与者的学习和联系经验？
  </p>
  <p class="graf graf--p">
   📌 从一开始就建立对行动目标的共同理解，将省去以后当您的计划改变和潜在冲突出现时的很多麻烦。
  </p>
  <figure class="graf graf--figure">
   <img class="graf-image aligncenter jetpack-lazy-image" data-height="523" data-image-id="0*uACZiKOoW9a_nCrK.png" data-lazy-src="https://i0.wp.com/cdn-images-1.medium.com/max/1067/0*uACZiKOoW9a_nCrK.png?w=1100&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" data-width="624" src="https://i0.wp.com/cdn-images-1.medium.com/max/1067/0*uACZiKOoW9a_nCrK.png?w=1100&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7"/>

  <h2 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     结构
    </strong>
   </span>
  </h2>
  <h3 class="graf graf--p">
   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     1、亲密团队 — — 与您熟悉和信任的人紧密合作
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   直接行动组织的最有效和最安全的模式之一就是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%82%a8%e9%9c%80%e8%a6%81%e7%9c%9f%e6%ad%a3%e7%9a%84-%e8%87%aa%e5%b7%b1%e4%ba%ba%ef%bc%9a%e6%95%b4%e4%bd%93%e5%ae%89%e5%85%a8%ef%bc%889%ef%bc%89-%e8%bf%9b%e5%85%a5%e6%88%98%e6%96%97%e7%8a%b6/" href="https://www.iyouport.org/%e6%82%a8%e9%9c%80%e8%a6%81%e7%9c%9f%e6%ad%a3%e7%9a%84-%e8%87%aa%e5%b7%b1%e4%ba%ba%ef%bc%9a%e6%95%b4%e4%bd%93%e5%ae%89%e5%85%a8%ef%bc%889%ef%bc%89-%e8%bf%9b%e5%85%a5%e6%88%98%e6%96%97%e7%8a%b6/" rel="noopener" target="_blank">
    亲密团队模式
   </a>
   （点击这个链接看到我们在系列教程对对它的讲述）。亲密团队是一群彼此间能够深深信任、有着相同目标的朋友，长期合作，彼此了解大家的特长和特点，整体会变得高效、行动更有效。
  </p>
  <p class="graf graf--p">
   📌 对于一个小的行动，亲密团队的成员可以承担不同的角色。对于一个较大的行动，您的亲密团队可以与其他亲密团队以 “集群” 的形式进行合作，每个小组扮演一个角色。这可以使决策比在一个大的群体中更容易，因为每个群体都可以派一名代表参加发言人会议。
  </p>
  <p class="graf graf--p">
   亲密团队的集群可以长期合作，建立信任和效力。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     2、招募 — — 慎重引进其他个人和团体
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   一旦您有了一个计划建议，就想好您
   <strong class="markup--strong markup--p-strong">
    需要多少人
   </strong>
   去完成它。
  </p>
  <p class="graf graf--p">
   📌 如果您的计划需要保密，就只邀请您信任的人加入，而且您确信他们会愿意加入 — — 您邀请的每一个人如果最终没有参加，那就是一个不必要的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%af%8f%e4%b8%80%e7%a7%8d%e5%8f%98%e5%8c%96%e9%83%bd%e6%9c%89%e5%8e%9f%e5%9b%a0%ef%bc%8c%e6%82%a8%e9%9c%80%e8%a6%81%e5%8f%8a%e6%97%b6%e6%8a%93%e5%8f%96%e5%a8%81%e8%83%81%e4%bf%a1%e5%8f%b7/" href="https://www.iyouport.org/%e6%af%8f%e4%b8%80%e7%a7%8d%e5%8f%98%e5%8c%96%e9%83%bd%e6%9c%89%e5%8e%9f%e5%9b%a0%ef%bc%8c%e6%82%a8%e9%9c%80%e8%a6%81%e5%8f%8a%e6%97%b6%e6%8a%93%e5%8f%96%e5%a8%81%e8%83%81%e4%bf%a1%e5%8f%b7/" rel="noopener" target="_blank">
    安全风险
   </a>
   。
  </p>
  <p class="graf graf--p">
   逐个发出邀请，
   <strong class="markup--strong markup--p-strong">
    不要
   </strong>
   群发邮件！这样那些决定不参加的人就不会知道其他参与者 的任何情况。
  </p>
  <p class="graf graf--p">
   先问一些关于潜在参与者可能感兴趣的一般性问题，在他或她准备好做出承诺
   <strong class="markup--strong markup--p-strong">
    之前，不要透露
   </strong>
   计划的任何关键细节，如确切的目标或日期等。
  </p>
  <p class="graf graf--p">
   当人们被带入计划并继续引入其他人时，要确保每个人
   <strong class="markup--strong markup--p-strong">
    对适当的安全程度有相同的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%8d%89%e6%9c%a8%e7%9a%86%e5%85%b5%e5%92%8c%e7%88%b1%e8%b0%81%e8%b0%81%e9%83%bd%e5%8f%aa%e4%bc%9a%e5%b8%a6%e6%9d%a5%e6%9b%b4%e5%a4%9a%e7%9a%84%e5%8d%b1%e9%99%a9%ef%bc%9a%e6%95%b4%e4%bd%93/" href="https://www.iyouport.org/%e8%8d%89%e6%9c%a8%e7%9a%86%e5%85%b5%e5%92%8c%e7%88%b1%e8%b0%81%e8%b0%81%e9%83%bd%e5%8f%aa%e4%bc%9a%e5%b8%a6%e6%9d%a5%e6%9b%b4%e5%a4%9a%e7%9a%84%e5%8d%b1%e9%99%a9%ef%bc%9a%e6%95%b4%e4%bd%93/" rel="noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     理解
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   随着越来越多的人参与到行动主义项目中来，重要的是每个人
   <strong class="markup--strong markup--p-strong">
    都要明白
   </strong>
   对他们的承诺有多大的期望。
  </p>
  <p class="graf graf--p">
   有时，最先提出计划的小组会比其他人更投入；如果他们做了几个月的准备工作，却在最后一刻让另一个他们所依赖的小组退出，那么所有的工作都白费了。
  </p>
  <p class="graf graf--p">
   每个人都有共同的责任，从一开始就坦诚地告诉大家，对他们的期望是现实的。同时，那些发起项目的人应该
   <strong class="markup--strong markup--p-strong">
    非常小心地与其他所有参与者分享所有权
   </strong>
   。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%bb%93%e8%af%86%e6%96%b0%e8%ae%a4%e8%af%86%e7%9a%84%e4%ba%ba%e6%97%b6%e6%82%a8%e5%ba%94%e8%af%a5%e6%9b%9d%e5%85%89%e8%87%aa%e5%b7%b1%e5%a4%9a%e5%b0%91%e4%bf%a1%e6%81%af%ef%bc%9f-%e5%8e%9f%e5%88%99/" href="https://www.iyouport.org/%e7%bb%93%e8%af%86%e6%96%b0%e8%ae%a4%e8%af%86%e7%9a%84%e4%ba%ba%e6%97%b6%e6%82%a8%e5%ba%94%e8%af%a5%e6%9b%9d%e5%85%89%e8%87%aa%e5%b7%b1%e5%a4%9a%e5%b0%91%e4%bf%a1%e6%81%af%ef%bc%9f-%e5%8e%9f%e5%88%99/" rel="noopener" target="_blank">
    信任分级
   </a>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     3、权力动态 — — 确保权力在您的团队中均匀分配
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   以参与性和协商一致的方式作出所有决定。如果您的小组规模足够大，可以使用非正式或正式的协商会议程序，以确保所有的声音都能被听到：共同制定每次会议的议程，并挑选一名主持人，以保证事情正常进行。📌
   <strong class="markup--strong markup--p-strong">
    每个人的参与度越高，您所做的决定就越有依据。
   </strong>
  </p>
  <p class="graf graf--p">
   📌 要注意可能不平衡的内部动态，例如 不同背景的人之间的动态，或本地组织者和外地参与者之间的动态。
  </p>
  <p class="graf graf--p">
   每个人越是深入地参与计划和准备行动，人们就会对行动的成功越是投入。
   <strong class="markup--strong markup--p-strong">
    一个具有良好内部动力的小组比任何个人都要聪明；个人可以带来想法，但小组可以一起找出应用这些想法的最佳方式。
   </strong>
  </p>
  <p class="graf graf--p">
   确保每个人在整个项目中都能感受到支持和舒适；在正式的结构之外和结构内部都要互相检查。虽然经常被忽视，但保持士气是成功的直接行动组织的一个重要方面。在面对意外和不确定性时，要保持
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%85%85%e5%88%86%e7%9a%84%e5%87%86%e5%a4%87%e5%92%8c%e5%ba%94%e6%80%a5%e5%8f%8d%e5%ba%94%e5%90%8c%e6%a0%b7%e9%87%8d%e8%a6%81%ef%bc%9a%e6%95%b4%e4%bd%93%e5%ae%89%e5%85%a8%ef%bc%887%ef%bc%89/" href="https://www.iyouport.org/%e5%85%85%e5%88%86%e7%9a%84%e5%87%86%e5%a4%87%e5%92%8c%e5%ba%94%e6%80%a5%e5%8f%8d%e5%ba%94%e5%90%8c%e6%a0%b7%e9%87%8d%e8%a6%81%ef%bc%9a%e6%95%b4%e4%bd%93%e5%ae%89%e5%85%a8%ef%bc%887%ef%bc%89/" rel="noopener" target="_blank">
    冷静的头脑
   </a>
   。
  </p>

  <h2>
   <span style="color: #339966;">
    <strong>
     基本准备
    </strong>
   </span>
  </h2>
  <h3 class="graf graf--p">
   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     1、安全文化 —— 以 “需要知道” 的方式传播信息
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   📌 安全文化是一种避免不健康的偏执狂的方法，在任何时候都要把风险降到最低。如果您和您的朋友们总是明智地进行自己的行为，您就可以减少对
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e6%b8%97%e9%80%8f%e6%8a%97%e8%ae%ae%e8%bf%90%e5%8a%a8%e7%ae%80%e5%8f%b2/" href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e6%b8%97%e9%80%8f%e6%8a%97%e8%ae%ae%e8%bf%90%e5%8a%a8%e7%ae%80%e5%8f%b2/" rel="noopener" target="_blank">
    渗透
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e8%af%86%e5%88%ab%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8%e4%b8%ad%e5%8f%af%e8%a7%81%e7%9a%84%e5%92%8c%e4%b8%8d%e5%8f%af%e8%a7%81%e7%9a%84%e7%9b%91%e8%a7%86/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e8%af%86%e5%88%ab%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8%e4%b8%ad%e5%8f%af%e8%a7%81%e7%9a%84%e5%92%8c%e4%b8%8d%e5%8f%af%e8%a7%81%e7%9a%84%e7%9b%91%e8%a7%86/" rel="noopener" target="_blank">
    监视
   </a>
   的恐惧。
  </p>
  <p class="graf graf--p">
   安全文化的精髓在于，
   <strong class="markup--strong markup--p-strong">
    信息是在 *需要知道* 的基础上共享的
   </strong>
   。在某些情况下，您的行动需要全城的人都知道才算成功；而在另一些情况下，关键是永远不要在直接参与者的圈子之外谈论这个行动。
  </p>
  <p class="graf graf--p">
   📌 每个参与行动的人都需要分享一种意识，即：认为什么程度的安全是合适的，并尊重他人的安全需求。
  </p>
  <p class="graf graf--p">
   在安全问题上，“同意” 和性亲密关系一样重要；
   <strong class="markup--strong markup--p-strong">
    在安全问题上违反他人的意愿是绝对不能接受的
   </strong>
   。从一开始就必须明确自己的安全需求；必要时一起宣誓保守秘密。
  </p>
  <p class="graf graf--p">
   📌 除非得到他们的明确许可，否则永远不要谈论您或他人参与过的行动，无论过去的时间有多长。
  </p>
  <p class="graf graf--p">
   当一个小组聚集在一起进行一个项目时，要确保在场的每个人都得到小组中其他人的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ae%89%e5%85%a8%e5%9c%b0%e4%bd%bf%e7%94%a8-signal%ef%bc%9a%e5%87%86%e5%88%99/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ae%89%e5%85%a8%e5%9c%b0%e4%bd%bf%e7%94%a8-signal%ef%bc%9a%e5%87%86%e5%88%99/" rel="noopener" target="_blank">
    担保 — — 担保他们是可靠的和值得信赖的
   </a>
   。为了保护所有人，您应该准备好在面对
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%9d%a5%e8%87%aa%e4%ba%ba%e8%ba%ab%e7%9a%84%e5%a8%81%e8%83%81%ef%bc%9a%e5%a6%82%e4%bd%95%e5%9c%a8%e6%95%8c%e5%af%b9%e7%8e%af%e5%a2%83%e4%b8%ad%e4%bf%9d%e6%8a%a4%e8%87%aa%e5%b7%b1%ef%bc%881%ef%bc%89/" href="https://www.iyouport.org/%e6%9d%a5%e8%87%aa%e4%ba%ba%e8%ba%ab%e7%9a%84%e5%a8%81%e8%83%81%ef%bc%9a%e5%a6%82%e4%bd%95%e5%9c%a8%e6%95%8c%e5%af%b9%e7%8e%af%e5%a2%83%e4%b8%ad%e4%bf%9d%e6%8a%a4%e8%87%aa%e5%b7%b1%ef%bc%881%ef%bc%89/" rel="noopener" target="_blank">
    审讯
   </a>
   和法律压力下保持沉默。
  </p>
  <p class="graf graf--p">
   📌 从一个项目开始，您就应该按照它可能要求的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/opsec-%e6%93%8d%e4%bd%9c%e5%ae%89%e5%85%a8%ef%bc%9a%e5%9c%a8%e5%8f%8d%e4%be%a6%e5%af%9f%e7%9a%84%e8%bf%87%e7%a8%8b%e4%b8%ad%e6%82%a8%e5%ba%94%e8%af%a5%e6%b3%a8%e6%84%8f%e5%93%aa%e4%ba%9b%e9%99%b7/" href="https://www.iyouport.org/opsec-%e6%93%8d%e4%bd%9c%e5%ae%89%e5%85%a8%ef%bc%9a%e5%9c%a8%e5%8f%8d%e4%be%a6%e5%af%9f%e7%9a%84%e8%bf%87%e7%a8%8b%e4%b8%ad%e6%82%a8%e5%ba%94%e8%af%a5%e6%b3%a8%e6%84%8f%e5%93%aa%e4%ba%9b%e9%99%b7/" rel="noopener" target="_blank">
    最高安全级别来操作
   </a>
   ；您以后总是可以变得不那么谨慎，但如果您一开始就粗心大意，就等于对自己关闭了很多选择。
  </p>
  <p class="graf graf--p">
   要知道您的所有行为都可能被监控或追踪：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%8c%87%e5%8d%97%ef%bc%9a%e5%bd%93%e6%82%a8%e5%87%86%e5%a4%87%e5%8f%82%e5%8a%a0%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8/" href="https://www.iyouport.org/%e6%8c%87%e5%8d%97%ef%bc%9a%e5%bd%93%e6%82%a8%e5%87%86%e5%a4%87%e5%8f%82%e5%8a%a0%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8/" rel="noopener" target="_blank">
    监控摄像头的记录
   </a>
   、您的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b4%ad%e7%89%a9%e4%b8%ad%e7%9a%84%e9%9a%90%e7%a7%81%e6%b3%84%e6%bc%8f%ef%bc%9a%e4%b8%8d%e8%ae%ba%e6%98%af%e5%9c%a8%e7%ba%bf%e5%89%81%e6%89%8b%e8%bf%98%e6%98%af%e5%ae%9e%e4%bd%93%e5%ba%97%ef%bc%8c/" href="https://www.iyouport.org/%e8%b4%ad%e7%89%a9%e4%b8%ad%e7%9a%84%e9%9a%90%e7%a7%81%e6%b3%84%e6%bc%8f%ef%bc%9a%e4%b8%8d%e8%ae%ba%e6%98%af%e5%9c%a8%e7%ba%bf%e5%89%81%e6%89%8b%e8%bf%98%e6%98%af%e5%ae%9e%e4%bd%93%e5%ba%97%ef%bc%8c/" rel="noopener" target="_blank">
    购物记录
   </a>
   、您
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/1200%e4%b8%87%e9%83%a8%e7%94%b5%e8%af%9d%ef%bc%8c%e4%b8%80%e4%b8%aa%e6%95%b0%e6%8d%ae%e9%9b%86%ef%bc%9a%e5%88%a9%e7%bb%b4%e5%9d%a6%e7%9c%bc%e4%b8%ad%e7%9a%84%e4%b8%96%e7%95%8c/" href="https://www.iyouport.org/1200%e4%b8%87%e9%83%a8%e7%94%b5%e8%af%9d%ef%bc%8c%e4%b8%80%e4%b8%aa%e6%95%b0%e6%8d%ae%e9%9b%86%ef%bc%9a%e5%88%a9%e7%bb%b4%e5%9d%a6%e7%9c%bc%e4%b8%ad%e7%9a%84%e4%b8%96%e7%95%8c/" rel="noopener" target="_blank">
    去的地方和与您见面的人
   </a>
   、会议的地点、您扔进
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e5%b7%b2%e6%88%90%e4%b8%ba%e7%9b%ae%e6%a0%87%ef%bc%9a%e6%9d%a5%e8%87%aa%e5%86%85%e9%83%a8%e5%92%8c%e5%a4%96%e9%83%a8%e7%9a%84%e5%a8%81%e8%83%81%e5%92%8c%e6%8a%b5%e5%be%a1/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e5%b7%b2%e6%88%90%e4%b8%ba%e7%9b%ae%e6%a0%87%ef%bc%9a%e6%9d%a5%e8%87%aa%e5%86%85%e9%83%a8%e5%92%8c%e5%a4%96%e9%83%a8%e7%9a%84%e5%a8%81%e8%83%81%e5%92%8c%e6%8a%b5%e5%be%a1/" rel="noopener" target="_blank">
    垃圾桶的物品
   </a>
   、您
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%8d%95%e5%90%91%e9%95%9c%e7%9a%84%e8%83%8c%e5%90%8e%ef%bc%9a%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e5%ae%b6%e5%92%8c%e6%94%bf%e5%ba%9c%e7%9a%84%e8%81%94%e6%89%8b%e4%b8%80%e7%9b%b4%e5%9c%a8%e5%a6%82/" href="https://www.iyouport.org/%e5%8d%95%e5%90%91%e9%95%9c%e7%9a%84%e8%83%8c%e5%90%8e%ef%bc%9a%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e5%ae%b6%e5%92%8c%e6%94%bf%e5%ba%9c%e7%9a%84%e8%81%94%e6%89%8b%e4%b8%80%e7%9b%b4%e5%9c%a8%e5%a6%82/" rel="noopener" target="_blank">
    访问的网站
   </a>
   、您电脑上的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%8es%e5%85%88%e7%94%9f%e5%85%b1%e5%ba%a6%e7%be%8e%e5%a5%bd%e4%b8%80%e5%a4%a9%e2%80%8a-%e2%80%8a%e5%8f%8d%e5%8f%96%e8%af%81%e6%8a%80%e5%b7%a7%ef%bc%9a%e5%8f%98%e5%be%97%e9%9a%be%e4%bb%a5%e8%a2%ab/" href="https://www.iyouport.org/%e4%b8%8es%e5%85%88%e7%94%9f%e5%85%b1%e5%ba%a6%e7%be%8e%e5%a5%bd%e4%b8%80%e5%a4%a9%e2%80%8a-%e2%80%8a%e5%8f%8d%e5%8f%96%e8%af%81%e6%8a%80%e5%b7%a7%ef%bc%9a%e5%8f%98%e5%be%97%e9%9a%be%e4%bb%a5%e8%a2%ab/" rel="noopener" target="_blank">
    文件
   </a>
   、您留下的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%8c%87%e7%ba%b9%e8%a7%a3%e9%94%81%e4%b8%8d%e5%ae%89%e5%85%a8/" href="https://www.iyouport.org/%e6%8c%87%e7%ba%b9%e8%a7%a3%e9%94%81%e4%b8%8d%e5%ae%89%e5%85%a8/" rel="noopener" target="_blank">
    指纹
   </a>
   （例如，在手电筒内的电池上），以及几乎所有
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%89%8b%e6%9c%ba%e4%b8%ad%e9%9a%90%e8%97%8f%e7%9a%84%e8%b7%9f%e8%b8%aa%e5%99%a8%ef%bc%9a%e4%bb%96%e4%bb%ac%e5%a6%82%e4%bd%95%e4%bd%bf%e7%94%a8%e6%82%a8%e6%89%8b%e6%9c%ba%e7%9a%84%e5%ba%94%e7%94%a8/" href="https://www.iyouport.org/%e6%89%8b%e6%9c%ba%e4%b8%ad%e9%9a%90%e8%97%8f%e7%9a%84%e8%b7%9f%e8%b8%aa%e5%99%a8%ef%bc%9a%e4%bb%96%e4%bb%ac%e5%a6%82%e4%bd%95%e4%bd%bf%e7%94%a8%e6%82%a8%e6%89%8b%e6%9c%ba%e7%9a%84%e5%ba%94%e7%94%a8/" rel="noopener" target="_blank">
    与手机有关的东西
   </a>
   。
   <strong class="markup--strong markup--p-strong">
    必要时设计强密码，准备好不在场证明。
   </strong>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     2、法律支持  — — 准备在行动期间和之后提供支持的基础设施
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   参与行动的每个人都应该意识到并准备好他们正在承担的风险以及与之相关的潜在刑事指控。
  </p>
  <p class="graf graf--p">
   📌 重要的是，不要把事情做得比您觉得准备好的更远：如果您在参与某种程度的风险时受伤或被捕，您对这种风险还没有做好情绪上的准备，其影响可能是衰弱的。
  </p>
  <p class="graf graf--p">
   比起匆忙地参与一项行动，有一个糟糕的经历，然后发誓放弃所有此类活动，您最好是能慢慢地开始，稳步地建立一个可持续参与的直接行动项目，最好是希望它可以持续一辈子。
  </p>
  <p class="graf graf--p">
   📌 如果您的行动可能会导致逮捕，为那些参与行动的人
   <strong class="markup--strong markup--p-strong">
    准备一个法律支持结构是必要的
   </strong>
   。这可以包括为被捕者提供一个法律援助号码，法律观察员监督和记录警察的行动，保释金，为被捕者提供即时支持并在法庭上代表他们的律师，以及一个准备在整个法庭案件中提供情感、财务和后勤支持的合作者圈子。
  </p>
  <p class="graf graf--p">
   📌 法律援助号码应该在整个行动中随时开放，接受来电；请记住，在某些情况下，您不能从监狱里打电话。
   <strong class="markup--strong markup--p-strong">
    法律援助号码不应指控被捕者或接电话的人 — — 如果您的不在场证明的一部分是您不认识对方（您的队友），你们就不能都从监狱拨打同一个号码。
   </strong>
  </p>
  <p class="graf graf--p">
   如果您担心自己会忘记这个号码，可以用永久记号笔
   <strong class="markup--strong markup--p-strong">
    写在身上的隐蔽部位
   </strong>
   。操作法律援助号码的人应该知道可能被逮捕的人的真实姓名，以便查询他们的情况。
  </p>
  <p class="graf graf--p">
   如果您有被逮捕的危险，请决定您是要带着身份证明，以加快处理速度，还是不带身份证明，这样他们就不能立即认出您。
   <strong class="markup--strong markup--p-strong">
    如果能有一大群拒绝提供信息的被捕者就会束缚法律程序，有时还会获得讨价还价的权力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果您需要吃药，可以考虑把药藏在身上，或者随身携带医生的说明，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%8a%97%e8%ae%ae%e4%b8%ad%e8%b0%a8%e9%98%b2%e5%b8%b8%e8%a7%81%e5%8d%b1%e9%99%a9%ef%bc%9a%e6%95%b4%e4%bd%93%e5%ae%89%e5%85%a8%ef%bc%8811%ef%bc%89-%e5%8d%8f%e4%bd%9c%e5%92%8c%e4%ba%92%e5%8a%a9/" href="https://www.iyouport.org/%e6%8a%97%e8%ae%ae%e4%b8%ad%e8%b0%a8%e9%98%b2%e5%b8%b8%e8%a7%81%e5%8d%b1%e9%99%a9%ef%bc%9a%e6%95%b4%e4%bd%93%e5%ae%89%e5%85%a8%ef%bc%8811%ef%bc%89-%e5%8d%8f%e4%bd%9c%e5%92%8c%e4%ba%92%e5%8a%a9/" rel="noopener" target="_blank">
    解释您需要什么
   </a>
   。
  </p>
  <p class="graf graf--p">
   找一个有同情心且值得信赖的律师 — — 或者
   <strong class="markup--strong markup--p-strong">
    可能找多个律师，因为一个律师不能代表同一个罪名的多个被告。
   </strong>
  </p>
  <p class="graf graf--p">
   📌 您可以研究一下哪些律师在过去处理过类似的案件。如果您不透露任何敏感的信息，可以向有同情心的律师询问与假设行为相关的指控，或者
   <strong class="markup--strong markup--p-strong">
    指定您可能需要他们服务的日期和时间 — — 但不要让他们知道任何可能牵扯到他们的事。
   </strong>
  </p>
  <p class="graf graf--p">
   📌 为了做好他们的工作，律师需要能够证明他们与任何非法行为无关。您应该记住这点。
  </p>
  <p class="graf graf--p">
   任何一个成员可能遭受逮捕的社区，
   <strong class="markup--strong markup--p-strong">
    最好提前建立一个保释基金/律师费基金；
   </strong>
   这可以可以省去很多在紧急情况下的奔波。举办义演，卖T恤，向有钱的同情者募捐，让大学里的朋友帮您预定在他们学校的演讲日期，以换取学生基金 ……
   <strong class="markup--strong markup--p-strong">
    确保资金留在一个公平的、值得信赖的、总是容易联系的人手中。
   </strong>
  </p>
  <p class="graf graf--p">
   📌 同样，考虑您的媒体策略是什么 — — 将公众的注意力和支持引向被捕者是否明智。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     3、媒体 —— 确立您想要的影响力覆盖面并获得它
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   早在行动之前，当您确立目标并确定优先次序时，就应准确地计算出您希望得到多少媒体报道，从哪些来源得到，以及您将如何获得或避免媒体报道。
  </p>
  <p class="graf graf--p">
   📌 这可能意味着撰写并发送一份
   <strong class="markup--strong markup--p-strong">
    新闻稿
   </strong>
   （谁、什么、何时、何地、如何、为什么，都要写清楚）、或
   <strong class="markup--strong markup--p-strong">
    公报
   </strong>
   ，在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%82%a8%e9%9c%80%e8%a6%81%e7%9c%9f%e6%ad%a3%e7%9a%84-%e8%87%aa%e5%b7%b1%e4%ba%ba%ef%bc%9a%e6%95%b4%e4%bd%93%e5%ae%89%e5%85%a8%ef%bc%889%ef%bc%89-%e8%bf%9b%e5%85%a5%e6%88%98%e6%96%97%e7%8a%b6/" href="https://www.iyouport.org/%e6%82%a8%e9%9c%80%e8%a6%81%e7%9c%9f%e6%ad%a3%e7%9a%84-%e8%87%aa%e5%b7%b1%e4%ba%ba%ef%bc%9a%e6%95%b4%e4%bd%93%e5%ae%89%e5%85%a8%ef%bc%889%ef%bc%89-%e8%bf%9b%e5%85%a5%e6%88%98%e6%96%97%e7%8a%b6/" rel="noopener" target="_blank">
    亲密团队
   </a>
   中推选一位发言人向媒体发言以代表您的行动主义项目，邀请独立记者参加行动或新闻发布会，传真公告或拨打新闻电话，提供采访（亲自或匿名通过
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b9%b0%e4%b8%8d%e8%b5%b7-%e4%b9%b0%e4%b8%8d%e5%88%b0%e4%b8%80%e6%ac%a1%e6%80%a7%e6%89%8b%e6%9c%ba%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9f%e8%bf%99%e9%87%8c%e6%98%af%e6%82%a8%e5%9c%a8%e5%8f%82%e5%8a%a0/" href="https://www.iyouport.org/%e4%b9%b0%e4%b8%8d%e8%b5%b7-%e4%b9%b0%e4%b8%8d%e5%88%b0%e4%b8%80%e6%ac%a1%e6%80%a7%e6%89%8b%e6%9c%ba%e6%80%8e%e4%b9%88%e5%8a%9e%ef%bc%9f%e8%bf%99%e9%87%8c%e6%98%af%e6%82%a8%e5%9c%a8%e5%8f%82%e5%8a%a0/" rel="noopener" target="_blank">
    一次性电话
   </a>
   ），或让您的团体成员自己报道。
  </p>
  <p class="graf graf--p">
   📌 如果您想避免某些类型的报道，这也可能意味着指派一名参与者，以确保摄影师不会将相机对准你。
  </p>
  <p class="graf graf--p">
   如果您正在与媒体沟通，编写谈话要点、您的发言人重复的声音片段等，以确保他们在媒体报道中得到呈现。
  </p>
  <p class="graf graf--p">
   给媒体代表尽可能少的材料，让他们不得不使用您希望他们使用的部分。追踪哪些记者倾向于提供积极的报道，并亲自接近他们，记得
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd%ef%bc%9a%e4%bd%a0%e5%ba%94%e8%af%a5%e7%9f%a5%e9%81%93%e7%9a%84%e6%8a%80%e6%9c%af%e3%80%81%e7%bb%8f%e9%aa%8c%e5%92%8c%e6%95%99%e8%ae%ad/" href="https://www.iyouport.org/%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd%ef%bc%9a%e4%bd%a0%e5%ba%94%e8%af%a5%e7%9f%a5%e9%81%93%e7%9a%84%e6%8a%80%e6%9c%af%e3%80%81%e7%bb%8f%e9%aa%8c%e5%92%8c%e6%95%99%e8%ae%ad/" rel="noopener" target="_blank">
    斯诺登等吹哨人当年怎么做的
   </a>
   吗？
  </p>
  <p class="graf graf--p">
   📌 如果您有一个网站，把这个网址弄到企业媒体的报道中，利用那媒体的影响力让他们的观众改道到您的媒体上。你们也可以自己通过张贴海报、海盗电台、演讲活动或挨家挨户开展对话等方式，向公众提供信息。
  </p>
  <p class="graf graf--p">
   📌 如果您的行动需要高度的安全性，请安全地发送您的公报：例如，从一台公共计算机上发送，不留下谁使用它的记录。要知道您使用的设备是如何让您入罪的。（在
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/1kod2L/iyp-direct-action5" href="https://start.me/p/1kod2L/iyp-direct-action5" rel="noopener" target="_blank">
    列表-5 “护盾”
   </a>
   栏目中看到专门提供给行动者的安全知识）
  </p>

  <h2 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     基础工作
    </strong>
   </span>
  </h2>
  <h3 class="graf graf--p">
   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     1、规划  — — 研究背景，制定战略，为不同的情况制定计划
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   适当的规划是安全、有效的直接行动的本质。📌
   <strong class="markup--strong markup--p-strong">
    牢记您的目标和优先事项、以及您所拥有的资源，规划和比较不同的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a1%ae%e5%ae%9a%e5%8f%98%e5%8c%96%e7%9a%84%e6%9d%a0%e6%9d%86%ef%bc%9a%e4%bf%a1%e6%81%af%e8%a1%8c%e5%8a%a8%e4%b8%bb%e4%b9%89%e6%8c%87%e5%8d%97%ef%bc%882%ef%bc%89-%e6%88%98%e7%95%a5%e6%95%88/" href="https://www.iyouport.org/%e7%a1%ae%e5%ae%9a%e5%8f%98%e5%8c%96%e7%9a%84%e6%9d%a0%e6%9d%86%ef%bc%9a%e4%bf%a1%e6%81%af%e8%a1%8c%e5%8a%a8%e4%b8%bb%e4%b9%89%e6%8c%87%e5%8d%97%ef%bc%882%ef%bc%89-%e6%88%98%e7%95%a5%e6%95%88/" rel="noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     战略
    </strong>
   </a>
   。权衡每一种战术的风险和潜在的回报：总是选择最安全的方式来完成给定的目标，并确保您能承受得起您选择的风险。
  </p>
  <p class="graf graf--p">
   📌 有时会发生这样的情况：随着规划过程的进行，一个项目会变得越来越雄心勃勃，越来越冒险，直到一些参与者开始产生怀疑；这时，可能需要制定一个更安全的或缩小的计划版本，这样它仍然可以进行。
  </p>
  <p class="graf graf--p">
   📌 在规划中，有无数的因素需要考虑。您必须根据当前的社会和政治形势选择最有效的战术。您必须选择最佳的行动地点，并考虑到它的所有属性；您必须选择最佳的日期和时间。您必须考虑到其他将在该地区的人，以及他们将如何反应 — — 他们是同情您，还是可能存在敌对以干扰您的活动？您必须协调行动中不同部分的时间，预测每个部分需要多长时间，并弄清楚参与行动的人将如何沟通。
  </p>
  <p class="graf graf--p">
   📌 在预测其他人 — — 比如说，警察 — —
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b4%bb%e5%8a%a8%e5%ae%b6%e5%a6%82%e4%bd%95%e7%9e%84%e5%87%866%e5%a4%a7%e6%9d%83%e5%8a%9b%e6%94%af%e6%9f%b1%e8%8e%b7%e5%8f%96%e6%9c%80%e5%bc%ba%e5%a4%a7%e7%9a%84%e6%8a%b5%e6%8a%97%e6%95%88%e6%9e%9c/" href="https://www.iyouport.org/%e6%b4%bb%e5%8a%a8%e5%ae%b6%e5%a6%82%e4%bd%95%e7%9e%84%e5%87%866%e5%a4%a7%e6%9d%83%e5%8a%9b%e6%94%af%e6%9f%b1%e8%8e%b7%e5%8f%96%e6%9c%80%e5%bc%ba%e5%a4%a7%e7%9a%84%e6%8a%b5%e6%8a%97%e6%95%88%e6%9e%9c/" rel="noopener" target="_blank">
    的反应
   </a>
   时，要考虑影响他们的因素。如果您有出其不意的优势，那么这种优势会持续多久？会不会有很多人关注这个事件？会不会马上知道您在做什么？周围会不会有中产阶级的市民或记者，他们的存在会不会对当局的反应造成影响？根据以往警
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%95%87%e5%8e%8b%e9%83%a8%e9%98%9f%e6%9c%80%e5%b8%b8%e7%94%a8%e5%93%aa%e4%ba%9b%e6%ad%a6%e5%99%a8%e5%af%b9%e4%bb%98%e6%8a%97%e8%ae%ae%e8%80%85%ef%bc%9f-%e6%a6%82%e8%a7%88/" href="https://www.iyouport.org/%e9%95%87%e5%8e%8b%e9%83%a8%e9%98%9f%e6%9c%80%e5%b8%b8%e7%94%a8%e5%93%aa%e4%ba%9b%e6%ad%a6%e5%99%a8%e5%af%b9%e4%bb%98%e6%8a%97%e8%ae%ae%e8%80%85%ef%bc%9f-%e6%a6%82%e8%a7%88/" rel="noopener" target="_blank">
    察在这种情况下的行为
   </a>
   先例，他们的战术可能是什么？他们的上司是否希望他们对您下狠手，或者避免挑起事端？他们的沟通能力如何，他们的行动速度如何，他们的位置在哪里，他们会走什么路线？
  </p>
  <p class="graf graf--p">
   📌 不要低估简单的后勤事务的挑战，比如在紧张的情况下运送人员或通讯。也别忘了规划一个撤离战略。
  </p>
  <p class="graf graf--p">
   因为计划很少会完全按照所制定的计划来进行，所以
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%85%85%e5%88%86%e7%9a%84%e5%87%86%e5%a4%87%e5%92%8c%e5%ba%94%e6%80%a5%e5%8f%8d%e5%ba%94%e5%90%8c%e6%a0%b7%e9%87%8d%e8%a6%81%ef%bc%9a%e6%95%b4%e4%bd%93%e5%ae%89%e5%85%a8%ef%bc%887%ef%bc%89/" href="https://www.iyouport.org/%e5%85%85%e5%88%86%e7%9a%84%e5%87%86%e5%a4%87%e5%92%8c%e5%ba%94%e6%80%a5%e5%8f%8d%e5%ba%94%e5%90%8c%e6%a0%b7%e9%87%8d%e8%a6%81%ef%bc%9a%e6%95%b4%e4%bd%93%e5%ae%89%e5%85%a8%ef%bc%887%ef%bc%89/" rel="noopener" target="_blank">
    制定出针对不同情况的备用计划
   </a>
   很重要。”如果____，我们就____；如果____，我们就____。”
   <strong class="markup--strong markup--p-strong">
    心中要有几个不同的目标，以防您的第一选择变成不可能
   </strong>
   。有一个基本的沟通和决策结构，将帮助您做好准备，以应对与您想象中的任何情况都不同的情况。
  </p>
  <p class="graf graf--p">
   小心不要让其他人因您的行动而处于危险之中；
   <strong class="markup--strong markup--p-strong">
    当局很可能会指控他们所掌握的任何人犯下最严重的罪行，所以既要让那些冒险的人安全地离开该地区，又要确保不能对其他人提出严重的指控。
   </strong>
  </p>
  <p class="graf graf--p">
   📌 在某些情况下，您可以把多层次的小组聚集在一起，每个人都知道大致的目标，但
   <strong class="markup--strong markup--p-strong">
    只有少数人知道关键的细节
   </strong>
   ，比如具体步骤是什么、或谁来进行最危险的活动等 — — 信任分级。
  </p>
  <p class="graf graf--p">
   📌 对最好的情况和最坏的情况都要做好准备。新的想法，如果是好的想法，往往会失败，只因为人们对这些想法的理解不够深入，而
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/1%e3%80%81%e8%ae%af%e6%81%af%e7%ba%aa%e5%be%8b%e4%b8%8d%e6%98%af%e5%88%9b%e9%80%a0%e5%8a%9b%e7%9a%84%e6%95%8c%e4%ba%ba%ef%bc%9a%e6%af%8f%e4%b8%80%e6%ac%a1%e5%bf%83%e8%b7%b3%e9%83%bd%e5%be%88/" href="https://www.iyouport.org/1%e3%80%81%e8%ae%af%e6%81%af%e7%ba%aa%e5%be%8b%e4%b8%8d%e6%98%af%e5%88%9b%e9%80%a0%e5%8a%9b%e7%9a%84%e6%95%8c%e4%ba%ba%ef%bc%9a%e6%af%8f%e4%b8%80%e6%ac%a1%e5%bf%83%e8%b7%b3%e9%83%bd%e5%be%88/" rel="noopener" target="_blank">
    旧的想法会失败的原因是大家对这些想法太熟悉了，包括当局！
   </a>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    有时，最好的结果来自于在全新的环境中应用熟悉的战术。
   </strong>
  </p>
  <p class="graf graf--p">
   回头看看有没有先例，在类似的环境下尝试类似行动的场合。这些可以起到很好的指导作用。当您积累了多年的经验，并从别人的成功和失败中学习时，您将发展出预测和准备各种情况的技能。（在我们的
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/1kod2L/iyp-direct-action5" href="https://start.me/p/1kod2L/iyp-direct-action5" rel="noopener" target="_blank">
    列表-5 中看到 “技巧、策略和原理”，以及镇压者的行动模式 “维稳面面观”，还有其他活动家的创意 “反抗者寻求经验”
   </a>
   ）
  </p>
  <h3 class="graf graf--p">
   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     2、准备工作  — — 准备好装备和适当的服装
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   一旦您的计划制定好了，就画出一个时间表，直到您的行动，从大日子开始倒数，
   <strong class="markup--strong markup--p-strong">
    确定所有必须到位的环节的最后期限。
   </strong>
  </p>
  <p class="graf graf--p">
   📌 在计划的早期，确定您需要哪些资金、材料和其他资源，以及如何获得它们。如果安全是优先考虑的问题，
   <strong class="markup--strong markup--p-strong">
    要以一种无法追踪到您的方式获得您所需要的东西。
   </strong>
  </p>
  <p class="graf graf--p">
   确保每个人都有
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%8a%97%e8%ae%ae%e4%b8%ad%e8%b0%a8%e9%98%b2%e5%b8%b8%e8%a7%81%e5%8d%b1%e9%99%a9%ef%bc%9a%e6%95%b4%e4%bd%93%e5%ae%89%e5%85%a8%ef%bc%8811%ef%bc%89-%e5%8d%8f%e4%bd%9c%e5%92%8c%e4%ba%92%e5%8a%a9/" href="https://www.iyouport.org/%e6%8a%97%e8%ae%ae%e4%b8%ad%e8%b0%a8%e9%98%b2%e5%b8%b8%e8%a7%81%e5%8d%b1%e9%99%a9%ef%bc%9a%e6%95%b4%e4%bd%93%e5%ae%89%e5%85%a8%ef%bc%8811%ef%bc%89-%e5%8d%8f%e4%bd%9c%e5%92%8c%e4%ba%92%e5%8a%a9/" rel="noopener" target="_blank">
    适合行动的衣服
   </a>
   ，包括必要时分层穿上不同的衣服 —— 以帮助您
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9c%a8%e4%bd%a0%e7%9a%84%e6%97%a5%e5%b8%b8%e7%94%9f%e6%b4%bb%e4%b8%ad%e5%ba%94%e7%94%a8%e9%9a%90%e8%94%bd%e9%87%8e%e6%88%98%e6%8a%80%e6%9c%af/" href="https://www.iyouport.org/%e5%9c%a8%e4%bd%a0%e7%9a%84%e6%97%a5%e5%b8%b8%e7%94%9f%e6%b4%bb%e4%b8%ad%e5%ba%94%e7%94%a8%e9%9a%90%e8%94%bd%e9%87%8e%e6%88%98%e6%8a%80%e6%9c%af/" rel="noopener" target="_blank">
    在撤离时甩掉尾随者
   </a>
   。
  </p>
  <p class="graf graf--p">
   📌 考虑到与服装有关的安全问题：如果大家为了匿名而穿黑衣服，要确保任何人的衣服都没有独特的可识别性特征；同样，如果您要随机冒充路人，请记住，您家乡城市的平民服装可能与其他城市不同。如果时间很重要，请确保每个人的手表都是同步的。
  </p>
  <p class="graf graf--p">
   仔细检查，确保在截止日期前一切准备就绪。如果参与者对准备行动的地区不熟悉，要先分发地图。如果需要的话，在行动前在该地区放置必要的材料 — —
   <strong class="markup--strong markup--p-strong">
    注意在此过程中不要泄露任何东西！
   </strong>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     3、侦察 — — 研究行动现场，掌握变化情况
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   📌 行动前，仔细研究该地区。画出进出的安全路线；寻找藏身之处、障碍物、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%96%b0%e5%9c%b0%e5%9b%be%e6%98%be%e7%a4%ba%e4%ba%86%e4%b8%8d%e5%8f%af%e9%81%bf%e5%85%8d%e7%9a%84%e7%9b%91%e8%a7%86%e8%a0%95%e5%8f%98/" href="https://www.iyouport.org/%e6%96%b0%e5%9c%b0%e5%9b%be%e6%98%be%e7%a4%ba%e4%ba%86%e4%b8%8d%e5%8f%af%e9%81%bf%e5%85%8d%e7%9a%84%e7%9b%91%e8%a7%86%e8%a0%95%e5%8f%98/" rel="noopener" target="_blank">
    潜在目标和监视摄像头（包括自动取款机和红绿灯内的摄像头）
   </a>
   。
  </p>
  <p class="graf graf--p">
   📌 注意关键距离需要多长时间，并注意关键地点的能见度。当局离我们有多近，他们需要多长时间才能到达？他们的接近会不会延迟？可不可以制造障碍让他们延迟？该地区还有哪些人？……
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在侦查时，注意不要引起别人的注意，也不要留下明显的经过记录
   </strong>
   。确保至少在计划行动的同一时间进行部分侦查，如果可能的话，在行动前再次进行快速检查，以确保没有任何变化。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%bf%e7%94%a8%e9%80%bb%e8%be%91%e5%bf%ab%e9%80%9f%e8%ae%b0%e5%bf%86%e5%a4%a7%e9%87%8f%e4%bf%a1%e6%81%af%ef%bc%9a%e5%a1%91%e9%80%a0%e8%b6%85%e5%bc%ba%e8%ae%b0%e5%bf%86%e5%8a%9b%e7%9a%84%e6%96%b9/" href="https://www.iyouport.org/%e4%bd%bf%e7%94%a8%e9%80%bb%e8%be%91%e5%bf%ab%e9%80%9f%e8%ae%b0%e5%bf%86%e5%a4%a7%e9%87%8f%e4%bf%a1%e6%81%af%ef%bc%9a%e5%a1%91%e9%80%a0%e8%b6%85%e5%bc%ba%e8%ae%b0%e5%bf%86%e5%8a%9b%e7%9a%84%e6%96%b9/" rel="noopener" target="_blank">
    行动者的记忆力非常重要！
   </a>
   侦查好的信息必须牢记在脑子里，您不可能一边看地图一边行动。
  </p>
  <p class="graf graf--p">
   如果您的行动需要完成艰巨的任务，比如爬上陡峭的屋顶，不妨在某个时间点进行
   <strong class="markup--strong markup--p-strong">
    实际的练习
   </strong>
   。
  </p>
  <p class="graf graf--p">
   📌 还可以从照片、地图和小册子中收集信息；可能会有航空地图或蓝图。在某些情况下，您可以从旅游中心获得信息，或者打电话，借口提问（例如作为学生想做报告，随便编什么都行），甚至接受导游的指导。一旦您收集了大量的信息，将重要的部分整合成适合您的地图会很有帮助。要注意安全地处理所有文件和文书工作。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     4、角色  — — 划分责任，建立决策结构
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   确定完成您的计划所需的所有角色，并确保每一个角色都得到填补。
  </p>
  <p class="graf graf--p">
   📌 一些潜在的角色包括：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    哨兵  — — 发现危险信号时能及时做出反应并安全地通知其他人；
   </li>
   <li class="graf graf--li">
    侦查军 — — 侦查者先行，并对地形做出判断，以制定最合理的下一步计划；
   </li>
   <li class="graf graf--li">
    媒体发言人  — — 确保媒体不会歪曲您的行动，发言人很重要，尤其是确保媒体的报道能覆盖您需要呈现给公众的意义和价值；
   </li>
   <li class="graf graf--li">
    自己的媒体管理员 — — 负责在行动过程中
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%85%ac%e6%b0%91%e5%8f%96%e8%af%81%e6%8a%80%e5%b7%a7%ef%bc%9a%e5%a6%82%e4%bd%95%e7%95%99%e4%bd%8f%e7%9c%9f%e7%9b%b8%ef%bc%9f/" href="https://www.iyouport.org/%e5%85%ac%e6%b0%91%e5%8f%96%e8%af%81%e6%8a%80%e5%b7%a7%ef%bc%9a%e5%a6%82%e4%bd%95%e7%95%99%e4%bd%8f%e7%9c%9f%e7%9b%b8%ef%bc%9f/" rel="noopener" target="_blank">
     记录真相
    </a>
    ，包括行动者的做法，也包括警察的行为，并有经验
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ae%89%e5%85%a8%e5%9c%b0%e7%95%99%e4%bd%8f%e8%af%81%e6%8d%ae%ef%bc%9a%e7%bb%99%e7%9c%9f%e7%9b%b8%e7%9a%84%e8%ae%b0%e5%bd%95%e8%80%85%e4%b8%80%e4%ba%9b%e6%8a%80%e6%9c%af%e6%8a%80/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ae%89%e5%85%a8%e5%9c%b0%e7%95%99%e4%bd%8f%e8%af%81%e6%8d%ae%ef%bc%9a%e7%bb%99%e7%9c%9f%e7%9b%b8%e7%9a%84%e8%ae%b0%e5%bd%95%e8%80%85%e4%b8%80%e4%ba%9b%e6%8a%80%e6%9c%af%e6%8a%80/" rel="noopener" target="_blank">
     安全地保存和呈现记录
    </a>
    ；
   </li>
   <li class="graf graf--li">
    法律援助联系人 — — 在有队友被捕的情况下需要联系的人；
   </li>
   <li class="graf graf--li">
    法律观察员 — —
    <strong class="markup--strong markup--li-strong">
     要熟悉法律，尤其是熟悉法律漏洞，以帮助行动者尽可能避免被起诉；
    </strong>
   </li>
   <li class="graf graf--li">
    医护协作 — — 如果有人受伤时，必须联系的人；
   </li>
   <li class="graf graf--li">
    掩护者 — —
    <strong class="markup--strong markup--li-strong">
     负责吸引镇压者的火力，以掩护行动尽可能多地持续，这一角色的队友需要具备
    </strong>
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%9c%a8%e5%a4%a7%e8%a7%84%e6%a8%a1%e6%9e%aa%e5%87%bb%e6%a1%88%e4%b8%ad%e5%b9%b8%e5%ad%98%ef%bc%9a%e5%9c%a8%e7%81%be%e9%9a%be%e4%b8%ad%e5%b9%b8%e5%ad%98%ef%bc%8810%ef%bc%89/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%9c%a8%e5%a4%a7%e8%a7%84%e6%a8%a1%e6%9e%aa%e5%87%bb%e6%a1%88%e4%b8%ad%e5%b9%b8%e5%ad%98%ef%bc%9a%e5%9c%a8%e7%81%be%e9%9a%be%e4%b8%ad%e5%b9%b8%e5%ad%98%ef%bc%8810%ef%bc%89/" rel="noopener" target="_blank">
     <strong class="markup--strong markup--li-strong">
      足够的逃生技能
     </strong>
    </a>
    ；
   </li>
   <li class="graf graf--li">
    自己的旁观者（例如，伪装成路人的队友，可以在必要时进行干预，或者在路障竖立在他们面前时礼貌地鸣喇叭） — —
    <strong class="markup--strong markup--li-strong">
     引导公众、并分散镇压者的注意力，还能调动真正的旁观者的情绪；
    </strong>
   </li>
   <li class="graf graf--li">
    逃亡时需要的司机 — — 驾驶技术必须足够好；
   </li>
   <li class="graf graf--li">
    运送物资的人 ；
   </li>
   <li class="graf graf--li">
    接受信息和做出战术决策的负责人 ；
   </li>
   <li class="graf graf--li">
    进行实际行动的人。
   </li>
  </ul>
  <p class="graf graf--p">
   📌 在某些情况下，明智的做法是在重要的角色上安排替补队员，以防在最后一刻发现有人不能参加。如果您事先不知道行动的日期会是什么 — — 例如，如果它与您无法事先预测的事件相吻合，如政治犯宣布判决的日期或宣战日 — — 则更是如此。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     5、考虑行动对他人的影响方式
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   如果您的行动是在一个较大的活动期间或作为其一部分进行的，那么可能会有一些大型会议，不同的团体会在这些会议上试图协调他们的努力。这些会议可能是有用的，但它们往往会消耗大量的时间和精力，所以
   <strong class="markup--strong markup--p-strong">
    确保您在参加会议时清楚地知道您希望达到什么目的。
   </strong>
  </p>
  <p class="graf graf--p">
   📌 无论您是在数千名其他活动家中间行动，还是在远离任何人的地方行动，都要考虑到您的行动将如何影响其他人。会不会危及他人？会不会引起警察的镇压？如果会，其他人是否会首当其冲，是否有可能抵消这种影响？您的行动会不会使其他人在某个社区做重要工作时更加困难？在行动前、行动中或行动后，您是否应该进行谈判或做出保证？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    尊重您与其他团体达成的所有协议；有些人可能愿意帮助您，不管是否知道您所做的具体细节。
   </strong>
   随着时间的推移，如果您证明了自己的可靠和体贴，您就会与他们建立联盟。
  </p>

  <h2 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     行动期间和之后
    </strong>
   </span>
  </h2>
  <h3 class="graf graf--p">
   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     1、意识  — — 在整个行动中保持警惕
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   📌 意识是任何行动成功的关键。通常情况下，气氛会很快发生变化。重要的是要跟上您周围发生的事，并事先确定您将如何对某一情况作出反应。
  </p>
  <p class="graf graf--p">
   例如，一辆警车的到来是大事吗？那十辆呢？在这个城市里，警察尾随游行者是常见的事吗？虽然您永远无法确定到底会发生什么，但
   <strong class="markup--strong markup--p-strong">
    提前回顾可能发生的所有情况，并了解您的小组想如何处理这些情况，将使每个人对如何应对 — — 以及如何避免过度反应 — — 有一个更坚实的想法
   </strong>
   。
  </p>
  <p class="graf graf--p">
   📌 在向他人通报事态发展时，要公布原始信息，而不是您可能从中得出的结论（是 “警察正在戴上防毒面具”，而不是 “他们要毒死我们！”），这样其他人就可以得出自己的结论。
  </p>
  <p class="graf graf--p">
   📌 抵制惊慌失措的冲动，也要抵制得意忘形的倾向。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     2、沟通 — — 互相通报情况
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在行动中，侦察员可以
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/1%e3%80%81%e4%ba%86%e8%a7%a3%e4%bd%a0%e7%9a%84%e5%9c%b0%e5%bd%a2%e6%98%af%e7%ad%96%e5%88%92%e8%a1%8c%e5%8a%a8%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" href="https://www.iyouport.org/1%e3%80%81%e4%ba%86%e8%a7%a3%e4%bd%a0%e7%9a%84%e5%9c%b0%e5%bd%a2%e6%98%af%e7%ad%96%e5%88%92%e8%a1%8c%e5%8a%a8%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" rel="noopener" target="_blank">
    跟踪地形的变化
   </a>
   做出反应，如警察的到来、人群的流动、附近其他人的活动和安全区等。他们可以使用通信系统，如一次性电话、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b3%84%e6%bc%8f%e6%96%87%e4%bb%b6%e6%98%be%e7%a4%ba%e9%97%b4%e8%b0%8d%e5%b7%b2%e7%bb%8f%e8%83%bd%e7%bb%95%e8%bf%87-signal-%e7%9a%84%e5%8a%a0%e5%af%86%ef%bc%8c%e9%82%a3%e4%b9%88%e7%ab%af%e5%af%b9/" href="https://www.iyouport.org/%e6%b3%84%e6%bc%8f%e6%96%87%e4%bb%b6%e6%98%be%e7%a4%ba%e9%97%b4%e8%b0%8d%e5%b7%b2%e7%bb%8f%e8%83%bd%e7%bb%95%e8%bf%87-signal-%e7%9a%84%e5%8a%a0%e5%af%86%ef%bc%8c%e9%82%a3%e4%b9%88%e7%ab%af%e5%af%b9/" rel="noopener" target="_blank">
    加密短信
   </a>
   、双向无线电、或哨子来保持联系；
   <strong class="markup--strong markup--p-strong">
    也可以使用音频或视觉信号，如汽车喇叭或烟花。
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.broadcastify.com/listen/" href="https://www.broadcastify.com/listen/" rel="noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     警用扫描仪
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    可以用来监控警方的通讯
   </strong>
   。
  </p>
  <p class="graf graf--p">
   📌 为了提高通信效率，侦察员可以向行动中心的个人或分队报告；在更大的环境中，他们可以将自己的发现打电话给一个中央信息枢纽，其他人可以带着问题打电话给这个中心。
  </p>
  <p class="graf graf--p">
   📌 就像通讯设备可以让您更有效率和效益一样，它也增加了被监视的风险。您可以使用代码和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%82%a8%e5%9c%a8%e4%bb%80%e4%b9%88%e6%97%b6%e5%80%99%e9%9c%80%e8%a6%81%e5%81%87%e5%90%8d%e3%80%81%e5%8c%bf%e5%90%8d%e5%92%8c%e5%85%ac%e5%bc%80%e5%9c%a8%e7%ba%bf%e8%ba%ab%e4%bb%bd%ef%bc%9f/" href="https://www.iyouport.org/%e6%82%a8%e5%9c%a8%e4%bb%80%e4%b9%88%e6%97%b6%e5%80%99%e9%9c%80%e8%a6%81%e5%81%87%e5%90%8d%e3%80%81%e5%8c%bf%e5%90%8d%e5%92%8c%e5%85%ac%e5%bc%80%e5%9c%a8%e7%ba%bf%e8%ba%ab%e4%bb%bd%ef%bc%9f/" rel="noopener" target="_blank">
    假名
   </a>
   ，但要慎重 — — 复杂的代码很容易被遗忘，
   <strong class="markup--strong markup--p-strong">
    检察官可以辩称您的代码比实际意义更激烈
   </strong>
   。
  </p>
  <p class="graf graf--p">
   📌 即使没有使用其他通信系统，在紧急情况下选择一个 “中止” 信号也是有用的。比如间谍的常见做法，在一辆车子上挂一个特殊的标记 —— 情报行动前所有参与者都理解这个标记，让其他人都不理解，然后开着车子途径行动者出没的地区，以提醒人们 “行动中止”。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     3、分散 — — 撤退
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%81%9a%e5%88%b0%e5%ae%8c%e7%be%8e%e9%9a%90%e8%ba%ab%ef%bc%9a%e6%80%9d%e8%80%83%e6%96%b9%e5%bc%8f%e6%98%af%e6%88%98%e7%95%a5%e6%80%a7%e7%9a%84%ef%bc%883%ef%bc%89/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%81%9a%e5%88%b0%e5%ae%8c%e7%be%8e%e9%9a%90%e8%ba%ab%ef%bc%9a%e6%80%9d%e8%80%83%e6%96%b9%e5%bc%8f%e6%98%af%e6%88%98%e7%95%a5%e6%80%a7%e7%9a%84%ef%bc%883%ef%bc%89/" rel="noopener" target="_blank">
    安全逃生
   </a>
   是直接行动组织中最常被忽视的部分。一定要事先制定好退出策略。
  </p>
  <p class="graf graf--p">
   如果您将在一个大的团体中，特别是和其他没有参与策划过程的人一起工作，想想如何
   <strong class="markup--strong markup--p-strong">
    避免从众心理
   </strong>
   ，保持表达自己的权利。
  </p>
  <p class="graf graf--p">
   📌 知道什么时候应该按捺住自己的优势，什么时候该退出 — — 什么时候该跑得越快越好，什么时候应该不紧不慢地走。
  </p>
  <p class="graf graf--p">
   📌 丢弃任何可能使您入罪的东西，如果可能的话，把它放在一个不会被发现的地方；等到您确定甩掉了监视跟踪时，再回来取。
  </p>
  <p class="graf graf--p">
   如果需要，
   <strong class="markup--strong markup--p-strong">
    事后在一个安全的地方集合，确保每个人都有交代
   </strong>
   ；寻求外界援助，写新闻稿。趁着每个人都还在，获取任何可能出庭作证或提供文件以协助被捕者的人的联系信息。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #cc99ff;">
    <strong class="markup--strong markup--p-strong">
     4、汇报 — — 重新聚集在一起，讨论什么是成功的，什么是可以吸取的教训
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   📌 行动结束后，销毁任何可能对您不利的证据；将可能与行动有关的工具放在隐蔽处。
  </p>
  <p class="graf graf--p">
   📌 如果您在未来的某一天可能需要出庭作证，请考虑将您可能需要记住的所有细节写在一张纸上，并将其藏在某个您可以确定永远不会被其他人发现的地方。
  </p>
  <p class="graf graf--p">
   在一个
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%89%be%e5%88%b0%e6%97%85%e5%ba%97%e6%88%bf%e9%97%b4%e9%87%8c%e9%9a%90%e8%97%8f%e7%9a%84%e6%91%84%e5%83%8f%e5%a4%b4%ef%bc%9f/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%89%be%e5%88%b0%e6%97%85%e5%ba%97%e6%88%bf%e9%97%b4%e9%87%8c%e9%9a%90%e8%97%8f%e7%9a%84%e6%91%84%e5%83%8f%e5%a4%b4%ef%bc%9f/" rel="noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     安全的环境
    </strong>
   </a>
   中聚在一起，复述发生的事。跟进正在进行的事，比如支持那些有法庭案件的人，向公众进一步澄清行动背后的目标和想法，以及整理冲突。
  </p>
  <p class="graf graf--p">
   庆祝你们的胜利，互相提出建设性的批评，从错误中学习，并为下一个项目制定计划。
  </p>
  <h2 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     最后
    </strong>
   </span>
  </h2>
  <p class="graf graf--p">
   IYP 近期延续了多个行动主义主题的系列，从 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%8a%97%e8%ae%ae%e4%b8%ad%e8%b0%a8%e9%98%b2%e5%b8%b8%e8%a7%81%e5%8d%b1%e9%99%a9%ef%bc%9a%e6%95%b4%e4%bd%93%e5%ae%89%e5%85%a8%ef%bc%8811%ef%bc%89-%e5%8d%8f%e4%bd%9c%e5%92%8c%e4%ba%92%e5%8a%a9/" href="https://www.iyouport.org/%e6%8a%97%e8%ae%ae%e4%b8%ad%e8%b0%a8%e9%98%b2%e5%b8%b8%e8%a7%81%e5%8d%b1%e9%99%a9%ef%bc%9a%e6%95%b4%e4%bd%93%e5%ae%89%e5%85%a8%ef%bc%8811%ef%bc%89-%e5%8d%8f%e4%bd%9c%e5%92%8c%e4%ba%92%e5%8a%a9/" rel="noopener" target="_blank">
    整体安全
   </a>
   ”、到 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bd%93%e6%b4%bb%e5%8a%a8%e5%ae%b6%e9%99%b7%e5%85%a5%e8%99%9a%e6%97%a0%e5%92%8c%e7%96%b2%e6%83%ab-%e5%af%bb%e6%89%be%e8%a7%a3%e5%86%b3%e6%96%b9%e6%a1%88%ef%bc%9a%e8%a1%8c/" href="https://www.iyouport.org/%e5%bd%93%e6%b4%bb%e5%8a%a8%e5%ae%b6%e9%99%b7%e5%85%a5%e8%99%9a%e6%97%a0%e5%92%8c%e7%96%b2%e6%83%ab-%e5%af%bb%e6%89%be%e8%a7%a3%e5%86%b3%e6%96%b9%e6%a1%88%ef%bc%9a%e8%a1%8c/" rel="noopener" target="_blank">
    行动主义
   </a>
   ”、到 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%e6%8a%80%e5%b7%a7%e5%b8%ae%e5%8a%a9%e6%82%a8%e5%9c%a8%e8%a1%8c%e5%8a%a8%e4%b8%ad%e6%9b%b4%e5%85%b7%e6%95%88%e8%83%bd%ef%bc%9a-%e4%bf%a1%e6%81%af/" href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%e6%8a%80%e5%b7%a7%e5%b8%ae%e5%8a%a9%e6%82%a8%e5%9c%a8%e8%a1%8c%e5%8a%a8%e4%b8%ad%e6%9b%b4%e5%85%b7%e6%95%88%e8%83%bd%ef%bc%9a-%e4%bf%a1%e6%81%af/" rel="noopener" target="_blank">
    信息行动主义
   </a>
   ”、再到 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%84%91%e8%a2%8b%e6%98%af%e4%ba%ba%e6%9c%80%e5%a4%a7%e7%9a%84%e5%bc%b1%e7%82%b9-%e4%ba%86%e8%a7%a3%e5%a4%b4%e7%9b%94%e5%92%8c%e4%bd%bf%e7%94%a8%e6%8a%80%e5%b7%a7%ef%bc%9a%e7%a4%ba%e5%a8%81/" href="https://www.iyouport.org/%e8%84%91%e8%a2%8b%e6%98%af%e4%ba%ba%e6%9c%80%e5%a4%a7%e7%9a%84%e5%bc%b1%e7%82%b9-%e4%ba%86%e8%a7%a3%e5%a4%b4%e7%9b%94%e5%92%8c%e4%bd%bf%e7%94%a8%e6%8a%80%e5%b7%a7%ef%bc%9a%e7%a4%ba%e5%a8%81/" rel="noopener" target="_blank">
    示威者指南
   </a>
   ”，（所有这些链接都是最近的一集，其中包含过往内容列表）；
   <strong class="markup--strong markup--p-strong">
    这些系列的侧重点不同，它们的目标都是一样的，尤其是，它们必须使用我们曾经一直以来不断介绍的各种其他知识 —— 只是当时看起来似乎像 “其他” 而已。
   </strong>
  </p>
  <p class="graf graf--p">
   📌 本文也一样。其中所有嵌入的链接都很重要！那是关于具体做法。
  </p>
  <p class="graf graf--p">
   没有哪种知识是单独存在于空气中的，它们会互相形成一个或多个不同的系统，以支持不同的目标（就连警察的审讯技巧培训手册也完全可以作为活动家
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%A6%82%E4%BD%95%E8%AE%A9%E4%B8%80%E4%B8%AA%E4%BA%BA%E5%BF%83%E7%94%98%E6%83%85%E6%84%BF%E5%90%90%E9%9C%B2%E9%87%8D%E8%A6%81%E4%BF%A1%E6%81%AF%EF%BC%9F%e2%80%8a-%e2%80%8a-%E7%A4%BE/" href="https://www.iyouport.org/%E5%A6%82%E4%BD%95%E8%AE%A9%E4%B8%80%E4%B8%AA%E4%BA%BA%E5%BF%83%E7%94%98%E6%83%85%E6%84%BF%E5%90%90%E9%9C%B2%E9%87%8D%E8%A6%81%E4%BF%A1%E6%81%AF%EF%BC%9F%e2%80%8a-%e2%80%8a-%E7%A4%BE/" rel="noopener" target="_blank">
    抵御审讯诱导
   </a>
   的秘笈，反着用就行了）。
  </p>
  <p class="graf graf--p">
   📌 换句话说，您必须有一个目标，有一个内心志向，这样您在接触到知识和技能的时候就能有意识地去捕获最有利于您的那部分内容，并充分地和有效地使用它们。
  </p>
  <p class="graf graf--p">
   对于行动主义来说，我们无法说的太详细，原因您懂的，正如您使用警察审讯培训手册来抵御审讯那样，您的对手也在拼命挖掘我们提供的信息以期掌握行动者的模式。我们能做的最好的事就是，
   <strong class="markup--strong markup--p-strong">
    把框架教给您，以供您用各种创造力去填充它，给您的对手一个出其不意。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    直接行动不是游戏，不是举牌子那么简单，但它是最可能有效的模式；它要求实践者必须具备足够的知识和技能，以及心理素质和应变能力，这需要日常的训练。
   </strong>
   希望我们提供的内容 —— 不论是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e8%a1%8c%e5%8a%a8%e6%b4%be-%e7%a4%be%e4%bc%9a%e5%8f%8d%e6%8a%97%e7%9a%84%e6%80%9d%e8%80%83/" href="https://www.iyouport.org/category/%e8%a1%8c%e5%8a%a8%e6%b4%be-%e7%a4%be%e4%bc%9a%e5%8f%8d%e6%8a%97%e7%9a%84%e6%80%9d%e8%80%83/" rel="noopener" target="_blank">
    教程系列
   </a>
   还是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.patreon.com/iyouport/posts" href="https://www.patreon.com/iyouport/posts" rel="noopener" target="_blank">
    书籍资源
   </a>
   —— 可以帮助您提升最值得提升的部分。
  </p>
