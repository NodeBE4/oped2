   <li>
    <strong>
     别忘了 iPhone 的，在这里：《
     <a href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e9%80%9a%e8%bf%87%e4%b8%80%e4%ba%9b%e8%ae%be%e7%bd%ae%e9%a2%84%e9%98%b2iphone%e8%a2%ab%e7%9b%97%e6%88%96%e8%a2%ab%e8%ad%a6%e6%96%b9%e6%9f%a5%e6%8a%84%e6%89%80%e9%80%a0%e6%88%90/" rel="noopener noreferrer" target="_blank">
      如何通过一些设置预防iPhone被盗或被警方查抄所造成的损失？
     </a>
     》
    </strong>
   </li>
  </ul>
  <p class="graf graf--p">
   虽然我们希望从未发生过这种事，但是，执法部门确实会毫无规则地查抄公民的手机，在很多国家都是如此；其中任何数据都有可能令您陷入“自证其罪”的困境。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种悲剧的发生时间完全无法预测，于是您应该在平时做好基本的防备工作
   </strong>
   。
  </p>
  <p class="graf graf--p">
   本文推荐一些可行的方法，您可以根据自己有可能面对的情况选择最适合的方案，或者将它们结合起来使用。
  </p>
  <p class="graf graf--p">
   但我们不是律师，所以请自行判断使用这些方法的风险，因为每种情况都不同，无法全面估测。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    提示1：在几秒钟内禁用生物识别
   </strong>
  </p>
  <p class="graf graf--p">
   虽然生物识别技术确实在安全性和便利性方面提供了很好的平衡，但是，在遭遇警察的时候，生物识别技术就可能会被用来对付您。
  </p>
  <p class="graf graf--p">
   这在美国也一样。
  </p>
  <p class="graf graf--p">
   虽然最近加利福尼亚州北区地方法院的一项裁决认为警察强迫您用指纹解锁手机违反了第四和第五修正案，这是第一次，但未来的法庭听证会结论如何还有待观察。
  </p>
  <p class="graf graf--p">
   在 Android 9 Pie 中，谷歌推出了一项称为锁定模式的功能。默认情况下不启用它，因此您需要手动打开它。
  </p>
  <p class="graf graf--p">
   激活后，您可以快速锁定手机，并一键禁用生物识别，将PIN、图案或密码作为解锁手机的唯一方法。
  </p>
  <p class="graf graf--p">
   最重要的是，它只会禁用您的指纹扫描仪（或面部识别）一次，因此您不必记得再去关闭该功能。
  </p>
  <p class="graf graf--p">
   这是 Android 的一个很好的补充，
   <strong class="markup--strong markup--p-strong">
    因为警察、抢劫犯和其他潜在的攻击者可以强迫你或者用你的指纹、面部或语音来解锁智能手机。或者，具有相似的面部或语音的人可以在没有生物识别签名的情况下解锁您的手机
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，在执法方面，由于第5修正案权利，警察无法合法地让您输入密码。
  </p>
  <p class="graf graf--p">
   即便对于非美国用户来说，没有第5修正案保护，使用该功能也能保护您免于如上所述的很多危险，至少能在警察面前拖延时间。
  </p>
  <p class="graf graf--p">
   此外在 Appl 设备要求您一起按下两个按钮几秒钟以触发其锁定模式（在较新的 iPhone 上）；或连续五次点击一个按钮（在较旧的 iPhone 上）。
  </p>
  <p class="graf graf--p">
   要在 Pixel，Pixel XL，Pixel 2 或 Pixel 2 XL 上解锁 Android 9.0 中的这一新安全功能，您只需转到设置中的“安全和位置”即可，然后点击设备安全性下的“锁屏首选项” 。
  </p>
  <p class="graf graf--p">
   接下来，只需确保切换到“显示锁定选项”，即可完成设置。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6391 jetpack-lazy-image" data-lazy-sizes="(max-width: 1082px) 100vw, 1082px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-5.png?resize=1082%2C614&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-5.png?w=1082&amp;ssl=1 1082w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-5.png?resize=300%2C170&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-5.png?resize=768%2C436&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-5.png?resize=1024%2C581&amp;ssl=1 1024w" data-recalc-dims="1" height="614" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-5.png?resize=1082%2C614&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1082"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6391" data-recalc-dims="1" height="614" sizes="(max-width: 1082px) 100vw, 1082px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-5.png?resize=1082%2C614&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-5.png?w=1082&amp;ssl=1 1082w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-5.png?resize=300%2C170&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-5.png?resize=768%2C436&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/1-5.png?resize=1024%2C581&amp;ssl=1 1024w" width="1082"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   如果您没有看到此选项，可能是因为您选择了滑动屏幕锁定而不是模式、PIN 或密码。
  </p>
  <p class="graf graf--p">
   现在，要实际使用锁定模式，只需长按电源按钮，直到电源菜单出现。
  </p>
  <p class="graf graf--p">
   您会看到锁定按钮出现，具体取决于运行的 Android Pie 版本。
  </p>
  <p class="graf graf--p">
   下次尝试解锁手机时，您需要先输入您的模式、PIN或密码。
  </p>
  <p class="graf graf--p">
   如果您的手机运行较旧的 Android 版本，还有另一种解决方法：只需重新启动即可。
  </p>
  <p class="graf graf--p">
   当然，不会建议您因为这一便利而拒绝更新。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    提示2：使用强密码
   </strong>
  </p>
  <p class="graf graf--p">
   在提示1 的前提下，手机提供的最安全的身份验证方法是字母数字组合的长密码。
  </p>
  <p class="graf graf--p">
   使用安全密码（最长可达16个字符），您可以放心，未经您的同意，您的手机将无法解锁，因为执法部门即便使用暴力破解也需要很长很长时间的猜测过程。
  </p>
  <p class="graf graf--p">
   纯数字（或使用常用词或名字）的8个字符的密码将需要一台计算机超过五个月才能破解；将此值增加到16个字符，破解需要几千年的时间。
  </p>
  <p class="graf graf--p">
   这可以是让任何人（包括执法部门）远离您手机的安全选择。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    提示3：禁用智能锁
   </strong>
  </p>
  <p class="graf graf--p">
   很多人喜欢使用 Android 的智能锁定功能，可以更轻松地解锁自己的设备，但是要知道，这是一把双刃剑：它还可以让警察更轻松地进入您的手机。
  </p>
  <p class="graf graf--p">
   特别是“On-Body Detection”设置，当运动传感器检测到自从上次使用 PIN、模式或密码解锁后一直在移动时（以与行走者一致的方式），可以保持手机未锁定状态。
  </p>
  <p class="graf graf--p">
   这种状态下如果您遇到警察的
   <strong class="markup--strong markup--p-strong">
    拦路检查
   </strong>
   ，情况就会变得很糟糕，他们可以通过简单的滑动解锁您的手机。
  </p>
  <p class="graf graf--p">
   智能锁的“可信位置”设置也存在风险。这可以让您的手机在家中等特定位置被自动解锁。
  </p>
  <p class="graf graf--p">
   如果警察闯进您的家里，就可以在没有PIN、模式或密码的情况下解锁您的手机。
  </p>
  <p class="graf graf--p">
   要禁用所有 Smart Lock 选项，请转至设置 →安全 →智能锁，然后单独选择每个项目并禁用它们。
  </p>
  <p class="graf graf--p">
   转至“可信任位置”，删除其中的设置。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    提示4：隐藏通知
   </strong>
  </p>
  <p class="graf graf--p">
   通知是很糟糕的东西，因为它设想的是永远只有您自己掌握手机，这显然非常不现实。
  </p>
  <p class="graf graf--p">
   虽然锁屏确实阻止了警察访问手机内容，但即使无法访问手机的其余部分，自动弹出通知也很可能成为重要信息来源。
  </p>
  <p class="graf graf--p">
   这是因为，默认情况下，通知（特别是消息）可以在锁定屏幕上阅读 —— 有时甚至是完整的消息。
  </p>
  <p class="graf graf--p">
   要更改这种情况，请打开“设置”并选择“安全性和位置”（在不同手机上可能会有不同的标记；搜索 “安全” 很容易找到）。
  </p>
  <p class="graf graf--p">
   选择“锁定屏幕首选项”和“锁定屏幕”。从弹出窗口中，选择“隐藏敏感内容”或“根本不显示通知”。
  </p>
  <p class="graf graf--p">
   如果您的手机配备了通知 LED，我们建议选择后者，因为它提供了最好的保护。
  </p>
  <p class="graf graf--p">
   LED 闪烁您就会知道有新的通知，但它们永远不会出现在锁定屏幕上。
  </p>
  <p class="graf graf--p">
   如果没有 LED 建议选择“隐藏敏感内容”。
  </p>
  <p class="graf graf--p">
   或者，如果您的手机使用的是 Android 9 或更高版本，则可以使用“锁定模式”选项作为临时禁用所有锁屏通知的方式，以防您被警察骚扰或扣留。
  </p>
  <p class="graf graf--p">
   触发锁定模式后，直到您下次使用模式、PIN 或密码解锁设备，通知才会显示在锁定屏幕上。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    提示5：保持您的引导加载程序锁定状态
   </strong>
  </p>
  <p class="graf graf--p">
   Android 有一个检查和平衡系统，以确保在启动时操作系统的每部分仍然是安全的。但它也是一个开放的操作系统。
  </p>
  <p class="graf graf--p">
   为了对希望 root 并同时强化启动安全性的用户保持开放状态，谷歌想出了一个默认情况下锁定 boot chain 的系统，但如果你愿意，你可以使用两部分设置来打开它。
  </p>
  <p class="graf graf--p">
   其中一个设置在 Android 操作系统内部，另一个设置在一个名为 bootloader 的预启动菜单上。
  </p>
  <p class="graf graf--p">
   Android 内部的设置称为OEM解锁。这里的想法是，如果您输入了模式、PIN 或密码以进入操作系统，才会启用此选项。因此如果它已启用，那么就是由您来启用的。
  </p>
  <p class="graf graf--p">
   从 Android 内部启用该设置后，您可以进入引导加载程序模式并发送 Fastboot 命令以解锁引导加载程序。
  </p>
  <p class="graf graf--p">
   这将允许您在设备上安装非官方软件，例如自定义恢复或自定义ROM。反过来，这些可以帮助您获得手机的 root 访问权限。
  </p>
  <p class="graf graf--p">
   为了使事情更安全，解锁引导加载程序的实际操作会自动擦除设备上的所有数据。
  </p>
  <p class="graf graf--p">
   ⚠️但是：如果您的引导程序当前已解锁，执法部门可以轻松地在设备上安装非官方固件，以绕过锁定屏幕并访问您的数据。
  </p>
  <p class="graf graf--p">
   OEM解锁设置并不值得担心，因为它只允许解锁引导加载程序。
  </p>
  <p class="graf graf--p">
   如果警察确实解锁了您的引导加载程序，您设备上存储的所有数据都将被自动删除。
  </p>
  <p class="graf graf--p">
   但是，如果您在启用 OEM 解锁后解锁了引导加载程序 —— 例如，如果您将手机与Magisk一起使用 —— 那么，非常建议在完成后返回并重新锁定引导加载程序。
  </p>
  <p class="graf graf--p">
   此外，如果您安装了自定义恢复（如 TWRP），则应确保使用密码保护恢复菜单。
  </p>
  <p class="graf graf--p">
   启用“OEM解锁”选项在技术上不太安全。理论上，技术娴熟的小偷可以在启用此选项的情况下完全擦除您的设备，这意味着您将无法再使用 Android 设备管理器跟踪设备的位置。但 Android 的其他安全措施仍然存在，因此没有您的模式、PIN或密码，任何人都无法访问您的数据。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    提示6：使用 Cerberus
   </strong>
  </p>
  <p class="graf graf--p">
   Cerberus 是Play商店中的第三方应用程序，提供 Google 的查找我的设备等所有功能。
  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">

  </p>

   这是一个很好的防盗功能，因为该软件几乎可以在设备开机并连接到互联网的任何环境中保持活动状态。
  </p>
  <p class="graf graf--p">
   但是，如果你的手机被一个精通技术的小偷或警察偷走，他们可以使用像 Titanium Backup 这样的 root 应用来卸载 Cerberus。
  </p>
  <p class="graf graf--p">
   或者，他们可以在第一时间使用您使用的相同root访问权限撤消刚刚进行的更改。
  </p>
  <p class="graf graf--p">
   所以，建议打开你的 SuperSU 应用程序，前往 Settings 选项卡，然后点击“Full unroot”按钮。
  </p>
  <p class="graf graf--p">
