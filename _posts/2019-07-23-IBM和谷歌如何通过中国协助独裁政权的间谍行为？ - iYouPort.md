  <p class="graf graf--p">
   这个东西只有一个小行李箱的大小，可以谨慎地放在汽车后备箱里。⚠️当该设备通电时，它就开始秘密监控附近的数百部手机，记录人们的所有私人谈话并吸取人们的所有短信。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该设备是由一家名为 Semptian 的中国监视技术公司生产的几种间谍工具之一，该公司已向中东和北非的威权政府提供了这种监视设备
   </strong>
   ：据两位了解公司运营情况的消息人士透露。
  </p>
  <p class="graf graf--p">
   ⚠️自2015年以来，
   <strong class="markup--strong markup--p-strong">
    Semptian 一直在美国技术的帮助之下建立更强大的大规模监控和审查系统
   </strong>
   ，并以一家名为 iNext 的前线公司的名义向政府出售这些监视技术。
  </p>

   这家中国监视巨头 Semptian 正在与美国寡头 IBM 和领先的美国芯片制造商 Xilinx 合作，推进一系列微处理器，使计算机能够更快地分析大量数据 —— 实现快速的大规模监控。
  </p>
  <p class="graf graf--p">
   这家中国监视技术公司是一家名为 OpenPower 基金会的美国组织的成员，该组织由谷歌和IBM高管创立。
  </p>
  <p class="graf graf--p">
   Semptian、Google 和 Xilinx 都没有回应置评请求。

  <p class="graf graf--p">
   ⚠️但该公司的业务范围远远超出中国。近年来，它一直在全球范围内推广其技术。
  </p>
  <p class="graf graf--p">
   在收到关于 Semptian 在大规模监控中的角色的机密消息来源的提示后，一名记者使用假名与该公司联系并冒充潜在客户。
  </p>
  <p class="graf graf--p">
   在电子邮件中，⚠️一位 Semptian 代表证实
   <strong class="markup--strong markup--p-strong">
    该公司已向中东和北非的间谍机构提供了监控工具 —— 并表示已在一个拒绝被透露名字的国家安装了大规模的监控系统，创建了笼罩其整个人口的数字圆形监狱
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这个名为 Aegis 的大规模监视系统旨在监控所有人的电话和互联网的使用。
  </p>
  <p class="graf graf--p">
   根据该公司
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/6188454-aegis1.html" href="https://www.documentcloud.org/documents/6188454-aegis1.html" rel="noopener noreferrer" target="_blank">
    提供的文件
   </a>
   ，该监视系统可以“存储和分析无限数据”、并“显示每个人的联系”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们已经在其他国家安装了 Aegis 监视系统，覆盖了整个国家，” Semptian 的 Zhu Wenying 在4月的一封电子邮件中说。
  </p>
  <p class="graf graf--p">
   他拒绝提供已安装该监视设备的国家的名字，并称其“高度敏感，我们受到非常严格的[保密协议]的约束”。
  </p>
  <p class="graf graf--p">
   ⚠️西方间谍机构和警察多年来一直在使用类似的监视设备。
  </p>
  <p class="graf graf--p">
   然而，部分要归功于像 Semptian 这样的公司，
   <strong class="markup--strong markup--p-strong">
    该技术越来越多地在专制国家被使用 —— 在这些国家里，持不同政见者被监禁、被折磨，并在某些情况下被处决。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们已经看到了一些普遍的和令人震惊的例子，说明世界各国政府如何通过强大的监视技术瞄准活动家、记者和反对派成员”，总部位于伦敦的人权组织 Privacy International 的执行董事 Gus Hosein 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">

   <strong class="markup--strong markup--p-strong">
    最新进入者
   </strong>
   ”。

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e4%b8%8e%e6%93%8d%e7%ba%b5/" href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e4%b8%8e%e6%93%8d%e7%ba%b5/" rel="noopener noreferrer" target="_blank">
     监视与操纵
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   当被问及中东和北非是否有任何国家拒绝这项交易时，Zhu 写道，伊朗和叙利亚是唯一两个。
  </p>
  <p class="graf graf--p">
   该公司显然愿意与该地区的其他国家合作 ——
   <strong class="markup--strong markup--p-strong">
    如沙特阿拉伯、巴林、摩洛哥、阿联酋、阿曼、苏丹和埃及 —— 这些政府经常滥用人权、打击言论自由、暴力镇压和平抗议。
   </strong>
  </p>
  <p class="graf graf--p">
   文件显示，Semptian 目前正在为政府提供购买四种不同监视系统的机会：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e7%a7%91%e6%8a%80%e5%b7%a8%e5%a4%b4%e5%a6%82%e4%bd%95%e5%b8%ae%e5%8a%a9%e4%b8%ad%e5%9b%bd%e5%bb%ba%e7%ab%8b%e5%a4%a7%e8%a7%84%e6%a8%a1%e7%9b%91%e8%a7%86%e7%b3%bb%e7%bb%9f/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e7%a7%91%e6%8a%80%e5%b7%a8%e5%a4%b4%e5%a6%82%e4%bd%95%e5%b8%ae%e5%8a%a9%e4%b8%ad%e5%9b%bd%e5%bb%ba%e7%ab%8b%e5%a4%a7%e8%a7%84%e6%a8%a1%e7%9b%91%e8%a7%86%e7%b3%bb%e7%bb%9f/" rel="noopener noreferrer" target="_blank">
    Aegis
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/6189037-iNext-Owlet.html" href="https://www.documentcloud.org/documents/6189037-iNext-Owlet.html" rel="noopener noreferrer" target="_blank">
    Owlet
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/6189052-iNext-Hawkeye.html" href="https://www.documentcloud.org/documents/6189052-iNext-Hawkeye.html" rel="noopener noreferrer" target="_blank">
    HawkEye
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/6189036-iNext-Falcon.html" href="https://www.documentcloud.org/documents/6189036-iNext-Falcon.html" rel="noopener noreferrer" target="_blank">
    Falcon
   </a>
   。
  </p>
  <p class="graf graf--p">
   正如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e7%a7%91%e6%8a%80%e5%b7%a8%e5%a4%b4%e5%a6%82%e4%bd%95%e5%b8%ae%e5%8a%a9%e4%b8%ad%e5%9b%bd%e5%bb%ba%e7%ab%8b%e5%a4%a7%e8%a7%84%e6%a8%a1%e7%9b%91%e8%a7%86%e7%b3%bb%e7%bb%9f/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e7%a7%91%e6%8a%80%e5%b7%a8%e5%a4%b4%e5%a6%82%e4%bd%95%e5%b8%ae%e5%8a%a9%e4%b8%ad%e5%9b%bd%e5%bb%ba%e7%ab%8b%e5%a4%a7%e8%a7%84%e6%a8%a1%e7%9b%91%e8%a7%86%e7%b3%bb%e7%bb%9f/" rel="noopener noreferrer" target="_blank">
    上篇文章介绍过的
   </a>
   那样，⚠️Semptian 的旗舰系统 Aegis 旨在安装在手机和互联网网络中，用于秘密收集人们的电子邮件记录、电话、短信、手机位置和网络浏览历史记录。
  </p>
  <p class="graf graf--p">
   ⚠️大多数国家的政府都有权*合法地强迫*电话和互联网提供商安装此类监视设备。
  </p>
  <p class="graf graf--p">
   Semptian 声称 Aegis 提供“全面了解虚拟世界”的能力，使政府间谍能够实时跟踪监视该国所有人。
  </p>
  <p class="graf graf--p">
   它还可以“阻止[访问]互联网上的某些信息” —— 审查政府不希望其公民看到的内容。
  </p>
  <p class="graf graf--p">
   Owlet 和 Falcon 设备规模较小；就如文章开头所描述，它们是便携式的，只专注于监视手机通信。
  </p>
  <p class="graf graf--p">
   它们的大小与行李箱相同，例如，可以通过放在汽车里或俯瞰城市广场的公寓，进行监视操作。
  </p>
  <p class="graf graf--p">
   当 Owlet 设备被激活时，它就开始在周围吸取手机通话和短信。 Semptian 的文件指出，Owlet 有能力
   <strong class="markup--strong markup--p-strong">
    同时监控200部不同的手机
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “
   <strong class="markup--strong markup--p-strong">
    大规模拦截系统用于拦截覆盖范围内的所有语音和短信，
   </strong>
   ”一份描述 Owlet 的文件说。
  </p>
  <p class="graf graf--p">
   该文件补充说，有一个“短信关键字过滤”功能，当局监视者可以根据特定的短语或单词来定位他们想要的任何人。
  </p>
  <p class="graf graf--p">
   与 Owlet 不同，Falcon 系统无法窃听电话或短信。
  </p>
  <p class="graf graf--p">
   相反，它被设计用于跟踪目标手机在近1英里半径范围内的位置，并可将其精确定位在5米范围内，⚠️与美国执法部门使用黄貂鱼的设备类似。
  </p>
  <p class="graf graf--p">
   延伸：《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ba%94%e4%bb%98%e6%94%bf%e5%ba%9c%e4%bd%bf%e7%94%a8%e9%bb%84%e8%b2%82%e9%b1%bc%e5%af%b9%e5%85%ac%e6%b0%91%e8%bf%9b%e8%a1%8c%e7%9a%84%e5%a4%a7%e8%a7%84%e6%a8%a1%e9%97%b4%e8%b0%8d/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ba%94%e4%bb%98%e6%94%bf%e5%ba%9c%e4%bd%bf%e7%94%a8%e9%bb%84%e8%b2%82%e9%b1%bc%e5%af%b9%e5%85%ac%e6%b0%91%e8%bf%9b%e8%a1%8c%e7%9a%84%e5%a4%a7%e8%a7%84%e6%a8%a1%e9%97%b4%e8%b0%8d/" rel="noopener noreferrer" target="_blank">
    如何应付政府使用黄貂鱼对公民进行的大规模间谍行为？#反监视技术
   </a>
   》
  </p>
  <p class="graf graf--p">
   根据 Semptian
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/6189036-iNext-Falcon.html" href="https://www.documentcloud.org/documents/6189036-iNext-Falcon.html" rel="noopener noreferrer" target="_blank">
    的文件
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    只要 Falcon 接通电源，它就可以 “强迫所有附近的移动电话和其他蜂窝数据设备连接到它”，并可以帮助政府当局 “找出任何目标[隐藏]的确切住所”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Falcon 还配备了一个只有口袋大小的设备，政府间谍可以直接把它放在口袋里徒步追踪所有人，定位在1米以内。
  </p>
  <p class="graf graf--p">
   ⚠️这个必需强调一下了。对于反监视，我们一直在强调小心异常缓慢行驶的汽车；现在还要加上：小心那些在一米范围内跟踪你的行人。
  </p>
  <p class="graf graf--p">
   ⚠️尤其是在大型群体事件中、参与敏感聚会途中、前往敏感地点途中，等等。
  </p>
  <p class="graf graf--p">
   ⚠️在这里看到：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    如何知道政府正在监视你？《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e7%9f%a5%e9%81%93%e8%87%aa%e5%b7%b1%e6%ad%a3%e5%9c%a8%e8%a2%ab%e7%9b%91%e8%a7%86%ef%bc%9f/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e7%9f%a5%e9%81%93%e8%87%aa%e5%b7%b1%e6%ad%a3%e5%9c%a8%e8%a2%ab%e7%9b%91%e8%a7%86%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     暗中监视的迹象
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%a3%80%e6%b5%8b%e6%82%a8%e6%98%af%e5%90%a6%e6%ad%a3%e5%9c%a8%e5%8f%97%e5%88%b0%e7%9b%91%e8%a7%86%ef%bc%9f%e2%80%8a-%e2%80%8a-%e7%9b%91%e8%a7%86%e6%a3%80%e6%b5%8b/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%a3%80%e6%b5%8b%e6%82%a8%e6%98%af%e5%90%a6%e6%ad%a3%e5%9c%a8%e5%8f%97%e5%88%b0%e7%9b%91%e8%a7%86%ef%bc%9f%e2%80%8a-%e2%80%8a-%e7%9b%91%e8%a7%86%e6%a3%80%e6%b5%8b/" rel="noopener noreferrer" target="_blank">
     如何检测您是否正在受到监视？ — — 监视检测和特殊保护行动
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%8c%87%e5%8d%97%ef%bc%9a%e5%bd%93%e6%82%a8%e5%87%86%e5%a4%87%e5%8f%82%e5%8a%a0%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8/" href="https://www.iyouport.org/%e6%8c%87%e5%8d%97%ef%bc%9a%e5%bd%93%e6%82%a8%e5%87%86%e5%a4%87%e5%8f%82%e5%8a%a0%e6%8a%97%e8%ae%ae%e6%b4%bb%e5%8a%a8/" rel="noopener noreferrer" target="_blank">
     指南：当您准备参加抗议活动…
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   Semptian 向政府出售的第四个系统 HawkEye 是一个基于摄像头的便携式平台，是的，⚠️也是便携式，还采用了面部识别技术。
  </p>
  <p class="graf graf--p">
   该公司的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.documentcloud.org/documents/6189052-iNext-Hawkeye.html" href="https://www.documentcloud.org/documents/6189052-iNext-Hawkeye.html" rel="noopener noreferrer" target="_blank">
    文件称
   </a>
   ，它被设计用于放置在任何位置，以创建“临时监视场景”。
  </p>
  <p class="graf graf--p">
   HawkEye 的功能是：⚠️在人们走过摄像机前时对他们进行扫描，并将他们的脸部图像
   <strong class="markup--strong markup--p-strong">
    实时地
   </strong>
   与“数百万数据的数据库”中包含的照片进行比较，如果识别出特定的嫌疑人，则立刻触发警报。
  </p>
  <p class="graf graf--p">
   ⚠️再次重申：这东西是便携式的，不是一个大电线杆子。
  </p>
  <p class="graf graf--p">
   Semptian 员工 Zhu 先生写道，其中一些监视工具已经提供给中东和北非地区的当局，即 MENA。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “Aegis、Falcon 和 HawkEye 是执法机构的新解决方案，”Zhu 写道，“所有这三种产品都有成功的故事，有些在中东和北非地区。”
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnas.org/support-cnas/cnas-supporters" href="https://www.cnas.org/support-cnas/cnas-supporters" rel="noopener noreferrer" target="_blank">
    政策智囊团新美国安全中心
   </a>
   的兼职高级研究员 Elsa Kania 表示，Semptian 的监视技术出口似乎符合
   <strong class="markup--strong markup--p-strong">
    更广泛的趋势 ——
   </strong>
   中国企业出口监控和审查技术以寻求新的出口市场、同时也在意识形态上推动中国的所谓的软实力。
  </p>
  <p class="graf graf--p">
   Kania 说：“中国共产党力寻求支持并支持与其本身并不相同的政权；我们看到监视技术的快速传播，在没有制衡机制和开放公民社会的政权中问题更加严重”。⚪️
  </p>
