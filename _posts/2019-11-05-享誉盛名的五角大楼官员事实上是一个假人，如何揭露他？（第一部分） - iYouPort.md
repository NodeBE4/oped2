  <ul class="postList">
   <li class="graf graf--li graf--startsWithDoubleQuote">
    “五角大楼俄罗斯问题资深分析师 David Jewberg” 保持着一个受欢迎的 Facebook 页面，并经常被乌克兰和俄罗斯媒体引用为与五角大楼有关的乌克兰和俄罗斯的内幕来源。他努力将自己描述为一个真正的人，有着一个合法的名字 “David Jewberg”，而不是一个角色扮演游戏。
   </li>
   <li class="graf graf--li">
    一些著名的俄罗斯反对派人士也经常引用 David Jewberg 作为受人尊敬的分析师和现实生活中的联系人。
   </li>
   <li class="graf graf--li">
    David Jewberg 实际上是一个想象中的角色，与美国境内的一大群人有关，其中心是美国金融家 Dan K. Rapoport，就是以他为模版被俄罗斯问题专家 David Satter 和华盛顿其他人
    <strong class="markup--strong markup--li-strong">
     编造出来的角色
    </strong>
    。与 Rapoport 相关的一些私人朋友和职业联系人帮助支撑了这个假人物；例如， Rapoport 的大学朋友的照片被用来代表 Jewberg，一些华盛顿地区的 Rapoport 朋友撰写了大量被署名为 Jewberg 的文章，让他看起来像是一个真人。
   </li>

   </li>
   <li class="graf graf--li">

    </strong>
    。

    温和的保留意见
   </strong>
   （“高级分析师……但谁知道是什么样的分析师……），但是，仍然对他声称的俄罗斯将在几周内落败北约的声明
   <strong class="markup--strong markup--p-strong">
    表示了热烈的回应
   </strong>
   。
  </p>
  <p class="graf graf--p">
   第二天，Bychkov 在 RIA Novosti 发表了一篇名为 “
   <a class="markup--anchor markup--p-anchor" data-href="https://ria.ru/accents/20161014/1479227916.html" href="https://ria.ru/accents/20161014/1479227916.html" rel="noopener noreferrer" target="_blank">
    Jewberg and Barbarossa
   </a>
   ” 的后续文章，“只是必要的澄清没有特别的理由”。
  </p>
  <p class="graf graf--p">
   Bychkov 说他意识到，Jewberg 的身份可能是乌克兰媒体捏造的，但他提到这对他个人而言“并不重要”，因为“ 如果我们信任乌克兰媒体的话 — Jewberg 的声明正是就目前在美国公开讨论的内容的精髓”。
  </p>
  <p class="graf graf--p">
   俄罗斯极受欢迎的共青团真理报也对 Jewberg 的声明发表了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.kp.ru/daily/26594.4/3609695/" href="https://www.kp.ru/daily/26594.4/3609695/" rel="noopener noreferrer" target="_blank">
    一篇文章
   </a>
   ，标题为“美国将在20天内征服俄罗斯？ 这是真的！”
  </p>
  <p class="graf graf--p">
   这篇文章的作者是俄罗斯著名军事分析家、俄罗斯军方的退役上校/发言人 Viktor Baranets。
  </p>
  <p class="graf graf--p">
   在他的专栏文章中，Baranets 瞄准了 David Jewberg（“我们正在处理一个不值得信赖的人的可信度”）和乌克兰媒体（称 Gazeta.ua 为反俄罗斯谣言的垃圾箱），但没有羞于宣布 David Jewberg 是假的。
  </p>
  <p class="graf graf--p">
   如上，Kseniya Kirillova 认为，
   <strong class="markup--strong markup--p-strong">
    俄罗斯媒体一直在设法有效地将 “David Jewberg”用于自己的目的：
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <em class="markup--em markup--p-em">
    “俄罗斯的宣传正在受益于[来自 Jewberg 的激进声明。
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    虽然在 Facebook 上，他正在一个相对较小的观众群面前辩论俄罗斯，以获得一个小范围的净积极效应，但是，他的言论所造成的损害，一旦被主流媒体引述，就会突然变得更大……
   </em>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     俄罗斯宣传人员以相当高的杠杆率利用了他[这个假人]的言论指责美国对俄罗斯的侵略，并为俄罗斯自己的侵略性姿态做辩护
    </em>
   </strong>
   <em class="markup--em markup--p-em">
    。”
   </em>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Jewberg 戏剧性的“辞职”
   </strong>
  </p>
  <p class="graf graf--p">
   在2016年的选举季，David Jewberg 继续保持着他的大型 Facebook 存在并与读者分享他的“内幕专家故事”。
  </p>
  <p class="graf graf--p">
   在2016年秋季，他在乌克兰和俄罗斯的媒体曝光高峰之前，他已经明确表示他不是唐纳德特朗普的粉丝。
  </p>
  <p class="graf graf--p">
   例如，2016年7月，Jewberg 将特朗普获胜的前景等同于“灾难性事件”，可与“种族间战争”和“核爆炸”相提并论。
  </p>
  <p class="graf graf--p">
   在特朗普竞选胜利后的第二天，这个假的“五角大楼官员”在 Facebook 上发表了一个声明，称他将从五角大楼辞职以抗议，并在社交网络上收集了
   <strong class="markup--strong markup--p-strong">
    超过一千个点赞。
   </strong>
  </p>
  <p class="graf graf--p">
   下图中俄语的翻译是这样的：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “可怕。怎么会这样？？？真的有很多人准备推特朗普上台吗？我之前写过，如果特朗普当选，我就会辞职。我不能接受被这个人指挥。”
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6573 jetpack-lazy-image" data-lazy-sizes="(max-width: 930px) 100vw, 930px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/21-1.jpg?resize=930%2C950&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/21-1.jpg?w=930&amp;ssl=1 930w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/21-1.jpg?resize=294%2C300&amp;ssl=1 294w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/21-1.jpg?resize=768%2C785&amp;ssl=1 768w" data-recalc-dims="1" height="950" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/21-1.jpg?resize=930%2C950&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="930"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6573" data-recalc-dims="1" height="950" sizes="(max-width: 930px) 100vw, 930px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/21-1.jpg?resize=930%2C950&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/21-1.jpg?w=930&amp;ssl=1 930w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/21-1.jpg?resize=294%2C300&amp;ssl=1 294w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/21-1.jpg?resize=768%2C785&amp;ssl=1 768w" width="930"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   一天后，Jewberg 写道，他“提交了辞职信”，并评论说：“特朗普就是我所鄙视的一切”。
  </p>
  <p class="graf graf--p">
   Jewberg 的反特朗普和以俄罗斯为重点的博客
   <strong class="markup--strong markup--p-strong">
    并未
   </strong>
   因他的所谓“辞职”而停止；
  </p>
  <p class="graf graf--p">
   然而，在特朗普当选后，Jewberg 以所谓的“内幕人士”采取行动，并持续向读者展示美国媒体对特朗普政府的报道，重点关注正在进行的“俄罗斯门”烟雾弹。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    尽管拥有超过 25,000 名 Facebook 粉丝，尽管在乌克兰和俄罗斯媒体上被广泛曝光，以及在特朗普当选后戏剧性地公开“辞职”，但是，David Jewberg 从来不是一个真正的人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   bellingcat 的调查联系了国务院（据称 Jewberg 工作过的地方）以及国防部（他据称在那里工作到2016年底，并且是美国陆军军官）。两边都说“没有这个人”。国务院甚至直接说，Jewberg 的在线活动
   <strong class="markup--strong markup--p-strong">
    是“虚构的”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-6574 jetpack-lazy-image" data-lazy-sizes="(max-width: 1099px) 100vw, 1099px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/22-1.jpg?resize=1099%2C311&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/22-1.jpg?w=1099&amp;ssl=1 1099w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/22-1.jpg?resize=300%2C85&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/22-1.jpg?resize=768%2C217&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/22-1.jpg?resize=1024%2C290&amp;ssl=1 1024w" data-recalc-dims="1" height="311" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/22-1.jpg?resize=1099%2C311&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1099"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-6574" data-recalc-dims="1" height="311" sizes="(max-width: 1099px) 100vw, 1099px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/22-1.jpg?resize=1099%2C311&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/22-1.jpg?w=1099&amp;ssl=1 1099w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/22-1.jpg?resize=300%2C85&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/22-1.jpg?resize=768%2C217&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/22-1.jpg?resize=1024%2C290&amp;ssl=1 1024w" width="1099"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   那五角大楼的这位所谓官员是怎么被创造出来的呢？
  </p>
  <p class="graf graf--p">
   为什么会有这么多人为这个海市蜃楼埋单？
  </p>
