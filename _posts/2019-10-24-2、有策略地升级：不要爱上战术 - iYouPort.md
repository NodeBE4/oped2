  <p class="graf graf--p">
   很多行动者追求“一步到位”，这不仅不现实，也容易萌生挫败感。
  </p>
  <p class="graf graf--p">
   对于组织者来说，每一步行动的可能性，以及如何应对尤其是负面的可能性，是需要预先介绍给所有参与者熟知的 —— 以做到心中有数，行动会更稳妥。
  </p>
  <p class="graf graf--p">
   当当局无视人民的要求时，行动者可以迫使他们倾听并采取行动。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    ⚠️但是要赢，人们必须保持住压力。
   </strong>
  </p>
  <p class="graf graf--p">
   你的目标很少能在你们一次行动后就放弃，因此，通常需要在逐步升级中战略性地增加对他们的压力。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种升级需要利用多种策略组合。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️请记得，你的目标也在预测你们的行动有可能如何升级，以便提早准备。
  </p>
  <p class="graf graf--p">
   但是一个组织良好的运动总能以新的方式增加压力。
   <strong class="markup--strong markup--p-strong">
    让目标措手不及
   </strong>
   。
  </p>
  <p class="graf graf--p">
   每一项新动作都会再次强调更大的需求，建立力量，以采取下一个升级步骤，并提醒目标：“在你屈服之前我们不会离开”。
  </p>
  <p class="graf graf--p">
   2017年，在乌干达西部举行的 Make Bushenyi Great Again（MBUGA）活动提供了一个很好的例子，该活动使用逐步战略升级来赢得行动者的需求。
  </p>
  <p class="graf graf--p">
   行动者的目标是：当选的区议长。他们的要求是：为 Bushenyi 设立服务委员会（处理政府工作人员相关问题的办公室）。
  </p>
  <p class="graf graf--p">
   首先，他们通过谈判桌向当选议长表达了他们的不满；后者答应他们“正在努力”。
  </p>
  <p class="graf graf--p">
   然而，一个月后，没有任何改变，所以活动家们选择提起诉讼。
  </p>
  <p class="graf graf--p">
   该议长嘲笑诉讼、并嘲笑公民浪费他们的时间和金钱。
  </p>
  <p class="graf graf--p">
   在接下来的五个月里，此案遭遇了不断的踢皮球，每次听证会都被推迟。
  </p>
  <p class="graf graf--p">
   当地公民变得愤怒，并迅速采取了压力升级战术。
  </p>
  <p class="graf graf--p">
   首先，人们选择了
   <strong class="markup--strong markup--p-strong">
    游行示威
   </strong>
   来吸引更高层领导人的注意力，他们可以向法院施压，要求法院采取行动。
  </p>
  <p class="graf graf--p">
   一个月后，活动家们制定了一项
   <strong class="markup--strong markup--p-strong">
    占领地区总部
   </strong>
   的直接行动。
  </p>
  <p class="graf graf--p">
   公民留下了要求服务委员会的标语牌。MBUGA 的活动家们还告诉司法部门，除非立即采取行动，否则他们也将
   <strong class="markup--strong markup--p-strong">
    占领法院
   </strong>
   。
  </p>
  <p class="graf graf--p">
   看到民众的决心，并担心丑闻不断升级，司法部门决定迅速采取行动。
  </p>
  <p class="graf graf--p">
   该案件于2017年7月在法庭上审理，公民赢得了此案。完胜。
  </p>
  <p class="graf graf--p">
   MBUGA 行动从对话升级为法律行动、再升级到街头抗议活动、再到占领建筑物，再到
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/1%E3%80%81%E5%A8%81%E8%83%81%E6%84%9F%E5%8F%AF%E8%83%BD%E6%AF%94%E4%BA%8B%E7%89%A9%E6%9C%AC%E8%BA%AB%E6%9B%B4%E6%9C%89%E6%95%88%EF%BC%9A%E6%88%98%E7%95%A5%E6%80%A7%E7%9B%B4%E6%8E%A5%E8%A1%8C%E5%8A%A8/" href="https://www.iyouport.org/1%E3%80%81%E5%A8%81%E8%83%81%E6%84%9F%E5%8F%AF%E8%83%BD%E6%AF%94%E4%BA%8B%E7%89%A9%E6%9C%AC%E8%BA%AB%E6%9B%B4%E6%9C%89%E6%95%88%EF%BC%9A%E6%88%98%E7%95%A5%E6%80%A7%E7%9B%B4%E6%8E%A5%E8%A1%8C%E5%8A%A8/" rel="noopener noreferrer" target="_blank">
    可信的威胁
   </a>
   …… 最后回到法庭，
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    整体上不仅升级了战斗力，还升级了行动参与者的规模和数量。
   </strong>
  </p>
  <p class="graf graf--p">
   而且，💡组织者非常明智 —— 只有在大多数参与者都意识到这是必要的并且是时候采取下一步措施后，才会升级。
  </p>
  <p class="graf graf--p">
   这并没有固定的模版，取决于您的具体行动是什么、您面临的状况是什么。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “伊莫卡利工人联盟”通过战略性地从基层组织升级到社区范围的罢工、绝食抗议、以及最终全国范围内的抵制运动，为佛罗里达州的农场工人赢得了更高的工资。
  </p>

  <p class="graf graf--p">
   Wangari Maathai 在肯尼亚的绿带运动通过
   <strong class="markup--strong markup--p-strong">
    一系列战略步骤逐步升级
   </strong>
   ，从种植树木到催化推翻独裁者的抵抗，步骤如下：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    与农村妇女联合种植树木。
   </li>
   <li class="graf graf--li">
    在这些树上安放政治意义。
   </li>
   <li class="graf graf--li">
    宣传反对公园和森林的私有化。
   </li>
   <li class="graf graf--li">
    绝食抗议。
   </li>
   <li class="graf graf--li">
    大量活动家占领公园以防止其关闭。
   </li>
   <li class="graf graf--li">
    脱光衣服抗议以要求释放政治犯。
   </li>
   <li class="graf graf--li">
    大规模采取行动结束莫伊独裁统治。
   </li>
  </ol>
  <p class="graf graf--p">
   ⚠️无论具体步骤是什么，原则都是一样的，正如 Saul Alinsky 在“激进规则”中所总结的：
   <strong class="markup--strong markup--p-strong">
    “保持压力。永远不要放松”。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️创造力和敏捷性是关键。
  </p>
  <p class="graf graf--p">
   正如 Alinsky 所说：“继续尝试新事物以令你的对手保持失衡。当你的对手掌握了一种方法时，你就用新的东西从侧翼击中他们”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    重复：创造力和敏捷性是关键。这里并没有固定模式，唯一固定的就是敏捷的应变能力。
   </strong>
  </p>

  <p class="graf graf--p">
   ⚠️不要爱上你的战术 —— 不要因为成功一次就使其为崇拜的模版。
  </p>
  <p class="graf graf--p">
   因为，很多情况下你只能成功一次。
  </p>
  <p class="graf graf--p">
