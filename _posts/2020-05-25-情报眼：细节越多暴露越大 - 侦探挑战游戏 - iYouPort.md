  <p class="graf graf--p">
   人们总是不知道自己发布的照片中都包含了哪些秘密 —— ⚠️事实上永远比你想象中要多。
  </p>
  <p class="graf graf--p">
   其实最近我们发现了一个很好的现象，一些中国网络用户在上传照片的时候的确会遮盖脸部、二维码和地址邮编的部分；但是，
   <strong class="markup--strong markup--p-strong">
    并非只有这些细节是致命的。
   </strong>
  </p>
  <p class="graf graf--p">
   👉这就是 *侦探挑战* 游戏的用意，即 邀请您亲手调查一张陌生的照片，以此追踪发布者的位置甚至曝光其真实身份。
  </p>
  <p class="graf graf--p">
   如果你尝试完成了相关的思考，下一次在准备上传任何照片的时候，您就会主动
   <strong class="markup--strong markup--p-strong">
    采取追踪者的眼光
   </strong>
   来审视您准备发布的内容。
  </p>
  <p class="graf graf--p">
   今天的内容继续旨在锻炼这一能力 ——
   <strong class="markup--strong markup--p-strong">
    您的自我保护直觉，和发现重要线索的敏锐。
   </strong>
  </p>
  <p class="graf graf--p">
   军事情报人员接受的训练就包括这点，他们会拿到一张看上去细节不多的照片，但是他们被要求从中找到二十个重要的情报：照片中的人是谁？什么时候拍摄的这种照片？目标当时的心情如何？为什么是这样？他和谁在一起？他在哪里？他穿的是什么鞋？他穿的衣服是在哪里买到的？等等。
  </p>
  <p class="graf graf--p">
   👉提出正确的问题有助于提取更多的情报线索；同时，只有当您拥有的信息越多的时候，才能越容易发现某些重要的东西。
  </p>
  <p class="graf graf--p">

  </p>

   很明显这是一个关于 when 的问题。但是在没有任何EXIF数据或可见时钟的情况下，
   <strong class="markup--strong markup--p-strong">
    唯一计算时间的方法
   </strong>
   …… 还记得是什么吗？
  </p>
  <p class="graf graf--p">
   没错，就是利用太阳的阴影。
  </p>
  <p class="graf graf--p">
   发布日期是
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/bayer_julia/status/1198998634143989760" href="https://twitter.com/bayer_julia/status/1198998634143989760" rel="noopener noreferrer" target="_blank">
    Nov 25, 2019
   </a>
   ，帖子中的提示是 “Last Thursday”，也就是说 **11月21日，星期四**。但是，
   <strong class="markup--strong markup--p-strong">
    如果需要知道具体拍摄时间，您首先要知道的就是具体地点
   </strong>
   。
  </p>
  <p class="graf graf--p">
   那么如何开始对该图像进行地理定位呢？
  </p>
  <p class="graf graf--p">
   你可以采用几种不同的技术，但是 nixintel 认为这是一个演绎军事版《二十个问题》的好机会 ——
   <strong class="markup--strong markup--p-strong">
    您可以从这张照片中提取二十条情报线索来帮助入门吗？
   </strong>
  </p>
  <p class="graf graf--p">
   来试试看：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    晴朗天气；
   </li>
   <li class="graf graf--li">
    现代建筑，玻璃和金属构造；
   </li>
   <li class="graf graf--li">
    它是什么样的建筑？不是住宅或工业建筑，看起来也不像是宏伟的公共建筑（比如法院、议会等）。
   </li>
   <li class="graf graf--li">
    在建筑物的上面部分正在进行一些建筑工作；
   </li>
   <li class="graf graf--li">
    右侧有 “Menu” 标志 —— 可能是咖啡馆或餐馆；
   </li>
   <li class="graf graf--li">
    外面有大型百事可乐标志；
   </li>
   <li class="graf graf--li">
    从左面玻璃窗的反射来看，这个建筑物位于道路旁的开放式绿色空间对面（比如公园或公共花园）；
   </li>
   <li class="graf graf--li">
    人行道上的某种电话或电气维护箱；
   </li>
   <li class="graf graf--li">
    带有YBS标志的黄色巴士– YBS代表什么？
   </li>
   <li class="graf graf--li">
    巴士服务路线号是79；
   </li>
   <li class="graf graf--li">
    公共汽车上的文字是非西方文字；
   </li>
   <li class="graf graf--li">
    车身上刊登的是 “100 Plus Active” 软饮料的广告 —— 该产品在哪里出售？广告上有一个 Facebook 的logo—— 该公司是否有一个 Facebook 页面可以告诉我们更多有关它们在世界上哪些地方经营的信息？
   </li>
   <li class="graf graf--li">
    公交车牌照的格式：红色背景上的白色文字，3N-2457。在此之上有一些较小的文字；
   </li>
   <li class="graf graf--li">
    红色和白色相间的混凝土路缘石。哪些城市/国家有这样的道路？
   </li>
   <li class="graf graf--li">
    主楼前有单独的道路或停车场？
   </li>
   <li class="graf graf--li">
    照片中的人似乎是亚洲人，可能是东南亚人；
   </li>
   <li class="graf graf--li">
    照片中的人穿着的衣服和绿树暗示着温暖的气候；
   </li>
   <li class="graf graf--li">
    台阶右侧好像是某种神祭物品；
   </li>
   <li class="graf graf--li">
    一个人坐在大楼外的台阶上。街上的人不多，交通也不多 —— 这是否意味着是一天中很早的时间？这栋楼还没有开门吗？
   </li>
   <li class="graf graf--li">
    有人体模特在建筑物的窗户中显现。说明该建筑物的部分或全部用于零售。
   </li>
  </ol>
