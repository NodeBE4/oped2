  <p class="graf graf--p">
   很高兴与IYP会面，尤其是能交谈到一些我们大家都感兴趣的问题。
   <strong>
    比如香港，比如直接行动的思考方式，比如透明度革命……等等，还有 IYP 自己的心里话。
   </strong>
  </p>
  <p class="graf graf--p">
   这次会面的时间是
   <strong class="markup--strong markup--p-strong">
    今年8月
   </strong>
   ，现在我们终于有机会将它翻译成中文，因为我们觉得这次访谈中涉及到的内容，中文读者值得了解。
  </p>

  <p class="graf graf--p">
   本文的标题是一张保险杠贴纸，IYP 就开着这么一辆80年代的老爷车 —— 顶着这个庸俗的贴纸 —— 来到这里。说实话一开始我想笑。
  </p>
  <p class="graf graf--p">
   我的名字缩写是 MR，以下我和IYP之间的对话（略去了理论性的和专业性的无聊内容）。
  </p>
  <h3>
   <span style="color: #ff9900;">
    <em>
     —— 以下是访谈的主要部分——
    </em>
   </span>
  </h3>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    MR
   </strong>
   ：“成为你想要在这个世界上看到的改变” —— 每个人都知道这是甘地的名言。简洁而有力。
  </p>
  <p class="graf graf--p">
   就如很多名言一样，很适合做成保险杠贴纸。你也这样觉得是吧？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    IYP
   </strong>
   ：的确如此。这些字很容易和大众内心的某些东西产生共鸣，就如冷读（Cold reading）的逻辑，它似乎在暗示某些真实而有价值的东西。
  </p>
  <p class="graf graf--p">
   但是，就像今天你能看到的许多其他简洁的名言贴纸一样，那些话从来不是它们声称的人所说的。
  </p>
  <p class="graf graf--p">
   甘地实际上是这样说的：
  </p>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote">
   <p>
    “如果我们能改变自己，那么世界的趋势也会随着发生变化。当一个人改变自己的本性时，世界也会随之趋向他。”
   </p>
  </blockquote>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    MR
   </strong>
   ：啊哦，这比上面那个普及版本更具对抗性，不是吗？改变我自己的本性？
  </p>
  <p class="graf graf--p">
   我还以为我们谈论的是那种简单轻松的事，比如不穿皮草、素食主义、或购买公平贸易咖啡豆 ……
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    IYP
   </strong>
   ：这就是今天“你希望在世界上看到的改变”的最大众化解释方式。
  </p>
  <p class="graf graf--p">
   这句话已经在我们的社会中如此普遍地反刍，它现在已经变成了陈词滥调，几乎没有意义，你在跳骚市场的廉价钥匙链上也能看到的东西；没有人再去仔细阅读它，但是大多数人都在
   <strong class="markup--strong markup--p-strong">
    假设自己已经理解
   </strong>
   ，因为你之前已经看过很多次了。
  </p>
  <p class="graf graf--p">
   如果想要真正去思考它，大多数人会说这意味着要“做出”你想要看到的世界变化。比如，如果你不喜欢工厂化养殖，那就成为素食主义者；如果你不喜欢贫困，就去在汤厨房做志愿者，等等。
  </p>
  <p class="graf graf--p">
   但这不是甘地所说的。这与甘地原来的说法完全不同。它甚至不是后来简化的保险杠贴纸版本所说的。
  </p>
  <p class="graf graf--p">
   你真的应该再看一下这个被广泛引用的推广版本，真的，用新鲜的眼光去看它，
   <strong class="markup--strong markup--p-strong">
    排除
   </strong>
   它被企业自由主义者轻率反刍并贴在 K-Mart 产品上的商业化部分 —— 你就会能得到与原版相同的信息。
  </p>
  <p class="graf graf--p">
   它没有说“
   <strong class="markup--strong markup--p-strong">
    做
   </strong>
   你想要在世界上看到的改变”；它也没有说“
   <strong class="markup--strong markup--p-strong">
    宣布
   </strong>
   你想要在世界上看到的改变”。
  </p>
  <p class="graf graf--p">
   它说的是：“
   <strong class="markup--strong markup--p-strong">
    成为
   </strong>
   ，你想要在世界上看到的改变”。
  </p>
  <p class="graf graf--p">
   它并
   <strong class="markup--strong markup--p-strong">
    不仅仅
   </strong>
   指的是行为或生活方式的改变，而是改变你作为一个人的身份。它说的是通过改变自己的本性来改变世界。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    MR
   </strong>
   ：嗯，这的确不同于传统的解释。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    IYP
   </strong>
   ：对引语的传统解释存在于一种无聊的陈词滥调中，人们会嘲笑它的过度使用，就像你刚才看到我的车时的表情。然而每个人都会结合自己的经验去理解这些话，最终得出各自完全不同的答案。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    MR
   </strong>
   ：我们可以稍后再考虑这个问题？我今天带来了一大堆问题，甘地可不在计划内。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    IYP
   </strong>
   ：好的。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-8802 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/下午茶2.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/下午茶2.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/下午茶2.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/下午茶2.jpg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/下午茶2.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/下午茶2.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/下午茶2.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/下午茶2.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-8802" data-recalc-dims="1" height="733" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/下午茶2.jpg?resize=1100%2C733&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/下午茶2.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/下午茶2.jpg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/下午茶2.jpg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/下午茶2.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/下午茶2.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/08/下午茶2.jpg?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <h3 class="graf graf--p graf--startsWithDoubleQuote">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     “记住：目标始终是要让异议思想成为主流”
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    MR
   </strong>
   ：我很喜欢你们的《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9c%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e4%b8%8a%e6%8a%97%e8%ae%ae%e6%9c%89%e4%bb%80%e4%b9%88%e9%97%ae%e9%a2%98%ef%bc%9f/" href="https://www.iyouport.org/%e5%9c%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e4%b8%8a%e6%8a%97%e8%ae%ae%e6%9c%89%e4%bb%80%e4%b9%88%e9%97%ae%e9%a2%98%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    在社交媒体上”抗议”有什么问题？
   </a>
   》，但是我觉得它没有说完全部。你觉得呢？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    IYP
   </strong>
   ：是这样。至少它说明白了两个关键部分之一。
  </p>
  <p class="graf graf--p">
   而另一个关键 — 缺失的部分 — 也许就是你感觉到的东西：即
   <strong class="markup--strong markup--p-strong">
    社交媒体促进的是自慰，而不是革命。
   </strong>
  </p>
  <p class="graf graf--p">
   令人惊讶的是，
   <strong class="markup--strong markup--p-strong">
    有太多的人将自慰与革命政治活动混为一谈了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   我们已经谈论了很多关于如何通过忽略他们的言语、并观察他们的行为来辨别特定的政治家、富豪或官员的真实意图的原则，但要知道，这个原则可以平等地
   <strong class="markup--strong markup--p-strong">
    适用于每一个人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果你忽略掉人们所说的话，忽略掉那些在革命思想的圈子中分享的不同意见，反过来，观察他们的行为，你就会产生这样的印象：即
   <strong class="markup--strong markup--p-strong">
    他们中的大多数人实际上只是对戏剧性、冲突、社交攀爬感兴趣，追求的只是社交媒体上的赞誉、转发和评分
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    MR
   </strong>
   ：他们想要的只是从他们自己的小部落圈子里获得支持和支配权，就像人类的祖先所追求的圈地统治那样。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    IYP
   </strong>
   ：是的，这是最原始的、和难以改变的冲动。
  </p>
  <p class="graf graf--p">
   大多数人都在专注于以某种方式维持现有的小圈子，这是上述冲动的
   <strong class="markup--strong markup--p-strong">
    首要动机
   </strong>
   ，这点上任何其它团体中的任何人都
   <strong class="markup--strong markup--p-strong">
    没有什么不同
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在社交网络上，再一次，没有国籍、语言、性别、代际、阶层，等等任何区别。没有，所有团体都是如此。
  </p>
  <p class="graf graf--p">
   显然，
   <strong class="markup--strong markup--p-strong">
    让一个人的自我骚动起来并不是政治激进主义的真正目的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    MR
   </strong>
   ：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%88%91%e6%b0%b8%e8%bf%9c%e6%ad%a3%e7%a1%ae%e8%83%8c%e5%90%8e%e7%9a%84%e5%8a%a8%e6%9c%ba/" href="https://www.iyouport.org/%e6%88%91%e6%b0%b8%e8%bf%9c%e6%ad%a3%e7%a1%ae%e8%83%8c%e5%90%8e%e7%9a%84%e5%8a%a8%e6%9c%ba/" rel="noopener noreferrer" target="_blank">
    “我永远正确”
   </a>
   。人们能在社交网络上看到的东西大抵都是如此。
  </p>
  <p class="graf graf--p">
   不论任何热点，不论有多激烈，即使评论的主题是行动者，也是如此；就如一场戏，所有人都在抢戏，而
   <strong class="markup--strong markup--p-strong">
    不是给予帮助
   </strong>
   。
  </p>
  <p class="graf graf--p">
   厌恶这种生态的人会因此远离。结果就是，剩下的几乎所有都是这场“戏”中的角色和道具。没有观众。
  </p>
  <p class="graf graf--p">

    让人们自己体验到它的重要性
   </a>

   </strong>

   <strong class="markup--strong markup--p-strong">
    MR

  <h3 class="graf graf--p graf--startsWithDoubleQuote">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     “表达者不考虑实用性”
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    MR
   </strong>
   ：你们强调“行动与表达不同”，但在中国，很多人会认为“表达就是行动”。你能解释下为什么你们认为不同吗？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    IYP
   </strong>
   ：首先澄清，我们认为表达和行动都具有价值；只是它们意味着对于我们这些倡导者来说需要
   <strong class="markup--strong markup--p-strong">
    不同的接触方向
   </strong>
   。
  </p>
  <p class="graf graf--p">
   举个最简单的例子，比如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%9a%90%e5%86%99%e6%9c%af%e7%9a%84%e5%b0%8f%e5%bf%83%e6%9c%ba/" href="https://www.iyouport.org/%e9%9a%90%e5%86%99%e6%9c%af%e7%9a%84%e5%b0%8f%e5%bf%83%e6%9c%ba/" rel="noopener noreferrer" target="_blank">
    隐写术
   </a>
   。就如你所知道的那样，这可以抵制几乎所有自动化的审查。
  </p>
  <p class="graf graf--p">
   理论上我们可以在这样的语境下推荐隐写术：即 当一个在线群组讨论审查问题时、并对“发出的内容只有自己可见”而感觉困惑时。
  </p>
  <p class="graf graf--p">
   这只是理论上。
  </p>
  <p class="graf graf--p">
   但是，事实上。多次测试后的结论却是，
   <strong class="markup--strong markup--p-strong">
    对审查*表达*困惑的人并不关心隐写术 —— 或者说解决问题的方法
   </strong>
   ；在上述情况下真正关心这一技术的人是目标行动的人。
  </p>
  <p class="graf graf--p">
   曾经有中国读者为我们解释这件事：对于表达者来说，被审查瞄准的内容被视为“具有更高价值”，如果能轻松绕避审查，这份价值就会被“削弱”；或者简单说，
   <strong class="markup--strong markup--p-strong">
    表达者考虑的不是实用性
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这是个惊人的逻辑，但也不难理解。
  </p>
  <p class="graf graf--p">
   IYP
   <strong>
    显然不会
   </strong>
   令纯粹的表达者满意，我们注重的是
   <strong class="markup--strong markup--p-strong">
    解决问题的方案
   </strong>
   ；我们推荐的技术不仅帮助更顺利无阻的表达，更为了顺利的行动。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “行动与表达不同”，这句话是关于权利倡导者对受压迫者的协助模式的思考，而非价值判断
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但这里显然还有一个问题。
  </p>
  <p class="graf graf--p">
   如果90%是表达者，那么理论上我们推荐的
   <strong class="markup--strong markup--p-strong">
    自我保护技术
   </strong>
   （比如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%bf%e7%94%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%97%b6%e4%bd%a0%e5%bf%85%e9%a1%bb%e6%b3%a8%e6%84%8f%e4%bb%80%e4%b9%88%ef%bc%9f%e2%80%8a/" href="https://www.iyouport.org/%e4%bd%bf%e7%94%a8%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%97%b6%e4%bd%a0%e5%bf%85%e9%a1%bb%e6%b3%a8%e6%84%8f%e4%bb%80%e4%b9%88%ef%bc%9f%e2%80%8a/" rel="noopener noreferrer" target="_blank">
    隐身
   </a>
   /
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" rel="noopener noreferrer" target="_blank">
    分身
   </a>
   等等），应该能得到更高的关注。但事实上也并非如此。
  </p>
  <p class="graf graf--p">
   对于个人影响力大的 — 包括期待更大个人影响力的表达者来说，隐身/分身和化名的保护性技术无法提供给他们想要的那种*将在线影响力在现实中变现*的能力；
  </p>
  <p class="graf graf--p">
   相反，
   <strong class="markup--strong markup--p-strong">
    是那些认为表达自由是纯粹的解放的人，才会对这类保护性技术更多关注。
   </strong>
  </p>
  <p>
   此外，在交流方面也是有区别的。比如我们经常会听到异议人士这样说 “团结要不得，恶意者才讲团结”，类似的观点。在讨论（辩论）的角度上似乎合理，也就是表达层面上；但是，
   <strong>
    在行动者听起来这是不可思议的，因为单枪匹马的行动注定无法成功
   </strong>
   。
  </p>
  <p>
   行动中不分左中右翼、派系或社区，只分敌我，站在同一方向上的所有人都是自己人。只有这样的联合才能有抵抗力。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    MR
   </strong>
   ：这就是你们为什么不关注网络名人？因为他们只是纯粹的表达者为了促进个人名气并没有行动力？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    IYP
   </strong>
   ：准确说是，“网络名人”不关心我们（主张的实用性）。
  </p>
  <p class="graf graf--p">
   任何内容提供者和倡导者如果声称不想要知名人士的协助推动，那都是在撒谎，这是由于互联网的传播模式  — — 关于你的内容能接触到多少受众。
  </p>
  <p class="graf graf--p">
   我们肯定期待实用性高的内容（而不是口水内容）能接触到更多读者，以帮助更多人改善安全性并增强行动力。但鉴于我们是一群0资金不盈利的义务劳动强迫症患者，于是我们至少肯定没有钱去购买影响力人士的支持。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    MR
   </strong>
   ：如果有这样的机会，你愿意去“买”吗？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">

  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    政府和寡头想要你相信的东西有很多，⚠️其中最不重要的就是观点 — — 真的一点都不重要；他们最想要你接受的是思考方式
   </strong>

   <strong>

   </strong>

  </p>
  <p class="graf graf--p">
   ⚠️你能发现一个国家的反对派团体和执政党具有同样的思考方式 — — 思考方式的模式，虽然他们彼此反对对方的主张；这种情况在很多国家都存在。
  </p>
  <p class="graf graf--p">
   人们将日复一日地争论政治意识形态、或应该对特定问题采取什么样的措施，但是，很少会转过身来检查一下自己
   <strong class="markup--strong markup--p-strong">
    用来形成这些意见的信息
   </strong>
   来源是什么。
  </p>
  <p class="graf graf--p">
   事实上，关于世界上正在发生的事的大部分信息都是由那些无可否认地想要维持现状的既得利益者所掌控的，不论你选择任何语种。
  </p>
  <p class="graf graf--p">
   人们基本不会质疑自己接收到的信息的性质，也同样不会质疑这些信息进入自己的头脑后会发生什么。长此以往便产生了认知偏见和
   <strong class="markup--strong markup--p-strong">
    感知过滤器
   </strong>
   — — ⚠️
   <strong class="markup--strong markup--p-strong">
    这些过滤器决定了所有未来信息的解释方式
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这就是宣传。它让你深刻认为那些想法就来自于本心。
  </p>
  <p class="graf graf--p">
   这也是为什么真相难以接触到人们。
  </p>
  <p class="graf graf--p">
   因为真相违反这种积攒而形成的解释方式。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对于我们大多数人来说，思想并不是推动进化的有用工具，而是作为整个“节目”的作家、导演和明星
   </strong>
   。就如 Ecknath Easwaran 曾经说过的：“
   <strong>
    <em>
     we don’t think our thoughts, our thoughts think us
    </em>
   </strong>
   .”
  </p>
  <p class="graf graf--p">
   当权者最可怕的地方正在于他们真的能做到控制思想而不需要审查，因为权力控制资本的派发，被权力不喜欢的东西会因缺乏资本而被边缘化并被淹没，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%aa%92%e4%bd%93%e5%b7%a8%e5%a4%b4%e5%92%8c%e7%a1%85%e8%b0%b7%e7%9a%84%e6%ad%a6%e5%99%a8%e5%8c%96%e7%ae%97%e6%b3%95%e6%b2%89%e9%bb%98%e5%bc%82%e8%ae%ae/" href="https://www.iyouport.org/%e5%aa%92%e4%bd%93%e5%b7%a8%e5%a4%b4%e5%92%8c%e7%a1%85%e8%b0%b7%e7%9a%84%e6%ad%a6%e5%99%a8%e5%8c%96%e7%ae%97%e6%b3%95%e6%b2%89%e9%bb%98%e5%bc%82%e8%ae%ae/" rel="noopener noreferrer" target="_blank">
    被垄断算法的阴谋所隐瞒
   </a>
