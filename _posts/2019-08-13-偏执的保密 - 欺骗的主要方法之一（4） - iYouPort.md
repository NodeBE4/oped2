  <p class="graf graf--p">
   战略性欺骗的主要方法可以分为两大类：拒止和欺骗。拒止就是保密，不让对手通过任何情报渠道获取到真实信息，以保证战略性欺骗的正常运行。
  </p>
  <p class="graf graf--p">
   由于存在很多人力情报和技术情报手段，
   <strong class="markup--strong markup--p-strong">
    针对这些手段
   </strong>
   的拒止措施也多种多样。关于情报手段的介绍，详见这篇文章《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e5%9c%b0%e5%9b%be%e6%98%af%e6%af%94%e8%a9%b9%e5%a7%86%e6%96%af%e9%82%a6%e5%be%b7%e6%9b%b4%e6%a3%92%e7%9a%84%e9%97%b4%e8%b0%8d/" href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e5%9c%b0%e5%9b%be%e6%98%af%e6%af%94%e8%a9%b9%e5%a7%86%e6%96%af%e9%82%a6%e5%be%b7%e6%9b%b4%e6%a3%92%e7%9a%84%e9%97%b4%e8%b0%8d/" rel="noopener noreferrer" target="_blank">
    “谷歌地图”是比詹姆斯邦德更棒的间谍
   </a>
   》。
  </p>
  <p class="graf graf--p">
   通常来说，大规模的行动部署是无法隐藏的，仅依靠保密不可能达到目的（游击队方案相比下更棒），还需要有相应的欺骗方法来迷惑对手的耳目。
  </p>
  <p class="graf graf--p">
   这就涉及各种欺骗措施。
   <strong class="markup--strong markup--p-strong">
    欺骗的目的是努力让对手相信假象而不是事实，引导其作出错误的判断。
   </strong>
  </p>
  <p class="graf graf--p">
   拒止虽然不是直接的、有意的欺骗行为，但是可以控制目标的知觉，是实施欺骗的重要前提和基础。正是拒止让欺骗成为可能 ——
   <strong class="markup--strong markup--p-strong">
    您现在知道了为什么专制国家的洗脑运动是以封锁为大前提的？更重要的是，知道了为什么我们要坚持不断推荐隐私保护和匿名措施？
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️不论是攻还是防，保密都是最重要的大前提。
  </p>
  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    —— 拒止措施 ——
   </strong>
  </h4>
  <p class="graf graf--p">
   拒止措施多种多样，可大致归为几大类：建立安全保密制度 —— 要为您的队友制订安全规则使所有人都具备足够强度的安全意识、物理安全 —— 重要东西/意图存放的地点、数字技术安全 —— 详见菜单“技术防身”、人事安全 —— 取信的技巧，免于对手的渗透。
   <strong class="markup--strong markup--p-strong">
    以上是基于反抗者群体而言的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️这里的*安全*指的是采取各种行动，给对手的情报搜集活动制造足够高的障碍。
  </p>
  <p class="graf graf--p">
   以下分别解释，基于战略性欺骗的各种措施。如果您的目标是防御并击溃战略性欺骗，了解其手段并反其道而行便可成功。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、安全保密制度
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️要成就一项欺骗计划需要大量的工作，然而要使之垮台却非常容易。
  </p>
  <p class="graf graf--p">
   一条不合时宜的新闻、一项不断发布的声明、一处小小的漏洞、一个隐藏的间谍……都可能使整个欺骗行动前功尽弃。
  </p>
  <p class="graf graf--p">
   因此，建立严格的保密制度是欺骗的第一要素。
  </p>
  <p class="graf graf--p">
   严守欺骗秘密的主要措施有以下几种：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    首先，建立信息知悉权制度，严格限制高级别知情者的人数 —— 也就是IYP曾经强调过的“信任分级”
   </li>
  </ul>
  <p class="graf graf--p">
   ⚠️知情人越少保密效果越好，时间越晚越有利于保密。
  </p>
  <p class="graf graf--p">
   非中心级参与者不得知晓任何相关信息，不论是你的家人还是某些重要领导人。
  </p>
  <p class="graf graf--p">
   比如情报部门策划的欺骗行动，其参与者/工具是一位外交官，那么该国总统和其他政府官员都会被排除在知情范围之外，该外交官的妻子和孩子也不应该知道任何事。
  </p>
  <p class="graf graf--p">
   这就是知悉权制度(need-to-know)。是否拥有接触秘密信息的资格，由秘密信息的法定持有人来决定 [Executive Order 12958, Sec. 4.1]
  </p>
  <p class="graf graf--p">
   在古巴导弹危机前，赫鲁晓夫严格限制了知悉范围。米高扬的政治秘书说：“讨论一开始就是秘密行动，甚至一些中央委员和部长都不知情。讨论只限制在10～15人的圈子里进行”。
  </p>
  <p class="graf graf--p">
   即使当时的驻美大使和联合国代表也不知情，他们一直坚持说苏联在古巴部署的是防御性武器。
  </p>
  <p class="graf graf--p">
   尼克松派基辛格访问中国，连他的国务卿罗杰斯都蒙在鼓里；埃及所有的作战命令在10月5日才传达到营级，而普通士兵是在开战前几个小时才知道这件事的。
  </p>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote">
   <p>
    <em>
     “易其事，革其谋，使人无识” —— 孙子
    </em>
   </p>
  </blockquote>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote">
   <p>
    <em>
     “为了保密，也要欺骗自己的部队” —— Greiffenberg 将军
    </em>
   </p>
  </blockquote>
  <ul class="postList">
   <li class="graf graf--li">
    其次，就是杀死透明度革命
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    任何政府都惧怕透明度革命，因为权力建立在谎言之上；不论其有没有采取战略性欺骗计划
   </strong>
   。
  </p>
  <p class="graf graf--p">
   战略性欺骗是一项系统工程，涉及多个环节，稍不留神就有可能导致欺骗失败。因此除了建立知悉权制度外，欺骗计划的各个环节都必须保密。
  </p>
  <p class="graf graf--p">
   1943年8月，盟军最高统帅部参谋部就成立了三军保密委员会“霸王行动小组”，负责起草有关保密规定。
  </p>
  <p class="graf graf--p">
   据此于当年9月采取了一项被称为 Bigot List 行动的特别程序，据此规定与突袭目标和时间相关的一切文件只能限于一小批官兵中传达。
  </p>
  <p class="graf graf--p">
   1944年春天开始，英国也推行了一系列严格保密措施，政府甚至发起了“乱说会丧命”的宣传运动。
  </p>
  <p class="graf graf--p">
   美国空军第九航空部队的军需官亨利米勒少将因为出言不慎，被降级为中校送回美国，巴顿受到警告。
  </p>
  <p class="graf graf--p">
   在向古巴部署导弹的过程中，赫鲁晓夫要求所有涉及行动的文件
   <strong class="markup--strong markup--p-strong">
    必须手写
   </strong>
   ，如国防部起草的致赫鲁晓夫的报告只有一份，
   <strong class="markup--strong markup--p-strong">
    禁止使用任何无线电传输
   </strong>
   。军队和装备都是在夜间通过火车运输的，其路线和目的地始终保密。沿途禁止发送任何信件和电报。
  </p>
  <p class="graf graf--p">
   一旦抵达目的地，所有官兵都被安置在军事基地里，任何人不准外出，直至登船。他们只知道自己要去遥远的海上参加战略演习，其余什么都不知道。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    保持正常状态，以免被察觉
   </li>
  </ul>
  <p class="graf graf--p">
   ⚠️在进行战略性欺骗的过程中，任何行动都必须出于正常状态，以免引起对手的注意。
  </p>
  <p class="graf graf--p">
   第四次中东战争前夕，萨达特总统照常进行访问，并透露将出席10月纽约举行的联合国大会。9月27日，埃及国防部长邀请所有内阁成员参观，向他们介绍总参谋部的组织和工作情况。
  </p>
  <p class="graf graf--p">

   第四次中东战争爆发后，埃及军队以迅雷不及掩耳之势攻入以色列，突破了以色列精心修筑的巴列夫防线。为扭转局势，以色列军队派出13辆坦克和数辆装甲车作为特遣队，欲潜入苏伊士运河西岸开始反击。
  </p>
  <p class="graf graf--p">
   特遣队队员身着埃及军服，讲一口流利的阿拉伯语，骗过了埃及的守桥士兵，顺利度过浮桥，对西岸的埃及后方发起了猛烈攻击，仅用了一天就摧毁了埃及重要的基础设施。以色列由此从战略被动转为了主动。
  </p>
  <p class="graf graf--p">
   ⚠️军事欺骗中的变形需要利用人力和物力资源，但是，战略性欺骗中的变形只需要动动口舌就可以达成目的。
  </p>
  <p class="graf graf--p">
   如苏德战前，德军大量出现在东线，这很容易使其与对苏战争联系起来，但希特勒表示，德军出现在东线，只是为了瞒过英国的侦查机构，隐蔽其对英战争的准备。
  </p>
  <p class="graf graf--p">
   1956年的苏伊士运河战争期间，在以色列对西奈半岛进行第一次打击后，达扬公开宣称，夺取米特拉通道只是为了实施“报复”，由此给埃及人带来一丝侥幸，从而拖延了埃及的全力反攻。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    迷惑，是指混淆、干扰、阻碍、困扰等行为，降低对于某事的真实本质的确定程度，从而达到隐藏的目的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   迷惑通过整体打乱或部分模糊的方法实现。
  </p>
  <p class="graf graf--p">
   一战期间，英国皇家海军在其战舰上使用Z字形迷惑性图案，以愚弄德国潜艇指挥官。布莱德雷将军让其无线电网络五次模拟巴顿的第三军行动，使德国人相信巴顿要参加阿登战役的救援。
  </p>
  <p class="graf graf--p">
   所有军事密码都属于一种迷惑；隐写术则属于隐形。
  </p>
  <p class="graf graf--p">
   下一篇文章将介绍示假措施。
   <strong class="markup--strong markup--p-strong">
    隐真只是一半工作，如果你让你的对手什么都看不见，对方肯定会心存疑惑，更加希望深挖你的秘密。这就会令你被置于目标的位置上。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️不论是现实中的隐身还是对峙行动中的隐藏，都少不了示假 —— 即 让对手看到错误的情报，并信以为真，以避免对真实情报的兴趣。
  </p>
