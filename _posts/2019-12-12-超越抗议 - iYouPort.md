  <p class="graf graf--p">
   我十四岁的时候参加了第一次抗议活动。那是一场温文尔雅的抗议在我当时居住的马里兰州郊区。
  </p>
  <p class="graf graf--p">
   那是1969年10月15日 — 第一次反越战抗议— 我和大约25个人站在街角，要求结束美国在越南的战争。
  </p>
  <p class="graf graf--p">
   我们宣读了美国战死者的名字。在那场抗议活动中，我是六名高中生参与者的一员。
  </p>
  <p class="graf graf--p">
   其余参与者还包括附近的大学生、当地天主教高中的修女、和几位参与过世界大战的退伍军人。
  </p>
  <p class="graf graf--p">
   大多数乘车经过此地的人都不知道发生了什么，就这么走过去了忽略了我们。一些人向我们展示了支持和平的信号，还有更多的人大声喊叫着给我们起绰号。
  </p>
  <p class="graf graf--p">
   随着岁月的流逝，我参加了许多次抗议活动。
  </p>
  <p class="graf graf--p">
   其中有些抗议是和平的，有些则涉及与警察和右翼抗议者发生的推搡冲突，还有一些涉及相当激烈的战斗，包括投掷石块、铺设路障、催泪瓦斯、警棍和橡皮子弹。
  </p>
  <p class="graf graf--p">
   然而，关于曾经这些抗议活动的有效性问题，如今仍是未知的。
  </p>
  <p class="graf graf--p">
   我真的相信没有任何方法能让完全非暴力是战术问题而不是道德问题。与此同时，对于抵抗执政权力的目标来说，似乎也没有什么比非暴力抗议看起来更简单的了。
  </p>
  <p class="graf graf--p">
   当公民群体考虑如何表达自己对统治阶级施加的某些政策或统治阶级本身的反对态度时，总是有许多因素在起作用。
  </p>
  <p class="graf graf--p">
   正如任何组织者所告诉你的那样，关于如何在特定时刻采取最有效的手段，是一种挑战。即 行动策划。
  </p>
  <p class="graf graf--p">
   策划所涉及的过程包括了解情况、反对意见、以及期望的结果。有时会失败，也有时就能成功。
  </p>
  <p class="graf graf--p">
   换句话说，
   <strong class="markup--strong markup--p-strong">
    有时，作为反对派一起工作的不同个人和团体有希望形成联盟以制定出具有胜算的战略；而其他时候，没有真正的团结存在，难以成功
   </strong>
   。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   这本书中包含组织手册和哲学讨论，McBay 将这些讨论与街头抗议和激进派的历史轶事交织在一起，这些轶事说明了斗争的长期性和关于如何动员的持久讨论。
  </p>
  <p class="graf graf--p">
   它同时呼吁
   <strong class="markup--strong markup--p-strong">
    从自由派行动主义转向直接行动，并邀请自由派活动家超越他们的舒适区，以便有机会阻止即将来临的灾难
   </strong>
   。这本书中引用了丰富的历史抗议故事。
  </p>
  <p class="graf graf--p">
   简而言之，
   <strong class="markup--strong markup--p-strong">
    自由派行动主义不仅被证明是无效的，并且，它实际上反而是加强了它假装反对的制度
   </strong>
   。
  </p>
  <p class="graf graf--p">
   虽然这种观点对于左派激进主义者和革命者来说并不陌生，但对它的重复，再次证明了它的真实性。
  </p>
  <p class="graf graf--p">
   在这个历史时刻，激进派的左翼战略手册似乎非常有用。McBay 的书中写得很清楚，就是为达到这个目的。
  </p>
  <p class="graf graf--p">
   如上所述，
   <strong class="markup--strong markup--p-strong">
    讨论的范围从纯粹实际的组织结构和安全性到关于战术和道德哲学的更宏大的对话
   </strong>
   。
  </p>
  <p class="graf graf--p">
   从文本中可以清楚地看出，McBay 不仅参与了各种激进组织和抗议活动，而且经历过许多他所评论的情况。
  </p>
  <p class="graf graf--p">
   从群体内部的人际关系到具有不同政治和方法的群体之间的关系，这本书在覆盖所有这些方面做得都不错。
  </p>
  <p class="graf graf--p">
   Full Spectrum Resistance 第一章的标题是“争取胜利”。这就是本文的原因和目的：
   <strong class="markup--strong markup--p-strong">
    建立一个致力于在世界上创造革命性社会变革的多维运动。
   </strong>
  </p>
  <p class="graf graf--p">
   大多数关心这类事的人都明白，无论我们多么假装，也无法否认，能做这件事的时间已经越来越短了。
  </p>
  <p class="graf graf--p">
   如果我们要与那些似乎有意摧毁世界以获取利益的人战斗，那么，我们
   <strong class="markup--strong markup--p-strong">
    不仅要战斗，而且必须争取胜利
   </strong>
   。
  </p>
  <p class="graf graf--p">
   要做到这一点，需要适当的工具和愿望。 Aric McBay 的书至少提供了值得参考的一部分。⚪️
