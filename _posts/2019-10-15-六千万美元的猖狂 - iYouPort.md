 <div class="entry-content">
  <ul>
   <li class="graf graf--p">
    <span style="color: #00ccff;">
     <em>
      <strong>
       他们说，如果你没有这串号码，将完全无法生活 — — 维持你生存的一切都需要这串号码。
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   但是目前还没有人看到这串号码，却被它恐吓出的“重要性”驱动着去排队等候注册，提交自己方方面面的一切信息，包括生物识别数据…
  </p>
  <p class="graf graf--p">
   <strong>
    这是全球反乌托邦大跃进的最新一幕 —— 由中国和印度等国家领军 —— 从哪里才能打破政权履带碾压一切的进程？
   </strong>
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5722 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/替代.png?resize=1100%2C586&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/替代.png?w=1400&amp;ssl=1 1400w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/替代.png?resize=300%2C160&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/替代.png?resize=768%2C409&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/替代.png?resize=1024%2C546&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/替代.png?resize=1100%2C586&amp;ssl=1 1100w" data-recalc-dims="1" height="586" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/替代.png?resize=1100%2C586&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   在内罗毕市中心的希尔顿酒店对面，一小群人在正午的阳光下排队，这可是该国异常炎热的四月天。
  </p>
  <p class="graf graf--p">
   穿着品牌黄色背心的注册代理人正忙着工作，
   <strong class="markup--strong markup--p-strong">
    手持法国跨国公司 IDEMIA 提供的平板电脑。
   </strong>
  </p>
  <p class="graf graf--p">
   这一场面意味着肯尼亚政府推出 Huduma Namba 的计划
   <strong class="markup--strong markup--p-strong">
    已经部分成功
   </strong>
   （斯瓦希里语意味“服务号码”）—— ⚠️这是一项雄心勃勃的大规模生物识别数据登记计划，一直饱受争议并面对法律挑战。但为什么还是能推动呢？
  </p>
  <p class="graf graf--p">
   肯尼亚执政党联盟已经将自己打造成了数字治理的先锋，而这项价值 6000万美元的项目 Huduma Namba 就是其最新举措。
  </p>
  <p class="graf graf--p">
   内阁官员 Fred Matiang’i
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/FredMatiangi/status/1105880115794378753" href="https://twitter.com/FredMatiangi/status/1105880115794378753" rel="noopener noreferrer" target="_blank">
    吹嘘
   </a>
   说，Huduma Namba不仅将加强安全、遏制腐败，还将“简化资源分配、改善服务提供”。这些都是1984最常见的BS。
  </p>
  <p class="graf graf--p">
   自从 Huduma Namba 于4月2日开始在全国范围内推出以来，政府官员已经开始
   <strong class="markup--strong markup--p-strong">
    广泛霸占广播、电视、平面媒体和 Twitter 以期说服人们接受
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对媒体的利用是当权者最拿手的（关于如何对付这点参见行动策划《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/1%e3%80%81%e5%88%ab%e8%ae%a9%e5%aa%92%e4%bd%93%e4%b8%ba%e4%bd%a0%e7%9a%84%e5%af%b9%e6%89%8b%e5%b7%a5%e4%bd%9c%ef%bc%9a%e5%88%9b%e9%80%a0%e6%80%a7%e7%a0%b4%e5%9d%8f/" href="https://www.iyouport.org/1%e3%80%81%e5%88%ab%e8%ae%a9%e5%aa%92%e4%bd%93%e4%b8%ba%e4%bd%a0%e7%9a%84%e5%af%b9%e6%89%8b%e5%b7%a5%e4%bd%9c%ef%bc%9a%e5%88%9b%e9%80%a0%e6%80%a7%e7%a0%b4%e5%9d%8f/" rel="noopener noreferrer" target="_blank">
    别让媒体为你的对手工作：创造性破坏
   </a>
   》）
  </p>
  <p class="graf graf--p">
   并且，当局已经在交通繁忙的地区建立了登记站，全国有35,000个登记站在使用。
  </p>
  <p class="graf graf--p">
   国家机构
   <a class="markup--anchor markup--p-anchor" data-href="https://www.standardmedia.co.ke/business/article/2001321603/huduma-namba-government-to-block-sim-cards" href="https://www.standardmedia.co.ke/business/article/2001321603/huduma-namba-government-to-block-sim-cards" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     威胁
    </strong>
    要切断
   </a>
   那些拒绝登记的人获取服务的权利。
  </p>
  <p class="graf graf--p">
   据政府消息来源称，到目前为止已有
   <strong class="markup--strong markup--p-strong">
    超过
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://techweez.com/2019/04/29/kenyans-signed-up-for-huduma-namba/" href="https://techweez.com/2019/04/29/kenyans-signed-up-for-huduma-namba/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     2000万人
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    接受了登记被收集生物识别数据
   </strong>
   。
  </p>
  <p class="graf graf--p">
   渐渐地，
   <strong class="markup--strong markup--p-strong">
    在哄骗和压力之下
   </strong>
   ，肯尼亚人正在排队等着被收集 —— 因为该国官员声称这一号码是“必不可少”的。
  </p>
  <p class="graf graf--p">
   如果证明成功，Huduma Namba 计划将捕获并将数百万肯尼亚人的数据整合到一个单一的数字系统 —— 国家综合身份管理系统中。
  </p>
  <p class="graf graf--p">
   根据总统乌胡鲁·肯雅塔（Uhuru Kenyatta）的说法，该数据库将成为“肯尼亚个人身份的唯一真值来源”。
  </p>
  <p class="graf graf--p">
   每个人都将获得一个独特的身份证号码，比
   <a class="markup--anchor markup--p-anchor" data-href="https://uidai.gov.in/" href="https://uidai.gov.in/" rel="noopener noreferrer" target="_blank">
    印度的 Aadhaar 号码
   </a>
   更厉害。正如一位助理负责人所说，Huduma Namba 将“成为一切号码的前提”。
  </p>
  <p class="graf graf--p">
   但仍有许多肯尼亚人持怀疑态度。在这个用宗教信仰做为道德评判和政治批评的国家里，传言使人们相信 Huduma Namba 是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.kenyans.co.ke/news/38448-uhuru-blasts-pastors-linking-huduma-namba-devils-666" href="https://www.kenyans.co.ke/news/38448-uhuru-blasts-pastors-linking-huduma-namba-devils-666" rel="noopener noreferrer" target="_blank">
    魔鬼的标志
   </a>
   。
  </p>
  <p class="graf graf--p">
   借助本土文化抵制官方的政治宣传是很棒的策略（相关解释详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/1%e3%80%81%e4%ba%86%e8%a7%a3%e4%bd%a0%e7%9a%84%e5%9c%b0%e5%bd%a2%e6%98%af%e7%ad%96%e5%88%92%e8%a1%8c%e5%8a%a8%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" href="https://www.iyouport.org/1%e3%80%81%e4%ba%86%e8%a7%a3%e4%bd%a0%e7%9a%84%e5%9c%b0%e5%bd%a2%e6%98%af%e7%ad%96%e5%88%92%e8%a1%8c%e5%8a%a8%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" rel="noopener noreferrer" target="_blank">
    了解你的文化地形是策划行动的第一步
   </a>
   》）
  </p>
  <p class="graf graf--p">
   非营利组织肯尼亚人权委员会（KHRC）
   <a class="markup--anchor markup--p-anchor" data-href="https://www.standardmedia.co.ke/article/2001319596/kenya-human-rights-commission-asks-kenyans-to-boycott-huduma-namba" href="https://www.standardmedia.co.ke/article/2001319596/kenya-human-rights-commission-asks-kenyans-to-boycott-huduma-namba" rel="noopener noreferrer" target="_blank">
    呼吁肯尼亚人抵制
   </a>
   这项倡议，而议会和法院则威胁要彻底破坏这一倡议。
  </p>
  <p class="graf graf--p">
   在Twitter上，政府咄咄逼人的 Huduma Namba 宣传已成为人们嘲笑的对象。模拟干旱诅咒的一位 Twitter 用户评论说：“政府居然没说如果拒绝 Huduma Namba 就不会下雨。”
  </p>
  <p class="graf graf--p">
   然而，肯尼亚官员仍在厚着脸皮坚持下去。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    Huduma Namba 是全球反乌托邦大跃进趋势中的最新动态 —— 这一趋势参见中国和印度等国家 —— 即 通过大数据实现彻底的1984
   </strong>
   。
  </p>
  <p class="graf graf--p">
   国家代表将 Huduma Namba 描述为收集、验证和数字化肯尼亚人生物数据的中立且有效的手段，可以替代所有其他注册号码的“一个数字万能钥匙”。
  </p>
  <p class="graf graf--p">
   解释一下这句话就是：该号码关联你的一切，你再也没可能在任何方面实现匿名和化名。
  </p>
  <p class="graf graf--p">
   尽管对其声称的好处感到困惑，但注册过程相对简单。任何六岁以上的人都可以注册。
  </p>
  <p class="graf graf--p">
   登记的人必须至少携带一张政府签发的身份证件 —— 无论是出生证、驾驶执照、国民身份证、护照、还是侨民卡。
  </p>
  <p class="graf graf--p">
   登记代理人协助人们填写两页的表格，其中
   <strong class="markup--strong markup--p-strong">
    要求提供一系列个人信息，包括出生详情、婚姻状况、地址、教育程度、和土地所有权，然后扫描到兼作相机和指纹扫描仪的平板电脑上
   </strong>
   。
  </p>
  <p class="graf graf--p">
   （去年底听闻中国部分地区出现了新的“人口普查”，来自北京消息人士称甚至被登记了工作地点、上班使用的交通工具等等信息，并且，这次登记使用的是电子设备，而非纸质表格。
   <strong class="markup--strong markup--p-strong">
    应该格外提起警惕，因为，还记得当年在大屠杀之前纳粹做过什么吗？详见《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E7%BA%B3%E7%B2%B9%E5%92%8C%E4%BD%A0/" href="https://www.iyouport.org/%E7%BA%B3%E7%B2%B9%E5%92%8C%E4%BD%A0/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     纳粹和你
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》
   </strong>
   ）
  </p>
  <p class="graf graf--p">
   在注册过程结束时，注册人会收到一张长条纸，让他们在未来某个未公开的日期申请他们的 Huduma Namba。
  </p>
  <p class="graf graf--p">
   很多人都
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/davidmakali1/status/1118936399489708034" href="https://twitter.com/davidmakali1/status/1118936399489708034" rel="noopener noreferrer" target="_blank">
    想知道
   </a>
   他们最终会如何以及何时才能收到那串难以捉摸的号码。
  </p>
  <p>
   还记得他吗👇？《血族》的主演，他的手臂上被纹了一串数字，A230385，那是奥斯威辛集中營的标志。那是IBM提供给纳粹的技术。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    十年禁令
   </strong>
  </p>
  <p class="graf graf--p">
   对效率和怀疑主义的担忧并不是围绕该计划的唯一问题。
  </p>
  <p class="graf graf--p">
   而且，⚠️法国跨国公司负责收集肯尼亚人的数据，这非常令人不安。
  </p>
  <p class="graf graf--p">
   IDEMIA，以前称为 OT-Morpho，在2017年的选举中首次引发争议，因为该公司被迫
   <a class="markup--anchor markup--p-anchor" data-href="https://techtrendske.co.ke/french-firm-ot-morpho-says-iebc/" href="https://techtrendske.co.ke/french-firm-ot-morpho-says-iebc/" rel="noopener noreferrer" target="_blank">
    否认
   </a>
   其生物识别选民身份登记系统被黑客攻击。
  </p>
  <p class="graf graf--p">
   反对派领导人指责该公司提供选民身份识别和结果传输工具包，指控其不当行为 —— 这是 IDEMIA 强烈否认的指控。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    有令人担忧的严重理由
   </strong>
   。隐私国际和知识产权与信息技术法中心（CIPIT）最近的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cipit.org/images/downloads/CIPIT-Elections-and-Biometrics-Report.pdf" href="https://www.cipit.org/images/downloads/CIPIT-Elections-and-Biometrics-Report.pdf" rel="noopener noreferrer" target="_blank">
    一份报告
   </a>
   显示，
   <strong class="markup--strong markup--p-strong">
    选民数据在选举前可供出售
   </strong>
   。就在此前一周，肯尼亚议会
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nation.co.ke/news/MPs-vote-block-Huduma-Namba-firm/1056-5086136-i81dnz/" href="https://www.nation.co.ke/news/MPs-vote-block-Huduma-Namba-firm/1056-5086136-i81dnz/" rel="noopener noreferrer" target="_blank">
    投票禁止 IDEMIA
   </a>
   在该国开展业务至少十年。
  </p>
  <p class="graf graf--p">
   但是，禁令并没有阻止政府。政府宣布，尽管议会将 IDEMIA 列入黑名单，Huduma Namba 仍将继续。
  </p>
  <p class="graf graf--p">
   尽管如此，这次投票还是对 IDEMIA 被授予政府招标的秘密程序进行了更严格的审查。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    谁的真值？
   </strong>
  </p>
  <p class="graf graf--p">
   几十年来，民事登记一直是争论谁是和谁不是“肯尼亚人”的主要场所。
  </p>
  <p class="graf graf--p">
   如果在肯尼亚没有合法身份，个人就不能自由行动、无法正式工作、不能开设银行账户、不能注册SIM卡、也不能访问许多其他私人和公共服务。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    不论是公民还是难民
   </strong>
   经常会被迫使用非法手段获取文件。
  </p>
  <p class="graf graf--p">
   在过去的七年里，穆罕默德（化名）一直在徒劳地试图获得令人垂涎的国民身份证。尽管他提供了能证明自己公民身份的文件，但他仍被怀疑为印第安人和穆斯林。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “今天他们说你是外国人，明天他们就说你是青年党”
   </strong>
   ，穆罕默德说（青年党是支持本拉登和基地组织的恐怖组织）。
  </p>
  <p class="graf graf--p">
   一名登记官拒绝了他的申请，神奇的理由是“许多难民都叫穆罕默德”，其他官员则向他索取贿赂。
  </p>
  <p class="graf graf--p">
   人权组织担心，Huduma Namba 进程只会将这些现有的歧视形式数字化。
  </p>
  <p class="graf graf--p">
   民权倡导者指出，不仅是无国籍和被边缘化的人应该警惕 Huduma Namba。
   <strong class="markup--strong markup--p-strong">
    任何人都应该警惕，因为肯尼亚尚未颁布任何数据保护立法
   </strong>
   。
  </p>
  <p class="graf graf--p">
   因此，政府正在“在没有任何真正的法律监督的情况下”收集信息。
  </p>
  <p class="graf graf--p">
   矛盾和不一致的政府声明进一步加剧了公众的不知情。
  </p>
  <p class="graf graf--p">
   虽然 IDEMIA 收集了数据，但ICT部声称“国家信息集成管理系统（NIIMS）的核心注册软件是由当地软件工程师设计和集成的。”
  </p>
  <p class="graf graf--p">
   4月初，Matiang’i 出现在一个受欢迎的穆斯林广播节目 Iqra FM 上，以向听众保证无证人员可以注册。尽管如此，民权团体报告说，那些缺乏身份证件的人已经被登记代理人拒之门外。没有身份证的穆罕默德连试一下的欲望都没有了。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5724 jetpack-lazy-image" data-lazy-sizes="(max-width: 706px) 100vw, 706px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/货币化数据.png?resize=706%2C712&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/货币化数据.png?w=706&amp;ssl=1 706w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/货币化数据.png?resize=150%2C150&amp;ssl=1 150w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/货币化数据.png?resize=297%2C300&amp;ssl=1 297w" data-recalc-dims="1" height="712" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/货币化数据.png?resize=706%2C712&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="706"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5724" data-recalc-dims="1" height="712" sizes="(max-width: 706px) 100vw, 706px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/货币化数据.png?resize=706%2C712&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/货币化数据.png?w=706&amp;ssl=1 706w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/货币化数据.png?resize=150%2C150&amp;ssl=1 150w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/06/货币化数据.png?resize=297%2C300&amp;ssl=1 297w" width="706"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    货币化数据
   </strong>
  </p>
  <p class="graf graf--p">
   人们纷纷猜测政府究竟想干什么。
  </p>
  <p class="graf graf--p">
   在任何国家都一样，当然包括肯尼亚，数据是一种宝贵的政治和经济商品，
   <strong class="markup--strong markup--p-strong">
    肯尼亚人口数据库的逐步数字化对肯尼亚最大的电信公司 Safaricom 等移动运营商来说是一个庞大的福利
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果用作人口普查、税务登记或公民身份证明，Huduma Namba 可能会影响肯尼亚宪法下的资源分配，扩大税基，或影响选民登记。
  </p>
  <p class="graf graf--p">
   政府批评者大卫·恩迪（David Ndii）指责执政党使用 Huduma Namba 推销与总统家族有关的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theeastafricanreview.info/op-eds/2019/04/02/crony-capitalism-and-state-capture-2-documents-reveal-the-kenyatta-familys-plans-to-take-over-lending-to-smes/" href="https://www.theeastafricanreview.info/op-eds/2019/04/02/crony-capitalism-and-state-capture-2-documents-reveal-the-kenyatta-familys-plans-to-take-over-lending-to-smes/" rel="noopener noreferrer" target="_blank">
    新手机借贷服务
   </a>
   。他指出
   <strong class="markup--strong markup--p-strong">
    这一登记是大规模监视系统的构建，其背后隐藏着商业利益。
   </strong>
  </p>
  <p class="graf graf--p">
   许多肯尼亚人同样怀疑政府的意图，这一点不足为奇。
  </p>
  <p class="graf graf--p">
   肯尼亚人聚集在内罗毕的乌胡鲁公园，参加
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/Maskani254/status/1122013589018165248" href="https://twitter.com/Maskani254/status/1122013589018165248" rel="noopener noreferrer" target="_blank">
    抗议
   </a>
   活动抵制政府贪污，其中就包括抵制 Huduma Namba 的活动。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    试验场
   </strong>
  </p>
  <p class="graf graf--p">
   许多批评者认为 Huduma Namba 是另一个白象计划。但
   <strong class="markup--strong markup--p-strong">
    该项目还涉及数据收集和生物识别技术日益广泛的巨大危险。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️许多技术和金融公司将全球南方国家视为“监视技术创新的试验台”。
  </p>
  <p class="graf graf--p">
   后殖民地非洲国家和其他所谓的“低权利环境”特别容易受到侵入性、不透明和不民主的数据收集方法的影响。
  </p>
  <p class="graf graf--p">
   ⚠️在西方，剑桥分析公司可能因其在英国脱欧和2016年美国大选中所扮演的心理操纵角色而臭名昭著，但是，很少有人知道，
   <strong class="markup--strong markup--p-strong">
    剑桥分析公司是首先在尼日利亚、印度和肯尼亚等国家磨练其算法的，这些国家没有隐私保护法，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://nymag.com/intelligencer/2018/03/what-was-cambridge-analytica-doing-in-developing-countries.html" href="https://nymag.com/intelligencer/2018/03/what-was-cambridge-analytica-doing-in-developing-countries.html" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     不会阻止这种大规模的数据收集
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
  </p>
  <p class="graf graf--p">
   更多在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
    心理操纵的秘密：”行为经济学”+PR=？》
   </a>
  </p>
  <p class="graf graf--p">
   非洲国家是监视资本主义巨头的磨刀石。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E9%9D%9E%E6%B4%B2%E4%BA%BA%E7%9A%84%E8%84%B8%E8%A2%AB%E7%94%A8%E6%9D%A5%E8%AE%AD%E7%BB%83%E4%B8%AD%E5%9B%BD%E8%80%81%E5%A4%A7%E5%93%A5%EF%BC%9A%E5%8C%97%E4%BA%AC%E6%AD%A3%E5%9C%A8%E5%8A%AA%E5%8A%9B/" href="https://www.iyouport.org/%E9%9D%9E%E6%B4%B2%E4%BA%BA%E7%9A%84%E8%84%B8%E8%A2%AB%E7%94%A8%E6%9D%A5%E8%AE%AD%E7%BB%83%E4%B8%AD%E5%9B%BD%E8%80%81%E5%A4%A7%E5%93%A5%EF%BC%9A%E5%8C%97%E4%BA%AC%E6%AD%A3%E5%9C%A8%E5%8A%AA%E5%8A%9B/" rel="noopener noreferrer" target="_blank">
    中国也在其中
   </a>
   磨刀。
  </p>
  <p class="graf graf--p">
   然而，肯尼亚人不是技术的被动接受者。由于法律宣传和人权团体的广泛反对，肯尼亚高等法院已经对登记工作施加了遏制。
  </p>
  <p class="graf graf--p">
   法院警告政府，不能将政府服务与 Huduma Namba 联系起来，不能规定登记截止日期或强迫任何人登记。法院还禁止收集DNA或GPS数据、并禁止与外界共享数据。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但是，案件的最终裁决仍未作出。与此同时，肯尼亚官员已经显示出藐视法庭裁决的迹象
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Nubian Rights Forum，NAMATI 和 Haki na Sheria 等公民权利和法律权益组织继续挑战 Huduma Namba 的合宪性，并推动提高透明度。
  </p>
  <p class="graf graf--p">
   最终，公民不服从可能是才最强大的抗议形式 —— 让更多肯尼亚人拒绝在登记站排队。⚪️
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://codastory.com/authoritarian-tech/kenya-biometric-project-shrouded-in-secrecy/" href="https://codastory.com/authoritarian-tech/kenya-biometric-project-shrouded-in-secrecy/" rel="noopener noreferrer" target="_blank">
    A $60 million government initiative to collect the data of millions in an effort to enhance services has been marked by criticism about its inefficiency and allegations of misconduct
   </a>
  </p>
