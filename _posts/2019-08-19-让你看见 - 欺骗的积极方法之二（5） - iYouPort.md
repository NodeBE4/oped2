   <li class="graf graf--p">
    <span style="color: #00ccff;">
     <em>
      <strong class="markup--strong markup--p-strong">
       隐真只是一半工作，如果你让你的对手什么都看不见，对方肯定会心存疑惑，更加希望深挖你的秘密。这就会令你被置于目标的位置上。
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   ⚠️不论是现实中的隐身还是对峙行动中的隐藏，都少不了示假 — — 即 让对手看到错误的情报，并信以为真，以避免对真实情报的兴趣。
  </p>

  <p class="graf graf--p">
   拒止的根本目的是隐真，但随着现代侦查技术的发展，以拒止方法隐藏行为意图以欺骗对手的难度越来越大。
  </p>
  <p class="graf graf--p">
   孙子兵法说“形人而我无形”。⚠️然而，
   <strong class="markup--strong markup--p-strong">
    “无形”往往会让对手认为是山雨欲来前的平静，反而更加提高了警惕、积极实施侦查
   </strong>
   。
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    在隐瞒真相的同时还必须通过“示形”的方法，展示虚假的、误导性的、甚至是真实的但经过剪裁的“半真相”，来吸引、转移欺骗对象的注意力，影响其侦查和战略决策的分析判断
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “示形”也是孙子“战胜策”中最常见的形式。孙子兵法的“十二诡道”都是示形。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    美军曾把信息战分为战略信息战和战场信息战两大类
   </strong>
   ，其中战略信息战包含：心理战、情报战、宣传战、战场伪装、指挥控制战，其中的战场伪装就是“示形”；其他几种作战样式也都离不开示形。
  </p>
  <p class="graf graf--p">
   前苏军总参谋长洛博夫大将的《战争中的谋略》一书将军事谋略定义为：隐蔽性和迷惑敌人的理论与实践，也是强调“示形”和用诈。
  </p>
  <p class="graf graf--p">
   隐形和示形的侧重点不同，功能也不一样。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    隐形代表的是战略性欺骗的消极方面，示形代表的是战略性欺骗的积极方面
   </strong>
   ；隐形的目的千篇一律，即 不让真相被对手发现，而示形则明确规定对手能看到什么、听到什么。
  </p>
  <p class="graf graf--p">
   隐形的宗旨是隐藏真相，而示形则是要展示假象。
  </p>
  <p class="graf graf--p">
   ⚠️隐形的方法带有强烈的技术色彩，示形则需要使用政治、外交、战术、技术等多种手段的综合。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    隐形可能不会引起对手的反应，而示形明确要求影响对手的行为
   </strong>
   。
  </p>
  <p class="graf graf--p">
   推荐一本书《The Deceivers: Allied Military Deception in the Second World War》, Thaddeus Holt
  </p>
  <p class="graf graf--p">
   因此，💡示形的目的更明确，手段更积极，目的与手段的联系更加直接。
  </p>
  <p class="graf graf--p">
   尽管如此，隐形和示形殊途同归，目的都是欺骗对手的视听，扰乱对手的思维，促使对手作出错误的反应。⚠️
   <strong class="markup--strong markup--p-strong">
    只有将隐形和示形结合起来，才能使战略性欺骗的效果最大化。
   </strong>
  </p>
  <p class="graf graf--p">
   我们提供过的《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" href="https://www.iyouport.org/%e5%88%86%e8%a3%82%e4%ba%ba%e6%a0%bc%e7%9a%84%e9%ad%85%e5%8a%9b/" rel="noopener noreferrer" target="_blank">
    分裂人格的魅力
   </a>
   》《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%a7%92%e8%89%b2%e6%89%ae%e6%bc%94%ef%bc%9a%e8%a1%8c%e5%8a%a8%e8%80%85%e5%88%9a%e9%9c%80video/" href="https://www.iyouport.org/%e8%a7%92%e8%89%b2%e6%89%ae%e6%bc%94%ef%bc%9a%e8%a1%8c%e5%8a%a8%e8%80%85%e5%88%9a%e9%9c%80video/" rel="noopener noreferrer" target="_blank">
    角色扮演
   </a>
   》《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E6%9F%90%E4%BA%9B%E8%A1%8C%E5%8A%A8%E8%80%85%EF%BC%8C%E4%BD%A0%E9%9C%80%E8%A6%81%E4%B8%80%E4%B8%AA%E8%A2%9C%E5%AD%90%E6%9C%A8%E5%81%B6/" href="https://www.iyouport.org/%E6%9F%90%E4%BA%9B%E8%A1%8C%E5%8A%A8%E8%80%85%EF%BC%8C%E4%BD%A0%E9%9C%80%E8%A6%81%E4%B8%80%E4%B8%AA%E8%A2%9C%E5%AD%90%E6%9C%A8%E5%81%B6/" rel="noopener noreferrer" target="_blank">
    袜子木偶
   </a>
   》都属于示形的一种；而《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89/" href="https://www.iyouport.org/%e8%87%aa%e6%88%91%e4%ba%ba%e8%82%89/" rel="noopener noreferrer" target="_blank">
    自我人肉
   </a>
   》则是隐形的第一步基础工作。
  </p>
  <p class="graf graf--p">
   真正的对峙中要比这更加负责和多样。
  </p>
  <p class="graf graf--p">
   制造假象的措施大致有三种，即 模仿 mimicking、无中生有 inventing、和设置假目标 decoying。在作战中它们被统称为模拟 simulation。它们都是主动的欺骗策略。
  </p>
  <p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、模仿
   </strong>
  </p>
  <p class="graf graf--p">
   模仿就是用一种事物复制另一种事物，复制多个方面，以此来展示虚假的事物。
   <strong class="markup--strong markup--p-strong">
    需要复制某一事物的多个特点，以无限接近被复制事物的类型
   </strong>
   。
  </p>
  <p class="graf graf--p">
   与机器一样，人类判断一件事物的时候也是通过抓取其各种特点来定位的，于是复制者需要能精准捕捉这些特点，⚠️准确说是你的对手做判断时所基于的特点，才能做到以假乱真。
  </p>
  <p class="graf graf--p">
   1941年，艾米希将军仅用了六个旅、区区两万人的部队就攻下了比利时的列日要塞，因为他的部队是从五个军团中抽出士兵组成的混合部队。比利时情报部门从俘虏的情况中分析，便以为他们面对的是五个军团的力量，也就是超过15万人，于是他们撤离了步兵。
  </p>
  <p class="graf graf--p">
   在“卫士”计划下，盟军派出了一名中尉冒充蒙哥马利前往直布罗陀和阿尔及尔视察。这名中尉不仅在相貌、体态、举止方面，都与蒙哥马利极其相似，而且，经过特殊训练后，对蒙哥马利的
   <strong class="markup--strong markup--p-strong">
    生活习惯、性格脾气、神态气韵、说话语气，掌握得非常精巧。
   </strong>
  </p>
  <p class="graf graf--p">
   英国情报机构还故意放出消息，称蒙哥马利此行旨在编组英美联军，准备在法国南部海岸登陆。
  </p>
  <p class="graf graf--p">
   这些举措果然骗过了训练有素的德国间谍，连蒙哥马利的密友 —— 直布罗陀总督沙拉尔将军，也信以为真了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、设置假目标
   </strong>
  </p>
  <p class="graf graf--p">
   模仿是模仿一个已存在的事物，设置假目标则是通过呈现另外一个事物来展示虚假的东西。
  </p>
  <p class="graf graf--p">
   它要创造一个或多个特点，形成一个全新的类型来实现。
  </p>
  <p class="graf graf--p">
   就如魔术师通过创造假人来展示一些“替代”效应，将军们会建造一些橡胶坦克、木枪、帆布飞机，以及虚假的无线电传输，来操纵对手的认知；海军将领也会制造一些假的战舰。
  </p>
  <p class="graf graf--p">
   1901年，美军范斯顿上校让他的士兵穿上菲律宾土著摩洛哥人的服装，抓获了其领导人吉纳尔多。
  </p>
  <p class="graf graf--p">
   ⚠️欧洲的游击队曾很多使用过这种方法。
   <strong class="markup--strong markup--p-strong">
    此举在细节方面应特别小心，因为你的对手具备很高的反渗透能力，他们懂得如何在细节特点中挖掘一个人的真实面目。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️美国战略情报局的外勤特工讲过一个故事，当时被派往法国的间谍要学会将香烟一直吸到烟蒂处。有一次，一名特工扔掉了吸了一半的香烟，结果因此被盖世太保抓捕。因为在香烟紧缺的法国，节俭的普通人是绝对不会这样做的。
  </p>
  <p class="graf graf--p">
   二战期间英军在中东地区的“A部队”就是一支专门从事模拟的专业部队。美军于1943年成立了第23特别大队，共有82名军官和1023名士兵，所使用的“武器”只是一些橡皮坦克、橡皮大炮、橡皮军车，不断变换的肩章符号和扩音设备。
  </p>
  <p class="graf graf--p">
   1944年和45年，这个大队先后*扮演*了第五装甲师、第四步兵师、第六装甲师等部队，由于模仿过于逼真，不仅迷惑了德军，连友军都信以为真了。
  </p>
  <p class="graf graf--p">
   在卢森堡南部，他们装扮美军装甲师，白天摆出几辆假坦克让德国人侦查，夜间播放坦克、车辆开动和人员喧闹的
   <strong class="markup--strong markup--p-strong">
    音响
   </strong>
   ，牵制德军一个师长达七天之久。
  </p>
  <p class="graf graf--p">
   在德黑兰会议上，斯大林向丘吉尔介绍苏联军事欺骗的经验，其中特别提到了设置假目标。为掩护从左翼向德军发动进攻的第20集军团，苏军在右翼集结了大量假坦克和假火炮，并用喇叭大放坦克马达的声音。为了将德军轰炸机引开，苏军使用了大量的伪装，引来了上千次德国空军轰炸。
  </p>
  <p class="graf graf--p">
   ⚠️使用假目标实施欺骗的要求非常严格。1、在配置上必须合乎技术要求；2、外貌要逼真，在高科技侦查的背景下尤其如此；3、道具必须像真实目标一样有活动的征候，让对手在视觉、听觉和仪器探测上都找不到破绽。
  </p>
  <p class="graf graf--p">
   和人物伪装一样，战略伪装一方面可以掩盖假目标的某些缺陷，另方面也合乎人的心理，太明显的目标反而容易引起怀疑。
  </p>
  <p class="graf graf--p">
   在科索沃战争中，南联盟军就是采取的伪装欺骗以保存实力的。结果是，南联盟军损失非常小，而北约空袭的效果很差，出乎各方面的意料。
   <strong class="markup--strong markup--p-strong">
    只因为他们的假目标足够逼真，骗了北约连续数天的全力轰炸
   </strong>
   。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-3972 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-6.jpg?resize=1100%2C825&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-6.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-6.jpg?resize=300%2C225&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-6.jpg?resize=768%2C576&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-6.jpg?resize=1024%2C768&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-6.jpg?resize=1100%2C825&amp;ssl=1 1100w" data-recalc-dims="1" height="825" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-6.jpg?resize=1100%2C825&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、牵制（佯动）
   </strong>
  </p>
  <p class="graf graf--p">
   牵制指的是在准备阶段同时在几个不同的方向上部署部队。对欺骗对象进行虚张声势的进攻，以逐渐麻痹对手，使其放松警惕，隐蔽己方的真实战略意图。
  </p>
  <p class="graf graf--p">
   ⚠️牵制的目的不是消灭对手，而是吸引对手，拖住对手，减轻对手对正面作战的压力，使对手处处薄弱，难以形成集中攻势。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    牵制的一个重要特征就是，以少牵多。就像打地鼠游戏，其实真实的“老鼠”并不在那些洞里
   </strong>
   。
  </p>
  <p class="graf graf--p">
   希特勒为了掩护巴巴罗萨计划采用的就是牵制欺骗行动。
  </p>
  <p class="graf graf--p">
   希特勒把“海狮”这一真正的作战行动变成了欺骗行动的序幕，他精心准备的欺骗行动还有“鱼叉计划”、“鲨鱼计划”、对英国大规模轰炸等；随机应变的欺骗行动有“巴尔干战争”、“默丘利”作战计划等，还有一些其他的计划，都在一定程度上
   <strong class="markup--strong markup--p-strong">
    掩饰了巴巴罗萨的实施，斯大林陷入了这一圈套
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️有时佯动是为了给对手制造一个“狼来了”的印象，不断继续，对方就会变得麻木，等到真正的攻击到来时仍一头雾水。
  </p>
  <p class="graf graf--p">
   第四次中东战争中埃及就采用了这一方法。他们每年秋天都进行一场为期一周的大型军演，而且规模一次比一次大。以色列人一开始对这些演习很紧张，之后开始警戒，再之后就变见怪不怪了。直到1973年埃及再次“演习”，而就在演习期间突然爆发了真正的战争。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、佯攻
   </strong>
  </p>
  <p class="graf graf--p">
   佯动可以转移对手的注意力，但由于它不与对手接触，缺乏进攻的真实感，容易被对手识破。
  </p>
  <p class="graf graf--p">
   ⚠️为使佯动具有可信性，就需要佯攻，这也是一种牵制性的手段。
  </p>
  <p class="graf graf--p">
   佯攻具有进攻的性质，需要与对手接触，从而使对手认为这是一场真正的进攻。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该计划应该符合整个作战目的，误导对手的反应。佯攻可以使对手错误地投入兵力，使其力量远离主攻方向，或者牵制对手的兵力，使其难以动弹
   </strong>
   。
  </p>
  <p class="graf graf--p">
   利用佯攻需要因时而异，灵活机智，变幻莫测。⚠️时机成熟时佯攻就可以变为主攻，以达到出其不意的目的。
  </p>
  <p class="graf graf--p">
   有时也可以反过来用，诱使对手误认为主攻才是佯攻，于是不敢轻易动用主力部队，从而陷入被动。
  </p>
  <p class="graf graf--p">
   1950年仁川登陆前，美军就使用了佯攻。美军在群山港附近的荒岛上举行登陆演习，同时派出联合突击队对群山港进行军事侦察，并对朝鲜东海岸和西海岸实施火力准备，英国轻型航空母舰和美国重巡洋舰攻击了平壤外港镇和达阳岛，以造成主要登陆点在群山地区的假象。
  </p>
  <p class="graf graf--p">
   这些欺骗行动都成功隐藏了仁川登陆的计划，牵制了朝鲜的精锐部队，并进一步削弱了北方的防御力量。
  </p>
  <p class="graf graf--p">
   苏联是佯攻欺骗的大师。1941年苏军最高统帅部在斯大林格勒执行“天王星计划”的同时，在莫斯科西北方向展开了具有牵制性的“火星战役”。
  </p>
  <p class="graf graf--p">
   在发起库尔斯克战役之前，苏军最高统帅部发布命令，一旦战争打响，相邻方向的各个方面都要发起进攻。最高统帅部命令崔可夫的62军团和库兹涅佐夫上将的近卫第一集团军利用德军主力被牵制在库尔斯克的机会，向顿巴斯发动进攻。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    通过实实在在的进攻使欺骗显得非常逼真
   </strong>
   ，这不同于盟军诺曼底登陆前的欺骗，这与苏联拥有大批人力资源密切相关。⚠️由于助攻和主攻拥有同样的实力，因此主从转换就格外容易了，令对手很难摸清主攻方向。
  </p>
  <p class="graf graf--p">
   为了掩护古巴部署进攻性武器的行动，苏联在柏林问题上向美国施加巨大压力。美苏在柏林对峙，战争几乎一触即发。赫鲁晓夫一再表示，在柏林问题上“美国已经拉开了战争序幕”。
  </p>
  <p class="graf graf--p">
   由于柏林局势紧张，美国政府的高级官员大多对苏联在古巴的军事行动不以为然，以为欧洲才是美苏对抗的“重点”。实际上柏林紧张局势只是赫鲁晓夫的障眼法，为转移美军注意力而设计的。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-3973 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-7.jpg?resize=1100%2C723&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-7.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-7.jpg?resize=300%2C197&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-7.jpg?resize=768%2C505&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-7.jpg?resize=1024%2C673&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-7.jpg?resize=1100%2C723&amp;ssl=1 1100w" data-recalc-dims="1" height="723" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/欺骗-7.jpg?resize=1100%2C723&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、假情报
   </strong>
  </p>
  <p class="graf graf--p">
   也就是通过各种手段发布假消息，以掩盖自己的战略企图，混淆视听，误导对手的决策。
  </p>
  <p class="graf graf--p">
   假情报这个词是第一次世界大战期间德国总参谋部采取的。20年代，苏联的契卡也采取了这一术语，自此一直为苏联国家安全部门和军事情报部门所使用。
  </p>
  <p class="graf graf--p">
   由于苏俄情报机构对这一概念运用得非常广泛，以致于美国官方直接将其翻译成欺骗。但是俄国人非常谨慎地
   <strong class="markup--strong markup--p-strong">
    将它与形态上的伪装区别开
   </strong>
   。
  </p>
  <p class="graf graf--p">
   假情报的载体有多种，作战地图、作战计划、命令、外交信函、会议记录、胶片、磁盘等存储器、甚至战俘，都可以用来传递假情报。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    随着科学技术的进步和新闻业的发展，新闻欺骗战也已经成为重要的战略性欺骗手段
   </strong>
   。通过新闻散步布假情报，不仅瓦解士气、制造混乱和矛盾，还可以迫使对手作出错误的判断。
  </p>
  <p class="graf graf--p">
   我们将在未来详细介绍渗透者的操作方式，其中就包括这点，虽然是辅助手段这一。
  </p>
  <p class="graf graf--p">
   早期的假情报活动带有浓厚的随机性，但是假情报的发布均由情报部门控制。比如二战时期的德军谍报局、伦敦监督处等，克格勃第一总局下属还有一个专门的假情报部门，以集中制造和散布假消息。
  </p>
  <p class="graf graf--p">
   ⚠️假情报往往很好用。由于对手的决心来自于他们掌握的情报，于是向对手传递假情报就成了诱使对手下定错误决心的极为重要的手段。
  </p>
  <p class="graf graf--p">
   主动提供现成的情报，既是假情报这种欺骗方法的基本特征，也是它的隐形法和示形法的主要区别。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    运用隐形法和示形法实施欺骗时，欺骗者向被骗者提供了*现象*，情况判断的结论是由被骗者做出的；而假情报提供给被骗者的不是现象，而是*结论*，
   </strong>
   其主要形式是以文字、图表、音像材料为载体的假情报。
  </p>
  <p class="graf graf--p">
   当然，情报形态并不是假情报方法的唯一标示。
  </p>
  <p class="graf graf--p">
   比如，把战俘关押在隔壁房间，使之偷听到欺骗者关于下一次行动的谈话，然后给战俘一个机会让他逃脱，把偷听到的内容报告给对手。尽管战俘没有带走文字形态的情报，但这种方法也属于假情报范畴。
  </p>
  <p class="graf graf--p">
   1943年，盟军计划在西西里岛登陆，为转移希特勒的视线，盟军实施了代号为“肉馅行动 operation mincement”的欺骗行动。
  </p>
  <p class="graf graf--p">
   一具尸体经过精心的伪造变成了所谓的“马丁上校”，尸体携带的消息是：“西西里岛只是一次佯攻，目的是为进攻撒丁岛和希腊做掩护”。德国人信以为真了，放松了对西西里岛的防备。
  </p>
  <p class="graf graf--p">
   还有一个伟大布局的假情报案例（想想看，
   <strong class="markup--strong markup--p-strong">
    主流媒体传统规则中的“两个及以上消息源确认”，是多么的无能和愚蠢
   </strong>
   ）
  </p>
  <p class="graf graf--p">
   1944年的“卫士”计划中，盟国欺骗机构在瑞典散布假消息，说，“英美的军事工程师正在调查瑞典火车和路基的负载压力，这些调查与盟军计划从挪威向波罗的海运送装甲部队有关”；“英国空军在视察瑞典军用机场的跑道和停机坪”；“西方国家正在为盟军就德国波罗的海海岸对面的哥得兰、厄兰德等地的过境权和港口设施使用权问题进行谈判”……等等。
  </p>
  <p class="graf graf--p">
   甚至有人劝说瑞典公民修筑防空掩体、储备食物和燃料。也就是刺激恐慌的心理战，越多人相信欺骗效果就越好。
  </p>
  <p class="graf graf--p">
   就在谣言越来越多的时候，瑞典的证券交易所又传来了新的信息：伦敦和纽约股票投机商们正在大量买进长期以来一直不景气的斯堪的纳维亚股票，这就使谣言变得更加可信了。
  </p>
  <p class="graf graf--p">
   这场精心策划的骗局使希特勒不仅没有将驻守在挪威的德军调往法国，而且还为了对付盟军“即将到来的入侵”加强了在那里的兵力。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    假情报的第二种表现形式是：破坏对手的威信、中伤其信誉、影响其与盟友的关系 —— 这些方面在警方对反对派的渗透中也往往是主要被采用的伎俩之一
   </strong>
   ，我们会在将来专门讲述关于渗透操控的手段。
  </p>
  <p class="graf graf--p">
   对于假情报来说，比如冷战时期，苏联情报机构利用著名作家 Mark Lane 编造肯尼迪被暗杀的理论，利用中情局的背叛者 Philip Agee 诋毁中情局，散步关于胡佛是同性恋的消息，并诋毁马丁路德金，说他是“政府的线人”。
  </p>
  <p class="graf graf--p">
   ⚠️对于这些手段中国读者应该都不陌生。是的，它们很有“传统”。
  </p>
  <p class="graf graf--p">
   1972年，尼克松访问加拿大前夕，一些暗示美国中央情报局与魁北克极端分子有联系的伪造文件到处散播。1982年的马岛战争后，克格勃伪造了里根总统与撒切尔夫人通话的录音带。录音里里根指责撒切尔夫人应该对谢菲尔德号巡洋舰的沉没负责，而撒切尔夫人则反驳说苏美中程核武器谈判的失败应该全怪在里根头上。
  </p>
  <p class="graf graf--p">
   其实全都是不存在的东西。但上述都是稍微用心就可以拆穿的。⚠️如今在机器学习的帮助下，伪造逼真的假货已经变得无比容易，甚至无法辨别真伪。在这看到更多介绍《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%bd%93%e6%81%b6%e6%84%8f%e8%80%85%e6%8e%8c%e6%8f%a1%e6%8a%80%e6%9c%af%ef%bc%9a%e4%bf%a1%e6%81%af%e6%88%98%e7%9a%84%e6%9c%aa%e6%9d%a5%e7%a9%b6%e7%ab%9f%e6%9c%89%e5%a4%9a%e5%8f%af%e6%80%95%ef%bc%9f/" href="https://www.iyouport.org/%e5%bd%93%e6%81%b6%e6%84%8f%e8%80%85%e6%8e%8c%e6%8f%a1%e6%8a%80%e6%9c%af%ef%bc%9a%e4%bf%a1%e6%81%af%e6%88%98%e7%9a%84%e6%9c%aa%e6%9d%a5%e7%a9%b6%e7%ab%9f%e6%9c%89%e5%a4%9a%e5%8f%af%e6%80%95%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
    当恶意者掌握技术：信息战的未来究竟有多可怕？
   </a>
   》
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    借刀杀人是这种欺骗手段的高级形式
   </strong>
   。例如30年代，德国情报机构利用斯大林的疑心，伪造图哈切夫斯基元帅与德军勾结的情报，借斯大林之手杀死了苏军最有远见的将领，从而为德军消除了一个极具威胁的对手。
  </p>
  <p class="graf graf--p">
   ⚠️想想看，如果反对派“得知”其最有成功希望的领袖“通敌”，其结果会是什么？
  </p>
  <p class="graf graf--p">
   假情报活动应与拒止紧密结合，因为拒止能消除、减弱、降低和破坏对手搜集情报的能力，有效降低对手的攻击水平。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    而在针对某一目标的侦查手段越单一、情报来源越少的情况下，对手就越是依赖那些极少数尚且通畅的信息渠道
   </strong>
   ，而且，随着拒止的深入，对手会更加急切地采取这些渠道，这就为欺骗方发布虚假消息提供了便利。
  </p>
  <p class="graf graf--p">
   ⚠️请注意，完全虚假的情报不可能达到欺骗目的，所以假情报不可以完全是假的。
  </p>
  <p class="graf graf--p">
   1942年，苏联特工杰米扬诺夫向德军谍报局报告，说苏军将与11月15日在北高加索和 Rzhev 附近发动进攻。德国人在 Rzhev 方向投入了重兵进行防卫，苏军进攻受挫，一个月内苏军伤亡193000人。
  </p>
  <p class="graf graf--p">
   正是这些人血提高了杰米扬诺夫在德军谍报机构的威信。东线外军处处长高度评价杰米扬诺夫为“主要情报来源”。然而，杰米扬诺夫此后提供的关于库尔斯克战役的情报都是假的，德军一再受挫。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    战争就是这么无耻，那193000人实际上是死于自己人之手。
   </strong>
  </p>
  <p class="graf graf--p">
   A部在当年的伪造技术就是炉火纯青的。一份伪造的《稳定行动特别情报作战训练》经过了美国中央情报局的鉴定，在文风、结构和用词上几乎没有任何破绽，并且使用的最是合适的打印机、纸张，于是外人完全不可能看出是假的。
  </p>
  <p class="graf graf--p">
   克格勃伪造的假情报大多有一个真实的文件为蓝本，但其内容、目标却与真实文件大相径庭。
  </p>
  <p class="graf graf--p">
   一份以绝密文件形式出现的《NR100–6》提到，一旦美国情报获悉苏联即将发动进攻，美国将进行先发制人的打击，北约国家也在美国的攻击范围内。
  </p>
  <p class="graf graf--p">
   这就是半真半假的：美国确实有这种先发制人的核打击计划，但美国的盟国不在攻击范围内。
  </p>
  <p class="graf graf--p">
   下一篇文章将介绍识别战略性欺骗的一些主要途径。既然是攻心的招数，其防御和反攻也大致基于心理角度。
  </p>
  <p class="graf graf--p">
   ⚠️中国有着悠久的战略性欺骗传统，中国的兵法知识一直被西方国家奉为圣典。但请注意，采取欺骗招数的往往都是权势，而反权势的人们却继续奉行着“行不更名坐不改姓才是光明正大”的被灌输价值观，不仅失去了重要的反击能力 即拒止，并且在行动的诸多方面严重被动。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    <em class="markup--em markup--p-em">
     —— 未完待续 ——
    </em>
   </strong>
  </p>
