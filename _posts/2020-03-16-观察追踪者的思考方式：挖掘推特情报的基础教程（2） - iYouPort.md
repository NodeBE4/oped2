  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     把推特作为开源情报数据库
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   制定好的计划的关键是好的研究员。如果你不能了解情况，就很难甚至不可能设计出一个资源高效的计划。
  </p>
  <p class="graf graf--p">
   优秀的研究人员可以采取一些基本步骤，将形成计划所需的各个部分整合在一起。
  </p>
  <p class="graf graf--p">
   首先，研究人员提出数据可以回答或证明的问题，然后，由此确定搜索答案的最佳信息来源。
  </p>
  <p class="graf graf--p">
   接下来，分析数据以寻找隐藏的线索或模式，并在必要时引入更多数据以全面了解目标。
  </p>
  <p class="graf graf--p">
   最后，
   <strong class="markup--strong markup--p-strong">
    借助对情况事实的了解，可以将数据转换为可操作的情报，以支持决策和行动计划。
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%83%85%e6%8a%a5%e4%b8%8d%e6%98%af%e7%ae%80%e5%8d%95%e7%9a%84%e4%bf%a1%e6%81%af%ef%bc%9aiyouport-%e7%ad%94%e8%af%bb%e8%80%85%e9%97%ae%ef%bc%9a%e5%85%b3%e4%ba%8e%e6%88%91%e4%bb%ac/" href="https://www.iyouport.org/%e6%83%85%e6%8a%a5%e4%b8%8d%e6%98%af%e7%ae%80%e5%8d%95%e7%9a%84%e4%bf%a1%e6%81%af%ef%bc%9aiyouport-%e7%ad%94%e8%af%bb%e8%80%85%e9%97%ae%ef%bc%9a%e5%85%b3%e4%ba%8e%e6%88%91%e4%bb%ac/" rel="noopener noreferrer" target="_blank">

  </p>
  <p class="graf graf--p">

  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     您将在 Twitter 上找到的数据有哪些
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   对于使用开源情报能力来决定行动计划的红队来说，Twitter 等社交媒体网站可以提供其他任何地方都无法获得的信息。
  </p>
  <p class="graf graf--p">
   您可以期望在 Twitter 上找到以下类型的信息（当然不止推特，所有社交媒体平台都有大量不注重隐私的人在不断发布敏感信息）：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    各种包括徽章、电子设备、门禁系统等关键信息的照片；
   </li>
   <li class="graf graf--li">
    被发布的电话号码和个人详细信息；
   </li>
   <li class="graf graf--li">
    办公室空间和私人住宅内部的照片和视频；
   </li>
   <li class="graf graf--li">
    通过主题标签连接的具有不同观点的同一事件中来自其他人的照片和视频（帮您补充不同视角的观察）；
   </li>
   <li class="graf graf--li">
    关于当前可能正在调查的任何主题的主要来源陈述；
   </li>
   <li class="graf graf--li">
    调查目标的闲暇时间花在哪里（概况定位）；
   </li>
   <li class="graf graf--li">
    显示了其他员工身份信息的公司或办公室的照片；
   </li>
   <li class="graf graf--li">
    旅行和即将发生的个人事件的记录；
   </li>
   <li class="graf graf--li">
    …… 等等
   </li>
  </ul>
  <p class="graf graf--p">
   想象一下，你可以观看一个视频，该视频向您显示了您无法物理进入的建筑物内部，您无法触摸的系统的硬件配置，或任何潜在的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    社交工程
   </a>
   目的的个人兴趣。这是有多丰厚的大礼呢。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     使用 Twint 分割数据
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   尽管 Twitter 中包含的数据很有价值，但查找和处理它们也很耗时。
  </p>
  <p class="graf graf--p">
   一条推文的发布时间，通常在一条推文是否有用、以及位置、主题和许多其他变量中，起着至关重要的作用。
  </p>
  <p class="graf graf--p">
   为了帮助研究人员快速了解这些选项，Twint 配备了搜索过滤器，您可以通过有用的方式组合它们以显示精确的信息。
  </p>
  <p class="graf graf--p">
   是否想知道自去年以来用户在某个目标城市附近标记的每条推文？或者，也许是所有被点赞提及了的特定主题的推文？…… 借助正确的搜索标记组合，您甚至可以搜索与目标帐户交互的所有人的信息。
  </p>
  <p class="graf graf--p">
   尽管许多搜索工具都需要使用您的 Twitter 帐户通过 Twitter 的API发出请求，但 Twint 不需要。如此一来您就可以
   <strong class="markup--strong markup--p-strong">
    绕过API的限制
   </strong>
   ，通过代理查询、以及以其他匿名方式，
   <strong class="markup--strong markup--p-strong">
    使自己与要研究的目标之间保持安全的距离
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Twint 能够快速生成文本和 CSV 文件以存档感兴趣的推文，因此它成为了一种很好的可服务于法医取证或调查性工作的 Twitter 工具。
  </p>
  <p class="graf graf--p">
   对于高级用户来说 Twint 是一个Python库，可以编写脚本来执行更多自定义的或复杂的操作。
  </p>
  <p class="graf graf--p">
   使用 Twint 编写和扩展 Twitter 搜索脚本的功能使其成为从社交媒体提取数据的简单而且强大的方法。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     步骤1：安装 Twint
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   要开始使用 Twint，我们将下载开发人员版本。
  </p>
  <p class="graf graf--p">
   我从
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/twintproject/twint" href="https://github.com/twintproject/twint" rel="noopener noreferrer" target="_blank">
    GitHub 页面
   </a>
   尝试了几种不同的安装方法，但发现只有一种方法可以在我使用的 Linux 系统上工作。您将需要 pip 包管理器，该程序应随 Python3 一起安装。
  </p>
  <p class="graf graf--p">
   打开一个终端窗口，然后键入以下内容以下载并安装开发者版本。
  </p>

   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%97%a0%e5%a4%84%e5%8f%af%e8%97%8f%e7%9a%84%e6%95%b0%e5%ad%97%e8%b6%b3%e8%bf%b9-%e4%ba%a6%e6%ad%a3%e4%ba%a6%e9%82%aa%e5%85%83%e6%95%b0%e6%8d%ae-%e4%b8%8a%e7%af%87/" href="https://www.iyouport.org/%e6%97%a0%e5%a4%84%e5%8f%af%e8%97%8f%e7%9a%84%e6%95%b0%e5%ad%97%e8%b6%b3%e8%bf%b9-%e4%ba%a6%e6%ad%a3%e4%ba%a6%e9%82%aa%e5%85%83%e6%95%b0%e6%8d%ae-%e4%b8%8a%e7%af%87/" rel="noopener noreferrer" target="_blank">
    无处可藏的数字足迹 — 亦正亦邪元数据 (上篇)
   </a>
   》。
  </p>
  <p class="graf graf--p">
   所以，⚠️您不仅需要对社交媒体情报的细致防御，还需要注意出现在您身边的所有人 —— 他们都应该具备与您保持一致的高水平的安全意识，否则，他们中任何人都可能出卖你。
  </p>
