  <div class="section-inner sectionLayout--insetColumn">
   <p class="graf graf--p graf-after--figure" id="99b4">
    愤世嫉俗的人们说：这是个比恶的世界，而不是比善。我们不支持这种过于极端的观点，但某些时候，这话似乎也有些道理。当最富有的人惯用压榨剥削、欺骗和霸权而成为“1%”时，剩下的“99%”正在面临时刻袭来的危险。
   </p>
   <p class="graf graf--p graf-after--p" id="b30b">
    <a class="markup--anchor markup--p-anchor" data-href="https://plus.google.com/b/104314277918209223775/104314277918209223775/posts/LWACwxfEqSJ" href="https://plus.google.com/b/104314277918209223775/104314277918209223775/posts/LWACwxfEqSJ" rel="noopener nofollow noreferrer" target="_blank">
     上周我们编译的报道
    </a>
    ：一家名为 Zerodium 的所谓漏洞收购公司公开披露了一个 Tor 浏览器零日漏洞，这是一个明显的公关噱头。
   </p>
   <p class="graf graf--p graf-after--p" id="eb8d">
    您可能早有了解，零日漏洞是已被发现的、但尚未向负责修补的开发人员透露的安全漏洞。零日漏洞的利用程序对网络安全具有巨大威胁，因此零日漏洞不但是骇客的最爱，掌握多少零日漏洞也成为评价骇客技术水平的一个重要参数。
   </p>
   <p class="graf graf--p graf-after--p" id="12b1">
    Zerodium 透露的零日是 Tor 浏览器的旧版本（7.x）。Zerodium 将他们的漏洞描述为能绕过 Tor / NoScript“最高”安全级别的保护，允许恶意代码在浏览器上运行。
   </p>
   <p class="graf graf--p graf-after--p" id="71dd">
    该漏洞利用程序需要与恶意 JavaScript 结合使用，
    <a class="markup--anchor markup--p-anchor" data-href="https://arstechnica.com/information-technology/2013/08/attackers-wield-firefox-exploit-to-uncloak-anonymous-tor-users/" href="https://arstechnica.com/information-technology/2013/08/attackers-wield-firefox-exploit-to-uncloak-anonymous-tor-users/" rel="noopener nofollow noreferrer" target="_blank">
     泄露用户的身份
    </a>
    ，以便“有效”。在给 ZDNet 的
    <a class="markup--anchor markup--p-anchor" data-href="https://www.zdnet.com/article/exploit-vendor-drops-tor-browser-zero-day-on-twitter/" href="https://www.zdnet.com/article/exploit-vendor-drops-tor-browser-zero-day-on-twitter/" rel="noopener nofollow noreferrer" target="_blank">
     声明
    </a>
    中 Zerodium 首席执行官 Chaouki Bekrar 表示，“这个 Tor 浏览器漏洞
    <strong class="markup--strong markup--p-strong">
     被 Zerodium 公司在几个月前做为零日漏洞收购，并与政府客户共享
    </strong>
    。”对于那些依赖 Tor 浏览器的人来说，这当然是一个非常令人不安的启示！
   </p>
   <p class="graf graf--p graf-after--p" id="2d47">
    <strong class="markup--strong markup--p-strong">
     使用 Tor 的人是为了抵抗压迫性政权对隐私的侵犯，这一信息说明人们已经有相当长一段时间处于危险之中了。
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="d5ff">
    好吧，现在我们应该承认这点了，这确实就是如今整个世界的状态：一个阴暗的组织先花大把的资金吸引安全研究人员，然后将他们发现的漏洞卖给政府客户，赚回更多的钱 — — 就是那些用于对抗他们的公民的地方。一切因为钱。
   </p>
   <p class="graf graf--p graf-after--p" id="7d30">
    零日漏洞及其利用代码不仅对犯罪骇客而言具有极高的利用价值，一些国家间谍和网军部队也非常重视这些信息。据路透社的报告，
    <strong class="markup--strong markup--p-strong">
     美国情报部门是零日漏洞黑市的最大买家
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="3543">
    而中国，在自己“制造”零日漏洞。详见 access now 组织的报告：
    <a class="markup--anchor markup--p-anchor" data-href="https://www.accessnow.org/china-tips-the-scale-of-global-cybersecurity-by-hoarding-vulnerabilities/" href="https://www.accessnow.org/china-tips-the-scale-of-global-cybersecurity-by-hoarding-vulnerabilities/" rel="noopener nofollow noreferrer" target="_blank">
     中国囤积漏洞的战略威胁全球网络安全
    </a>
    。
   </p>
   <p>
   </p>

   <div class="section-inner sectionLayout--insetColumn">
    <p class="graf graf--p graf-after--figure" id="5d53">
     2017 年11月 Recorded Future 发布了中国国家漏洞数据库出版速度研究（缩写CNNVD）。最初他们得出的结论是，中国实际上比美国更快地评估和报告漏洞。然而，当重新审视调查结果时才
     <a class="markup--anchor markup--p-anchor" data-href="https://www.recordedfuture.com/chinese-vulnerability-data-altered/" href="https://www.recordedfuture.com/chinese-vulnerability-data-altered/" rel="noopener nofollow noreferrer" target="_blank">
      发现
     </a>
     ，
     <strong class="markup--strong markup--p-strong">
      大多数数据都被修改了，以隐藏更长的处理时间，在此期间中国政府可以评估漏洞是否对情报行动有用
     </strong>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="17e4">
     这种处理手段很可能是中国国家安全部（MSS）的命令。该部门需要更长的时间来评估这些漏洞，主要看如何利用它们，然后才会公开。通过此过程，他们会延迟发布公共通知、修补和补救指导，从而大幅增加公众所面临的危险。
    </p>
    <p class="graf graf--p graf-after--p" id="1c03">
     中国国家机构对脆弱性的战略性使用方案并不是原创，
     <strong class="markup--strong markup--p-strong">
      现在这种卑劣的方法在全球都有强烈的推动力
     </strong>
     ，各国政府都在制定披露政策、制定具体规则来管理国家对这种高度敏感信息的处理，如美国的 Vulnerability Equity Process（
     <a class="markup--anchor markup--p-anchor" data-href="https://www.accessnow.org/meltdown-spectre-need-better-vulnerability-disclosure-stronger-u-s-cyber-framework/" href="https://www.accessnow.org/meltdown-spectre-need-better-vulnerability-disclosure-stronger-u-s-cyber-framework/" rel="noopener nofollow noreferrer" target="_blank">
      VEP
     </a>
     ）。
    </p>
    <p class="graf graf--p graf-after--p" id="6f82">
     我们
     <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/DonEvansWm/status/972451031803224064" href="https://twitter.com/DonEvansWm/status/972451031803224064" rel="noopener nofollow noreferrer" target="_blank">
      报道过这个消息
     </a>
     ：今年3月，由于来自中国的安全研究人员此前占据了黑客领域的主导地位，因此今年中国没有派出团队参加 Pwn2Own 国际黑客大赛 — — 当局要控制技术高手在自己的权力范围内。5月，第一次在美国境外举行的 DEFCON 是与百度安防合作在中国北京举行的，此事则进一步暴露了政府对控制安全漏洞信息流动的明显渴望。
    </p>
    <p class="graf graf--p graf-after--p" id="53c9">
     2018年7月，美国国会听证会上透露，中国政府在中国公司向用户（包括外国政府）披露漏洞之前，已经知道了微处理器中如今臭名昭着的 Meltdown 和 Spectre 漏洞。这一举措对中国当局来说有明显的战略优势，同时，它极有可能危及全球数字生态系统的安全。
    </p>
    <p class="graf graf--p graf-after--p" id="148d">
     当漏洞未被修补时，它们会构成严重的安全威胁，这种威胁影响深远且具有潜在的经济破坏性。漏洞披露的及时和准确是防止这种破坏发生的最有效方法。然而，
     <strong class="markup--strong markup--p-strong">
      包括中国在内的越来越多的国家则采取了相反的方法
     </strong>
     。中国拥有一个相当大的黑客社区，其技术实力绝不容忽视。
    </p>
    <p class="graf graf--p graf-after--p" id="d2af">
     全球范围内，政府和情报机构对零日漏洞的需求量非常大，从而推动了一个巨大的交易市场的出现，就如 Zerodium 这样的公司，就是该市场的交易中间人，这些公司已经赚得盆满钵满。
    </p>
    <p class="graf graf--p graf-after--p" id="0bfa">
     Zerodium 毫不掩饰他们卑劣的经营，该网站上就直接写着，他们的客户“主要是政府组织……以及来自国防、技术和金融部门的大公司”。Zerodium 甚至为他们的高价支出而感到自豪，他们在自己的网站上说：“我们支付大笔的奖金，真金白银不是虚的”。
    </p>
    <p class="graf graf--p graf-after--p" id="a4a2">
     <strong class="markup--strong markup--p-strong">
      但必须知道！这个模式与像 HackerOne 或 Bugcrowd 那些传统的 bug 赏金流程有根本性的不同！
     </strong>
     后者会直接向开发人员报告漏洞，以便立即修补它们。传统程序通常也会向提交漏洞的安全研究人员支付（相比下少很多的）费用。而前者，直接卖给了必然将这些漏洞武器化压制本国人民的的政府当局，赚来大把的金钱。
    </p>
    <p class="graf graf--p graf-after--p" id="314e">
     那么对于此案例来说，Tor 的用户究竟前后总共多长时间处于危险之中？做个拼图并不难：
    </p>
    <p class="graf graf--p graf-after--p" id="c12b">
     2017 年 9 月 13 日，Zerodium 的网站宣布了限时优惠：2017 年 9 月 13 日至 2017 年 11 月30日期间，他们将为 Tor Browser 漏洞发现者支付总计 1,000,000 美元。是的，你没有看错：一百万美元。
    </p>
    <p class="graf graf--p graf-after--p" id="37da">
     为了能看得更清楚一点，我们来做个悲惨的比较。在
     <a class="markup--anchor markup--p-anchor" data-href="https://hackerone.com/torproject" href="https://hackerone.com/torproject" rel="noopener nofollow noreferrer" target="_blank">
      Tor Project HackerOne 页面上
     </a>
     ，Tor Browser 的开发人员表示为发现高危严重性漏洞的研究人员提供
     <strong class="markup--strong markup--p-strong">
      4,000 美元
     </strong>
     的资金，自 2017 年7月20日他们的漏洞赏金计划启动以来，他们只支付出了 7,100 美元……
     <strong class="markup--strong markup--p-strong">
      他们根本无法与政府客户的财务能力相竞争
     </strong>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="9229">
     而 Zerodium 的声明还指出，如果能发现一个功能更强的漏洞可以获得更高的奖金，需要是可以绕过阻止 JavaScript 的 Tor / NoScript “最高”安全级别的那种。这就是正是发生的事。
    </p>
    <p class="graf graf--p graf-after--p" id="e6a2">
     一名安全研究人员发现了这样一个漏洞，并将其出售给了 Zerodium。虽然 Zerodium 当时没有透露他们对 Tor 浏览器漏洞支付的结果，但 Bekrar 上周向 ZDNet 表示，在限时优惠报价期间和之后，他们
     <strong class="markup--strong markup--p-strong">
      获得了许多符合他们要求的 Tor 漏洞
     </strong>
     。这意味着政府实体在 2017 年 9 月 13 日限时优惠开始后的任何时间都可以使用这个 Tor 浏览器漏洞。
    </p>
    <p class="graf graf--p graf-after--p" id="1a84">
     Bekrar 声明不受漏洞影响的 Tor Browser 8.0 于 2018 年9月5日发布 — — 也就是在 Zerodium 于 2018 年9月10日向公众发布 Tor 浏览器零日漏洞之前的仅仅 5 天！
     <strong class="markup--strong markup--p-strong">
      这意味着可能差不多有一年的时间里，Tor Browser 用户试图保护自己免受压迫性政权危害的目的都失败了！却全然不知道自己正在危险之中。
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="78d7">
     如果出售 Tor 浏览器漏洞的安全研究人员将其提交给 Tor 浏览器开发者，他们可能会获得更少的经济回报，但会为世界做出善事。但事实上……
     <strong class="markup--strong markup--p-strong">
      他们选择了金钱，而不是道德
     </strong>
     。
    </p>
    <p class="graf graf--p graf-after--p" id="c6d8">
     当然，一些政府间谍机构可能会自己发现 Tor 浏览器漏洞，并且在 Zerodium 开始暗中兜售它之前就已经知道了。与 Tor 浏览器捆绑在一起的 NoScript 扩展开发人员 Giorgio Maone 告诉 ZDNet，该漏洞是在 2017 年5月11日发布的 NoScript 5.0.4 中引入的。如果间谍机构或其他任何人在 Zerodium 收购之前发现了这个漏洞的话，用户可能在那时开始就已经陷入危险了。
    </p>
    <p class="graf graf--p graf-after--p" id="9956">
     合理合法的 bug 赏金程序应该以这样的方式运行：当安全研究人员发现漏洞时，他们会立即报告给开发人员，并尽快修补。但随着 Zerodium 等组织的“蓬勃发展”，安全研究人员正在被大笔的资金诱惑，将他们的漏洞利用出售给拥有“精英客户群”的漏洞收购公司。这些组织购买的安全漏洞通常是
     <strong class="markup--strong markup--p-strong">
      故意不提交给相应的开发人员，以防止它们被修补
     </strong>
     。由于漏洞未得到修补，政府买家可以在他们认为合适的情况下秘密获取并利用其发起针对目标的攻击 — — 政府的“目标”是谁？调查记者、影响力很高的公民活动家、反对派组织、公民社会组织、地缘政治对手……一切有可能威胁到政权稳固的人和群体。
    </p>
    <p class="graf graf--p graf-after--p" id="c01e">
     Zerodium 并不是唯一一家迎合政府的漏洞收购公司。众所周知，像 Northrop Grumman，General Dynamics 和 Raytheon 这样的主要国防承包商也向政府实体出售致命的漏洞。可能还有其他公司我们尚未了解到的。由于这些强大的公司秘密出售零日攻击，隐私对于生活在压迫政权下的人来说是一场越来越艰苦的战斗。
    </p>
    <p class="graf graf--p graf-after--p" id="2d4e">
     <strong class="markup--strong markup--p-strong">
      零日漏洞交易有三种市场
     </strong>
     ：犯罪黑客交易侵入系统、盗取密码和信用卡号的漏洞利用代码和漏洞信息的
     <strong class="markup--strong markup--p-strong">
      黑市
     </strong>
     ；充斥着漏洞悬赏活动的
     <strong class="markup--strong markup--p-strong">
      “白市”
     </strong>
     ，研究人员和黑客在这里把漏洞信息提供给供应商，可获得相应的赏金，同时漏洞也得以修复 — — 在这个市场活动的还有安全公司，他们购买零日漏洞利用来进行渗透测试，以确定客户系统是否易受攻击；而最致命的是

   </p>
   <p class="graf graf--p graf-after--p" id="9153">
    我们在推荐自由开源软件作为数字极权的替代品的时候，收到了来自中国高级知识分子的神奇回复，对方说：“你提到的那些东西我完全不知道，又怎么能信任呢？”苦笑。谷歌母公司 Alphabet 去年全年的销售额高达 1109 亿美元，谷歌的在线广告是 Alphabet 最赚钱的业务。这还是在受到特朗普税改和欧盟反垄断起诉（27 亿欧元罚款）的影响之下的收入。相比下大多数独立的开源自由软件只是大树下的蚂蚁，它们拥有的只有道德感、对互联网民主化的信念和广大技术人士的热情奉献，在这种情况下我们该如何打造能压过数字极权的知名度以令其被中国用户“信任”？
   </p>
   <p class="graf graf--p graf-after--p" id="b0f8">
    如果您还觉得不够：最新消息，
    <span class="markup--quote markup--p-quote is-other" data-creator-ids="e5835fc6e3b2">
     为中国大型“集中营”提供监控摄像头的中国科技巨头海康威视（Hikvision）得到了美国朋友的大力帮助。公关巨头博雅公关公司(Burson-Marsteller)在帮助海康威视的美国子公司完成“战略规划和指导”等任务。游说公司 Sidley Austin
     <a class="markup--anchor markup--p-anchor" data-href="https://projects.propublica.org/represent/lobbying/301017994" href="https://projects.propublica.org/represent/lobbying/301017994" rel="nofollow noopener noreferrer" target="_blank">
      任命
     </a>
     前14届国会议员 Rick Boucher (D-Va.) 帮助海康威视公司反击美国对他们的监控摄像头的禁止
    </span>
    （关于海康威视，请见这篇文章《
    <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E6%98%AF%E8%B0%81%E5%9C%A8%E4%B8%BA%E4%B8%AD%E5%9B%BD%E6%94%BF%E5%BA%9C%E7%9A%84%E7%9B%91%E8%A7%86%E7%A4%BE%E4%BC%9A%E7%B3%BB%E7%BB%9F%E6%8F%90%E4%BE%9B%E6%8A%80%E6%9C%AF%E6%94%AF%E6%8C%81-54121bfaffba" href="https://medium.com/@iyouport/%E6%98%AF%E8%B0%81%E5%9C%A8%E4%B8%BA%E4%B8%AD%E5%9B%BD%E6%94%BF%E5%BA%9C%E7%9A%84%E7%9B%91%E8%A7%86%E7%A4%BE%E4%BC%9A%E7%B3%BB%E7%BB%9F%E6%8F%90%E4%BE%9B%E6%8A%80%E6%9C%AF%E6%94%AF%E6%8C%81-54121bfaffba" rel="noopener noreferrer" target="_blank">
     是谁在为中国政府的监视社会系统提供技术支持？
    </a>
    》）
   </p>
   <p class="graf graf--p graf-after--p" id="d440">
    目前美国公司与中国老大哥海康威视的合作看起来是一项“明智的商业决策”。向美国司法部提交的一份文件显示，美国公司
    <a class="markup--anchor markup--p-anchor" data-href="https://www.odwyerpr.com/story/public/11129/2018-08-10/b-m-focuses-chinas-top-maker-spy-gear.html" href="https://www.odwyerpr.com/story/public/11129/2018-08-10/b-m-focuses-chinas-top-maker-spy-gear.html" rel="nofollow noopener noreferrer" target="_blank">
     每月收到 25,000美元
    </a>
    的酬劳支付，从高级管理人员每小时650美元、到实习生每小时85美元……现任博雅公关公司的首席执行官 Don Baer 是前克林顿政府的通讯主管。
   </p>
   <p class="graf graf--p graf-after--p" id="7cce">
    <span class="markup--quote markup--p-quote is-other" data-creator-ids="anon">
     最令人无语的是，中国人在等着美国公司来拯救自己，而美国公司在等着北京的钱。
    </span>
   </p>
   <p class="graf graf--p graf-after--p" id="d175">
    这种事自古以来都不新奇。1937年6月，IBM 创始人托马斯·沃森（Thomas Watson Jr.）
    <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/archive/politics/2001/02/11/ibm-technology-aided-holocaust-author-alleges/6addc414-ecee-4058-bea6-7f4708912d6f/?utm_term=.cfed55ea94d5" href="https://www.washingtonpost.com/archive/politics/2001/02/11/ibm-technology-aided-holocaust-author-alleges/6addc414-ecee-4058-bea6-7f4708912d6f/?utm_term=.cfed55ea94d5" rel="nofollow noopener noreferrer" target="_blank">
     接受了
    </a>
    阿道夫·希特勒（Adolf Hitler）为外国合伙人颁发的奖章。纳粹使用 IBM 制造的机器帮助统计犹太人 —— 直到第二次世界大战开始，美国技术人员都在为纳粹提供机器维修服务。
   </p>
   <p class="graf graf--p graf-after--p" id="9cd0">
    我们相信 IBM 没有爱上希特勒、不恨犹太人，谷歌也并不是喜欢中国政府、不“恨”中国人，他们只是喜欢钱。就如那些将零日漏洞卖给压迫性政府的公司，仅仅是因为政府当局和其情报部门是出价最高的人。如果按照物质主义的逻辑来说，这样做会被认为“无可厚非”。然而，更严重的问题是，追求正义的人们没有那么多钱和当权者竞争，
   </p>
   <ul class="postList">
    <li class="graf graf--li graf-after--p" id="66b2">
     就如 Tor 的四千美元和 Zerodium 的一百万美元的天壤之别……
    </li>
    <li class="graf graf--li graf-after--li" id="3fb9">
     就如帮助独裁政府工作的人自称只为“养家糊口”；
    </li>
    <li class="graf graf--li graf-after--li" id="e8c0">
     就如威权国家公务员考试的挤破门槛被解释成只图一个“铁饭碗”
    </li>
   </ul>
   <p class="graf graf--p graf-after--li" id="007c">
    但是，我们还能用什么来捍卫良善之心？
   </p>
