  <p class="graf graf--p">
   2015年3月，Hicham Mansouri 通过电子邮件向一家反恶意软件公司投诉，他怀疑有人可能未经许可远程访问其设备。
  </p>
  <p class="graf graf--p">
   他还记得与该软件公司交换了一些信息，但几天之后通信就被打断了。
  </p>
  <p class="graf graf--p">
   大约10名穿着便服的警察闯入他在拉巴特的家中，逮捕了他。
  </p>
  <p class="graf graf--p">
   这是他讲给巴黎的CPJ的故事。他现在在这里寻求政治庇护。
  </p>
  <p class="graf graf--p">
   Mansouri 当时担任摩洛哥调查新闻协会的项目经理，负责协调安全报道的应用程序 StoryMaker，并调查数字监控问题 —— 他因所谓的通奸被判入狱，这是诬陷的罪名以报复他的调查工作。
  </p>
  <p class="graf graf--p">
   2016年1月，当他重新拿回自己的计算机时发现，与反恶意软件公司通信的电子邮件丢失了，显然是被人从他的帐户中删除了。
  </p>
  <p class="graf graf--p">
   这不是个案。多位记者描述了电子设备发生故障的方式，他们认为这些设备受到了监视。

  <p class="graf graf--p">
   独立记者遭受的监视和被迫逃亡严重打击了摩洛哥的新闻自由，该国记者几十年来一直受到骚扰。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们已经无法明确表示在摩洛哥有任何独立新闻，”加利福尼亚州的学者 Samia Errazzouki 说。
  </p>
  <p class="graf graf--p">
   摩洛哥王国因支持温和的伊斯兰教和提高其性别平等记录以及其他被认为是使该地区走向进步的政策而受到国际上的广泛赞誉。
  </p>
  <p class="graf graf--p">
   然而，曾经是摩洛哥记者的 Errazzouki 表示，那个被称为 makhzen 的强大国家机构 —— 君主制、政府部门和维稳部门、以及与他们关系密切的富商 —— 自20世纪90年代以来一直在积极参与对独立媒体的打击。
  </p>
  <p class="graf graf--p">
   摩洛哥当局长期
   <strong class="markup--strong markup--p-strong">
    操纵广告市场以鼓励当局满意的报道
   </strong>
   ，那些选择独立的记者受到逮捕和恐吓。
  </p>
  <p class="graf graf--p">
   根据 Errazzouki 和调查间谍软件的国际组织（如 Privacy International 和 Citizen Lab）的研究，摩洛哥当局还部署了
   <strong class="markup--strong markup--p-strong">
    复杂的监控软件
   </strong>
   。
  </p>
  <p class="graf graf--p">
   很遗憾，这种情况在很多国家都不是新闻。

  <p class="graf graf--p">
   2016年底，al-Hoceima 市的一名鱼贩被一辆垃圾车碾压致死，他只是试图收回警方被没收的商品，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bbc.com/news/world-africa-37819308" href="https://www.bbc.com/news/world-africa-37819308" rel="noopener noreferrer" target="_blank">
    此事件引发了
   </a>
   整个北部 Rif 地区的大型抗议活动。
  </p>
  <p class="graf graf--p">
   截至2017年12月，至少有三名记者 —— Hamid al-Mahdaoui、Mohamed al-Asrihi 和 Abdelkabir al-Hor —— 因参与示威活动而被监禁。
  </p>
  <p class="graf graf--p">
   自由职业者 Ali Lmrabet 经常从西班牙巴塞罗那前往他的摩洛哥家乡得土安。 Lmrabet 写下了很多关于摩洛哥事务的评论，包括 Rif 抗议活动。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “每次他们都会试图对我使用间谍软件，”他说，他指的是摩洛哥当局。
  </p>
  <p class="graf graf--p">
   Lmrabet 因在2003年“侮辱国王”的罪名被判处三年徒刑；虽然他在2004年获得了赦免，但他在2005年又因撰写关于西撒哈拉地区的报道而被禁止从事新闻工作10年。
  </p>
  <p class="graf graf--p">
   Lmrabet 描述了他以前的
   <strong class="markup--strong markup--p-strong">
    笔记本电脑即使在他不使用时也表现出奇怪的行为
   </strong>
   ，比如忽然在屏幕上打开难以识别的文件，或者莫名其妙地激活网络摄像头。
  </p>
  <p class="graf graf--p">
   他说，反病毒软件在该计算机上检测到至少三个恶意程序。他表示自己遵循“严格的安全协议”。
  </p>
  <p class="graf graf--p">
   总部位于伦敦的 Al-Quds al-Arabi 报记者 Saida Elkamel 表示，她有时会注意到，
   <strong class="markup--strong markup--p-strong">
    在某些手机对话中会出现回声
   </strong>
   ，比如与 Nasser Zefzafi 的父亲的对话，Nasser 是一名活动家，由于领导 Rif 抗议活动而被判监禁20年。
  </p>
  <p class="graf graf--p">
   记者说，她的手机经常摸起来很烫，而且电力消耗特别快。
  </p>
  <p class="graf graf--p">
