  <p class="graf graf--p graf--startsWithDoubleQuote">
   “人类只有一种真正有效的武器，那就是笑声” —— 马克吐温
  </p>
  <p class="graf graf--p">
   人类历史上一直在使用幽默给那些被征服和被压迫的人们带来希望，并鼓励抵抗。
  </p>
  <p class="graf graf--p">
   幽默可以成为一种强大的武器，从心理层面削减压迫者的规模，破坏他们的社会合法性 —— 难怪在第三帝国期间禁止反纳粹的笑话。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    幽默也可以滋养抵抗
   </strong>
   。
  </p>
  <p class="graf graf--p">
   幽默是一种高级别的反抗，但不是“段子手”那种。推荐一本书《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.jstor.org/stable/2769536?seq=1#page_scan_tab_contents" href="https://www.jstor.org/stable/2769536?seq=1#page_scan_tab_contents" rel="noopener noreferrer" target="_blank">
    “Gallows Humor”-A Sociological Phenomenon
   </a>
   》
  </p>
  <p class="graf graf--p">
   弗洛伊德确定幽默有三种：身体幽默、思想幽默、和精神幽默。
  </p>
  <p class="graf graf--p">
   这三种都可以用来破坏权力，并使受压迫者的精神浮现。
  </p>
  <p class="graf graf--p">
   身体很有趣，因为一些身体现象是无法控制的：即使是强大的权威、看起来不可一世的权力，也很难控制放屁、打嗝、跌跤 …… 抗议者往往通过指出它来颠覆他们的权威。
  </p>
  <p class="graf graf--p">
   津巴布韦总统罗伯特穆加贝绊倒的照片在互联网上疯狂传播，他要求摄影师快速删除已经传遍世界各地的照片来控制局面，然而一切都是徒劳的；越是删除，人们的欢闹声就越高。
  </p>
  <p class="graf graf--p">
   想想看，在现场直播的重大场合上，一个悠扬的长屁会带来什么样的效果？吃下一整盘煮豆子后任何活人都能做到。
  </p>
  <p class="graf graf--p">
   当言论自由受到限制，或者直接与权威对抗被视为非法时，思想幽默就很有用。
  </p>
  <p class="graf graf--p">
   用
   <strong class="markup--strong markup--p-strong">
    最巧妙的方式
   </strong>
   羞辱他们就很难被当权者抨击为冒犯或非法（因此也难以实施惩罚）。
  </p>
  <p class="graf graf--p">
   中国官员已经决定在新闻媒体中禁止双关语和文字游戏，足见，这种幽默形式的颠覆性力量已经得到了充分的体现。
  </p>
  <p class="graf graf--p">
   最后，精神幽默就是反守为攻。
  </p>
  <p class="graf graf--p">
   一个很好的例子是 Anna Gensler 的 Instagram 艺术项目 Instagranniepants。
  </p>
  <p class="graf graf--p">
   由于厌倦了在线性骚扰，Gensler 开始将她收到的骚扰信息整合到她绘制的骚扰者的有趣的裸体漫画中，并将结果发布在网上；这些作品迅速成为了一个受欢迎的漫画meme。
  </p>
  <p class="graf graf--p">
   有效的幽默可以为观众提供他们渴望的正义；提供唯一安全的方式来藐视权威，或作为模版，激励公众在真实世界中寻求正义。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/?p=7242&amp;preview=true" href="https://www.iyouport.org/?p=7242&amp;preview=true" rel="noopener noreferrer" target="_blank">
    反叛小丑军（CIRCA）
   </a>
   就是一个伟大的幽默行动实例。
  </p>
  <p class="graf graf--p">
   CIRCA 已经出现在整个欧洲的抗议游行中，以破坏警察（以及相关暴力镇压）的气势，并且通常会增加一种狂欢节般的反独裁气氛。
  </p>
  <p class="graf graf--p">
   例如，在英国军队招募办公室举行的反战抗议活动中，CIRCA 成员使用鸡毛掸子“清理”士兵和警车，并巧妙地模仿官员的行走和姿势，甚至连官员自己都在努力憋住不要笑出来。
  </p>
  <p class="graf graf--p">
   一旦警察将小丑们从大楼赶出去，招聘办公室就会提前关门，所以小丑们就在门外放了一张桌子，上面竖一块牌子写着 “在这里签字”，然后人们开始与小丑军签约，而不是正规军。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    让人们发笑的东西更容易被记住 —— 更容易被传播。
   </strong>
   大多数人更容易在社交媒体上发布一些有趣的内容，而不是严肃的新闻和信息。
  </p>
  <p class="graf graf--p">
   活动家应该利用这点。更多见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/1%e3%80%81%e5%88%ab%e8%ae%a9%e5%aa%92%e4%bd%93%e4%b8%ba%e4%bd%a0%e7%9a%84%e5%af%b9%e6%89%8b%e5%b7%a5%e4%bd%9c%ef%bc%9a%e5%88%9b%e9%80%a0%e6%80%a7%e7%a0%b4%e5%9d%8f/" href="https://www.iyouport.org/1%e3%80%81%e5%88%ab%e8%ae%a9%e5%aa%92%e4%bd%93%e4%b8%ba%e4%bd%a0%e7%9a%84%e5%af%b9%e6%89%8b%e5%b7%a5%e4%bd%9c%ef%bc%9a%e5%88%9b%e9%80%a0%e6%80%a7%e7%a0%b4%e5%9d%8f/" rel="noopener noreferrer" target="_blank">
    别让媒体为你的对手工作：创造性破坏
   </a>
   》
  </p>
  <p class="graf graf--p">
   再一次，就如我们一贯强调的，⚠️
   <strong class="markup--strong markup--p-strong">
    幽默战略应该与其他战略一起协调运用，以服务于一场设计完好的抗议行动 —— 否则只能混成一个微博段子手或抖音网红。
   </strong>
  </p>
  <p class="graf graf--p">
   下面举一个例子：内裤行动。
  </p>
  <p class="graf graf--p">
   2007年，在军方对广泛的民主抗议活动进行了残酷的镇压之后，传统的公众抗议活动已经在缅甸变得不可能。
  </p>
  <p class="graf graf--p">
   尽管如此，异议仍然很普遍，严酷的镇压反而激发了许多创造性的想法，颠覆性的和风险较低的行动不断激增。
  </p>
  <p class="graf graf--p">
   虽然这些行动还不足以摧毁军方对权力的控制，但它们有助于
   <strong class="markup--strong markup--p-strong">
    进一步孤立执政的将军，帮助人们克服恐惧并治愈创伤，感受到成为积极抵抗的一部分
   </strong>
   。
  </p>
  <p class="graf graf--p">
   军事领导层有一种迷信的恐惧，认为接触女性内衣可能会夺走他们的权力；于是，这就产生了极大的幽默感 —— Lanna Action 组织呼吁缅甸各地的妇女通过把自己的内裤邮寄给将军，以抗议该政权及其使用的性暴力。
  </p>
  <p class="graf graf--p">
   活动家还呼吁世界各地的支持者将自己的内裤邮寄到他们所在国家的缅甸大使馆或领事馆。
  </p>
  <p class="graf graf--p">
   Lanna 对缅甸的行动也使用了其他象征手段来嘲笑军队并对抗政府。
  </p>
  <p class="graf graf--p">
   再一次，强调了我们前述的原理：
   <strong class="markup--strong markup--p-strong">
    文化地形，它非常重要
   </strong>
   ；见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/1%e3%80%81%e4%ba%86%e8%a7%a3%e4%bd%a0%e7%9a%84%e5%9c%b0%e5%bd%a2%e6%98%af%e7%ad%96%e5%88%92%e8%a1%8c%e5%8a%a8%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" href="https://www.iyouport.org/1%e3%80%81%e4%ba%86%e8%a7%a3%e4%bd%a0%e7%9a%84%e5%9c%b0%e5%bd%a2%e6%98%af%e7%ad%96%e5%88%92%e8%a1%8c%e5%8a%a8%e7%9a%84%e7%ac%ac%e4%b8%80%e6%ad%a5/" rel="noopener noreferrer" target="_blank">
    了解你的地形是策划行动的第一步
   </a>
   》。
  </p>
  <p class="graf graf--p">
   与流浪狗的组合在缅甸文化中是极具攻击性的，因此，活动人士将丹瑞大将的照片贴在流浪狗的项圈上。
  </p>
  <p class="graf graf--p">
   在丹瑞大将的生日那天，活动家们做了一个内裤形状的烤蛋糕，上面贴上丹瑞大将的照片，然后放到街头，让流浪狗吃了个饱。
  </p>
  <p class="graf graf--p">
   后来，为了抗议腐败的 2010 年选举，Lanna Action 的活动家组办了一系列选举游击队剧目，试图登记 Lanna 内裤党，传播“pantifesto”，并发誓要“争夺并赢得 2010 年缅甸选举，形成一个过渡政府并将权力移交给缅甸人民。
  </p>
  <p class="graf graf--p">
   选举游击队的力量在很大程度上不是为试图赢得国家权力的，而是要质疑其核心政权。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在极端的国家暴力时期，当公共集会被取缔时，“内裤”行动提供了简单且低风险的表达异议的方式。
   </strong>
  </p>
  <p class="graf graf--p">
   它出色地利用迷信和文化禁忌将幽默注入了最危险的区域，并破坏了一个极其强大的对手。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    关键的战术：分布式行动
   </strong>
  </p>
  <p class="graf graf--p">
   在这里看到更多解释《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/2%E3%80%81%E5%88%86%E5%B8%83%E5%BC%8F%E8%A1%8C%E5%8A%A8%E7%9A%84%E8%A1%97%E5%A4%B4%E6%B4%BE%E5%AF%B9/" href="https://www.iyouport.org/2%E3%80%81%E5%88%86%E5%B8%83%E5%BC%8F%E8%A1%8C%E5%8A%A8%E7%9A%84%E8%A1%97%E5%A4%B4%E6%B4%BE%E5%AF%B9/" rel="noopener noreferrer" target="_blank">
    分布式行动的街头派对
   </a>
   》。
  </p>
  <p class="graf graf--p">
   在本案例中，在公共集会被宣布为非法的时候，内裤行动让人们有了一种在缅甸和国外以分散的方式、在自己的时间空间中进行抗议的方式。
  </p>
  <p class="graf graf--p">
   这无疑会增强人们的信心和参与热情。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    主要原理：幽默
   </strong>
  </p>
  <p class="graf graf--p">
   就像皇帝的新衣一样。
  </p>
  <p class="graf graf--p">
   统治者的力量被证明极易受到嘲弄，无论是一个孩子大声说皇帝没有穿衣服、还是女人们指出将军害怕内裤。
  </p>
  <p class="graf graf--p">
   没有什么比笑声更能打破强大的权力了。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、抓住道德制高点
   </strong>
  </p>
  <p class="graf graf--p">
   如前文中介绍过的要点，
   <strong class="markup--strong markup--p-strong">
    充分利用文化仪式和符号就可以增加你在冲突中的合法性和潜在的领导力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这对于绝大多数为佛教徒的缅甸来说至关重要，因为佛教僧侣在传统意义上固定了该国的道德指南针。
  </p>
  <p class="graf graf--p">
   僧侣是政治启蒙者，并在该国作为人民神圣权利和责任的守护者。
  </p>
  <p class="graf graf--p">
   最着名的是，缅甸尽人皆知的藏红花革命，就是由全缅甸僧侣联盟领导的，该联盟的立场是服务于人民并抵制政权强加的痛苦。
  </p>
  <p class="graf graf--p">
   特别是，两种宗教传统被用来谴责军政府严重违反佛陀的教义。
  </p>
  <p class="graf graf--p">
   首先，僧侣的公开诵经祈祷是在与受苦的公众结盟；由于这种反抗团结的行动，僧侣遭到政权的残酷殴打。
  </p>
  <p class="graf graf--p">
   其次，可能更为著名的是，僧侣联合的“推翻施舍碗”行动，体现了在激进主义传播中使用文化仪式的力量。就是上面的图片，缅甸僧侣在1990年和2007年两次推翻了他们的施舍碗，拒绝接受军人和他们的家属的捐款。
  </p>
  <p class="graf graf--p">
   这种简单的拒绝行动有效地剥夺了当局获得宗教价值的能力，给执政的政权带来了严重的道德和政治挑战。
  </p>
  <p class="graf graf--p">
   由于政府和大多数国家官员都是佛教徒，因此使用宗教仪式可以更直接和更有力地对政权喊话。

    2、利用偏见来获得优势
   </strong>
  </p>
  <p class="graf graf--p">
   如果你是一群认为触摸女人的内裤会削弱自己“力量”的将军，你就会自动落入内裤运动构建的陷阱。
  </p>
  <p class="graf graf--p">
   Lanna Action 组织的内裤运动在当地文化中非常有效。
  </p>
  <p class="graf graf--p">
   据报道，有数百条内裤被送到了大使馆。
  </p>
  <p class="graf graf--p">
   Panty Power 是一种幽默的方式，表现出对民主抗议者的严重支持，用蕾丝哄骗政权，推翻性别歧视，并利用了军政府的偏见。
  </p>
  <p class="graf graf--p">
   在公共集会被禁止的时候，内裤行动提供了一种简单而低风险的方式来参与抗议。
  </p>
  <p class="graf graf--p">
   它出色地利用了迷信和文化禁忌来对抗巨大的权力，并将幽默注入困境。
  </p>
  <p class="graf graf--p">
   如上所述，流浪狗的行动也是如此。
   <strong class="markup--strong markup--p-strong">
    当局疯狂地满街跑，试图抓住那些“参与抗议运动”的狗，因为狗脖子上挂在将军的照片
   </strong>
   。这些幽默的场景被迅速捕捉并上传到互联网，成为热门。
  </p>
  <p class="graf graf--p">

   <strong class="markup--strong markup--p-strong">
    3、用鲜花与仇恨言论作斗争
   </strong>
  </p>
  <p class="graf graf--p">
   众所周知，种族冲突在缅甸非常广泛，大多数佛教徒和少数民族穆斯林人口之间的冲突可以说举世闻名，特别是在若开邦。
  </p>
  <p class="graf graf--p">
   随着国家的开放，互联网的访问和使用呈指数级增长；据预测，到2015年底，缅甸的互联网和移动电话普及已经从2013年的10％上升到了50％。
  </p>
  <p class="graf graf--p">
   当然，
   <strong class="markup--strong markup--p-strong">

   </strong>

  </p>
  <p class="graf graf--p">
   在过去几年中，极端主义团体一直在通过传播错误信息呼吁暴力，特别是煽动对穆斯林罗兴亚少数民族的仇恨。
  </p>
  <p class="graf graf--p">
   Facebook 的极端主义帖子导致了无数的混乱和仇杀，政府关闭了该网站以试图限制暴力。
  </p>

   </strong>
   。

   2014年4月，Nay Phone Latt —— 一位著名的博客作者兼缅甸信息通信技术发展组织负责人，此前曾因撰写反政府文章被监禁四年 —— 他开展了一场以 #FlowerSpeech 为主题的打击仇恨言论的在线活动。
  </p>
  <p class="graf graf--p">
   活动家鼓励人们用龙船花发布自己的照片；作为缅甸人崇拜的国花，它们传统上代表着力量和诚实。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这场运动不止有 Facebook 帖子，还有贴纸、海报、T恤、腕带、和歌曲
   </strong>
   ，活动家们试图借此打击“危险言论”和种族间紧张关系的蔓延，并说服人们接受缅甸的多样性。
  </p>

   这是一个动员技巧。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、用笑声抵制镇压
   </strong>
  </p>
  <p class="graf graf--p">
   在缅甸，一个名为“胡子兄弟”的喜剧三人组提供了有力的反抗。
