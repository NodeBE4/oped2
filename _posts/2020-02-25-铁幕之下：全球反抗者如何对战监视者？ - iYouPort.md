   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️在情报饥渴和大规模监视痴迷的问题上，全球所有国家都一样 —— 它们以所谓的国家安全的名义从事大规模的信息收集和处理，同时又过分保密政府的阴暗面，缺乏监督和问责制。
  </p>
  <p class="graf graf--p">
   所有这些都强调了对
   <strong class="markup--strong markup--p-strong">
    有意义的控制机制
   </strong>
   的需求。
  </p>
  <p class="graf graf--p">
   迫切需要创建和执行易于访问，清晰且具有包容性的控制机制，以使人们知道自己的数据正在被如何收集、如何由公共机构处理和共享，尤其是在执法和情报机构内部；赋予人们有效的补救机制，并在必要时有权访问、纠正、甚至彻底删除其数据。
  </p>
  <p class="graf graf--p">
   为了拥有强大的基本隐私权，我们需要在机构何时以及如何使用我们的个人信息方面保持真正的透明。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">

   </strong>
   。
  </p>
  <p class="graf graf--p">

  <h3 class="graf graf--h4">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--h4-strong">
     阿根廷：没有到来的变革
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   来自阿根廷公民民主协会（ADC）的合作伙伴对阿根廷情报系统进行了研究，该情报系统早就应该进行改革以改善其工作模式和对基本权利的尊重。
  </p>
  <p class="graf graf--p">
   阿根廷任命新政府18个月后，情报系统对窃听活动的控制方式进行了改革，并任命了新的主管机构。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    然而，这些都未能带来必要的变革，仍然没有透明度、没有效率，没能将民主进程整合到系统内
   </strong>
   。
  </p>
  <p class="graf graf--p">
   公民权利组织 ADC 的报告显示，⚠️改革过程实际上已被用来加强政府对情报服务的控制，恰恰与民主背道而驰。
  </p>
  <p class="graf graf--p">
   阿根廷是有数据保护法的，也有宪法规定保护公民“私人行动的自由”；但是，该国同时还有一个国家生物识别数据库，称为SIBIOS；
   <strong class="markup--strong markup--p-strong">
    各种监听监视和管制模式与中国非常相似。
   </strong>
  </p>
  <p class="graf graf--p">
   早在美国之前，布宜诺斯艾利斯就宣布了将采用空中气球老大哥，以实现对某些区域的360度无死角实时监控。（去年美国采取的气球监视引起了公民极大的愤怒；中国早在2015年就在新疆采取了浮空器监视，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b5%ae%e7%a9%ba%e5%99%a8%e3%80%81%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e3%80%81%e5%b0%84%e9%a2%91%e3%80%81%e6%97%a0%e4%ba%ba%e6%9c%ba-%e5%94%90%e5%ae%81%e8%a1%97%e8%81%9a%e9%9b%86/" href="https://www.iyouport.org/%e6%b5%ae%e7%a9%ba%e5%99%a8%e3%80%81%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e3%80%81%e5%b0%84%e9%a2%91%e3%80%81%e6%97%a0%e4%ba%ba%e6%9c%ba-%e5%94%90%e5%ae%81%e8%a1%97%e8%81%9a%e9%9b%86/" rel="noopener noreferrer" target="_blank">
    在这里看到所有“热门”监视技术的报告
   </a>
   ）
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    间谍机构 ——
   </strong>
  </p>
  <p class="graf graf--p">
   情报秘书处由 Juan Perón 将军于1946 年
   <a class="markup--anchor markup--p-anchor" data-href="http://www.lanacion.com.ar/1764868-guia-para-entender-los-servicios-de-inteligencialos-duenos-de-todos-los-secretosconcepto-elasdequam-sim-zzrillan-veros-nostrudconceptotitulo-rit-aliquam-ad-tatver-illandre-magna-faccumdf" href="http://www.lanacion.com.ar/1764868-guia-para-entender-los-servicios-de-inteligencialos-duenos-de-todos-los-secretosconcepto-elasdequam-sim-zzrillan-veros-nostrudconceptotitulo-rit-aliquam-ad-tatver-illandre-magna-faccumdf" rel="noopener noreferrer" target="_blank">
    成立
   </a>
   ，当时是国家信息协调秘书处（SIDE）。
  </p>
  <p class="graf graf--p">
   它是一个民用情报机构，其任务是提供国内和国外情报。它在阿根廷军政府独裁时期（1976–1983年）演变成为一支
   <a class="markup--anchor markup--p-anchor" data-href="http://www.bbc.co.uk/news/world-latin-america-30995722" href="https://www.bbc.co.uk/news/world-latin-america-30995722" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     秘密警察部队
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，并被军政府用来追踪其对手，并监视包括工会主义者和其他左翼活动家在内的所有人，他们把活动家称为“颠覆分子”。
   </strong>
  </p>
  <p class="graf graf--p">
   该秘书处在1983年向民主过渡期间幸存下来了。
  </p>
  <p class="graf graf--p">
   但是，在军事独裁统治结束之后，新政府致力于明确重新定义政府与军方的关系，以防止军方再次镇压政治异议人士。
  </p>
  <p class="graf graf--p">

   <strong class="markup--strong markup--p-strong">
    身份证和数据库 ——
   </strong>
  </p>
  <p class="graf graf--p">
   1、国民身份证
  </p>
  <p class="graf graf--p">
   自1968年以来，阿根廷公民有义务从内政部下属的国家公民登记处（RENAPER）获得国民身份证（DNI）。
  </p>
  <p class="graf graf--p">
   2014年，RENAPER 发布了
   <a class="markup--anchor markup--p-anchor" data-href="http://servicios.infoleg.gob.ar/infolegInternet/anexos/235000-239999/237457/norma.htm" href="http://servicios.infoleg.gob.ar/infolegInternet/anexos/235000-239999/237457/norma.htm" rel="noopener noreferrer" target="_blank">
    第3020/14号决议
   </a>
   ，其中确定唯一有效的身份证明文件是*新的数字身份证*，该身份证将公民的生物识别数据数字化并收集到一个统一的数据库中。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     如今生物识别身份证甚至大范围普及了
    </strong>
    ，见《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e8%ba%ab%e4%bb%bd%e8%af%81/" href="https://www.iyouport.org/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e8%ba%ab%e4%bb%bd%e8%af%81/" rel="noopener noreferrer" target="_blank">
     生物识别身份证
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   自2009年11月以来，RENAPER
   <strong class="markup--strong markup--p-strong">
    已发行了超过4100万张生物识别身份证
   </strong>
   。
  </p>
  <p class="graf graf--p">
   有问题的数据库是
   <strong class="markup--strong markup--p-strong">
    联邦生物识别系统或 SIBIOS
   </strong>
   （Sistema Federal deIdentificaciónparamése la Seguridad）由
   <a class="markup--anchor markup--p-anchor" data-href="http://servicios.infoleg.gob.ar/infolegInternet/anexos/185000-189999/189382/norma.htm" href="http://servicios.infoleg.gob.ar/infolegInternet/anexos/185000-189999/189382/norma.htm" rel="noopener noreferrer" target="_blank">
    1766/11号行政命令
   </a>
   于2011年创建隶属于安全部。
  </p>
  <p class="graf graf--p">
   ⚠️SIBIOS 收集的生物识别数据主要包括指纹和面部图像。SIBIOS 的主要用户是联邦警察、国家宪兵队、国家海岸警卫队、机场警察、RENAPER 和国家移民局。
  </p>
  <p class="graf graf--p">
   自2017年4月起，
   <a class="markup--anchor markup--p-anchor" data-href="http://servicios.infoleg.gob.ar/infolegInternet/anexos/270000-274999/273446/norma.htm" href="http://servicios.infoleg.gob.ar/infolegInternet/anexos/270000-274999/273446/norma.htm" rel="noopener noreferrer" target="_blank">
    第243/17号行政命令
   </a>
   扩大了对 SIBIOS 的访问范围，使其“覆盖国家、省和自治市行政部门或司法机关的所有机构（…），以便他们能够
   <strong class="markup--strong markup--p-strong">
    实时进行生物特征查询
   </strong>
   ”。
  </p>
  <p class="graf graf--p">
   2018年12月， 议会批准了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hcdn.gob.ar/prensa/noticias/noticias-podio/noticias_0873.html" href="https://www.hcdn.gob.ar/prensa/noticias/noticias-podio/noticias_0873.html" rel="noopener noreferrer" target="_blank">
    一项法律
   </a>
   ， 以实施“国民学生证”，记录学生的出勤情况。
  </p>
  <p class="graf graf--p">
   2、选民登记
  </p>
  <p class="graf graf--p">
   在2017年8月18日的立法选举中，国家选举委员会（CNE）开展了一项生物识别选民身份登记的试点计划。
  </p>
  <p class="graf graf--p">
   通过
   <a class="markup--anchor markup--p-anchor" data-href="http://adc.org.ar/wp-content/uploads/2017/10/AE-050-17.pdf" href="http://adc.org.ar/wp-content/uploads/2017/10/AE-050-17.pdf" rel="noopener noreferrer" target="_blank">
    第50/17号特别协议
   </a>
   ，在选民识别过程中使用了生物识别数据，尤其是指纹。该过程是自愿的，但是收集了 362,460 人的生物特征数据。
  </p>
  <p class="graf graf--p">
   在公民权利组织 ADC 向CNE提出信息自由请求之后他们发现，此过程存在问题。
  </p>
  <p class="graf graf--p">
   例如，CNE 给出指定行动的原因之一是移民问题。然而，
   <a class="markup--anchor markup--p-anchor" data-href="http://adcdigital.org.ar/2017/10/18/prueba-biometrica-las-paso-la-camara-electoral-respondio-adc/" href="http://adcdigital.org.ar/2017/10/18/prueba-biometrica-las-paso-la-camara-electoral-respondio-adc/" rel="noopener noreferrer" target="_blank">
    正如ADC所言
   </a>
   ，这方面尚未被证明是选举进程正确发展的障碍。
  </p>
  <p class="graf graf--p">
   实际上，该项计划的目的之一就是在生物识别方面构建数据库。
  </p>
  <p class="graf graf--p">
   ⚠️就如中国的诸多实践一样，包括刷脸才能乘坐地铁、北京通，等等，他们的目的是
   <strong class="markup--strong markup--p-strong">
    采取一种人们难以拒绝的方式进行大规模收集
   </strong>
   。
  </p>

   <a class="markup--anchor markup--p-anchor" data-href="http://servicios.infoleg.gob.ar/infolegInternet/anexos/60000-64999/64790/texact.htm" href="http://servicios.infoleg.gob.ar/infolegInternet/anexos/60000-64999/64790/texact.htm" rel="noopener noreferrer" target="_blank">

  </p>

   2017年5月，安全部再次与 ENACOM 一起发起了
   <a class="markup--anchor markup--p-anchor" data-href="http://www.lanacion.com.ar/2028468-como-hay-que-registrar-un-celular-para-evitar-su-reventa-en-el-mercado-negro" href="http://www.lanacion.com.ar/2028468-como-hay-que-registrar-un-celular-para-evitar-su-reventa-en-el-mercado-negro" rel="noopener noreferrer" target="_blank">

   </a>

  </p>
  <p class="graf graf--p">

   <a class="markup--anchor markup--p-anchor" data-href="https://www.lanacion.com.ar/2200878-las-fuerzas-de-seguridad-sufrieron-un-ciberataque-y-ahora-investigan-su-origen" href="https://www.lanacion.com.ar/2200878-las-fuerzas-de-seguridad-sufrieron-un-ciberataque-y-ahora-investigan-su-origen" rel="noopener noreferrer" target="_blank">

   </a>
   ，政府内部消息人士指出这是DDoS攻击。调查的进一步细节未知，但已知开始的是刑事调查。
  </p>

    行业许可 ——
   </strong>
  </p>
  <p class="graf graf--p">

   2014年12月16日通过的《阿根廷数字法案》确定，联邦信息技术和通信管理局（FAITC）将继续执行对 SECOM 和 CNC 先前执行的任务。

  </p>

    促进和规范对包括宽带和互联网在内的信息通信技术和电信服务的获取；
   </li>
   <li class="graf graf--li">
    授予和宣布电信运营商的许可证和许可证的有效期；
   </li>
   <li class="graf graf--li">
    确保公司符合预期的质量标准。
   </li>

   <strong class="markup--strong markup--p-strong">

  <h3 class="graf graf--h4">
   <span style="color: #ff6600;">
    <strong>
     智利：数据保留与人权
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   来自智利的 Derechos Digitales 将他们的研究重点放在整个拉丁美洲的数据保留和手机注册方案上。
  </p>
  <p class="graf graf--p">
   一般而言，数据保留，尤其是移动用户数据，会严重影响人权，例如隐私权以及表达和思考的自由权利。
  </p>
  <p class="graf graf--p">
   数据保留政策的任何实施都应至少遵守一系列前提条件，以保证公民能充分享受这些人权，首先要证明这些政策是否必要和成比例。
  </p>
  <p class="graf graf--p">
   该研究还对墨西哥、巴西、哥伦比亚、秘鲁、阿根廷和智利进行了比较关于数据保留政策的分析，根据美洲背景下的国际人权标准对其进行了评估，设定了参数以评估美洲国家的类似举措，例如 作为一项法案，要求在巴拉圭建立移动用户生物识别数据库，该数据库最终被巴拉圭总统否决。
  </p>
  <p class="graf graf--p">
   智利宪法的第19条明确承诺保护公民享有私生活自由的权利。该宪法于2018年进行了改革，为个人数据保护添加了特定权利；而且，1999年， 智利成为第一个通过全面数据法的南美国家。

   </a>

  <h3 class="graf graf--h4">
   <span style="color: #ff6600;">
    <strong>
     南非：挑战老大哥
    </strong>
   </span>
  </h3>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “知情权运动（R2K）’’一直在多个方面挑战南非的国家监视。
  </p>
  <p class="graf graf--p">
   通过在全国范围内举行的多次简报、分析和讨论活动，他们的研究主要集中在 RICA（通信截取规定和与通信相关的信息的提供）法案上。
  </p>
  <p class="graf graf--p">
   在 RICA 规定中（与 Derechos Digitales 的研究一致）授权创建SIM卡注册数据库。
  </p>
  <p class="graf graf--p">
   研究报告首先指出：“严格执行 SIM 卡实名注册可能对穷人、我们国家的经济增长、每个人的私人信息的安全、以及最后但并非最不重要的 — 您信用记录，具有毁灭性影响”。
  </p>
  <p class="graf graf--p">
   这些问题是由 R2K 和 Privacy International 在向联合国普遍定期审议工作组提交的联合呈件中提出的，该过程引起联合国对南非提出了有关其监视做法的两项建议，并呼吁对其情报机构进行监督。
  </p>
  <p class="graf graf--p">
   南非共和国宪法第14条承诺保护隐私权；而且，《个人信息保护法》（2013年第4号法令）是规范南非数据保护的主要手段。
  </p>
  <p class="graf graf--p">
   但是，大规模法外监视被曝光时……哎呀。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    关于元数据的规定 ——
   </strong>
  </p>
  <p class="graf graf--p">
   主要的通信监视法 RICA 区分了通信内容和元数据，该法案将其定义为“与通信相关的信息”。
  </p>
  <p class="graf graf--p">
   ⚠️RICA要求所有电信和互联网服务提供商保留所有用户的元数据至少三年。防止元数据被拦截的保护措施远低于通信内容的保护措施。
  </p>
  <p class="graf graf--p">
   根据该法案，超过90天的元数据被归类为“存档”信息，⚠️执法机构可以向任何高等法院法官或地方法官寻求拦截指示。
  </p>
  <p class="graf graf--p">
   2017年5月
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dailymaverick.co.za/article/2017-05-10-op-ed-big-brother-is-watching-your-phone-call-records/#.WfCPVEzMxTY" href="https://www.dailymaverick.co.za/article/2017-05-10-op-ed-big-brother-is-watching-your-phone-call-records/#.WfCPVEzMxTY" rel="noopener noreferrer" target="_blank">
    进行的
   </a>
   一项
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dailymaverick.co.za/article/2017-05-10-op-ed-big-brother-is-watching-your-phone-call-records/#.WfCPVEzMxTY" href="https://www.dailymaverick.co.za/article/2017-05-10-op-ed-big-brother-is-watching-your-phone-call-records/#.WfCPVEzMxTY" rel="noopener noreferrer" target="_blank">
    调查
   </a>
   显示，根据 RICA 第15条，执法人员
   <strong class="markup--strong markup--p-strong">
    经常使用
   </strong>
   《
   <a class="markup--anchor markup--p-anchor" data-href="http://www.justice.gov.za/legislation/acts/1977-051.pdf" href="http://www.justice.gov.za/legislation/acts/1977-051.pdf" rel="noopener noreferrer" target="_blank">
    刑事诉讼法》
   </a>
   第205
   <a class="markup--anchor markup--p-anchor" data-href="http://www.justice.gov.za/legislation/acts/1977-051.pdf" href="http://www.justice.gov.za/legislation/acts/1977-051.pdf" rel="noopener noreferrer" target="_blank">
    条
   </a>
   来处理
   <strong class="markup--strong markup--p-strong">
    数量惊人通话数据记录
   </strong>
   。
  </p>
  <p class="graf graf--p">
   根据第205条的规定，执法人员可以向高等法院法官、地方法院检查官申请获取记录的授权。
  </p>
  <p class="graf graf--p">
   在权利组织“知情权”运动提出质疑之后，南非主要的移动服务提供商（Vodacom，MTN，Cell C 和 Telkom）透露，⚠️他们已经根据205条的执法要求

    <strong class="markup--strong markup--p-strong">
     集体披露了70,000个用户呼叫记录
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   ⚠️法律没有规定要通知某人其通讯已经被截获，而 RICA 还禁止任何人透露有关通信拦截方向的任何信息。这意味着用户不仅不会收到通知，服务提供商也不会发布合规性报告。
  </p>
  <p class="graf graf--p">

  <h3 class="graf graf--h4">
   <span style="color: #ff6600;">
    <strong>
     哥伦比亚：过渡时期的隐私权危机
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   随着2016年《和平协定》的签署，哥伦比亚面临新的挑战和问题。
  </p>
  <p class="graf graf--p">
   其中之一就是关于隐私在过渡环境中意味着什么。
  </p>
  <p class="graf graf--p">
   DeJusticia 一直在探索这些棘手的问题。已经出现的中心问题是，
   <strong class="markup--strong markup--p-strong">
    如何处理在过去的四十年中情报档案里注册的个人数据
   </strong>
   。
  </p>
  <p class="graf graf--p">
   哥伦比亚的宪法包含对隐私权的明确保护（1991年宪法第15条）；而且哥伦比亚有两项数据保护法：2008年第1266号法律、和2012年第1518号法律。哥伦比亚的数据保护机构是工商联合会（SIC）。
  </p>
  <p class="graf graf--p">
   但是，警察从事间谍活动的指控依旧令人困惑。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    监控能力 ——
   </strong>
  </p>
  <p class="graf graf--p">
   哥伦比亚既主持又参加了许多
   <strong class="markup--strong markup--p-strong">
    监视技术的贸易展览
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ISS World 是最大的贸易展览之一。哥伦比亚警察于2012年参加了 ISS World，其中有3家哥伦比亚公司展示了他们的产品：Biotekne SAS、哥伦比亚 ASOTO 技术集团、以及其 Esperanza 监视系统，还有 Inteligencia＆Tecnología（STAR）的供应商。
  </p>
  <p class="graf graf--p">
   ⚠️
   <a class="markup--anchor markup--p-anchor" data-href="http://www.export.gov.il/heb/Branches/Technologies/DefenceIndustries/Events/events,1418" href="http://www.export.gov.il/heb/Branches/Technologies/DefenceIndustries/Events/events,1418" rel="noopener noreferrer" target="_blank">
    由以色列驻波哥大大使馆赞助
   </a>
   的年度 Cibercolombia 贸易展览会主要展示以色列的监视产品。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    哥伦比亚的许多监视设备由国际公司、尤其是美国公司提供
   </strong>
   。
  </p>
  <p class="graf graf--p">
   据报道，在过去的十年中，向哥伦比亚情报部门的精英部队提供的美国资金、设备、甚至还有培训，
   <strong class="markup--strong markup--p-strong">
    被用来监视最高法院的大法官，时任总统阿尔瓦罗·乌里韦的政治反对者、和民间社会团体
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在哥伦比亚和美国中央情报局（CIA）
   <a class="markup--anchor markup--p-anchor" data-href="http://www.washingtonpost.com/sf/investigative/2013/12/21/covert-action-in-colombia/" href="https://www.washingtonpost.com/sf/investigative/2013/12/21/covert-action-in-colombia/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     针对FARC的
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    秘密行动中
   </strong>
   ，拦截通信至关重要。
  </p>
  <p class="graf graf--p">
   ⚠️以色列也是重要的监视技术供应国。
  </p>
  <p class="graf graf--p">
   至少从2005年起，以色列公司 Verint Systems 就提供了情报部门 DAS、DIPOL 和 DIJIN 所使用的关键侦听基础设施。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    国家情报部门 DAS 使用的软件是取证工具包（FTK），这是由位于美国的AccessData生产的计算机取证软件
   </strong>
   。该软件的远程功能使分析师能够进行远程分析，以法医技术分析实时数据。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    该软件还可用于解密 PGP 加密的磁盘
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    黄貂鱼 ——
   </strong>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://privacyinternational.org/node/991" href="https://privacyinternational.org/node/991" rel="noopener noreferrer" target="_blank">
    隐私国际组织的一项调查显示
   </a>
   ，
   <strong class="markup--strong markup--p-strong">
    许多公司
   </strong>
   在哥伦比亚提供IMSI捕手移动监视设备 。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    总部位于新西兰的 Spectra Group
   </strong>
   通过哥伦比亚公司 Maicrotel Ltda 在2005年9月向情报机构 DIPOL 提供了其 Laguna IMSI 捕获器。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    英国监视公司 Smith Myers
   </strong>
   制造的 Bulldog 和 Nesie 是在哥伦比亚出售的另外两个受欢迎的IMSI捕手。
  </p>
  <p class="graf graf--p">
   2010年，DAS 准备以
   <strong class="markup--strong markup--p-strong">
    25万美元以上
   </strong>
   的价格购买 Bulldog 拦截系统，并以32万美元以上的价格购买 Nesie 监视系统。
  </p>
  <p class="graf graf--p">
   Fiscalía 还计划以略高于280,000美元的价格购买 Bulldog 监视系统，DIJIN 在波哥大的部门也是如此。
  </p>
  <p class="graf graf--p">
   2014年，
   <strong class="markup--strong markup--p-strong">
    加拿大电信公司 Exfo 的芬兰分公司
   </strong>
   将其 NetHawk F10 IMSI 捕手出口到了哥伦比亚。
  </p>
  <p class="graf graf--p">
   据报道，哥伦比亚情报人员利用 IMSI 捕手追踪了哥伦比亚平民，并影响了周围居民的权利。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    入侵恶意软件和黑客攻击 ——
   </strong>
  </p>
  <p class="graf graf--p">
   意大利公司 Hacking Team 生产的入侵系统已被哥伦比亚警察收购。
  </p>
  <p class="graf graf--p">
   ⚠️该公司的远程控制系统（RCS）可以绕过常见的防病毒程序和加密，因此可用于劫持计算机和移动设备，同时使用户无法察觉。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    通过感染目标设备，RCS套件可以捕获目标设备上的所有数据，远程打开和关闭网络摄像头和麦克风，复制文件和输入密码
   </strong>
   。
  </p>
  <p class="graf graf--p">
   2014年，Hacking Team 就有一名驻哥伦比亚的现场工程师，并与哥伦比亚警察签有有效合同。
  </p>
  <p class="graf graf--p">
   Hacking Team 向DEA提供了其技术，据报道，
   <a class="markup--anchor markup--p-anchor" data-href="https://wikileaks.org/hackingteam/emails/?q=DEA+colombia&amp;mfrom=&amp;mto=&amp;title=&amp;notitle=&amp;date=&amp;nofrom=&amp;noto=&amp;count=50&amp;sort=0#searchresult" href="https://wikileaks.org/hackingteam/emails/?q=DEA+colombia&amp;mfrom=&amp;mto=&amp;title=&amp;notitle=&amp;date=&amp;nofrom=&amp;noto=&amp;count=50&amp;sort=0#searchresult" rel="noopener noreferrer" target="_blank">
    内部电子邮件
   </a>
   的曝光显示，该间谍部门
   <strong class="markup--strong markup--p-strong">
    使用间谍软件从美国驻波哥大大使馆进行监视
   </strong>
   。
  </p>
  <p class="graf graf--p">
   公民实验室于2014年
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2014/02/mapping-hacking-teams-untraceable-spyware/" href="https://citizenlab.ca/2014/02/mapping-hacking-teams-untraceable-spyware/" rel="noopener noreferrer" target="_blank">
    进行
   </a>
   的一项
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2014/02/mapping-hacking-teams-untraceable-spyware/" href="https://citizenlab.ca/2014/02/mapping-hacking-teams-untraceable-spyware/" rel="noopener noreferrer" target="_blank">
    调查
   </a>
   得出的结论是，⚠️
   <strong class="markup--strong markup--p-strong">
    自2012年以来，这些间谍技术已被发现实施针对新闻记者、维权人士、和人权捍卫者的袭击，并有证据表明这些技术至少在21个国家中得到了可疑部署，包括哥伦比亚
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Hacking Team 还与哥伦比亚警察有两个项目，其中一个似乎与PUMA监视系统有关。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    哥伦比亚军队还雇用了黑客
   </strong>
   。在陆军情报与反情报学校（Escuela de Inteligencia y Contrainteligencia）军队训练学员进行黑客入侵。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    网络拦截 ——
   </strong>
  </p>
  <p class="graf graf--p">
   哥伦比亚最可见的通讯拦截系统是埃斯佩兰萨（Sistema Esperanza）；它得到了美国毒品执法机构（DEA）的大力支持。
  </p>
  <p class="graf graf--p">
   总检察长办公室（FiscalíaGeneral de laNación，“Fiscalía”）管理和维护该平台，⚠️该平台可以获取移动和固定电话呼叫的数据和内容。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    各个执法机构都可以访问的拦截系统 Esperanza 已与该国的电信运营商相连接
   </strong>
   。它用于逐案获取司法起诉的证据。
  </p>
  <p class="graf graf--p">
   ⚠️PUMA 于2007年从以色列监视公司 Verint Systems Ltd 收购，并由哥伦比亚公司 Compania Comercial Curacao de Colombia 维护。2013年，警方提出了扩展该系统的建议，
   <strong class="markup--strong markup--p-strong">
    声称扩展后的 PUMA 能够捕获到三倍多的电话和数据
   </strong>
   。
  </p>
  <p class="graf graf--p">
   扩展后的 PUMA 包括一个针对互联网服务提供商（ISP）的监视模块，以及遍布全国的多达700个工作站。
  </p>
  <p class="graf graf--p">
   扩展合同是与另一家以色列监视公司 NICE Systems 以及哥伦比亚公司 Eagle Comercial 签订的。然而，Fiscalía 和警方之间在管理上的分歧使扩张停滞不前，该项目被搁置。
  </p>
  <p class="graf graf--p">
   此外，警察情报局（Direcciónde Inteligencia Policial，“ DIPOL”）还获得并部署了
   <strong class="markup--strong markup--p-strong">
    自己的大规模自动通信监视系统，即 综合记录系统（“ IRS”）
   </strong>
   。
  </p>
  <p class="graf graf--p">
   IRS 创立于2005年，负责监控E1线路和3G移动电话的大量通信流量。
  </p>
  <p class="graf graf--p">
   隐私国际组织对这些技术的分析是，⚠️该监视系统每天能够收集1亿个呼叫数据记录，每天能够拦截2000万条SMS。然后处理庞大的数据存储、并将其与其他类型的数据（
   <strong class="markup--strong markup--p-strong">
    包括图像、视频和生物特征详细信息
   </strong>
   ）组合。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    监视实例 ——
   </strong>
  </p>
  <p class="graf graf--p">
   自90年代以来，通讯拦截丑闻就是哥伦比亚维稳政治的一个特征。
  </p>
  <p class="graf graf--p">
   ⚠️至少从1971年以来，当局
   <a class="markup--anchor markup--p-anchor" data-href="http://www.eldiarioexterior.com/articulo.asp?idarticulo=%2026464&amp;accion=ext" href="http://www.eldiarioexterior.com/articulo.asp?idarticulo=%2026464&amp;accion=ext" rel="noopener noreferrer" target="_blank">
    一直在窃听电话
   </a>
   线，近年来，监视在针对哥伦比亚革命武装力量的军事行动中发挥了重要作用。
  </p>
  <p class="graf graf--p">
   据报道，在2011年，被截获的电话对找到哥伦比亚革命武装力量的最高领导人
   <a class="markup--anchor markup--p-anchor" data-href="http://www.bbc.com/news/world-15604456" href="https://www.bbc.com/news/world-15604456" rel="noopener noreferrer" target="_blank">
    Alfonso Cano
   </a>
   至关重要，后者已经在军事袭击中
   <strong class="markup--strong markup--p-strong">
    丧生
   </strong>
   。
  </p>
  <p class="graf graf--p">
   据报道，军方使用 Esperanza 拦截系统找到了哥伦比亚革命武装力量的军事领导人
   <a class="markup--anchor markup--p-anchor" data-href="http://www.semana.com/nacion/articulo/chuzadas-asi-fue-la-historia/376548-3" href="http://www.semana.com/nacion/articulo/chuzadas-asi-fue-la-historia/376548-3" rel="noopener noreferrer" target="_blank">
    Mono Jojoy
   </a>
   ，后者后来也被
   <strong class="markup--strong markup--p-strong">
    杀害
   </strong>
   。
  </p>
  <p class="graf graf--p">
   不仅如此，关于非法拦截私人通讯的报道充斥着
   <strong class="markup--strong markup--p-strong">
    法外失踪和杀戮
   </strong>
   的指控。有不同的机构参与了这些非法拦截。
  </p>
  <p class="graf graf--p">
   在一个
   <a class="markup--anchor markup--p-anchor" data-href="http://www.acnur.org/t3/uploads/media/COI_53.pdf" href="http://www.acnur.org/t3/uploads/media/COI_53.pdf" rel="noopener noreferrer" target="_blank">
    著名的案例
   </a>
   中，超过2,000条电话线被军事警察 GAULA 非法窃听。
  </p>
  <p class="graf graf--p">
   然而，最臭名昭著的拦截丑闻涉及DAS，并于2009年2月被 Semana 揭露。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    DAS 的特殊战略情报小组对大约600名公众人物进行了有针对性的监视，其中包括议员、记者、人权活动家、和律师、以及法官
   </strong>
   。
  </p>
  <p class="graf graf--p">
   根据 Fiscalía 进行调查期间检索到的文件，
   <strong class="markup--strong markup--p-strong">
    DAS截获了电话、电子邮件流量、以及国际和国内的联系人列表，并使用这些信息来编制目标的心理概况，以便对受试者及其家人（包括儿童）进行贴身监视
   </strong>
   。
  </p>
  <p class="graf graf--p">
   通信监视是DAS滥用的关键。
   <strong class="markup--strong markup--p-strong">
    记者霍尔曼·莫里斯（Hollman Morris）的电话受到近乎不间断的监视
   </strong>
   。莫里斯后来多次被迫流放。
  </p>
  <p class="graf graf--p">
   克劳迪亚·杜克（Claudia Duque）是一名律师和新闻工作者，以前曾与 CCAJAR 律师团队合作，她虽幸免于绑架事件的发生，但是收到了电话暴力威胁。⚠️DAS关于她的档案中包含有关通讯和物理监视的大量证据。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    身份追踪 ——
   </strong>
  </p>
  <p class="graf graf--p">
   1、身份证和数据库
  </p>
  <p class="graf graf--p">
   哥伦比亚的人口登记处成立于1938年，由
   <a class="markup--anchor markup--p-anchor" data-href="http://www.registraduria.gov.co/" href="http://www.registraduria.gov.co/" rel="noopener noreferrer" target="_blank">
    国家民事登记处
   </a>
   （Registraduríadel Estado Civil）管理。
  </p>
  <p class="graf graf--p">
   注册表由
   <a class="markup--anchor markup--p-anchor" data-href="http://www.alcaldiabogota.gov.co/sisjur/normas/Norma1.jsp?i=8256" href="http://www.alcaldiabogota.gov.co/sisjur/normas/Norma1.jsp?i=8256" rel="noopener noreferrer" target="_blank">
    三个主要部分
   </a>
   组成：出生、婚姻和死亡。
  </p>
  <p class="graf graf--p">
   出生登记处除了记录一般的识别性信息（例如姓名、出生日期、父母的姓名、和身份，职业、以及负责出生时医疗程序的医生）以外，还记录了新生儿的脚印。
  </p>

   注册表信息由
   <a class="markup--anchor markup--p-anchor" data-href="http://www.registraduria.gov.co/-Consultas-ANI-.html" href="http://www.registraduria.gov.co/-Consultas-ANI-.html" rel="noopener noreferrer" target="_blank">
    国家身份档案
   </a>
   （Archivo Nacional deIdentificación ）和
   <a class="markup--anchor markup--p-anchor" data-href="http://consultasrc.registraduria.gov.co:8080/ProyectoSCCRC/" href="http://consultasrc.registraduria.gov.co:8080/ProyectoSCCRC/" rel="noopener noreferrer" target="_blank">
    民事登记数据库
   </a>
   （Base de datos del Registro Civil）管理。即使注册表是公开的，法律也对发行副本施加了限制，以保护隐私权。但是，与国家民事登记处达成协议后，官方和私人团体均可查阅国家身份证档案库。
  </p>
  <p class="graf graf--p">
   自1970年以来，哥伦比亚的每个新生儿都被分配了一个唯一的识别码。
  </p>
  <p class="graf graf--p">
   在2000年之前，该数字由两部分组成：第一部分是日期（例如，对于1984年7月1日出生的人而言为 840701），第二部分是5位数字，用于区分同日出生的所有人，并允许性别识别。
  </p>
  <p class="graf graf--p">
   自2000年以来，标识符为10位数字。这些号码中的许多号码都分配给了每个登记处，登记处则将这些号码分配给新生儿或要求提供身份证的任何人。
  </p>
  <p class="graf graf--p">
   哥伦比亚的成年年龄为
   <a class="markup--anchor markup--p-anchor" data-href="http://www.secretariasenado.gov.co/senado/basedoc/constitucion_politica_1991_pr003.html#98" href="http://www.secretariasenado.gov.co/senado/basedoc/constitucion_politica_1991_pr003.html#98" rel="noopener noreferrer" target="_blank">
    18岁
   </a>
   ，这意味着该人具有充分的法律行为能力，可以在公开选举中投票。证明这种情况的媒介是
   <a class="markup--anchor markup--p-anchor" data-href="http://www.registraduria.gov.co/-Cedula-de-ciudadania-.html" href="http://www.registraduria.gov.co/-Cedula-de-ciudadania-.html" rel="noopener noreferrer" target="_blank">
    身份证
   </a>
   （“céduladeciudadanía”）。
  </p>
  <p class="graf graf--p">
   2、选民登记
  </p>
  <p class="graf graf--p">
   为了确保投票所需的身份证件，
   <a class="markup--anchor markup--p-anchor" data-href="http://www.secretariasenado.gov.co/senado/basedoc/ley_1475_2011_pr001.html#47" href="http://www.secretariasenado.gov.co/senado/basedoc/ley_1475_2011_pr001.html#47" rel="noopener noreferrer" target="_blank">
    国家民政登记处
   </a>
   进行了选举普查，并添加了未出现在登记中的满足合法年龄的人。
  </p>
  <p class="graf graf--p">
   登记处还删除了死者的身份证、在国外的军事人员的身份证、以及由于确认的刑事判决而无法行使公共权利的其他人的身份证、和其他不合法身份证。
  </p>
  <p class="graf graf--p">
   人口普查仅包含身份证号码。
  </p>
  <p class="graf graf--p">
   3、SIM卡实名注册
  </p>
  <p class="graf graf--p">
   目前，在哥伦比亚，SIM卡注册不是强制性的，但是有IMEI注册系统。
  </p>
  <p class="graf graf--p">
   自2011年以来（根据2011年
   <a class="markup--anchor markup--p-anchor" data-href="https://www.mintic.gov.co/portal/604/articles-3558_documento.pdf" href="https://www.mintic.gov.co/portal/604/articles-3558_documento.pdf" rel="noopener noreferrer" target="_blank">
    第2630号法令
   </a>
   ） ，哥伦比亚政府一直在开发手机注册系统，旨在避免和阻止手机盗窃。
  </p>
  <p class="graf graf--p">
   该系统有两个主要部分：IMEI数据库和验证方案。
  </p>
  <p class="graf graf--p">
   IMEI数据库 ——
  </p>
  <p class="graf graf--p">
   有两种数据库：白名单数据库和黑名单数据库。
  </p>
  <p class="graf graf--p">
   白名单数据库包含允许在哥伦比亚移动网络上有效的IMEI。除了注册导入的设备外，该数据库还将IMEI与用户身份直接联系起来。因此，个人信息包括，例如：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    全名;
   </li>
   <li class="graf graf--li">
    ID类型和编号；
   </li>
   <li class="graf graf--li">
    地址;
   </li>
   <li class="graf graf--li">
    联系电话。
   </li>
  </ul>
  <p class="graf graf--p">
   电信运营商需要在以下任何来源或数据库上验证此信息：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    国家身份证档案；
   </li>
   <li class="graf graf--li">
    民事国家登记处；
   </li>
   <li class="graf graf--li">
    信用历史和风险数据库；
   </li>
   <li class="graf graf--li">
    运营商收集的数据库。
   </li>
  </ul>
  <p class="graf graf--p">
   黑名单数据库中包含：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    该设备被报告为失窃或丢失；
   </li>
   <li class="graf graf--li">
    该设备的IMEI被认为是不规则的：格式不对，也没有合格证、或重复的；
   </li>
   <li class="graf graf--li">
    设备未在白名单数据库中注册的。
   </li>
  </ul>
  <p class="graf graf--p">
   ⚠️警察和司法机关以及ICT部门和电信监管机构等行政机关
   <strong class="markup--strong markup--p-strong">
    都可以
   </strong>
   访问包含此数据的数据库。此访问没有任何形式的监督。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    验证系统 ——
   </strong>
  </p>
  <p class="graf graf--p">
   由于IMEI可以重新编辑，因此对于监管机构和政府而言，上述两个数据库不足以防止手机失窃。为了调试数据库并最终确保每个合法IMEI都在白名单数据库中注册（这意味着用户标识），他们建立了一个验证系统。
  </p>
  <p class="graf graf--p">
   验证系统要求所有运营商注意其网络上的每个IMEI活动。这是通过呼叫详细日志分析来实现的。该过程分为三个步骤：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    内部网络分析：每个运营商分析其CDR；
   </li>
   <li class="graf graf--li">
    网络间分析：每个运营商将其CDR发送给运营商选择的第三方，第三方对其进行分析以发现重复项；
   </li>
   <li class="graf graf--li">
    控制措施：对于每种类型的不规则IMEI（无格式、未经认证、重复），操作人员应采取监管机构定义的措施。
   </li>
  </ul>
  <p class="graf graf--p">
   需要注意到，白名单数据库中包含IMEI、用户真实身份和电话号码的关联。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    智能警务 ——
   </strong>
  </p>
  <p class="graf graf--p">
   数十年的武装冲突导致哥伦比亚的监视技术产业蓬勃发展，尤其是CCTV、视频监视和生物识别技术。
  </p>
  <p class="graf graf--p">
   2016年10月，麦德林市宣布已
   <strong class="markup--strong markup--p-strong">
    从日本NEC公司购买了
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://asia.nikkei.com/Business/Companies/NEC-surveillance-system-detects-soccer-hooligans-at-Colombia-stadium" href="http://asia.nikkei.com/Business/Companies/NEC-surveillance-system-detects-soccer-hooligans-at-Colombia-stadium" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     生物特征面部识别技术
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。
   </strong>
   据报道，竞技场运营商已创建了一个足球迷黑名单，NEC系统将使用该黑名单与入口处的监控摄像头捕获的面孔进行比较。
  </p>
  <p class="graf graf--p">
   更广泛地说，在2017年期间，负责组织、管理和规范哥伦比亚职业足球锦标赛（DIMAYOR）的私人实体在该国体育场内推进了其监视维稳战略。该策略包括为引入照片ID卡，以及安装带有生物识别控制如指纹识别设备和面部识别的摄像头和门禁。
  </p>
  <p class="graf graf--p">
   在这里看到详细报告：
   <a class="markup--anchor markup--p-anchor" data-href="https://docs.wixstatic.com/ugd/0d468a_03c8a9167bd040e4a16739460bb21a51.pdf" href="https://docs.wixstatic.com/ugd/0d468a_03c8a9167bd040e4a16739460bb21a51.pdf" rel="noopener noreferrer" target="_blank">
    “Acceso a los archivos de inteligencia y contrainteligencia en el marco del posacuerdo” (in Spanish)
   </a>
  </p>
  <p>
   <img alt="" class="aligncenter wp-image-9750 jetpack-lazy-image" data-lazy-sizes="(max-width: 750px) 100vw, 750px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-4.png?resize=750%2C1063&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-4.png?w=285&amp;ssl=1 285w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-4.png?resize=212%2C300&amp;ssl=1 212w" data-recalc-dims="1" height="1063" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-4.png?resize=750%2C1063&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="750"/>
   <noscript>
    <img alt="" class="aligncenter wp-image-9750" data-recalc-dims="1" height="1063" sizes="(max-width: 750px) 100vw, 750px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-4.png?resize=750%2C1063&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-4.png?w=285&amp;ssl=1 285w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/10/7-4.png?resize=212%2C300&amp;ssl=1 212w" width="750"/>
   </noscript>
  </p>
  <h3 class="graf graf--h4">
   <span style="color: #ff6600;">
    <strong>
     如何在联合国争取隐私权
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   《世界人权宣言》（第12条）和《公民权利和政治权利国际公约》（第17条）都包含了隐私权。
  </p>
  <p class="graf graf--p">
   近年来隐私权在联合国已经成为中心焦点。联合国大会在2013年和2014年通过了有关数字时代隐私权的决议，继而人权理事会建立了促进和保护隐私权的专门机制，2015年引入了联合国隐私权问题特别报告员。
  </p>
  <p class="graf graf--p">
   此外，其他特别报告员更加关注非法干扰隐私的影响，特别是在通信监视的情况下；人权事务委员会加强了对《公民权利和政治权利国际公约》缔约国隐私权的审查；普遍定期审议机制的建议开始引起人们对监视和人权的关注。
  </p>
  <p class="graf graf--p">
   隐私国际组织及其合作伙伴组织对这些事态的发展起到了重要作用，向这些人权机构提供了主题简报，并呼吁联合国更多地关注隐私权。
  </p>
  <p class="graf graf--p">
   以下是一份简报，
   <strong class="markup--strong markup--p-strong">
    着重介绍了非政府组织有机会向某些选定的联合国人权机构提出与隐私权有关的问题，这些机构的任务和能力是监测、提供建议和补救措施。
   </strong>
  </p>
  <p class="graf graf--p">
   该简报会根据国际隐私组织的经验提供了一些示例，并指向其他资源和指南。
  </p>
  <p class="graf graf--p">
   尽管这些指南的重点是非政府组织的工作，但是完全可以通过其他民间社会参与者以及个人向联合国人权机制提供信息。
  </p>
  <p class="graf graf--p">
   希望能对中国社会的反抗者有所帮助。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国是全球最大的监视国家之一
   </strong>
   ，在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%92%8c%e4%b8%ad%e5%9b%bd%e5%9c%a8%e7%9b%91%e8%a7%86%e5%85%ac%e6%b0%91%e6%96%b9%e9%9d%a2%e5%bc%95%e9%a2%86%e5%85%a8%e7%90%83/" href="https://www.iyouport.org/%e7%be%8e%e5%9b%bd%e5%92%8c%e4%b8%ad%e5%9b%bd%e5%9c%a8%e7%9b%91%e8%a7%86%e5%85%ac%e6%b0%91%e6%96%b9%e9%9d%a2%e5%bc%95%e9%a2%86%e5%85%a8%e7%90%83/" rel="noopener noreferrer" target="_blank">
    美国和中国在监视公民方面引领全球
   </a>
   》；
   <strong class="markup--strong markup--p-strong">
    但中国的监视状态绝不是特殊的
   </strong>
   —— 您在上面看到的那些案例中每一个都和中国很相似，甚至更严重，但是都得到了联合国的重视和帮助。
  </p>
  <p class="graf graf--p">
   此前我们
   <strong class="markup--strong markup--p-strong">
    已经发布过一个经验
   </strong>
   《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%9d%9e%e6%94%bf%e5%ba%9c%e7%bb%84%e7%bb%87%e5%a6%82%e4%bd%95%e4%b8%8e%e8%81%94%e5%90%88%e5%9b%bd%e4%ba%ba%e6%9d%83%e5%a7%94%e5%91%98%e4%bc%9a%e5%90%88%e4%bd%9c%e4%bb%a5%e6%94%b9%e5%96%84%e5%9b%bd/" href="https://www.iyouport.org/%e9%9d%9e%e6%94%bf%e5%ba%9c%e7%bb%84%e7%bb%87%e5%a6%82%e4%bd%95%e4%b8%8e%e8%81%94%e5%90%88%e5%9b%bd%e4%ba%ba%e6%9d%83%e5%a7%94%e5%91%98%e4%bc%9a%e5%90%88%e4%bd%9c%e4%bb%a5%e6%94%b9%e5%96%84%e5%9b%bd/" rel="noopener noreferrer" target="_blank">
    非政府组织如何与联合国人权委员会合作以改善国家的人权状况
   </a>
   ？》；下面是隐私国际组织介绍的经验 —— 这是全球最好的隐私权组织之一：《
   <a class="markup--anchor markup--p-anchor" data-href="https://privacyinternational.org/sites/default/files/2017-12/UN_Guide_Final.pdf" href="https://privacyinternational.org/sites/default/files/2017-12/UN_Guide_Final.pdf" rel="noopener noreferrer" target="_blank">
    HOW TO TALK ABOUT THE
    <br/>
    RIGHT TO PRIVACY AT THE UN
   </a>
   》。⚪️
  </p>
