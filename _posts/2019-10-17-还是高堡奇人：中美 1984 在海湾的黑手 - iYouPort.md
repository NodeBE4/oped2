  </p>
  <p class="graf graf--p">
   巴勒斯坦计算机工程师 Ahmad 为一家由阿联酋政府高级官员控制的公司提供技术支持，而后在阿布扎比他被拘留几个月，没有任何解释。
  </p>
  <p class="graf graf--p">
   他说他被蒙住眼睛并带到一个他现在仍然不知道是什么地方的秘密监狱。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在审讯期间 —— 警察似乎相信他可以获得敏感信息 —— 他被绑在电椅上，一次电击20秒钟，当他在痛苦中昏倒时，他们在他的脸上泼冷水，强迫他恢复意识。
   </strong>
  </p>
  <p class="graf graf--p">
   Ahmad 说，
   <strong class="markup--strong markup--p-strong">
    警察做的第一件事就是没收他的手机和电脑。
   </strong>
  </p>
  <p class="graf graf--p">
   警察要求密码，他给了警察不正确的密码，希望能够拖延时间。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    但是审讯人员后来告诉他，无论如何他们都已经打开了这些设备。他说，他相信他们真的做到了无需密码破解—— 因为其中一名官员开始和他开玩笑，讲述一位女性朋友，就是在他手机上的照片。这位官员说，“她很性感”。
   </strong>
  </p>
  <p class="graf graf--p">
   在他第一次被拘留五个月后，Ahmad 被蒙住眼睛，开车送回到阿布扎比的机场。
  </p>
  <p class="graf graf--p">
   他被驱逐到黎巴嫩，没有任何解释。警察甚至道歉并告诉他，他是无辜的。
  </p>
  <p class="graf graf--p">
   Ahmad 最终搬到了瑞典，他现在住在那里；在他成为瑞典公民之后，他开始撰写关于他在狱中的苦难。
  </p>
  <p class="graf graf--p">
   阿联酋的前雇主给他发了一封信，他与 BuzzFeed News 分享了这封信，信中声称根据阿联酋的网络犯罪法可能会因涉嫌窃取该公司的机密信息而起诉他。
  </p>
  <p class="graf graf--p">
   虽然他现在住在国外，但一直在警惕着自己的设备被监视的可能性。
  </p>
  <p class="graf graf--p">
   去年，
   <strong class="markup--strong markup--p-strong">
    该国最著名的人权活动家之一 Ahmed Mansoor 被判处10年监禁，并因在 Facebook 和 Twitter 上发布的一些帖子被罚款约27万美元
   </strong>
   ，法庭认定这些帖子“损害了阿联酋在国外的声誉及其“
   <a class="markup--anchor markup--p-anchor" data-href="https://gulfnews.com/uae/crime/man-gets-10-years-jail-for-defaming-uae-through-social-media-1.2229192" href="https://gulfnews.com/uae/crime/man-gets-10-years-jail-for-defaming-uae-through-social-media-1.2229192" rel="noopener noreferrer" target="_blank">
    社会和谐统一
   </a>
   ”。
  </p>
  <p class="graf graf--p">
   当局可以根据该国的网络犯罪法逮捕任何在互联网上发帖的人，该法禁止任何人在社交媒体上的所谓“诽谤言论”。
  </p>
  <p class="graf graf--p">
   Mansoor
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/ahmed_mansoor?lang=en" href="https://twitter.com/ahmed_mansoor?lang=en" rel="noopener noreferrer" target="_blank">
    在他的推特账户中
   </a>
   的陈述引起了全世界对整个中东地区侵犯人权行为的关注，
   <strong class="markup--strong markup--p-strong">
    他的政府一直在
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2016/08/million-dollar-dissident-iphone-zero-day-nso-group-uae/" href="https://citizenlab.ca/2016/08/million-dollar-dissident-iphone-zero-day-nso-group-uae/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     严密监视他
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，他的电子设备一直是间谍软件的攻击目标。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “Mansoor 毫不畏言他所做的事和他为什么这样做，但他非常清楚自己受到监视。他会在酒店大堂与人会面，但他会把手机放在家里，“曾在该地区为人权观察组织工作的研究员 Nicholas McGeehan 说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当局已经囚禁了所有这些人、或对其进行了恐吓，骚扰他们以令他们保持沉默”。
  </p>
  <p class="graf graf--p">
   自由言论倡导者和前阿联酋囚犯说，对社交媒体的控制以及当局可能在窥探自己手机的认识已经让人感到脊背发冷。人权组织表示，他们担心在物理空间中越来越多的监控技术可能会使问题变得更加严重。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “政府已表现出要压制一切和平反对的意愿和愿望，”公民实验室的 Marczak 说，他曾与 Mansoor 合作过。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果他们有强大的工具来跟踪你在户外的一举一动，并且他们有强大的间谍软件可以破解你家里的所有设备，那么你就几乎没有任何地方可以躲藏了。”⚪️
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.buzzfeednews.com/article/meghara/dubai-facial-recognition-technology-ibm-huawei-hikvision" href="https://www.buzzfeednews.com/article/meghara/dubai-facial-recognition-technology-ibm-huawei-hikvision" rel="noopener noreferrer" target="_blank">
    Face recognition has been banned in San Francisco and linked by Alexandria Ocasio-Cortez to a rise in global fascism. But it’s being marketed in Dubai, which has spied on hundreds of dissidents, by American and Chinese tech giants.
   </a>
  </p>
