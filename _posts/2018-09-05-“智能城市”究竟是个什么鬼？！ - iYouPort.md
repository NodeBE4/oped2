 <div class="entry-content">
  <ul>
   <li>
    <em>
     如今很多人都知道了，“智能城市”和全方面的侵入性监视之间没什么不同。但究竟被监视的都是些什么内容？要多丰富的数据才能把 IBM、谷歌等监视资本主义巨头美成那样？以及我们可以怎么办
    </em>
   </li>
  </ul>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--figure" id="df41">
   “智能城市”只是一个营销术语，用于定义技术的使用 — — 特别是数据收集技术 — — 他们告诉你，这是为了“改善城市的运作”。就像扎克伯格告诉你：“监视资本主义是为了改善服务质量”。智能城市所基于的理念是，“地方政府对城市居民了解得越多，他们提供的服务就越好”。然而，现实情况是，该营销术语对于从公司到政府的不同参与者来说意味着不同的事物。
  </p>
  <p class="graf graf--p graf-after--p" id="c672">
   世界银行提出了智能城市的两种可能性定义。第一个是“
   <a class="markup--anchor markup--p-anchor" data-href="http://www.worldbank.org/en/topic/ict/brief/smart-cities" href="http://www.worldbank.org/en/topic/ict/brief/smart-cities" rel="noopener nofollow noreferrer" target="_blank">
    技术密集型城市
   </a>
   ，无处不在的传感器和高效的公共服务，这得益于数千个互联设备实时收集的信息。”第二个是“ 一个
   <a class="markup--anchor markup--p-anchor" data-href="http://www.worldbank.org/en/topic/ict/brief/smart-cities" href="http://www.worldbank.org/en/topic/ict/brief/smart-cities" rel="noopener nofollow noreferrer" target="_blank">
    培养公民和政府之间良好关系的城市
   </a>
   ” — 利用现有技术。他们依靠公民的反馈来帮助改善服务提供，并建立收集这些信息的机制。“你信吗？相信政府在全心全意为人民服务？即便是民选政府也应该随时接受人民的监督，而不是寄希望于他们主动正义。权力集中的结果只能是权力滥用。
  </p>
  <p class="graf graf--p graf-after--p" id="aba3">
   另一方面，大型科技公司也给出了关于智能城市的定义，例如 IBM 的“城市工作新方式”、或西门子的“ 露天电脑 ”。
  </p>
  <p class="graf graf--p graf-after--p" id="c576">
   智能城市项目的范围从小型初创企业创建的应用程序（例如本地服务级）到非常广泛的基础设施，由 IBM，Oracle 和 Microsoft 等公司设计，
   <strong class="markup--strong markup--p-strong">
    集中收集公民数据并提供城市数据分析服务
   </strong>
   。所有这一切都共同依赖于数据：无论是生成和收集数据还是基于已有数据提供服务。
  </p>
  <p class="graf graf--p graf-after--p" id="f7ab">
   <strong class="markup--strong markup--p-strong">
    智能城市是如何发展的？
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="e2fa">
   虽然智能城市极为多样化，但它们最常用于三个部门：运输、能源和健康。看起来没什么要命的？恰恰相反，这三点足够控制住你的整个人生。
  </p>
  <p class="graf graf--p graf-after--p" id="9dd3">
   <strong class="markup--strong markup--p-strong">
    — — 运输 — —
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="04f3">
   由英国商业创新与技能部委托编写的一份研究论文将智能交通系统描述为“通过信息收集、存储、操控和随后的传播，为旅行者/客户提供附加价值、引导、提高安全性或效率“。
  </p>
  <p class="graf graf--p graf-after--p" id="e337">
   智能公共交通项目通常很大程度上依赖于“智能卡”。智能卡是旅行通行证，其中集成了电路。这种东西是
   <strong class="markup--strong markup--p-strong">
    识别个人身份
   </strong>
   用的：要么是因为必须以您自己的真实身份注册它们（如巴黎的“
   <a class="markup--anchor markup--p-anchor" data-href="http://www.navigo.fr/je-gere-ma-carte/" href="http://www.navigo.fr/je-gere-ma-carte/" rel="noopener nofollow noreferrer" target="_blank">
    Navigo ”卡
   </a>
   ），要么是因为它们可以通过其他方式追溯到您的真实身份 — 例如，需要您使用信用卡充值。
  </p>
  <p class="graf graf--p graf-after--p" id="c49d">
   今天，智能城市的发展进一步向“一卡通”迈进了。人们现在可以使用非接触式信用卡或智能手机支付
   <a class="markup--anchor markup--p-anchor" data-href="https://tfl.gov.uk/fares-and-payments/contactless/other-methods-of-contactless-payment" href="https://tfl.gov.uk/fares-and-payments/contactless/other-methods-of-contactless-payment" rel="noopener nofollow noreferrer" target="_blank">
    伦敦
   </a>
   或
   <a class="markup--anchor markup--p-anchor" data-href="https://techcrunch.com/2017/08/15/beijing-public-transport-app/" href="https://techcrunch.com/2017/08/15/beijing-public-transport-app/" rel="noopener nofollow noreferrer" target="_blank">
    北京
   </a>
   等城市的公共交通工具，或者以其他形式关联身份，如布宜诺斯艾利斯市的学生证。与系统地收集公共交通数据一样，越来越多的城市也在开发
   <strong class="markup--strong markup--p-strong">
    自行车共享
   </strong>
   计划。中国有什么大家都知道，在纽约，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.citibikenyc.com/system-data" href="https://www.citibikenyc.com/system-data" rel="noopener nofollow noreferrer" target="_blank">
    Citibike 一直在分享
   </a>
   他们收集的自行车用户数据，从而揭示收集到的数据的范围：旅行持续时间; 开始和停止时间，日期和地点; 性别和用户的出生年份等等一大堆信息。
  </p>
  <p class="graf graf--p graf-after--p" id="6c0d">
   智能城市也在收集道路使用者的数据，使用
   <a class="markup--anchor markup--p-anchor" data-href="https://aclum.org/uncategorized/cameras-used-electronic-toll-collection-raise-privacy-concerns/" href="https://aclum.org/uncategorized/cameras-used-electronic-toll-collection-raise-privacy-concerns/" rel="noopener nofollow noreferrer" target="_blank">
    电子收费
   </a>
   来检测汽车何时开车。
  </p>
  <p class="graf graf--p graf-after--p" id="4d24">
   ⚠️ 在这样的城市里，你的一举一动都在监视者的掌握之下，你去了哪里、见了谁、持续多长时间等等，显示你的社交关系图谱；你最喜欢去哪里、多久去一次，显示了你的偏好兴趣，这些数据汇总起来就可以全面操纵你了。在这样的地方，汽车和自行车都不再能够帮你逃离危险，反而帮助了追踪你的人。
   <strong class="markup--strong markup--p-strong">
    当中国的共享单车出现时，人们在讨论路权、道德问题等等，是的这没错，但却忽略了其支付方式的危险 — — 辅助监视定位的能力，实在不应该
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="5db3">
   <strong class="markup--strong markup--p-strong">
    — — 能源和水 — —
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="1b68">
   能源和水管理技术
   <a class="markup--anchor markup--p-anchor" data-href="https://ec.europa.eu/energy/en/topics/markets-and-consumers/smart-grids-and-meters" href="https://ec.europa.eu/energy/en/topics/markets-and-consumers/smart-grids-and-meters" rel="noopener nofollow noreferrer" target="_blank">
    已经在开始实施了
   </a>
   ，以优化能源使用并更好地了解需求和供应。能源管理技术的范围从智能电网 — 由国家实施 — 到家庭能源管理系统 — 也称为智能电表。智能电表允许各国获取更多数据，以帮助他们管理储能，并帮助他们解决能量损失和停电的问题。允许公司减少计费错误，并为客户提供更便宜的能源交易。
  </p>
  <p class="graf graf--p graf-after--p" id="163b">
   ⚠️ 但是，
   <strong class="markup--strong markup--p-strong">
    持续监控您的能源消耗也意味着国家政府、您的能源供应商以及服务供应链中的任何其他第三方，都知道您何时在家、您什么时候醒来，上床睡觉之前是不是喜欢淋浴或泡澡……
   </strong>
   根据这些数据，可以推断出更多关于你、你的家人和你关系网络中的其他人的情报，例如关于宗教信仰及活动、你家庭的组成等等一切。
  </p>
  <p class="graf graf--p graf-after--p" id="c8b1">
   <strong class="markup--strong markup--p-strong">
    这些数据足够做出几万字的社交工程攻击演示
   </strong>
   。你将遭受什么样的攻击和操纵，取决于你的数据最终落在谁的手里。
  </p>
  <p class="graf graf--p graf-after--p" id="3b3f">
   <strong class="markup--strong markup--p-strong">
    — — 智能健康 — —
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="fdc1">
   技术和数据收集也被用于让老年人留在自己的房子里而不是搬到养老院去。欧盟是鼓励发展“辅助生活”的机构之一。老年人被要求佩戴可穿戴式皮肤接触设备，将有关其健康的信息（心率、血压等）传递给医疗中心，以便专业人员可以在患者面临风险时立即提供帮助。这些技术还可以检测出可能非自主的人何时离开家、在哪里迷路。放置在房子周围的监视设备能检测到一个人是否摔倒……
  </p>
  <p class="graf graf--p graf-after--p" id="b86f">
   在新加坡，作为所谓的“智能国家”计划的一部分，一些老年人的公共管理住宅周围都
   <a class="markup--anchor markup--p-anchor" data-href="https://www.wsj.com/articles/singapore-is-taking-the-smart-city-to-a-whole-new-level-1461550026" href="https://www.wsj.com/articles/singapore-is-taking-the-smart-city-to-a-whole-new-level-1461550026" rel="noopener nofollow noreferrer" target="_blank">
    安装了传感器
   </a>
   。这些监视设备能监测到老年人什么时候停止移动，甚至监控他们是不是定时上厕所！
  </p>
  <p class="graf graf--p graf-after--p" id="87a7">
   也许你家里有老人，你会担心他们的安全，这完全是可以理解的。但是你有没想过，这些监视数据不是由你 — 亲人 — 来掌握的，而是政府部门和技术公司。
  </p>
  <p class="graf graf--p graf-after--p" id="a79e">
   ⚠️如果你被当局瞄准了，他们却无法直接令你妥协的时候，你家老人的一举一动就将变成操控你的杠杆。中国人大多明白这意味着什么，尤其是被当局骚扰过的人。
  </p>
  <p>

   </noscript>
  </p>
  <div class="section-inner sectionLayout--insetColumn">
   <p class="graf graf--p graf-after--figure" id="ebe6">
    <strong class="markup--strong markup--p-strong">
     谁建造和供应智能城市？
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="57f4">
    智能城市是公共和私营部门之间合作的结果 — —
    <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/bigdata/@iyouport/5ufmfe" href="https://steemit.com/bigdata/@iyouport/5ufmfe" rel="noopener nofollow noreferrer" target="_blank">
     公司监控伙伴关系
    </a>
    的合法化。
   </p>
   <p class="graf graf--p graf-after--p" id="3010">
    虽然智能城市的资金和倡议往往来自地方政府，希望“改善城市”，但国家政府有时可以成为领导其发展的关键力量。例如，在印度，政府启动了一项计划，即“
    <a class="markup--anchor markup--p-anchor" data-href="http://smartcities.gov.in/content/" href="http://smartcities.gov.in/content/" rel="noopener nofollow noreferrer" target="_blank">
     智能城市使命
    </a>
    ”，在全国范围内开发了100多个智能城市。
   </p>
   <p class="graf graf--p graf-after--p" id="4730">
    此外，地方和国家政府也不是唯一参与这一领域的权力，这种大规模监视的营销参与者还包括
    <a class="markup--anchor markup--p-anchor" data-href="http://ec.europa.eu/eip/smartcities/" href="http://ec.europa.eu/eip/smartcities/" rel="noopener nofollow noreferrer" target="_blank">
     欧盟
    </a>
    、
    <a class="markup--anchor markup--p-anchor" data-href="https://unhabitat.org/kenyas-proposed-smart-city-signs-agreement-with-un-habitat/" href="https://unhabitat.org/kenyas-proposed-smart-city-signs-agreement-with-un-habitat/" rel="noopener nofollow noreferrer" target="_blank">
     联合国
    </a>
    和
    <a class="markup--anchor markup--p-anchor" data-href="http://www.worldbank.org/en/topic/ict/brief/smart-cities" href="http://www.worldbank.org/en/topic/ict/brief/smart-cities" rel="noopener nofollow noreferrer" target="_blank">
     世界银行
    </a>
    等超国家机构，使用所谓的“利大于弊”的理想化。
    <strong class="markup--strong markup--p-strong">
     于是这事儿就严重了。成为了一个已经很难避免的趋势。
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="3b80">
    更要命的是，无论是中央政府还是地方政府或超国家机构，往往都缺乏内部专业知识和资源，因此他们必须依靠私营部门来设计、实施和维护设备、系统以及基础设施，以促成这些智能城市举措的建立。
   </p>
   <p class="graf graf--p graf-after--p" id="7d49">
    然而，公司不仅是推动者和提供者，而且他们也运行他们自己的角色。
   </p>
   <p class="graf graf--p graf-after--p" id="a418">
    虽然提供集成到智能城市的服务的公司多种多样，但
    <strong class="markup--strong markup--p-strong">
     值得注意的是，提供数据处理服务的大型企业（如 IBM，Alphabet 谷歌母公司，微软和甲骨文）如何推动智能城市背后的叙述，这些叙述专注于增加数据的生成、收集和处理
    </strong>
    。
   </p>
   <p class="graf graf--p graf-after--p" id="e890">
    <strong class="markup--strong markup--p-strong">
     &gt; IBM
    </strong>
    ：IBM 最初创造了“智能城市”这一营销术语。凭借其智能城市挑战计划，该公司已经开发了诸多他们的城市化愿景 — 基于数据集中化，甚至目标是遍布全球。从那时起，该公司已成为市场中最大的参与者之一，预计到 2020 年其规模将达到近 1.56 万亿美元。
   </p>
   <p class="graf graf--p graf-after--p" id="f927">
    IBM 对“智能城市”的设想是以数据为中心的。根据公司材料，它依靠传感器 — 能够检测环境变化的电子元件，如温度、运动、光线和声音 — 放置在整个城市，为市政府的使用创建数据。放置在传输交通流量的高速公路上的摄像机和实时记录水和能源使用的数字仪表（也称为“智能仪表”）是 IBM 提供给当地政府可以使用的数据类型的两个例子。
   </p>
   <p class="graf graf--p graf-after--p" id="7d33">

     <p class="graf graf--p graf-after--p" id="2a7f">
      在新加坡，政府目前正计划与法国软件公司 DassaultSystèmes 合作创建 Virtual Singapore，以实现“城市仪表板”的下一步。该平台不仅会使用传感器和摄像机汇总来自该市的所有数据，还允许政府官员深入公寓，理由是“分析其能耗并检测可能的气体泄漏情况”。传感器和摄像头将允许政府追踪在新加坡驾驶的每一辆车，并且还允许他们控制人群并进行实验以预测流行病和自然灾害或灾难。政府甚至能够通过监控抓到从高层建筑乱扔垃圾或在禁区内吸烟的人……
      <strong class="markup--strong markup--p-strong">
       这些高度侵入性的监控很少能真正实现利他主义
      </strong>
      。
     </p>
     <p class="graf graf--p graf-after--p" id="7b07">
      物联网（IoT）正在改变智能城市的本质。这是连接到互联网的对象的开发和扩展。电脑和手机不再是访问互联网的唯一途径。现在这个监视网络还连接了手表、冰箱、婴儿监视器、电视、甚至牙刷和烤面包机。物联网深深影响了智能城市计划的构思。
      <strong class="markup--strong markup--p-strong">
       每个连接对象都成为了当地政府可能利用的数据来源
      </strong>
      。人们随身携带的手机、有时候是手表都是监视器。在 IBM 对雅加达智能城市的描述中，该公司甚至称将公民称作为“行走的传感器”！
     </p>
     <p class="graf graf--p graf-after--p" id="add9">
      对 IBM 说了很多，主要因为它是这一营销概念的创世者，其他公司也正在加紧跟上。
     </p>

      <strong class="markup--strong markup--p-strong">

      </strong>

     <p class="graf graf--p graf-after--p" id="a872">
      <strong class="markup--strong markup--p-strong">
       已经建立的智能城市都有哪些？
      </strong>

     <p class="graf graf--p graf-after--p" id="e28d">
      — — 雅加达 — —
     </p>
     <p class="graf graf--p graf-after--p" id="178c">
      雅加达智能城市计划于 2014 年启动，并且是作为 2012 年就开始的更广泛的政府工作的一部分。该市有一系列目标，到 2025 年将实现的目标分为五个类别：智能生活，智能移动，智能治理，智能经济和智能人才。
     </p>
     <p class="graf graf--p graf-after--p" id="c2d4">
      雅加达智能城市计划非常依赖于一个名为 Qlue 的应用程序 — 由同名公司开发。 Qlue 可在手机上提供人们所面临问题的报告：从废物处理到破碎的路灯或者交通拥堵等等。政府使用由印度尼西亚官方谷歌云合作伙伴 TerraLogiq 设计的 CROP 程序回应报告。印度尼西亚政府已委托 IBM 公司来分析公民使用 Qlue 的数据。
     </p>
     <p class="graf graf--p graf-after--p" id="6919">
      但是，
      <strong class="markup--strong markup--p-strong">
       Qlue 已经成为了监控城市工人潜在负面影响的工具
      </strong>
      。公民反馈该技术被用来评估城市工人是否有效率（工作是否努力），城市可以在此基础上判断是否解雇工人。仅此一点就留下了滥用的空间，因为恶意者可能会发送大量报告和负面反馈以令员工被解雇 — — 就像网络水军利用 Twitter 和 Facebook 的举报功能促使社交媒体封锁异议人士那样。
     </p>
     <p class="graf graf--p graf-after--p" id="4b92">
      曾经有报道指，一名 Qlue 用户反复发送多份琐碎事件的报告 — 污水中的塑料袋、丢失的钢笔等等 — 以便该分区被显示为一个有问题的地区。后来才知道该用户原来是分区选举的候选人。
     </p>
     <p class="graf graf--p graf-after--p" id="5326">
      — — 里约热内卢 — —
     </p>
     <p class="graf graf--p graf-after--p" id="dd77">
      里约热内卢 2010 年就成为了 IBM 签约的旗舰城市。IBM 还是警务基础设施的提供商，并为执法目的设计数据分析软件。
     </p>
     <p class="graf graf--p graf-after--p" id="7bb1">
      智能城市技术 — 包括 CCTV 交通网络 — 都集中在城市较富裕的地区。因此，城市管理者更多地在更富裕的地区进行干预 — 他们从中获取数据 — 而不是更加贫困的地区，从而加剧了已经存在的不平等。里约行动中心还揭示了建立一个日益受到监管的公共空间的危险，在对指挥与控制综合中心（里约的另一个“控制室”）的工作人员进行采访发现，里约的许多紧迫的社会和安全问题 — — 这座城市正在遭遇洪灾和世界杯前的
      <strong class="markup--strong markup--p-strong">
       抗议活动，多次遭到暴力镇压
      </strong>
      。
     </p>
     <p class="graf graf--p graf-after--p" id="7c82">
      — — 菲律宾 — —
     </p>
     <p class="graf graf--p graf-after--p" id="2b4c">
      菲律宾的达沃市是另一个案例，智能城市倡议已经导致一个更加苛刻的监视城市的形成。建立运营中心后杜特尔特总统祝贺他的女儿也就是该智能城市市长 Sara Duterte 购买了一套“先进的监控和警务系统”，其 CCTV 摄像头据称帮助警方抓到了一名达沃商人。 该监视系统的效率已归功于 IBM 的设备支持。
     </p>
     <p class="graf graf--p graf-after--p" id="ac9c">
      与此同时，达沃市也被目睹了
      <strong class="markup--strong markup--p-strong">
       对政治异议的严重压制以及安全部门高度的法外杀戮和滥用行为
      </strong>
      。罗德里戈·杜特尔特早早已常因其政府骇人听闻的人权记录而受到批评。
     </p>
     <p class="graf graf--p graf-after--p" id="1984">
      — — 印度 — —
     </p>
     <p class="graf graf--p graf-after--p" id="951f">
      印度的现实反映了当前对智能城市项目的普遍担忧：这些城市在营销期限之外会是什么样子？这些城市实际上是在为谁服务？
     </p>
     <p class="graf graf--p graf-after--p" id="168a">
      2015 年，莫迪启动了一项计划，在印度建立 100 个智能城市。根据在线媒体
      <a class="markup--anchor markup--p-anchor" data-href="http://scroll.in/" href="https://scroll.in/" rel="noreferrer noopener nofollow" target="_blank">
       Scroll.in
      </a>
      ，印度标准局最初试图创建一个可居住指数来评估应该为一个城市提供的服务和基础设施。该委员会设计了包括经济在内的各个部门的指标，教育，能源，环境，健康，治理，运输，住房和安全……对于每个指标，都规定了如何衡量数据的方法。
     </p>
     <p class="graf graf--p graf-after--p" id="942e">
      然而，智能城市举措可能会增加弱势群体被边缘化的现状。例如，据报道，古吉拉特邦的土地被强行收购，用于为富裕的投资者和居民建设智能城市。就像中国发生的事那样，工人阶级社区已经被从中心区迁移出去，街头小贩也被赶出了公路。然而，根据一些报道，正是他们的存在确保了妇女出行更加安全。
     </p>
     <p class="graf graf--p graf-after--p" id="7e2c">
      — — 中国 — —
     </p>
     <p class="graf graf--p graf-after--p" id="fec1">
      中国已经成为全球智能城市建设最为积极的国家之一。近年来，智能城市已上升为北京的国家战略，随着政策红利的进一步释放、资金的大量投入，智能城市产业正在变成一块越来越大的肥肉。
     </p>
     <p class="graf graf--p graf-after--p" id="9d66">
      2015 年，中兴通讯的智能城市项目就在银川启动：银川市民大厅 — — 这座巨大的、现代化的市民大厅实施全数字化管理，被称为“电子政务的典范、简便市民生活”。该公司给 100 余个中国城市配备了所谓的智能城市技术，整体形成大规模监控。
     </p>
     <p class="graf graf--p graf-after--p" id="3281">
      中兴不仅向超过百个中国城市卖出了智能城市运营平台，也把产品卖到了尼日利亚、加纳和阿尔及利亚等地。法国的马赛和东南亚国家印度尼西亚也是中兴的客户。
     </p>
     <p class="graf graf--p graf-after--p" id="85b1">
      去年底，多家美国科技巨头和中国国有企业齐聚深圳参加第十六届中国国际公共安全博览会（简称：安博会）展示大规模监控领域的未来，这些企业都想在全球最大的监控设备市场分一杯羹。
     </p>
     <p class="graf graf--p graf-after--p graf--trailing" id="e9d4">
      两年前，中国监控市场的规模就达到了64亿美元，政府希望在中国蓬勃发展的技术产业的帮助下，建立一个规模庞大、规模空前的国家监控系统。估计到2020年，中国将安装近三亿个摄像头。中国官方媒体引用的一位“专家”的说法，中国警方在未来几年里将在提升跟踪活动的技术能力上再花 300 亿美元。
     </p>
    </div>
   </div>
  </section>
  <section class="section section--body section--last">
   <div class="section-divider">
   </div>
   <div class="section-content">
    <div class="section-inner sectionLayout--insetColumn">
     <blockquote class="graf graf--blockquote graf--leading" id="b1cf">
      <p>
       <strong class="markup--strong markup--blockquote-strong">
        <em class="markup--em markup--blockquote-em">
         政府声称他们正在创造智能城市以提供更好的服务质量，然而，如果公众不参与讨论，他们是在为谁制作智能城市？
        </em>
       </strong>
      </p>
     </blockquote>
     <p class="graf graf--p graf-after--blockquote" id="6879">
      曾经我们的行为只能在我们在线时被跟踪，而现在我们面对的是永久的监视。
     </p>
     <p class="graf graf--p graf-after--p" id="ad0a">
      <strong class="markup--strong markup--p-strong">
       越来越多的监控不仅发生在公共场所，也发生在私人空间
      </strong>
      ，就如新加坡做的那样，无线传感器已经被安装在老人家中，用于监控居民的移动、睡眠模式甚至浴室的使用情况。中国很快也会实施，鉴于中国的老龄化问题，当局有充分的理由这样做。
     </p>
     <p class="graf graf--p graf-after--p" id="2793">
      Blum-Dumontet 在报告“ 智能城市 — 乌托邦视野，反乌托邦现实 ”中写道：“虽然这些举措有明显的预期积极目标，但它们也具有高度的侵入性，并且很可能会滥用或转向较少利他的目的。”
     </p>
     <p class="graf graf--p graf-after--p" id="5e9a">
      对于人权记录不佳且没有保护公民数据的法律框架的国家公民来说，思考这些问题尤为重要。“老大哥”效应并不仅限于专制政权。即使在人民有权抗议和组织的民主国家，智能技术也会侵蚀人们的异议能力。就如在美国和英国发生的事那样。
     </p>
     <p class="graf graf--p graf-after--p" id="fbb2">
      而且，对智能设施的担忧已经超出了他们可以跟踪我们的行为这一事实。如今人们越多地使用自动化对象取代非自动化对象，就越
      <strong class="markup--strong markup--p-strong">
       容易受到恶意软件攻击，数据泄露，勒索软件，物联网僵尸网络拒绝服务攻击
      </strong>
      。只要你拥有一个操作系统，现在大多数是 Android，iOS 或微软，你就会受到相当常见的攻击形式的影响。
     </p>
     <p class="graf graf--p graf-after--p" id="fa89">
      通过智能传感器和摄像头检测市民的一举一动，并收集大量数据，如果这些数据落入恶意者手中、骇客会劫持我们的智能基础设施，那会发生什么？
     </p>
     <p class="graf graf--p graf-after--p" id="8927">
      曾经发生的诸多网络攻击表明，即使是最高科技的基础设施也是如此脆弱。例如，2014 年韩国一家核电厂运营商的计算机系统遭到破坏。虽然当局坚称没有关键数据被泄露，但事件引发了严重的安全恐惧。
     </p>
     <p class="graf graf--p graf-after--p" id="7f88">
      如上，所谓的智能城市的趋势很可能已经无法扭转，但
      <strong class="markup--strong markup--p-strong">
       技术进步永远不能以剥夺人权为代价。我们需要将个人需求带回城市的核心，当考虑城市化时，需要重点考虑的是解决方案如何影响人们对权利的享受，例如避免歧视的权利，行动自由，隐私，集会或言论自由等。
      </strong>
     </p>
     <p class="graf graf--p graf-after--p" id="23e7">
      城市应该为每个人服务而不是为政权服务，为此，决策过程应该是开放、透明和包容的，法律和道德义务涉及所涉及的行为者。各国政府应寻求公民、民间社会以及技术和安全专家、学术界以及私营部门的投入和专业知识。在提出解决方案时，隐私部门应确保进行彻底的问题分析和需求评估，并尊重基本的基础数据保护和安全标准。促进并资助此类举措的其他参与者有责任确保他们这样做。
     </p>
     <p class="graf graf--p graf-after--p" id="7dbb">
      如果做不到，没有法律保护、没有公民社会揭露，那就必需发动民众一起来抵制这些项目。没有别的办法。
     </p>
    </div>
   </div>
