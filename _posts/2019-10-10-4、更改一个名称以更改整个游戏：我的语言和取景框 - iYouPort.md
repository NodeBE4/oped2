  <p class="graf graf--p">
   使用他们的语言就相当于被困在他们的框架内。
  </p>
  <p class="graf graf--p">
   当局很擅长控制概念，以此来控制“事实”。
  </p>
  <p class="graf graf--p">
   比如我们多次举例过的，“舆情分析”，听起来就像当权者在倾听民意似的；事实上它是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%BB%80%E4%B9%88%E6%98%AF%E7%A4%BE%E4%BA%A4%E5%AA%92%E4%BD%93%E6%83%85%E6%8A%A5%EF%BC%9F%E4%BD%A0%E7%9A%84%E6%8E%A8%E7%89%B9%E5%A6%82%E4%BD%95%E6%88%90%E4%B8%BA%E6%8A%93%E6%8D%95%E4%BD%A0%E7%9A%84/" href="https://www.iyouport.org/%E4%BB%80%E4%B9%88%E6%98%AF%E7%A4%BE%E4%BA%A4%E5%AA%92%E4%BD%93%E6%83%85%E6%8A%A5%EF%BC%9F%E4%BD%A0%E7%9A%84%E6%8E%A8%E7%89%B9%E5%A6%82%E4%BD%95%E6%88%90%E4%B8%BA%E6%8A%93%E6%8D%95%E4%BD%A0%E7%9A%84/" rel="noopener noreferrer" target="_blank">
    社交媒体情报
   </a>
   ，它是有可能严重侵害公民权利的。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “情报”意味着侦查、窥视，并借此制定行动以针对任何监视者想要的人。
  </p>
  <p class="graf graf--p">
   当我们换一个名词时，就能揭露当权者试图掩盖的行为本质。
  </p>
  <p class="graf graf--p">
   ⚠️如果你沿用当权者制定的名词，即便论述得非常清晰，也很难真正触动广大利益受损群体以联合反抗和/或自我保护。
  </p>
  <p class="graf graf--p">
   这只是一个最简单的例子。
  </p>
  <p class="graf graf--p">
   人们理解新思想和概念的主要过滤器是用他们的第一语言，即母语。
  </p>
  <p class="graf graf--p">
   反抗者的母语也是他们所反对的人的母语。于是使用语言游戏会非常有趣。
  </p>
  <p class="graf graf--p">
   💡变革者必须解开复杂的理论并将抽象的，官僚的术语翻译成简单的日常用语，这样人们才能更容易识别、讨论和对抗不公正和压迫性的系统。
  </p>
  <p class="graf graf--p">
   举一个典型的例子。在乌干达，“腐败”一词在该国的任何一种民族语言中都没有相应的术语。
  </p>
  <p class="graf graf--p">
   这个词本身是相当广泛和具象的，因为它的拉丁词根意味着简单的 break down。
  </p>
  <p class="graf graf--p">
   但是当你把腐败的政治含义剥离到最基本的形式时，你会发现，它意味着盗窃。
  </p>
  <p class="graf graf--p">
   2012年，在反对广泛滥用公共资源的运动中，民间社会组织决定将“腐败”作为盗窃、将腐败官员作为小偷来处理。
  </p>
  <p class="graf graf--p">
   如果贪污是抢劫，那么腐败就是盗窃 —— 乌干达的每一种当地语言中都有盗贼在这个词。
  </p>
  <p class="graf graf--p">
   使用语言的微小变化在这场运动中的影响是惊人的。
  </p>
  <p class="graf graf--p">
   人们可能会突然明白腐败的真正含义：有人在从其他人那里窃取资源，特别是那些最需要这些资源的人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果面对不公正事件却迷失在法律条款中，就很容易错过腐败的实际影响。
   </strong>
  </p>
  <p class="graf graf--p">
   通过指出腐败就是抢劫，该运动暴露了腐败的实际影响 —— 例如，因为无法获得产妇护理，全国范围内每一天就有16名母亲死亡；如果资源没有被私人偷窃，可以建造数千所农村学校，孩子们不至于没学上；如果某个公职人员没有将这些资源汇集到个人银行账户，那么农民本来可以获得更好的设备、穷人可以拥有更好的医疗保健。
  </p>
  <p class="graf graf--p">
   💡通过更改问题的名称，该活动使每个人都能够识别出错误行为、并联合起来推动解决方案的实施。
  </p>
  <p class="graf graf--p">
   人们从来知道如何处理他们社区中的小偷，现在也很容易看出应该如何应对政府中的小偷。
  </p>
  <p class="graf graf--p">
   突然之间，在抽象的不公正面前，人们不再无能为力；恶行会呈现一个准确的名字、清晰的脸，和后果。
  </p>
  <p class="graf graf--p">
   一旦问题出现在更有意义的文化背景下，人们就可以更容易想象出适当的补救措施。
  </p>
  <p class="graf graf--p">
   窃贼们自己的回应也是史无前例的。鉴于与抢劫等卑鄙行为直接有关的尴尬和羞耻，政府官员表现得尤其恐慌，为了摆脱罪名歪招叠出。
  </p>
  <p class="graf graf--p">
   他们越是挣扎，其结果就越尴尬。
  </p>
  <p class="graf graf--p">
   并且，媒体也接受了这一术语，开始将腐败称为盗窃。这种新语言迅速传播到大多数媒体，人们至今仍在使用这一术语。
  </p>
  <p class="graf graf--p">
   2012年，在乌干达，精明的活动家通过更改一个名字从而改变了游戏规则。
  </p>
  <p class="graf graf--p">
   虽然此案例只是一个国家的一场斗争，但
   <strong class="markup--strong markup--p-strong">
    这一原则适用于全世界
   </strong>
   。
  </p>
  <p class="graf graf--p">
   您正在策划的运动主题能否通过修改名字以提升动员效果？要不要试试看？
  </p>
  <p class="graf graf--p">
   肯定有一个最准确的词能指向胜利的目标。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5642 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-2.jpg?resize=1100%2C550&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-2.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-2.jpg?resize=300%2C150&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-2.jpg?resize=768%2C384&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-2.jpg?resize=1024%2C512&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-2.jpg?resize=1100%2C550&amp;ssl=1 1100w" data-recalc-dims="1" height="550" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1-2.jpg?resize=1100%2C550&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   此外，加拿大社会学家 Erving Goffman 写过一本书
   <em class="markup--em markup--p-em">
    Frame Analysis，
   </em>
   如果您是活动家们并为组织动员发愁，值得阅读这本书。
  </p>
  <p class="graf graf--p">
   人类是叙事的动物，总是用故事来理解身处的世界，
   <strong class="markup--strong markup--p-strong">
    概念框架就是这个过程的关键。
   </strong>
  </p>
  <p class="graf graf--p">
   就如 George Lakoff 所描述的，框架是“允许人类理解现实的心理结构 —— 有时是创造我们所认为的现实。”
  </p>
  <p class="graf graf--p">
   无论你是否意识到框架的存在，框架都嵌入在我们描述和理解世界的任何方式中，⚠️它被强加了一种通常有益于那些有权力的人的意义。
  </p>
  <p class="graf graf--p">
   例如，当经济学家给出“经济预测”时，经常称之为“晴雨表”，这就巧妙地引导人们将资本主义视为超越人类控制的自然力量。
  </p>
  <p class="graf graf--p">
   ⚠️这种框架有利于银行业和寡头企业，因为它模糊了他们如何实际操纵市场以获得自己的优势的事实。
  </p>
  <p class="graf graf--p">
   框架可以是一个有用的工具，不仅可以增强自身力量，还可以进行竞争。
  </p>
  <p class="graf graf--p">
   💡在制定行动或活动时，
   <strong class="markup--strong markup--p-strong">
    您必须考虑如何最具说服力地描绘问题或冲突，以便破坏主导框架，并使用有利于运动目标的框架取而代之。
   </strong>
  </p>
  <p class="graf graf--p">
   参见这里：“关于框架有一个基本的事实：如果你接受了别人的框架，你就输了” — — 《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/3%E3%80%81%E6%A1%86%E6%9E%B6%E6%98%AF%E4%B8%80%E7%A7%8D%E6%9D%83%E5%8A%9B/" href="https://www.iyouport.org/3%E3%80%81%E6%A1%86%E6%9E%B6%E6%98%AF%E4%B8%80%E7%A7%8D%E6%9D%83%E5%8A%9B/" rel="noopener noreferrer" target="_blank">
    框架是一种权力
   </a>
   》
  </p>
  <p class="graf graf--p">
   💡您要问的一些问题包括：谁是好人？谁是坏人？冲突点是什么？利害关系有哪些？必须揭示哪些隐藏的力量或新的解决方案？我们的基本价值观是什么？是否有一个统一的主题可以为我们的故事创建一个框架结构？
  </p>
