  <p class="graf graf--p graf-after--figure" id="31b2">
   如果新浪微博和其他中国数字极权巨头永久性封锁了周小平，你会赞同吗？
  </p>
  <p class="graf graf--p graf-after--p" id="7d08">
   这也许是个很不准确的类比。首先周小平影响社会舆论的能力连 Alex Jones 的脚趾甲都比不上；其次，周小平是被塑造出来的代表权力抢占舆论市场的玩偶，而 Alex Jones 的追随者数量之高是真的代表了一部分确实存在的社会心态；最后，这件事是基本不可能发生的，中国的 BAT 不会这么做，鉴于这些平台的政治任务和周小平账户可观的引流，并且他们在中国没有也不会受到舆论压力，而苹果、Facebook 等平台截然相反……
  </p>
  <p class="graf graf--p graf-after--p" id="4f54">
   但是我们依旧想这样做一下对比，以此申明：究竟应该如何认识审查制度。不幸的事，
   <strong class="markup--strong markup--p-strong">
    当下中国社会并没有在普遍的程度上理解审查，也就难以理解和追求民主自由。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="8b0c">
   谁是 Alex Jones？
  </p>
  <p class="graf graf--p graf-after--p" id="f1f2">
   这是近期一直在持续的英文网络头号舆论热点：苹果，YouTube，Spotify 和 Facebook 纷纷删除了美国最著名阴谋论网络名人 Alex Jones 的所有内容。这位以阴谋论著称的名人拥有一个广播节目，并运行着一些网站包括 Infowars。他有数百万跟随者。但忽然之间全消失了。
  </p>
  <p class="graf graf--p graf-after--p" id="2150">
   此事引发了一个极大规模的辩论，以至于 iYouPort 已经无法坐视不理。我们粗略观察了一下，相关论点集中在几个方面。首先毫无疑问那就是，第一修正案保护的言论自由。
  </p>
  <p class="graf graf--p graf-after--p" id="449c">
   第一修正案禁止“剥夺言论自由”，意味着政府不得审查或惩罚言论。在为公共论坛设定规则时，国家的任何一方都不得通过观点对人群加以区分。
   <strong class="markup--strong markup--p-strong">
    根据最高法院对自由表达的广泛概念，即使是冒犯性和仇恨性的言论也是允许的，除非直接煽动暴力。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="4a61">
   但问题在于，私营公司不是宪法里说的“国家”。Apple，Facebook 和 YouTube 可以根据自己的意愿编写其服务条款，并根据自己的偏好审查其平台上的内容。政府和私人审查制度之间存在着一条小而迷人的灰色地带 ……
  </p>
  <p class="graf graf--p graf-after--p" id="bf48">
   <strong class="markup--strong markup--p-strong">
    Facebook 和 YouTube 不是国家，宪法管不了它们，但如果它们的审查反复无常、或亲近某一党派而显示出严重不公正，对民主的影响之后果将不堪设想
   </strong>
   。这些私人实体如今的威力已经基本和大多数国家的政府一样强大，如何让它们尊重宪法，是一个迫切的问题。
  </p>
  <p class="graf graf--p graf-after--p" id="bda9">
   但是，即便宪法承诺的言论自由，在近三百年后的今天，依旧着存在对其理解的误差。
  </p>
  <p class="graf graf--p graf-after--p" id="da16">
   这是言论自由圈中最古老的陈词滥调，即：“你不能在拥挤的剧院里大喊大叫”。换句话说就是，即使根据第一修正案，言论自由也是有限制的。
  </p>
  <p class="graf graf--p graf-after--p" id="f8ff">
   在剧院的类比中，风险是造成伤害，不是直接造成伤害而是通过你的陈述可能导致的恐慌。类似的规则可以适用于针对特定群体的暴力或仇恨的煽动言论。
  </p>
  <p class="graf graf--p graf-after--p" id="3c6b">
   按照这个标准，Alex Jones 和他的 Infowars 出版物对言论自由形成了“可怕的挑战”。他们通过在线基本等同于每天尖叫“着火啦”来创造自己的职业生涯。
  </p>
  <p class="graf graf--p graf-after--p" id="5ddd">
   不论是 9/11 truthers、推动 Pizzagate 阴谋论，导致自动武器在华盛顿餐厅出现、还是说 Sandy Hook 是 “false flag” 、煽动对犯罪受害者父母的骚扰，等等，他们真的做了很多。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4398 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-155.jpeg?resize=1100%2C616&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-155.jpeg?w=1920&amp;ssl=1 1920w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-155.jpeg?resize=300%2C168&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-155.jpeg?resize=768%2C430&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-155.jpeg?resize=1024%2C573&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-155.jpeg?resize=1100%2C616&amp;ssl=1 1100w" data-recalc-dims="1" height="616" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-155.jpeg?resize=1100%2C616&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4398" data-recalc-dims="1" height="616" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-155.jpeg?resize=1100%2C616&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-155.jpeg?w=1920&amp;ssl=1 1920w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-155.jpeg?resize=300%2C168&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-155.jpeg?resize=768%2C430&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-155.jpeg?resize=1024%2C573&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-155.jpeg?resize=1100%2C616&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="0219">
   但最终，对 Infowars 和 Jones 采取行动的不是美国政府，而是美国最大的科技公司，硅谷巨头。Jones 和 Infowars 现已被从 YouTube，Spotify，Facebook 和 Apple 中删除 — 所有这些都在几小时内发生并完成。
  </p>
  <p class="graf graf--p graf-after--p" id="d67e">
   必需知道，政府和数字极权，这两者是一样的可怕。在集中化互联网吞噬了所有人的今天，这个问题必需被所有人了解。
  </p>
  <p class="graf graf--p graf-after--p" id="bc72">
   <strong class="markup--strong markup--p-strong">
    任何科技巨头都没有控制您的内容的主动权
   </strong>
   。但它们是私人公司，因此只要它符合法律，就可以完全合法地决定其网络上什么样的内容可用可见。
  </p>
  <p class="graf graf--p graf-after--p" id="7a25">
   然而，尽管如此，法律论证 — 而且这是一个合理的论证 — 实际上，
   <strong class="markup--strong markup--p-strong">
    如果你以这种方式被淘汰出技术巨头的平台，你几乎没有机会再重新获得任何观众。
   </strong>
   中国的转世党的确存在，但就如你知道的，他们中绝大多数只会不断地在转世中衰落，最终失去所有机会。
  </p>
  <p class="graf graf--p graf-after--p" id="529f">
   <strong class="markup--strong markup--p-strong">
    如果你的政府审查你，你还有很多其他国家可以尝试传播你的信息，利用地缘政治冲突，在本国的“敌国”方面获得言论自由
   </strong>
   。但如果是 Facebook 和其他科技公司审查你，你就没有任何可能在地球上找到另一个一模一样的影响力机会。
  </p>
  <p class="graf graf--p graf-after--p" id="a6a9">
   拥有这种权力的公司就不应该存在。
  </p>
  <p class="graf graf--p graf-after--p" id="c162">
   并且，对于前半句话，不得不指出的是，
   <strong class="markup--strong markup--p-strong">
    美国除外
   </strong>
   ，因为美国在全球的盟友太多了，而且其敌国都是显著的流氓国家，公信力非常低，于是，不论是批评大规模监控、美国情报部门对大众实施的间谍行为、还是揭露战争的真相，美国公民的言论自由一直被集中化控制着。（的确，就如这篇文章中
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E7%BD%91%E7%BB%9C%E6%88%98-%EF%B8%8F%E7%A4%BE%E4%BA%A4%E5%B7%A5%E7%A8%8B%E5%AD%A6%E6%AD%A3%E8%A2%AB%E5%85%A8%E7%90%83%E5%BD%93%E6%9D%83%E8%80%85%E5%88%A9%E7%94%A8%E6%9D%A5%E6%94%BB%E5%87%BB%E6%B4%BB%E5%8A%A8%E5%AE%B6-%E5%BC%82%E8%AE%AE%E4%BA%BA%E5%A3%AB%E5%92%8C%E8%AE%B0%E8%80%85-%E5%A6%82%E4%BD%95%E6%8A%B5%E5%BE%A1-2baa68643142" href="https://medium.com/@iyouport/%E7%BD%91%E7%BB%9C%E6%88%98-%EF%B8%8F%E7%A4%BE%E4%BA%A4%E5%B7%A5%E7%A8%8B%E5%AD%A6%E6%AD%A3%E8%A2%AB%E5%85%A8%E7%90%83%E5%BD%93%E6%9D%83%E8%80%85%E5%88%A9%E7%94%A8%E6%9D%A5%E6%94%BB%E5%87%BB%E6%B4%BB%E5%8A%A8%E5%AE%B6-%E5%BC%82%E8%AE%AE%E4%BA%BA%E5%A3%AB%E5%92%8C%E8%AE%B0%E8%80%85-%E5%A6%82%E4%BD%95%E6%8A%B5%E5%BE%A1-2baa68643142" rel="noopener noreferrer" target="_blank">
    嵌入的视频
   </a>
   所显示的那样，美国异议人士经常借助俄罗斯媒体表达自己的态度，但是它的观感肯定不会那么完美对吧。我们支持这种“敌人的敌人”玩法，
   <strong class="markup--strong markup--p-strong">
    但我们应该追求本国对宪法的承诺落在实处，而不是满足于“我有地方说话就够了”
   </strong>
   ）
  </p>
  <p class="graf graf--p graf-after--p" id="e3d5">
   Alex Jones 不应该是任何人的自由言论殉道者 — 他的内容是私人的（他的阴谋咆哮是用于出售定价过高的补品）虽然也是危险的。但
   <strong class="markup--strong markup--p-strong">
    对他的禁令确实揭示了我们所有人所面对的在互联网上权力集中化已经到了令人震惊的程度。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="92c5">
   以上反驳了这场舆论冲突中最为常见的思路，它们主要由于抓错了重点。
  </p>
  <p class="graf graf--p graf-after--p" id="8e3b">
   实际上，Jones 的故事中最重要的部分是他和 Infowars 被禁止的方式。多年来，人们一直试图让科技公司解释其网络上的危险错误信息、仇恨言论和滥用行为，不仅美国，全球很多国家都在要求这些跨国公司根据形形色色的本地法律对内容做出管制。作为回应，我们不断听到的是，“问题很复杂，公司已经制定了广泛的程序来解决这些问题”……等等。
  </p>
  <p class="graf graf--p graf-after--p" id="5983">
   然而，Infowars 从几乎所有主要网络中消失的方式令任何说过或谴责过网络言论问题的人都变成了傻瓜。在撰写本文时，Twitter 是最后一个尚未禁止它的主要平台。
  </p>
  <p class="graf graf--p graf-after--p" id="1fd9">
   不要就此以为 Twitter 是言论自由胜地，它并不是。
   <a class="markup--anchor markup--p-anchor" data-href="https://dailyarchives.org/index.php/reports/1002-shut-it-down-twitter-shadow-banning-accounts-with-politically-incorrect-posts" href="https://dailyarchives.org/index.php/reports/1002-shut-it-down-twitter-shadow-banning-accounts-with-politically-incorrect-posts" rel="noopener nofollow noreferrer" target="_blank">
    推特到目前为止仍在禁止发布政治上不正确的材料的账户
   </a>
   。这不该令人惊讶。
   <strong class="markup--strong markup--p-strong">
    Twitter 最近还宣布了一个奥威尔思想委员会式的组织，以关闭任何他们不喜欢的言论类型
   </strong>
   。这些早前只是传言的消息现在已经得到了内部人士的确认。
  </p>
  <p class="graf graf--p graf-after--p" id="6948">
   消息来源称，Twitter 维持了一个受欢迎的 Twitter 帐户的“白名单”、和一个不受欢迎的帐户“黑名单”。白名单上的帐户在搜索结果中具有优先级，即使它们
   <strong class="markup--strong markup--p-strong">
    不是
   </strong>
   在用户群体中最受欢迎的。同时，黑名单上的帐户很难在搜索中找到，他们的帖子被降级。
  </p>
  <p class="graf graf--p graf-after--p" id="4c8b">
   几周来，一些用户一直在报告保守派、另类右翼、文化自由主义者和其他反 PC 持不同政见者的推文已从他们的时间线上消失。
  </p>
  <p class="graf graf--p graf-after--p" id="6665">
   也许你不喜欢另类右翼、不喜欢保守派，就如你不喜欢周小平，我们也不喜欢，但这不是他们被剥夺言论自由的理由。（除了周小平）他们依旧是公民，其权利受宪法保护，而如今宪法无法保护他们，
   <strong class="markup--strong markup--p-strong">
    因为夺取其言论自由权的实施者是大公司。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="9bcf">
   在美国，甚至这些大公司的做法最近还得到了立法者的赞同。美国参议员克里斯·墨菲（Chris Murphy）呼吁硅谷进行更多的审查，称“封锁 Infowars 是一个良好的开端”……虽然同时有很多评论家坚定地谴责硅谷的做法，并指出：
   <strong class="markup--strong markup--p-strong">
    审查 Infowars 可能导致对主流媒体不喜欢的人采取进一步的遏制。
   </strong>
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4399 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-156.jpeg?resize=1100%2C604&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-156.jpeg?w=1518&amp;ssl=1 1518w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-156.jpeg?resize=300%2C165&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-156.jpeg?resize=768%2C422&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-156.jpeg?resize=1024%2C563&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-156.jpeg?resize=1100%2C604&amp;ssl=1 1100w" data-recalc-dims="1" height="604" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-156.jpeg?resize=1100%2C604&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4399" data-recalc-dims="1" height="604" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-156.jpeg?resize=1100%2C604&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-156.jpeg?w=1518&amp;ssl=1 1518w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-156.jpeg?resize=300%2C165&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-156.jpeg?resize=768%2C422&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-156.jpeg?resize=1024%2C563&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-156.jpeg?resize=1100%2C604&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="9ae4">
   Murphy 的帖子下面充满了指责的留言。一位用户说：
   <strong class="markup--strong markup--p-strong">
    “这意味着你在做到控制一切言论思想之前是不会停下来的，意味着如果我们说了你不同意的观点你就会不高兴，意味着你想要决定我们的思想和认知……我不喜欢 Alex Jones，但我现在觉得你比他还可怕”
   </strong>
   ；另一位用户补充说：“他在提倡审查制度！”著名评论员 Ben Shapiro 留言说：“你是政府官员，你应该知道有一种东西叫第一修正案”。
  </p>
  <p class="graf graf--p graf-after--p" id="22b4">
   今年六月，加州一位社交媒体用户 Jared Taylor 对 Twitter 提起诉讼，而法官拒绝了 Twitter 驳回诉讼的请求。此案颇为瞩目，虽然目前只是赢在了第一轮，但明显是反审查诉讼的巨大成就。
  </p>
  <p class="graf graf--p graf-after--p" id="e46c">
   法官明确指出：“这是我们的第一修正案在很久以前就承诺下的言论自由”。
  </p>
  <p class="graf graf--p graf-after--p" id="7cce">
   此案是
   <strong class="markup--strong markup--p-strong">
    第一次
   </strong>
   通过反击社交媒体平台进行的审查根据州或联邦法律被认定是可行的，真的是第一次 — — 这些审查是一种越来越广泛的做法，被许多人视为
   <strong class="markup--strong markup--p-strong">
    观点歧视
   </strong>
   。希望此案能对其他互联网平台产生更深远的影响，这些互联网平台已经成为所有人表达思想的必要工具，但他们却沉默了他们自己不同意的声音。
  </p>
  <p class="graf graf--p graf-after--p" id="525d">
   就在最近，
   <strong class="markup--strong markup--p-strong">
    Twitter 已经在其平台上封锁了三个非常重要的反战带头人账户
   </strong>
   ：罗恩保罗研究所的执行董事 Daniel McAdams、Scott Horton、著名反战网站作家
   <a class="markup--anchor markup--p-anchor" data-href="https://plus.google.com/b/104314277918209223775/104314277918209223775/posts/ZRe8bMtL3mz" href="https://plus.google.com/b/104314277918209223775/104314277918209223775/posts/ZRe8bMtL3mz" rel="noopener nofollow noreferrer" target="_blank">
    Peter Van Buren
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="0e59">
   在一个社团主义的政府体系中，公司权力与国家权力之间没有明显清晰的界限，公司审查就是国家审查。因为企业以游说和竞选捐款的形式将贿赂合法化，给了富人控制政府的政策和行为的能力，而普通人没有有效的影响力。大型的有影响力的公司与国家不可分割，因此他们对审查制度的使用与国家的审查密不可分。
  </p>
  <p class="graf graf--p graf-after--p" id="c821">
   <strong class="markup--strong markup--p-strong">
    对于硅谷巨头来说尤其如此，硅谷与情报机构的广泛关系
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://surveillancevalley.com/blog" href="https://surveillancevalley.com/blog" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     早有详细记载
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。一旦你
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.gizmodo.com.au/2018/03/google-is-helping-the-pentagon-build-ai-for-drones/" href="https://www.gizmodo.com.au/2018/03/google-is-helping-the-pentagon-build-ai-for-drones/" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     协助建造
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    军方的无人机计划、接受中央情报局和国家安全局的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://qz.com/1145669/googles-true-origin-partly-lies-in-cia-and-nsa-research-grants-for-mass-surveillance/" href="https://qz.com/1145669/googles-true-origin-partly-lies-in-cia-and-nsa-research-grants-for-mass-surveillance/" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     拨款进行大规模监视
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    、或者让你的网站内容
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.zerohedge.com/news/2018-05-04/atlantic-council-explains-why-we-need-be-propagandized-our-own-good" href="https://www.zerohedge.com/news/2018-05-04/atlantic-council-explains-why-we-need-be-propagandized-our-own-good" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     受到北约宣传部门的监管
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，你就不能再假装自己是一个独立于政府权力的私人公司。
   </strong>
   在当前的系统中，有可能拥有价值几百万美元的正常业务，但如果你想在一个资金直接转化为政治权力的系统中获得数十亿美元的财富控制，你需要尽可能使用现有的权力结构，像中央情报局和五角大楼一样，否则他们将与你的竞争对手而不是你合作。
  </p>
  <p class="graf graf--p graf-after--p" id="ce86">
   在中国也是同样的逻辑。
  </p>
  <figure aria-describedby="caption-attachment-4400" class="wp-caption aligncenter" id="attachment_4400" style="width: 700px">
   <img alt="" class="wp-image-4400 jetpack-lazy-image" data-lazy-sizes="(max-width: 700px) 100vw, 700px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-157.jpeg?resize=700%2C615&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-157.jpeg?w=1150&amp;ssl=1 1150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-157.jpeg?resize=300%2C263&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-157.jpeg?resize=768%2C675&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-157.jpeg?resize=1024%2C899&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-157.jpeg?resize=1100%2C966&amp;ssl=1 1100w" data-recalc-dims="1" height="615" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-157.jpeg?resize=700%2C615&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="700"/>
   <noscript>
    <img alt="" class="wp-image-4400" data-recalc-dims="1" height="615" sizes="(max-width: 700px) 100vw, 700px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-157.jpeg?resize=700%2C615&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-157.jpeg?w=1150&amp;ssl=1 1150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-157.jpeg?resize=300%2C263&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-157.jpeg?resize=768%2C675&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-157.jpeg?resize=1024%2C899&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-157.jpeg?resize=1100%2C966&amp;ssl=1 1100w" width="700"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-4400">
    这就是我们对审查的态度
   </figcaption>
  </figure>
  <p class="graf graf--p graf-after--figure" id="dffb">
   整件事中唯一幸运的是，人们终于有机会就大型互联网平台监管内容和封锁帐户这一棘手问题展开公开辩论了。但是，
   <strong class="markup--strong markup--p-strong">
    必须说，围绕右翼阴谋理专家 Alex Jones 的 Infowars、也就是围绕内容节制进行辩论，是一个严重的错误！
   </strong>
   真正的议题应该是苹果，Facebook，谷歌拥有的 YouTube 和 Spotify 等在线巨头，这些数字极权做出的成千上万的审核决定。
  </p>
  <p class="graf graf--p graf-after--p" id="8673">
   互联网公司在过去十年中删除了数以百万的帖子和图像，并暂停了数百甚至数千个用户帐户。
   <strong class="markup--strong markup--p-strong">
    这些被沉默的声音跨越意识形态政治光谱，而且是在全球范围内
   </strong>
   ：摩洛哥无神论者、讨论在线骚扰的女性、以十字架为主的广告、黑人和穆斯林活动家重新发布他们收到的种族主义信息、 异装表演者、土著妇女、分娩图片、母乳喂养照片……这些平台已经被用来记录叙利亚、缅甸和克什米尔的战争罪行，北达科他州的逮捕，以及整个美国的警察暴行。
  </p>
  <p class="graf graf--p graf-after--p" id="0d64">
   就如华盛顿邮报的观点专栏明确指出的，对于这种由少数私营公司把持的互联网，我们应该非常谨慎，其中控制如此多公共话语的平台会因为对内容的反对而定期删除帖子和停用帐户。
   <strong class="markup--strong markup--p-strong">
    一旦像内容审核这样的系统成为常态，掌权者就不可避免地利用它们。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="10d4">
   一次又一次，这些平台已经屈服于专制政权的审查要求，强大的权力操纵程序以有效地审查他们的政治对手。鉴于这种现实，以及这个世界包括美国诸多政治审查的悲惨历史，人们不应该为自己可能同意的一个审查决定而欢呼 — — Alex Jones，或者周小平。
  </p>
  <p class="graf graf--p graf-after--p" id="b9bc">
   <strong class="markup--strong markup--p-strong">
    很多人的确不喜欢他们，也不应该喜欢，但他们的存在 — — 拥有耀眼的追随者数字 — — 恰恰揭示了这个社会所存在的问题。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="5021">
   对于中国人熟知的披萨门，事实上不应该通过遏制 Alex Jones 的言论以避免其出现，正相反，这样做是无效的。Jones 的阴谋论之所以能成功，正因为社会上的确已经存在不得不引起注意的政治不信任，于是应该被审查的是为什么政府/党派失信，而不是 Jones。周小平的追随者中大多数人不过是在看他的笑话，但
   <strong class="markup--strong markup--p-strong">
    这正是中国社会舆论低效的重要原因：人们不是在以舆论做武器、追求真相，而是仅仅在看热闹消遣
   </strong>
   。这就是为什么周小平愚蠢的表演能在如此长的时间内浪费掉如此多公众的宝贵时间。应该被审视的是中国社会对公民权的理解，对舆论的理解，而不是周小平。
  </p>
  <p class="graf graf--p graf-after--p" id="ba45">
   这些集中化的巨头公司、不透明的审查制度，能采取行动以更积极地管制其网站上显示的内容，这种能力必需令人担忧。即使我门也会有同样的担忧，即关于虚假信息、极端主义、骚扰和对骚扰的激励。但审查永远不是正确的解决方案。
  </p>
  <p class="graf graf--p graf-after--p" id="603c">
   虽然占主导地位的社交媒体平台主要位于美国，但他们的大多数用户都在国外，而平台采取的政策在其他地方的影响力也同样很大。
   <strong class="markup--strong markup--p-strong">
    无论我们是否同意这些平台对某个内容所采取的决定，都无法忽视，这些平台对在线公共领域的影响力已经需要我们所有人提起警惕
   </strong>
   。任何一方的决定都会产生巨大的影响。
  </p>
  <blockquote class="graf graf--blockquote graf-after--p" id="f8fe">
   <p>
    <em class="markup--em markup--blockquote-em">
     任何基于意识形态的、基于对被封锁账户之审美的、任何纠结于“公司不是国家的”的观点，都是在偏离基本的中心问题，即 不透明的集中化的审查是对言论自由权的公然侵犯，是对宪法的侮辱。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p graf-after--blockquote" id="e6be">
   <strong class="markup--strong markup--p-strong">
    观点歧视
   </strong>
   是 Alex Jones 事件在中国读者中最容易产生误解的关键。其重点甚至不是歧视，而是观点。中国舆论场（虽然没有正常的思想市场）中真正的观点是极少的，大多数表达基于认知局限性、概念不清和逻辑荒谬，要知道，真正的观点是认知清晰的、概念准确的、逻辑科学的，在此基础上的不同意见，而局限和荒谬恰恰是长期被审查带来的问题。
  </p>
  <p class="graf graf--p graf-after--p" id="4702">
   人们对战斗/对抗的兴趣远低于八卦，这是最令我们感觉困惑的一点。我们致力于传播知识技能，以提升民间实力，目的是希望更多人尽快开始使用这些技能做正确的事，因为我们看到了人们的期待，对民主的向往。然而不幸的是，至少目前为止，单调的传闻、浅薄的讥讽和无效的围观依旧占据绝大多数，
   <strong class="markup--strong markup--p-strong">
    很多积极人士甚至还没能理解技术技能对其期待之目标的巨大价值。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="4097">
   抱歉我们以周小平做出的类比，因为在中国（至少互联网诞生后的这些年里）还从来没有过一个自由的思想市场，而对自由的理解追求正是中国社会急需获得的东西。为了便于中国读者的理解，这个类比只是不得已。
  </p>
  <p class="graf graf--p graf-after--p" id="20f7">
   中国社会被困在多个怪圈中，很多人无法获得自由思想的经验进而不再能理解自由，无法获得平等的观点交流进而扭曲对平等的认知，无法获得有效的知识从而混淆了观点和智商的概念……必需能在某一个或几个方面做出突破，否则无法打破这个螺旋状下降的趋势。
  </p>
  <p class="graf graf--p graf-after--p" id="8927">
   希望我们对 Alex Jones 事件的简要梳理能帮上一些忙。⚪️
  </p>
