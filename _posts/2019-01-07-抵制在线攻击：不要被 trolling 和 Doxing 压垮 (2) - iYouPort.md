    我曾经提出过一个问题：如何向活动家普及匿名安全知识。但这是有难度的，活动家往往需要通过实名的知名度来获得更广泛的动员能力，另外塑造一个虚拟身份期望它拥有同样的号召能力是不现实的。
   </p>
   <p class="graf graf--p graf-after--p" id="a82d">
    但是，
    <strong class="markup--strong markup--p-strong">
     实名有着明显的危险，不只有众所周知的 trolling，还有 Doxing 这类从在线追踪到线下迫害的严重人身威胁
    </strong>
    。目前正在大多数国家泛滥。
   </p>
   <p class="graf graf--p graf-after--p" id="7097">
    Doxing 是一种基于互联网的暴力行为，用于挖掘和广播关于个人或组织的私人或可识别信息，以便骚扰和伤害活动家及其组织活动。此外，
    <strong class="markup--strong markup--p-strong">
     这种攻击还可能伴随着现实世界的暴力，并传播关于个人和/或社会运动的虚假信息，搅乱联合、抹黑名誉
    </strong>
    。
   </p>
   <ul class="postList">

      网络“水军”的布局
     </a>
     》
    </li>
   </ul>
   <p class="graf graf--p graf-after--li" id="d7cb">
    敌对方的水军可以通过搜索公共数据、和社交媒体网站（如 Facebook）以及使用黑客工具、和社交工程攻击，来获取关于活动家的私密信息。也就是说，
    <strong class="markup--strong markup--p-strong">

    </strong>

   </p>
   <p class="graf graf--p graf-after--p" id="3d7f">
    <strong class="markup--strong markup--p-strong">
     以下是一个能保护您的清单
    </strong>
    ：
   </p>
   <p class="graf graf--p graf-after--p" id="4985">
    ✔
    <strong class="markup--strong markup--p-strong">
     创建自助计划
    </strong>
    ，并呼吁您的家人和朋友来帮助您。让他们知道最新的情况，因为 trolling 和 doxing 很可能是创伤性的，你必须优先考虑自己的身心健康，以便继续战斗。
   </p>
   <p class="graf graf--p graf-after--p" id="91f0">
    同时，我们依旧要强调
    <strong class="markup--strong markup--p-strong">
     合作和联盟
    </strong>
    。当你孤军奋战的时候就很容易被有组织的攻击压垮，当你知道有很多志同道合的人和你站在一起的时候，就能感受到更多的力量和勇气。
   </p>
   <p class="graf graf--p graf-after--p" id="66b5">
    如果可以通过评分来说明的话，中国社会将被列入“协作和联盟能力较差”的行列。
    <strong class="markup--strong markup--p-strong">
     如今的联盟仅仅限于本地是不够的，更多的需要全球联盟
    </strong>
    ，很多互联网权利组织都提供跨国协助，不论是快速响应、互联网审查和阻塞、还是在线攻击。比如
    <a class="markup--anchor markup--p-anchor" data-href="https://stoplapdspying.org/" href="https://stoplapdspying.org/" rel="noopener nofollow noreferrer" target="_blank">
     Stop LAPD Spying Coalition
    </a>
    ，
    <a class="markup--anchor markup--p-anchor" data-href="https://www.equalitylabs.org/" href="https://www.equalitylabs.org/" rel="noopener nofollow noreferrer" target="_blank">
     Equality labs
    </a>
    ，都是协作被在线攻击迫害的活动家的组织。
   </p>

     <strong class="markup--strong markup--p-strong">

     </strong>
     。您可以在
     <a class="markup--anchor markup--p-anchor" data-href="http://www.haveibeenpwned" href="http://www.haveibeenpwned/" rel="nofollow noopener noreferrer" target="_blank">

    </p>

    </p>

    </p>
    <p class="graf graf--p graf-after--p" id="a453">
     ✔ 为您的所有帐户开启 2FA。这意味着您在登录帐户时添加了其他验证方法。当有 Trolls 试图侵入你的帐户时，就能阻止他们。
    </p>
    <p class="graf graf--p graf-after--p" id="c712">
     还是那句话，您需要像骇客那样思考。如果受到损害，您有哪些帐户会造成损失最严重？通过接管您的电子邮件，攻击者可以释放私密信息并干扰您的沟通，通过接管您的银行帐户，他们会肆无忌惮地处理您的财务状况等等。因此，如果可以的话，锁定所有这些。

     <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/crypto/@iyouport/tor" href="https://www.iyouport.org/%E4%BD%BF%E7%94%A8-tor-%E4%BF%9D%E6%8A%A4%E8%87%AA%E5%B7%B1%E6%97%B6%E5%8D%83%E4%B8%87%E4%B8%8D%E8%A6%81%E5%81%9A%E8%BF%99%E4%B9%9D%E4%BB%B6%E4%BA%8B%EF%BC%81/" rel="noopener nofollow noreferrer" target="_blank">
      <strong class="markup--strong markup--p-strong">
       使用 Tor 保护自己时千万不要做这九件事！
      </strong>
     </a>
     <strong class="markup--strong markup--p-strong">
      》
     </strong>
    </p>
    <p>
     <img alt="" class="aligncenter size-full wp-image-2545 jetpack-lazy-image" data-lazy-sizes="(max-width: 703px) 100vw, 703px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-27.png?resize=703%2C343&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-27.png?w=703&amp;ssl=1 703w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-27.png?resize=300%2C146&amp;ssl=1 300w" data-recalc-dims="1" height="343" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-27.png?resize=703%2C343&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="703"/>
     <noscript>
      <img alt="" class="aligncenter size-full wp-image-2545" data-recalc-dims="1" height="343" sizes="(max-width: 703px) 100vw, 703px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-27.png?resize=703%2C343&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-27.png?w=703&amp;ssl=1 703w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/01/0-27.png?resize=300%2C146&amp;ssl=1 300w" width="703"/>
     </noscript>
    </p>
    <p class="graf graf--p graf-after--figure" id="6ed7">
     ✔
     <strong class="markup--strong markup--p-strong">
      安装 Signal
     </strong>
     。安装在手机和计算机上，此安全消息传递和语音应用程序可以取代短信、电话和电子邮件。重要的是您必须先在手机上安装它，并确保验证所有用户。Signal 最大的问题在于使用手机号注册，您可以避免使用自己真正的手机号，方法：《🔐
     <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%A6%82%E4%BD%95%E5%9C%A8%E4%B8%8D%E5%BF%85%E6%9A%B4%E9%9C%B2%E7%9C%9F%E5%AE%9E%E7%94%B5%E8%AF%9D%E5%8F%B7%E7%A0%81%E7%9A%84%E6%83%85%E5%86%B5%E4%B8%8B%E4%BD%BF%E7%94%A8-signal-5ad500d41f09" href="https://www.iyouport.org/%f0%9f%94%90-%E5%A6%82%E4%BD%95%E5%9C%A8%E4%B8%8D%E5%BF%85%E6%9A%B4%E9%9C%B2%E7%9C%9F%E5%AE%9E%E7%94%B5%E8%AF%9D%E5%8F%B7%E7%A0%81%E7%9A%84%E6%83%85%E5%86%B5%E4%B8%8B%E4%BD%BF%E7%94%A8-signal/" rel="noopener noreferrer" target="_blank">
      如何在不必暴露真实电话号码的情况下使用 Signal
     </a>
     》
    </p>
    <p class="graf graf--p graf-after--p" id="b9e9">
     ✔ 使用加密的电子邮件，⚠️
     <strong class="markup--strong markup--p-strong">
      切勿使用谷歌的 Gmail
     </strong>
     !!! GMail 与政府就许多监控计划（包括 PRISM 项目）上进行合作。因此，尽管谷歌的广泛保护可以帮助您解决个别黑客的问题，但仍然存在这样的威胁：您的帐户中的所有数据都可以被搜索并存储到 NSA 的服务器上，而无需您的知情。
    </p>
    <p class="graf graf--p graf-after--p" id="e316">
     如果您必需使用 Gmail，请安装
     <strong class="markup--strong markup--p-strong">
      额外的加密
     </strong>
     ，MAC 的 GPG 加密或 PC 的https://gnupg.org。这些是安全的，但你自己的 GPG 的设置可能对非专业人士来说有点麻烦。在这种情况下，使用 Tuanota 或 ProtonMail 等加密电子邮件服务。这两种服务都将您的加密密钥作为服务的一部分嵌入，其界面类似于 g-mail。
    </p>
    <p class="graf graf--p graf-after--p" id="abd5">
     ✔ 安全的团队通话使用 TALKY.IO或 ZOOM。
     <strong class="markup--strong markup--p-strong">
      ⚠️ 包括免费电话会议在内的所有其他协议都不安全！包括 Skype，谷歌环聊和 facetime
     </strong>
     。 Talky.io是免费的，但其免费版本的时间有限。如果您使用 Zoom，请确保转到设置并启用加密。
    </p>
    <p class="graf graf--p graf-after--p" id="edd9">
     ✔ 在您的所有社交网络上更改您的隐私设置。访问您的 Facebook，Twitter，Snapchat 和 Instagram 的隐私设置到 PRIVATE，并阻止所有已关注您的 Trolls。
    </p>
    <p class="graf graf--p graf-after--p" id="807c">
     <strong class="markup--strong markup--p-strong">
      LINKEDIN
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="1ca9">
     对于许多人来说，如果被发现从事政治活动，那么他们就会面临特别的风险。如今这在任何一个国家都一样。
    </p>
    <p class="graf graf--p graf-after--p" id="73e0">
     要禁用公开可见的个人资料，请访问
     <a class="markup--anchor markup--p-anchor" data-href="https://www.linkedin.com/public-profile/settings" href="https://www.linkedin.com/public-profile/settings" rel="noreferrer noopener nofollow" target="_blank">
      https://www.linkedin.com/public-profile/settings
     </a>
     ，在右侧，您会看到“您的个人资料的公开可见度”。将其切换为“关闭”。
    </p>
    <p class="graf graf--p graf-after--p" id="3d1e">
     <strong class="markup--strong markup--p-strong">
      Facebook
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="db82">
     对于您的所有 Facebook 隐私设置，可以在这里找到它们：
     <a class="markup--anchor markup--p-anchor" data-href="https://www.facebook.com/help/325807937506242/" href="https://www.facebook.com/help/325807937506242/" rel="noreferrer noopener nofollow" target="_blank">
      https://www.facebook.com/help/325807937506242/
     </a>
    </p>
    <p class="graf graf--p graf-after--p" id="2832">
     <strong class="markup--strong markup--p-strong">
      关键是以下这些
     </strong>
     ：
    </p>
    <ul class="postList">
     <li class="graf graf--li graf-after--p" id="1998">
      更改您的设置，以便只有您的朋友才能看到您当前的帖子。当您想要发布与公共相关的工作内容时，请将这些单独的帖子设置为公开。观看此视频，保护过去的时间线帖子：
      <a class="markup--anchor markup--li-anchor" data-href="https://www.eff.org/deeplinks/2017/03/most-powerful-single-click-your-facebook-privacy-settings" href="https://www.eff.org/deeplinks/2017/03/most-powerful-single-click-your-facebook-privacy-settings" rel="noopener nofollow noreferrer" target="_blank">
       How To Video
      </a>
     </li>
     <li class="graf graf--li graf-after--li" id="c304">
      请查看您的朋友列表。如果有人关注了你或者你的朋友，但你不记得他们是谁，或者模糊地记得与他们进行过一些尴尬的互动，那么立刻拉黑他们。仔细检查您的每个朋友是否都是唯一的，
      <strong class="markup--strong markup--li-strong">
       查看是否有人伪造了和您的朋友相似的名字和照片的帐户
      </strong>
      ，如果有，立即拉黑。
     </li>
     <li class="graf graf--li graf-after--li" id="fb5c">
      同时浏览您的个人资料信息，确保您的电话号码和电子邮件设置为“仅限您自己可见”。
      <strong class="markup--strong markup--li-strong">
       删除您在个人资料网站上不希望被任何人看到的信息，包括精选照片什么的
      </strong>
      。这是因为常见的战术 Trolls 将会搜集你的专辑照片并将其传播到互联网上。他们会这样做，要么是为你创建一个虚假的个人资料，用于抹黑，要么就是骚扰 memes 等。
     </li>
     <li class="graf graf--li graf-after--li" id="13a8">
      删除您的 Facebook 所有公开照片，并将其替换为没有您在内的通用照片，并删除您的完整生日信息（或用不准确的信息替换它）。
     </li>
    </ul>
    <p class="graf graf--p graf-after--li" id="c525">
     <strong class="markup--strong markup--p-strong">
      Twitter
     </strong>
    </p>
    <ul class="postList">
     <li class="graf graf--li graf-after--p" id="5618">
      在您的帐户设置中，请确保您已验证所有登录请求，以便标记任何试图进入您帐户的人。
     </li>
     <li class="graf graf--li graf-after--li" id="fb9b">
      在您的隐私和安全设置中，请确保禁用位置信息。这可以防止通过推文泄露您的位置。
     </li>
     <li class="graf graf--li graf-after--li" id="8c8a">
      调整照片标记，以便随机的攻击帐户无法把您标记在骚扰内容或状态中。
     </li>
     <li class="graf graf--li graf-after--li" id="6392">
      关闭“通过电子邮件/电话找到我”选项。
     </li>
     <li class="graf graf--li graf-after--li" id="ccdc">
      如果您担心被假冒 Antifa 帐户关注，请考虑安装像
      <a class="markup--anchor markup--li-anchor" data-href="https://blocktogether.org/" href="https://blocktogether.org/" rel="noopener nofollow noreferrer" target="_blank">
       block together
      </a>
      这样的应用程序。 block together 可确保您可以更安心地操作，因为他们的目标是阻止再次骚扰和传播虚假信息。您还可以使用 Troll Busters 等服务反击大规模的 trolls，其中包含可以帮助淹没滥用行为的正面消息。有关详情，请访问
      <a class="markup--anchor markup--li-anchor" data-href="http://www.troll-busters.com/" href="http://www.troll-busters.com/" rel="noreferrer noopener nofollow" target="_blank">
       http://www.troll-busters.com
      </a>
      。
     </li>
    </ul>
    <p class="graf graf--p graf-after--li" id="44f2">
     ✔
     <strong class="markup--strong markup--p-strong">
      取消所有ORPHAN帐户
     </strong>
     。请记住，trolls 将使用他们在网上获得的任何信息来攻击您，他们会搜罗尽可能多的您拥有的账户。您长时间未使用的帐户可能会使您容易受到攻击，所以要保持安全并关闭它们。您可以访问
     <a class="markup--anchor markup--p-anchor" data-href="https://namechk.com/" href="https://namechk.com/" rel="noreferrer noopener nofollow" target="_blank">
      https://namechk.com
     </a>
     获取您可能忘记注册过的帐户列表。
    </p>
    <p class="graf graf--p graf-after--p" id="2f66">
     ✔ 最后，确保备份您的硬件。
    </p>
    <p class="graf graf--p graf-after--p" id="ff3b">
     很好就这样！也许你会觉得这很难记住，数字安全是一个系统，您正在创建并实施作为组织者必备的核心技能的一部分。
     <strong class="markup--strong markup--p-strong">
      数字安全没有灵丹妙药，它是一种意识和实践，通过不断重复及社区共同致力于保持安全而不断变得更好。
     </strong>
    </p>
    <p class="graf graf--p graf-after--p" id="d8cf">
     <strong class="markup--strong markup--p-strong">
      现在最好的防守是集体防守，我们要联合起来
     </strong>
     。⚪️
    </p>
