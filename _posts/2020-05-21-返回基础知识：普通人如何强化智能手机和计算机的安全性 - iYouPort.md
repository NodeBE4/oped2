  <p class="graf graf--p">
   信息安全专业人员撰写的文章经常会服务于已经对网络安全原理有高度了解的同行。
  </p>
  <p class="graf graf--p">
   如果您正在写技术白皮书或其他内容，那么会很好，但是，我认为
   <strong class="markup--strong markup--p-strong">
    对于每天使用电脑和手机的普通用户而言，他们需要的是任何人都可以采取的步骤以加强计算机和智能手机的安全性
   </strong>
   。在这方面，专业人士做得不够好。
  </p>
  <p class="graf graf--p">
   本文希望在这方面提供一些帮助：您的个人信息和电子设备的基础安全性。
  </p>
  <p class="graf graf--p">
   本文侧重于基础级指南，重点关注各种技能和知识水平的互联网用户的相关基本网络安全做法，对日常普通用户很有用。
  </p>
  <p class="graf graf--p">
   正如定义所指出的，强化某些东西是为了使其更安全以免受攻击。这就像使房屋免受飓风侵袭，或为军事车辆增添重型装甲以防弹。
  </p>
  <p class="graf graf--p">
   房屋或车辆可能仍会遭受一些损坏，但是重要的是，风暴或袭击结束后，房屋或车辆在保护房屋内的所有物品的同时，仍可以做到功能正常。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     基本计算机安全原理
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   作为计算机技术的日常用户，您必须对正在使用的系统和设备的安全概念和原理有一个基本了解，不需要深入，以便知道它们为什么存在以及如何帮助提供多种功能。这样您就可以利用它们来进行基本安全防御了。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     机密性
    </strong>
    —— 保护信息免遭未经授权的访问
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     完整性
    </strong>
    —— 信息保护防止未经授权修改数据
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     可用性
    </strong>
    —— 保护您访问信息的能力
   </li>
  </ul>
  <p class="graf graf--p">
   上述模型是网络安全的基础。当然，还有很多，但不要让事情复杂化。
  </p>
  <p class="graf graf--p">
   每当您考虑计算机、应用程序、智能手机、平板电脑或任何类型的电子设备时，都可以将其置于上述模型的环境中，并为它们分配风险级别，例如中低至高。
  </p>
  <p class="graf graf--p">
   在家用计算机浏览器上查看 Facebook 的机密性风险是否处于低、中或高水平？您需要能根据您自己的具体区别尽可能准确地回答这些问题。
  </p>
  <p class="graf graf--p">
   请注意，您访问的每个网站都将需要跟踪您的使用活动和偏好的cookie。
  </p>
  <p class="graf graf--p">
   众所周知，即使用户关闭了 Facebook 浏览器选项卡，某些 Cookie（例如 Facebook 的 cookie）也可以跟踪您的浏览器活动 …… 所以，要当心。
  </p>
  <p class="graf graf--p">
   重复同样的问题以获取对完整性和可用性的判断，您将能获得想法；在很多情况下，
   <strong class="markup--strong markup--p-strong">
    您需要能比较
   </strong>
   ，上述安全模型中的三根支柱中哪一根对您来说在当前情况下最重要？也就是合理地权衡。
  </p>
  <p class="graf graf--p">
   如果用户最关心的是能够访问其数据（即可用性），那么他们可能更愿意接受较少的完整性和机密性，因为在安全性方面，任何技术都总会取舍。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     删除所有不必要的应用程序
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   通过删除不必要的软件应用程序来减少计算机的攻击面。
  </p>
  <p class="graf graf--p">
   卸载您不需要或不常用的所有应用程序，在智能手机上执行相同的操作。但是，其中有些已经进入了操作系统，这需要更高级的技能和知识来消除。
  </p>
  <p class="graf graf--p">
   例如，Windows 操作系统预装有许多用户不知道的 “膨胀” 软件，手机也一样。
  </p>
  <p class="graf graf--p">
   可以将计算机和智能手机上运行的每项服务和应用程序视为唯一的
   <strong class="markup--strong markup--p-strong">
    攻击途径
   </strong>
   。所以，卸载不使用的程序，否则，如果存在可以利用的未修补漏洞，它们就会变成攻击者的门户。
  </p>
  <p class="graf graf--p">
   所有类型的操作系统都具有类似的漏洞。 Mac 用户曾经认为 Mac 没有病毒或恶意软件，然后网络罪犯开始将注意力集中在该操作系统上了 ……
  </p>

  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     最低特权原则
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   最低特权就是听起来的那样。它仅涉及为用户提供执行任何与工作相关的任务所需的最低访问权限。
  </p>
  <p class="graf graf--p">
   现在网络上存在着一些真正危险的威胁，例如掠夺性天敌、网络欺凌、和金融诈骗，那些不那么明智的人更容易陷入危险。
  </p>
  <p class="graf graf--p">
   尤其是如果您家里有孩子的话，请注意设定最低权限，以避免小孩的误操作。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     纵深防御
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   1、请定期确定是否需要安装更新以确保计算机的操作系统是最新的。
  </p>
  <p class="graf graf--p">
   2、Anti-Virus- Avast 具有出色的免费版本 —— 但现在 Avast 被曝光将用户私密数据出售以盈利，请谨慎考虑该公司的服务。实际上，任何顶级的AV供应商都可以做到这一点。您只需要做好定时检查，因为每天都有成千上万的
   <strong class="markup--strong markup--p-strong">
    新
   </strong>
   恶意软件被释放出来。
  </p>
  <p class="graf graf--p">
   3、像免费版本的 Spybot 这样的反间谍软件也可以考虑。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     更改默认帐户密码
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   实际上，您应该在家用 Wi-Fi 路由器或任何电子设备上要做的第一件事就是更新固件并更改其上的默认密码。
  </p>
  <p class="graf graf--p">
   请记住，您将其连接到互联网了，世界上任何地方的人都可能会扫描它的漏洞。如果他们发现您使用的是默认密码，他们就会侵入您的路由器和系统。不要让这种事情发生！
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     启用自动更新
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   这是很不言自明的。经常更新软件，否则将遭受漏洞利用的后果…… 保持最新状态的最佳方法是启用自动更新。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     使用密码管理器
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   使用长密码或更复杂的密码仍然是一种最佳做法，但也不要忘记，不可以两次使用相同的密码，这一点同样重要！
  </p>
  <p class="graf graf--p">
   那么应该如何记住所有这些密码呢？使用信誉良好的密码管理器服务，例如 1Password 或 LastPass，两者均提供免费服务级别。
  </p>
  <p class="graf graf--p">
   您只需要设置一个非常好的密码短语，即 “主密码”，它将用作解密密钥以解锁其余所有密码。
  </p>
  <p class="graf graf--p">
   有些人不赞成这个概念，因为将所有“鸡蛋”放在一个篮子里似乎非常危险，但是密码管理器和其他任何密码存储方法一样安全。
  </p>
  <p class="graf graf--p">
   请勿在浏览器中启用保存的密码！
  </p>
  <p class="graf graf--p">
   这只是邀请黑客来入侵。实际上任何以未锁定状态接触您的计算机的人都可以在浏览器中访问您的密码。如果攻击者能够通过恶意软件或其他方式远程破坏您的计算机，他们就很容易访问这些密码。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   KeePass Password Safe 是可与您的计算机和手机一起使用的另一个选项。
  </p>
  <p class="graf graf--p">
   KeePass 的不同之处在于，您可以将密码管理器应用程序下载到计算机或智能手机上（或同时下载到这两者），然后将所有密码本地保存在 C:\ 上的加密库中，这比将其存储在网站中或者写在纸上更好。
  </p>
  <p class="graf graf--p">
   写下密码并非完全不安全，但是大多数人都很懒，只是将它们放在计算机旁边或键盘下方。尽管这似乎是“简单”的选择，但是显然，非常不安全，如果您在工作中这样做，很可能会受到纪律处分。
  </p>
  <p class="graf graf--p">
   使用密码管理器的最有利效果之一就是更改密码非常容易。您想要一个50个字符的密码，包括大小写字母、数字和符号吗？没问题。这里有大把的选项提供给您。
  </p>
  <p class="graf graf--p">
   此外，还有一些内置功能可以测试您的密码的强度，并检查是否在其他地方重复了密码 —— 如果是，可能使攻击者在不同的网站上执行 “凭证填充”。
  </p>
  <p class="graf graf--p">
   您肯定不希望发生这种情况，因此最好立即养成对拥有帐户的每个网站使用
   <strong class="markup--strong markup--p-strong">
    唯一密码
   </strong>

  <p class="graf graf--p">
   5、安装付费的 VPN 应用程序，以保护您使用公共Wi-Fi热点时不会受到窥探者的攻击。
  </p>
  <p class="graf graf--p">
   这可能无法阻止警察和政府间谍跟踪您，但基本可以防止您的互联网服务提供商（ISP）和星巴克的 Hacker Bob 看到您在网上做什么。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    参见《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%a3%80%e6%9f%a5%e6%82%a8%e7%9a%84vpn%e8%bf%9e%e6%8e%a5%e6%98%af%e5%90%a6%e5%ae%9e%e9%99%85%e5%8a%a0%e5%af%86/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e6%a3%80%e6%9f%a5%e6%82%a8%e7%9a%84vpn%e8%bf%9e%e6%8e%a5%e6%98%af%e5%90%a6%e5%ae%9e%e9%99%85%e5%8a%a0%e5%af%86/" rel="noopener noreferrer" target="_blank">
     如何检查您的VPN连接是否实际加密
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   您应该知道，有许多不道德的VPN供应商您应该小心。在选择VPN提供商之前，请先做点调查，以便对得起您的付费。您需要考察的是以下这些方面：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e9%aa%8c%e8%af%81%e6%82%a8%e7%9a%84-vpn-%e8%bf%9e%e6%8e%a5%e6%98%af%e5%90%a6%e5%ae%89%e5%85%a8%ef%bc%9f/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e9%aa%8c%e8%af%81%e6%82%a8%e7%9a%84-vpn-%e8%bf%9e%e6%8e%a5%e6%98%af%e5%90%a6%e5%ae%89%e5%85%a8%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     如何验证您的 VPN 连接是否安全？
    </a>
    》
   </li>
  </ul>
  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     七步原则
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在您考虑自己的数字安全性时，需要一些基本的技巧。
  </p>
  <p class="graf graf--p">
   最基础的就是威胁建模 —— 全面分析您自己的需求，以便获得最恰当的防御模式。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bf%9d%e6%8a%a4%e5%ae%89%e5%85%a8%e9%9c%80%e8%a6%81%e4%b8%80%e7%b3%bb%e5%88%97%e6%80%9d%e8%80%83%e6%96%b9%e5%bc%8f%ef%bc%8c%e8%80%8c%e4%b8%8d%e6%98%af%e5%93%aa%e4%b8%aa%e8%bd%af%e4%bb%b6/" href="https://www.iyouport.org/%e4%bf%9d%e6%8a%a4%e5%ae%89%e5%85%a8%e9%9c%80%e8%a6%81%e4%b8%80%e7%b3%bb%e5%88%97%e6%80%9d%e8%80%83%e6%96%b9%e5%bc%8f%ef%bc%8c%e8%80%8c%e4%b8%8d%e6%98%af%e5%93%aa%e4%b8%aa%e8%bd%af%e4%bb%b6/" rel="noopener noreferrer" target="_blank">
     保护安全需要一系列思考方式，而不是哪个软件
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   首先，
   <strong class="markup--strong markup--p-strong">
    请问自己以下问题
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    我要保护什么？
   </li>
   <li class="graf graf--li">
    我想防的人是谁(们)？
   </li>
   <li class="graf graf--li">
    我做到保护的可能性有多大？
   </li>
   <li class="graf graf--li">
    如果我失败了，后果有多严重？
   </li>
   <li class="graf graf--li">
    我愿意付出多少以防止潜在的后果？
   </li>
  </ul>
  <p class="graf graf--p">
   回答这些问题后，您可以更好地评估您的数字安全需求、并创建安全计划或威胁建模。
  </p>
  <p class="graf graf--p">
   请始终记得以下6点：
  </p>
  <p class="graf graf--p">
   1、
   <strong class="markup--strong markup--p-strong">
    最薄弱的一环
   </strong>
   ——
  </p>
  <p class="graf graf--p">
   记住那句老话：最薄弱的一环会毁掉整个链条。例如，如果您根本没安窗闩，那么即便安装了最好的门锁也没有用。
  </p>
  <p class="graf graf--p">
   同样，如果您在笔记本电脑上存储未加密的副本并且笔记本电脑被盗，那么仅仅对电子邮件进行加密并没有用。
  </p>
  <p class="graf graf--p">
   请考虑信息和电子设备使用的每一个部分，尝试找出数字安全实践中的任何薄弱环节。
  </p>
  <p class="graf graf--p">
   2、
   <strong class="markup--strong markup--p-strong">
    更简单，更安全，更轻松
   </strong>
   ——
  </p>
  <p class="graf graf--p">
   每一个闪亮的、全新的安全解决方案都会对某些人产生诱惑。但是很快，人们发现自己使用了太多的工具，尝试了很多事，以至于已经错乱。
  </p>
  <p class="graf graf--p">
   尤其是，拥有复杂的安全系统使识别薄弱环节变得更加困难。
  </p>
  <p class="graf graf--p">
   因此，保持简单。
  </p>
  <p class="graf graf--p">
   有时，最安全的解决方案可能是技术含量最低的解决方案。就如在电脑摄像头上贴黑胶布、不要买任何所谓的的智能家居、出门前把手机放入法拉第袋…… 等等。
  </p>
  <p class="graf graf--p">
   也许计算机可以在很多方面发挥作用，但是有时简单的笔和便条纸的安全性问题更容易理解，因此更易于管理。
  </p>
  <p class="graf graf--p">
   3、
   <strong class="markup--strong markup--p-strong">
    更昂贵并不意味着更安全
   </strong>
   ——
  </p>
  <p class="graf graf--p">
   不要以为最昂贵的解决方案是最好的。特别是如果它占用了其他地方所需的资源的话。
  </p>
  <p class="graf graf--p">
   诸如在将垃圾丢在路边之前将其彻底切碎的低成本措施就可以为您带来很多好处。
  </p>
  <p class="graf graf--p">
   4、
   <strong class="markup--strong markup--p-strong">
    可以信任某人，但需要始终知道您信任的人 “是谁”
   </strong>
   ——
  </p>
  <p class="graf graf--p">
   计算机安全建议最终听起来像是您应该绝对不信任任何人，仅仅相信自己。
  </p>
  <p class="graf graf--p">
   在现实世界中，几乎可以肯定的是，至少从您的近亲、同伴到医生或律师，很多人都知道您的某些信息。
  </p>
  <p class="graf graf--p">
   而在数字空间中，棘手的是：如何了解您信任的人以及信任的对象。
  </p>
  <p class="graf graf--p">
   您可能会给您的律师提供密码列表：但是您应该考虑一下这样做可能赋予他们的权力 —— 是否有坏人可以轻易地访问您的密码。
  </p>
  <p class="graf graf--p">
   您可能会在被认为是 “私人” 的 Dropbox 或 Google 之类的云服务中编写文档：但是，与此同时您还允许了 Dropbox 和 Google 访问您的文档。
  </p>
  <p class="graf graf--p">
   请随时记得，不论是在线还是离线，与您共享秘密的人越少，保密的机会就越大。
  </p>
  <p class="graf graf--p">
   5、并没有一个完美的安全计划 ——
  </p>
  <p class="graf graf--p">
   您需要做的是制定一个对您有用的、对您面临的风险以及您和您的同事必然能够采取实施步骤的安全计划。
  </p>
  <p class="graf graf--p">
   如果不能做到每天都遵循该计划，这一计划就行不通。纸上谈兵不解决问题。
  </p>
  <p class="graf graf--p">
   6、
   <strong class="markup--strong markup--p-strong">
    今天安全的东西明天可能不安全
   </strong>
   ——
  </p>
  <p class="graf graf--p">
   不断重新评估您的安全实践至关重要。仅仅因为它们在去年或上周是安全的，并不意味着它们现在仍然安全。
  </p>
  <p class="graf graf--p">
   请不断关注您信任的安全知识平台发布的最新消息。请记住：安全性是一个连续的过程。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     最后
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   本文中所有这些安全措施都是给普通人 —— 它们是保护数据所需的 **最低限度** 的步骤。
  </p>
  <p class="graf graf--p">
   我们不想说这是个弱肉强食的世界，但如果您真的像绵羊一样，那就几乎肯定会招来狼。
  </p>
  <p class="graf graf--p">
   购买计算机和智能手机时不要仅仅研究性能方面哪个更好，您需要多钻研下如何正确地保护它们，以免遭受不必要的伤害。
  </p>
