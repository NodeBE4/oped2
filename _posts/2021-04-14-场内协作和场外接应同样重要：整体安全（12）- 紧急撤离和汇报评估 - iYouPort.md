   这里将是本套课程的最后一集。
  </p>
  <p class="graf graf--p">
   当团体成员处于压力之下，或感受到恐惧和受到威胁时，就会对集体动态产生负面影响。
   <strong class="markup--strong markup--p-strong">
    这样的群体可能会变得更加专制，人们可能会说出和做出伤害对方的事。
   </strong>
  </p>
  <p class="graf graf--p">
   您需要避免这种情况。
  </p>
  <p class="graf graf--p">
   当您在抗议活动中，需要快速做出决定时，恐惧、压力以及它们所导致的人际问题，可能会阻碍您做出决定的能力，使您陷入危险之中。
  </p>
  <p class="graf graf--p">
   📌 因此，在抗议活动之前，您应该先建立一个清晰的决策结构或共识程序，以及沟通渠道，这是很重要的。决定一套 “不可谈判” 措施  — — 针对给定场景的行动方案，这些方案不会在行动开始后进行辩论（如紧急撤离战略）。
  </p>
  <p class="graf graf--p">
   如果必须对某个问题进行辩论，考虑谁需要参与讨论：整个团队，还是一个工作组？
   <strong class="markup--strong markup--p-strong">
    如果辩论中不必要的声音太多，就会成为一种阻碍，它会减缓决策的速度。
   </strong>
  </p>
  <p class="graf graf--p">
   辩论时，尽量练习
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%9b%b4%e6%8e%a5%e8%a1%8c%e5%8a%a8%e7%9a%84%e5%85%b3%e9%94%ae%e5%9c%a8%e4%ba%8e%e5%8d%8f%e4%bd%9c%ef%bc%8c%e4%bd%86%e5%8d%8f%e4%bd%9c%e7%9a%84%e8%83%bd%e5%8a%9b%e5%b9%b6%e9%9d%9e%e5%a4%a9%e7%94%9f/" href="https://www.iyouport.org/%e7%9b%b4%e6%8e%a5%e8%a1%8c%e5%8a%a8%e7%9a%84%e5%85%b3%e9%94%ae%e5%9c%a8%e4%ba%8e%e5%8d%8f%e4%bd%9c%ef%bc%8c%e4%bd%86%e5%8d%8f%e4%bd%9c%e7%9a%84%e8%83%bd%e5%8a%9b%e5%b9%b6%e9%9d%9e%e5%a4%a9%e7%94%9f/" rel="noopener" target="_blank">
    积极倾听
   </a>
   ，以避免误解，确保每个人都能理解所谈的内容。作为一个过程，📌 可以考虑
   <a class="markup--anchor markup--p-anchor" data-href="https://wikipedia.org/wiki/Consensus_decision-making" href="https://wikipedia.org/wiki/Consensus_decision-making" rel="noopener" target="_blank">
    共识决策法
   </a>
   ：每个参与者都可以否决一项提案，但否决权只能与反提案一起使用。也就是说，如果没有更好的建议，否决权就不成立。
  </p>
  <p class="graf graf--p">
   偶尔，恐惧可以非常有用地帮助我们逃离危险的环境  — — 只是基于一种本能的感觉，认为有什么不对劲。如果您的直觉告诉您正处于危险之中，这种情况下
   <strong class="markup--strong markup--p-strong">
    建议您与自己的亲密团队中的某个人分享，并检查您的感知
   </strong>
   ：您所感知的指标是什么，这一指标告诉您有什么不对劲？与您的朋友或亲密团队一起分析，并共同做出决定。
  </p>
  <p class="graf graf--p">
   📌 亲密团队应该为分享抗议前后的恐惧和压力影响提供明确的空间。此外，为了放松和管理压力，单独或与亲密团队一起进行愉快的活动是健康的（selfcare）。考虑让您的亲密团队中的一个或多个人负责组织这项活动。
  </p>
  <p class="graf graf--p">
   关于恐惧和对威胁的本能响应的更多信息，可以查看本系列中的第2集。
  </p>
  <p class="graf graf--p">
   在抗议或示威活动中，您经常可能目睹当局或其他反对您的抗议目标的团体的侵犯人权行为。为了协调反应，您必须尽可能准确地报告所发生的事。而当您就是当事人时，很可能会感到害怕或痛苦，这种情况下报告事实可能会非常困难。
  </p>
  <p class="graf graf--p">
   为了避免这种情况，您需要在行动前对此拟定一个框架/准则，在您的压力水平正常、足够冷静的情况下确立这个框架，在现场状态下您只需要按照框架的要求去回答问题就可以了，就像做填空题。
  </p>
  <p class="graf graf--p">
   📌 在紧急情况下，准确的信息流是至关重要的。确定需要联系的人和优先顺序，优先考虑能够提供有效和即时支持的人。
  </p>
  <p class="graf graf--p">
   在试图报告任何事情之前，您必须做的第一件事就是尽量到一个安全的地方；然后，与 *最有可能在安全空间中的人* 取得联系  — — 最好是一开始就不在抗议现场内的人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    您应该事先准备好他们的联系方式，非常建议写下来，写在自己身上或确保不会丢失的便条上，如果只放在手机里可能是不够安全的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   📌 事先花点时间整理一下您的想法，在您的框架内加入以下问题：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    你是谁，你在哪里？
   </li>
   <li class="graf graf--li">
    发生了什么事？
   </li>
   <li class="graf graf--li">
    在哪里发生的？
   </li>
   <li class="graf graf--li">
    什么时候发生的？
   </li>
   <li class="graf graf--li">
    谁受到了影响？
   </li>
   <li class="graf graf--li">
    谁是肇事者，你怎么知道？
   </li>
   <li class="graf graf--li">
    你需要什么？
   </li>
   <li class="graf graf--li">
    如何以及何时能联系到你？
   </li>
  </ul>
  <p class="graf graf--p">
   如果您认为某些情况很有可能发生，那么在抗议前练习一下可能会很有帮助。
  </p>
  <p class="graf graf--p">
   📌 如果您的抗议者同伴被捕，请考虑以下几点：
  </p>
  <p class="graf graf--p">
   1. 记下逮捕的地点和时间，以及有关逮捕官员和任何有关车辆的信息  — — 姓名或警号通常印在警服上，不难识别。这些信息对于在亲密团队、人权组织、和您想要合作的媒体中传播，至关重要。
  </p>
  <p class="graf graf--p">
   2. 如果被逮捕的人是您的亲密团队的成员，
   <strong class="markup--strong markup--p-strong">
    请立刻进行点名，确认其他人都没事
   </strong>
   。正如我们在
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ae%89%e5%85%a8%e5%9c%b0%e4%bd%bf%e7%94%a8-signal%ef%bc%9a%e5%87%86%e5%88%99/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ae%89%e5%85%a8%e5%9c%b0%e4%bd%bf%e7%94%a8-signal%ef%bc%9a%e5%87%86%e5%88%99/" rel="noopener" target="_blank">
    安全通信准则
   </a>
   中讲述的，
   <strong class="markup--strong markup--p-strong">
    您最好能做到定时点名，以便能够及时发现失踪的队友
   </strong>
   。
  </p>
  <p class="graf graf--p">
   3. 确认被拘留者的位置、他或她的身体状况的细节、以及当局对他或她的指控是什么。如果您认为可能会出现被逮捕的情况，您应该在抗议之前制定一个全面的策略，可以有渠道要求立即释放，最好是让本国和/或国际人权组织参与进来。
  </p>
  <p class="graf graf--p">
   4. 如果您不希望逮捕事件阻断您的行动，则应
   <strong class="markup--strong markup--p-strong">
    进行重新组合，调整计划，必要时重新为每一位队友分配职责
   </strong>
   。所以，您的战略中需要有Plan B，把这些可能性及应对措施都加入进去。
  </p>
  <p class="graf graf--p">
   📌 具体方案可以参见我们此前提供的《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%9b%b4%e6%8e%a5%e8%a1%8c%e5%8a%a8%e5%88%86%e6%ad%a5%e6%8c%87%e5%8d%97%ef%bc%9a%e5%ae%83%e6%98%af%e4%bb%80%e4%b9%88%e3%80%81%e5%ae%83%e6%9c%89%e4%bb%80%e4%b9%88%e7%94%a8%e3%80%81%e5%a6%82%e4%bd%95/" href="https://www.iyouport.org/%e7%9b%b4%e6%8e%a5%e8%a1%8c%e5%8a%a8%e5%88%86%e6%ad%a5%e6%8c%87%e5%8d%97%ef%bc%9a%e5%ae%83%e6%98%af%e4%bb%80%e4%b9%88%e3%80%81%e5%ae%83%e6%9c%89%e4%bb%80%e4%b9%88%e7%94%a8%e3%80%81%e5%a6%82%e4%bd%95/" rel="noopener" target="_blank">
    直接行动分布指南
   </a>
   》，那是一份比较详尽的行动中战略建议。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在抗议活动结束时，或者如果您提前离开，您的第一反应可能是放松，这会导致您分心。当局的镇压者或您的任何对手都可能会利用这个时机来识别和骚扰抗议者，特别是当人们分成小团体或独自离开时
   </strong>
   。
  </p>
  <p class="graf graf--p">
   为了保持安全，请考虑以下几点：
  </p>
  <p class="graf graf--p">
   1.
   <strong class="markup--strong markup--p-strong">
    保持警惕
   </strong>
   ：继续监测您周围的情况，直到可以确定您已经抵达一个更安全的空间。
  </p>
  <p class="graf graf--p">
   2.
   <strong class="markup--strong markup--p-strong">
    不要独自离开
   </strong>
   ：至少成双成对地离开，或与您的亲密团队一起离开。
  </p>
  <p class="graf graf--p">
   3.
   <strong class="markup--strong markup--p-strong">
    沟通
   </strong>
   ：让抗议活动之外的人知道您要准备离开了，以及他们应该期待您何时到达预定地点。尽量使用安全的通信渠道进行！这样一来如果您没能按照预期时间抵达预期地点，您的联络人应该立刻提高警惕，推测可能出事了。
  </p>
  <p class="graf graf--p">
   4.
   <strong class="markup--strong markup--p-strong">
    反监视
   </strong>
   ：如果您觉得自己被跟踪了，不要直接走小路，而是要尽量迷惑跟踪您的人，让他们难以确定您的真实目的地。对此我们有过详细介绍，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9c%a8%e4%bd%a0%e7%9a%84%e6%97%a5%e5%b8%b8%e7%94%9f%e6%b4%bb%e4%b8%ad%e5%ba%94%e7%94%a8%e9%9a%90%e8%94%bd%e9%87%8e%e6%88%98%e6%8a%80%e6%9c%af/" href="https://www.iyouport.org/%e5%9c%a8%e4%bd%a0%e7%9a%84%e6%97%a5%e5%b8%b8%e7%94%9f%e6%b4%bb%e4%b8%ad%e5%ba%94%e7%94%a8%e9%9a%90%e8%94%bd%e9%87%8e%e6%88%98%e6%8a%80%e6%9c%af/" rel="noopener" target="_blank">
    在你的日常生活中应用隐蔽野战技术
   </a>
   》。
  </p>
  <p class="graf graf--p">
   5.
   <strong class="markup--strong markup--p-strong">
    不要逗留
   </strong>
   ：不要在抗议活动发生的区域逗留太久。
  </p>
  <p class="graf graf--p">
   6.
   <strong class="markup--strong markup--p-strong">
    考虑如何处理材料
   </strong>
   ：横幅、传单和其他材料可能会把您和抗议活动联系起来。尽量安全地处理它们，或将它们打包藏到外人看不见的地方。
  </p>
  <p class="graf graf--p">
   7.
   <strong class="markup--strong markup--p-strong">
    照顾好您拍摄的照片和录音或录像
   </strong>
   ：把它们藏在身上，或者在拍摄后立即让一个不显眼的 “慢跑者” 把它们带到安全的地方。详细做法我们曾经介绍过，见：

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     紧急情况下的安全
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   规划一个安全去处，就如作为联络点的安全屋，这可以在紧急情况下节省宝贵的时间。选择安全场所的过程也可以培养您的技能，以防万一，您必须随机应变。
  </p>
  <p class="graf graf--p">
   📌 在决定安全地点时请考虑以下几点：
  </p>
  <p class="graf graf--p">
   1.
   <strong class="markup--strong markup--p-strong">
    您为什么要去
   </strong>
   ：您要逃避什么，这个地方能以什么方式保护您？如果此地会导致您与您的关系网络断开、或增加他人的风险，应该尽量避免选择。
  </p>
  <p class="graf graf--p">
   2.
   <strong class="markup--strong markup--p-strong">
    您要去的地方
   </strong>
   ：考虑您可以从抗议地点去哪里，在那里您是否容易被找到，您是否可以或应该带着您的电子设备去那个地方（电子设备可能会导致容易跟踪您），您在那里可以或不可以执行什么职责（职责就是为每一位队友安排的角色），以及当您到达那里时，您有希望够得到什么样的医疗协助或情感协助。
  </p>
  <p class="graf graf--p">
   3.
   <strong class="markup--strong markup--p-strong">
    如何到达那里
   </strong>
   ：考虑您要逃离的对手是谁，他们的资源如何，以及您自己的资源如何（您的知识、您的车辆、您的朋友、您带的现金）：如何以及和谁一起前往最好？
  </p>
  <p class="graf graf--p">
   4.
   <strong class="markup--strong markup--p-strong">
    联系和支持结构
   </strong>
   ：尽量提前让一位足够信任的队友知道您在紧急情况下可能去的地方，这样您就不必在紧急状态下通过不安全的渠道进行沟通了。如果您必须临时告诉其他人您在哪里，请尽量通过安全的渠道，让对方把消息传播给最需要知道这件事的人  — —
   <strong class="markup--strong markup--p-strong">
    此人必须确定能够认同并熟练使用《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e7%9a%84%e5%9b%a2%e9%98%9f%e9%99%b7%e5%9c%a8%e8%a2%ab%e7%9b%91%e8%a7%86%e5%92%8c%e8%a2%ab%e6%b8%97%e9%80%8f%e7%9a%84%e6%81%90%e6%85%8c%e4%b8%ad%ef%bc%8c%e5%8f%aa%e6%98%af/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e7%9a%84%e5%9b%a2%e9%98%9f%e9%99%b7%e5%9c%a8%e8%a2%ab%e7%9b%91%e8%a7%86%e5%92%8c%e8%a2%ab%e6%b8%97%e9%80%8f%e7%9a%84%e6%81%90%e6%85%8c%e4%b8%ad%ef%bc%8c%e5%8f%aa%e6%98%af/" rel="noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     安全文化
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》！
   </strong>
  </p>
  <p class="graf graf--p">
   5.
   <strong class="markup--strong markup--p-strong">
    最低限度的资源
   </strong>
   ：您需要什么样的最低限度的资源才能到达那个安全的地方？
  </p>

   📌 上述内容我们曾经有过非常详细的介绍，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%81%9a%e5%88%b0%e5%ae%8c%e7%be%8e%e9%9a%90%e8%ba%ab%ef%bc%9a%e6%89%ab%e8%8d%a1%e5%8d%b1%e9%99%a9%e5%8c%ba%e5%9f%9f%ef%bc%887%ef%bc%89/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%81%9a%e5%88%b0%e5%ae%8c%e7%be%8e%e9%9a%90%e8%ba%ab%ef%bc%9a%e6%89%ab%e8%8d%a1%e5%8d%b1%e9%99%a9%e5%8c%ba%e5%9f%9f%ef%bc%887%ef%bc%89/" rel="noopener" target="_blank">

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     汇报、评估和学习
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   抗议结束后，最好与您的亲密团队举行一次 “签出” 汇报会议。当然，这应该在一个安全的空间里进行。
  </p>
  <p class="graf graf--p">
   这种汇报很重要，因为它有助于：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    防止进一步的伤害，并在抗议活动后做最后的威胁评估；
   </li>
   <li class="graf graf--li">
    照顾身处困境中的人；
   </li>
   <li class="graf graf--li">
    吸取教训，改进今后的抗议或行动。
   </li>
  </ul>
  <p class="graf graf--p">
   📌 汇报应该在抗议活动结束后不久进行。最好事先商定好要问的问题。一个
   <strong class="markup--strong markup--p-strong">
    样本议程
   </strong>
   可能是这样的：
  </p>
  <p class="graf graf--p">
   1. 感觉检查  — — 每个成员都向小组汇报自己的情况。任何紧急的需求（医疗或情感）都应该优先得到处理。
  </p>
  <p class="graf graf--p">
   2. 威胁评估  — — 分享关于当前安全形势的指标，评估抗议活动后团队成员可能面临的任何威胁，以及如何处理这些威胁（即：更新安全计划）。
  </p>
  <p class="graf graf--p">
   3. 学习和改进的机会 — — 在这次抗议活动中哪些地方做得很好？下次应该改进哪些地方？是否出现了任何没有预料到的或计划到的安全指标或威胁？
  </p>
  <p class="graf graf--p">
   4. 接下来的步骤 — — 在您的团队再次重聚之前，现在必须做出哪些决定？
  </p>
  <p class="graf graf--p">
   确保你们耐心地相互沟通，并积极倾听，以确保每个人都清楚所吸取的教训和下一步行动。
  </p>
  <p class="graf graf--p">
   如果您或您的团体受到不公正的待遇，考虑向有关权利组织或公民社会团体投诉。在这样做之前，先从您的律师或当地的人权团体那里获得法律建议和支持。如果您担心分享您所收集的信息的安全性和法律影响，请就如何最好地开展工作征求专家意见。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     最后
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   恭喜您完成 “整体安全” 系列课程！现在您应该已经掌握了一个科学的安全框架，从策划到部署、从组织协调到任务分配、从技术安全到心理安全、从个人到团队再到联盟，这一框架将适用于绝大多数行动计划，不论是直接行动、传统行动、还是网络
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%e6%8a%80%e5%b7%a7%e5%b8%ae%e5%8a%a9%e6%82%a8%e5%9c%a8%e8%a1%8c%e5%8a%a8%e4%b8%ad%e6%9b%b4%e5%85%b7%e6%95%88%e8%83%bd%ef%bc%9a-%e4%bf%a1%e6%81%af/" href="https://www.iyouport.org/%e7%a4%be%e4%ba%a4%e5%aa%92%e4%bd%93%e6%83%85%e6%8a%a5%e6%8a%80%e5%b7%a7%e5%b8%ae%e5%8a%a9%e6%82%a8%e5%9c%a8%e8%a1%8c%e5%8a%a8%e4%b8%ad%e6%9b%b4%e5%85%b7%e6%95%88%e8%83%bd%ef%bc%9a-%e4%bf%a1%e6%81%af/" rel="noopener" target="_blank">
    信息行动主义
   </a>
   。
  </p>
  <p class="graf graf--p">
   本系列内容中加入的所有链接都很重要，它们是相关部分的详细介绍和操作方法，如果您准备使用这套课程在自己的组织中安排培训，请将其中嵌入的链接部分作为知识点进行。
  </p>
  <p class="graf graf--p">
   祝您的行动一切顺利安全。⚪️
  </p>
