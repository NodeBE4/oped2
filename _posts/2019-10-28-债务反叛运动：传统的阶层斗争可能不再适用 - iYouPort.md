  <p class="graf graf--p">
   人们讨论债务问题已经有很多年了，可惜目前为止依旧停留在讽刺“负翁”的层面上。这就失去了一个重要的机会。
  </p>
  <p class="graf graf--p">
   介绍一下美国的反抗运动，也许能打开一些思路。
  </p>
  <p class="graf graf--p">
   **
   <strong>
    当许多人被债务压垮时，需要反击的是金融剥削
   </strong>
   。**
  </p>
  <p class="graf graf--p">
   说到反压迫的公民不合作，你能想到什么？没错，几乎数不胜数。它可以有无数种形式。
  </p>
  <p class="graf graf--p">
   就如激进主义黑人活动家 Rosa Parks 拒绝在种族隔离的公交车上放弃那些“white only”的座位，这种看似简单的不合作就能产生巨大的效果。同样，拒绝支付每月的债务也是不合作的直接行动。
  </p>
  <p class="graf graf--p">
   由于许多国家的工资自20世纪70年代以来一直停滞不前，人们越来越多地转向了债务融资来支付教育、住房和医疗保健费用。
  </p>
  <p class="graf graf--p">
   银行积极追求并从这种债务爆炸中获利，助长了经济不平等，加剧了大规模的信贷泡沫，并以契约奴役形式俘获了数百万美元。
  </p>
  <p class="graf graf--p">
   无论成本如何，大多数人都觉得有义务偿还贷款，或担心违约的持久后果，但是，金融危机已经开始改变这一点。
  </p>
  <p class="graf graf--p">
   在看到政府砍掉了数万亿美元的救助资金、并向大银行提供廉价贷款之后，越来越多的人认识到：我们的
   <strong class="markup--strong markup--p-strong">
    债务负担是一个结构性问题、一个大规模的骗局，而不是个人失败或合法义务。
   </strong>
  </p>
  <p class="graf graf--p">
   但要求政治家和银行的宽恕是不可能的，因为我们的付款是他们的利润。我们需要的是
   <strong class="markup--strong markup--p-strong">
    杠杆作用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   美国活动家 Sarah Jaffe 和 Matthew Skomarovsky 提出建议：进入债务罢工，债务人集体谈判的反抗实验。
  </p>
  <p class="graf graf--p">
   这个想法很简单：我们停止向银行支付账单，直到他们答应进行谈判。
  </p>
  <p class="graf graf--p">
   因为没有这些付款他们就无法运作 —— 不论是学生贷款、抵押贷款或消费者信贷 —— 他们面临着很大的谈判压力。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种罢工可以与改革金融体系、废除掠夺性和高利贷贷款条件、或提供直接债务免除的要求联系起来。
   </strong>
  </p>
  <p class="graf graf--p">
   💡罢工者甚至可以汇集他们未支付的部分或全部资金，并将其投入“罢工基金”以支持该活动、或启动基于社区的替代信贷系统。
  </p>
  <p class="graf graf--p">
   协调是关键。
  </p>
  <p class="graf graf--p">
   ⚠️独自行动是无效的，只能使自己暴露于报复和分裂。
  </p>
  <p class="graf graf--p">
   相反，参与者应该签署承诺 —— 不论是公开的还是保密的 —— 以停止支付某些账单。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当足够多的人注册以提供真正的杠杆时，罢工效应就会出现。
   </strong>
  </p>
  <p class="graf graf--p">
   与此同时，组织要活跃，不断宣传总计数字，累计来自各方面的不满，收集无耻的债务故事，并观察金融精英的恐慌。
  </p>
  <p class="graf graf--p">
   债务罢工大胆、简单、易于参与 —— 比支付账单更容易，因为您所要做的就是不支付账单。
  </p>
  <p class="graf graf--p">
   这需要勇气和社会支持，但需要提供即时的满足感以形成凝聚力。
  </p>
  <p class="graf graf--p">
   在美国，人们多年来一直在动员取消抵押品赎回权和掠夺性贷款；
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “占领学生债务运动”组织旨在收集一百万名学生拒绝债务的承诺；
  </p>
  <p class="graf graf--p">
   还有另一个社会组织正在建立一个社会承诺系统，通过邻里、共同贷款人和需求来连接债务人。
  </p>
  <p class="graf graf--p">
   互联网上，像 ThePoint 这样的承诺行动平台、和像各大社交网络这样的故事聚合器，很快就可以成为债务战场上的武器。
  </p>
  <p class="graf graf--p">
   人们的愤怒、组织者、技术和工具都已经存在，有可能在全球范围内发起行动。
  </p>
  <p>
   <img alt="" class="aligncenter wp-image-6273 size-full jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/债务.png?resize=1100%2C620&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/债务.png?w=1252&amp;ssl=1 1252w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/债务.png?resize=300%2C169&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/债务.png?resize=768%2C433&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/债务.png?resize=1024%2C577&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/债务.png?resize=1100%2C620&amp;ssl=1 1100w" data-recalc-dims="1" height="620" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/10/债务.png?resize=1100%2C620&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    关键原理 ——
   </strong>
  </p>
  <p class="graf graf--p">
   债务经常被视为个人失败、被羞于公开讨论，而不是反对系统性剥削的共同斗争。
  </p>
  <p class="graf graf--p">
   大多数人倾向于将债务视为不可谈判的事实，而不是社会结构问题。
  </p>
  <p class="graf graf--p">
   这些根深蒂固的思考方式需要在行动中改变。
  </p>
  <p class="graf graf--p">
   一旦人们能意识到债务是可以重新协商甚至完全拒绝的，就会发现，
   <strong class="markup--strong markup--p-strong">
    人们有能力取消这个依赖于大众的羞耻感和忍耐才能运作的系统。
   </strong>
  </p>
  <p class="graf graf--p">
   美国的家庭债务约占GDP的90％，增长率几乎是实际收入的两倍，正如 Mike Konczal 所指出的那样，
   <strong class="markup--strong markup--p-strong">
    不成比例地影响了社会底层的99％
   </strong>
   。
  </p>
  <p class="graf graf--p">
   正如“占领学生债务”运动的口号所说的：“付不起钱吗？那就不要付钱？不会付钱！”
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    潜在危险 ——
   </strong>
  </p>
  <p class="graf graf--p">
   虽然最初的注册参与与签署在线请愿书一样容易，但是与请愿书不同，这一注册可能会产生严重后果。
  </p>
  <p class="graf graf--p">
   贷款违约会影响您的信用评级、这会严重影响您未来获得信用卡、租房、买车甚至找工作的能力。不论在美国还是中国都是如此。
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    成功的债务罢工必须有强大的支持网络
   </strong>
   ，就像工会有一个支持罢工工人的罢工基金一样。
  </p>
  <p class="graf graf--p">
   达到战术有效性所需的临界质量也可能是一项挑战 ——
   <strong class="markup--strong markup--p-strong">
    这项运动只能是大规模才能有效。
   </strong>
  </p>
  <p class="graf graf--p">
   还有一个在动员方面容易滑入的陷阱。
  </p>
  <p class="graf graf--p">
   由于今天的阶级意识越来越多地出现在债务人 — 债权人的界限上、而不是工人 — 资本家的界限上。
  </p>
  <p class="graf graf--p">
   于是行动者在策划方面也很容易被这一思路所牵制。
  </p>
  <p class="graf graf--p">
   许多激进主义者的公报都采用了阶级斗争的经典语言。但是，⚠️这种语言不仅经常无法调动参与，甚至很容易疏远那些有可能会焕发同情心的人。
  </p>
  <p class="graf graf--p">
   于是，
   <strong class="markup--strong markup--p-strong">
    建议以债务为中心的组织
   </strong>
   ，有可能重振二十一世纪的激进斗争。
  </p>
  <p class="graf graf--p">
   劳工运动的语言出现在电力织机还是工业驱动力的时代，贵族控制着土地和国家的时代，那时候大多数劳动人民都是直接生产者。
  </p>
  <p class="graf graf--p">
   而今天，发达国家的大多数人都是非直接生产者，从事客户服务、财务和其他行政或技术领域的工作。
  </p>
  <p class="graf graf--p">
   因此，他们不再能够直接目睹从他们身上偷走并被资本家囤积起来的劳动成果，
   <strong class="markup--strong markup--p-strong">
    剥削正在以越来越阴险的方式深入和细分
   </strong>
   。
  </p>
  <p class="graf graf--p">
   今天的人们并不认为“劳动产品”是雇主出售的实际商品；在他们看来，他们劳动的产物就是他们的薪水。
  </p>
  <p class="graf graf--p">
   这就是他们生产的东西，即从他们的手中获取的东西 —— 即：他们的账单、债务、税收。
  </p>
  <p class="graf graf--p">
   在美国，
   <strong class="markup--strong markup--p-strong">
    二十年的轻松信贷和泡沫经济使大多数人深陷债务之中，这通常是因为人们必须支付教育、儿童保育、住房和医疗等必需品
   </strong>
   。
  </p>
  <p>
   预计中国也可能很快体验到这点：
  </p>

  <p class="graf graf--p">
   对于活动家来说，这是一个真实的机会，可以证明资本主义根本无法公平有效地提供必需品，他们的债务是不公正的，是人们被迫接受的。
