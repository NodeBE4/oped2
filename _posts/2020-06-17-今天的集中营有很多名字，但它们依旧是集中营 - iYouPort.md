  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote">
   <p>
    “集中营的使用改变了世界，但展望未来，使用集中营最可预见的结果是一个拥有更多集中营的世界。”
   </p>
  </blockquote>
  <p class="graf graf--p">
   今天的人们使用了许多委婉语来描述这种设施：再教育营、拘留中心、劳动营、监狱营、国内无家可归者营地等等。但是，在第一次世界大战之前，所有这些东西都被简单地称为集中营，它们于1890年代在古巴开始，当时的目的是控制针对西班牙殖民者的本地起义。
  </p>
  <p class="graf graf--p">
   然而从那时起，
   <strong class="markup--strong markup--p-strong">
    集中营就在全球范围内激增，并已成为试图控制大型人群的政府的首选工具。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在过去的100年中，整个种族都被围捕、未经审判就被监禁、并且被无限期羁押，慢慢饿死、或死于疾病和忽视，甚至在美国也是如此
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些营地有什么共同点？是什么使它们与众不同？最重要的是，我们如何与之抗争？
  </p>
  <p class="graf graf--p">
   这篇访谈是与 Andrea Pitzer 交流关于她的著作《漫漫长夜：全球集中营历史》。她追溯了集中营的演变，从古巴最早的化身到当今中国、缅甸和美国关塔那摩的现代形式。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 访谈笔录 ——
   </strong>
  </h3>
  <p class="graf graf--p">
   <span style="color: #3366ff;">
    <strong class="markup--strong markup--p-strong">
     匿名人士
    </strong>
   </span>
   ：我们没有工作，也没有人权，我们不幸生活在这些难民营中。其他社区可以自由行走，而我们不能，我们不能做事，不能学习。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     主持人
    </strong>
   </span>
   ：在罗兴亚人过去居住的郊区实兑，坐落着一个庞大的国内流离失所者营地 IDP Camps。2012年，缅甸军方开始对罗兴亚人的村庄进行恐怖统治，强奸并杀害他们。其余的罗兴亚人被集中在一起，被迫进入国内流离失所者营地。这位匿名人士是该营地的幸存者之一。
  </p>
  <p class="graf graf--p">
   <span style="color: #3366ff;">
    <strong class="markup--strong markup--p-strong">
     匿名人士
    </strong>
   </span>
   ：这种情况就像 …… 动物。我们活得就像动物。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     主持人
    </strong>
   </span>
   ：他和他的所有家人​​都在这个营地里，等他们被释放时，他们的生活已经非常艰难。
  </p>
  <p class="graf graf--p">
   <span style="color: #3366ff;">
    <strong class="markup--strong markup--p-strong">
     匿名人士
    </strong>
   </span>
   ：我的生活被彻底毁了。我今年38岁。我有四个孩子。我担心孩子的未来生活。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     主持人
    </strong>
   </span>
   ：他们不能离开。他们没有工作，也没有机会接受教育。武装警卫用步枪在边界巡逻。他们每个月收到一次口粮，这意味着他们经常饿。这种情况使营地居民疲惫不堪，即使他们能组织起来进行反击（也没有力气）。
  </p>
  <p class="graf graf--p">
   <span style="color: #3366ff;">
    <strong class="markup--strong markup--p-strong">
     匿名人士
    </strong>
   </span>
   ：我们如何获得人权和正义？
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     主持人
    </strong>
   </span>
   ：定量口粮、按种族隔离人群、武装警卫。如果这听起来很熟悉，那就是。
   <strong class="markup--strong markup--p-strong">
    它们正是我们所谓的集中营的标志
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     Andrea Pitzer
    </strong>
   </span>
   ：集中营的构想首先不是针对战俘的，而是针对平民，被关押者并不是战斗人员。
   <strong class="markup--strong markup--p-strong">
    这是未经审判就大规模拘留平民的行为，通常是基于种族、族裔、宗教或政治背景，而不是他们所做的任何事
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     主持人
    </strong>
   </span>
   ：集中营。这个词应该是二战和大屠杀的幽灵。实际上，仅使用这个概念就会让人产生强烈的情感反应。但是，并非所有集中营都像我们在大屠杀中所看到的那样。
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     Andrea Pitzer
    </strong>
   </span>
   ：每当我与人交谈这件事时，我总是喜欢预先说明，我们不是在谈论奥斯威辛集中营，我们不是在谈论死亡工厂，也许我们以后再谈。但是首先，你知道，这不是什么新发明。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     主持人
    </strong>
   </span>
   ：事实上，
   <strong class="markup--strong markup--p-strong">
    认识集中营的迹象很重要，因为根据 Andrea Pitzer 的说法，全世界目前都充满了集中营
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     Andrea Pitzer
    </strong>
   </span>
   ：在冷战时期，你真的可以看到，殖民地的集中营制度与共产主义的集中营制度共同分裂了整个世界。每个国家都在打造集中营。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    主持人
   </span>
   ：每个国家都在打造集中营 —— 中国有关押维吾尔人和政治对手的集中营；美国有关押移民的集中营；还有朝鲜。您可能也会认为关塔那摩湾拘押中心是一个集中营。这就是我们今天所谈论的。我们到底是怎么走到这一步的？
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e5%92%8c%e7%be%8e%e5%9b%bd%ef%bc%8c%e4%b8%a4%e4%b8%aa%e9%9b%86%e4%b8%ad%e8%90%a5/" href="https://www.iyouport.org/%e4%b8%ad%e5%9b%bd%e5%92%8c%e7%be%8e%e5%9b%bd%ef%bc%8c%e4%b8%a4%e4%b8%aa%e9%9b%86%e4%b8%ad%e8%90%a5/" rel="noopener noreferrer" target="_blank">
     中国和美国，两个集中营
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   Pitzer 写了一本有关该主题的书，名为 “漫漫长夜：全球集中营历史”。在本节目的第一部分中，我们将探讨
   <strong class="markup--strong markup--p-strong">
    集中营的兴起及其如何演变成了世界各地几乎是永久性的机构
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     Andrea Pitzer
    </strong>
   </span>
   ：当我写第一本书时，那是写洛丽塔的 Vladimir Nabokov 的政治传记，他的兄弟死于德国集中营，第一次世界大战期间的集中营出现在他的作品中，还有苏联古拉格。这让我想到，集中营的概念什么时候进入世界的，如何
   <strong class="markup--strong markup--p-strong">
    被大众所默许
   </strong>
   ？
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     主持人
    </strong>
   </span>
   ：因此，在深入探讨集中营的历史之前，让我们先谈谈它们的一些定义特征，或使它们与众不同的地方。因为在某些方面，集中营和现代监狱非常相似。
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     Andrea Pitzer
    </strong>
   </span>
   ：许多大型集中营系统发生的事情之一就是它们变成了自己的小世界。因此，实际上就像是单独监禁的监禁室，有时甚至大型集中营系统中也有监狱。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     主持人
    </strong>
   </span>
   ：但是它们之间有很大的不同。正如我们所提到的，集中营针对的是种族群体。但是也……
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     Andrea Pitzer
    </strong>
   </span>
   ：一般而言，它与监狱的区别在于，在监狱中的人应该首先经过审判。
   <strong class="markup--strong markup--p-strong">
    他们应该有机会提出某种辩护
   </strong>
   。而集中营通常是围绕该过程的最终结果。有些时候会集中进行 “审判”，例如，在肯尼亚的英国人，有时会在一个领域一次性判处300人，但这并不是真正的审判。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     主持人
    </strong>
   </span>
   ：还有不可避免的死亡人数 —— 即使在集中营的最早期表现中，甚至在有计划地杀死被拘留者之前。
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     Andrea Pitzer
    </strong>
   </span>
   ：尽管他们没有使用子弹、绞刑，而疾病、营养不良以及人们所处的恶劣条件，这些困境足以杀死许多人。其中大多数是妇女和儿童。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     主持人
    </strong>
   </span>
   ：Andrea Pitzer 所说的最后一点非常重要。
   <strong class="markup--strong markup--p-strong">
    政府不必主动杀人
   </strong>
   。仅仅在没有充足食物、水和卫生条件的情况下将人们围困在肮脏的地方，就足以导致大规模死亡。
  </p>
  <p class="graf graf--p">
   在所有我们要考察的营地中，这种死亡是一个普遍因素。
  </p>
  <p class="graf graf--p">
   我们将快速概述大多数人从未听说过的深刻历史，并继续下去。
  </p>
  <p class="graf graf--p">
   因此，第一阶段是：实际上第一个集中营的创建。
  </p>
  <p class="graf graf--p">
   历史上大概有数十个政府聚集种族群体并将其隔离的例子。例如美国原住民被隔离。但是，集中营一词的首个实际用法是古巴的西班牙人使用的。西班牙人称之为
   <em class="markup--em markup--p-em">
    reconcentracion
   </em>
   。
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     Andrea Pitzer
    </strong>
   </span>

  </p>
  <p class="graf graf--p">
   二战后，世界承诺 “绝不会再有种族灭绝的发生、绝不会再使用集中营，绝不会再发生大屠杀了”。
  </p>
  <p class="graf graf--p">
   但是 ……
  </p>
  <p class="graf graf--p">
   我们开始了 “
   <em class="markup--em markup--p-em">
    Making Contact
   </em>
   ” 的这一章，谈论的是现代集中营，例如缅甸的罗兴亚人集中营。所以发生了什么？
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     Andrea Pitzer
    </strong>
   </span>
   ：世界在1945年重新设置，并从半个世纪前的殖民地开始回归。这部分原因是您正确地掌握了冷战二元论，那就是俄罗斯与美国。
  </p>
  <p class="graf graf--p">
   因此，⚠️你要么是共产主义者，要么是反共产主义者。如果你是共产主义者，那么你就有以古拉格为模型的这些大型集中营系统；如果你是反共产主义者，那么你将从50年前的殖民地集中营中发展出反殖民集中营。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     主持人
    </strong>
   </span>
   ：这是第四阶段，也是我们今天经历的阶段。这些新的集中营令人担忧：
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     Andrea Pitzer
    </strong>
   </span>
   ：旧式集中营的延续是我们现在从隔离游击队、通过围捕一群人来隔离恐怖分子开始的。我认为值得了解。
  </p>
  <p class="graf graf--p">
   例如，⚠️关塔那摩（Guantanamo）起初是大规模拘留寻求庇护者的场所，而这种地方 —— 政府被赋予权力将人们拘禁在这里 —— 正是它在9/11之后成为酷刑场所的基础。
  </p>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    我们在那些针对维吾尔族人口的集中营中也再次体验到了这种以反恐为借口实施集中营的观念
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ⚠️中国人已经意识到，如果你说这些人是 “恐怖分子” 或声称要面对恐袭威胁，那么全世界也很少有国家会对集中营说不。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-12211 jetpack-lazy-image" data-lazy-sizes="(max-width: 790px) 100vw, 790px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.png?resize=790%2C1334&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.png?w=790&amp;ssl=1 790w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.png?resize=178%2C300&amp;ssl=1 178w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.png?resize=606%2C1024&amp;ssl=1 606w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.png?resize=768%2C1297&amp;ssl=1 768w" data-recalc-dims="1" height="1334" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.png?resize=790%2C1334&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="790"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-12211" data-recalc-dims="1" height="1334" sizes="(max-width: 790px) 100vw, 790px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.png?resize=790%2C1334&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.png?w=790&amp;ssl=1 790w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.png?resize=178%2C300&amp;ssl=1 178w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.png?resize=606%2C1024&amp;ssl=1 606w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/2-11.png?resize=768%2C1297&amp;ssl=1 768w" width="790"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   结果就是，超过一百万维族穆斯林和其他少数民族人口被关押在这样的地方。因此，我认为，
   <strong class="markup--strong markup--p-strong">
    这种恐怖威胁仍然是一种可以让当权者构建野蛮残酷的折磨的集中营、并获得世界许可的方式
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%ad%a6%e6%83%95%e5%be%ae%e5%a6%99%e7%9a%84%e5%8a%9d%e5%af%bc%e6%9c%af%ef%bc%9a%e6%81%90%e6%83%a7%e8%af%89%e6%b1%82-%e5%9f%8b%e4%b8%8b%e6%83%85%e7%bb%aa-%e6%93%8d%e6%8e%a7%e6%80%9d/" href="https://www.iyouport.org/%e8%ad%a6%e6%83%95%e5%be%ae%e5%a6%99%e7%9a%84%e5%8a%9d%e5%af%bc%e6%9c%af%ef%bc%9a%e6%81%90%e6%83%a7%e8%af%89%e6%b1%82-%e5%9f%8b%e4%b8%8b%e6%83%85%e7%bb%aa-%e6%93%8d%e6%8e%a7%e6%80%9d/" rel="noopener noreferrer" target="_blank">
     警惕微妙的劝导术：恐惧诉求 — — 埋下情绪 操控思想
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    （中国维族集中营幸存者）Mihrigul Tursun
   </strong>
   ：我被带到一个没有窗户的地下牢房。四面墙上都有摄像头，因此官员们可以看到房间的每个角落。我被关押的其中一个牢房里大约有60个人。
  </p>
  <p class="graf graf--p">
   到了晚上，要有15位妇女站着，才能让我们其余的人侧身睡觉，然后我们每两个小时轮换一次。有些人一年没有洗过澡。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     主持人
    </strong>
   </span>
   ：您正在聆听的是 Mihrigul Tursun 的证词，她在2018年11月国会面前谈到了她在维吾尔族 “再教育营” 中的经历。她的证词是通过口译员朗读的。
  </p>
  <p class="graf graf--p">
   <span style="color: #ff9900;">
    <strong class="markup--strong markup--p-strong">
     Mihrigul Tursun
    </strong>
   </span>
   ：我第二次被拘留时，也曾在老虎凳上遭受过酷刑。我被带到一个特别的房间，被放在高脚椅上。他们将我的手臂和腿固定到位，并按下按钮以收紧。卫兵把头盔戴在我被剃光的头上。每次触电时，我的全身都会剧烈晃动，我会感觉到静脉的疼痛。
  </p>
  <p class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     主持人
    </strong>
   </span>
   ：但是 Andrea Pritzer 并不感到绝望。她的希望使我感到惊讶，我问她 —— 重温所有这段历史并重复看一遍是什么感觉 —— 针对持不同政见者和少数族裔的相同模式、相同的仇恨、相同的酷刑和虐待？
  </p>
  <p class="graf graf--p">
   她回答说，事实上，⚠️这些宏伟的设计 —— 大规模监禁、包围整个社区、强迫失踪 —— 它们需要我们的同谋。（
   <strong class="markup--strong markup--p-strong">
    如上所述，同谋指的是，人们接受了这种遭遇，没有反抗
   </strong>
   ）
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     Andrea Pitzer
    </strong>
   </span>
   ：⚠️政府必须 *积极培训* 人们接受这一点。因此，必须训练警卫人员、并使其坚定服从、让他们愿意对人们做这种事。
  </p>
  <p class="graf graf--p">
   ⚠️而且还必须训练被关押者 “愿意” 以这种方式被关押。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    坏消息是，人们真的很容易受到那种宣传的影响和折磨，这种宣传使政府和政党促使我们相信这样做是必要的
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%91%86%e8%84%b1%e6%93%8d%e7%ba%b5%e5%92%8c%e5%ae%a3%e4%bc%a0%ef%bc%9a28%e4%b8%aa%e9%95%bf%e6%9c%9f%e6%8a%80%e5%b7%a7/" href="https://www.iyouport.org/%e6%91%86%e8%84%b1%e6%93%8d%e7%ba%b5%e5%92%8c%e5%ae%a3%e4%bc%a0%ef%bc%9a28%e4%b8%aa%e9%95%bf%e6%9c%9f%e6%8a%80%e5%b7%a7/" rel="noopener noreferrer" target="_blank">
     摆脱操纵和宣传：28个长期技巧
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%88%90%e4%b8%ba%e4%bd%a0%e6%83%b3%e8%a6%81%e5%9c%a8%e8%bf%99%e4%b8%aa%e4%b8%96%e7%95%8c%e4%b8%8a%e7%9c%8b%e5%88%b0%e7%9a%84%e6%94%b9%e5%8f%98%ef%bc%9a%e4%b8%8eiyp%e4%b8%8b%e5%8d%88%e8%8c%b6/" href="https://www.iyouport.org/%e6%88%90%e4%b8%ba%e4%bd%a0%e6%83%b3%e8%a6%81%e5%9c%a8%e8%bf%99%e4%b8%aa%e4%b8%96%e7%95%8c%e4%b8%8a%e7%9c%8b%e5%88%b0%e7%9a%84%e6%94%b9%e5%8f%98%ef%bc%9a%e4%b8%8eiyp%e4%b8%8b%e5%8d%88%e8%8c%b6/" rel="noopener noreferrer" target="_blank">
     “你是在有史以来最重要的战斗中战斗”
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   因此，
   <strong class="markup--strong markup--p-strong">
    我认为，这真的是我们必须集中精力的地方 —— 如果我们想阻止这种情况发生 —— 必须从阻止政治宣传方面努力，这些宣传使人们以为其他人是动物、或者是肮脏的，接受他们 “应该遭受处罚” 的结论。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️我从支持今天一些集中营设置的人们那里听到了他们的想法。例如，当我在缅甸的时候，我确切地听到了这些想法。这些想法正是我们所有人从美国总统嘴里所听到的说法，但是，这种说法已经非常古老，这正是那些集中营赖以生存的理论根基。
  </p>
  <p class="graf graf--p">
   我认为集中营真的很难消除，
   <strong class="markup--strong markup--p-strong">
    任何政治体制都很容易激发出集中营的想法。所以仅仅因为你是一个民主国家并不意味着你就不会这样做
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，如果你是一个民主国家，那么理论上你应该有更多的手段可以撤销它。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    最后
   </strong>
  </h3>
  <p class="graf graf--p">
   主持人：您正在听的是《漫漫长夜：全球集中营历史》的作者 Andrea Pitzer 的访谈。
  </p>
  <p class="graf graf--p">
   在今天结束节目之前，我们想要感谢有机会使用 Abdul Mozid 的歌曲，这首歌是在孟加拉国的罗兴亚难民营中录制的。
  </p>
  <p class="graf graf--p">
   我们很乐意收到您的来信。您对边境拘留营有何看法，
   <strong class="markup--strong markup--p-strong">
    我们如何才能关闭当下位于全世界的集中营？
   </strong>
   ⚪️
  </p>
