      </strong>

  <p class="graf graf--p">
   首先回顾我们曾经推荐过的一套视频，关于算法专制的噩梦《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e6%ad%a3%e5%9c%a8%e5%a6%82%e4%bd%95%e8%a2%ab%e6%93%8d%e7%ba%b5%ef%bc%9a%e8%bf%99%e6%98%af%e6%af%8f%e4%b8%aa%e4%ba%ba%e9%83%bd%e9%9a%be%e4%bb%a5%e8%ba%b2%e9%81%bf%e7%9a%84%e6%97%a0%e5%bd%a2/" href="https://www.iyouport.org/%e4%bd%a0%e6%ad%a3%e5%9c%a8%e5%a6%82%e4%bd%95%e8%a2%ab%e6%93%8d%e7%ba%b5%ef%bc%9a%e8%bf%99%e6%98%af%e6%af%8f%e4%b8%aa%e4%ba%ba%e9%83%bd%e9%9a%be%e4%bb%a5%e8%ba%b2%e9%81%bf%e7%9a%84%e6%97%a0%e5%bd%a2/" rel="noopener noreferrer" target="_blank">
    你正在如何被操纵：这是每个人都难以躲避的无形专制
   </a>
   》。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.orfonline.org/cyfy-africa/" href="https://www.orfonline.org/cyfy-africa/" rel="noopener noreferrer" target="_blank">
    CyFyAfrica 2019
   </a>
   — 技术、革新和社会大会 — 于6月在摩洛哥丹吉尔举行。
  </p>
  <p class="graf graf--p">
   今年的会议是第二届。这是一个充满活力的多元化的聚会，由来自65个国家（主要是非洲和亚洲国家）的各种政策制定者、联合国代表、部长、政府代表、外交官、媒体、科技公司代表和学者参加。
  </p>
  <p class="graf graf--p">
   会议的中心目标是在全球语话中提高非洲大陆的声音。
  </p>
  <p class="graf graf--p">
   Observer Research Foundation 主席（该会议的共同主持人之一）在开幕致辞中强调，随着非洲大陆越来越依赖高端技术以解决其社会、教育、健康、经济和金融问题，非洲年轻人的声音需要被放在首位。
  </p>
  <p class="graf graf--p">
   该会议的目的部分是为这些年轻人提供一个平台，让他们与来自各大学的许多西方学者和工商业部门的技术开发人员一起获得提高的机会。
  </p>
  <blockquote class="graf graf--blockquote">
   <p>
    技术倡导者通常会提供各种理由，不惜一切代价地试图将生活的各个方面数字化……
   </p>
  </blockquote>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    相当于硅谷的科技创业公司如今可以在非洲大陆的任何角落找
   </strong>
   到 —— 亚的斯亚贝巴的“Sheba Valley”，拉各斯的“Yabacon Valley”、以及内罗毕的 “Silicon Savannah”，这是仅举几例 —— 并且非洲正在追求“银行、金融、健康护理和教育等各个领域的前沿创新。
  </p>
  <p class="graf graf--p">
   非洲大陆确实可以从各种数字技术和人工智能发展中受益：例如，埃塞俄比亚农民可以使用众包数据分析来预测和生产更好的作物。
  </p>
  <p class="graf graf--p">
   正如
   <a class="markup--anchor markup--p-anchor" data-href="https://afro.who.int/news/who-and-itu-use-digital-technology-strengthen-public-health-services-africa" href="https://afro.who.int/news/who-and-itu-use-digital-technology-strengthen-public-health-services-africa" rel="noopener noreferrer" target="_blank">
    世界卫生组织公告所讲述
   </a>
   的那样，大数据有助于改善教育和医疗保健领域的服务。数据可以帮助弥合困扰每个社会、政治和经济领域的巨大不平等…
  </p>
  <p class="graf graf--p">
   但是，所有这些都不是我想在这里讨论的内容。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-9050 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-1.png?resize=1100%2C617&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-1.png?w=1198&amp;ssl=1 1198w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-1.png?resize=300%2C168&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-1.png?resize=768%2C431&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-1.png?resize=1024%2C574&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-1.png?resize=1100%2C617&amp;ssl=1 1100w" data-recalc-dims="1" height="617" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-1.png?resize=1100%2C617&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9050" data-recalc-dims="1" height="617" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-1.png?resize=1100%2C617&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-1.png?w=1198&amp;ssl=1 1198w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-1.png?resize=300%2C168&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-1.png?resize=768%2C431&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-1.png?resize=1024%2C574&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/2-1.png?resize=1100%2C617&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   在非洲大陆内外已经有
   <strong class="markup--strong markup--p-strong">
    无数的顽固技术崇拜者，他们非常乐意盲目地采用任何所谓的“AI”、“智能”或“数据驱动”，而不愿重新思考可能出现的意外后果
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这很像你目前在中国看到的情况。
  </p>
  <p class="graf graf--p">
   无论技术创新的主题出现在哪里，你通常都会发现这种情况，技术倡导者提供各种理由，试图将生活的各个方面都变成数字化，不惜一切代价。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他们在“创新”的旗帜下盲目地接受道德上完全可疑的和危险的做法和应用、“破坏性的实践”和“改变游戏规则”，几乎不会受到批评或怀疑
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在整个非洲大陆发生的一些最大规模的人工智能或机器学习会议的演讲者阵容和讨论主题中，
   <strong class="markup--strong markup--p-strong">
    几乎没有任何人
   </strong>
   提到道德考虑、数据隐私、或人工智能的意外后果。
  </p>
  <p class="graf graf--p">
   再一次，和中国一样。
  </p>
  <p class="graf graf--p">
   鉴于已经有足够多的科技崇拜者掌握着非洲大陆的技术未来，重要的是要指出需要采取的警告以及需要从世界其他地方吸取的教训。
  </p>
  <p class="graf graf--p">
   非洲不需要也不应该亲自经历灾难性的警示故事才能发现数字化的黑暗面和生活全面技术化所带来的危机。
  </p>
  <p class="graf graf--p">
   大数据和AI似乎为复杂的社会问题提供了快速的解决方案。但是，这正是问题出现的地方。
  </p>
  <p class="graf graf--p">
   在全球范围内，人工智能技术正在逐步融入保险、移动银行、医疗保健和教育服务等各种领域的决策过程中。
  </p>
  <p class="graf graf--p">
   来自非洲大陆各地的各种初创公司正在兴起 —— 例如在尼日利亚的
   <a class="markup--anchor markup--p-anchor" data-href="https://disrupt-africa.com/2016/07/printivo-is-nigerias-most-investable-startup/" href="https://disrupt-africa.com/2016/07/printivo-is-nigerias-most-investable-startup/" rel="noopener noreferrer" target="_blank">
    Printivo
   </a>
   、在肯尼亚的
   <a class="markup--anchor markup--p-anchor" data-href="https://mydawa.com/home#/home" href="https://mydawa.com/home#/home" rel="noopener noreferrer" target="_blank">
    Mydawa
   </a>
   等等 —— 它们旨在开发下一个“前沿”应用程序、工具或系统。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    与所有
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e6%8e%a7%e5%88%b6%e7%9a%84%e4%b8%96%e7%95%8c%e5%92%8c%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89-2/" href="https://www.iyouport.org/%e8%b0%b7%e6%ad%8c%e6%8e%a7%e5%88%b6%e7%9a%84%e4%b8%96%e7%95%8c%e5%92%8c%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89-2/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     监视资本主义
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    一样，他们收集尽可能多的数据来分析、推断、预测 “用户”的各种行为和习惯
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，在建立最新的招聘应用程序或最先进的移动银行系统的竞赛中，创业公司和大公司
   <strong class="markup--strong markup--p-strong">
    均忽视了
   </strong>
   每个数据点背后的人类/人权。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “数据”被视为可以争夺的东西，一种无可争议地属于科技公司和政府的东西，完全抹去了创造数据的个人。
  </p>
  <p class="graf graf--p">
   这使得行为操纵和诱导变得非常容易，一切都是为了公司的盈利，以牺牲个人权利为代价。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    从诱导到操纵，一切都是开发自动化系统的私营部门工程师才有权决定什么是“正确”。
   </strong>
  </p>
  <p class="graf graf--p">
   在整个过程中，每一种被认为是“老土陈旧”的人和行为都会被迅速排斥，并加深边缘化。每当异议试图进入讨论，都会被推到一边。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-9051 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-4.png?resize=1100%2C735&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-4.png?w=1300&amp;ssl=1 1300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-4.png?resize=300%2C201&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-4.png?resize=768%2C513&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-4.png?resize=1024%2C685&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-4.png?resize=1100%2C735&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-4.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="735" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-4.png?resize=1100%2C735&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9051" data-recalc-dims="1" height="735" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-4.png?resize=1100%2C735&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-4.png?w=1300&amp;ssl=1 1300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-4.png?resize=300%2C201&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-4.png?resize=768%2C513&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-4.png?resize=1024%2C685&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-4.png?resize=1100%2C735&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/09/3-4.png?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   与大数据一样，人工智能只是一个不经意间被抛出的流行语；众所周知的只是跨不同学科的持久争论，这个词根本无法最终定义。
  </p>
  <p class="graf graf--p">
   它可以指从高度夸张的欺骗性机器人到 Facebook 和谷歌的内容 feeds 算法的所有一切，是这些算法在决定你能在互联网上看到什么样的内容。
  </p>
  <p class="graf graf--p">
   所有混迹该领域的研究人员和媒体记者都有助于过度夸大和吹嘘所谓的人工智能的能力，往往将其归功于神一般的力量。
  </p>
  <p class="graf graf--p">
   但是，正如 Melanie Mitchell 等领先的AI学者所强调的那样，我们离“超级智能”还有很远。
  </p>
  <p class="graf graf--p">
   同样，Jeff Bigham 指出，在许多被广泛讨论的“自动”系统中 —— 无论是机器人还是语音识别算法 ——
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%9b%86%e4%bd%93%e7%9b%91%e5%90%ac/" href="https://www.iyouport.org/%e9%9b%86%e4%bd%93%e7%9b%91%e5%90%ac/" rel="noopener noreferrer" target="_blank">
    大部分的工作都是由人类来完成的
   </a>
   ，那些通常只有微薄薪水的工人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    对所谓的人工智能系统功能的夸大，将人们的注意力从这些技术构成的真正危险中转移开来，这些危险比“杀手机器人”更加隐蔽、细致、和渐进
   </strong>
   。
  </p>
  <p class="graf graf--p">
   人们将算法默认为是更加客观的，毕竟那些规则“适用于每一个人”。但这是关于自动化系统的最错误和最有害的误解之一。
  </p>
  <p class="graf graf--p">
   正如哈佛数学家 Cathy O’Neil 在《Weapons of Math Destruction》中所解释的那样，“算法是嵌入代码中的意见。”
  </p>
  <p class="graf graf--p">
   在“AI”和“数据驱动”的幌子下，系统被假装为政治中立，但由于数据被解释、收集和用于产生某些结果的方式本身具有固有的政治倾向，其结果只是加深了人类固有的偏见。
  </p>
  <p class="graf graf--p">
   在分析数据时，这些系统改变了社会结构，强化了社会刻板印象，并进一步使那些已经处于社会等级底层的人群处于劣势。
  </p>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote">
   <p>
    “数据”被视为可以争夺的东西
   </p>
  </blockquote>
  <p class="graf graf--p">
   人工智能是人类创造、控制和负责的工具，与任何其他工具一样，它嵌入并反映了人类的不一致、局限、偏见以及政治和情感的欲望。
  </p>
  <p class="graf graf--p">
   人类对世界的看法和选择就构建在算法模型中。
  </p>
  <p class="graf graf--p">
   因此，由谁来建立系统并选择所使用的数据和来源，将深刻影响技术可能产生的效果。
  </p>
  <p class="graf graf--p">
   人工智能系统产生的结果反映了社会和文化上的刻板印象，而非客观事实。
  </p>
  <p class="graf graf--p">
   社会领域中的技术使用经常是有意或无意地集中在惩罚性实践上，无论是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a4%be%e4%bc%9a%e4%bf%a1%e7%94%a8%e7%9b%91%e8%a7%86%ef%bc%9a%e8%8b%b1%e5%9b%bd%e3%80%81%e4%b8%ad%e5%9b%bd%e3%80%81%e7%be%8e%e5%9b%bd%e5%92%8c%e6%97%a5%e6%9c%ac%e3%80%90viedo%e3%80%91/" href="https://www.iyouport.org/%e7%a4%be%e4%bc%9a%e4%bf%a1%e7%94%a8%e7%9b%91%e8%a7%86%ef%bc%9a%e8%8b%b1%e5%9b%bd%e3%80%81%e4%b8%ad%e5%9b%bd%e3%80%81%e7%be%8e%e5%9b%bd%e5%92%8c%e6%97%a5%e6%9c%ac%e3%80%90viedo%e3%80%91/" rel="noopener noreferrer" target="_blank">
    预测谁将犯罪
   </a>
   ，还是谁不能支付抵押贷款。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    宣称为了 实现“安全”和“秩序”而建立和应用的技术发展，往往旨在惩罚而不是恢复
   </strong>
   。
  </p>
  <p class="graf graf--p">
   此外，这些技术必然会给某些人带来残忍、歧视和不人道的做法。
  </p>
  <p class="graf graf--p">
   在这方面，中国维吾尔人遭受的残酷待遇和全世界穷人所遭受的不公平待遇就是例证。
  </p>
  <p class="graf graf--p">
   同样，随着约翰内斯堡和坎帕拉等城市引入面部识别技术的使用，对边缘群体的歧视排斥和过度监管是不可避免的。
  </p>
  <p class="graf graf--p">
   无论是否明确承认，商业公司开发人工智能的核心目标不是一般地纠正偏见，而是要推断个体“用户”的弱点和不足，好像人们只是作为被操纵的对象而存在。
  </p>
  <p class="graf graf--p">
   这些公司理所当然地认为，他们能搜罗到的所有数据都应该“属于”他们。
  </p>
  <p class="graf graf--p">
   围绕“数据挖掘”和“数据丰富的大陆”（非洲科技领域的共同语言）的讨论，显示了每个数据点背后的个体在多大程度上从这些寡头公司的角度来看仍然是无关紧要的。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e9%9d%9e%e6%b4%b2%e4%ba%ba%e7%9a%84%e8%84%b8%e8%a2%ab%e7%94%a8%e6%9d%a5%e8%ae%ad%e7%bb%83%e4%b8%ad%e5%9b%bd%e8%80%81%e5%a4%a7%e5%93%a5%ef%bc%9a%e5%8c%97%e4%ba%ac%e6%ad%a3%e5%9c%a8%e5%8a%aa%e5%8a%9b/" href="https://www.iyouport.org/%e9%9d%9e%e6%b4%b2%e4%ba%ba%e7%9a%84%e8%84%b8%e8%a2%ab%e7%94%a8%e6%9d%a5%e8%ae%ad%e7%bb%83%e4%b8%ad%e5%9b%bd%e8%80%81%e5%a4%a7%e5%93%a5%ef%bc%9a%e5%8c%97%e4%ba%ac%e6%ad%a3%e5%9c%a8%e5%8a%aa%e5%8a%9b/" rel="noopener noreferrer" target="_blank">
     非洲人的脸被用来训练中国老大哥：北京正在努力将其监视技术输出到发展中国家
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这是殖民者的态度，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%95%b0%e5%ad%97%e6%ae%96%e6%b0%91%e4%b8%bb%e4%b9%89%e5%a8%81%e8%83%81%e5%85%a8%e7%90%83%e5%8d%97%e6%96%b9%e5%9b%bd%e5%ae%b6/" href="https://www.iyouport.org/%e6%95%b0%e5%ad%97%e6%ae%96%e6%b0%91%e4%b8%bb%e4%b9%89%e5%a8%81%e8%83%81%e5%85%a8%e7%90%83%e5%8d%97%e6%96%b9%e5%9b%bd%e5%ae%b6/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     数字殖民
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，即 宣称人类是可以被自由采集的原材料
   </strong>
   。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-9052 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-1.png?resize=1100%2C586&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-1.png?w=1400&amp;ssl=1 1400w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-1.png?resize=300%2C160&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-1.png?resize=768%2C409&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-1.png?resize=1024%2C546&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-1.png?resize=1100%2C586&amp;ssl=1 1100w" data-recalc-dims="1" height="586" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-1.png?resize=1100%2C586&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-9052" data-recalc-dims="1" height="586" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-1.png?resize=1100%2C586&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-1.png?w=1400&amp;ssl=1 1400w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-1.png?resize=300%2C160&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-1.png?resize=768%2C409&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-1.png?resize=1024%2C546&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/09/4-1.png?resize=1100%2C586&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   当我们将关于社会问题的决策交给由利润驱动的寡头公司开发的自动化系统时，这意味着不仅允许社会关注的重要问题全部由这些公司的激励（利润）决定，而且，也将复杂的道德问题交给了寡头体制。
  </p>
  <p class="graf graf--p">
   与民主背道而驰 —— 任何一个寡头公司的负责人和算法的构建者都不是民选的，不会也没有没有义务为人民的权益负责。
  </p>
  <p class="graf graf--p">
   人类不再有自主权，仅仅被视为等待被采集的资源。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
     心理操纵的秘密：”行为经济学”+PR=？
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" rel="noopener noreferrer" target="_blank">
     看不见的独裁
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   行为算法模型的推论并不反映世界的中立状态没有提供任何深入的因果解释，而是重新描述已经强烈持有的社会性和历史性的不公正。
  </p>
  <p class="graf graf--p">
   技术总体上不是中立的或客观的；它是反映社会偏见、不公平和不公正的镜子。
  </p>
  <p class="graf graf--p">
   例如，在会议期间，一名联合国代表处理了正在制定的打击在线恐怖主义的工作，但令人失望地明确仅仅关注伊斯兰群体，描绘了一种不切实际和有害的在线恐怖主义形象：在过去九年中，欧洲、澳大利亚和北美洲
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/interactive/2019/04/03/world/white-extremist-terrorism-christchurch.html" href="https://www.nytimes.com/interactive/2019/04/03/world/white-extremist-terrorism-christchurch.html" rel="noopener noreferrer" target="_blank">
    发生了350起白人主导的极端恐怖主义袭击事件
   </a>
   ；在美国这个时期，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.scribd.com/document/356288299/White-Supremacist-Extremism-JIB" href="https://www.scribd.com/document/356288299/White-Supremacist-Extremism-JIB" rel="noopener noreferrer" target="_blank">
    所有此类攻击中
   </a>
   有三分之一是由于白人极端主义。
  </p>
  <p class="graf graf--p">
   这说明了一个令人担忧的现状，即
   <strong class="markup--strong markup--p-strong">
    刻板的观点驱动了所谓的问题以及我们为应对问题而开发的技术类型。进而通过一系列技术将我们所发现的东西作为偏见直觉的证据，并进一步强化刻板印象
   </strong>
   。
  </p>
  <p class="graf graf--p">
   社会最脆弱的群体正遭受各种数字化技术的不成比例的影响。然而，许多适用于人工智能的道德原则是坚定的功利主义。
  </p>
  <p class="graf graf--p">
   他们宣称所关心的是“对于最多人来说最大的幸福”，显然这意味着从不寻求以少数群体为中心的解决方案。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e9%9a%90%e7%a7%81%e7%a9%b6%e7%ab%9f%e6%98%af%e7%bb%99%e8%b0%81%e7%9a%84%ef%bc%9f/" href="https://www.iyouport.org/%e9%9a%90%e7%a7%81%e7%a9%b6%e7%ab%9f%e6%98%af%e7%bb%99%e8%b0%81%e7%9a%84%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     隐私究竟是给谁的？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   即使在算法决策过程中的不公平和歧视被提到了前台，“解决方案”中也几乎找不到边缘人士的利益：企业和学术道德委员会制定的缓解建议通常是在没有受影响人员的协商和参与的情况下制定的。
  </p>
  <p class="graf graf--p">
   但是，边缘人士的声音需要在每一步都被优先考虑，包括设计、开发和实施任何技术，以及政策制定。
  </p>
  <p class="graf graf--p">
   随着非洲努力追赶最新的技术发展，他们还必须保护非洲大陆最脆弱人群免受技术可能造成的伤害性后果。
  </p>
  <p class="graf graf--p">
   如果真的保护和尊重 CyFyAfrica 会议领导人想要重视的年轻人的权利，那么自由和隐私就应该被优先考虑。
  </p>
  <p class="graf graf--p">
   只有通过考虑当地价值观、背景和生活方式，以及通过将批判性声音纳入技术叙事的重要部分，才能实现个人权利和自由的指导和保障。
  </p>
  <p class="graf graf--p">
   公平地说，CyFyAfrica 确实允许某些小组中的一些批评声音。然而，更大范围的批评声音被埋没在压倒性的科技爱好者声音之下。
  </p>
  <p class="graf graf--p">
   会议结束后，发言人有机会在观察研究基金会运作的半学术期刊上分享他们的观点和研究成果。
  </p>
  <p class="graf graf--p">
   具有讽刺意味的是，
   <strong class="markup--strong markup--p-strong">
    声称为非洲青年发声的平台只有在其声音与强大的科技寡头公司、政策制定者和政府的叙述和动机保持一致时，才能实现
   </strong>
   。
  </p>
  <p class="graf graf--p">
   流行语就像一场流行病，目前已经席卷全球。在利益的驱动下所有人都变得疯狂。
  </p>
  <p class="graf graf--p">
   制止技术发展是不可能的，大数据和所谓的AI必然继续被更广泛的利用；如何才能使其民主化以为大众的利益服务，也许是一个长期且艰难的问题。⚪️
  </p>
