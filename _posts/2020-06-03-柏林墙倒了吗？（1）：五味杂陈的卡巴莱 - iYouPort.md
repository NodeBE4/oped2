  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     实验
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   一对双胞胎，在出生时被分开，一个送到富裕、宽容的家庭，而一个则由贫穷、严厉的家庭收养。
  </p>
  <p class="graf graf--p">
   45年后，富有的兄弟找到了贫穷的兄弟，邀请他去参观自己在郊外的别墅。
  </p>
  <p class="graf graf--p">
   一开始，他们为互相陪伴、为似乎重新找到失去的另一半而快乐欢欣。然而，过了个把小时，他们开始发现彼此的陪伴令人烦恼。
  </p>
  <p class="graf graf--p">
   穷兄弟被富兄弟的喧闹和咄咄逼人的笑声所折磨，富兄弟则为穷兄弟的顺从和充满怨恨的沉默而恼怒。
  </p>
  <p class="graf graf--p">
   对于父母为什么抛弃他们，当富兄弟诉说他那个版本的故事时，被告知另一个不同版本故事的穷兄弟生气地予以否认。
  </p>
  <p class="graf graf--p">
   很快，他们沉默无言地坐着，都在想，如果他们从未遇到，也许结果会更好些。除了碰巧拥有一对粗心大意的父母之外，他们俩似乎毫无共同之处。
  </p>
  <p class="graf graf--p">
   然后，富兄弟的妻子出现在这个郊外别墅的草坪上，给她贫穷的小叔子带来了饮料。穷兄弟看到嫂子和他自己的妻子一样有着红色的头发，有着同样对高跟鞋的偏好 —— 可以展示涂得红亮的脚趾甲。
  </p>
  <p class="graf graf--p">
   随后他还注意到，虽然他自己的院子比这个别墅花园要小得多，但布景几乎完全一样；更让人迷惑的是，他注意到他的富兄弟躺在一把草地椅上喝啤酒，稀疏的沙黄色头发分开的方式和自己一模一样，从左耳掠到右边。
  </p>
  <p class="graf graf--p">
   最后，当富兄弟的儿子出来借车钥匙时，穷兄弟再次发现，自己和富兄弟为各自的儿子起了同样的名字……
  </p>
  <p class="graf graf--p">
   ****
  </p>
  <p class="graf graf--p">
   设想一个民族，把它分成两个相隔的国家；确保这两个国家实施对立的哲学和社会组织形式。努力确保这个民族的居民被一再告知：这个实验将是永久性的。
  </p>
  <p class="graf graf--p">
   在两个国家之间竖立一道墙，尽可能防止他们之间有任何交流。45年后，移除这道墙，告诉人们实验结束，因而他们再次成为一个单一民族。
   <strong class="markup--strong markup--p-strong">
    他们还会是一个民族吗？
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    设计这样一个实验是想回答一个问题：是民族塑造国家，还是国家塑造民族？
   </strong>
  </p>
  <p class="graf graf--p">
   这个问题类似于环境和遗传在个人塑造中的相对重要性。
  </p>
  <p class="graf graf--p">
   应当有可能询问接受实验的人们 —— 就像问出生时被分开的双胞胎一样 —— 他们是否还能互相认得兄弟？他们应当能够告诉你答案。
  </p>
  <p class="graf graf--p">
   实验应该能得出结论。
  </p>

  <p class="graf graf--p">
   1989年11月，德国电影导演 Wim Wenders 正在澳大利亚荒漠中一个叫火鸡溪（Turkey Creek）的地方，消息传来，说柏林墙倒了。
  </p>
  <p class="graf graf--p">
   一开始他不相信这个消息，随之感觉很焦虑。是坦克推到的吗？他觉得一切都是不真实的。必须得到一些这事的确发生了的证据。做为一名导演，他必须看到照片。
  </p>
  <p class="graf graf--p">
   火鸡溪的飞行医生正好有一台传真机，Wenders 让柏林的一个朋友把所有能够剪下来的照片都发给他了。
  </p>
  <p class="graf graf--p">
   很快，照片开始在传真机上滚印出来，卷曲在导演的手上。在如此遥远的澳大利亚腹地，这些照片显得更加有意义。它们使他想起了自己一个月前刚去世的父亲错失了这样一个似乎让德国两代人和解的时刻。
  </p>
  <p class="graf graf--p">
   电影导演记得，当时他落泪了，长久以来从未有过的。他记得自己当时想：“出错的世界修补好了，出错的我的内心修补好了”。
  </p>
  <p class="graf graf--p">

  <blockquote class="graf graf--blockquote">
   <p>
    所有这些站在墙头的让人落泪的面容，它们如此令人难忘。它们似乎比我们的脸更为真实，它们看上去就像40年代、50年代电影里的脸庞。为什么更为真实？我真的不知道。苦难让人们变得真实。也许这就是为什么。这些人经历了我们未曾经历的，是这些让他们更为真实。 ——
    <strong class="markup--strong markup--blockquote-strong">
     Volker Schlöndorff 德国电影导演, 1989
    </strong>
   </p>
  </blockquote>
  <blockquote class="graf graf--blockquote">
   <p>
    我的成长过程中曾为德国感觉羞耻。但这是一个我感觉到可能有些不一样的时刻。简单的事实是，在过去的一百年中，德国人为了自由和解放承担某种风险的时候并不多见。——
    <strong class="markup--strong markup--blockquote-strong">
     Peter Schneider 德国作家，1989
    </strong>
   </p>
  </blockquote>
  <p class="graf graf--p">
   在 “冷战” 的所有年月里。在其后所有充斥着犬儒和苦涩的年月里，
   <strong class="markup--strong markup--p-strong">
    只有一个毫无顾忌的快乐之夜
   </strong>
   ：柏林墙倒下的时候。人们穿梭来往，不敢相信这一切，怀疑自己是不是在做梦，却又发现不是做梦。
  </p>
  <p class="graf graf--p">
   在欧洲战后的整个历史上，只有这样一个纯粹快乐的夜晚。
   <strong class="markup--strong markup--p-strong">
    只有一个
   </strong>
   。
  </p>
  <p class="graf graf--p">
   自那之后，一切都未能如我们所愿，我们开始重写自己的情感历史。我们几乎忘却了那个夜晚。如果它能重新回到记忆中，我们问自己：为什么我们会用仅仅用一个夜晚的 *幻象* 来款待自己？为什么我们不论如何都要拥有一丝希望？我们多么愚蠢 ……
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     相册
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   1989年的德国革命以莱比锡为中心，这个城市有200万人口，位于德意志民主共和国（DDR）东南角的萨克森州。
  </p>
  <p class="graf graf--p">
   那里举行了规模最大的反旧政权游行。游行静默而有序，从晚上开始，每个人都结束工作之后。成千上万的人集合起来，占领了整条整条的街道，向卡尔马克思广场和歌剧院前进。
  </p>
  <p class="graf graf--p">
   一开始，在1989年9月末的时候，标语牌上只是要求 “媒体自由”、“签证自由”。也有些胆小的标牌上写着 “戈比帮助我们”。（戈比 Gorbi 是对当时苏共总书记戈尔巴乔夫的昵称，他在两德统一中起到了决定性的作用）
  </p>
  <p class="graf graf--p">
   9月末，当局用高压水龙头、大规模军警队伍、盾牌、警棍、警犬对付游行。游行者被抓起来，双手扭曲着反绑在身后，扔进人民警察（Volkspolizei）卡车的后备箱里。
  </p>
  <p class="graf graf--p">
   但游行继续。口号开始逐渐超出圣尼古拉教堂的路德派秘密反抗中形成的谨慎要求。
  </p>
  <p class="graf graf--p">
   到1989年10月中旬，新的横幅上写着 “我们要改革！” “我们就是人民！”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    分化人们、让每个人感觉到自己是孤立无援的，这就是当局维系自身生存的主要手段，而这些游行成为了集体之自我发现的一种演习。
   </strong>
  </p>
  <p class="graf graf--p">
   一夜又一夜，人群聚集起来，人们四处张望，看着从街道一边延伸到另一边有多少像他们一样的人，他们尽可能地向后和向前看。最后会有领导人准备冒点险：莱比锡歌剧院的导演、一些路德派的牧师、当地表演卡巴来的喜剧演员。
  </p>
  <p class="graf graf--p">
   其他城市效仿莱比锡的例子，市民们也走上了街头。旧政权哄骗、引诱、咆哮、威吓，但一切似乎都不再管用。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11916 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4.jpeg?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4.jpeg?w=1400&amp;ssl=1 1400w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4.jpeg?resize=1024%2C576&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4.jpeg?resize=768%2C432&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4.jpeg?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4.jpeg?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11916" data-recalc-dims="1" height="619" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4.jpeg?resize=1100%2C619&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4.jpeg?w=1400&amp;ssl=1 1400w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4.jpeg?resize=1024%2C576&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4.jpeg?resize=768%2C432&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/4.jpeg?resize=1100%2C619&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    服从和逆来顺受神秘地消失了
   </strong>
   。当局在电视上呼吁 “维持秩序” 之后，更多的标语牌出现在人群中，上面写着：“不要演讲！” “不要唧唧哇哇！” “不要粉饰太平！” “该动手术了！”
  </p>
  <p class="graf graf--p">
   10月下着雨，但人们继续在伞下游行。他们举行守夜，要求改革，把数千支蜡烛放置在教堂的墙上。蜡滴下来汇集在一起，融入人行道上半透明的池子里。
  </p>
  <p class="graf graf--p">
   同时，摄影师们注意到，警察脸上的表情出现了变化。他们在巡逻车前面摆出各种姿势拍照，警棍被塞进腰带里，而且随着时间一天天过去，越来越不清楚谁在负责。
  </p>
  <p class="graf graf--p">
   在莱比锡周围的学校。老师告诉学生们 “不要去参加游行”，
   <strong class="markup--strong markup--p-strong">
    当学生们不管不顾去参加的时候，他们发现，自己的老师与他们并肩游行
   </strong>
   。
  </p>
  <p class="graf graf--p">
   第二天早上，他们会在课堂上互相挤挤眼。
  </p>
  <p class="graf graf--p">
   到10月末，卡尔马克思广场上的标语牌写道：“德意志民主共和国属于人民，不属于党！”
  </p>
  <p class="graf graf--p">
   然后，当柏林墙在11月9日倒下时，示威者们极为惊讶地发现，他们带来了一场革命，而没有牺牲掉任何一条性命。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这场革命的象征非常恰如其分，是我们在保加利亚和罗马尼亚的人群中也见到过的怪异旗帜
   </strong>
   ：锤子和镰刀被剪掉的那个旗帜，就像人们在床单上剪了两个洞，给孩子做了一件鬼怪服。
  </p>
  <p class="graf graf--p">
   一个国旗上有洞的国家是一个不再知道自己是什么的国家。一群人挥舞着这个旗帜，他们真的希望仍然保留红色？其他人仅仅是不知道还有什么别的东西能让自己挥舞的。
  </p>
  <p class="graf graf--p">
   但是，也有另一些人开始挥舞另一个德国的旗帜 —— 柏林墙对面的旗帜。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11917 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5.jpeg?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5.jpeg?w=1400&amp;ssl=1 1400w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5.jpeg?resize=1024%2C576&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5.jpeg?resize=768%2C432&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5.jpeg?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5.jpeg?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

    <img alt="" class="aligncenter size-full wp-image-11917" data-recalc-dims="1" height="619" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5.jpeg?resize=1100%2C619&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5.jpeg?w=1400&amp;ssl=1 1400w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5.jpeg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5.jpeg?resize=1024%2C576&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5.jpeg?resize=768%2C432&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/01/5.jpeg?resize=1100%2C619&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   柏林墙倒掉一个月后，在卡尔马克思广场上出现了一个新的标语。首先是写在一个潦草的牌子上，然后是几十个牌子，再然后，是成千上万个牌子：
   <strong class="markup--strong markup--p-strong">
    “
    <em class="markup--em markup--p-em">
     Deutschland, einig Vaterland
    </em>
    ” （德意志，一个祖国）
   </strong>
   。在六个星期的时间里，“我们就是人民” 的呼喊变成了圣歌：“我们是一个民族！”
  </p>
  <p class="graf graf--p">
   德意志，一个祖国 …… 现在看，这个标语确实神秘。
   <strong class="markup--strong markup--p-strong">
    一个祖国。这到底意味着什么？它是一个事实的宣告吗？一种愿望的表达？还是一个令人困惑的难题？……
   </strong>
  </p>
  <p class="graf graf--p">
   在莱比锡老市政厅下面的拱廊，有一个商店，你可以在这里买到一个相册，它记录了从1989年9月到12月那段时间的几乎每一天。
  </p>
  <p class="graf graf--p">
   在相册中，最令人难忘的面庞是那些在人群中盯着摄像机看的人们。在每一张照片中，他们的脸都布满了恐惧和紧张。恐惧使他们的微笑半途凝结，让他们说的话停留在嘴边，令他们的反抗姿态折断双翼。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    恐惧不仅是担忧被捕，不仅是怀疑摄影师正在为警察工作。它是历史性的恐惧，是人们也许曾经在1848年的欧洲兵营中看到的那种恐惧，是一个民族将要抬脚步入未知并且不知道接下来会发生什么的恐惧
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果所有的革命都开始于迈入未知的那一步，你在这些面庞上看到的恐惧会让人感觉奇怪，他们如何敢于迈出第一步？
  </p>
  <p class="graf graf--p">
   然而，当你翻动相册，当游行进入第二周、第三周、第四周时，你可以看到
   <strong class="markup--strong markup--p-strong">
    那种恐惧开始慢慢消散；面庞开始放松，姿态开始轻盈，笑声变得洪亮
   </strong>
   。
  </p>
  <p class="graf graf--p">
   到了11月，人们不再担心自己将走向何方。现在他们开始相信历史正在与他们一起前进。成千上万的莱比锡人涌上大街，进入卡尔马克思广场，对着摄像机欢笑挥手。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11918 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-scaled.jpeg?resize=1100%2C791&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-scaled.jpeg?w=2048&amp;ssl=1 2048w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-scaled.jpeg?resize=300%2C216&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-scaled.jpeg?resize=1024%2C736&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-scaled.jpeg?resize=768%2C552&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-scaled.jpeg?resize=1536%2C1105&amp;ssl=1 1536w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-scaled.jpeg?resize=1100%2C791&amp;ssl=1 1100w" data-recalc-dims="1" height="791" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/01/6-scaled.jpeg?resize=1100%2C791&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     卡尔马克思广场
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   一辆电车把我放在了卡尔马克思广场。道路清扫车在排水沟上扫过。雨水抽打在空荡荡的喷泉里的青铜马雕像上。我周围只有空无一人的雨水扫过的人行道。
  </p>
  <p class="graf graf--p">
   在关于这个广场的旧纪录片中，这儿有更多的喷泉，有煤气灯，和一个巨大的带有科林斯式柱廊的歌剧院，柱廊下面可以停靠四轮马车。
  </p>
  <p class="graf graf--p">
   在纪录片中，戴呢帽、腋下夹着报纸的人们正是在我站的这个电车站上上下下，有一个小姑娘卖紫罗兰，有一个手风琴演奏师，一个大胡子的乞丐在向路人伸手乞讨，一个男人站在手推车后面，拿着报纸折成的装满葵花籽的圆筒。你几乎可以听到女人们的长裙拖在人行道上发出的沙沙声。那是1913年4月。
  </p>
  <p class="graf graf--p">
   莱比锡的书店出售一本书，是莱比锡人在元年 —— 1945年 —— 的照片。盟军已经轰炸了整个城市，数英亩的碎石堆积在卡尔马克思广场的每一个角落。
  </p>
  <p class="graf graf--p">
   照片上，清理广场残渣的 “碎石女人” 在工作间隙凝视着你，她们把砖块整齐地拍成一排排。她们的围裙是破旧的，头发上有厚厚的灰尘，穿着工装裤和工作靴。她们的耳朵后面夹着卷烟，粗糙的手握紧砖块和锤子。
  </p>
  <p class="graf graf--p">
   她们凝望着未来，仿佛可以比我们看得更清楚。也许就是在那个时刻 —— 在刚刚开始的零年，工人国家真的像是一个美好的梦想。
  </p>
  <p class="graf graf--p">
   而现在看来，似乎令人难以相信这儿曾一度有过梦想，每个人都曾一度相信过它。
  </p>
  <p class="graf graf--p">
   早在1953年，工人国家就派出坦克对付柏林的工人。即使在那个时候，这个国家实际上什么样，也是显而易见的。
  </p>
  <p class="graf graf--p">
   但碎石女人和退伍老兵想要相信点什么。即使他们的领导人不相信，他们仍坚持到底。
   <strong class="markup--strong markup--p-strong">
    因为，要轻易地怀疑你自己的整个人生都是在虚无中度过的，那也太过痛苦，太过荒谬了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   不相信的人和醒悟过来的人离开了这里，去往西方，他们的离去悄无声息。大部分留下的人不再有幻想，他们安慰自己，如果民主德国很糟糕，那么波兰会更糟糕，匈牙利会特别糟糕，俄国会无比糟糕 ……
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    政权的合法性依赖于负面比较带来的安慰。
   </strong>
  </p>
  <p class="graf graf--p">
   在60年代，民主德国当局重建了广场，用的是粗野的混凝土，非常符合他们的政治风格。它炸毁了广场一角有300年历史的巴洛克式教堂，在废墟上建造了一座30层高的钢铁摩天大楼。
  </p>
  <p class="graf graf--p">
   卡尔马克思广场仍然作为一个已消失的政权心脏地带的公共荒漠保存下来了。
   <strong class="markup--strong markup--p-strong">
    它是对民主德国加之于公共空间的恐惧和人类自发性的纪念碑
   </strong>
   。
  </p>
  <p class="graf graf--p">
   卖香肠的小贩、兜售小册子的人、艺术家、妓女、反叛少年 …… 如果给他们半点机会，他们就可以把它推翻。
  </p>

   <p>
    In time, the division of Germany came to seem eternal. Indeed, two generations grew up on either side of the Wall who actually feared what their nation might become if it were ever allowed to unite. The dream of a united Germany was not merely renounced; it was officially anathematized by the Ostpolitik pursued by both sides.
   </p>

  <p class="graf graf--p">
   时间的冰河作用缓慢地创造了两个民族、两个国家。当然，有柏林墙，你还是可以在电视屏幕上看到西德的照片。但是
   <strong class="markup--strong markup--p-strong">
    到80年代末，如果你没有离开，那么，你已经把你的孪生兄弟/姐妹的记忆从你的脑海中抠出去了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一种必要的巨型架构 —— 帝国式划分欧洲 —— 使这种遗忘相当容易。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    德国的分裂看上去将会永远持续
   </strong>
   。
  </p>
  <p class="graf graf--p">
   事实上，在墙的两边有两代人成长起来，他们真的担忧，如果被许可统一，他们的民族将会如何。
  </p>
  <p class="graf graf--p">
   一个统一德国的梦想不只是被抛弃了，它还受到双方所追求的东方政策（Ostpolitik）的正式谴责。（“东方政策” 是西德前总理维利勃兰特在60年代提出的对东欧保持正常关系的政策）
  </p>
  <p class="graf graf--p">
   所以，当帝国必需的伟大架构开始在他们头顶颤抖、摇晃，当莱比锡的人们占领街头，他们从未有一刻想到，他们将最终打倒一个国家，并带来一个民族的统一。他们从未想到，如果他们全部一起挤靠那扇被锁住的门，它也许会忽然打开，让他们滚入一个陌生的新世界 ……
  </p>

  <p class="graf graf--p">
   1989年之后，卡尔马克思广场恢复了1914年之前的旧名 —— 奥古斯都广场。
  </p>
  <p class="graf graf--p">
   梅赛德斯奔驰冷蓝色的霓虹灯光环绕漂浮在对面的保险大厦上。甚至有些方案想炸毁这个大楼，按照老照片和平面图纸，用石头一块块重砌已经消失的巴洛克式教堂。
  </p>
  <p class="graf graf--p">
   但某种不怀好意的幽灵继续使这些方案受到轻视。
  </p>
  <p class="graf graf--p">
   莱比锡的其他地方都被扔到了建筑商手里，而卡尔马克思广场，仍然顽固地保持不变。
  </p>
  <p class="graf graf--p">
   在改造这个荒漠的任务面前，历史记忆仿佛退缩了。
  </p>
  <p class="graf graf--p">
   在通往莱比锡大学建筑的入口处，一群热烈工人的浮雕中间，卡尔马克思乱蓬蓬犹如牛头一般的巨大青铜头像继续俯视着广场。某种程度上他仍是这个风掠之地的天才领袖。当游行者们用歌唱和横幅占领这里的时候，他收缩了自己的权威。
  </p>
  <p class="graf graf--p">
   现在，这里又变成了荒漠，好像他的灵魂又收回了这个地方。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     卡巴莱
    </strong>
   </span>
  </h3>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “它是伟大的，它是一场革命。有一个月的时间，革命是掌握在莱比锡人的手里。然后它结束了。
   <strong class="markup--strong markup--p-strong">
    而现在，制造革命的人们不拥有任何权力。但他们仍然在那儿
   </strong>
   ：商店售货员、图书馆员、教授。他们仍旧想要我所信仰的东西”。
  </p>
  <p class="graf graf--p">
   Böhnke 先生停顿了一下，看上去有点尴尬，厚实的手擦了擦前额，“谈到你信仰的东西，现在有点令人悲哀。但我说的是，一个为了人民的德意志”。
  </p>
  <p class="graf graf--p">
   从卡尔马克思广场出发，沿着一条狭窄的鹅卵石街道，在一个卡巴莱地下室的酒吧里，我找到了 Gunther Böhnke，他在酒吧后部的一个圆桌上喝啤酒。
  </p>
  <p class="graf graf--p">
   Böhnke 先生是 “学院派” 卡巴莱的明显，他跟我讲 “冷战” 时代东欧地区有板有眼、精确的英语。那种还没有受过本土接触影响的英语，那是完全从磁带里学的英语。
  </p>
  <p class="graf graf--p">
   白天，他在一家出版公司翻译儿童书籍；晚上，他是一位卡巴莱艺术家。
   <strong class="markup--strong markup--p-strong">
    过去，卡巴莱艺术家是威权国家持有执照的小丑
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在15米深的地下，在小小的舞台上，褪色的灰丝绒幕布之前，
   <strong class="markup--strong markup--p-strong">
    卡巴莱是窃窃私语的地方，是不可说的东西可以说出来的地方
   </strong>
   。
  </p>
  <p class="graf graf--p">
   过去，剧院的票会提前10年卖光，你没看错。一个卡巴莱艺术家可以把他自己分到的票当成钱用：有多少座位，从屠夫手里就可以换多少香肠。
  </p>
  <p class="graf graf--p">
   现在，票价太贵，当地人承受不起，座位都是被成群的西德人所占据。他们来到这里，为那些嘲笑他们自己的笑话发出不自然的笑声。
  </p>
  <p class="graf graf--p">
   Böhnke 先生赋予 “东德佬 Ossi” 以人性和讽刺性。他个头矮小，秃头，穿一件不合身的花昵夹克，系一条超大的东方式领带（East-bloc tie）。他的肚子凸起，将衬衫纽扣崩得紧紧的，
   <strong class="markup--strong markup--p-strong">
    忧郁的脸上交织着顺从和精明
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在舞台上，他的固定套路是关于一个贫穷、呆傻的东德人寻找工作，一个西德的人事主管面试他。东德佬随口说他从未入过党。认为这是对方想要的，却只听西德人回复：“你有什么毛病吗？你的积极性在哪里？”
  </p>
  <p class="graf graf--p">
   他所有笑话都像这样，辛酸地反映一种本不应该存在的分歧：
   <strong class="markup--strong markup--p-strong">
    一群人民，一种语言，一个民族，然而却经历了45年不同的国度之后，几乎无法认识彼此
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们都是好人”。莱比锡最古老餐馆的
   <strong class="markup--strong markup--p-strong">
    西德新店主
   </strong>
   在晚餐结束后、卡巴莱开始之前对我说。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “好人。只不过，他们不知道该怎么工作。我跟你发誓。我必须一切从头开始。开始教他们（东德人）如何准时上班，如何正确处理客人的账单，让他们的手指不要伸进汤里。
   <strong class="markup--strong markup--p-strong">
    我不是在做餐馆生意，我简直是一个社会工作者……”
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他谈到他的德国同胞带有的那种深情的傲慢，与英国殖民管理者在谈及 Tanganyikans 时惯用的那种语气，如出一辙
   </strong>
   。
  </p>

  </p>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote">
   <p>
    “Unification has not been the disquieting reunion of two lost twins on a suburban lawn but a colonial occupation. ”
   </p>
  </blockquote>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    统一不是失散的双胞胎在郊区草坪上揣揣不安的重逢，而是一种殖民占领
   </strong>
   。
  </p>
  <p class="graf graf--p">
   当你在莱比锡醒来，听到的声音是老建筑的内脏 —— 机床、石膏、钉子、窗框、木板 —— 被扔进长长的带回响的塑料斜槽里，滚进建筑商的废料桶的声音。
  </p>
  <p class="graf graf--p">
   建筑物的正面被保留下来 —— 必须能把贝纳通的牌子钉在上面 —— 但城市的内脏已经被去除了。
  </p>
  <p class="graf graf--p">
   Böhnke 先生认为，当一个社会体制崩溃时，我们只能预计到，那些主要是体制受害者的人们会为体制的失败而受到谴责。
   <strong class="markup--strong markup--p-strong">
    世界就是这样，事实上拖垮了政权的人们，现在被当作爱发牢骚的乞丐给打发走了 —— 是被那些曾经坐在电视机前为他们的公民勇气鼓掌的西德人打发走的。
   </strong>
  </p>
  <p class="graf graf--p">
   正如德国作家 Peter Schneider 所说，柏林墙是面镜子。西德人问道：镜子，墙上的镜子，谁是所有人里最公正的？镜子一成不变地回答：是你。
  </p>
  <p class="graf graf--p">
   45年的时间，一个民族分裂为两个国家，双方都必须塑造对方的负面形象。为什么只因墙倒了，这种行为就应该结束呢？为什么只因每个人现在生活在同一个国家，这种行为就应该结束呢 ……
  </p>
  <p class="graf graf--p">
   但是，在这个镜子的游戏里，有什么东西已经发生了改变。
  </p>
  <p class="graf graf--p">
   革命之前，在西德，负面形象是民主德国这个国家自身和其令人憎恶的制度。
   <strong class="markup--strong markup--p-strong">
    而现在，负面形象是民族，是人民本身；他们的抱怨和消极。
   </strong>
  </p>
  <p class="graf graf--p">
   现在那个国家已经消失，而人民本身 —— 那个民族 —— 为国家的曾经存在受到责备。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-11923 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?w=2048&amp;ssl=1 2048w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?resize=1024%2C683&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?resize=1536%2C1024&amp;ssl=1 1536w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?resize=1100%2C733&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-11923" data-recalc-dims="1" height="733" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?resize=1100%2C733&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?w=2048&amp;ssl=1 2048w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?resize=1024%2C683&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?resize=1536%2C1024&amp;ssl=1 1536w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?resize=1100%2C733&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/01/11-scaled.jpeg?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   说来奇怪，
   <strong class="markup--strong markup--p-strong">
    这种责备经常来自东德一
   </strong>
   方。前东德人表达自己对从前在西德混得好的兄弟姐妹的憎恶，并不少见。
  </p>
  <p class="graf graf--p">
   因此，前东德小说家 Monika Maron 说：“我最不喜欢我的前东德同胞们的一点是，他们认为这个世界亏欠他们，尤其是亏欠他们的尊严。他们似乎忘了，直到几年前，他们还没有真正摸到那种尊严”。
  </p>
  <p class="graf graf--p">
   东德精神病学家 Hans Joachim Maatz 写了一本书，告诉西德公众，45年的极权主义制造了一种东德的人格结构，特征为 “情感被压抑，无安全感，潜在攻击性”。他说：
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    在那个社会中，一个人的基本人权 —— 保持自己、有自己意见、作为个体被理解和接受的权利 —— 毫无保障 …… 在这个体制中能够安全生存的，只有那些调整和牺牲他们自己的自发活力、他们的诚实、他们的批判能力、顺从生活的人。
   </em>
  </p>
  <p class="graf graf--p">
   像某些陈腐的心理学形式一样，这只对某些人来说是正确的。那么，
   <strong class="markup--strong markup--p-strong">
    那些不安的、神经质的、顺从的个体 —— 像 Böhnke 先生那样的人 —— 是如何找到拖垮整个政权的勇气呢？
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    东德的负面形象当然迎合西德人的自恋情结，但它使统一的历史变得难以理解。
   </strong>
  </p>
  <p class="graf graf--p">
   Böhnke 先生对于1989年以来国家命运的悲喜剧淡然处之：“墙倒了之后，我们的人民去西德，他们带回来难以置信的故事。在那里，你可以在1月份就买到新鲜的莴苣和番茄，你能想象吗？
   <strong class="markup--strong markup--p-strong">
    现在这里也有超市了，我们也能做到了。只是我们买不起，而且房租已经涨了五倍”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他耸耸肩，笑了：“当我们10月在莱比锡的雨中游行时，我们想要的只是多一点民主，多一点尊重。没人想要统一”。
  </p>
  <p class="graf graf--p">
   统一了。
  </p>
  <p class="graf graf--p">
   也没人想要卡巴莱消亡，但现在，Böhnke 先生说，你可以在德国电视上看到一切。那儿的讽刺更为尖锐，时效更为迅速。现在谁还想看卡巴莱呢？
  </p>
  <p class="graf graf--p">
   Böhnke 先生曾经把他的演出带到西德，但效果并不好。它太本地化、太萨克森化、太乡土了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他并不想回到旧时代
   </strong>
   ，但环顾酒吧 —— 他度过一生中最好时光的地方 —— 烟熏过的黑色墙壁，看到从前演出的海报，看到桌子上方的吊灯（老斯塔西经常在这里藏麦克风：“喂喂鲍里斯，你听得到吗？我的声音够大吗？”）；很显然，藏有异议文化的玻璃罐 —— 让他从前的生活产生意义，让他从前可以用卡巴莱的票子从屠夫那里换香肠的日子 —— 所有这一切被 “统一” 掉了。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    —— 未完待续 ——
   </em>
  </p>
