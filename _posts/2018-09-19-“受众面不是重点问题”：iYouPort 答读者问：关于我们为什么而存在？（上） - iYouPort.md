  <p class="graf graf--p graf-after--figure" id="ebb7">
   还记得当中国刚刚建造 GFW 的时候，人们普遍关切的问题是如何翻墙？于是技术人士开始推广各种能突破审查封锁的工具，并传播它们；随着越来越多的翻墙技术的普及，人们关切的问题变成了“墙外”有什么不同？于是内容提供者开始着眼于“我们要在 Twitter(etc.) 上提供墙内必定会被审查的内容”。比如墙内禁止批评政策，Twitter 中文就几乎完全变成了调侃八卦。没人考虑它究竟有什么意义？
  </p>
  <p class="graf graf--p graf-after--p" id="b778">
   这一过程看起来很合理。但它暴露了一个问题，即 我们的思路是否一直在被当局的管制进度所绑架 — — 封锁、和反封锁；审查、和反审查 — — 在这场猫鼠游戏中人民一直是老鼠，以至于全部注意力被集中在“当局又有什么新动作了吗”这个狭小的范围内。
  </p>
  <p class="graf graf--p graf-after--p" id="b08b">
   不，这不是反抗。这不是互联网的正确用法。
  </p>
  <p class="graf graf--p graf-after--p" id="69c4">
   互联网是一个巨大的并迅速增长的数据金矿，社交媒体应该是各种动员、连接、知识共享、并汇集大众智慧的绝佳场域，它们合起来就是我们最有力的武器。当我们有效利用它们时，我们就变成了猫，逃跑的角色应该是当权者。
  </p>
  <p class="graf graf--p graf-after--p" id="6bfa">
   iYouPort 曾经在多篇文章中反映过上述思路，比如《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E4%BA%92%E8%81%94%E7%BD%91%E7%A9%B6%E7%AB%9F%E6%98%AF%E4%BB%80%E4%B9%88-%E6%9D%A5%E8%87%AA-%E7%9B%91%E8%A7%86%E8%B0%B7-%E6%83%8A%E4%BA%BA%E7%9A%84%E5%90%AF%E7%A4%BA-cdd6b5b6bd9d" href="https://medium.com/@iyouport/%E4%BA%92%E8%81%94%E7%BD%91%E7%A9%B6%E7%AB%9F%E6%98%AF%E4%BB%80%E4%B9%88-%E6%9D%A5%E8%87%AA-%E7%9B%91%E8%A7%86%E8%B0%B7-%E6%83%8A%E4%BA%BA%E7%9A%84%E5%90%AF%E7%A4%BA-cdd6b5b6bd9d" rel="noopener noreferrer" target="_blank">
    互联网究竟是什么
   </a>
   》、《
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/1007" href="https://pao-pao.net/article/1007" rel="noopener nofollow noreferrer" target="_blank">
    为什么说当心不要被恶意愚蠢拉低智商？
   </a>
   》、《
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/922" href="https://pao-pao.net/article/922" rel="noopener nofollow noreferrer" target="_blank">
    中国人什么都知道
   </a>
   》等等。
  </p>
  <p class="graf graf--p graf-after--p" id="8149">
   而现在，我们是在实践它：关于互联网应该如何使用的实践。我们能力有限，对中国社会的了解肯定没有广大中文读者更多，但技术是无国界的，我们有想法，这些技术曾经帮助了其他威权国家的反抗者，那么理论上它们就能帮助中国的反抗者。
   <strong class="markup--strong markup--p-strong">
    当所有人都处在互联网上时、当技术老大哥被全球专制政权共享时，国界的概念已经变得非常模糊
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="a98d">
   没有任何一个政权能被口水击败。当权者什么都不怕，除了真相。
  </p>
  <p class="graf graf--p graf-after--p" id="c9ff">
   以下是我们收到的一些读者提问，以及我们的回复。
  </p>
  <p class="graf graf--p graf-after--p" id="0a16">
   <strong class="markup--strong markup--p-strong">
    — — 为什么我们不需要更广泛的受众群？ — —
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="a33b">
   Q1:
   <em class="markup--em markup--p-em">
    如今几乎所有媒体平台包括账户都在努力宣传自己，甚至购买机器人炒作，相比下你们的受众量很低，为什么不宣传？你们以什么为骄傲？或者说以什么为验证标准以衡量你们的知识推广的价值？
   </em>
  </p>
  <p class="graf graf--p graf-after--p" id="87e9">
   A: 一直以来我们是在有意缩窄而不是放大受众面，这与绝大多数平台相反。原因有几个：
  </p>
  <p class="graf graf--p graf-after--p" id="51f5">
   首先，我们的定位是技术工具，专注于安全性、技术性和知识性，
   <strong class="markup--strong markup--p-strong">
    包括防御和攻势，而且这两部份往往高度相关
   </strong>
   。于是整体上很明显，我们的读者应该是民间的技术爱好者，加之长期以来，通过我们对反审查、言论自由、新闻自由、互联网民主化等理念的推广和解析，相信我们已经吸引到了一些富有正义感的读者。于是整体上我们是在为用知识技术追求正义这一思路的支持者提供协助。这一群体本身就不会很大，在任何一个国家都是如此，所以专注于传播推广的意义并不高。需要这些知识的朋友完全可以通过搜索关键字以找到我们。
  </p>
  <p class="graf graf--p graf-after--p" id="f286">
   其次，我们无法确认目标受众的分布，但有一个比较有意义的参数那就是人们对常用平台的选择，它意味着日常信息量、思考习惯、兴趣倾向等这些特征。对于中文读者来说相对自由度高一些的平台比如 Twitter，Medium 等，与微信等严格审查中文的平台，是有明显区别的。于是我们将传播限制在前者，将更有希望提高读者受益的可能性。（当然，这不是个严谨的标准，尤其是在
   <strong class="markup--strong markup--p-strong">
    Twitter 中文内容一直在加速无意义化
   </strong>
   的背景下，但我们现在还没有别的办法。下图这篇文章写得不够好，但这句按语是真实的）
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-2797 jetpack-lazy-image" data-lazy-sizes="(max-width: 630px) 100vw, 630px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/09/0-15.jpeg?resize=630%2C888&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/09/0-15.jpeg?w=630&amp;ssl=1 630w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/09/0-15.jpeg?resize=213%2C300&amp;ssl=1 213w" data-recalc-dims="1" height="888" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/09/0-15.jpeg?resize=630%2C888&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="630"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-2797" data-recalc-dims="1" height="888" sizes="(max-width: 630px) 100vw, 630px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/09/0-15.jpeg?resize=630%2C888&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/09/0-15.jpeg?w=630&amp;ssl=1 630w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2018/09/0-15.jpeg?resize=213%2C300&amp;ssl=1 213w" width="630"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="931c">
   第三，我们没有流量指标，因为没有投资人也没有老板，于是无需追求过高的阅读量。虽然这样会艰苦一些，但这种运作形式能让我们更加自由地专注于我们的信念，即
   <strong class="markup--strong markup--p-strong">
    提升民间的反抗实力
   </strong>
   。
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/advocacy/@iyouport/is-democracy-in-its-death-throes-what-should-we-do" href="https://steemit.com/advocacy/@iyouport/is-democracy-in-its-death-throes-what-should-we-do" rel="noopener nofollow noreferrer" target="_blank">
    我们看到了人们对民主自由的向往
   </a>
   ，同时也看到了技术能力、知识和方法的欠缺以至于难以实现这一期待。希望 iyouport 能成为一个有效的工具箱，为有志者带来切实的帮助。
  </p>
  <p class="graf graf--p graf-after--p" id="65af">
   于是，我们的骄傲将来自于中国的读者能从中获益，
   <strong class="markup--strong markup--p-strong">
    使用这些工具在正义的目标上
   </strong>
   ，不论是自我保护、主动出击、挖掘真相赢得公民权、还是应对现场侦查和追踪。而不是表面优势的点击量和转发量。并且鉴于我们的主题，也不可能支持更高的转发评论，就如大家所知，如果一个账户显示出对反侦察技术的高度偏好，对于监视者来说（如果存在的话）那将意味着什么？所以，我们相信“隐性读者”的力量，价值将更有可能在这一范围内发挥。
  </p>
  <p class="graf graf--p graf-after--p" id="ddf2">
   至于价值验证，短时间内无法做到。
   <strong class="markup--strong markup--p-strong">
    iyouport 是一个匿名的无中心的小型网络
   </strong>
   ，汇聚大众智慧和能力，集思广益，只要有创意并实用于我们的目标，我们将来者不拒。不论您擅长翻译、技术操作、情报分析、还是具备各方面的实战经验，都可以在此得到发挥。
  </p>
  <p class="graf graf--p graf-after--p" id="6e8f">
   我们认为如果能完成这一步就是极大的价值。因为它并不容易做到，尤其是
   <strong class="markup--strong markup--p-strong">
    对于中国的现状来说，即 粉丝经济；人们对结论的过度追求，而不是获取结论的方法；对信息刺激性的过度追求，而不是真相，以及挖掘真相的方法，等等。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="47fe">
   这一网络将由创造者、技术共享者、志愿者和读者组成，随着其不断扩大，不断接近我们的目标。
  </p>
  <p class="graf graf--p graf-after--p" id="1ccb">
   <strong class="markup--strong markup--p-strong">
    — — 我们对“影响”的认识可能与众不同 — —
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="71a9">
   Q 2、
   <em class="markup--em markup--p-em">
    你们相信影响是价值吗？你们曾经带来过什么样的影响？可以举几个例子吗，谢谢。
   </em>
  </p>
  <p class="graf graf--p graf-after--p" id="69ed">
   A：是的，影响的确有强大的力量。当目之所及的人群都在拥护谷歌时，你很难想象得到，其中的确有不少中国读者对监视资本主义的认识是透彻的。在我们发表了一系列分析之后，很明显越来越多的反馈显示出这一影响的存在。
  </p>
  <p class="graf graf--p graf-after--p" id="8985">
   <strong class="markup--strong markup--p-strong">
    也许与大众的理解不同，这里的影响指的并非是致力于对认知的改变，更多是对发言的勇气的调动
   </strong>
   — — 突破恶意愚蠢的禁锢。由于社交媒体上人们倾向于与社交范围内的主流保持一致，以获得群体内的认同感，于是很多时候只是没有机会说话，而非误解。
  </p>
  <p class="graf graf--p graf-after--p" id="fe75">
   当我们呈现一种基于事实而非迎合任何一种普遍认知的表达方式的时候，影响就出现了。其实更准确的说法应该是带动。最近有一部很棒的纪录片推出，名叫‘Free Speech, Parler Sans Peur’（后半句法语的意思是‘毫无恐惧地说话’）。自由表达曾经的意思仅仅限于对集中化审查的反对，而如今它变成了一种复杂且棘手的问题 — — 群体认同被夸大了，它带来了强大的自我审查，舆论变成了“你支持还是反对”的两边倒，即便有想法的人也难以有机会就事论事。
  </p>
  <p class="graf graf--p graf-after--p" id="cc4d">
   我们认为这种状况不利于思考，从而不利于解决问题。
  </p>

  <p class="graf graf--p graf-after--p" id="5179">
   我们
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/iyouport_news/status/969809148660498432" href="https://twitter.com/iyouport_news/status/969809148660498432" rel="noopener nofollow noreferrer" target="_blank">
    在早前的文章中写过
   </a>
   ，很快，人类将被分为两部分：懂电脑的人、和害怕电脑的人。如果不能成为前者，将遇到很多阻碍，甚至人身危险。
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/DonEvansWm/status/983568586827599873" href="https://twitter.com/DonEvansWm/status/983568586827599873" rel="noopener nofollow noreferrer" target="_blank">
    事实早已显示
   </a>
   ，当权者更倾向于利用技术来对付老百姓，然而老百姓如果普遍不懂技术、不重视技术，只能意味着，连自我保护都无法做到，必输无疑。唯一的关键是，究竟想不想保护自己？想不想摆脱天罗地网？因为摆脱是可行的，只有摆脱的意愿才能促成我们在这里的对话，才有变革的可能。
  </p>
  <p class="graf graf--p graf-after--p" id="0eb0">
   另一方面，随着全球政府不透明的程度都在提升，公民的无知程度也正在被加大，公民无法得知政府干了什么，也就无法作出有效的决定，民主因此被破坏。
   <strong class="markup--strong markup--p-strong">
    这就是透明度革命的起源，也是它被严重打压的原因
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="c7de">
   2⃣️ 集中化的社交网络就意味着封闭，但对于中国社会来说，封闭是尤其明显的。既存在来自当局的封锁、审查和 GFW，也有来自中国公民的自我封闭。详见我们早前的文章《
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/773" href="https://pao-pao.net/article/773" rel="noopener nofollow noreferrer" target="_blank">
    隧道生态和心理操控
   </a>
   》以及《
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/1007" href="https://pao-pao.net/article/1007" rel="noopener nofollow noreferrer" target="_blank">
    恶意愚蠢
   </a>
   》、《
   <a class="markup--anchor markup--p-anchor" data-href="https://pao-pao.net/article/706" href="https://pao-pao.net/article/706" rel="noopener nofollow noreferrer" target="_blank">
    极权社会的心理阴影
   </a>
   》。
  </p>
  <p class="graf graf--p graf-after--p" id="7417">
   并且，随着突破审查封锁的技术一直在发展，上述后者 即 自我封闭 的影响已经高过前者。很多人显然还没有学会如何使用互联网，使用习惯仍在局域网下，加之自我营销的欲求显然高于知识交换、被认同的欲求高于批判性思考，于是自我封闭的社交媒体气泡效应在中国尤其突出。这种状况意味着人们非常容易被欺骗和诱导，也是假消息在 Twitter 中文圈爆炸性增长的原因（关于为什么，
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E6%94%BF%E6%B2%BB%E7%9A%84%E4%B8%A4%E6%9E%81%E5%88%86%E5%8C%96%E5%A6%82%E4%BD%95%E5%9C%A8%E7%A4%BE%E4%BA%A4%E7%BD%91%E7%BB%9C%E4%B8%8A%E4%BD%93%E7%8E%B0-%E4%B8%BA%E4%BB%80%E4%B9%88%E8%BF%99%E6%A0%B7%E7%9A%84%E7%94%9F%E6%80%81%E5%BE%88%E5%AE%B9%E6%98%93%E8%A2%AB%E6%93%8D%E7%BA%B5-%E5%9B%B4%E8%A7%82-twitter-%E6%B4%BB%E5%8A%A8%E5%9C%B0%E5%9B%BE-ba5cb3aebb54" href="https://medium.com/@iyouport/%E6%94%BF%E6%B2%BB%E7%9A%84%E4%B8%A4%E6%9E%81%E5%88%86%E5%8C%96%E5%A6%82%E4%BD%95%E5%9C%A8%E7%A4%BE%E4%BA%A4%E7%BD%91%E7%BB%9C%E4%B8%8A%E4%BD%93%E7%8E%B0-%E4%B8%BA%E4%BB%80%E4%B9%88%E8%BF%99%E6%A0%B7%E7%9A%84%E7%94%9F%E6%80%81%E5%BE%88%E5%AE%B9%E6%98%93%E8%A2%AB%E6%93%8D%E7%BA%B5-%E5%9B%B4%E8%A7%82-twitter-%E6%B4%BB%E5%8A%A8%E5%9C%B0%E5%9B%BE-ba5cb3aebb54" rel="noopener noreferrer" target="_blank">
    详见这篇文章
   </a>
   ）
  </p>
  <p class="graf graf--p graf-after--p" id="ae60">
   越来越多的用户开始抱怨“中国民主派”集中的 Twitter 中文圈混乱的生态，我们不想多说，但这些现象的存在意味着
   <strong class="markup--strong markup--p-strong">
    阻碍变革的因素正在膨胀
   </strong>
   。技术技能的缺乏和其引发的对安危的恐慌、对真相的漠视以及挖掘真相的方法和能力的欠缺，都导致人们更倾向于刺激的、无厘头的信息，有意制造的阴谋论就在其中。
  </p>
  <p class="graf graf--p graf-after--p" id="f289">
   当下全球很多国家都遭遇了社会对真相的无视，兰德专门做了一个长达三百多页的研究报告分析这一问题，详见我们中文摘要分析《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E7%9C%9F%E7%9B%B8%E8%A1%B0%E5%8F%98-%E5%BD%93%E6%B0%91%E4%B8%BB%E7%97%85%E5%8D%B1-%E5%8F%98%E9%9D%A9%E9%81%A5%E9%81%A5%E6%97%A0%E6%9C%9F%E6%97%B6-%E6%88%91%E4%BB%AC%E5%BA%94%E8%AF%A5%E6%80%9D%E8%80%83%E4%BB%80%E4%B9%88-9f1990b6e9f1" href="https://medium.com/@iyouport/%E7%9C%9F%E7%9B%B8%E8%A1%B0%E5%8F%98-%E5%BD%93%E6%B0%91%E4%B8%BB%E7%97%85%E5%8D%B1-%E5%8F%98%E9%9D%A9%E9%81%A5%E9%81%A5%E6%97%A0%E6%9C%9F%E6%97%B6-%E6%88%91%E4%BB%AC%E5%BA%94%E8%AF%A5%E6%80%9D%E8%80%83%E4%BB%80%E4%B9%88-9f1990b6e9f1" rel="noopener noreferrer" target="_blank">
    真相衰变
   </a>
   》，以及这个问题应该怎么办。
  </p>
  <p class="graf graf--p graf-after--p" id="c13f">
   3⃣️ 羊群效应是最为普遍的，没有民主经验的社会尤其如此。人们很容易因被诱导而作出错误的选择。一直以来这就是威权国家难以变革为民主国家的最大障碍之一。令社会欠缺批判性思考教育显然是独裁者的一大杀器。
  </p>
  <p class="graf graf--p graf-after--p" id="2fe3">
   可以肯定的是，
   <strong class="markup--strong markup--p-strong">
    阴谋论和假消息不能也无法通过审查来解决
   </strong>
   ，审查只能促进人们对假消息的兴趣。今年以来迅速高升的审查问题一直是西方网络的热门舆论，而且其中出现了为数不少的支持审查的观点，非常令人困惑，
   <strong class="markup--strong markup--p-strong">
    这意味着批判性思考必需被不断加强、重申，而不是离开课堂就能丢弃的
   </strong>
   。我们的详细分析
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%A6%82%E6%9E%9C%E5%85%A8%E7%BD%91%E5%B0%81%E9%94%81%E5%91%A8%E5%B0%8F%E5%B9%B3-%E4%B8%80%E4%B8%AA%E8%88%86%E8%AE%BA%E7%83%AD%E7%82%B9%E6%88%96%E8%AE%B8%E8%83%BD%E5%B8%AE%E5%8A%A9%E4%B8%AD%E5%9B%BD%E7%A4%BE%E4%BC%9A%E6%9B%B4%E6%B7%B1%E5%B1%82%E7%90%86%E8%A7%A3%E5%AE%A1%E6%9F%A5-5180387d2aac" href="https://medium.com/@iyouport/%E5%A6%82%E6%9E%9C%E5%85%A8%E7%BD%91%E5%B0%81%E9%94%81%E5%91%A8%E5%B0%8F%E5%B9%B3-%E4%B8%80%E4%B8%AA%E8%88%86%E8%AE%BA%E7%83%AD%E7%82%B9%E6%88%96%E8%AE%B8%E8%83%BD%E5%B8%AE%E5%8A%A9%E4%B8%AD%E5%9B%BD%E7%A4%BE%E4%BC%9A%E6%9B%B4%E6%B7%B1%E5%B1%82%E7%90%86%E8%A7%A3%E5%AE%A1%E6%9F%A5-5180387d2aac" rel="noopener noreferrer" target="_blank">
    在这里
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="fa41">
   另附：为什么中国社会使用互联网的方法是错的，正确的使用方法是什么，详见《为
   <a class="markup--anchor markup--p-anchor" data-href="https://steemit.com/life/@iyouport/6icpu8" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e4%b8%ad%e5%9b%bd%e7%a4%be%e4%bc%9a%e9%9a%be%e4%bb%a5%e7%bb%84%e7%bb%87%e8%b5%b7%e6%9d%a5/" rel="noopener nofollow noreferrer" target="_blank">
    什么如今的中国社会组织不起来
   </a>
   》
