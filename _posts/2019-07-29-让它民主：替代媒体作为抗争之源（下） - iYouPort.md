    上一篇文章中
   </a>
   我们讨论了一些让新闻再次服务于民主的建议。主要是给公民的建议。当然，也可以给职业媒体一些建议，但理论上他们绝对做不到。于是我们专注于公民，毕竟民主意味着公民的行动，而非任何特权阶层。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E8%AE%A9%E5%AE%83%E6%B0%91%E4%B8%BB%EF%BC%9A%E5%9C%A8%E7%9C%9F%E7%9B%B8%E4%BA%8B%E4%B8%9A%E4%B8%AD%E6%9E%84%E6%83%B3%E6%9C%AA%E6%9D%A5%E5%9B%BE%E6%99%AF%EF%BC%88%E4%B8%8A%EF%BC%89/" href="https://www.iyouport.org/%E8%AE%A9%E5%AE%83%E6%B0%91%E4%B8%BB%EF%BC%9A%E5%9C%A8%E7%9C%9F%E7%9B%B8%E4%BA%8B%E4%B8%9A%E4%B8%AD%E6%9E%84%E6%83%B3%E6%9C%AA%E6%9D%A5%E5%9B%BE%E6%99%AF%EF%BC%88%E4%B8%8A%EF%BC%89/" rel="noopener noreferrer" target="_blank">
    上一次
   </a>
   分析到一些不现实的和失败的传播实验，本文将介绍一些比较成功的、可以作为参考的实验案例，即：替代媒体方案。
   <strong class="markup--strong markup--p-strong">
    理论上这一民主实践有希望在中国获得成功。
   </strong>
  </p>
  <p class="graf graf--p">
   替代媒体是指在内容、制作或发行方面与已建立的或主导类型的媒体完全不同的媒体 —— *完全不同*就是关键字，据我们了解中国目前没有这样的项目，中国的自媒体不是替代媒体、公众号和私人博客只是主流媒体的复制品。下面会解释为什么。
  </p>
  <p class="graf graf--p">
   替代媒体有多种形式，包括印刷、音频、视频、互联网和街头艺术等等，都包括。
  </p>
  <p class="graf graf--p">
   这不是新鲜事物，而是有传统的。人民的自由之心自古以来都很强盛。
  </p>
  <p class="graf graf--p">
   一些早前的例子包括20世纪60年代的反文化杂志、少数族裔和土著人媒体，如 加拿大的第一人民电视网（后来更名为土著人民电视网），以及最近出现的一些在线的开放出版新闻网站，如 Indymedia。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    关键的区别在于：主流大众传媒总体上代表政府和寡头企业的利益，而替代媒体则往往是非商业性项目 —— 主张那些被排斥在主流之外的人的利益
   </strong>
   ，例如 穷人、政治反对派、少数民族、劳工团体、LGBTQI 等等。
  </p>
  <p class="graf graf--p">
   ⚠️所以，并不是说你没有资金靠众筹维持并没有商业化就可以称得上替代媒体的，关键在于你为谁服务。
  </p>
  <p class="graf graf--p">
   这些媒体传播的是被边缘化社区的观点，例如在公民社会组织 Democracy Now 中你
   <strong class="markup--strong markup--p-strong">
    才有
   </strong>
   机会听到的观点，并创建身份社区，例如在 YouTube 上创建的 It Gets Better Project，以应对同性恋青少年自杀事件的增加现象。类似案例非常多。
  </p>
  <p class="graf graf--p">
   简单说，替
   <strong class="markup--strong markup--p-strong">
    代媒体是行动主义者，而不是嘴炮
   </strong>
   。
  </p>
  <p class="graf graf--p">
   另类媒体挑战了主流文化的信念和价值观，并被 Antonio Gramsci 的文化霸权理论的拥护者们描述为“反霸权”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “反”，就是中心意义。
  </p>
  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    —— 定义 ——
   </strong>
  </h4>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “替代媒体”有各种各样的定义。例如，John Downing 将“激进的替代媒体”定义为“表达反霸权政策、优先事项和观点的另类愿景的媒体”；Chris Atton 在评估该术语的各种定义时，反复注意到源自小规模反霸权群体和个人的替代媒体制作的重要性。[Atton, C. (2002). “Approaching Alternative Media: Theory and Methodology.” In Alternative Media. Thousand Oaks: Sage Publications]
  </p>
  <p class="graf graf--p">
   Christian Fuchs 还认为，
   <strong class="markup--strong markup--p-strong">
    替代媒体必须具有四种不同的特性
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    首先，这些媒体的受众必须参与创造替代媒体所发布的内容。
   </li>
   <li class="graf graf--li">
    第二是它必须与主流不同。
   </li>
   <li class="graf graf--li">
    第三，它应该创造一个与国家和寡头公司完全不同的视角。
   </li>
   <li class="graf graf--li">
    第四个属性是替代媒体必须“与市场和/或国家建立不同类型的关系”。
   </li>
  </ul>
  <p class="graf graf--p">
   这四点非常重要。也就是说，具备这四点才能称之为替代媒体，而不仅仅是独立媒体。独立媒体有可能无需做到全部这四点。
  </p>
  <p class="graf graf--p">
   于是您能看清，为什么我们说中国目前没有这样的公民社会项目。并且，您可以理解到替代媒体的参与者在技能方面的重要性，
   <strong class="markup--strong markup--p-strong">
    也就是IYP不断介绍技术技能的用意，这是民主实践中关键的基础设施
   </strong>
   。
  </p>
  <p class="graf graf--p">
   您一定熟悉一句流行语叫“医生不会告诉你的事”，人们普遍对这样的流行语所涵盖的内容感兴趣，因为绝大多数人能根深蒂固对认识到，那些拥有强大权力的人不会完全为普通人着想（只是类比，此处无关医患关系）。
  </p>
  <p class="graf graf--p">
   媒体也一样。对于那些垄断者和具有强大后台的媒体，你不用指望从它们那里看到有利于公民的完整真相，而且它们很擅长愚弄和操纵公民，以利于权势的利益。
  </p>
  <p class="graf graf--p">
   于是，打破它们的第一步是应该认识到它们是如何操弄你的，这方面非常微妙，并非显而易见，我们在“How To Win”部分有很多介绍。了解并能准确识别国家和寡头公司的影响力手段，是打造合规替代媒体的重要基础。
  </p>
  <p class="graf graf--p">
   简而言之，当你看到异议记者主张“像其他国家的主流媒体学习什么以抵制本国的主流媒体”的时候，你能知道，这里尚且不会诞生替代媒体，没有公民社会。
  </p>

  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    —— 常见的方法和做法 ——
   </strong>
  </h4>
  <p class="graf graf--p">
   替代媒体学术研究的方法试图理解这些媒体的重要性，每一个研究都强调媒体的不同方面，包括公共领域的作用、社会运动方面的作用、以及创造媒体的社区的参与。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    民主理论与公共领域 ——
   </strong>
  </p>
  <p class="graf graf--p">
   理解替代媒体的一种方式是考虑他们在民主传播过程中的作用。
  </p>
  <p class="graf graf--p">
   哲学家 Jürgen Habermas 提出，一个健康的民主社会需要一个可以积极发动公民参与、在公民之间进行理性辩论的空间。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    至关重要的是，公共领域的对话不受任何当局的控制，以便公民能够平等地交换意见
   </strong>
   。这意味着需要言论自由和新闻自由。
  </p>
  <p class="graf graf--p">
   在哈贝马斯关于公共领域的想法中，参与对所有人开放，所有参与者都被认为是平等的，任何问题都可以提出来进行辩论。
  </p>
  <p class="graf graf--p">
   但是，这种观点没有注意到女性和少数群体（及其利益）被固有地被排除在公共领域的辩论之外。
  </p>
  <p class="graf graf--p">
   鉴于这种社会不平等，哲学家 Nancy Fraser 认为：
   <strong class="markup--strong markup--p-strong">
    多个独立公共领域是十分重要的，其中下属群体的成员可以首先在他们自己之间审议问题和关注问题，然后将这些问题主张带到更大的公共领域中
   </strong>
   。
  </p>
  <p class="graf graf--p">
   替代媒体对于发展群体的需求和身份以及挑战更大的主导公共领域至关重要。例如，一个女权主义的反公共领域负责传播这样一种观点，即家庭虐待和生殖权利等妇女问题在大公共领域值得辩论。一个政治异议的反公共领域负责传播的观点是，权势在隐瞒什么、我们如何才能反击。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    社运媒体 ——
   </strong>
  </p>
  <p class="graf graf--p">
   社会运动是一种集体行动。涉及大型、有时是非正式的团体或组织，它们关注特定的政治或社会问题，并推动、抵制或消除社会变革。前者指的是进步运动，后者指得是保守运动。在中国发生的更多是后者。
  </p>
  <p class="graf graf--p">
   ⚠️社会运动媒体指的是社会运动如何使用媒体，而且，由于社会运动的本质，媒体往往是一种重要的替代方式。
  </p>
  <p class="graf graf--p">
   沟通对社会运动的成功至关重要。
  </p>
  <p class="graf graf--p">
   研究表明，社会运动在通过主流媒体进行交流时遇到了很大的困难，
   <strong class="markup--strong markup--p-strong">
    因为主流媒体经常会系统地歪曲、诬蔑或忽视社会运动的观点
   </strong>
   。 [Stein, Laura (2009). “Social movement web use in theory and practice: a content analysis of US movement websites”.

  <p class="graf graf--p">
   主流媒体很可能会否认社会运动在其发展的关键时刻的代表性意义，利用消息框架破坏、或削弱公众对运动合法性的看法，或隐含地鼓励寻求报道的运动行为者迎合社会活动主流报道的可疑价值，包括对暴力、情绪和口号的高度关注。
   <strong class="markup--strong markup--p-strong">
    你能在黄背心运动中看到主流媒体的集体抹黑和冷漠，理解这点并不难。
   </strong>
  </p>
  <p class="graf graf--p">
   这种有问题的社会运动报道通常被称为抗议范式：
   <strong class="markup--strong markup--p-strong">
    大众媒体通过对抗议者的偏颇描绘使抗议团体被边缘化，并通过这样做而支持现状
   </strong>
   。因此，💡社会运动经常需要转向替代媒体形式和实践，以便更有效地实现其目标。
  </p>
  <p class="graf graf--p">
   黄背心并不是特例。主流媒体对社会运动的抵制在新闻业史上比比皆是。另一个最典型的例子就是“占领运动”，它始于2011年9月的占领华尔街。
  </p>
  <p class="graf graf--p">
   占领运动抗议世界各地的社会和经济不平等，其主要目标是使所有社会中的经济和政治关系不那么垂直分层，分布得更加平坦。
  </p>
  <p class="graf graf--p">
   地方团体往往有不同的侧重点，但在该运动的主要关注点中，涉及大公司和全球金融体系如何控制世界的方式不成比例地使极少数人受益，这种不公平破坏了民主、并且导致不稳定。
  </p>

  </p>
  <p class="graf graf--p">

   还记得我们在上篇文章中所举的例子吗？我们的记者留言给主流媒体的一带一路峰会驻场记者，要求他们关系一下会场外封路所造成的民生问题，但是他们不会关心这点，这不是他们的利益所在。
  </p>
  <p class="graf graf--p">
   💡替代媒体是激进主义者。人权、环境运动、和公民权利等领域的社会运动产生了替代媒体，以促进其目标，传播其意识，并激发公民参与和支持。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    人权 ——
   </strong>
  </p>
  <p class="graf graf--p">
   使用替代媒体进行人权社会运动的一个例子是 WITNESS 组织。
  </p>
  <p class="graf graf--p">
   WITNESS 是一个非营利人权组织，其使命是与实地组织合作，支持记录侵犯人权的行为及其后果，
   <strong class="markup--strong markup--p-strong">
    以促进公众参与、政策变革和公正
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他们使用手持式便携式摄像机和智能手机等技术录制视频，以吸引全世界的注意力，并更深入传播侵犯人权的行为。
  </p>
  <p class="graf graf--p">
   他们记录了巴西贫民窟的警察暴力、刚果民主共和国的儿童兵、巴西和美国的人口贩运、以及许多其他侵犯人权行为，
   <strong class="markup--strong markup--p-strong">
    使用的都是替代媒体
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    环保运动 ——
   </strong>
  </p>
  <p class="graf graf--p">
   使用替代媒体的环保运动的一个典型例子是绿色和平组织。
  </p>
  <p class="graf graf--p">
   绿色和平组织是一个非政府环境组织，将其活动集中在全球问题上，如气候变化、森林砍伐、过度捕捞、商业捕鲸、基因工程和反核等问题。
  </p>
  <p class="graf graf--p">
   它使用直接行动、游说和研究并用的方法来实现其目标，尤其是：替代媒体。
   <strong class="markup--strong markup--p-strong">
    他们使用在线策略，如 播客和博客以及表演艺术
   </strong>
   。后面的“形式“部分有更多介绍。[Waltz, M (2005).
   <em class="markup--em markup--p-em">
    Alternative and Activist Media
   </em>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    民权运动 ——
   </strong>
  </p>
  <p class="graf graf--p">

  <h4 class="graf graf--h4">
   <strong>
    —— 参与式文化 ——
   </strong>
  </h4>
  <p class="graf graf--p">
   替代媒体经常被研究作为参与文化的一种表现形式，最重要的是，💡
   <strong class="markup--strong markup--p-strong">
    其中公民不仅仅是消费者，而是作为贡献者或生产者
   </strong>
   。
  </p>
  <p class="graf graf--p">
   通过开放媒体制作的渠道，参与式文化被认为可以促进民主、公民参与、和创造性表达。
  </p>
  <p class="graf graf--p">
   参与式文化很古老，远远早于互联网出现。业余新闻协会就是一种参与式文化形式，出现在19世纪后期。这些协会的成员排版并打印他们自己的出版物，这些出版物通过订户网络邮寄。
  </p>
  <p class="graf graf--p">
   Zines，一个谈话广播节目和小组项目，也先于博客、播客、维基和社交网络。借助维基百科，Tumblr，Imgur，Reddit，Vine 和 YouTube 等网络服务，所有这些服务都允许用户分发原创内容，使媒体制作更具参与性。
  </p>
  <p class="graf graf--p">
   随着公民在收集、报告、分析和传播新闻及信息方面发挥更积极的作用，
   <strong class="markup--strong markup--p-strong">
    参与式新闻也创造了替代媒体。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种形式的替代和激进的新闻采访和报道功能在主流媒体机构是绝对找不到的，往往是对专业新闻主要缺点的回应
   </strong>
   。
  </p>
  <p class="graf graf--p">
   💡它参与新闻实践，但是不会受到利润目标的驱动，具有不同的理想，并依赖于其他合法性来源。
  </p>
  <p class="graf graf--p">
   参与式媒体方法考虑参与制作媒体内容以及将媒体制作流程作为替代媒体的定义特征，以做出决策。
  </p>
  <p class="graf graf--p">
   💡参与式文化可以通过多种方式实现。其中媒体素养是通过了解媒体系统的惯例和生产方式以开始参与的途径之一，也就是我们前文中分析到的公民社会行动者必须具备充足的技能。
   <strong class="markup--strong markup--p-strong">
    学习自己制作媒体的个人是将公民从扫盲转向参与的重要步骤
   </strong>
   。
  </p>
  <p class="graf graf--p">
   粉丝小说、社区广播或FM、家庭视频等等，这些都是公民可以制作的媒体内容，实现参与文化，并逐步转向制作替代媒体。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    通过促进参与，替代媒体有助于加强公民态度，使公民能够积极参与与日常生活相关的主要领域之一，并将他们的沟通权利付诸实践
   </strong>
   。
  </p>
  <p class="graf graf--p">
   为了说明民主与参与媒体制作之间的关系，“公民媒体”一词表明：替代媒体可以帮助那些制作媒体的人也成为积极的公民。这一概念与社区媒体密切相关。
  </p>
  <h4 class="graf graf--h4">
   <strong>
    —— 社区媒体 ——
   </strong>
  </h4>
  <p class="graf graf--p">
   社区媒体包括公民媒体、参与媒体、活动家和激进派媒体、以及地方或区域特定平台参与的更广泛的传播形式。
   <strong class="markup--strong markup--p-strong">
    与其他形式的替代媒体一样，社区媒体一直在努力避免商业化。
   </strong>
  </p>
  <p class="graf graf--p">
   消除或避免单一所有权或赞助的动机，⚠️是希望不受到监视、不需要某种义务满足特定议程。
  </p>
  <p class="graf graf--p">
   社区媒体通常被归类为草根，这种描述既适用于财务结构，也适用于内容创建过程。虽然社区媒体的多样性因媒体平台而异（广播、电视、网络或印刷品），但媒体来源
   <strong class="markup--strong markup--p-strong">
    通常向公众/社区开放
   </strong>
   以提交材料和内容。
  </p>
  <p class="graf graf--p">
   这项开放的政策需要符合社区媒体的价值观，以维持民主的方法和精神。
  </p>
  <p class="graf graf--p">
   历史上，社区媒体有助于提供另一种政治声音。在世界各地的社区形式中，这类媒体更多被用来提升特定空间的需求和语话，通常与地理、文化、社会或经济相似性密切联系。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    民族和土著媒体 ——
   </strong>
  </p>
  <p class="graf graf--p">
   少数族裔社区媒体可以是本地化的，也可以是国家的，用于向目标人群传播信息。
   <strong class="markup--strong markup--p-strong">
    它们为少数群体社区以及少数群体和多数群体之间的讨论和交流，提供了平台
   </strong>
   。
  </p>
  <p class="graf graf--p">
   请注意一点，⚠️少数族裔替代媒体并非自娱自乐的封闭圈子，它的目的是与多数族裔群体形成更友好的沟通和协作。
  </p>
  <p class="graf graf--p">
   通常，以少数族裔为重点的媒体提供必要的资源，以其自己的语言为其受众提供基本信息，帮助特定群体作为其居住国的平等公民参与。
  </p>
  <p class="graf graf--p">
   这些媒体平台和网点根据种族、民族或文化特征，
   <strong class="markup--strong markup--p-strong">
    为被剥夺权利或被边缘化的群体提供文化交流和提升或赋权的机会
   </strong>
   。
  </p>
  <p class="graf graf--p">
   从历史上看，这些形式的媒体有双重目的：💡一方面向传统上被主流媒体忽视或绕避的社区传播信息，另方面作为政治抗议或社会改革的工具。
  </p>
  <p class="graf graf--p">
   为解决少数群体发声而创建的空间通常跨越替代媒体和激进主义媒体的路线，致力于通过主流措施提供无法获得的资源，并转变普遍可接受的观点或对特定人群的理解。
  </p>
  <p class="graf graf--p">
   在整个20世纪，媒体空间的开发都是明显活跃的，是为了适应不断增长的多元文化状态。
  </p>
  <p class="graf graf--p">
   比如在美国，非裔美国人创建了像 Chicago Defender 这样的当地出版物来分享关键信息，以保护公民免受警察和政策制定者的歧视行为，而 Jet &amp; Ebony 的杂志则赋予了全国黑人以身份，赞扬了美国黑人的成就和思想领导力。
  </p>
  <p class="graf graf--p">
   而且，类似的做法在拉丁裔/亚裔群体中也变得越来越普遍。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-3614 jetpack-lazy-image" data-lazy-sizes="(max-width: 1066px) 100vw, 1066px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-1.png?resize=1066%2C722&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-1.png?w=1066&amp;ssl=1 1066w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-1.png?resize=300%2C203&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-1.png?resize=768%2C520&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-1.png?resize=1024%2C694&amp;ssl=1 1024w" data-recalc-dims="1" height="722" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/07/3-1.png?resize=1066%2C722&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1066"/>

   </noscript>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    从属者研究 ——
   </strong>
  </p>
  <p class="graf graf--p">
   在替代媒体研究和从属者研究中发现了相关的目标，因为两个领域都普遍存在对被剥夺权利和受压迫的声音的关注。
  </p>
  <p class="graf graf--p">
   从属者研究借鉴了安东尼奥·葛兰西（Antonio Gramsci）对“下层”群体的讨论，即 被认为在社会、经济和政治上处于劣势地位的群体。
  </p>
  <p class="graf graf--p">
   从属者研究中最重要的问题之一是 Gayatri Chakravorty Spivak 提出的，“从属者有机会说话吗？”她在她的同名论文中提出了这个问题。
  </p>
  <p class="graf graf--p">
   Spivak 调查
   <strong class="markup--strong markup--p-strong">
    从属者是否在霸权政治话语中有发言权，如果有，他们的声音是否能被听到，是否允许他们参与。⚠️这很重要，因为从属者参与政治和其他社会和文化实践的能力是挑战他们的从属地位的关键
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这种特殊的学术机构对于替代媒体的研究和讨论是非常有用的，因为他们共同关注被剥夺权利的人民参与和抑制主流霸权话语的能力。[Spivak, G. (2010). “Can the Subaltern Speak?” In
   <em class="markup--em markup--p-em">
    Can the Subaltern Speak? Reflections on the History of an Idea.
   </em>
   Ed. by Rosalind Russel.]
  </p>
  <p class="graf graf--p">
   在替代媒体学者 Clemencia Rodriguez 的研究中这种联系被加强。在对公民身份的讨论中，Rodriguez 评论说：
   <strong class="markup--strong markup--p-strong">
    “公民必须通过参与日常政治实践，在日常生活中才能获取公民身份……
   </strong>
  </p>
  <p class="graf graf--p">
   当公民积极参与重塑自己身份、他人身份和社会环境的行为时，他们就会产生出力量。
  </p>
  <p class="graf graf--p">
   中国公民为什么倾向于“吃瓜”，曾经有很多观察人士作出过解释（包括我们），但是不可否认的一点就在于，缺乏参与的积极性和条件，以至于被动地处于事不关己的旁观地位。
  </p>
  <p class="graf graf--p">
   因此，可以说，通过创建替代媒体的从属组织，人们确实可以表达自己的公民身份，由此产生了力量，并让他们发出自己的声音。希望中国公民能通过本文获得启示。[Rodriguez, C. (2001). “From Alternative Media to ‘Citizens’ Media.” In
   <em class="markup--em markup--p-em">
    Fissures in the Mediascape.
   </em>
   Creskill, NJ: Hampton Press.]
  </p>

  <h4 class="graf graf--h4">
   <strong>
    —— 媒体形式 ——
   </strong>
  </h4>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、地下出版物
   </strong>
  </p>
  <p class="graf graf--p">
   另类媒体由印刷出版物组成，
   <strong class="markup--strong markup--p-strong">
    提供与主流媒体和企业媒体的报纸、杂志和其他印刷品所提供的不同或持不同意见的观点
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Factsheet Five 出版商 Mike Gunderloy 将另类新闻描述为
   <strong class="markup--strong markup--p-strong">
    成熟的
   </strong>
   地下出版社。 Whole Earth, the Boston Phoenix, 和 Mother Jones 都属于这类。
  </p>
  <p class="graf graf--p">
   地下出版物出现的数量很少，通常难以辨认，踩在无法辨认的主题的薄冰上，从来不会为设计师的牛仔裤那种东西刊登广告。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    替代媒体的一个例子就是战术媒体，它使用“
   </strong>
   hit-and-run
   <strong class="markup--strong markup--p-strong">
    ”战术来引起人们对新出现问题的关注
   </strong>
   。通常，战术媒体试图揭露那些操控主流媒体来源的大公司。对此我们后面还有具体分析，关于《守门人作为认知杀手》。
  </p>
  <p class="graf graf--p">
   致力于战术媒体实践和信息活动的一个著名非政府组织就是 Tactical Technology Collective，
   <strong class="markup--strong markup--p-strong">
    它的理念和 IYP 差不多，即 协助人权倡导者使用技术保护自己，并在必要时进攻
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他们向全球社区免费发布了多个工具包，包括南亚非政府组织，并且协助建立一个自主的非政府组织框架，即 Security-In-A-Box。这是一个软件集合。为在潜在敌对政治气候下运作的非政府组织保护数据安全，以及一个简短形式的工具包 10 Tactics —— 为权利倡导者提供原创和巧妙的方式来吸引公众注意力、并达成事业目标。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、无线电
   </strong>
  </p>
  <p class="graf graf--p">
   无线电由于其低成本，易于使用和近乎无处不在而成为替代媒体的重要形式。
  </p>
  <p class="graf graf--p">
   例如，在20世纪70年代早期的澳大利亚，一个新的替代无线电部门诞生，由那些
   <strong class="markup--strong markup--p-strong">
    被排除在两个国家广播系统之外的人
   </strong>
   创建。在美国，第一个由听众支持的独立电台 KPFA 始于1949年，
   <strong class="markup--strong markup--p-strong">
    旨在提供真正自由的声音，不受主流电台商业利益的束缚
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他们的内容范围很广泛；虽然一些电视台的主要目标明确是政治性的和激进的，但其他电台则试图播放他们认为被排除在主流电台之外的音乐。
  </p>
  <p class="graf graf--p">
   替代无线电通常（尽管不总是）采用社区广播的形式，通常被理解为
   <strong class="markup--strong markup--p-strong">
    参与性、开放性、非营利性，并由社区制作
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些无线电台可能合法也可能非法播放，作为海盗电台。
  </p>
  <p class="graf graf--p">
   替代无线电是一种
   <strong class="markup--strong markup--p-strong">
    全球现象
   </strong>
   。社区和替代无线电活动的例子包括TilosRádió（匈牙利），Missinipi 广播公司（加拿大），Pacifica Radio 和 Prometheus Radio Project（均在美国）、以及 Radio Sagarmatha（尼泊尔）。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、视频和电影
   </strong>
  </p>
  <p class="graf graf--p">
   另类电影和视频通常是在主流电影和视频行业所覆盖范围之外制作的，
   <strong class="markup--strong markup--p-strong">
    其特点是主流产品中很少见的内容和/或风格
   </strong>
   。但是，它们的特定类型、内容和形式差异很大。
  </p>
  <p class="graf graf--p">
   它们通常在非营利性组织环境中产生，例如 视频艺术集体（举例 Videotage，洛杉矶电影制片人合作社）或草根社会正义组织（举例 Line Break，CINEP-研究和大众教育中心）。
  </p>
  <p class="graf graf--p">
   💡
   <strong class="markup--strong markup--p-strong">
    参与式视频项目中被边缘化群体或资源不足的群体通过视频讲述自己的故事，参与者都有相关的传媒技能，并增强相关人员的能力，传播主流媒体中人们通常看不到的表现形式，最重要的是，挑战现有的权力关系
   </strong>
   。[社会组织讲述故事的重要性，详见这里的分析《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%88%A9%E7%94%A8%E8%AE%B2%E8%BF%B0%E6%95%85%E4%BA%8B%E6%8E%A8%E5%8A%A8%E7%A4%BE%E4%BC%9A%E5%8F%98%E9%9D%A9%E7%9A%84%E7%A7%98%E8%AF%80%EF%BC%9A%E6%9B%B4%E5%A5%BD%E7%9A%84%E5%90%88%E4%BD%9C%E4%BC%99/" href="https://www.iyouport.org/%E5%88%A9%E7%94%A8%E8%AE%B2%E8%BF%B0%E6%95%85%E4%BA%8B%E6%8E%A8%E5%8A%A8%E7%A4%BE%E4%BC%9A%E5%8F%98%E9%9D%A9%E7%9A%84%E7%A7%98%E8%AF%80%EF%BC%9A%E6%9B%B4%E5%A5%BD%E7%9A%84%E5%90%88%E4%BD%9C%E4%BC%99/" rel="noopener noreferrer" target="_blank">
    利用讲述故事推动社会变革的秘诀
   </a>
   》]
  </p>
  <p class="graf graf--p">
   美国的另类电影在20世纪30年代的 The Film &amp; Photo League 组织中很明显，通过社会纪录片和新闻片的编辑引起社会对工会和阶级问题的关注。[Leshne, C (2006). “The Film &amp; Photo League of San Francisco”.
   <em class="markup--em markup--p-em">
    Film History
   </em>
   .
   <strong class="markup--strong markup--p-strong">
    18
   </strong>
   (4): 361.
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Digital_object_identifier" href="https://en.wikipedia.org/wiki/Digital_object_identifier" rel="noopener noreferrer" target="_blank" title="Digital object identifier">
    doi
   </a>
   :
   <a class="markup--anchor markup--p-anchor" data-href="https://doi.org/10.2979%2FFIL.2006.18.4.361" href="https://doi.org/10.2979%2FFIL.2006.18.4.361" rel="nofollow noopener noreferrer" target="_blank">
    10.2979/FIL.2006.18.4.361
   </a>
   .]
  </p>
  <p class="graf graf--p">
   随着技术的普及，激进的视频制作在80年代达到了顶峰。公共电视提供了广播渠道，通常是朋克和嘻哈影响的激进派文化批评。
  </p>
  <p class="graf graf--p">
   例如，Deep Dish TV是一个电视网络，旨在通过公共电视提供接入基层组织的媒体，放大被边缘化的观点、纠正误传。如今，便携式的录制技术和互联网给了全球参与替代视频内容制作的更多机会，并促成了更加高效的流通和内容消费。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、互联网
   </strong>
  </p>
  <p class="graf graf--p">
   随着数字技术日益增加的重要性，出现了关于数字媒体适应替代媒体和主流媒体之间二分法的问题。
  </p>
  <p class="graf graf--p">
   首先，博客、Facebook、Twitter 和其他类似网站，虽然不一定被创建为信息媒体，但越来越多地被用来传播新闻和信息，理论上都可以充当替代媒体，因为它们
   <strong class="markup--strong markup--p-strong">
    允许普通公民绕过传统主流媒体的守门人高墙，并分享这些公民认为重要的信息和观点
   </strong>
   。
  </p>
  <p class="graf graf--p">
   其中最重要的是：⚠️避免集中化网络平台监控审查，做好安全防御工作。这点详见我们的介绍《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/?p=3164" href="https://www.iyouport.org/?p=3164" rel="noopener noreferrer" target="_blank">
    大清洗：反侦察的第一步
   </a>
   》、《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/?p=3336" href="https://www.iyouport.org/?p=3336" rel="noopener noreferrer" target="_blank">
    角色扮演：行动者刚需
   </a>
   》，以及“Knowledge Node”栏目中介绍的诸多安全知识。
  </p>
  <p class="graf graf--p">
   其次，互联网通过人际网络的培养、社会变革的集体行动，并使信息更易于获取，为动员提供了另一个空间。通常，在具有异议、持不同政见或非传统观点的人群中，互联网平台允许
   <strong class="markup--strong markup--p-strong">
    创建新的替代性社区，可以为那些通常被主流媒体边缘化的人提供发言权
   </strong>
   。
  </p>
  <p class="graf graf--p">
   此外，互联网还导致了另一种编程形式，使专业人员和业余爱好者都能够
   <strong class="markup--strong markup--p-strong">
    颠覆或逃避对信息和信息技术的开放获取形成阻碍的商业和政治限制
   </strong>
   。
  </p>
  <p class="graf graf--p">
   最后，互联网还孕育了一种创造和传播知识的新方式 —— 共同知识 —— 这与自上而下的灌输方式不同。它寻求并鼓励多个用户的参与，促进协作知识生产的形式。维基百科是这一类型的一个很好的例子。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、街头艺术
   </strong>
  </p>
  <p class="graf graf--p">
   街头艺术通常被认为是游击艺术，不受正规艺术界的限制。
  </p>
  <p class="graf graf--p">
   街头艺术以涂鸦、模板、壁画和印刷品的形式占用或改变公共空间，
   <strong class="markup--strong markup--p-strong">
    作为抗议和社会评论的手段。
   </strong>
  </p>
  <p class="graf graf--p">
   街头艺术作为另一种形式的重要方面是它融合了美学和社会参与、城市空间的使用、以及与艺术所在地区的社会景观的互动。
  </p>
  <p class="graf graf--p">
   街头艺术运动在20世纪80年代作为一种与高级艺术和商业场所截然不同的艺术形式而受到欢迎，之后，随着人气的增长，一些街头艺术家从街道的其他场所搬到了画廊和博物馆展览。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    巴黎、布宜诺斯艾利斯和圣保罗等城市通过艺术家集体行动和竞赛的形式将街头艺术作为合法的替代媒体突显出来，引起人们对其他声音的关注
   </strong>
   。[
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20151027052728/http://woostercollective.com/" href="https://web.archive.org/web/20151027052728/http://woostercollective.com/" rel="nofollow noopener noreferrer" target="_blank">
    “Wooster Collective”
   </a>
   .
   <em class="markup--em markup--p-em">
    Wooster Collective
   </em>
   . Archived from
   <a class="markup--anchor markup--p-anchor" data-href="http://www.woostercollective.com/" href="http://www.woostercollective.com/" rel="nofollow noopener noreferrer" target="_blank">
    the original
   </a>
   on 2015–10–27. Retrieved 2015–10–27.]
  </p>
  <p class="graf graf--p">
   艺术家和粉丝在互联网上分享来自世界各地的街头艺术图片，极大地影响了街头艺术这一替代媒体形式的发育。 Streetsy.com和WoosterCollective.com等网站是最受欢迎的街头艺术分享网站。
  </p>
  <p class="graf graf--p">
   你可以看到
   <strong class="markup--strong markup--p-strong">
    这是极具感召力的直接行动模式；最重要的是，它所激发的公民参与和由此形成的联合力量
   </strong>
   。这些都是泼墨所远不能及的。泼墨等个人行为艺术并非无效，但如果不能引发联合行动，只能成为一个会迅速过时的符号。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    6、表演
   </strong>
  </p>
  <p class="graf graf--p">
   表演也是媒体？当然是。
  </p>
  <p class="graf graf--p">
   作为替代媒体的表演使用戏剧、歌曲和表演艺术作为吸引观众和
   <strong class="markup--strong markup--p-strong">
    促进社会议程的手段
   </strong>
   。
  </p>
  <p class="graf graf--p">
   表演艺术是一种前卫艺术形式，通常使用现场表演来挑战传统形式的视觉艺术。它的运作是“戏剧的对立面，挑战正统的艺术形式和文化规范”。
  </p>
  <p class="graf graf--p">
   从达达派和超现实主义到后极简主义的社会和文化运动中，
   <strong class="markup--strong markup--p-strong">
    表演艺术都发挥着重要的作用，反映了当时的政治环境
   </strong>
   。
  </p>
  <p class="graf graf--p">
   虽然表演艺术通常被归为高级艺术，但街头戏剧通常以草根方式运作，利用当地社区进行表演或对话。💡
   <strong class="markup--strong markup--p-strong">
    它可以作为一种抗议形式而存在
   </strong>
   ，就像致力于通过实验剧场改变社会权力等级的 Living Theatre 一样。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    7、音乐
   </strong>
  </p>
  <p class="graf graf--p">
   某些类型的音乐和音乐表演也可以归类为替代媒体。
  </p>
  <p class="graf graf--p">
   独立音乐是与商业唱片公司分开制作的音乐。采取的是颠覆性的声音和歌词。
  </p>
  <p class="graf graf--p">
   David Hesmondhalgh 教授将独立音乐的另类性质描述为“务实的后朋克公司网络”，对主要唱片公司所青睐的文化生产商业组织构成了重大的挑战。
  </p>
  <p class="graf graf--p">
   用艺术反抗 —— IYP 推荐一系列抵制监制保护隐私的歌曲，即 替代媒体，传播这些歌曲，将他们设为手机铃声，这就是一场权利运动。详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%B8%80%E8%B5%B7%E5%90%AC%E6%AD%8C/" href="https://www.iyouport.org/%E4%B8%80%E8%B5%B7%E5%90%AC%E6%AD%8C/" rel="noopener noreferrer" target="_blank">
    一起听歌
   </a>
   》。
  </p>
  <p class="graf graf--p">
   💡想想看，中国在上述这些方面都拥有哪些创作空间？💡
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-3617 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6.jpg?resize=1100%2C619&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6.jpg?resize=300%2C169&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6.jpg?resize=768%2C432&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6.jpg?resize=1024%2C576&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6.jpg?resize=1100%2C619&amp;ssl=1 1100w" data-recalc-dims="1" height="619" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/07/6.jpg?resize=1100%2C619&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <h4 class="graf graf--h4">
   <strong>
    —— 类型 ——
   </strong>
  </h4>
  <p class="graf graf--p">
   通信学者 Leah Lievrouw 主要关注新媒体在替代媒体项目中日益重要的作用，他确定了
   <strong class="markup--strong markup--p-strong">
    基于当代新媒体的五种类型的替代媒体活动
   </strong>
   ：文化干扰，替代计算、参与式新闻、中介动员和共同知识。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     文化干扰
    </strong>
    —— 通常试图批评娱乐、广告和艺术等流行文化。它倾向于评论企业资本主义和消费主义问题，并寻求提供政治评论。文化干扰文本的特征包括图像、视频、声音或文本的挪用或重新利用，并且在某种意义上它们具有讽刺意味或讽刺性。比如如今的互联网meme 是为最简单的文化干扰。
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     替代计算
    </strong>
    —— 涉及信息和通信技术的物质基础设施。它试图批评和重新配置系统，旨在颠覆或规避对信息和信息技术的开放获取产生阻碍作用的商业和政治限制。比如开源运动、以及洪流。海盗党崛起。
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     参与式新闻
    </strong>
    —— 是指基于网络的关键或激进新闻来源，无论是在线新闻服务还是博客。这些新闻的替代渠道通常采用公民新闻的哲学，并将自己视为主流新闻和观点的替代品。参与式新闻项目可涵盖被报道不足的群体和社会问题。在此一些替代媒体项目的作者和读者具有相似的能力，因此具有参与性或互动性的特征。参与式新闻的一个例子是著名的 Indymedia。
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     中介动员
    </strong>
    —— 涉及通过使用 Facebook 或 YouTube 等新媒体工具和平台动员或组织社交活动、身份或文化项目的沟通实践。这种类型的特征包括人际网络的培养、社会变革的集体行动，并使信息易于获取。
   </li>
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     共同知识
    </strong>
    —— 是指为传统的自上而下创造和传播知识的形式提供替代方案的项目。它寻求并鼓励多个用户的参与，促进协作知识生产。例如维基百科。
   </li>
  </ul>
  <p class="graf graf--p">
   从类型的角度思考当前形式的替代媒体，不仅可以让我们识别某些交流方式的特征和惯例，而且还可以“让人们能够恰当地表达自己，实现各种目的或意图”。
  </p>
  <p class="graf graf--p">
   换句话说，💡
   <strong class="markup--strong markup--p-strong">
    通过本文，您可以开始了解替代新媒体项目的创作者和参与者如何积极地塑造他们的沟通实践。在您的国家和社区实现它们
   </strong>
   。
  </p>
  <p>
   <img alt="" class="aligncenter wp-image-3618 jetpack-lazy-image" data-lazy-sizes="(max-width: 700px) 100vw, 700px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/最后.png?resize=700%2C680&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/最后.png?w=1148&amp;ssl=1 1148w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/最后.png?resize=300%2C292&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/最后.png?resize=768%2C747&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/最后.png?resize=1024%2C995&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/最后.png?resize=1100%2C1069&amp;ssl=1 1100w" data-recalc-dims="1" height="680" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/07/最后.png?resize=700%2C680&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="700"/>

  <h4>
   <strong>
    —— 审美 ——
   </strong>
  </h4>
  <p class="graf graf--p">
   💡与实验和创新的生产和协作模式相结合，
   <strong class="markup--strong markup--p-strong">
    替代媒体中的美学可以成为用来颠覆主导权力的政治工具。
   </strong>
  </p>
  <p class="graf graf--p">
   像许多替代媒体制作者一样，学者 Crispin Sartwell 将政治视为一种美学环境。因此，这些艺术政治系统不仅使用美学作为获得权力的工具，而且通过所有媒体中的审美形式产生。
  </p>
  <p class="graf graf--p">
   因此，替代媒体寻求新的艺术、非传统或前卫的手段来代表其内容的情况并不少见。在这种情况下，美学的使用允许替代媒体以重新调整、重新协商或暴露其内部政治的方式，来处理其他平庸的内容。
  </p>
  <p class="graf graf--p">
   学者们将 Avante-garde 艺术运动联系起来，将
   <strong class="markup--strong markup--p-strong">
    另类美学作为一种政治工具
   </strong>
   。
  </p>
  <p class="graf graf--p">
   像未来主义、达达和情境主义这样的运动看起来像是挑战关于“艺术是什么”的陈规定论，
   <strong class="markup--strong markup--p-strong">
    以便从根本上改变公共和政治意识形态的正式规则。即 💡拒绝那些由支配性阶级强制执行的风格和审美的逻辑、理由和规则。
   </strong>
  </p>
  <p class="graf graf--p">
   一些替代媒体制造者希望通过拒绝主流的视觉教条，从而彻底摆脱主导阶级的令人窒息的限制，其他的则通过模仿、嘲弄和讽刺来颠覆主流语言和信息。
  </p>
  <p class="graf graf--p">
   强调观众参与的前卫运动包括未来主义、达达主义、超现实主义、情境主义、波普艺术、新混合主义和受压迫者剧场。
  </p>
  <p class="graf graf--p">
   通过邀请观众参与媒体的创作，
   <strong class="markup--strong markup--p-strong">
    合作者希望通过拥抱民主的生产方式来颠覆或批评社会中的等级结构（资本主义、象牙塔）
   </strong>
   。
  </p>
  <p class="graf graf--p">
   好了。希望您能通过本文了解到反抗意味着什么、如何寻找社会空间以发挥反抗的能力、如何在身边开启民主实践，祝福中国社会拥有美好的未来。⚪️
  </p>
