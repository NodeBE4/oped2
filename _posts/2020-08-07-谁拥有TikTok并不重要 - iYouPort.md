  <p class="graf graf--p">
   【按】您能看到，本文的存在超出了我们每日的预定发布数量。因为我们认为它很重要 —— 并非由于所谓的热点（TikTok的确是热点），但一向对网络热点毫无兴趣的 IYP 实际上关心的是这件事中暴露的思考方式。
  </p>
  <p class="graf graf--p">
   中文舆论对封锁和收购 TikTok 的广泛兴奋和支持程度透露，依旧有很多人没能理解数据的民族主义地缘政治。事实上我们在2年前对谷歌重返中国的传闻进行的分析中已经做出过相关的解释，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%af%b4-google-%e6%90%9c%e7%b4%a2%e8%83%bd%e5%90%a6%e6%88%90%e5%8a%9f%e5%9b%9e%e5%bd%92%e4%b8%ad%e5%9b%bd%e5%b8%82%e5%9c%ba%e6%a0%b9%e6%9c%ac%e4%b8%8d%e9%87%8d%e8%a6%81/" href="https://www.iyouport.org/%e4%b8%ba%e4%bb%80%e4%b9%88%e8%af%b4-google-%e6%90%9c%e7%b4%a2%e8%83%bd%e5%90%a6%e6%88%90%e5%8a%9f%e5%9b%9e%e5%bd%92%e4%b8%ad%e5%9b%bd%e5%b8%82%e5%9c%ba%e6%a0%b9%e6%9c%ac%e4%b8%8d%e9%87%8d%e8%a6%81/" rel="noopener noreferrer" target="_blank">
    为什么说 Google 搜索能否成功回归中国市场根本不重要？
   </a>
   》。
  </p>
  <p class="graf graf--p">
   然而，
   <strong class="markup--strong markup--p-strong">
    人们再一次被绕进去了，被绕进了 matrix 语话的浆糊里，而忘记了基本的思考方式
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    封锁或收购能制止 TikTok 这类监视资本家的数据交易和审查吗？—— 不；
   </li>
   <li class="graf graf--li">
    你能在任何技术的世界里竖起 **绝对的** 边界墙吗？—— 不；
   </li>
   <li class="graf graf--li">
    美国人的数据真的能做到都放在美国吗，或者中国人的数据都放在中国？—— 不；
   </li>
   <li class="graf graf--li">
    那么数据政治之战的结果是谁赢了？—— 民族主义；
   </li>
   <li class="graf graf--li">
    谁会从民族主义中获利？……
   </li>
  </ul>
  <p class="graf graf--p">
   这句话更简单地说是，我们应该追求准确识别任何当权者操纵的 matrix，并以我们这些互联网公民自己的利益为出发点进行思考。
  </p>
  <p class="graf graf--p">
   📌
   <strong class="markup--strong markup--p-strong">
    这里是2篇文章，它们是同一个主题，详细解释了上述；只是角度不同
   </strong>
   。
   <strong class="markup--strong markup--p-strong">
    请注意，这是给美国读者的分析。而我们需要在中国有更多基于公民权益的思考方式。而不是跟着当权者的宣传跑
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些分析在美国社会一定程度上已经形成共识。而我们的翻译依旧在冒着足够大的风险，因为我们明白，这些道理在中文舆论场可能很不受待见 ……
  </p>

   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%91%86%e8%84%b1%e6%93%8d%e7%ba%b5%e5%92%8c%e5%ae%a3%e4%bc%a0%ef%bc%9a28%e4%b8%aa%e9%95%bf%e6%9c%9f%e6%8a%80%e5%b7%a7/" href="https://www.iyouport.org/%e6%91%86%e8%84%b1%e6%93%8d%e7%ba%b5%e5%92%8c%e5%ae%a3%e4%bc%a0%ef%bc%9a28%e4%b8%aa%e9%95%bf%e6%9c%9f%e6%8a%80%e5%b7%a7/" rel="noopener noreferrer" target="_blank">
     摆脱操纵和宣传：28个长期技巧

   </li>

  <p class="graf graf--p">
   总之，您不应该使用 TikTok，就如您应该 deleteFacebook 一样。这就是公民权益。

  <p class="graf graf--p">
   当微软上
   <a class="markup--anchor markup--p-anchor" data-href="https://www.gizmodo.com.au/2020/08/microsofts-takeover-would-be-a-win-for-tiktok-and-tech-giants-not-users/" href="https://www.gizmodo.com.au/2020/08/microsofts-takeover-would-be-a-win-for-tiktok-and-tech-giants-not-users/" rel="noopener noreferrer" target="_blank">
    周
   </a>
   早些时候
   <a class="markup--anchor markup--p-anchor" data-href="https://blogs.microsoft.com/blog/2020/08/02/microsoft-to-continue-discussions-on-potential-tiktok-purchase-in-the-united-states/" href="https://blogs.microsoft.com/blog/2020/08/02/microsoft-to-continue-discussions-on-potential-tiktok-purchase-in-the-united-states/" rel="noopener noreferrer" target="_blank">
    正式成为
   </a>
   潜在收购 TikTok 的领先者时，全球的技术评论家发现自己面临着无尽的问题，似乎没人能回答。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.gizmodo.com.au/2020/08/microsofts-takeover-would-be-a-win-for-tiktok-and-tech-giants-not-users/" href="https://www.gizmodo.com.au/2020/08/microsofts-takeover-would-be-a-win-for-tiktok-and-tech-giants-not-users/" rel="noopener noreferrer" target="_blank">
    为什么
   </a>
   像微软这样的公司会对 TikTok 感兴趣呢？是否真的要求美国财政部从这一有望成为数十亿美元的交易中分一杯羹（即使根本不可能）？买断难道就能解决 TikTok相关的任何所谓隐私问题吗，还是直接
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2020/8/4/21354255/microsoft-tiktok-trump-ban-bytedance-china-us-canada" href="https://www.theverge.com/2020/8/4/21354255/microsoft-tiktok-trump-ban-bytedance-china-us-canada" rel="noopener noreferrer" target="_blank">
    爆发出
   </a>
   一连串新的问题？……
  </p>
  <p class="graf graf--p">
   最后一个问题已经困扰我好一阵子了。
  </p>
  <p class="graf graf--p">
   虽然我对地缘政治的细枝末节有点生疏，但我已经花了一年多的时间密切关注 TikTok 不断发展的计划，该计划的目标是追踪和锁定美国和国外的青少年。
  </p>
  <p class="graf graf--p">
   随着时间的推移，变得非常非常清楚的是，虽然说，谷歌和 Facebook 与 TikTok 最终都受到彼此国家监管机构的排斥，但
   <strong class="markup--strong markup--p-strong">
    相同的是，这些公司都是监视资本主义体制的驱动机器，它们的盈利模式都是靠出卖我们每个人的数据以获得高额利润。
   </strong>
  </p>
  <p class="graf graf--p">
   目前的数字经济在一定程度上依赖于全球联系，
   <strong class="markup--strong markup--p-strong">
    这些联系的建立远比任何禁令或收购所能触及的更深
   </strong>
   。
  </p>
  <p class="graf graf--p">
   或者，说得更直白一点。
   <strong class="markup--strong markup--p-strong">
    如果特朗普真的关心让我们 “干净的美国手机的数据” 不被那个肮脏的、坏透了的共产主义对手中国抓到的话，那么微软买下 TikTok 是没有用的，我们的数据正在不断地从美国公司转移到中国的服务器上，不管 TikTok 的主人是谁
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一般来说，当我们谈论某个社交网络所做的无节制的数据搜刮时，我们谈论的是以定向广告的名义进行的数据搜刮。这就是应该发生的事。
  </p>
  <p class="graf graf--p">
   但是。由于部分广告生态系统的荒谬扩张， 事实证明，法律制度在解决蔓延的问题上是远远不够的，而且许多广告技术参与者普遍持不以为然的态度，
   <strong class="markup--strong markup--p-strong">
    我们已经看到许多此类数据已重新被用于政府监视
   </strong>
   。【这是指在美国，而这点上任何国家都一样】
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    对此的具体解释 见
    <a class="markup--anchor markup--li-anchor" data-href="https://start.me/p/RMQrDD/iyp-2" href="https://start.me/p/RMQrDD/iyp-2" rel="noopener noreferrer" target="_blank">
     列表-2 “公私监控伙伴关系”
    </a>
   </li>
  </ul>
  <p class="graf graf--p">
   但正如多起诉讼（以及 Mike Pompeo ）向我们展示的那样，问题并不在于被囤积的数据，而在于这些数据的存储地点。12月，中国当局正式颁布了一项立法，允许政府在一定程度上不受限制地访问存储在总部位于中国的服务器上或通过中国网络传输的数据。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    数据本地存储就是所谓的互联网主权的典型特征，
    <strong class="markup--strong markup--li-strong">
     这种 “主权” 声称正在全世界流行
    </strong>
    ，见《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e4%b8%bb%e6%9d%83%e7%9a%84%e5%85%a8%e7%90%83%e5%b4%9b%e8%b5%b7%ef%bc%9a%e8%bf%99%e6%84%8f%e5%91%b3%e7%9d%80%e4%bb%80%e4%b9%88%ef%bc%9f/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e4%b8%bb%e6%9d%83%e7%9a%84%e5%85%a8%e7%90%83%e5%b4%9b%e8%b5%b7%ef%bc%9a%e8%bf%99%e6%84%8f%e5%91%b3%e7%9d%80%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     互联网主权的全球崛起：这意味着什么？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   尽管 TikTok 反复
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/technology/2020/07/13/tiktok-privacy/" href="https://www.washingtonpost.com/technology/2020/07/13/tiktok-privacy/" rel="noopener noreferrer" target="_blank">
    坚持
   </a>
   要在美国本地和新加坡的服务器上存储美国用户的数据，并且，尽管完全缺乏任何证据表明任何美国用户的数据存储在中国大陆，但是专家们还是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/technology/2019/12/31/us-army-bans-tiktok-military-devices-signaling-growing-concern-about-apps-chinese-roots/?itid=lk_inline_manual_46" href="https://www.washingtonpost.com/technology/2019/12/31/us-army-bans-tiktok-military-devices-signaling-growing-concern-about-apps-chinese-roots/?itid=lk_inline_manual_46" rel="noopener noreferrer" target="_blank">
    指出
   </a>
   该公司的 Bytedance 母公总部设在中国，足以作为值得警惕的理由。
  </p>
  <p class="graf graf--p">
   单从这一点来看，我就能明白，为什么微软的收购是有意义的：
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    用一家总部设在西雅图的公司取代一家总部设在北京的母公司，可以平息人们对中国服务器的偏执
   </strong>
   。
  </p>
  <p class="graf graf--p">
   📌 但这只是一个幻想，只有当数据的古怪世界存在任何该死的意义时它才会有效。
  </p>
  <p class="graf graf--p">
   当一个应用程序将你的数据从你的手机传送到 TikTok（或一些广告商的）手里时，它要走的弯路几乎是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.google.com/url?sa=i&amp;url=https%3A%2F%2Flumapartners.com%2Fcontent%2Flumascapes%2Fmobile-lumascape%2F&amp;psig=AOvVaw1rh_3Ak96negoIDiIKOkLg&amp;ust=1596726849855000&amp;source=images&amp;cd=vfe&amp;ved=0CA0QjhxqFwoTCNCYrLathOsCFQAAAAAdAAAAABAD" href="https://www.google.com/url?sa=i&amp;url=https%3A%2F%2Flumapartners.com%2Fcontent%2Flumascapes%2Fmobile-lumascape%2F&amp;psig=AOvVaw1rh_3Ak96negoIDiIKOkLg&amp;ust=1596726849855000&amp;source=images&amp;cd=vfe&amp;ved=0CA0QjhxqFwoTCNCYrLathOsCFQAAAAAdAAAAABAD" rel="noopener noreferrer" target="_blank">
    无穷无尽的
   </a>
   —— 而且，如果你的数据碰巧在途中某个地方跨越了国际边界，那么这些合作关系就会变得
   <a class="markup--anchor markup--p-anchor" data-href="https://www.aarki.com/blog/the-state-of-programmatic-advertising-in-china" href="https://www.aarki.com/blog/the-state-of-programmatic-advertising-in-china" rel="noopener noreferrer" target="_blank">
    更加纠结
   </a>
   。
  </p>
  <p class="graf graf--p">
   您可以通过自己检查下应用程序的后端来部分理解这一点，但这实际上只说明了一半的故事。【文章用鼠王 “rat king” 来形容这种关系；鼠王是传说中若干只老鼠的尾巴连在一起而形成的古怪生物。您可以脑补那个状况……】
  </p>
  <p class="graf graf--p">
   要解开剩下的部分，具有讽刺意味的是，需要两样东西，而这两样东西都是当前美国政府无法满足的：商业洞察力和
   <a class="markup--anchor markup--p-anchor" data-href="https://edition.cnn.com/videos/politics/2020/08/04/swan-intv-axios-trump-coronavirus-deaths-newday-vpx.cnn" href="https://edition.cnn.com/videos/politics/2020/08/04/swan-intv-axios-trump-coronavirus-deaths-newday-vpx.cnn" rel="noopener noreferrer" target="_blank">
    图表
   </a>
   。
  </p>
  <p class="graf graf--p">
   为了说明这件事，让我们先假设中国的官员们迫不及待地想从任何应用程序中获取任何美国人的数据，并且不惜为此窃听他们国家境内的任何数据中介公司。
  </p>

    Clean Network
   </a>
   的一部分，据其声称为了 “确保美国人的数据不受中共监控国家的影响”，以及其他外国实体的影响。现在您明白了，这根本做不到。
  </p>
  <p class="graf graf--p">
   在过去的几个月里，我因为我的立场 —— 即 TikTok 不是唯一的问题 —— 而受到了很多的指责，但我坚持自己的观点。
   <strong class="markup--strong markup--p-strong">
    禁止 TikTok（或将其转移到美国所有权下）相当于将一栋燃烧的摩天大楼卖给新的业主，并期望以此来灭火。
   </strong>
  </p>
  <p class="graf graf--p">
   它做不到。
  </p>
  <p class="graf graf--p">
   当然，所有这些都是基于这样的论点，即 “美国人的隐私是特朗普政府希望禁止 TikTok 或任何其他与中国有关系的产品的核心”。可惜并非如此。
  </p>
  <p class="graf graf--p">
   即使是这样，您现在也可以明白了：对 TikTok 的禁令将对美国人的个人数据产生几乎为零的影响。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://gizmodo.com/it-doesn-t-matter-who-owns-tiktok-1844595163" href="https://gizmodo.com/it-doesn-t-matter-who-owns-tiktok-1844595163" rel="noopener noreferrer" target="_blank">
    It Doesn’t Matter Who Owns TikTok
   </a>
  </p>
  <p class="graf graf--p">
   下面是 The Intercept 的文章。
  </p>

  <p class="graf graf--p">
   国务院对 “净网” 有一个新的愿景 —— 它的意思是一个没有中国的互联网。
  </p>
  <p class="graf graf--p">
   这个新的民族专属网络声称 “是特朗普政府为保护我们公民的隐私和公司最敏感的信息而采取的全面方法”，通过确保中国无法利用美国及其盟友多年来从事的技术制造一连串的颠覆和暴力。
  </p>
  <p class="graf graf--p">
   作为一份政策文件，它是无稽之谈；但作为一份道德文件，一份编纂的伪善文件，它的主旨是非常清楚的：如果要建立一个跨越整个世界的监控国家，那最好是美国制造。
  </p>
  <p class="graf graf--p">
   国务卿 Mike Pompeo 的一
   <a class="markup--anchor markup--p-anchor" data-href="https://www.state.gov/announcing-the-expansion-of-the-clean-network-to-safeguard-americas-assets/" href="https://www.state.gov/announcing-the-expansion-of-the-clean-network-to-safeguard-americas-assets/" rel="noopener noreferrer" target="_blank">
    份声明
   </a>
   中包括了一个五方面的计划，以击退红色中国对你的数据进行窃取和滥用的企图：
  </p>
  <p class="graf graf--p">
   努力阻止中国手机运营商进入美国市场，将侵犯隐私的中国应用从美国应用商店中踢出，将美国应用从中国公司运营的应用商店中删除，将美国公民的数据从 “我们的外国对手可以访问的” 中国云服务器中删除，并确保在各大洲之间传输互联网信号的海底电缆不会被诡异的中国情报部门秘密窃听。
  </p>
  <p class="graf graf--p">
   真正的问题，甚至比国务院的指令如何能够实际完成这些工作更重要的是（如上文所示，无法实际完成）：
   <strong class="markup--strong markup--p-strong">
    为什么世界上有人会认真对待这一倡议？这个
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     被美国间谍机构渗透了几十年的网络
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    怎么可能被认为是干净的？
   </strong>
  </p>
  <p class="graf graf--p">
   美国政府有胆量谴责 “威胁我们的隐私、扩散病毒、传播宣传和虚假信息的应用程序”，这实在是略显惊人，啼笑皆非。美国无一例外地参与了上述每一种做法，并违反了 “干净互联网” 的每一点美德。我们从哪里下手？
  </p>
  <p class="graf graf--p">
   提供9–11事件后的NSA电话监控项目，AT&amp;T 和 Verizon 等大型电信运营商与政府密切合作，提供了数亿条电话和短信的敏感数据，该项目直到去年才被关闭 —— 仅仅是据称的关闭，实际上并没有关闭。
  </p>
  <p class="graf graf--p">
   尽管对源自中国的社交网络 TikTok 的批评者很快就会指出中国科技公司对其政府的数据共享义务，但我们美国人完全不需要到世界的另一端去寻找这种不透明的安排。
  </p>
  <p class="graf graf--p">
   正如《纽约时报》在终止NSA监视项目的文章中指出的那样，“从2006年开始，外国情报监视法庭开始发布秘密命令，要求这些公司参与，其依据是对《爱国者法案》第215条的新颖解释，即 只要声称反恐，F.B.I.就可以获得 ‘相关’ 的任何数据。”
  </p>
  <p class="graf graf--p">
   长期以来，人们担心中国华为制造的预先篡改的通信设备可能会入侵美国的网络，但当你回忆起NSA如何
   <a class="markup--anchor markup--p-anchor" data-href="https://www.nytimes.com/2014/03/23/world/asia/nsa-breached-chinese-servers-seen-as-spy-peril.html" href="https://www.nytimes.com/2014/03/23/world/asia/nsa-breached-chinese-servers-seen-as-spy-peril.html" rel="noopener noreferrer" target="_blank">
    通过入侵华为本身
   </a>
   、同时出口
   <a class="markup--anchor markup--p-anchor" data-href="https://arstechnica.com/tech-policy/2014/05/photos-of-an-nsa-upgrade-factory-show-cisco-router-getting-implant/" href="https://arstechnica.com/tech-policy/2014/05/photos-of-an-nsa-upgrade-factory-show-cisco-router-getting-implant/" rel="noopener noreferrer" target="_blank">
    美国公司思科的预先篡改网络设备
   </a>
   时，就开始觉得有点道德上的空洞了。心理学家可能会把美国在这方面的担忧描述为 “投射”。
  </p>
  <p class="graf graf--p">
   不止中国，在向其他国家兜售侵犯隐私的应用程序时，我们也同样处于弱势，而这个行业是美国引以为豪的先锋。
  </p>
  <p class="graf graf--p">
   几乎没有任何美国公民的日常行动、习惯、品味和欲望不会
   <strong class="markup--strong markup--p-strong">
    被一大群盈利公司昼夜监视
   </strong>
   ，而这些公司的名字你永远都不会知道，他们的利益也很少分享。
  </p>
  <p class="graf graf--p">
   在最近一次关于美国科技公司垄断行为的国会听证会上，Facebook 首席执行官马克-扎克伯格（他表示自己的公司代表着 “美国价值观”）
   <a class="markup--anchor markup--p-anchor" data-href="https://techcrunch.com/2020/07/29/zuckerberg-unconvincingly-feigns-ignorance-of-data-sucking-vpn-scandal/" href="https://techcrunch.com/2020/07/29/zuckerberg-unconvincingly-feigns-ignorance-of-data-sucking-vpn-scandal/" rel="noopener noreferrer" target="_blank">
    被问及
   </a>
   该公司在智能手机上部署的一个
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%97%b4%e8%b0%8dvpn/" href="https://www.iyouport.org/%e9%97%b4%e8%b0%8dvpn/" rel="noopener noreferrer" target="_blank">
    间谍VPN以监视儿童和成年人的行为
   </a>
   。扎克伯格一开始否认他曾听说过这一事件，即使他在宣誓后告诉国会，该公司没有使用 cookies 来追踪私人信息（
   <a class="markup--anchor markup--p-anchor" data-href="https://twitter.com/ashk4n/status/1288616328874487809" href="https://twitter.com/ashk4n/status/1288616328874487809" rel="noopener noreferrer" target="_blank">
    它确实使用了
   </a>
   ）。蓬佩奥对 “中国监控国家” 的谴责，听起来不过是监控国家保护主义。
  </p>
  <p class="graf graf--p">
   谷歌的安卓手机预装了一长串永远开机、永远监听、永远追踪的功能，这些功能悄悄地监视着每个人的智能手机使用习惯和实时位置：
   <strong class="markup--strong markup--p-strong">
    数据越来越多地被移交给警方，并且一如既往地被一系列令人瞠目结舌的公司用来做定位广告。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这里并没有被认为是 “干净” 的东西
   </strong>
   ，包括你的手机现在向你推荐的 Dove Soap 肥皂精油的惊人清洁力，它也同样是被 Facebook、Google 和无数营销人员持续追踪的结果。【《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e4%bd%a0%e4%b8%8d%e5%81%8f%e6%89%a7%ef%bc%8c%e9%82%a3%e5%b0%b1%e6%98%af%e7%96%af%e4%ba%86/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e4%bd%a0%e4%b8%8d%e5%81%8f%e6%89%a7%ef%bc%8c%e9%82%a3%e5%b0%b1%e6%98%af%e7%96%af%e4%ba%86/" rel="noopener noreferrer" target="_blank">
    “如果你不偏执，那就是疯了”
   </a>
   》】
  </p>
  <p class="graf graf--p">
   Skype 是世界上最受欢迎的和曾经被认为值得信赖的软件之一，但是，为了让国家安全局有效地
   <strong class="markup--strong markup--p-strong">
    监视它所携带的任何私密对话
   </strong>
   ，它的美国所有者臭名昭著地
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2013/jul/11/microsoft-nsa-collaboration-user-data" href="https://www.theguardian.com/world/2013/jul/11/microsoft-nsa-collaboration-user-data" rel="noopener noreferrer" target="_blank">
    破坏了它
   </a>
   。当然，世界上其他国家也肯定感受到了这种 “干净”。
  </p>
  <p class="graf graf--p">
   蓬佩奥呼吁建立一个 “干净的云”，不能 “被我们的外国对手访问”。听听蓬佩奥的宣传吧，为什么其他国家应该转而使用比如说亚马逊的AWS云服务 ——
   <strong class="markup--strong markup--p-strong">
    这个中情局和国家安全局使用的云服务
   </strong>
   —— 会很有意思。
  </p>
  <p class="graf graf--p">
   美国政府应该非常仔细地考虑，是否要告诉世界，
   <strong class="markup--strong markup--p-strong">
    一家公司与情报界的关系和长期合作承诺应该是做生意的基本门槛？
   </strong>
  </p>
  <p class="graf graf--p">
   蓬佩奥决心 “确保连接我国与全球互联网的海底电缆不被中国超大规模地颠覆用于情报收集”，这也许是他最道德沦丧的举措。他承诺说：“我们还将与外国伙伴合作，确保世界各地的海底电缆不会同样遭到破坏。”
  </p>
  <p class="graf graf--p">
   而他没有注意到，几十年来正是我们自己和
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2018/03/01/nsa-global-surveillance-sigint-seniors/" href="https://theintercept.com/2018/03/01/nsa-global-surveillance-sigint-seniors/" rel="noopener noreferrer" target="_blank">
    “五眼” 间谍联盟中的这些 “外国伙伴”
   </a>
   确保了海底电缆遭到破坏。
  </p>
  <p class="graf graf--p">
   斯诺登披露的以 FAIRVIEW、STORMBREW 和 OAKSTAR 等名字命名的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/business/economy/the-nsa-slide-you-havent-seen/2013/07/10/32801426-e8e6-11e2-aa9f-c03a72e2d342_story.html" href="https://www.washingtonpost.com/business/economy/the-nsa-slide-you-havent-seen/2013/07/10/32801426-e8e6-11e2-aa9f-c03a72e2d342_story.html" rel="noopener noreferrer" target="_blank">
    NSA监视项目
   </a>
   ，显示了美国在绝对秘密地监视互联网物理光纤骨干上的庞大投入。
  </p>
  <p class="graf graf--p">
   这种对互联网的做法 即
   <strong class="markup--strong markup--p-strong">
    “照我们说的做，而不是照我们做的做”
   </strong>
   一直存在，但在 “禁止” TikTok 的争论中达到了顶点，TikTok 是一款中国的视频分享应用，算得上是历史上使用最广泛的软件之一。我发现，随便摘取任何一篇国家安全中间派反对 TikTok 的共识，将 “TikTok” 换成 比如说 Facebook，都是一件很有意义的事 —— 毫不违和。
  </p>
  <p class="graf graf--p">
   智囊团的论点是，无论哪家公司被划入国家安全的疯狂中，都要对不诚实的科技公司进行全国性的禁止，因为这些公司会尽可能多地采集数据，并与间谍机构和警方合作，这一点非常成立。
  </p>
  <p class="graf graf--p">
   没有理由相信 TikTok 比 Facebook 或任何其他寡头公司吸纳了更多的数据，其所有者在没有任何有意义的监管的情况下可以对这些信息永远为所欲为。
  </p>
  <p class="graf graf--p">
   把马克·扎克伯格、桑达·皮采和其他硅业高层视为中国秘密、破坏和诡计的有意义的对手，视为民主理想的屏障，这是对任何关注者的侮辱。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国技术专制主义的噩梦，丝毫不能为我们在国内建立的所谓 “简版监控国家” 开脱。
   </strong>
  </p>
  <p class="graf graf--p">
   自从七年前NSA承包商 Edward Snowden 帮助揭露了各种监控计划以来，我们已经知道，来自谷歌、Facebook 和苹果等硅谷巨头公司的信息被定期和系统性地提供给NSA，在许多情况下只有秘密法庭、秘密程序和秘密调查结果知晓。
  </p>
  <p class="graf graf--p">
   要想论证为什么我们不透明的公私国家安全数据漏斗是 “有意义的民主和真正的开放”，需要一个比蓬佩奥更伟大的头脑。
  </p>
  <p class="graf graf--p">
   不那么听话的公司可能会发现自己成为了联邦 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.eff.org/issues/national-security-letters" href="https://www.eff.org/issues/national-security-letters" rel="noopener noreferrer" target="_blank">
    国家安全信函
   </a>
   ” 的接收者，
   <strong class="markup--strong markup--p-strong">
    这些信函不仅强制他们交出所要求的一切数据，而且甚至在监禁的惩罚下禁止他们提及他们已经被要求交出数据。
   </strong>
  </p>
  <p class="graf graf--p">
   如果我们谴责的是以国家安全为名的两面派、以情报为名的强暴行为，以及不断侵犯公民自由和个人网络隐私的普遍做法，那么让经营硅谷的那些斯坦福大学毕业生们如此彻底地逍遥法外就很奇怪了。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    没有必要试图在中国和美国之间进行道德比较；
   </strong>
   中国城市中竖立的极权主义、无处不在的监控和镇压性的维稳网络，目前只能让 Palo Alto 的高管和警察工会的老板们羡慕不已，尽管我们不能责怪他们不努力。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    【尚未发布】《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%bb%96%e4%bb%ac%e5%b0%86%e4%b8%ad%e5%9b%bd%e8%a7%86%e4%b8%ba%e6%a6%9c%e6%a0%b7%ef%bc%9a%e6%b3%84%e9%9c%b2%e6%96%87%e4%bb%b6%e6%8f%ad%e7%a4%ba%e6%94%bf%e5%ba%9c%e9%97%b4%e8%b0%8d%e6%9c%ba%e6%9e%84/" href="https://www.iyouport.org/%e4%bb%96%e4%bb%ac%e5%b0%86%e4%b8%ad%e5%9b%bd%e8%a7%86%e4%b8%ba%e6%a6%9c%e6%a0%b7%ef%bc%9a%e6%b3%84%e9%9c%b2%e6%96%87%e4%bb%b6%e6%8f%ad%e7%a4%ba%e6%94%bf%e5%ba%9c%e9%97%b4%e8%b0%8d%e6%9c%ba%e6%9e%84/" rel="noopener noreferrer" target="_blank">
     他们将中国视为榜样：泄露文件揭示政府间谍机构如何利用冠状病毒推动大规模监视的内幕
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   新疆维吾尔族集中营的恐怖 —— 中国当局的残暴、非人化以及更糟糕的情况 —— 在这个国家是无与伦比的。中国国内对人权的践踏和对公民自由的瓦解，比美国自己的同类状况要大上好几个数量级。但别人比我们差得多，并不意味着我们就是好的、是干净的、就是值得效仿的。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%ab%98%e6%98%8e%e7%9a%84%e5%8f%8d%e6%8a%97/" href="https://www.iyouport.org/%e9%ab%98%e6%98%8e%e7%9a%84%e5%8f%8d%e6%8a%97/" rel="noopener noreferrer" target="_blank">
    对穆斯林群体的系统性监控和骚扰
   </a>
   ，要么是错的，要么不是。
  </p>
  <p class="graf graf--p">
   把少数民族群体
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2020/07/09/the-trump-child-abuse-scandal/" href="https://theintercept.com/2020/07/09/the-trump-child-abuse-scandal/" rel="noopener noreferrer" target="_blank">
    关在笼子里
   </a>
   ，要么是错的，要么不是。
  </p>
  <p class="graf graf--p">
   与军方和间谍机构关系密切的技术公司对用户
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/RMQrDD/iyp-2" href="https://start.me/p/RMQrDD/iyp-2" rel="noopener noreferrer" target="_blank">
    进行持续的监控
   </a>
   ，要么是严重问题，要么不是。
  </p>
  <p class="graf graf--p">
   窃听海底光缆、后门网络设备，把智能手机变成追踪设备，要么是没人该做的事，要么不是。
  </p>
  <p class="graf graf--p">
   向年轻人推销应用，然后囤积他们的数据，让他们接受面部识别，或者谁知道还有什么其他的东西，不管是扎克伯格还是 TikTok 的老板张一鸣实施的，都应该要么得罪我们，要么不得罪我们。
  </p>
  <p class="graf graf--p">
   硅谷的工业队长们早就放弃了对中国同行的道德优越感。至少，我不知道张一鸣曾经告诉他的朋友们，人们 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.businessinsider.com/embarrassing-and-damaging-zuckerberg-ims-confirmed-by-zuckerberg-the-new-yorker-2010-9" href="https://www.businessinsider.com/embarrassing-and-damaging-zuckerberg-ims-confirmed-by-zuckerberg-the-new-yorker-2010-9" rel="noopener noreferrer" target="_blank">
    像傻逼一样信任我
   </a>
   ”（这是扎克伯格的名言）。
  </p>
  <p class="graf graf--p">
   美国可以而且应该能够摆脱上述所有这些丑闻，因为我们有民主价值观，或对人权和人的基本尊严的承诺，我们公正的司法机构，我们有原则的行政部门，以及我们勤奋的立法部门，这种论点会说服一些渴望与中国打仗的人 —— 对这些人来说，只有好人（我们）和坏人（他们）—— 但对我们其他人来说，应该感到的是酸楚，特别是考虑到当前的政府。
  </p>
  <p class="graf graf--p">
   当不明身份的联邦突击队把公民塞进货车后座上、司法部由白宫的公开亲信指挥时，鼓吹美国对中国式数据做法的优越 “法治” 就不那么令人放心了。
  </p>
  <p class="graf graf--p">
   未来几周你会听到和读到很多关于蓬佩奥的 “净网” 计划如何意味着两个分裂的互联网的开始：一个是红色的中国人，一个是骄傲的美国人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    请记住，中国对技术未来的愿景是绝对令人反感的、危险的，无论如何都值得与之对抗；但美国的网络却并不干净。这一点至关重要
   </strong>
   。⚪️
  </p>

    The Filthy Hypocrisy of America’s “Clean” China-Free Internet

  </p>
