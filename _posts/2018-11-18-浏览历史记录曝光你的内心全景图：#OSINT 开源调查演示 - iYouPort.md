  <h4 class="graf graf--h4 graf-after--blockquote" id="a08b">
   – I — 探索浏览历史
  </h4>
  <p class="graf graf--p graf-after--h4" id="a0d7">
   以上这个小故事基于一个小小的摘录，只有一个两小时的样本，
   <strong class="markup--strong markup--p-strong">
    完全来自瑞士记者 J 先生的互联网浏览历史
   </strong>
   。2015 年6月下旬，他参观了柏林的战术科技办公室，并且他参加了这个开放私人生活的游戏 — — 即 看到通过他在自己的设备上创建的数据，老大哥可以得到什么。
  </p>
  <p class="graf graf--p graf-after--p" id="201c">
   我们探索的其中一个数据集是 J 先生的浏览历史收集。我们的目标是找出我们可以从一个人的浏览历史中挖到多少东西，或者，换句话说，任何人可以仅仅通过利用一个人的浏览记录得到哪些信息。
  </p>
  <p class="graf graf--p graf-after--p" id="ff83">
   <strong class="markup--strong markup--p-strong">
    查找浏览历史记录背后的真实姓名和社交图
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="1619">
   查看数据集只需要花几分钟时间，以关联此浏览历史记录背后的人的真实姓名。只需对他的 Facebook 流量进行排序，即他访问的个人资料页面，就能识别到真人。
  </p>
  <p class="graf graf--p graf-after--p" id="5ac7">
   由于 Facebook 正在实施“实名制政策”，这是将某人的浏览历史记录与其真实姓名联系起来的一种邪恶方式。
   <a class="markup--anchor markup--p-anchor" data-href="http://www.mimuw.edu.pl/~wjaworski/papers/2011_IJCIS.pdf" href="http://www.mimuw.edu.pl/~wjaworski/papers/2011_IJCIS.pdf" rel="noopener nofollow noreferrer" target="_blank">
    有许多学术论文
   </a>
   讲述了如何根据用户的浏览模式和行为识别身份的模型。 探索 Facebook URL 则更容易曝光某人的隐私，仅仅根据 URL 的结构，我们就能够重建这个人的社交关系图的一部分 — —
   <strong class="markup--strong markup--p-strong">
    是的，这种方法可以揭示任何追求实名上网的中国用户的私密生活，以及任何使用实名制的应用程序的用户，包括没能按照基本步骤采取匿名的用户
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="4ab3">
   这个故事仅仅依据 Facebook。
   <strong class="markup--strong markup--p-strong">
    对于中国的微信来说，您将“收获”更多
   </strong>
   。

  <h4 class="graf graf--h4 graf-after--figure" id="5722">
   —— 探索模式：算法眼中的习惯生物 ——
  </h4>
  <p class="graf graf--p graf-after--h4" id="622c">
   我们人类都是习惯的生物，我们倾向于在日常行为中创造重复的模式。我们倾向于在相似的时间睡觉和醒来，相似的早晨惯例、相似的社交互动仪式。如今，由于我们生活的许多部分都是由技术调节的，因此这些模式通过不同的数字足迹得以复制和可见。
  </p>
  <p class="graf graf--p graf-after--p" id="ea0a">
   <strong class="markup--strong markup--p-strong">
    当识别出模式时，异常检测就诞生了
   </strong>
   。正如 Pasquinelli 所说，模式和异常的两个认知极点是算法治理的同一枚硬币的两面，只能在模式规律性的基础上检测到意外的异常（“Anomaly Detection: The Mathematization of the Abnormal in the Metadata Society”, Matteo Pasquinelli (2015)）。
  </p>
  <p class="graf graf--p graf-after--p" id="1250">
   模式识别和异常检测都被用作理解大量数据的方法，我们所有人的数字足迹正被许多参与者 — — 全球政府机构、互联网公司和服务提供商或数据交易商 即第三方，所收集。
  </p>
  <p class="graf graf--p graf-after--p" id="b428">
   <strong class="markup--strong markup--p-strong">
    在算法的眼中被认为是“异常”的东西就可以将您列入政府机构的重点监视列表
   </strong>
   ，或者某些行为模式可以导致您被标记为在线广告的投放目标 — —
   <strong class="markup--strong markup--p-strong">
    这也是为什么我们自己也许都无法识别自己的模式“异常”，但政府知道，数据中间商知道，任何一个老大哥都知道
   </strong>
   。 在 J 先生的简单条形图和热图的情况下，基于浏览动作的次数就可以揭示出几种行为模式。
  </p>

  <h3 class="graf graf--h3 graf-after--p" id="71e7">
   谁有权浏览这些数据？
  </h3>
  <p class="graf graf--p graf-after--h3" id="5855">
   了解谁可以访问我们的浏览历史以及分析它的可能性，将使我们深入了解信息社会中新的权力结构和财富分配。
  </p>
  <p class="graf graf--p graf-after--p" id="b4f0">
   <strong class="markup--strong markup--p-strong">
    浏览器
   </strong>
   — — 用于 Web 浏览的基本软件是浏览器。本质上，它是一种软件，可将用户的命令（最常见的 URL）转换为服务器可以理解的请求，然后以用户可理解的方式表示服务器的响应。因此，浏览器是这个拼图中的单个部分，其中包含有关用户浏览习惯的所有数据。大多数现代浏览器允许创建配置文件，即允许用户登录，并在他们使用的所有设备上具有相同的设置、书签和历史记录。
   <strong class="markup--strong markup--p-strong">
    这意味着他们正在将这些数据传递到一个中心点，该中心点由构建特定浏览器的公司/组织所拥有。我们现在都知道这家公司是谁……没错，Google
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="69b5">
   <strong class="markup--strong markup--p-strong">
    Cookies
   </strong>
   — — 超过 50％ 的网站使用 cookies . Cookie 收集的数据有不同的应用程序，其中一些确实影响功能（本地化和用户特定设置，但最广泛使用 Cookie 的是广告。更准确地说，对用户的习惯进行分析。许多 cookie 收集有关引荐来源 URL 地址的数据。
  </p>
  <p class="graf graf--p graf-after--p" id="ebc4">
   <strong class="markup--strong markup--p-strong">
    Network
   </strong>
   — — 互联网作为一个网络是一个非常物理的东西，它由服务器、电缆、路由器和其他网络设备组成，使流量能够流动。这些网段由不同的实体建立，大多数时候是控制网络基础设施的公司，但不是所有网络在物理和技术上都是安全的。
   <strong class="markup--strong markup--p-strong">
    攻击者可以通过多种方式连接到网络，并嗅探数据流量，虽然密码和财务详细信息等数据通常是加密的，但 URL 则不会加密，这就会将它们带到网络嗅探器的触及范围之内
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="70f2">
   <strong class="markup--strong markup--p-strong">
    悲剧的法律
   </strong>
   — — 许多国家的立法都有所谓的数据保留法，要求 ISP 和电信公司保留6个月到2年的元数据，具体取决于您在哪个国家/地区。这意味着，如果访问您的保留数据，就能看到在两年前您正在访问的所有网站，所有在线行为，也就是您的全部生活，因为如今每个人都将内心留在互联网上了。对于使用 SSL / TLS 的网站，只有域名可见，而不是完整的 URL，
   <strong class="markup--strong markup--p-strong">
    但这并不会使这些保留的数据不那么具有侵入性，因为其逻辑映射可能导致任何种类的不同结论
   </strong>
   。
  </p>

  </blockquote>
  <p class="graf graf--p graf-after--blockquote" id="c86b">
   <strong class="markup--strong markup--p-strong">
    19 世纪的根源
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="9943">
   1850 年代美国海军上尉 Matthew Fontaine Maury 通过美国海军天文台发现了数千艘旧船的原木。当时，在航行完成后，日志不被视为重要信息。源于他的痴迷，他开发了一种方法，系统地从每个日志中提取关键信息，并开始手动绘制地图信息，使用超过 1,200 万个数据点，以提高船舶的导航速度和安全性。
  </p>
  <p class="graf graf--p graf-after--p" id="1ded">
   <strong class="markup--strong markup--p-strong">
    他被认为是我们今天称之为大数据分析的先驱之一
   </strong>
   ，他是第一个认识到由数千个较小数据块创建的信息之价值的人。但对于我们的背景来说，这个故事还有另一个有趣的方面。
  </p>
  <p class="graf graf--p graf-after--p" id="c3d1">
   他的地图被证明是非常有用和成功的，不仅在海军内部，而且在商船之间。了解新数据收集的重要性，Maury 建立了交换船舶原木地图的原则。这种提供产品或服务的做法，在他的情况下映射，以换取航行日志，就如今天的浏览历史数据，
   <strong class="markup--strong markup--p-strong">
    这就是 150 年后 Google 或 Facebook 等当代数字极权的主要商业模式的基本组成部分
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="3f85">
   再倒退十年，19世纪40年代，在大西洋的另一边，在英国，还有另一个与我们的故事相关的重要历史事件。根据 D
   <a class="markup--anchor markup--p-anchor" data-href="http://www.historyandpolicy.org/policy-papers/papers/surveillance-privacy-and-history" href="http://www.historyandpolicy.org/policy-papers/papers/surveillance-privacy-and-history" rel="noopener nofollow noreferrer" target="_blank">
    avid Vincent 的说法
   </a>
   ，这一时期
   <strong class="markup--strong markup--p-strong">
    促进了我们现在称之为社交网络的创造
   </strong>
   ，利用当时的信息技术（邮政服务）来扩展个人互动领域，通过在一天内来回交换邮件，可以在城市内进行对话、安排和参与会议。
  </p>
  <p class="graf graf--p graf-after--p" id="2ef3">
   在 1840 年之前，邮政服务主要由政府控制之外的非正式信函分散网络运行，旨在规避皇家邮政的高额费用。当 Penny Post 作为一个集中的，低成本的政府邮政服务被引入时，由于担心日益增长的工人阶级运动，“保护国家免受内部威胁”（也就是维稳），因此抹杀了隐私问题。
   <strong class="markup--strong markup--p-strong">
    这使政府能够获得公民的邮政通信，并且第一次系统性地统计了一个国家的通信实践并生成了统计数据
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="4007">
   正如 Vincent 所描述的那样，现在可以进行同样的统计测试。
   <strong class="markup--strong markup--p-strong">
    它更加微粒化、体积更大、更快速，与十九世纪不同，它还涉及跨国公司的利润
   </strong>
   。
  </p>
  <figure class="wp-block-image">
   <img alt="" class="wp-image-3205 jetpack-lazy-image" data-lazy-sizes="(max-width: 957px) 100vw, 957px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-45.jpeg?resize=957%2C820&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-45.jpeg?w=957&amp;ssl=1 957w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-45.jpeg?resize=300%2C257&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-45.jpeg?resize=768%2C658&amp;ssl=1 768w" data-recalc-dims="1" height="820" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-45.jpeg?resize=957%2C820&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="957"/>
   <noscript>
    <img alt="" class="wp-image-3205" data-recalc-dims="1" height="820" sizes="(max-width: 957px) 100vw, 957px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-45.jpeg?resize=957%2C820&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-45.jpeg?w=957&amp;ssl=1 957w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-45.jpeg?resize=300%2C257&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-45.jpeg?resize=768%2C658&amp;ssl=1 768w" width="957"/>
   </noscript>
  </figure>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--figure" id="6f49">
   “秘密办公室”在 1650 年代形成，并在邮政总局内作为
   <strong class="markup--strong markup--p-strong">
    卧底国家间谍机构
   </strong>
   运作。
   <strong class="markup--strong markup--p-strong">
    该办公室的主要工作就是拦截英国和海外之间的邮件
   </strong>
   ，并阅读它们。在19世纪40年代，秘密办公室以某种方式被暴露，并就其行为
   <a class="markup--anchor markup--p-anchor" data-href="http://www.historyextra.com/feature/5-things-you-didn%E2%80%99t-know-about-secret-spying-arm-post-office" href="http://www.historyextra.com/feature/5-things-you-didn%E2%80%99t-know-about-secret-spying-arm-post-office" rel="noopener nofollow noreferrer" target="_blank">
    接受了调查
   </a>
   。
  </p>
  <h3 class="graf graf--h3 graf-after--p" id="78dd">
   现在：走向思想警察
  </h3>
  <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote graf-after--h3" id="b200">
   <em class="markup--em markup--blockquote-em">
    “There was of course no way of knowing whether you were being watched at any given moment. How often, or on what system, the Thought Police plugged in on any individual wire was guesswork. It was even conceivable that they watched everybody all the time. But at any rate they could plug in your wire whenever they wanted to. You had to live — did live, from habit that became instinct — in the assumption that every sound you made was overheard, and, except in darkness, every movement scrutinized.” — — 1984, George Orwell
   </em>
  </blockquote>
  <figure class="wp-block-image">
   <img alt="" class="wp-image-3207 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-211.png?resize=1024%2C634&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-211.png?resize=1024%2C634&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-211.png?resize=300%2C186&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-211.png?resize=768%2C476&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-211.png?w=1040&amp;ssl=1 1040w" data-recalc-dims="1" height="634" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-211.png?resize=1024%2C634&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>
   <noscript>
    <img alt="" class="wp-image-3207" data-recalc-dims="1" height="634" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-211.png?resize=1024%2C634&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-211.png?resize=1024%2C634&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-211.png?resize=300%2C186&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-211.png?resize=768%2C476&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/0-211.png?w=1040&amp;ssl=1 1040w" width="1024"/>
   </noscript>
   <figcaption>
    George Orwell’s 1984–1954 BBC TV Movie
   </figcaption>
  </figure>
  <p class="graf graf--p graf-after--figure" id="e382">
   人们一直在努力减少“电子通信交易记录”，仅仅是关于一个人行踪的额外信息，就像调查人员从线人那得到的东西。或者通过所谓的“国家安全”借口获得的，这是一种行政传票，使美国联邦机构能够
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/world/national-security/fbi-wants-access-to-internet-browser-history-without-a-warrant-in-terrorism-and-spy-cases/2016/06/06/2d257328-2c0d-11e6-9de3-6e6e7a14000c_story.html" href="https://www.washingtonpost.com/world/national-security/fbi-wants-access-to-internet-browser-history-without-a-warrant-in-terrorism-and-spy-cases/2016/06/06/2d257328-2c0d-11e6-9de3-6e6e7a14000c_story.html" rel="noopener nofollow noreferrer" target="_blank">
    在没有事先司法监督的情况下收集公民的隐私信息
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="e4b0">
   2016 年2月举行的听证会上，坐在美国参议院特别委员会面前，FBI 负责人提到增加有争议的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/world/national-security/fbi-wants-access-to-internet-browser-history-without-a-warrant-in-terrorism-and-spy-cases/2016/06/06/2d257328-2c0d-11e6-9de3-6e6e7a14000c_story.html" href="https://www.washingtonpost.com/world/national-security/fbi-wants-access-to-internet-browser-history-without-a-warrant-in-terrorism-and-spy-cases/2016/06/06/2d257328-2c0d-11e6-9de3-6e6e7a14000c_story.html" rel="noopener nofollow noreferrer" target="_blank">
    关键字过滤
   </a>
   ；当时的六年前，在类似的意图失败之前，美国政府对通信内容完全漠不关心，只寻求其技术性记录，认为“获取元数据会更快更容易”; 所有已经存在的数据，每次点击都会大规模生成。（详见我们本周后期的具体演示）
  </p>
  <p class="graf graf--p graf-after--p" id="fc53">
   但是电子通信交易记录或通信数据 — 例如拨打的号码、发送的文本消息的接收者，所涉及的设备的 IP 地址，特别是所访问的网络域名的记录 — 经常能
   <strong class="markup--strong markup--p-strong">
    显示出比内容本身更多的隐私信息
   </strong>
   ，
   <strong class="markup--strong markup--p-strong">
    我们将就这一问题做一系列详细说明和开源调查演示
   </strong>
   。
   <a class="markup--anchor markup--p-anchor" data-href="https://assets.documentcloud.org/documents/2853157/Read-the-letter-from-privacy-advocates-opposing.pdf" href="https://assets.documentcloud.org/documents/2853157/Read-the-letter-from-privacy-advocates-opposing.pdf" rel="noopener nofollow noreferrer" target="_blank">
    用隐私组织的话来说
   </a>
   ：“这些信息可以揭示一个人的政治关系，医疗条件，宗教信仰，药物滥用史，性取向，甚至他或她的
   <strong class="markup--strong markup--p-strong">
    全天活动的细节
   </strong>
   ，”描绘出一个人最私密的生活全景图。
  </p>
  <p class="graf graf--p graf-after--p" id="dc81">
   Snowden 揭露的国家安全局 一个名为 Xkeyscore 的计算机系统，用于搜索和分析全球互联网数据时（美国国家安全局每时每刻都收集的东西），对通信数据的渴望的真正范围被揭示出来。作为“从互联网开发情报的最广泛的系统”，包括电子邮件的内容、访问和搜索的网站，以及他们的元数据，Xkeyscore 允许 NSA 分析师
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2013/jul/31/nsa-top-secret-program-online-data" href="https://www.theguardian.com/world/2013/jul/31/nsa-top-secret-program-online-data" rel="noopener nofollow noreferrer" target="_blank">
    在没有事先授权的情况下搜索其庞大的数据库
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="f692">
   由 DARPA 资助的另一个项目可以让我们深入了解数据收集和分析的未来应用。多尺度异常检测（ADAMS）程序可创建、调整和应用技术，
   <strong class="markup--strong markup--p-strong">
    以便在海量数据集中进行异常表征和检测
   </strong>
   。数据中的异常提示在各种现实世界环境中
   <a class="markup--anchor markup--p-anchor" data-href="http://www.darpa.mil/program/anomaly-detection-at-multiple-scales" href="http://www.darpa.mil/program/anomaly-detection-at-multiple-scales" rel="noopener nofollow noreferrer" target="_blank">
    收集额外的可操作信息
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="d207">
   <strong class="markup--strong markup--p-strong">
    该项目基本上通过分析个人的浏览习惯以及其他数据源（例如手机日志或位置数据）
   </strong>
   ，创建了在军事等大系统中识别下一个 Edward Snowden 或 Chelsea Manning 的平台。
  </p>
  <p class="graf graf--p graf-after--p" id="d117">
   数据热潮绝不仅限于西方所谓的“反恐战争”管理者和其他政党，但它拥有同样的普遍借口，
   <strong class="markup--strong markup--p-strong">
    即所谓的“国家安全” — — 是的，这是当今世界最大和最流行的狡辩之词。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="b113">
   例如，中国认为现在是时候将游戏向前推进一步了：最大的国营防务承包商之一中国电子科技集团
   <a class="markup--anchor markup--p-anchor" data-href="http://chinadigitaltimes.net/2016/03/china-tries-hand-pre-crime/" href="http://chinadigitaltimes.net/2016/03/china-tries-hand-pre-crime/" rel="noopener nofollow noreferrer" target="_blank">
    正致力于开发收集和整合数据的软件
   </a>
   ，关于工作、爱好、消费习惯、和普通公民的其他行为数据。官员们宣布，这个被称为预测性警务数据平台的“统一信息环境”将首先在少数民族人口的地区
   <a class="markup--anchor markup--p-anchor" data-href="http://www.bloomberg.com/news/articles/2016-03-03/china-tries-its-hand-at-pre-crime" href="https://www.bloomberg.com/news/articles/2016-03-03/china-tries-its-hand-at-pre-crime" rel="noopener nofollow noreferrer" target="_blank">
    进行测试
   </a>
   。
   <strong class="markup--strong markup--p-strong">
    除了传统的数据收集方式，该平台还整理了中国公民的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.defenseone.com/technology/2016/03/thanks-america-china-aims-tech-dissent/126491/" href="http://www.defenseone.com/technology/2016/03/thanks-america-china-aims-tech-dissent/126491/" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     在线行为数据
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="4f52">
   您还认为“技术性解决方案只是 IT 界书呆子才应该具备的知识”吗？是否能掌握基本的隐私保护技术已经事关您的安危！
  </p>
  <p class="graf graf--p graf-after--p" id="494c">
   <strong class="markup--strong markup--p-strong">
    谁是 J 先生？
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="efbd">
   那么，我们真的可以通过浏览历史记录中的 URL 来真正了解 J 先生是谁吗？
  </p>
  <p class="graf graf--p graf-after--p" id="a552">
   您已经看到，从 J 先生的浏览历史中收集的数据就能提供对他在某一天内的意识流的惊人洞察。
   <strong class="markup--strong markup--p-strong">
    我们只是在做最简单的演示，而真正的老大哥可以拿到更多的数据来确认这些思想所具有的实际意义的任何可能性
   </strong>
   。无论哪种方式，J 先生仍然暴露。J 先生可能只是一个普通的、体面的、有点疲惫的家伙，寻求从职业跑步机上获得短暂的喘息。而完全放弃了自己的隐私权。
  </p>
  <p class="graf graf--p graf-after--p" id="6491">
   我们只希望您不是他。
  </p>
