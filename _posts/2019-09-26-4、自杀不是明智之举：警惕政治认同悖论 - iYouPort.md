     自杀不是明智之举：警惕政治认同悖论
    </a>
    》
   </li>
   <li>
    “如果真正的激进派发现长发会给沟通和组织活动设置心理障碍，他们就会减掉头发”《
    <a href="https://www.iyouport.org/5%E3%80%81%E6%8A%97%E8%AE%AE%E8%A1%8C%E5%8A%A8%E4%B8%8D%E6%98%AF%E5%B9%BF%E5%9C%BA%E8%88%9E%EF%BC%9A%E8%A1%A8%E8%BE%BE%E6%80%A7%E5%92%8C%E5%B7%A5%E5%85%B7%E6%80%A7%E7%9A%84%E7%9F%9B%E7%9B%BE-2/" rel="noopener noreferrer" target="_blank">
     抗议行动不是广场舞：表达性和工具性的矛盾

   </li>

   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5173" data-recalc-dims="1" height="729" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1.jpg?resize=1100%2C729&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1.jpg?resize=300%2C199&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1.jpg?resize=768%2C509&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1.jpg?resize=1024%2C678&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/1.jpg?resize=1100%2C729&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <blockquote>
   <p class="p1">
    <em>
     <span class="s1">
      “
     </span>
     <span class="s2">
      少数激进派行动者在敌人的领土上英勇就义，就像孤独的游击战士。也许能让某些人感到光荣，但是无疑，这是一次自杀任务
     </span>
     <span class="s1">
      ”
     </span>
    </em>
   </p>
  </blockquote>
  <p class="p1">
   <span class="s2">
    中国的社区间协作为什么很差？这是曾经被观察者问及的。原因也许有很多，但组织运作方式问题也许是根本性的。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    任何接触或参与过中国社交圈子的人大概都会有这样的印象，很多圈子会越变越小，圈子的
   </span>
   <span class="s1">
    “
   </span>
   <span class="s2">
    中心级人物
   </span>
   <span class="s1">
    ”
   </span>
   <span class="s2">
    会不断与群体中其他人失去连接关系；直至他们彼此之间也出现裂痕，最终崩溃。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    <b>
     尤其是政治异议的群体，这种现象是格外有害的
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    几乎所有人都明白，团队组织需要凝聚力，越多越好，团结才能鼓舞士气。
   </span>
  </p>
  <p class="p4">
   <span class="s2">
    任何严肃的社会运动都需要相应严肃的群体认同，这种认同可以鼓励其成员在长期斗争中作出特殊的承诺和牺牲。
   </span>
  </p>
  <p class="p4">
   <span class="s2">
    但是，
   </span>
   <span class="s4">
    ⚠️
   </span>
   <span class="s2">
    强大的团体认同是一把双刃剑
   </span>
   <span class="s5">
    ——
   </span>
   <span class="s2">
    群体的身份和凝聚力越强，人们就越有可能与其他群体和社会疏远。加速自我边缘化。
   </span>
  </p>
  <p class="p6">
   <span class="s3">
    这就是
    <b>
     政治认同的悖论
    </b>
    。
   </span>
  </p>
  <p class="p4">
   <span class="s2">
    这种现象在所有群体中都是如此，
   </span>
   <span class="s4">
    ⚠️
   </span>
   <span class="s2">
    但往往会对参与政治斗争的团体产生非常特殊的影响，因为，
    <b>
     反抗团体不仅需要培养强大的内部认同，还必须赢得盟友
    </b>
    。
   </span>
  </p>
  <p class="p4">
   <span class="s2">
    在政治团体中，孤立的倾向可能会迅速升级，因为对峙的斗争会激发出更多对峙的心理（就如在美国民权运动期间活动家们忍受的残酷抵抗）。
   </span>
  </p>
  <p class="p4">
   <span class="s2">
    一方面，参与者需要比以往更多地相互传递力量和支持，在冲突升级的时刻里，他们需要对自己的群体身份感到自豪并由此产生凝聚力；而另一方面，他们也需要保持外向，与广泛的且不断增长的大众保持联系。
   </span>
  </p>
  <p class="p7">
   <span class="s2">
    即使领导者专注于完成这项任务，也可能很难做到这一点，更不用说当他们毫无准备的情况下了，往往就是突出的悖论。
   </span>
  </p>
  <p class="p7">
   <span class="s2">
    以民主社会学生组织
   </span>
   <span class="s5">
    SDS
   </span>
   <span class="s2">
    为例（原始的
   </span>
   <span class="s5">
    SDS
   </span>
   <span class="s2">
    在
   </span>
   <span class="s5">
    1969
   </span>
   <span class="s2">
    年以戏剧性的方式分崩离析）。
   </span>
  </p>
  <p class="p7">
   <span class="s2">
    这个庞大的学生组织从中心崩溃了
   </span>
   <span class="s5">
    ——
   </span>
   <span class="s2">
    充分体现了政治认同悖论。当时，关键领导者已经被他们的对立身份所包围，并且变得越来越孤立。
    <b>
     他们失去了与更广泛的会员（即
    </b>
   </span>
   <b>
   </b>
   <span class="s2">
    <b>
     学生）相连接的能力和倾向，更不用说更广泛的社会了
    </b>
    。
   </span>
  </p>
  <p class="p7">
   <span class="s2">
    那一代中一些最忠诚的潜在领导人认为，与组织大量学生采取协调行动相比，少数人制造炸弹会更有价值。
   </span>
  </p>
  <p class="p7">
   <span class="s4">
    ⚠️
   </span>
   <span class="s2">
    这是一种极端孤立的趋势。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    少数激进派行动者在敌人的领土上英勇就义，就像孤独的游击战士。也许能让某些人感到光荣，但是，这是一次自杀任务。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    <b>
     政治认同悖论提出了警告
    </b>
    ：政治团体需要发展强大的联系和强有力的衔接。如果没有强大的团队内部联系，团队成员就会缺乏严肃斗争所需的承诺水平；但是，如果没有强大的超越群体本身的衔接，该组织就会变得迅速孤立和边缘，无法建立更广泛的联盟。
   </span>
  </p>
  <p class="p3">
   <span class="s4">
    💡
   </span>
   <span class="s3">
    <b>
     优秀的领导者必须在这两种刚需相互冲突的条件下进行非凡的平衡，以便在团队中建立强烈的认同感，并且同时与团队之外的盟友和潜在盟友建立联系
    </b>
    。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    组织运作需要很多技巧，其一点不差行动本身所需要的技巧。毕竟，你想要做的是与更强大和沉重的机器对抗，而你首先就应该让自身变成更加敏捷和灵巧的机器。螺丝钉是无法完成这项任务的。
   </span>
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5174 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2.jpg?resize=1100%2C688&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2.jpg?resize=300%2C188&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2.jpg?resize=768%2C480&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2.jpg?resize=1024%2C640&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2.jpg?resize=1100%2C688&amp;ssl=1 1100w" data-recalc-dims="1" height="688" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2.jpg?resize=1100%2C688&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-5174" data-recalc-dims="1" height="688" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2.jpg?resize=1100%2C688&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2.jpg?w=1280&amp;ssl=1 1280w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2.jpg?resize=300%2C188&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2.jpg?resize=768%2C480&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2.jpg?resize=1024%2C640&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2.jpg?resize=1100%2C688&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="p1">
   <span class="s2">
    招募协作者
   </span>
   <span class="s1">
    ——
   </span>
   <span class="s2">
    更多螺丝钉
   </span>
   <span class="s1">
    ——
   </span>
   <span class="s2">
    是第一步。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    引入新参与者对于任何希望增加规模和能力的活动家团体都至关重要
   </span>
   <span class="s1">
    ——
   </span>
   <span class="s2">
    但必须知道这只是第一步。
   </span>
  </p>
  <p class="p1">
   <span class="s4">
    ⚠️
   </span>
   <span class="s2">
    将人们融入已建立的团队，将是下一步更大的挑战，需要对其进行有意识的处理。你需要的不仅是自身的魅力
   </span>
   <span class="s1">
    ——
   </span>
   <span class="s2">
    让人们喜欢上你愿意接近你，并且，你还需要一些基本的程序，能够让加入你的新人感受到被欢迎的融洽。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    对于初学者来说，当有人说他们有兴趣了解更多信息或参与你的小组时，不要只是邀请他们参加你的下一次会议并将其留在原地等待；这可能会突出
   </span>
   <span class="s1">
    “
   </span>
   <span class="s2">
    主
   </span>
   <span class="s1">
    –
   </span>
   <span class="s2">
    客
   </span>
   <span class="s1">
    ”
   </span>
   <span class="s2">
    关系并令情感保持生疏。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    即使是最热情和最具包容性的群体也倾向于发展自己的内部文化，这种文化无意中会让新人感觉自己像个外人。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    <b>
     就如中国的圈子所呈现的那样
    </b>
    ：新来的人往往插不上话，你完全不知道原有的成员之间交流的是什么意思；他们使用特定的彼此才理解的措辞和表达方式，它产生了鲜明的排斥感。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    为了提高您的新成员保留率，请在新的成员加入小组之前安排一对一的交流。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    作为组织者你需要结识这个人。了解是什么吸引了他们，他们喜欢或擅长什么样的任务，以及他们有多少时间。然后告诉他们更多有关该小组的信息，并讨论他们的参与情况。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    虽然这一过程可能需要花费你更多的时间，但是，
    <b>
     从长远来看，它可以节省时间
    </b>
    ：因为这样做能让人们倾向于更快地进入工作、并且更长时间地坚持下去。
   </span>
  </p>
  <p class="p1">
   <span class="s4">
    💡
   </span>
   <span class="s2">
    让您的团队中的一个或两个成员作为持续的角色承担这一责任，可能是有意义的。
   </span>
  </p>
  <p class="p3">
   <span class="s3">
    其次，
    <b>
     如果你想激励人们保持参与，你需要让他们感受到被重视和被赞赏
    </b>
    。人们喜欢和善待自己的人在一起。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    我们大多数人都不缺少可以填满空闲时间的杂事：
   </span>
   <span class="s4">
    💡
   </span>
   <span class="s2">
    如果你希望人们优先考虑你的团体，你就必须善待他们。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    注意观察捕捉并承认新人的贡献，无论多么微小。请抽出时间在聚会之外与他们一起合作和交流。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    经常询问他们的意见：他们对组织的看法是什么？对哪个行动比较感兴趣？有些什么希望？在倾听的过程中您就可以将组织的理念融合进新人的意识。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    技术是你的武器，人员是你的力量，缺一不可。
   </span>
  </p>
  <p class="p1">
   <span class="s2">
    你的对手最擅长的就是孤立你和你的队友，让你们先入无助和分裂（
    <b>
     我们将在后面开启
    </b>
   </span>
   <span class="s1">
    <b>
     “
    </b>
   </span>
   <span class="s2">
    <b>
     攻心计
    </b>
   </span>
   <span class="s1">
    <b>
     ”
    </b>
   </span>
   <span class="s2">
    <b>
     的独立话题，详细这些手段
    </b>
    ）；如果你想赢，就有必要努力抵御这些攻势，切勿给对手以可乘之机。
   </span>
   <span class="s4">
    ⚪️
   </span>
  </p>
