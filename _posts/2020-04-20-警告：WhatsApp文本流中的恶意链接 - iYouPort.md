  <p class="graf graf--p">
   在2018年11月至2019年5月之间，藏族团体的高级成员在私人的 WhatsApp 文本交流中收到恶意链接，攻击者
   <strong class="markup--strong markup--p-strong">
    冒充非政府组织工作人员、记者和其他虚假角色
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这些恶意链接利用了网络浏览器的漏洞，可在iOS和Android设备上安装间谍软件，其中有些案例是OAuth网络钓鱼页面。公民实验室称攻击者为 POISON CARP。
  </p>
  <p class="graf graf--p">
   公民实验室观察到 POISON CARP 总共使用了八个安卓浏览器漏洞和一个安卓间谍软件套件，以及一个iOS漏洞利用链和iOS间谍软件。
   <strong class="markup--strong markup--p-strong">
    被观察到的漏洞利用都不是0day
   </strong>
   。
  </p>
  <p class="graf graf--p">
   POISON CARP 的攻击与去年广泛报道的针对维吾尔族人的两次网络攻击活动之间存在重叠。
   <strong class="markup--strong markup--p-strong">
    此次攻击中使用的Android恶意软件是功能齐全的间谍软件套件，以前没有被记录过
   </strong>
   。
  </p>
  <p class="graf graf--p">
   POISON CARP 似乎使用了来自各种来源的 Android 浏览器漏洞。在一种情况下，POISON CARP 使用了 Exodus Intelligence 公开发布的有效漏洞利用，这个漏洞在源代码中得到了修复 ，但尚未将其补丁分发给 Chrome 用户。
  </p>
  <p class="graf graf--p">
   在其他情况下，POISON CARP 使用的是经过轻微修改的Chrome漏洞利用代码版本，这些代码发布在个人 GitHub 页面上，包括奇虎360 Vulcan 团队的成员、腾讯 Xuanwu Lab 的成员、还有 Chrome Bug Tracker 上的一个 Google Project Zero 成员。
  </p>
  <p class="graf graf--p">
   这是第一起针对藏人群体的一键式手机攻击案件，反映了针对藏人群体的数字间谍威胁的复杂程度不断升级。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     摘要
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   西藏社区已经被数字间谍活动包围了
   <a class="markup--anchor markup--p-anchor" data-href="https://isc.sans.edu/diary/Overview+of+cyber+attacks+against+Tibetan+communities/4177" href="https://isc.sans.edu/diary/Overview+of+cyber+attacks+against+Tibetan+communities/4177" rel="noopener noreferrer" target="_blank">
    十多年
   </a>
   。
  </p>
  <p class="graf graf--p">
   2009年，《信息战监测》发表了
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2009/03/tracking-ghostnet-investigating-a-cyber-espionage-network/" href="https://citizenlab.ca/2009/03/tracking-ghostnet-investigating-a-cyber-espionage-network/" rel="noopener noreferrer" target="_blank">
    Tracking GhostNet
   </a>
   的报告，详细描述了一个针对西藏组织的恶意软件操作，包括在达兰萨拉的达赖喇嘛私人办公室、以及在103个国家的政府机构。
  </p>
  <p class="graf graf--p">
   当时很少有公开报道针对性的恶意软件攻击，关于这些威胁如何影响民间社会的文件也很有限。
  </p>
  <p class="graf graf--p">
   在过去的十年里，GhostNet 使用的策略已经为西藏人所熟悉：充满旧漏洞利用的电子邮件被用来向没有及时更新的电脑传送定制的恶意软件。
  </p>
  <p class="graf graf--p">
   通常，这些操作中使用的恶意软件针对的是 Windows 系统，一些罕见的恶意软件针对
   <a class="markup--anchor markup--p-anchor" data-href="https://targetedthreats.net/media/2-Extended%20Analysis-Full.pdf" href="https://targetedthreats.net/media/2-Extended%20Analysis-Full.pdf" rel="noopener noreferrer" target="_blank">
    MacOS
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2013/04/permission-to-spy-an-analysis-of-android-malware-targeting-tibetans/" href="https://citizenlab.ca/2013/04/permission-to-spy-an-analysis-of-android-malware-targeting-tibetans/" rel="noopener noreferrer" target="_blank">
    Android
   </a>
   系统。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些间谍活动有一个共同点，那就是关注巧妙的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     社交工程
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.usenix.org/system/files/conference/usenixsecurity14/sec14-paper-hardy.pdf" href="https://www.usenix.org/system/files/conference/usenixsecurity14/sec14-paper-hardy.pdf" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     而不是
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    漏洞利用或恶意软件的技术复杂性
   </strong>
   。（简单说它是交互式的攻击，需要目标人的点击操作才能实现）
  </p>
  <p class="graf graf--p">
   虽然这些模式很常见，⚠️但
   <strong class="markup--strong markup--p-strong">
    此次观察到的战术转变似乎与社区防御姿态的变化有关
   </strong>
   。
  </p>
  <p class="graf graf--p">
   从历史上看，恶意软件以电子邮件附件的形式发送是藏族群体经历的最常见的威胁。
  </p>
  <p class="graf graf--p">
   作为回应，社区中的团体发起了一场提高用户安全意识的运动，建议人们使用云平台，如 Google Drive 或 DropBox，来分享文档以替代电子邮件附件。
  </p>
  <p class="graf graf--p">
   之后逐渐观察到针对藏族群体的恶意软件攻击有所减少，但凭证式钓鱼有所增加，这表明操作者正在改变他们的应对策略。
  </p>
  <p class="graf graf--p">
   最近，公民实验室还观察到恶意 OAuth 应用的攻击，可能是为了
   <strong class="markup--strong markup--p-strong">
    绕过用户在谷歌账户上使用的双因素身份验证
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e4%bd%bf%e7%94%a8%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b%e6%9d%a5%e6%94%bb%e5%87%bb-2fa%ef%bc%9f/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e4%bd%bf%e7%94%a8%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b%e6%9d%a5%e6%94%bb%e5%87%bb-2fa%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     如何使用社交工程来攻击 2FA？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   这些变化表明了藏人组织的数字防御和针对他们的攻击者的能力之间的内在不对称 —— ⚠️
   <strong class="markup--strong markup--p-strong">
    改变一个社区的安全习惯是一个缓慢而渐进的过程，而对手可以在一夜之间发展
   </strong>
   。
  </p>
  <p class="graf graf--p">
   为了应对这些挑战，藏人组织最近成立了西藏计算机应急小组（TibCERT），这是一个由藏人组织组成的联盟，旨在通过事件响应协作和数据共享来改善数字安全。
  </p>
  <p class="graf graf--p">
   2018年11月，TibCERT 收到了通过 WhatsApp 发送给西藏组织高级成员的可疑信息。
  </p>
  <p class="graf graf--p">
   在目标群体的同意下，TibCERT 与公民实验室共享了这些信息的样本。
  </p>
  <p class="graf graf--p">
   分析发现，这些信息包含了旨在针对 iPhone 和 Android 设备安装间谍软件的恶意链接。
  </p>
  <p class="graf graf--p">
   这一攻击似乎是由单一的操作者执行的，公民实验室称为 POISON CARP。这是第一次被记录在案的针对藏族群体的一键式移动攻击。
  </p>
  <p class="graf graf--p">
   与通常观察到的用来对付藏族社区的手段相比，
   <strong class="markup--strong markup--p-strong">
    它代表了
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" href="https://www.iyouport.org/category/%e5%bf%83%e7%90%86%e5%ad%a6%e5%92%8c%e7%a4%be%e4%ba%a4%e5%b7%a5%e7%a8%8b/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     社交工程策略
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    和技术复杂性的显著升级
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Poison CARP 使用的漏洞、间谍软件和基础设施将它与去年报道的两个针对维吾尔族团体的数字间谍活动联系起来。
  </p>
  <p class="graf graf--p">
   2019年8月，谷歌 Project Zero 报道了谷歌威胁分析小组发现的一场数字间谍活动，利用 iOS 漏洞（其中一个案例中为
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%8d%b1%e9%99%a9%ef%bc%9a%e7%9b%91%e6%8e%a7%e6%8a%80%e6%9c%af%e5%85%ac%e5%8f%b8%e6%ad%a3%e5%9c%a8%e6%8e%a8%e5%8a%a8%e5%af%b9%e9%9b%b6%e6%97%a5%e6%bc%8f%e6%b4%9e%e7%9a%84%e5%88%a9%e7%94%a8/" href="https://www.iyouport.org/%e5%8d%b1%e9%99%a9%ef%bc%9a%e7%9b%91%e6%8e%a7%e6%8a%80%e6%9c%af%e5%85%ac%e5%8f%b8%e6%ad%a3%e5%9c%a8%e6%8e%a8%e5%8a%a8%e5%af%b9%e9%9b%b6%e6%97%a5%e6%bc%8f%e6%b4%9e%e7%9a%84%e5%88%a9%e7%94%a8/" rel="noopener noreferrer" target="_blank">
    零日漏洞
   </a>
   ）在受害者的 iphone 上植入间谍软件。
  </p>
  <p class="graf graf--p">
   随后的媒体报道援引匿名消息人士的话说，这场攻击针对的是维吾尔族社区，同样的方法也被用来传播针对 Android 和 Windows 的恶意软件。
  </p>
  <p class="graf graf--p">
   在这些报道之后，Volexity 公布了针对维吾尔族人的数字间谍活动的细节，被入侵的网站用 Android 恶意软件感染目标。虽然 Volexity 没有提供任何与谷歌报告重叠的技术指标，但他们推测，在这两种情况下，操作者可能是相同的。公民实验室的报告提供了这些缺失的环节。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     1、瞄准目标
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在2018年11月11日至14日期间，公民实验室观察到15起针对达赖喇嘛私人办公室、西藏中央政府、西藏议会和西藏人权组织的入侵企图。
  </p>
  <p class="graf graf--p">
   在2019年4月22日和5月21日期间观察到两次额外的攻击尝试。受到攻击的大多数人在各自的组织中担任高级职务。
  </p>
  <p class="graf graf--p">
   这些入侵企图是通过 WhatsApp 上的七个虚假人物发送的信息来实现的，
   <strong class="markup--strong markup--p-strong">
    这些假人物被设计成记者、国际倡导组织的工作人员、西藏人权组织的志愿者、以及前往印度的游客
   </strong>
   。
  </p>
  <p class="graf graf--p">
   虚假角色使用的都是带有香港区号（+ 852）的 WhatsApp 电话号码。
  </p>
  <p class="graf graf--p">
   在整个攻击活动中，POISON CARP 展示了在社交工程方面的重大努力。虽然这种手段早已不是新鲜事。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">

  <p class="graf graf--p">
   经过后续点击，链接只是重定向到一个合法的谷歌登录页面。
  </p>
  <p class="graf graf--p">
   第二个 bit.ly 短链接为 MOONSHINE 提供服务，可以确定这些 OAuth 攻击是由同一操作者实施的。
  </p>
  <p class="graf graf--p">
   基于诱骗页面的相似性和使用来自 RiskIQ 的被动 DNS 数据，确定下列网站似乎是由同一个操作者使用的：

  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     结论
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   一个重要发现是 POISON CARP 和 Google Project Zero 和 Volexity 报道的攻击活动之间的联系。它们使用相同的 iOS 漏洞和类似的 iOS 间谍软件植入，可以确定这三个攻击很可能是由同一个操作者执行的、或者一个密切合作的攻击小组共享了资源。
  </p>
  <p class="graf graf--p">
   除了技术上的重叠之外，这些攻击的目标都是与中国有关的少数民族群体：维吾尔族和藏族。
  </p>
  <p class="graf graf--p">
   这些社区经历了数字间谍威胁超过十年，以前的报告经常发现相同的攻击者和恶意软件工具包针对他们。然而，POISON CARP 和相关威胁等级的提升改变了一些游戏规则。
  </p>
  <p class="graf graf--p">
   多年来，
   <strong class="markup--strong markup--p-strong">
    西藏组织已经对可疑的电子邮件、附件和网络钓鱼的迹象了如指掌。然而，POISON CARP 使用了社区并不擅长应对的移动威胁，如果设备运行易受攻击的 iOS 或安卓版本，这种攻击会导致严重的危害
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Poison CARP 使用的社交工程之所以成功，部分原因是虚拟人物角色、并努力使目标个体感到舒适。这种针对性攻击形式在移动聊天应用程序上比通过电子邮件更容易实现。
   </strong>
  </p>
  <p class="graf graf--p">
   针对移动平台的攻击也反映了在世界各地民间社会面临的信息安全威胁中所呈现的一般模式。
  </p>
  <p class="graf graf--p">
   已经有许多报告显示，商业间谍软件供应商专门向政府出售服务，他们的间谍技术产品被用来监视活动人士和记者。
  </p>
  <p class="graf graf--p">
   这些攻击事件表明了
   <strong class="markup--strong markup--p-strong">
    针对移动设备间谍活动之需求的日益增长
   </strong>
   。从对手的角度来看，驱动这种需求的因素是显而易见的 ——
   <strong class="markup--strong markup--p-strong">
    民间社会的组织和动员更多使用的是移动设备。手机数据能揭示一切。
   </strong>
  </p>
  <p class="graf graf--p">
   应对这些威胁需要民间社会和私营企业一起采取行动。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Tibcert 等类似的公民社会安全防御组织的努力是加强西藏组织数字安全的重要步骤，可以作为其他民间社会团体的榜样。
   </strong>
  </p>
  <p class="graf graf--p">
   与此同时，平台提供商应特别关注针对民间社会的威胁。不仅民间社会用户受到数字间谍活动负面影响的风险增加，而且，
   <strong class="markup--strong markup--p-strong">
    这些监视工具会使使用用户都面临危险
   </strong>
   。⚪️
  </p>
