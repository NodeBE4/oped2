  <p class="graf graf--p">
   2002年，Johnny Long 开始研究谷歌搜索查询，揭露易受攻击的系统或易于披露的敏感信息。他把这些方法标记为 Google Dorks。

    </em>

  <ul class="postList">
   <li class="graf graf--li">

    </em>

     filetype:DOC —
    </em>

   </li>
   <li class="graf graf--li">
    <em class="markup--em markup--li-em">
     intext:word1 —
    </em>
    找到包含您要搜索的特定单词的网页和网站。
   </li>
   <li class="graf graf--li">

    </em>

    <em class="markup--em markup--li-em">

    </em>

    <em class="markup--em markup--li-em">

    </em>
    — 显示所有子域。 Asterisk 可以替代搜索查询中的单个或多个单词。

   为什么有用？作为一名调查员，您可能会需要通过头像找到这个人，因为人们通常不会为他们使用的各种社交网络更改个人资料图片；作为记者，您可以执行与过滤配对的图像搜索以揭露虚假新闻。
  </p>
  <p class="graf graf--p">
   如果您需要在社交网络上进行狭窄的搜索，可以使用针对 Vkontakte 的 Findclone 和 Findmevk.com 或者针对 Reddit 的 karmadecay 进行搜索。
  </p>
  <p class="graf graf--p">
   此外，值得一提的是浏览器扩展：适用于 Chrome 的 RevEye 和适用于 Firefox 的“图像搜索”选项。

  <p class="graf graf--p">
   像 CamFind for iOS 这样的移动应用可能会帮助您搜索物理世界中的内容。此外，还有一个
   <a class="markup--anchor markup--p-anchor" data-href="https://www.imageidentify.com" href="https://www.imageidentify.com" rel="nofollow noopener noreferrer" target="_blank">
    Image Identification Project
   </a>
   ，用AI识别图像上的内容。
  </p>
  <p class="graf graf--p">
   图像本身包含许多有用的信息，如相机信息、地理坐标等等 —— 它被称为EXIF数据，如果没有删除，您可能会从中发现很多有趣的信息。
  </p>
  <p class="graf graf--p">
   例如，映射地理坐标以找出拍摄照片的位置或获取相机序列号，有一项特殊服务：stolencamerafinder.com。
  </p>
  <p class="graf graf--p">
   图像编辑工具允许查看元数据，如果您不想要安装复杂程序，跨平台免费软件 Exiftool 可能比较适合您。
  </p>
  <p class="graf graf--p">
   第三种方法是在线查看 EXIF 数据：exifdata.com 或 viewexifdata.com。要删除EXIF数据，您可以使用本地安装的工具：exifpurge.com 或在线进行：verexif.com。
  </p>

    <img alt="" class="aligncenter size-full wp-image-4004" data-recalc-dims="1" height="660" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/隐私4.jpg?resize=1100%2C660&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/隐私4.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/隐私4.jpg?resize=300%2C180&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/隐私4.jpg?resize=768%2C461&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/隐私4.jpg?resize=1024%2C614&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/05/隐私4.jpg?resize=1100%2C660&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 自动化OSINT ——
   </strong>
  </p>
  <p class="graf graf--p">
   互联网是数据的海洋，手动查找信息可能既费时又无效，而且自动化工具可以实现肉眼不会发现的相关性。
  </p>
  <p class="graf graf--p">
   这一切都取决于你的情况，你是否需要使用这些工具，因为大多数工具都需要学习，需要解决复杂的问题。
  </p>
  <p class="graf graf--p">
   因此，如果您只想完成几个简单的任务，前面提到的东西就已经足够了。
  </p>
  <p class="graf graf--p">
   以下是更好的工具。
  </p>
  <p class="graf graf--p">
   SpiderFoot ——
  </p>
  <p class="graf graf--p">
   如果你想自动化OSINT，它是最好的侦察工具之一，因为它可以用来同时查询100多个公共数据源，它的模块化允许微调查询源。
  </p>
  <p class="graf graf--p">
   有四种不同的用例：获取目标的所有内容，了解目标暴露在互联网上的内容，查询黑名单和其他来源以检查目标的恶意与否，通过不同的开源渠道收集情报。最后一个是被动侦察的理想选择。
  </p>
  <p class="graf graf--p">
   theHarvester ——
  </p>
  <p class="graf graf--p">
   这是一个非常简单但有效的工具，用于在信息收集阶段获取有关目标的有价值信息。
  </p>
  <p class="graf graf--p">
   对于被动侦察，theHarvester 使用许多资源来获取Bing、百度、雅虎和谷歌搜索引擎等平台的数据，以及 LinkedIn、Twitter 和 Google Plus 等社交网络数据。
  </p>
  <p class="graf graf--p">
   对于主动侦察，它执行DNS反向查找、DNS TDL扩展和DNS暴力破解。
  </p>
  <p class="graf graf--p">
   Recon-ng ——
  </p>
  <p class="graf graf--p">
   这是另一个很棒的命令行工具，用于彻底、快速地进行信息收集。
  </p>
  <p class="graf graf--p">
   这个功能齐全的Web侦察框架包括一系列用于被动侦察、便利功能、和交互式帮助的模块，可指导您如何正确使用它。
  </p>
  <p class="graf graf--p">
   对于那些熟悉 Metasploit 的人来说，Recon-ng 会更容易学习，因为它们具有相似的使用模式。
  </p>
  <p class="graf graf--p">
   如果您正在寻找能够快速检查公司在互联网上的可见性的强大功能 —— 这是一个可行的工具。
  </p>
  <p class="graf graf--p">
   Maltego ——
  </p>
  <p class="graf graf--p">
   这是一个为分析复杂环境而开发的高级平台。
  </p>
  <p class="graf graf--p">
   除了数据挖掘之外，它还可以进行数据关联并直观地呈现数据。
  </p>
  <p class="graf graf--p">
   Maltego 与您连接的实体（人员，公司，网站，文档等）合作，以便从不同来源收集有关它们的更多信息，以获得有意义的结果。
  </p>
  <p class="graf graf--p">
   该工具的独特之处在于“transforms” — 一个插件库，可帮助运行与外部应用程序不同类型的测试和数据集成。
  </p>
  <p class="graf graf--p">
   FOCA ——
  </p>
  <p class="graf graf--p">
   这是一种从分析的文档中提取隐藏信息和元数据的工具。
  </p>
  <p class="graf graf--p">
   分析所有文档并提取元数据后，它会进行自动元数据分析，以确定哪些文档是由同一用户创建的。
  </p>
  <p class="graf graf--p">
   最新版本仅适用于 Windows。
  </p>
  <p class="graf graf--p">
   Metagoofil ——
  </p>
  <p class="graf graf--p">
   也是一个命令行工具。对网站下载的公共文档进行分析和元数据提取。适用于 pdf，doc，xls，ppt 和其他格式。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— 总结 ——
   </strong>
  </p>
  <p class="graf graf--p">
   在后隐私世界中保持私密权并控制在这个数字海洋中漂浮的信息是很困难的。
  </p>
  <p class="graf graf--p">
   虽然你无法控制关于你的一切，但至少要了解它，这非常重要。
  </p>
  <p class="graf graf--p">
   不言而喻，⚠️ 在数字时代，信息起着关键作用，因此那些知道如何找到信息的人将始终领先一步。
  </p>
  <p class="graf graf--p">
   这就是本文的目的，以展示OSINT如何帮助解决广泛的问题：从深度调查到隐私人身安全。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    文章中介绍的大部分技巧都很简单，但是会非常强大。因此，如果被用于恶意目的就会造成严重的损害，因此希望您能合理使用它们
   </strong>
   。
  </p>
  <p class="graf graf--p">
   您喜欢使用什么工具和技术来收集情报？欢迎留言讨论。⚪️
  </p>
