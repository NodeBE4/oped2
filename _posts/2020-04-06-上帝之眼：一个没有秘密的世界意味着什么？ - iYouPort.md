  <p class="graf graf--p">
   星期六早晨大约10:30，在伦敦北部的伊斯灵顿市，两名骑着轻便摩托车的男子沿着上街的购物走廊飞奔。
  </p>
  <p class="graf graf--p">
   他们戴着头盔、手套、穿着夹克，看起来就像狂躁的电子游戏人物。他们飞快地穿越交通流，其他车子各种避让。
  </p>
  <p class="graf graf--p">
   三到四分钟后，他们突然进入了一片绿树成荫的安静住宅区。他们跳到路边，关闭了引擎，在人行道上卸下头盔的同时开始交谈。
  </p>
  <p class="graf graf--p">
   他们以为自己的对话只有自己知道。但是，他们可能还不知道的是：大约一英里之外，还有另外两个人通过一个没有窗户的房间在看着他们。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们出动了”，Sal 对 Eric 说。
  </p>
  <p class="graf graf--p">
   这俩人坐在伊斯灵顿闭路电视监视（CCTV）控制室的长条形控制台后面，该控制台涂有灰色的油漆，下面铺着地毯，没有装饰。
  </p>
  <p class="graf graf--p">
   Sal 是中年人，Eric 非常年轻。两人都穿着休闲的办公室装束。
  </p>
  <p class="graf graf--p">
   他们可没有闲聊。当那两个骑摩托车的人在街上狂奔时，Sal 正在他的电脑键盘上打字，提示 Eric 看 Camera 10，摩托车人就出现在那个监视器上。
  </p>
  <p class="graf graf--p">
   当骑摩托车的人从 Sal 的视线中消失时，Eric 迅速将他们定位在 Camera 163 监视器上。他用操纵杆将摄像机的距离拉近，直到画面上的车牌清晰可辨。
  </p>
  <p class="graf graf--p">
   Sal 接通了警察局：“我们看到两个可疑的骑轻便摩托车的人在伊斯林頓上街飙车做特技。”
  </p>

  <p class="graf graf--p">
   在这两个监视者面前，是一排带有16个屏幕的巨大显示屏。
   <strong class="markup--strong markup--p-strong">
    它可以从伊斯林頓的180个CCTV摄像机网络中传输实时的监控图像
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这个星期六的早晨相对比较平静。而在这周的早些时候，一名年轻人在公寓被刺伤后死亡；另一名男子从被称为“自杀桥”的地方跳下身亡。
  </p>
  <p class="graf graf--p">
   当天晚些时候，在芬斯伯里公园，这些监控摄像机将花费数小时在35,000名节日游客身上平移，以寻找扒手、醉酒的斗殴者、以及其他各种各样的小恶作剧。
  </p>
  <p class="graf graf--p">
   但目前，那两个骑摩托车耍酷的人是伊斯灵顿唯一的“活动”。
  </p>
  <p class="graf graf--p">
   尽管分别从事这项工作已经15年和4年的 Sal 和 Eric 都在以单调的动作监视着一台又一台摄像机，但我几乎可以看到他们的激动。
  </p>
  <p class="graf graf--p">
   他们认为，那是困扰伊斯灵顿超过一年的两个所谓的帮派成员。他们从行人手中抢走智能手机，然后在黑市上出售。在近233,000名居民的社区中，这类事件每周发生约50次。
  </p>
  <p class="graf graf--p">
   我被两个人的基本奇异之处所吸引，
   <strong class="markup--strong markup--p-strong">
    那两个人似乎不知道他们的一举一动都在受到监视。
   </strong>
  </p>
  <p class="graf graf--p">
   也许他们是罪犯，也许有点变态，但对于这些问题，监视尚无定论。
   <strong class="markup--strong markup--p-strong">
    唯一可以确定的是 —— 监视者看到了他们，但他们没有看到监视者。
   </strong>
  </p>
  <p class="graf graf--p">
   就像狩猎步枪瞄准镜中的鹿一样，摩托车骑手没有表现出任何担惊受怕的迹象。而他们已经被深深地暴露了。
  </p>
  <p class="graf graf--p">
   那个晚上，我坐在几英里外伦敦西南部的一辆移动拖车中，就在沃克斯豪尔地铁站（Vauxhall Underground Station）街对面。几个闭路电视监视屏摆在我们面前，显示来自10个针对附近夜总会进行监视的摄像机提供的图像。
  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">
   Haz 在一个月内有两个周末在这里。这个夜店希望能避免光顾的顾客进行毒品交易而造成法律麻烦，因此，他们委托了一名移​​动CCTV运营商和前警察 Gordon Tyermam，让他的手下人 Haz 密切监视人群。
  </p>
  <p class="graf graf--p">
   有时，夜总会顾客碰巧注意到了其中一个监视摄像头，于是就
   <strong class="markup--strong markup--p-strong">
    通过对着摄像头树中指或裸露乳房以做出回应
   </strong>
   ，这些画面都在 Haz 的视野内。
  </p>
  <p class="graf graf--p">
   此外，成千上万进出俱乐部的男男女女都是夜店的普通娱乐者。
  </p>
  <p class="graf graf--p">
   Haz 说：“这是我迄今为止最出色的、最激动人心的工作。一切都是不可预测的。一直很安静，然后突然就爆发了战斗 ……”。
  </p>
  <p class="graf graf--p">
   Haz 在那辆用来监视的拖车里坐了10个小时，眼睛一直盯着显示器上夜店里的顾客。如果发现毒品交易或打架的情况，他会立即通过对讲机通知夜店的内部保安人员。
  </p>
  <p class="graf graf--p">
   今晚没有毒品交易，没有打架，只有年轻人和醉汉的小愚蠢。
  </p>
  <p class="graf graf--p">
   他惊叹道：“你在闭路电视监视器上看到的东西，使你想到‘这不是成年人的举止。’ 他们往往会忘记自己是谁。”
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    他们真的会忘记他们是谁吗？还是他们只是倾向于忘记有人正在监视着自己？
   </strong>
  </p>

  <p class="graf graf--p">
   1949年，在欧洲专制主义的阴影中，英国小说家乔治·奥威尔（George Orwell）出版了他的反乌托邦杰作《1984》，其严厉的告诫是：“老大哥在看着你。”
  </p>
  <p class="graf graf--p">
   尽管这个想法可能令人不安，但 “watching” 已经历史悠久。
   <strong class="markup--strong markup--p-strong">
    1949年，一家美国公司发布了第一个商用CCTV监视系统。两年后的1951年，柯达向狂暴的公众推出了其布朗尼便携式电影摄影机。
   </strong>
  </p>
  <p class="graf graf--p">
   如今，每年在互联网上共享或存储的图像超过2.5万亿张，更不用说人们保留的数十亿张照片和录像了。
  </p>
  <p class="graf graf--p">
   一家电信公司估计，到2020年，将有61亿人拥有
   <strong class="markup--strong markup--p-strong">
    具有拍照功能的手机
   </strong>
   。
  </p>
  <p class="graf graf--p">
   同时，估计一年内就售出了1.06亿个
   <strong class="markup--strong markup--p-strong">
    新的监控摄像机
   </strong>
   。
  </p>
  <p class="graf graf--p">
   地球上有超过300万台
   <strong class="markup--strong markup--p-strong">
    ATM机
   </strong>
   盯着他们的客户。
  </p>
  <p class="graf graf--p">
   成千上万个被称为
   <strong class="markup--strong markup--p-strong">
    自动车牌识别装置（ANPR）
   </strong>
   的摄像机悬挂在道路上，以抓取超速驾驶者或违规停车者，但在英国，还可以跟踪任何当局感兴趣的人的踪迹。
  </p>
  <p class="graf graf--p">
   现在，佩戴
   <strong class="markup--strong markup--p-strong">
    身体摄像头
   </strong>
   的人数一直在迅速增多，不仅包括警察，还包括医院工作人员和其他许多非执法人员。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    私人用监控设备
   </strong>
   （如 行车记录仪、自行车头盔摄像机、配有镜头以捕捉小偷的门铃）的数量也在迅速增加，它们已迅速成为许多城市居民日常武器库的一部分。
  </p>
  <ul class="postList">

  </p>

   哥伦比亚大学第一修正案研究所（Knight First Amendment Institute）的创办人 Jameel Jaffer 说：“我确实认为我们的生活越来越多地被记录和追踪。我还认为，我们只是开始努力解决这一问题的影响，因此在采用新技术或允许新的监视形式在我们的社会中扎根之前，我们应该考虑一下这些监视的长期影响如何成为现实。”
  </p>
  <p class="graf graf--p">
   如何做出这样的判断？当发生突破而使人们对如何看待世界的观念发生剧变时，做到这点特别容易。实际上，这一关键角色已经出现。
  </p>
  <p class="graf graf--p">

   Marshall 和 Schingler 和我坐在一起。前者是戴着金属框眼镜的瘦高英国人，后者是随和的加利福尼亚人。他们俩都是39岁，前一天的晚餐是庆祝在 Planet 从事全职工作五周年。

  <p class="graf graf--p">
   他们进行了实验，将相对便宜的相机送入轨道，确认它们可以在外太空工作的状况。 “我们考虑可以用这些图像做些什么？” Schingler 说。

  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我们如何利用这些东西造福人类？列出世界上的问题：贫困、住房、营养不良、森林砍伐。如果您拥有有关我们星球的最新信息，那么所有这些问题都将更容易解决。比如如果你提早发现了亚马逊森林的漏洞就可以在火灾发生之前向政府提供有关此信息，结果是什么？”
  </p>
  <p class="graf graf--p">
   Marshall 和 Schingler 在硅谷的一个车库里开发了他们的第一个模型。这个想法是设计一种成本相对较低、鞋盒大小的卫星，以最大程度地减少设计此类技术所需的军事级规模的预算，然后，如 Marshall 告诉我的那样，“发射人类历史上最大规模的卫星群。 ”
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   如今，Planet拥有200多颗在轨卫星，其中约有150颗被称为 Doves，可以在条件适当的情况下每天对每一块土地进行成像。 Planet 拥有远至冰岛和南极洲的地面站。它的客户也一样分布在全球。
  </p>
  <p class="graf graf--p">
   该公司与亚马逊保护协会合作，追踪秘鲁的森林砍伐情况。它已向国际特赦组织提供图像，记录缅甸安全部队对罗兴亚人村庄的袭击。
  </p>
  <p class="graf graf--p">
   在米德尔伯里研究所（Middlebury Institute）的防扩散研究中心，重复进行的全球成像有助于智囊团监视伊朗或朝鲜导弹测试场的突然出现。

  </p>

  </p>

    <img alt="" class="size-full wp-image-10353" data-recalc-dims="1" height="735" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/17-1.png?resize=1100%2C735&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/17-1.png?w=1400&amp;ssl=1 1400w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/17-1.png?resize=300%2C201&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/17-1.png?resize=768%2C513&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/17-1.png?resize=1024%2C685&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/17-1.png?resize=1100%2C735&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/17-1.png?resize=272%2C182&amp;ssl=1 272w" width="1100"/>

The data are overlaid onto maps (below) to provide accurate fire perimeters so firefighters can plan their attacks, predict fire behavior, and identify threats.
   </figcaption>
  </figure>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10354 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-1.png?resize=1100%2C692&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-1.png?w=1400&amp;ssl=1 1400w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-1.png?resize=300%2C189&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-1.png?resize=768%2C483&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-1.png?resize=1024%2C644&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-1.png?resize=1100%2C692&amp;ssl=1 1100w" data-recalc-dims="1" height="692" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-1.png?resize=1100%2C692&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10354" data-recalc-dims="1" height="692" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-1.png?resize=1100%2C692&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-1.png?w=1400&amp;ssl=1 1400w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-1.png?resize=300%2C189&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-1.png?resize=768%2C483&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-1.png?resize=1024%2C644&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/18-1.png?resize=1100%2C692&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10355 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19-1.png?resize=1100%2C717&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19-1.png?w=1100&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19-1.png?resize=300%2C196&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19-1.png?resize=768%2C501&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19-1.png?resize=1024%2C667&amp;ssl=1 1024w" data-recalc-dims="1" height="717" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19-1.png?resize=1100%2C717&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-10355" data-recalc-dims="1" height="717" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19-1.png?resize=1100%2C717&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19-1.png?w=1100&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19-1.png?resize=300%2C196&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19-1.png?resize=768%2C501&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/19-1.png?resize=1024%2C667&amp;ssl=1 1024w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   这是Planet自己的
   <strong class="markup--strong markup--p-strong">
    道德准则
   </strong>
   将发挥作用的地方。它不仅可以拒绝与具有恶意动机的客户合作，而且还不允许客户对所购买的图片拥有唯一的所有权主张。
  </p>
  <p class="graf graf--p">
   另一个重要的限制是技术。Planet 以10英尺的分辨率监视世界，足以分辨出一辆卡车的轮廓，但不能分辨出人类的轮廓。从分辨率上讲，能看清一只脚的技术由另一家卫星成像公司 DigitalGlobe 提供。
  </p>
  <p class="graf graf--p">
   但目前，
   <strong class="markup--strong markup--p-strong">
    只有 Planet 拥有强大的卫星部署能力，能提供整个地球所有陆地的每日图像。
   </strong>
  </p>
  <p class="graf graf--p">
   尽管如此，Planet 还是取得了进展。其他人将有一天跟随它。到那时，他们将如何利用每时每刻监看世界各地的权力？他们的目标会和 Planet 的目标一样仁慈吗？他们会尝试完善分辨率更高、因而具有更高侵入性的卫星摄影吗？
  </p>
  <p class="graf graf--p">
   Marshall 告诉我：“要识别300英里以外的人，您需要一个像公共汽车一样大的摄像头。” 他补充说，无论如何，一家寻求实现这一目标的美国公司将遇到相当大的联邦监管障碍。
  </p>
  <p class="graf graf--p">
   当然，法规可以更改；技术极限的边界也可以更改。就在一两年前，在轨运行中数量最多的卫星的所有者是美国政府，拥有约170颗卫星。如今，Planet 运行的卫星数量超过地球上最强大的国家。
  </p>
  <p class="graf graf--p">
   谁会成为最大的老大哥？
  </p>
  <figure aria-describedby="caption-attachment-10356" class="wp-caption aligncenter" id="attachment_10356" style="width: 1100px">
   <img alt="" class="size-full wp-image-10356 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/20.png?resize=1100%2C737&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/20.png?w=1100&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/20.png?resize=300%2C201&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/20.png?resize=768%2C515&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/20.png?resize=1024%2C686&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/20.png?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="737" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/20.png?resize=1100%2C737&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="size-full wp-image-10356" data-recalc-dims="1" height="737" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/20.png?resize=1100%2C737&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/20.png?w=1100&amp;ssl=1 1100w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/20.png?resize=300%2C201&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/20.png?resize=768%2C515&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/20.png?resize=1024%2C686&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/20.png?resize=272%2C182&amp;ssl=1 272w" width="1100"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-10356">
    Dubbed the “fifth largest intelligence agency,” more than 850,000 volunteers—retirees outfitted with official red vests or armbands—are the eyes and ears of their Beijing, China, neighborhoods. Out in force on holidays, they help direct traffic, give visitor information, and check in on the infirm. But they’re most known for monitoring the streets for suspicious behavior.
   </figcaption>
  </figure>
  <p class="graf graf--p">
   一个名叫 John Goolgasian 的高个子年轻人想向我展示弗吉尼亚州机构 GeoSpark Analytics 如何将犯罪数据与 Planet 图像相匹配。几次点击之后就看到了被极端主义组织博科圣地占领的尼日利亚社区。
  </p>
  <p class="graf graf--p">
   再点击几下，图像就更加熟悉：我在华盛顿特区的邻居，特别是离我家只有几个街区的地方，刚刚接到了一份入室盗窃报告。
  </p>
  <p class="graf graf--p">
   首席运营官 Tom Barton 说， “我希望从现在开始的一年后，我们可以站在这里说，‘天哪，我们确实改变了世界。’”
  </p>
  <p class="graf graf--p">
   当一个年轻的女人向我展示她笔记本电脑上的东西时，我正在思考这个问题的含义。她的名字叫 Annie Neligh，是空军资深人士，现任 Planet 的 “客户解决方案工程”负责人。
  </p>
  <p class="graf graf--p">
   Neligh 的客户之一是德克萨斯州的一家保险公司。该公司怀疑它正在为隐瞒自己安装游泳池的房主续签保险，这会对公司的保单造成40％的损失。因此，它要求 Planet 提供有关德克萨斯州普莱诺市房屋的卫星图像。
  </p>
  <p class="graf graf--p">
   Neligh 向我展示了她的发现。附近1,500处房产的卫星图像，可以清楚地看到520个游泳池闪烁的形状 —— 远远超出了保险公司客户声称的比例。 Neligh 耸了耸肩，露出淡淡的微笑。 “人们撒谎，你知道，” 她说。
  </p>
  <p class="graf graf--p">
   现在她的客户获得了真相。此信息将被如何处理？对普莱诺如此清醒的小村庄进行突袭吗？抬高保费？这些图像可能会显示施工人员正在安装的新的按摩浴缸和西班牙瓦屋顶吗？
  </p>
  <p class="graf graf--p">
   未来就在这里，在其中，真相不仅仅是一个善良的教育者。它是一种武器，既可以对抗偷猎者、盗贼、疯狂的轰炸机，也可以击溃人类天性中的小把戏。人们撒谎，你知道。
   <strong class="markup--strong markup--p-strong">
    这个完全透明的时代已经强加给所有人。
   </strong>
  </p>
  <p class="graf graf--p">
   当我回到旅馆时，我想到了伊斯灵顿那两个轻便摩托车骑手，我想知道他们是否被捕了。我想知道他们是否被认为犯了任何罪行，除了在一个本来很沉闷的早晨做了些有趣的特技之外。
  </p>
  <p class="graf graf--p">
   我想知道他们是否会知道有看不见的陌生人一直在注视着他们，就像一个陌生人现在正在注视着我一样 —— 就在对面那个闭路电视监控器中，看到一个孤独的人在黑暗中空旷的街道上快速行走的景象。寒冷的夜晚，没有穿外套，似乎正在逃离什么。⚪️
  </p>
  <p class="graf graf--p">

    They Are Watching You — and Everything Else on the Planet
   </a>
  </p>
