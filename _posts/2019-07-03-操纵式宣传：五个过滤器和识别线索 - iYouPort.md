      </strong>
     </em>

   </li>

  <p>
   审查是宣传的基础，审查造成宣传、宣传反过来再促进审查。
  </p>

  <p>
   正如 IYP 曾经在
   <a href="https://www.iyouport.org/%E7%8B%AC%E5%AE%B6%EF%BC%9A%E5%8C%97%E4%BA%AC%E7%9A%84%E6%94%BF%E6%B2%BB%E5%AE%A3%E4%BC%A0%E6%A8%A1%E5%BC%8F%E5%8F%91%E7%94%9F%E6%94%B9%E5%8F%98/" rel="noreferrer noopener" target="_blank">
    独家消息中
   </a>
   所分析的，审查并不必需一个“中宣部”，而中国目前已经掌握到了这一关键操作点。反审查的社区和活动家群体应该尽快跟上当局的变化速度。
  </p>
  <p>
   反审查是
   <a href="https://www.iyouport.org/category/%E5%85%B3%E4%BA%8E%E9%80%8F%E6%98%8E%E5%BA%A6%E9%9D%A9%E5%91%BD/" rel="noreferrer noopener" target="_blank">
    透明度革命
   </a>
   启动的媒体改革中最重要的目标这一。更多详见《
   <a href="https://www.iyouport.org/%E8%AD%A6%E6%83%95%E5%BE%AE%E5%A6%99%E7%9A%84%E5%8A%9D%E5%AF%BC%E6%9C%AF%EF%BC%9A%E6%81%90%E6%83%A7%E8%AF%89%E6%B1%82-%E5%9F%8B%E4%B8%8B%E6%83%85%E7%BB%AA-%E6%93%8D%E6%8E%A7%E6%80%9D/" rel="noreferrer noopener" target="_blank">
    大部分审查并非如你平日的吐槽对象那般显而易见 — — 为自由而战（二）
   </a>
   》。审查在呈现不完整的和扭曲的“现实”，审查存在的地方都是宣传，而难以有真相。
  </p>
  <p>
   <strong>
    中国最近正在火热主张的“融媒”想要利用的是一个被称为 propaganda model 的理论，当然，这次是在算法专制的基础上加强它
   </strong>
   。
  </p>
  <p>
   “宣传模型”是爱德华·赫尔曼（Edward S. Herman）和诺姆·乔姆斯基（Noam Chomsky）提出的政治经济概念模型，用于解释宣传和系统性偏见在企业大众媒体中的作用 —— 即 主流媒体。
  </p>
  <p>
   该模型试图解释
   <strong>
    人口是如何被操纵的、以及操纵者如何通过这种宣传在公众心目中“制造”对经济、社会和政治政策的同意
   </strong>
   。
  </p>
  <p>
   因为，
   <strong>
    企业媒体的结构方式会产生内在的利益冲突，这种利益冲突本身就起到了非民主力量的宣传作用
   </strong>
   —— 也就是说，真的不需要中宣部也能达成同样的“作用”。
  </p>
  <p>
   这一模型的提出最早出现在1988年出版的《Manufacturing Consent: The Political Economy of the Mass Media》“制造共识”一书中，
  </p>
  <p>
   Chomsky 在描述媒体的“社会目的”时写道，“……除了边缘元素或相对模糊的学术文献之外，必须严格忽视对制度及其运作方式的深入研究”。该理论假定了五种一般类别的“过滤器”，它们决定了在新闻媒体中呈现的新闻类型。
  </p>
  <p>
   这五个类别是：媒体所有权、盈利模式、采购关系、过度抨击和“恐惧意识形态”。下面逐一分析这些类别，您将更容易理解为什么
   <a href="https://www.iyouport.org/%E7%8B%AC%E5%AE%B6%EF%BC%9A%E5%8C%97%E4%BA%AC%E7%9A%84%E6%94%BF%E6%B2%BB%E5%AE%A3%E4%BC%A0%E6%A8%A1%E5%BC%8F%E5%8F%91%E7%94%9F%E6%94%B9%E5%8F%98/" rel="noreferrer noopener" target="_blank">
    审查并不需要中宣部
   </a>
   。
  </p>
  <figure class="wp-block-image">
   <img alt="" class="wp-image-3139 jetpack-lazy-image" data-lazy-sizes="(max-width: 828px) 100vw, 828px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-1.png?fit=764%2C1024&amp;ssl=1&amp;is-pending-load=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-1.png?w=828&amp;ssl=1 828w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-1.png?resize=224%2C300&amp;ssl=1 224w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-1.png?resize=768%2C1030&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-1.png?resize=764%2C1024&amp;ssl=1 764w" height="1110" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/1-1.png?fit=764%2C1024&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="828"/>

   </noscript>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、所有权
   </strong>
  </p>
  <p class="graf graf--p">
   主流媒体公司的庞大规模和寻求利润的饥渴产生了明确的偏见。
  </p>
  <p class="graf graf--p">
   在19世纪早期，出现过一个激进的英国媒体，解决了工人们的担忧。但是，过度的印花税出现了，
   <strong class="markup--strong markup--p-strong">
    旨在将报纸所有权限制为“受人尊敬的”富人群体
   </strong>
   ，开始改变了新闻界的面貌。尽管如此，那个年代仍然存在一定程度的多样性。
  </p>
  <p class="graf graf--p">
   在第二次世界大战后的英国，激进的或工人友好的报纸，如“每日先驱报”，“新闻纪事报”，“周日公民报”（自此后便失败了或被其他出版物收购了）还有“每日镜报”（至少活到了20世纪70年代后期）都在定期发表文章质疑资本主义制度。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    它们的共同特点是，这些早期的激进派报纸并不受公司所有权的约束，
   </strong>
   因此可以自由地批评资本主义制度。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    而目前的主流媒体，全部都是大公司和大企业集团的一部分，它们向公众提供的信息将必然由于这些利益关系而产生偏见。
   </strong>
  </p>
  <p class="graf graf--p">
   这些企业集团经常超越传统媒体领域，因此具有广泛的经济利益，他们会很担心曝光一些真相。也是
   <strong class="markup--strong markup--p-strong">
    为什么这些企业集团都倾向于将主流传媒机构收入靡下
   </strong>
   ，因为只有这样他们才能继续操纵大众对“事实”的认识。
  </p>
  <p class="graf graf--p">
   换句话说，一切最能危害拥有媒体的寡头之经济利益的新闻项目，都将面临最大的偏见处理和审查。
  </p>
  <p class="graf graf--p">
   如果最大化利润意味着牺牲新闻的客观性，那么最终能生存下来的新闻来源*必须*从根本上带有偏见。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、盈利模式
   </strong>
  </p>
  <p class="graf graf--p">
   第二个过滤器是广告收入为主的盈利模式。大多数报纸都必须吸引广告才能支付生产成本；如果没有足够多的广告，他们将不得不提高报纸的价格。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    整个传媒业都在激烈竞争，争的目标不是真实性和专业性，而是吸引多少广告商。与竞争对手相比，广告收入较少的报纸处于严重的劣势
   </strong>
   。
  </p>
  <p class="graf graf--p">
   十九世纪和二十世纪死掉了很多真正代表工人阶级利益的报纸，都是因为缺乏广告收入而饿死的 —— 显然，贫苦的工人阶级不是广告的完美消费者，他们需要联合对抗的富人阶层才是。
  </p>
  <p class="graf graf--p">
   根据这个过滤器，
   <strong class="markup--strong markup--p-strong">
    新闻是一种“填充物”，广告才是主题，目的是使特权读者看到构成内容的广告，并因此采取任何最有利于吸引目标受众的形式
   </strong>
   。
  </p>
  <p class="graf graf--p">
   于是，任何与“买家情绪”相冲突的内容，比如边缘化社区、少数族裔的声音、反抗力量等等内容，都会被排斥；因为它们与广告商的利益相冲突。结果是，在读者眼中出现的世界并不是真实和完整的。
  </p>
  <p class="graf graf--p">
   简单说，
   <strong class="markup--strong markup--p-strong">
    购买报纸的人是销售给购买广告空间的企业的*产品*
   </strong>
   ；而新闻作为“产品”只要能吸引到眼球就行了，无所谓事实真相和公民知情权。
  </p>
  <p class="graf graf--p">
   如今，最大“媒体” — 即 吸引广告的黑洞是监视资本主义寡头谷歌和 Facebook，于是，和传统的媒介理论一样，这个互联网的经济模式正在杀死异议。换句话说，我们将只剩下官方生成的和批准的政治宣传、以及”被允许”的异议。详细分析见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%BC%82%E8%AE%AE%E7%9A%84%E6%B6%88%E4%BA%A1%EF%BC%9A%E4%B8%BA%E4%BB%80%E4%B9%88%E7%BD%91%E7%BB%9C%E5%8F%98%E5%BE%97%E5%90%8C%E8%B4%A8%E5%8C%96/" href="https://www.iyouport.org/%E5%BC%82%E8%AE%AE%E7%9A%84%E6%B6%88%E4%BA%A1%EF%BC%9A%E4%B8%BA%E4%BB%80%E4%B9%88%E7%BD%91%E7%BB%9C%E5%8F%98%E5%BE%97%E5%90%8C%E8%B4%A8%E5%8C%96/" rel="noopener noreferrer" target="_blank">
    异议的消亡：为什么网络变得同质化
   </a>
   》
  </p>
  <p class="graf graf--p">
   已经悲催到极致了，然而，这些还不是全部。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、采购关系
   </strong>
  </p>
  <p class="graf graf--p">
   第三个过滤器更加致命，与大众媒体新闻的采购有关：“大众媒体通过经济必要性和利益互惠而与强大的信息来源
   <strong class="markup--strong markup--p-strong">
    形成共生关系
   </strong>
   。”
  </p>
  <p class="graf graf--p">
   即使像BBC这样的大型媒体公司也无法在任何地方安置记者。他们只将资源集中在可能发生新闻故事的地方：白宫、五角大楼、唐宁街10号、和其他所谓的中央新闻“终端”。
  </p>
  <p class="graf graf--p">
   某国际大型传媒集团的记者在中国“一带一路”高峰论坛期间于中国的社交媒体上发布了多张该论坛内部的照片。我们的记者留言表示：“人们对里面的事不那么关心，请关注下外面的状况，外面正在封路，去听听老百姓的埋冤吧”。显然，没有被回复。
   <strong class="markup--strong markup--p-strong">
    对于这些大型企业媒体而言，“老百姓”的心声是微不足道的，权势财团才是与它们产生共生关系的关键
   </strong>
   。
  </p>
  <p class="graf graf--p">
   您还觉得主流媒体长期关心马云、王健林的小心思而无视民间反抗力量和苦难这件事“很奇怪”吗？
  </p>
  <p class="graf graf--p">
   顺便说，将马云收购南华早报解释为中宣部的渗透至少是不全面的，收购主流传媒只是作为大型企业的必经之路；阿里巴巴首先是寡头，其次才是“党员”，当然，中国执政党一直在努力将这一顺序颠倒过来。
  </p>
  <p class="graf graf--p">
   这在绝大多数国家都一样。虽然英国报纸可能偶尔抱怨新工党的“spin-doctoring”，但它们依旧要依赖于“总理的个人发言人”对政府新闻的声明。
  </p>
  <p class="graf graf--p">
   商业公司和贸易组织也是这一共生关系中的主体。
   <strong class="markup--strong markup--p-strong">
    冒犯这些强大新闻来源的编辑和记者
   </strong>
   （通过质疑所提供材料的准确性或偏见性）
   <strong class="markup--strong markup--p-strong">
    很可能会而受到威胁。
   </strong>
  </p>
  <p class="graf graf--p">
   媒体已经不愿意发表会损害企业集团利益的文章，这些利益为他们提供了他们所赖以生存的资源。
  </p>
  <p class="graf graf--p">
   这种共生关系也导致了“道德分工”，其中官员被赋权拥有并提供了所谓的“事实“，记者只是“得到了“它们。然后，记者采取的是不加批判的态度，使内容能够有利于企业价值而不会遇到认知失调。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    4、过度抨击
   </strong>
  </p>
  <p class="graf graf--p">
   这就是第四个过滤器 ‘flak’，即 对媒体声明或[电视/广播]节目的否定回应。采取的形式包括信件、电报、电话、请愿、诉讼、演讲和法案的形式，在国会和其他地方‘投诉、威胁和惩罚的行动’。[
   <a class="markup--anchor markup--p-anchor" data-href="https://www.uow.edu.au/~sharonb/columns/probe11.html" href="https://www.uow.edu.au/~sharonb/columns/probe11.html" rel="nofollow noopener noreferrer" target="_blank">
    “The Decline of the Global Climate Coalition”
   </a>
   ]
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    商业组织经常聚集在一起形成高射炮机器。一个例子就是美国的全球气候联盟（GCC），包括化石燃料和汽车公司，如埃克森、德士古和福特等
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这个 GCC 由全球最大的公关公司之一美国博雅公关公司（Burson-Marsteller）发起的，目的就是攻击气候科学家的可信度、将全球变暖的警告称为“恐慌故事”。[在这里看到：
   <strong class="markup--strong markup--p-strong">
    美国博雅公关公司帮助中国海康威视的美国子公司完成“战略规划和指导”等任务
   </strong>
   《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%8F%AF%E6%80%9C%E6%AD%A3%E4%B9%89%E6%B2%A1%E6%9C%89%E9%92%B1/" href="https://www.iyouport.org/%E5%8F%AF%E6%80%9C%E6%AD%A3%E4%B9%89%E6%B2%A1%E6%9C%89%E9%92%B1/" rel="noopener noreferrer" target="_blank">
    可怜正义没有钱
   </a>
   》]
  </p>
  <p class="graf graf--p graf--startsWithSingleQuote">
   ‘flak’ 是指对媒体声明或节目的否定回应。 这个术语被用来描述那些诋毁组织或个人的努力，这些组织或个人由于不同意或怀疑有利于既定权力的普遍假设，而成为被攻击目标（例如反权威和反建制的社会组织或个人） 。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、恐惧意识形态
   </strong>
  </p>
  <p class="graf graf--p">
   这个过滤器就非常简单了，任何了解冷战中史上最强大的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%E5%BF%83%E7%90%86%E6%88%98%E5%92%8C%E4%BF%A1%E6%81%AF%E6%88%98/" href="https://www.iyouport.org/category/%E5%BF%83%E7%90%86%E6%88%98%E5%92%8C%E4%BF%A1%E6%81%AF%E6%88%98/" rel="noopener noreferrer" target="_blank">
    心理战
   </a>
   历史的人都明白它意味着什么。它也是西方公民社会对主流媒体的抨击中一个最大的目标。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    恐惧意识形态让媒体成为了政权信息战中的武器
   </strong>
   。
  </p>
  <p class="graf graf--p">
   制造恐惧有一种特别高效的作用，我们有过具体分析，详见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ad%a6%e6%83%95%e5%be%ae%e5%a6%99%e7%9a%84%e5%8a%9d%e5%af%bc%e6%9c%af%ef%bc%9a%e6%81%90%e6%83%a7%e8%af%89%e6%b1%82-%e5%9f%8b%e4%b8%8b%e6%83%85%e7%bb%aa-%e6%93%8d%e6%8e%a7%e6%80%9d/" href="https://www.iyouport.org/%e8%ad%a6%e6%83%95%e5%be%ae%e5%a6%99%e7%9a%84%e5%8a%9d%e5%af%bc%e6%9c%af%ef%bc%9a%e6%81%90%e6%83%a7%e8%af%89%e6%b1%82-%e5%9f%8b%e4%b8%8b%e6%83%85%e7%bb%aa-%e6%93%8d%e6%8e%a7%e6%80%9d/" rel="noopener noreferrer" target="_blank">
    警惕微妙的劝导术：恐惧诉求 — — 埋下情绪 操控思想
   </a>
   》；
   <strong class="markup--strong markup--p-strong">
    人工恐惧的创造方式具有双重目的：一边是为了摆脱你不喜欢的人，另一边则是吓唬其余人。当人们受到惊吓时，他们就会依附于权威。
   </strong>
  </p>
  <p class="graf graf--p">
   当前的伊斯兰恐惧症正在成为新的“恐惧意识形态”来源。
  </p>
  <blockquote class="graf graf--blockquote">
   So I think when we talked about the “fifth filter” we should have brought in all this stuff — the way artificial fears are created with a dual purpose… partly to get rid of people you don’t like but partly to frighten the rest.
   <br/>
   Because if people are frightened, they will accept authority.
  </blockquote>
  <blockquote class="graf graf--blockquote">
   — Noam Chomsky
  </blockquote>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    宣传是为了操纵人群
   </strong>
   。对人群的操纵在人类历史上可以追溯到公元前5世纪，锡拉丘兹的诉讼当事人试图在法庭上提高他们的说服力。[ Aristotle, The Art of Rhetoric, translated with an introduction by H.C. Lawson-Tencred (New York, NY: Penguin Group, 2004): 1–13.；Cheryl Glean (1997). Rhetoric Retold: Regendering the Tradition from Antiquity through the Reniassance. Illinois: SIU Press. pp. 33, 60.]
  </p>
  <p class="graf graf--p">
   操纵意味着“通过巧妙的、不公平的或阴险的手段来控制或玩弄目标群体/个人，特别是为了自己的优势。” 听起来很消极。
   <strong class="markup--strong markup--p-strong">
    但是必须澄清，人群操纵并非完全贬义的。
   </strong>
  </p>
  <p class="graf graf--p">
   政治就是这样一种东西，即 在正确的意义上是“正确的东西并不足以让你赢得胜利。是*政治技术*在决定政治成功。要想成功需要了解如何组织和如何沟通的技术。大多数政治技术都是哲学中立的。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “你应该学习如何获胜，这是你的理念” —— 这句话是 Morton C. Blackwell 在一个著名的相关演讲中所说的。也许听起来特别马基雅维利，不过从一个平衡的角度上看，它是事实。就如 IYP 介绍的社会行动技巧（下个月还有更多具体介绍）。但必须平衡，
   <strong class="markup--strong markup--p-strong">
    如果你使用阴暗的手段即便达成了正义的目的（也许很难达成）也不会持久立足
   </strong>
   。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    与宣传的结盟 ——
   </strong>
  </p>
  <p class="graf graf--p">
   人群操纵者和宣传者的结盟可以达成更高的效果。
  </p>
  <p class="graf graf--p">
   根据 Edward Bernays 的说法，
   <strong class="markup--strong markup--p-strong">
    宣传者必须让他的目标群体预先测试一条信息。消息本身必须提前试水，因为无效的消息比没有消息更糟糕
   </strong>
   。
  </p>
  <p class="graf graf--p">
   社会科学家 Jacques Ellul 将这种试水活动称为“前宣传”，如果要使主要信息有效，这一点就至关重要：
  </p>
  <blockquote class="graf graf--blockquote">
   <em>
    旨在改变观点和态度的直接宣传必须具有先前宣传，这种宣传具有社会性、缓慢性、一般性，
    <strong class="markup--strong markup--blockquote-strong">
     目的是寻求创造气氛，有利于初步态度的氛围。没有预先宣传，任何直接宣传都不会有效
    </strong>
    ……
   </em>
  </blockquote>
  <p class="graf graf--p">
   Jacques Ellul 在《宣传：人的态度的形成》一书中
   <strong class="markup--strong markup--p-strong">
    将社会宣传比作耕作，将直接宣传比作播种，也就是说，如果没有前者，后者即便再卖力也是结不出果实的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   社会学宣传是一种社会寻求通过根据一种模式统一其成员的行为，渗透其生活方式，从而将其自身强加于其目标群体，将最大数量的个体融入宣传目标自身的现象。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    基本上社会学宣传旨在通过利用所有社会潮流，通过长期渗透和渐进适应来发展对既定秩序的遵守或捍卫，从而增强一致性
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该宣传的要素是渗透个人的生活方式，然后，个人就会开始在电影、写作或艺术中表达它，而没有意识到它压根不是自己的想法
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这种非自愿行为通过广告、电影、教育和杂志在社会中扩张。 “无论是有意的还是无意的，整个人群都在以这种方式表达自己；并且其次表明，它的影响力更多地体现在整个生活方式上。”
  </p>
  <p class="graf graf--p">
   这种类型的宣传有可能不是故意的，而是在文化或国家内自发地或不知不觉地涌现出来。这种宣传强化了个人的生活方式，并将这种生活方式表现得最好。社会学宣传为个人根据个人生活方式做出的善恶判断创造了无可争议的标准。
  </p>
  <p class="graf graf--p">
   请注意，
   <strong class="markup--strong markup--p-strong">
    社会学宣传不一定会导致行动，但是，它可以为直接宣传做好准备
   </strong>
   。从那时起，这种社会学宣传的个人便会认为，以这种方式生活的人都站在天使一边，而那些不这样做的人则是坏人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这类例子数不胜数。最眼下的比如
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%E7%9B%91%E8%A7%86%E5%8C%BA%E5%9D%97%E9%93%BE%E5%92%8C%E5%8F%8D%E7%8E%B0%E9%87%91%E4%B9%8B%E6%88%98/" href="https://www.iyouport.org/category/%E7%9B%91%E8%A7%86%E5%8C%BA%E5%9D%97%E9%93%BE%E5%92%8C%E5%8F%8D%E7%8E%B0%E9%87%91%E4%B9%8B%E6%88%98/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     反现金之战
    </strong>
   </a>
   。想想看支持反现金的都是谁？—— 金融权力的垄断者；无现金社会是什么时候出现的概念？—— 是在在线支付如微信、支付宝变成“超酷”亚文化之后而不是之前；以至于隐私倡导者惊讶地发现，他们对现金的支持被大众认为是保守的和落伍的，是“老太太才会拿着钱包出门”的可笑。这就是社会学宣传的成功 —— 当直接宣传 即 无现金社会出现时，人们只有拥护，就仿佛扔掉现金是来自他们“自己的”想法一般自然而然。
  </p>
  <p class="graf graf--p">
   再想想，强烈宣传“融媒”的是谁？—— 中国执政党，习近平的原话是：“要着力打造一批形态多样、手段先进、具有竞争力的新型主流媒体，建成几家拥有强大实力和传播力、公信力、影响力的新型媒体集体”；流量经济的深入人心是在什么时候出现的？—— 是在“融媒”倡导之前、所谓的打赏创造新中产之后；以至于去中心化互联网自由倡导者发现这件事的时候已经无法将人们拉出微信了，“去Mastodon干什么？你写东西有人给你打赏吗？”但是，那极少数所谓“成功”的微信公众号都是谁？当你持有执政党不喜欢的意见时，甭说“成功”，连存活都没有机会。
  </p>
  <p class="graf graf--p">
   很简单，
   <strong class="markup--strong markup--p-strong">
    是人们先“知道”什么样的内容才能赚钱，这一盈利模型的社会学宣传无比地成功之后才有的直接宣传，此时便无人会反驳它了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    中国发生的事就如很久以前 Bernays 所指出的：通过*识别*和签约那些影响力大的公众舆论人物（关键专家、名人、博主等）即可加快这一过程
   </strong>
   。还记得中国微博崛起后执政党做的第一件事是什么吗？没错，收编大V。还有人觉得曾经的“微博言论自由鼎盛时段”有什么值得怀念的吗？那不过是为了引诱影响力出现 —— 即 收编目标的出现 —— 而故意松动的闸口。
  </p>
  <p class="graf graf--p">
   集中化社交媒体是没有言论自由的。因为控制权不属于用户。
  </p>

  <p>
   <strong>
    权威 ——
   </strong>
  </p>
  <p>
   操纵者可以是演说者、团体、名人，或者任何可以通过号召将人群引入一个特定位置的人。亚里士多德认为，操纵者的精神或信誉有助于他的说服力。
  </p>
  <p>
   声望极高的操纵者可以使他的目标人群的重要能力瘫痪，并引起尊重和敬畏。
  </p>
  <p>
   权威来自声望，这可以通过“获得的声望”（例如职称头衔、制服、法官的长袍）和“个人声望”（即 内在权力）所产生。
  </p>
  <p>
   成功是影响个人声望的最重要因素。Le Bon 写道：
   <strong>
    “从声望开始受到质疑的第一分钟起，它就不再是声望了。”
   </strong>
   因此，操纵者应该阻止这种讨论并与人群保持一定距离，以免他的错误破坏他的声望。
  </p>
  <p>
   操纵者控制人群的能力尤其取决于他或她的视觉效果、声音和口头传递能力。温斯顿丘吉尔和阿道夫希特勒在这方面是最强势的，成为了主要的修辞学家。
  </p>
  <p>
   温斯顿丘吉尔将自己的当众发言经验命名为“修辞的脚手架”，并概述了他认为是任何有效言论的基本要素。[Winston S. Churchill, “The Scaffolding of Rhetoric”, in Randolph S. Churchill, Companion Volume 1, pt. 2, to Youth: 1874–1900, vol. 1 of
   <em>
    the Official Biography of Winston Spencer Churchill
   </em>
   (London: Heinmann, 1967): 816–21.]
  </p>
  <p>
   阿道夫希特勒相信他可以应用他在第一次世界大战期间痛苦地从盟军那里学到的宣传教训，然后将这些教训应用于德国。以下几点有助于深入了解他在舞台表演背后的想法 —— ⚠️对于读者来说，以下是识别宣传的关键：
  </p>
  <p>
   <strong>
    呼吁群众
   </strong>
   ：“[宣传]必须始终专注于群众”，而不是“受过科学训练的知识分子”;
  </p>
  <p>
   <strong>
    瞄准情绪
   </strong>
   ：“[宣传]必须针对情感，而且只能在非常有限的程度上瞄准所谓的智力”;
  </p>
  <p>
   <strong>
    保持你的信息简单
   </strong>
   ：“宣传多方面是错误的…大众的接受度非常有限，他们的智力很少，但他们的遗忘能力是巨大的”;
  </p>
  <p>
   <strong>
    为最坏的情况做好准备
   </strong>
   ：“[准备]战斗，从而[帮助]保护他免于失望。在此之后，对他使用的最可怕的武器似乎只是确认他的宣传者告诉他的是什么；于是同样强化了他对政府主张的信念，而另一方面又加剧了他对邪恶敌人的愤怒和仇恨”;
  </p>
  <p>
   <strong>
    不要做半个陈述
   </strong>
   ：“宣传的任务绝对不是要对真理进行客观研究，它的任务是始终如一地坚定不移地为自己的权利服务。“
  </p>
  <p>
   <strong>
    不断重复你的信息
   </strong>
   ：“[宣传技术]必须将自己局限于几个点上并反复重复。在这个世界中，持续存在是成功的第一个也是最重要的要求。通过例子而不是通过论证来引导人群。为了巩固影响，不能过分远离观众。如果是的话，影响力将为零……
  </p>
  <p>
   前面都比较简单，最后这句话，即 关于“不能过分远离观众”的观点就是景观政治，当下最值得警惕的变化之一，它在互联网媒介下被史无前例地抬高了能力。请参加 IYP 早前做过的分析《
   <a href="https://www.iyouport.org/%E4%B8%AD%E5%9B%BD%E6%94%BF%E5%BA%9C%E5%B7%B2%E6%94%80%E4%B8%8A%E6%99%AF%E8%A7%82%E6%94%BF%E6%B2%BB%E7%9A%84%E8%BE%B9%E7%BC%98/" rel="noreferrer noopener" target="_blank">
    中国政府已攀上景观政治的边缘
   </a>
   》。
  </p>
  <p>
   纳粹党利用宣传在希特勒周围发展了一种人格崇拜。伊恩·克肖（Ian Kershaw）等历史学家强调了希特勒作为演说家的技巧所带来的
   <strong>
    心理影响
   </strong>
   。Roger Gill 曾对此评论说：“他[希特勒]动人的演讲吸引了大量德国人的思想和心灵：
   <strong>
    他几乎催眠了所有观众
   </strong>
   ”。[Roger Gill (2006).
   <a href="https://books.google.com/books?id=vIbI2qSvYPIC&amp;pg=PA259" rel="noreferrer noopener" target="_blank">
    <em>
     Theory and Practice of Leadership
    </em>
   </a>
   . SAGE. p. 259.]
  </p>
  <p>
   IYP 已经对宣传做出了很多分析，接下来还会有其他分析，这些内容综合起来也许仍不是全部；因为，就如威胁情报一样，攻击者会在不断更新的手段中获得升级，识别和抵制的方案也需要不断被更新。
  </p>
  <p>
   希望这些内容能帮助您保持清醒，和/或在必要的时候为了正义的目的合理利用。⚪️
  </p>
