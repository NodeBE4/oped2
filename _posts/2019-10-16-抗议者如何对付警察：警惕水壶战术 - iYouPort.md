  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    1、什么是水壶？
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “水壶”是警察用来控制街头抗议者的战术。
  </p>
  <p class="graf graf--p">
   穿着全套防暴装备的警察利用他们的身体在一群抗议者周围形成“警戒线”。警戒线内的区域被称为“水壶”，以这种方式封闭抗议者的过程被称为 Kettling。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “水壶”可以很大 —— 装下几千人，也可以很小 —— 比如高度定位于组织者、或者在最具战略性的位置，装下几十人。
  </p>
  <p class="graf graf--p">
   在设置水壶时，警察会缓慢前进，移动编队，有时敲打他们的盾牌以恐吓抗议者。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一旦抗议者进入水壶，他们就被强行关在那里。任何人都不得离开。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在水壶内，抗议者没有任何食物或水，也无法使用厕所。
   </strong>
  </p>
  <p class="graf graf--p">
   很快就会逼迫人群沸腾 —— 这就是“烧水”。
  </p>
  <p class="graf graf--p">
   接下来就是大规模逮捕，暴力冲突。
  </p>
  <p class="graf graf--p">
   这种战术有着非常悠久的历史。
  </p>
  <p class="graf graf--p">
   在美国和欧洲很流行。
  </p>
  <p class="graf graf--p">
   1986年，德国警察在汉堡 Heiligengeistfeld 的反核抗议示威活动中第一次使用了这种方法。
  </p>
  <p class="graf graf--p">
   最近的使用是法国的黄背心运动中，在这里看到图文和视频介绍《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E7%8E%B0%E4%BB%A3%E7%BB%B4%E7%A8%B3%E6%88%98%E6%9C%AF%E4%B8%80%E7%9E%A5/" href="https://www.iyouport.org/%E7%8E%B0%E4%BB%A3%E7%BB%B4%E7%A8%B3%E6%88%98%E6%9C%AF%E4%B8%80%E7%9E%A5/" rel="noopener noreferrer" target="_blank">
    现代维稳战术一瞥
   </a>
   》；直到今年初的第16次抗议活动，已经有
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E4%BB%96%E4%BB%AC%E4%B8%8D%E5%85%B3%E5%BF%83%E4%BD%A0%E7%9A%84%E6%AD%BB%E6%B4%BB%EF%BC%8C%E5%9B%A0%E4%B8%BA%E4%BB%96%E4%BB%AC%E6%98%AF1/" href="https://www.iyouport.org/%E4%BB%96%E4%BB%AC%E4%B8%8D%E5%85%B3%E5%BF%83%E4%BD%A0%E7%9A%84%E6%AD%BB%E6%B4%BB%EF%BC%8C%E5%9B%A0%E4%B8%BA%E4%BB%96%E4%BB%AC%E6%98%AF1/" rel="noopener noreferrer" target="_blank">
    8,000多人被捕，500人受重伤，2000多人被监禁，12人死亡，20多人失明，6人失去了手
   </a>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    从80年代到现在，这种战术从未间断。而且最近两年还在美国重新引起了震惊
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果抗议者试图突破围困，警察就会使用盾牌、警棍等武器进行回击。
  </p>
  <p class="graf graf--p">
   这些全副武装的防暴警察都是经过专门的打斗训练的。一般抗议者很难赢过他们。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-5786 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-7.png?resize=1100%2C511&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-7.png?w=1528&amp;ssl=1 1528w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-7.png?resize=300%2C139&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-7.png?resize=768%2C357&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-7.png?resize=1024%2C476&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-7.png?resize=1100%2C511&amp;ssl=1 1100w" data-recalc-dims="1" height="511" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/06/2-7.png?resize=1100%2C511&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    2、何时以及为什么警察会使用水壶战术？
   </strong>
  </p>
  <p class="graf graf--p">
   这本身是个维稳战术，如果简单回答这个问题的话那就是：维稳。
  </p>
  <p class="graf graf--p">
   任何时候警察如果觉得和平抗议队伍中“出现了骚乱”，他们就会采取这种方法。
  </p>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    请注意：骚乱有可能是警察当局故意制造的。
   </strong>
  </p>
  <p class="graf graf--p">
   大多数群体运动都会被恶意渗透，以帮助当局造就镇压的合理性。
  </p>
  <p class="graf graf--p">
   这不意味着运动本身有什么问题，而是说组织者应该及时反应过来，抓住渗透者并排除渗透者。
  </p>
  <p class="graf graf--p">
   被渗透并被故意制造混乱的可能性是大规模行动的最明显弱点，因为掌握一切是很难的；而且随着行动加深，很多活动家本身会变得更加激进。
  </p>
  <p class="graf graf--p">
   要想避免这种弱点也有办法，详见我们曾经介绍过的（这里是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/1%e3%80%81%e5%a8%81%e8%83%81%e6%84%9f%e5%8f%af%e8%83%bd%e6%af%94%e4%ba%8b%e7%89%a9%e6%9c%ac%e8%ba%ab%e6%9b%b4%e6%9c%89%e6%95%88%ef%bc%9a%e6%88%98%e7%95%a5%e6%80%a7%e7%9b%b4%e6%8e%a5%e8%a1%8c%e5%8a%a8/" href="https://www.iyouport.org/1%e3%80%81%e5%a8%81%e8%83%81%e6%84%9f%e5%8f%af%e8%83%bd%e6%af%94%e4%ba%8b%e7%89%a9%e6%9c%ac%e8%ba%ab%e6%9b%b4%e6%9c%89%e6%95%88%ef%bc%9a%e6%88%98%e7%95%a5%e6%80%a7%e7%9b%b4%e6%8e%a5%e8%a1%8c%e5%8a%a8/" rel="noopener noreferrer" target="_blank">
    汇总版链接
   </a>
   ）
  </p>
  <p class="graf graf--p">
   根据法律，警方不应该使用水壶“直接或间接地扼杀或阻止抗议”。Kettling一直受到许多民权和激进组织的强烈批评。
   <strong class="markup--strong markup--p-strong">
    它涉及让抗议者在身体严重不适、甚至生命危险的情况下持续数小时，并对抗议产生强大的威慑作用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   但是，只要警方能找到任何理由确认“骚乱”，就可以采取这种方法。
  </p>
  <p class="graf graf--p">
   所以，⚠️非暴力直接行动的抗议者需要有成熟的管理机制，避免被恶意渗透制造混乱。
  </p>
  <p class="graf graf--p">
   ⚠️请注意，有些时候警方会故意制造“假水壶” —— 也就是说，故意让抗议者见到似乎“水壶”正在形成，这种情况下任何有点经验的活动家就会下意识迅速撤离此地；但是如果它是假的，这意味着警方可以很方便地将抗议者驱逐出一个（有可能是很重要的）战略位置。
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/uk/2011/apr/18/police-kettling-protest-liberty" href="https://www.theguardian.com/uk/2011/apr/18/police-kettling-protest-liberty" rel="noopener noreferrer" target="_blank">
    在这看到
   </a>
   一个在伦敦发生的案例。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    3、警察可以将人们关在水壶里多久？
   </strong>
  </p>
  <p class="graf graf--p">
   只要警方认为“需要”，他们就可以持续“烧水”。
  </p>
  <p class="graf graf--p">
   法律角度上警方无权进一步拘禁，但是，
   <strong class="markup--strong markup--p-strong">
    这其中有很大的自由裁量权
   </strong>
   。2010年12月的学生抗议等大型示威活动中发生的“水壶”战略，抗议者被围困了十个小时。
  </p>
  <p class="graf graf--p">
   ⚠️尤其是缺乏经验的抗议者，非常容易被困在“水壶”里，因为他们对警方的行动迹象观察不周。
  </p>
  <p class="graf graf--p">
   在黄背心抗议中之所以有八千多人被捕、致残，正是因为其中很多参与者都是缺乏经验的。
  </p>
  <p class="graf graf--p">
   记者也经常被困在“水壶”中，也会遭到殴打，警方会声称记者“没有出示证件”，事实上大多数时候是警察不给记者出示证件的机会。
  </p>
  <p class="graf graf--p">
   当发生“烧水”时，医务人员也很难接近以救援。
  </p>
  <p class="graf graf--p">
   但是，⚠️请注意，对付这种战术必须使用智慧，而非携带杀伤性武器；因为你很可能没有机会使用武器的情况下就被*合法*镇压了 —— 携带致命武器是镇压最合理的理由，不管它有没有被使用。
  </p>

  <p class="graf graf--p">
   如果他们“ 合理地怀疑 ”您携带的物品是非法的，或者有明确被授权，警察就可随时进行搜查。
  </p>
  <p class="graf graf--p">
   1994年英国“刑事司法和公共秩序法”第60条允许警察搜查任何人，⚠️如果警察认为存在严重暴力的可能性，或者他们有理由相信抗议者携带武器或“危险工具”，就可以搜查。
  </p>
  <p class="graf graf--p">
   上述授权通常被用于镇压大规模抗议。
  </p>
  <p class="graf graf--p">
   但是理论上警方无权知道你的真实姓名和地址，也不应该给你拍照。
