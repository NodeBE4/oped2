  <p class="graf graf--p">
   前面的内容您可能看累了。那么我们来轻松一下；在愉快中学习。
  </p>
  <p class="graf graf--p">
   目前这个世界在个人和政治上已经筋疲力尽了吗？或者只是需要一点休眠以熬过漫长的寒冬？
  </p>
  <p class="graf graf--p">
   不一定。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当你在现实中感觉无所作为的时候，你会做点什么？你会抱怨这个世界的堕落，还是躲起来悄悄打磨你的枪法？
   </strong>
  </p>
  <p class="graf graf--p">
   我不知道；但可以肯定，抱怨没有用，💡而技能的精进可以帮助你在任何时候获得突破性的灵感。
  </p>
  <p class="graf graf--p">
   这就是游戏能帮你做到的事。
  </p>
  <p class="graf graf--p">
   本文希望能探索一些能帮您打磨技能的东西。也就是
   <strong class="markup--strong markup--p-strong">
    抵抗游戏
   </strong>
   。
  </p>
  <p class="graf graf--p">
   任何人都能使用它们。一些人已经开始将它们
   <strong class="markup--strong markup--p-strong">
    用于教育工具来教授政治、社会或经济的现实
   </strong>
   ；另一些人用它们来
   <strong class="markup--strong markup--p-strong">
    挖掘灵感，希望能克服现实中有问题的系统以寻求出路
   </strong>
   。
  </p>
  <p class="graf graf--p">
   不论如何，最终这些游戏是为培训注入一些趣味性的好方法，💡同时很有可能为您的战略规划带来创造力。
  </p>
  <p class="graf graf--p">
   这些游戏的玩法可以以一种低风险（并且压力较小）的方式来开始规划新的社会运动、或锻炼团队的集体力量，没有实质上的威胁性。
  </p>
  <p class="graf graf--p">
   本文即将介绍的游戏涵盖各种风格 —— 从合作到角色扮演到标准的游戏都包括；每一种都有自己的个性。有些能描绘你内心的甘地，有些会让你生活在最疯狂的起义幻想中，有些只会让你发笑。
  </p>
  <p class="graf graf--p">
   所以，现在开始探索属于你的游戏吧。
  </p>

  <p class="graf graf--p">
   这款游戏诞生刚刚超过一年，是 TESA 游戏团队的成果，也被称为
   <strong class="markup--strong markup--p-strong">
    *教育和社会行动工具箱*
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在游戏中，所有玩家都在同一个团队中，运行不同部分的运动，联合起来努力击败“这个体制”。
  </p>
  <p class="graf graf--p">
   首先，你需要选择一个问题或重点关注故事：它可以是像反政府运动那样现实的东西，也可以是像“小龙的正义”这样虚构的战斗。当然会推荐您选择实战模拟。
  </p>
  <p class="graf graf--p">
   玩家
   <strong class="markup--strong markup--p-strong">
    通过合作
   </strong>
   实现挑战，不论输赢所有人共进退。
  </p>
  <p class="graf graf--p">
   玩家的每一次进击，“这个体制”就会反击，导致游戏发生紧张局面 ——
   <strong class="markup--strong markup--p-strong">
    玩家会面临逮捕、内讧、渗透、监视和其他各种障碍
   </strong>
   。
  </p>
  <p class="graf graf--p">
   是的，非常真实。
  </p>
  <p class="graf graf--p">
   💡如果您想探索组织者和活动家在创建反抗行动中共同努力的尝试和有可能遭遇的磨难，这个游戏的挑战就能发挥作用。
  </p>
  <p class="graf graf--p">
   显然对于缺乏实战经验的社会来说它会很有用。
  </p>
  <p class="graf graf--p">
   将您的抵抗活动（或重点故事）集中在一起可以帮助非组织者或新活动家立即参与到游戏中来；它也是解决教育游戏巨大挑战的关键方法：如何让它们真正有趣且有吸引力，而不仅仅是强大的训练工具。
  </p>
  <p class="graf graf--p">
   对于 Rise Up!，五个玩家一起上会效果很好，即便只有两个玩家也可以运行，但趣味性会少一些。所以你能感受到，
   <strong class="markup--strong markup--p-strong">
    它很侧重联合能力的培养
   </strong>
   。
  </p>
  <p class="graf graf--p">
   年轻人和年长的人可以
   <strong class="markup--strong markup--p-strong">
    共同制定战略
   </strong>
   ，思考最紧迫的是什么，增加成员还是深入新的地区，
   <strong class="markup--strong markup--p-strong">
    如何在互联网上组织或与媒体合作
   </strong>
   。
  </p>
  <p class="graf graf--p">
   你最终可能会辩论出最有效的战术选择。
  </p>
  <p class="graf graf--p">
   此外，如果你最终入狱或进了医院，游戏可能会在你没有被释放的情况下结束。
  </p>
  <p class="graf graf--p">
   这就是为什么我建议添加一条规则，即 除非每个人都平安回家，否则不能结束。要抵抗到底。
  </p>
  <p class="graf graf--p">
   当然，它依旧是希望激发人们反侦查反追捕的技能。您能从中获得的知识和灵感都可以运用在现实生活中。因为游戏很真实。
  </p>
  <p class="graf graf--p">
   一个很大的优点是该套装包括两个游戏板，一个用于标准游戏，另一个用于更简化的版本。您可以从简单的开始练习。
  </p>
