   学校和真实生活之间的一个重要区别在于：学校先教你知识然后考试，生活则是先考试再教你知识。
  </p>
  <p class="graf graf--p">
   你不需要亲自被考糊了才能学到东西，要知道一直不断有人被“考糊”，他们的教训应该成为更多人的知识。
  </p>
  <p class="graf graf--p">
   ⚠️我们必须重点推荐这篇报告，向更多人推荐；因为，这其中分析到的所有政府抓人和取证的手段，
   <a href="https://twitter.com/DonEvansWm/status/1174606337730154496" rel="noopener noreferrer" target="_blank">
    是任何政府都惯常使用的 —— 包括中国政府
   </a>
   —— 而且，全球只有中国和美国能使用它们全部。
  </p>
  <p class="graf graf--p">
   这是一系列血淋淋的教训。⚠️提醒读者特别注意本报告中含有的诸多超链接；它们就是所有人能否活在这个世界的基本功。
  </p>
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>

      </strong>
     </em>
    </span>
   </li>

  <p class="graf graf--p">
   越来越多的人受到“间谍法”的政府指控，但是他们并不是什么“间谍”。
  </p>
  <p class="graf graf--p">
   他们只是很普通的人，携带着自动备份到云端的智能手机；当他们想与某人交谈时，会给对方发短信或打电话；他们使用 Gmail 并在 Facebook 上分享 meme 和谈论政治；有时他们甚至从工作用的计算机登录这些帐户。
  </p>
  <p class="graf graf--p">
   就像绝大多数人那样活着。
  </p>
  <p class="graf graf--p">
   然后，他们在工作过程中看到了一些令人不安的事。也许是因为政府
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/drone-papers/the-assassination-complex/" href="https://theintercept.com/drone-papers/the-assassination-complex/" rel="noopener noreferrer" target="_blank">
    经常不知道
   </a>
   无人机袭击杀死的人是否是平民；或者国家安全局在2016年目睹了针对地方选举官员的网络攻击，情报部门认为这是由俄罗斯精心策划的，尽管总统总是在电视上反驳；再或者，只是看到了
   <strong class="markup--strong markup--p-strong">
    FBI 如何
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2017/01/31/hidden-loopholes-allow-fbi-agents-to-infiltrate-political-and-religious-groups/" href="https://theintercept.com/2017/01/31/hidden-loopholes-allow-fbi-agents-to-infiltrate-political-and-religious-groups/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     利用隐藏的漏洞
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    渗透监视政治异议人士和宗教团体
   </strong>
   ……
  </p>
  <p class="graf graf--p">
   因此，这些普通人开始搜索政府数据库以获取更多信息，并可能打印出了他们找到的一些文档，真相和证据。
  </p>
  <p class="graf graf--p">
   他们使用 Google 搜索了相关信息；也许他们甚至会向朋友发送短信，告诉他们在考虑可能的后续步骤时这是多么的疯狂……
  </p>
  <p class="graf graf--p">
   他们应该联系记者吗？他们想要说出真相。
  </p>
  <p class="graf graf--p">
   对真相的追求驱使他们开始查找自己喜欢的新闻机构的提示页面，并开始研究如何使用Tor浏览器。
  </p>
  <p class="graf graf--p">
   所有这一切都发生在他们第一次联系记者之前。
  </p>
  <p class="graf graf--p">
   ⚠️目前为止大多数人仍不太了解
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e4%b8%8e%e6%93%8d%e7%ba%b5/" href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e4%b8%8e%e6%93%8d%e7%ba%b5/" rel="noopener noreferrer" target="_blank">
    监视
   </a>
   ，但可以肯定，我们所有人都是监视的受害者。电信公司和技术巨头几乎可以访问我们所有私人数据，从我们在任何特定时间的确切物理位置、到我们的短信和电子邮件的所有内容。
  </p>
  <p class="graf graf--p">
   ⚠️即使私人数据没有直接发送给科技公司，我们的设备仍在本地保存了这些数据。你还记得两个月前的今天下午3:05，你在电脑上做了什么吗？你不记得了，但是你的网络浏览器会记得。
  </p>
  <p class="graf graf--p">
   然而，虽然所有人都在受到广泛的监视，但是对于政府雇员和承包商来说 —— 特别是那些有安全许可的人 —— 隐私几乎是不存在的。
  </p>
  <p class="graf graf--p">
   他们正是能说出巨大真相的人。因为他们有机会接触到真相。
  </p>
  <p class="graf graf--p">
   ⚠️每当他们搜索数据库时，他们使用的每一个搜索词和他们搜索的确切时间，都会被记录下来，并与他们的个人身份直接联系。
  </p>
  <p class="graf graf--p">
   当他们访问一个秘密文件时，或者他们打印任何东西时，或者当他们将USB记忆棒插入他们的工作计算机时 …… 一切都被监视着，一切都被记录下来了。
  </p>
  <p class="graf graf--p">
   可能存在员工截取屏幕截图或复制粘贴某些内容的确切日志。⚠️即使他们试图通过直接拍摄屏幕照片来绕过工作电脑监视，他们的工作场所无处不在的摄像机也可能记录他们的一举一动。
  </p>
  <p class="graf graf--p">
   这就是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e9%80%8f%e6%98%8e%e5%ba%a6%e9%9d%a9%e5%91%bd/" rel="noopener noreferrer" target="_blank">
    透明度革命
   </a>
   面前的那堵墙。
  </p>
  <p class="graf graf--p">
   拥有
   <a class="markup--anchor markup--p-anchor" data-href="https://fas.org/sgp/isoo/new_sf312.pdf" href="https://fas.org/sgp/isoo/new_sf312.pdf" rel="noopener noreferrer" target="_blank">
    安全许可
   </a>
   的政府工作人员需要承诺 “绝不向任何未经授权接收的人透露任何机密信息”。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    对于许多透明度革命的支持者而言，公开这些秘密的决定源于对政府阴谋的疑虑，同时深刻的认识到，只要该阴谋保持机密，这个体制就不会改变。
   </strong>
  </p>
  <p class="graf graf--p">
   虽然对于在内部提出问题或向国会提出申诉的举报人提出了一些保护措施，但是，
   <strong class="markup--strong markup--p-strong">
    这些人仍然会因说真话而受到惩罚，这一惩罚的历史是很悠久的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   所谓的“间谍法”正在被越来越多地使用。这是一项1917年颁布的法律，规定：如果“有意或有理由相信该信息将被用于对美国的伤害或给外国人的利益”，那么公开该信息的任何人都会违反“间谍法”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    该法律本身充分体现了体制针对说出真相的人的打压。
   </strong>
  </p>
  <p class="graf graf--p">
   根据该法律规定，政府内部人士
   <strong class="markup--strong markup--p-strong">
    不得辩称
   </strong>
   ，他们决定分享这些信息的决定是出于帮助广大公民对抗和结束被政府虐待之厄运的热情。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “该法案对公众有兴趣了解政府的无能、腐败或犯罪的可能性视而不见，政府保护特定秘密的政治利益被置于超越一切的位置上”
   </strong>
   ，第一修正研究所所长
   <a class="markup--anchor markup--p-anchor" data-href="https://www.newyorker.com/news/news-desk/the-espionage-act-and-a-growing-threat-to-press-freedom" href="https://www.newyorker.com/news/news-desk/the-espionage-act-and-a-growing-threat-to-press-freedom" rel="noopener noreferrer" target="_blank">
    Jameel Jaffer 最近写道
   </a>
   。
  </p>
  <p class="graf graf--p">
   在特朗普时代涉嫌泄密的四起所谓的间谍法案件中，最不寻常的关注就是：前中央情报局软件开发商 Joshua Schulte，他被指控泄漏 CIA 文件和黑客工具，也就是 Wikileaks 公开的著名 Vault 7 披露。
  </p>
  <p class="graf graf--p">
   在这里看到《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/cia%e5%89%8d%e9%9b%87%e5%91%98%e8%a2%ab%e6%80%80%e7%96%91%e6%b3%84%e5%af%86%ef%bc%8c%e5%94%af%e4%b8%80%e8%af%81%e6%8d%ae%e6%98%af%e4%bb%96%e4%bd%bf%e7%94%a8%e4%ba%86tor%e6%b5%8f/" href="https://www.iyouport.org/cia%e5%89%8d%e9%9b%87%e5%91%98%e8%a2%ab%e6%80%80%e7%96%91%e6%b3%84%e5%af%86%ef%bc%8c%e5%94%af%e4%b8%80%e8%af%81%e6%8d%ae%e6%98%af%e4%bb%96%e4%bd%bf%e7%94%a8%e4%ba%86tor%e6%b5%8f/" rel="noopener noreferrer" target="_blank">
    CIA前雇员被怀疑泄密，唯一“证据”是他使用了Tor浏览器
   </a>
   》
  </p>
  <p class="graf graf--p">
   Schulte 的案例与其他案件有所不同，因为在FBI于2017年3月
   <strong class="markup--strong markup--p-strong">
    扫荡他的家并没收了他的台式电脑、电话和其他设备之后
   </strong>
   ，政府使用所谓的儿童色情来诽谤他（显然当局的扫荡并没有拿到泄密证据）。
  </p>
  <p class="graf graf--p">
   检察官最初对 Schulte 提出了多项与儿童色情制品有关的罪名，后来又在另一起案件中受到所谓的性侵犯指控。
  </p>
  <p class="graf graf--p">
   直到2018年6月，政府才最终根据所谓的“间谍法”指控他泄漏黑客工具。他对所有指控都表示不认罪。
  </p>
  <p class="graf graf--p">
   其他三起因涉嫌泄露政府机密的所谓“间谍法”案件，均涉及被认为是独立媒体 Intercept 的消息来源。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    intercept 是为了专门发布斯诺登文件而创立的出版物，此后一直在公开披露符合公共利益的各种来源的秘密政府文件，也一直是特朗普政府对透明度革命的战争中具有吸引力的目标
   </strong>
   ，这并不奇怪。
  </p>
  <p class="graf graf--p">
   在这里看到斯诺登文件汇总版《

   ⚠️端到端加密只能保护消息从一部电话传输到另一部电话的过程，但是，通信各方的每部电话仍然拥有所有这些消息的纯文本副本，这使得它们很容易受到当局物理搜查的攻击。
  </p>
  <p class="graf graf--p">
   阅后即焚是一个很好的开始，但它们仍然需要进行改进。
  </p>
  <p class="graf graf--p">
   用户应该可以选择自动让所有聊天记录消失，而不必记住每次开始对话时设置消失的消息，并且应该询问用户是否需要在首次设置应用时启用此消息。
  </p>
  <p class="graf graf--p">
   尤其是，⚠️当对话中的所有消息都消失时，与该人进行对话的所有取证跟踪也应该消失。
  </p>
  <p class="graf graf--p">
   同样，在保护元数据方面也还有许多工作要做。
  </p>
  <p class="graf graf--p">
   💡Signal 的“
   <a class="markup--anchor markup--p-anchor" data-href="https://signal.org/blog/sealed-sender/" href="https://signal.org/blog/sealed-sender/" rel="noopener noreferrer" target="_blank">
    sealed sender
   </a>
   ”功能可以加密 Signal 服务可访问的大部分元数据，比任何其他流行的消息传递应用程序都要强大，但它仍然不完美。
  </p>
  <p class="graf graf--p">
   消息传递应用程序需要重新设计其服务，以便他们无法访问有关其用户的
   <strong class="markup--strong markup--p-strong">
    任何元数据
   </strong>
   ，包括IP地址。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    只有当消息服务本身也无法访问*任何*元数据时，才能保证数据不会泄漏给FBI 这类间谍组织
   </strong>
   。
  </p>
  <p class="graf graf--p">
   默认情况下，Web 浏览器会保留您访问过的每个网页的详细历史记录。
  </p>
  <p class="graf graf--p">
   这就是
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" href="https://www.iyouport.org/category/%e7%9b%91%e8%a7%86%e8%b5%84%e6%9c%ac%e4%b8%bb%e4%b9%89/" rel="noopener noreferrer" target="_blank">
    监视资本主义
   </a>
   的盈利模式。他们不会在此做出任何改变 —— ⚠️这就是为什么每个人都应该
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e6%8a%80%e6%9c%af%e9%98%b2%e8%ba%ab-%e8%87%aa%e6%88%91%e4%bf%9d%e6%8a%a4%e6%96%b9%e6%b3%95/" href="https://www.iyouport.org/category/%e6%8a%80%e6%9c%af%e9%98%b2%e8%ba%ab-%e8%87%aa%e6%88%91%e4%bf%9d%e6%8a%a4%e6%96%b9%e6%b3%95/" rel="noopener noreferrer" target="_blank">
    自己保护好自己
   </a>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    本报告中的几乎所有超链接都应该给您带来帮助
   </strong>
   。
  </p>
  <p class="graf graf--p">
   目前，Tor浏览器是保护用户隐私的最佳选择。只要
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%bf%e7%94%a8-tor-%e4%bf%9d%e6%8a%a4%e8%87%aa%e5%b7%b1%e6%97%b6%e5%8d%83%e4%b8%87%e4%b8%8d%e8%a6%81%e5%81%9a%e8%bf%99%e4%b9%9d%e4%bb%b6%e4%ba%8b%ef%bc%81/" href="https://www.iyouport.org/%e4%bd%bf%e7%94%a8-tor-%e4%bf%9d%e6%8a%a4%e8%87%aa%e5%b7%b1%e6%97%b6%e5%8d%83%e4%b8%87%e4%b8%8d%e8%a6%81%e5%81%9a%e8%bf%99%e4%b9%9d%e4%bb%b6%e4%ba%8b%ef%bc%81/" rel="noopener noreferrer" target="_blank">
    操作方面不要出错
   </a>
   —— 人始终是最大的 bugs，而非技术。
  </p>
  <p class="graf graf--p">
   但是，不幸的是，仅仅在计算机上安装 Tor 浏览器或其他隐私专用工具，就可以作为所谓的证据攻击说真话的人。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   将来，
   <strong class="markup--strong markup--p-strong">

   </strong>
   。
  </p>
  <p class="graf graf--p">
