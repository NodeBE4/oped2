  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    Leo 和 Leech
   </strong>
  </h3>
  <p class="graf graf--p">
   Leo 有一个新的伤疤，从他的前额向下穿到右颊。那是几天前的夜晚在一个出租车车站的打斗中留下的。
  </p>
  <p class="graf graf--p">
   他还有很多其他的伤疤，在发梢处、在胳膊上，是他在东德的儿童之家和监狱里留下的。
  </p>
  <p class="graf graf--p">
   在前臂上，
   <strong class="markup--strong markup--p-strong">
    他有一个英文的刺青 “Skinhead”（光头党），另一条胳膊上的刺青是一个铁十字架和一幅1937年的德国地图 —— 地图向波兰扩展了一半
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他的汗衫上写着 “FAP”，自由工人党（Frei Arbeiter Partei，德国的新纳粹政党）。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “他们是唯一维护工人的政党”，他说，“其他党都是狗屁”。
  </p>
  <p class="graf graf--p">
   Leo 没有失业，如果我说新纳粹主义（neo-Nazism）是东德的失业问题催生的，他会嗤之以鼻（译者注：这个说法是很多外国知识分子普遍认为的）。
  </p>
  <p class="graf graf--p">
   对于 Leo 而言，新纳粹主义是一种信条、一种生活方式，完全与一个稳定的工作合拍。他说，毕竟德国人最好的地方就是他们知道怎么工作。
  </p>
  <p class="graf graf--p">
   他认为自己是一个好德国人，所以为了保住在西德一家小公司做组合家具平板包装装配的工作，他让自己的头发长成了体面的平头。
  </p>
  <p class="graf graf--p">
   今天他休息，所以在 Leo 位于莱比锡北部劳工阶层社区的这个小公寓里，他和他的同伴，一个16岁、有着一双招风耳的中学生 Leech，正在喝 “煤油” —— 用几品脱啤酒和樱桃酒混合而成的酒饮料。
  </p>
  <p class="graf graf--p">
   这儿正在发生的事 —— 一个外国作家（我）谈论新纳粹主义 —— 对他们来说并不新鲜。我给他们提供了宣传的机会，他们给我提供了鲜明的、并令人很不愉快的 “拷贝” 答案。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我问他们标准性的问题，而他们报之以标准的怒斥。
   </strong>
  </p>
  <p class="graf graf--p">
   奥斯维辛？从未发生过！那只是抹黑德国人名誉的谎言！“所有这些用毒气杀人的说法都是屁话，从未发生过”。他们在学校里告诉我们这个，但如果你相信老师告诉你的事，那你就是个傻瓜！
  </p>
  <p class="graf graf--p">
   关于德国，没有什么会让你感觉羞耻的吗？没有。国家发动战争，国家输了战争。那又怎样？
  </p>
  <p class="graf graf--p">
   对话结束，我得说，
   <strong class="markup--strong markup--p-strong">
    像他那样的人代表了正在侵蚀脆弱的德国民主之心脏的癌症。
   </strong>
  </p>
  <p class="graf graf--p">
   这已经足够真实，但还有些可以说的。
  </p>
  <p class="graf graf--p">
   首先，看看 Leech 的背带，那上面装饰着的是英国的国旗。对于像 Leo 和Leech 这样的 “好德国人”，要承认他们有多感激英国人，那肯定是件尴尬的事。
  </p>
  <p class="graf graf--p">
   新纳粹主义也许在英国已经式微，但是光头文化（skin culture）—— 种族主义、发型、和音乐 —— 正席卷从旧民主德国东德波兰-德国边境到法兰克福的地区。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    光头文化可能是英国对德国以及新欧洲所做的最为持久的 “贡献”。
   </strong>
  </p>

  <blockquote class="graf graf--blockquote">
   <p>
    Leo may just be what a country gets when it loses peaceful ways of being proud about itself, when the language of national pride is forced underground, when patriotism is hijacked by criminals.
   </p>
  </blockquote>
  <p class="graf graf--p">
   听 Leo 说话的时候，我试图理解，为什么他是我
   <strong class="markup--strong markup--p-strong">
    在莱比锡遇到的唯一的对国家有着强烈自豪感的人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这是他让我警惕的地方。而不是他的伤疤、不是他的铁血言论，也不是他放在门口的短棍。
  </p>
  <p class="graf graf--p">
   如果你从他的视角看这个世界，他来自欧洲唯一不允许对自身有好感的国家。
  </p>
  <p class="graf graf--p">
   在民主德国，他们告诉他民族主义等于种族主义；在西德，他们说民族必须为其罪恶赎罪，而且他们开始建立一个
   <strong class="markup--strong markup--p-strong">
    以德国马克为基础的后民族主义身份
   </strong>
   。
  </p>
  <p class="graf graf--p">
   当国家不再以和平的方式为其自身感到自豪的时候，当民族自豪的宣言被迫转入地下，当爱国主义被罪恶所绑架，也许，就会产生像 Leo 这样的人。
  </p>
  <p class="graf graf--p">
   Leo 也代表了被压迫者的回归。在西德，大多数人都接受了1945年战败之后加之于它的边界。
   <strong class="markup--strong markup--p-strong">
    而在东德不是这样
   </strong>
   。
  </p>
  <p class="graf graf--p">
   Leo 的母亲出生于上西里西亚（Upper Silesia），那儿曾经是德国的一个省，现在属于波兰。Leo 的父亲在上西里西亚拥有一个农场，用他的话说就是 “一座产业”。
  </p>
  <p class="graf graf--p">
   Leo 听着这样的故事长大：1945年春天，俄罗斯人把他们驱赶到莱比锡的时候，家人是如何被俄罗斯人俯冲轰炸，把他们赶到壕沟里逃命。
  </p>
  <p class="graf graf--p">
   我问 Leo，是否想要回到上西里西亚参观，
   <strong class="markup--strong markup--p-strong">
    他说：“不做游客，绝不。必须跟德国国旗一起回去”。
   </strong>
  </p>
  <p class="graf graf--p">
   但 Helmut Kohl 不是签署了条约承认与波兰的边界吗？Leo 轻蔑地吸了一口万宝路，“没有政客能将德国人忽悠出他们自己的土地”，他说。
  </p>
  <p class="graf graf--p">
   如果不是成长过程中受到一个工人国家的谎言的蒙骗，Leo 本应该是一个好的社会主义者。他对身为工人有着强烈的自豪感，对于从前的日子，他最痛恨的是党的领导人把阶级团结友爱的言辞变成了对自身特权令人厌恶的辩护。
  </p>
  <p class="graf graf--p">
   在昂纳克政权之下做一个光头党，不仅是一种民族主义东德反抗标志；它也是真正的德国工人的一种战斗方式，Leo 激动地说。
  </p>
  <p class="graf graf--p">
   Leo 确实有两年 —— 他所称之为 “我被浪费的两年” —— 在昂纳克的监狱里度过，
   <strong class="markup--strong markup--p-strong">
    原因是试图逃往西方
   </strong>
   。
  </p>
  <p class="graf graf--p">
   统一之后，Leo 要求作为政治犯获得赔偿。
   <strong class="markup--strong markup--p-strong">
    拿1万马克作为补偿给一个会兴奋地点燃宪法的光头党，是对德国好心的民主的又一个讽刺
   </strong>

  </p>

   </strong>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    德国民族主义历史的间歇期
   </strong>
  </h3>
  <blockquote class="graf graf--blockquote">
   <p>
    This Volk — simple, clean, pure, ardently communitarian — was the happy projection of an unhappy intelligentsia, seeking the emotional belonging denied them in real life.
   </p>
  </blockquote>
  <p class="graf graf--p">
   种族民族主义是德国浪漫主义知识分子在拿破仑1792～1813年入侵德意志诸公国期间的创造。
  </p>
  <p class="graf graf--p">
   德国知识分子站出来反对法国启蒙运动将政治社会视为一个平等契约社会的观点，他们将德意志文化提高到普世文化的高度，反对冰冷的、机械的法兰西个人主义。
  </p>
  <p class="graf graf--p">
   在这过程中，浪漫派代表诗人 Novalis、“狂飙突进运动” 代表人物 Schiller、德国哲学家 Fichte 和社会学家 Müller 等人自觉地将自己看作民族的真正喉舌。
  </p>
  <p class="graf graf--p">
   而他们的民族主义信条倾向于捍卫已有的秩序、反对法国人，后来激发出统一和变革的激情。消解了等级制度和德意志诸公国。
  </p>
  <p class="graf graf--p">
   对于浪漫派而言，社会权威的新的主体是 “人民”，一种德意志民众的高度浪漫、抽象的形象。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “人民” 是法国 “民族 la nation” 理念以种族为决定因素的变种。
   <strong class="markup--strong markup--p-strong">
    这个 “人民” —— 简单、干净、纯粹、热忱于公有社会 —— 是一个不快乐东德知识界的快乐投影。他们寻求在现实生活中遭到拒绝的情感归属
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在法国、英国和美国，民族主义是一种现代化了的意识形态。在浪漫派看来，它变成了一种反现代化的信条，表达了他们对在法国出现的世俗民主的嫉妒和憎恨。
  </p>
  <p class="graf graf--p">
   浪漫民族主义成为一种对个人主义和个人权利的飞跃，倾向于一种个人通过归属于 “人民” 的强烈体验，获得内在自由的社会景象。
  </p>
  <p class="graf graf--p">
   德国历史学的传统之一，是将希特勒的恶性民族主义追溯到显然无辜的德国浪漫派情感主义，然而，我们如何从浪漫主义先驱、哲学家 Herder 那里得出希特勒？
  </p>
  <p class="graf graf--p">
   至少是从那个给出著名评论—— “因此从根本而言，所有的比较都不恰当，每个民族都有根源于其内在的幸福核心” —— 的 Herder 那里？
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    希特勒扩张性、攻击性的民族主义是一个普遍化的信条。德意志要成为所有一切的比较标准。这恰恰是 Herder 历史化、相对化的民族意识理念所否认的。
   </strong>
  </p>
  <p class="graf graf--p">
   希特勒统治下的大屠杀结果也不应让我们倒过来重写 “人民 Volkisch” 民族主义的历史，他对把从 “人民” 的定义中排除犹太人付出了始终不懈的努力。
  </p>
  <p class="graf graf--p">
   事实上，反犹主义直到1871年之后还未成为德国民族主义的中心，
   <strong class="markup--strong markup--p-strong">
    而它成为中心，盎格鲁撒克逊的反犹主义、法国的优生学、和美国的新达尔文主义，与德国本土的舆论潮起到了同样重要的作用
   </strong>
   。
  </p>
  <p class="graf graf--p">
   而且，犹太人只是德国民族主义可以抓来定义 “他者” 的族群之一。就如法国大革命后，是法国人；1871年后，是斯拉夫人。
  </p>
  <p class="graf graf--p">
   德国人始终对他们东部与斯拉夫各民族东德边界高度关注。神圣罗马帝国时代，德国人大规模迁移到东部临近斯拉夫人的边界，让德国人相信他们自己的种族特殊性。
  </p>
  <p class="graf graf--p">
   东部的大型德国人社区保留了他们的语言、宗教和文化，尽管事实上许多斯拉夫 — 德意志的民族同化确实发生了，但是，民族自我认知的决定性因素是并未发生的那种同化。
  </p>
  <p class="graf graf--p">
   东部的斯拉夫边界地带让德意志的核心地带将自己界定为一个拥有清晰的种族边界的国家，这是法国无法比拟的。
  </p>
  <p class="graf graf--p">
   随着斯拉夫人自己从德国人那里了解了种族民族主义，轮到德国人为他们东部边界上包括波兰人、捷克人、匈牙利人的 “民族统一蓝图” 而恐惧了。
  </p>
  <p class="graf graf--p">
   只要两大德意志帝国霍亨索伦和哈布斯堡保持对斯拉夫新兴民族的控制，这种对东部的恐惧似乎是可以容忍的。而一旦这些帝国在1918年解体，
   <strong class="markup--strong markup--p-strong">
    一个新的斯拉夫帝国在列宁及其后斯大林的领导之下，在衰弱的德意志边界崛起，德国人对斯拉夫野蛮的古老恐惧，复活了。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些恐惧被希特勒熟练地加以操控
   </strong>
   ，他在多种族的帝国熔炉维也纳长大，早已开始讨厌其浓重的多种族性，视其为对德意志人民天然优越性的冒犯。
  </p>
  <p class="graf graf--p">
   即使我们注意到德意志对斯拉夫人千年以来的恐惧，以及他们对犹太人根深蒂固的仇恨，但是像最近一些历史学家那样，宣称 “德国从德意志民族身份存在一刻起就在准备大屠杀”，也不过是某种媚俗的形式而已。
  </p>
  <p class="graf graf--p">
   这无异于说，从现代性存在的那一刻起，从有铁路和煤气炉开始，大屠杀就准备要发生。
  </p>
  <p class="graf graf--p">
   但真正的问题是，为什么德意志民族的种族性定义会清除其他所有可能的竞争选择？对此如何解释？
  </p>
  <p class="graf graf--p">
   因为竞争始终存在。在1848年席卷德意志各公国的革命之后，法兰克福议会走得如此之远，其集会甚至是依据德意志公民身份，不仅仅是那些种族上的德意志人，而是 “所有那些在德意志居住的人 …… 即使按出生和语言他们并非德意志人”。
  </p>
  <p class="graf graf--p">
   战后德国转向这样的传统 —— 即使它们已被扫地出门 ——
   <strong class="markup--strong markup--p-strong">
    试图为自己界定一个可以接受的形象。
   </strong>
  </p>

   <p>
    Unlike the reactionary ethnic nationalism of the Romantics, “state nationalism” had a strongly modernizing impetus. It sought above all to forge a nation-state, not by calling up the romance of the Volk, but by creating collective civic attachment to the institutions of the Reich.
   </p>

  <p class="graf graf--p">
   除了明确的自由派和公民传统，还有一种也许可以称为
   <strong class="markup--strong markup--p-strong">
    “国家民族主义”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这种形式的民族主义为拿破仑战败之后普鲁士的施泰因 — 登贝格改革、以及俾斯麦1871年达到顶点的统一计划，提供了意识形态动力。
  </p>
  <p class="graf graf--p">
   与浪漫派保守的种族民族主义不同，“国家民族主义” 拥有强烈的现代化冲动；
   <strong class="markup--strong markup--p-strong">
    它力图首先形成一个民族国家，不是通过召唤浪漫的 “人民”，而是通过创造对帝国机制的集体性公民依附
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    希特勒魔鬼式的成就就是将德意志 “帝国” 民族主义的和 “人民” 民族主义的意识形态，强行捏合在一起，形成德意志极权主义的意识形态
   </strong>
   。
  </p>
  <p class="graf graf--p">
   种族民族主义的另一个竞争来源由德国社会民主主义提供，作为1941年以前欧洲最强大的工人运动，它有强烈的国际主义和反民族主义意识的特征。
  </p>
  <p class="graf graf--p">
   社会民主主义传统一直坚持 —— 如果不算成功的话 —— 反对种族特殊主义，
   <strong class="markup--strong markup--p-strong">
    它的余脉仍在民主德国的社会主义意识形态中延续
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一方面，工人支持的党派拒绝表达民族性的言论，认为它是沙文主义的、资产阶级的和反动的；另一方面，传统的德国保守主义未能在德国工人中为宪法民族主义赢得支持。
  </p>
  <p class="graf graf--p">
   希特勒趁虚而入。
  </p>
  <p class="graf graf--p">
   德国左派和右派
   <strong class="markup--strong markup--p-strong">
    都没有能够
   </strong>
   在德国劳工阶层中为宪法、公民的民族主义赢得决定性支持，这一灾难性的失败为希特勒打开了道路 —— 放弃它们，转向一种极其恶毒的种族民族主义。
  </p>
  <p class="graf graf--p">
   希特勒是德国第一个将民族主义转化为流行大众运动的政客。
  </p>
  <p class="graf graf--p">
   他的
   <strong class="markup--strong markup--p-strong">
    仇恨政治
   </strong>
   指责战胜的协约国对惩罚性的《凡尔塞和约》负有责任，指责犹太人的阴谋对大萧条负有责任，
   <strong class="markup--strong markup--p-strong">
    从而 “精明” 地将人民划入高贵的受害者之列。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    受到误导和挫败的人民转向希特勒，因为他的民族主义取悦他们，消除他们的不安，使他们相信其他人应该为他们所遭受的苦难负责。
   </strong>
  </p>
  <p class="graf graf--p">
   民族主义提供给党的信徒一种政治麻痹剂，一种永远愤慨、欢欣、偶尔暴力的兴奋状态；使他们从正常的、沉闷的普通生活中逃离出来。
  </p>
  <p class="graf graf--p">
   德国自由主义和德国社会民主主义
   <strong class="markup--strong markup--p-strong">
    都未能
   </strong>
   证明自己强大到足以阻止希特勒，但如果人们将它们排除到故事之外，德国民族主义的历史会变成一次漫长的、严酷的、毫无特征的通向希特勒的进军。
  </p>
  <p class="graf graf--p">
   而且，如果你从故事中排除德国自由派和社会民主派的传统，你就不能解释战后自由派、社会民主派意识的复兴。
  </p>
  <p class="graf graf--p">
   换句话说，你就不能解释45年来在联邦德国的民主的真正事实。这个故事的寓意是什么？德国历史不是其命运，未来不是过去的囚徒。只有民族主义者才会那样认为。
  </p>

  </blockquote>
  <p class="graf graf--p">
   在法兰克福萨克森豪森（Sachsenhausen）区的美杜莎土耳其乐师正在为琵琶、手鼓、竖琴、齐特琴挑选乐曲。
  </p>
  <p class="graf graf--p">
   男人和男人、女人和女人在跳舞、穿梭在桌子间，来往于这个位于街道之下、粉刷成白色迷宫一样的通道中，侍者们艰难地在沸腾的舞池中移动，托盘上放着啤酒、烤肉串和土耳其沙拉。
  </p>
  <p class="graf graf--p">
   20多岁的 Huseyn 和 Zu 都
   <strong class="markup--strong markup--p-strong">
    说着一口完美的德语
   </strong>
   。还是孩子的时候，他们从土耳其来到这里。德国是他们所了解的一切。
  </p>
  <p class="graf graf--p">
   Zu 可以 —— 像人们说的 —— “完美地冒充德国人”：她有着俏皮的金发和浅色的皮肤，曾经为美国运通公司工作过，
   <strong class="markup--strong markup--p-strong">
    说那种怪怪的美式英语。这是法兰克福的第二语言，这种英语事实上是在宣称：“我不只是德国人，还是欧洲人”。
   </strong>
  </p>
  <p class="graf graf--p">
   Huseyn 握着她的手，她轻抚着他胡须之上的面颊。我能感受到他们不愿意谈论摩恩的新纳粹袭击事件。他们宁愿把它抛开，这是又一个夜晚，让他们可以听齐特琴、跳舞，忘记外面大街上实际发生的事。
  </p>
  <p class="graf graf--p">
   对于他们，实际问题不是愤怒或恐惧，尽管有很多这种东西。
   <strong class="markup--strong markup--p-strong">
    “当我听到总理表达他的歉意时，仅此足以令我爆发”，Huseyn 咕哝着说。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    实际问题是他们现在归属哪里？他们是否可以归属任何地方。
   </strong>
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我不知道去哪里”，Huseyn 说，“我在这里长大，我讲德语。我爱土耳其，但那里不再是我的家园”。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    随后他又说，“但我始终想着回去”。
   </strong>
  </p>
  <p class="graf graf--p">
   他伸出手，再次握住 Zu 的手。
  </p>
  <p class="graf graf--p">
   在德国，大街上他们可以在一起。由于父母不允许，他们不能住在一起，但是有一个通情达理的叔叔有时候让他们在他的住处过夜。
  </p>
  <p class="graf graf--p">
   Huseyn 和 Zu 羞涩地笑了。现在，如果在土耳其，Zu 玩着她的啤酒杯垫说：“我是不可能这样做的。对他不是问题，但对我是问题”。
  </p>
  <p class="graf graf--p">
   即使 Huseyn 最终能在这里获得公民身份，他觉得也不会有太大差异。“我应该拿护照怎么着呢？” 他痛苦地说，“把它挂在脖子上吗？”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一张护照会让人们不再叫他 “肮脏的土耳其人” 吗？一张护照会让德国工人在工厂里和他一起抽烟吗？他有一张土耳其人的脸孔，一张护照授予的正式权利不会改变德国人看到他的样子。
   </strong>
  </p>
  <p class="graf graf--p">
   将来，Huseyn 也许会归属于德意志国家，但绝不会归属于德意志民族。
  </p>
  <figure aria-describedby="caption-attachment-11991" class="wp-caption aligncenter" id="attachment_11991" style="width: 1200px">
   <img alt="" class="size-full wp-image-11991 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/9.jpeg?resize=1100%2C1100&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/9.jpeg?w=1200&amp;ssl=1 1200w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/9.jpeg?resize=300%2C300&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/9.jpeg?resize=1024%2C1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/9.jpeg?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/9.jpeg?resize=768%2C768&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/9.jpeg?resize=1100%2C1100&amp;ssl=1 1100w" data-recalc-dims="1" height="1100" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/9.jpeg?resize=1100%2C1100&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="size-full wp-image-11991" data-recalc-dims="1" height="1100" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/9.jpeg?resize=1100%2C1100&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/9.jpeg?w=1200&amp;ssl=1 1200w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/9.jpeg?resize=300%2C300&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/9.jpeg?resize=1024%2C1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/9.jpeg?resize=150%2C150&amp;ssl=1 150w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/9.jpeg?resize=768%2C768&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/9.jpeg?resize=1100%2C1100&amp;ssl=1 1100w" width="1100"/>

   <figcaption class="wp-caption-text" id="caption-attachment-11991">

   </figcaption>
  </figure>
  <p class="graf graf--p">
   第二天早晨，法兰克福当地一所学校的操场上，下着雨。孩子们和他们的父母跪在人行道上制作标语牌，牌子在雨中有点被淋花了。
  </p>
  <p class="graf graf--p">
   标语牌和海报是孩子们写的，他们的话包括 “外国人是我们的朋友” 和 “我们都是土耳其人”。
  </p>
  <p class="graf graf--p">
   大约有15名老师、父母和孩子，他们从操场出发，游行穿过社区。
  </p>
  <p class="graf graf--p">
   我加入了游行队伍，穿过郊区下雨的街道。与这些孩子们肩并肩，他们的面孔满怀希望和天真，带有高尚情感举着雨迹斑斑的海报。
  </p>
  <p class="graf graf--p">
   我告诉自己，我们正在行动。我们不会把自己算到沉默的大多数里面去。我们不是那些什么都不敢做的人。
  </p>
  <p class="graf graf--p">
   当然，问题并不是体面的德国人会不会站出来。在数周和数月的时间里，有数以百万计的人站出来游行。但这种天真的、荣耀的道德自恋形式到底意味着什么？
   <strong class="markup--strong markup--p-strong">
    它说：我为我的国家感到羞耻，但我不对其最糟糕的行为负责。
   </strong>
  </p>
  <p class="graf graf--p">
   所以当我们游行时，我们将自己与摩恩、罗斯托克、索林根，与一系列那晚的火苗、火把、弹簧刀、尖叫和靴子的踩踏产生联系的城镇，区别开来了。
  </p>
  <p class="graf graf--p">
   在人群中，我遇到一个秃头的中年工程师，他和女儿拉手走着。我告诉他我刚从莱比锡过来。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “刚从东部回来，是吗？我一个人在这里待了六个月。真是个乱七八糟的地方。没有希望。没有人有体面的日间工作。我们不得不重新开始。他们[东德人]绝望、不可救药。从不停止抱怨”。
   </strong>
  </p>
  <p class="graf graf--p">
   他战栗了。他说：“欢迎回家”。然后握紧女儿的手。
  </p>
  <p class="graf graf--p">
   当然，他绝不会像如此随意地谈论他的德意志同胞那样谈论土耳其人。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    奇怪的人们，互相之间如此地不喜欢。
   </strong>
  </p>
  <p class="graf graf--p">
   小小的游行队伍在街道上行进，沉默的旁观者从旁边经过，脸上看不出他们对这些标语怎么想。
  </p>
  <p class="graf graf--p">
   我开始和一个名叫 Sabine 的年轻女人交谈。她解释说，她的孩子太小，现在还没上学。那么她为什么来参加游行呢？
  </p>
  <p class="graf graf--p">
   她有一张修长的漂亮面孔，锐利的蓝色眼睛，挽起的头发是褐色的，辫子垂在背后。
   <strong class="markup--strong markup--p-strong">
    说话的时候，她保持着一种微妙的、悲痛的距离
   </strong>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我不太确定，只不过是来了。只不过是没有坐在家里。”
  </p>
  <p class="graf graf--p">
   她将黑色长外套拉紧，裹住肩头，打了一个寒颤：“现在这个社会有那么多的丑恶，那么多的仇恨”。
  </p>
  <p class="graf graf--p">
   她的声音里带着苦恼：“我没办法让我的孩子们远离这些。我想要，但是没办法”。
  </p>
  <p class="graf graf--p">
   我们沉默着走了很长一段时间，然后她停下来，抹了抹头发上的雨，似乎终于找到了解释自己为什么在这里的话：
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我害怕。在德国，我第一次感到害怕”。⚪️
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    —— 未完待续 ——
   </em>
  </p>
