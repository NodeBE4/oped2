 <div class="entry-content">
  <ul>
   <li>
    <span style="color: #00ccff;">
     <em>
      压制性政权越来越擅长利用数字间谍来镇压异议并扼杀国内外的反对，甚至在民主国家也是如此。美国公民社会组织报告的威胁所占比例为第二大，仅次于中国 — — 最新报告，分析技术局势
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p graf-after--figure" id="14ca">
   十多年来，民间社会组织（CSO）一直受到来自大公司和政府的几乎相同强度的攻击，不仅威胁言论自由，而且威胁民主本身。
  </p>
  <p class="graf graf--p graf-after--p" id="3c5b">
   2009 年因窃取了谷歌源代码而“走红”的 Aurora 行动，其另一个主要目标似乎鲜为人知，那就是：西藏社区。
  </p>
  <p class="graf graf--p graf-after--p" id="9046">
   <strong class="markup--strong markup--p-strong">
    从那以后，国家支持的数字间谍活动越来越多地被用来对付全球的记者、活动家和人道主义团体。重要的是，尽管面临着与大公司相同的高级威胁，但公民社会组织
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://targetedthreats.net/" href="https://targetedthreats.net/" rel="noopener nofollow noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     非常脆弱
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，缺乏资源难以建立有效应对措施和获得外部帮助，使其极易受到攻击 — 并且随着威胁的不断发展，应对难度越来越大。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="e82e">
   一种令人担忧的趋势已经非常明显：专制政权越来越擅长利用数字间谍来镇压异议并扼杀国内外的反对意见。
  </p>
  <p class="graf graf--p graf-after--p" id="380c">
   今年早些时候，哥伦比亚大学国际和公共事务学院与公民实验室的研究人员举办了一次研讨会，研究民间社会组织面临的数字威胁，以及他们在减轻这些威胁方面遇到的障碍。确定了以下六项挑战。
  </p>
  <p class="graf graf--p graf-after--p" id="b581">
   <strong class="markup--strong markup--p-strong">
    首先是，资源匮乏
   </strong>
   。学者们需要花时间和精力来推测网络冲突，更不用说还要收集和分析经验数据了。与此同时，像公民实验室这样的独立研究中心以及电子前沿基金会、人权观察和 Access Now 等非营利组织已经创建了一系列关于公民社会组织目标威胁的宝贵报告。这些组织提供高质量的研究，但由于资金和人力限制，其产出有限。
   <span class="markup--quote markup--p-quote is-other" data-creator-ids="81d8af433c04">
    私人网络安全行业获得了更好的资金支持，有能力收集有关目标威胁的大量数据，但它是由商业利益驱动的，而这些利益通常不会优先考虑民间社会。
   </span>
  </p>
  <p class="graf graf--p graf-after--p" id="a2dd">
   <strong class="markup--strong markup--p-strong">
    其次，针对民间社会组织的有针对性的数字威胁不仅仅局限于专制国家，而是一种全球性的威胁，因为这些政权越来越警惕民间社会，还能跨国攻击
   </strong>
   。下图显示了网络安全公司和独立研究中心对公共威胁报告的分析数据。
  </p>
  <p class="graf graf--p graf-after--p" id="a5f7">
   <strong class="markup--strong markup--p-strong">
    很大一部分威胁活动是针对自由民主国家的公民社会组织的 — — 在美国，公民社会组织报告的威胁所占比例为第二大，仅次于中国。
   </strong>
   由于民间社会对于运作民主至关重要，专制政权通过数字手段向海外投射权力的趋势是对全球民主的严重攻击。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-1183 jetpack-lazy-image" data-lazy-sizes="(max-width: 680px) 100vw, 680px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/08/0-1.jpeg?resize=680%2C397&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/08/0-1.jpeg?w=680&amp;ssl=1 680w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/08/0-1.jpeg?resize=300%2C175&amp;ssl=1 300w" data-recalc-dims="1" height="397" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/08/0-1.jpeg?resize=680%2C397&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="680"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-1183" data-recalc-dims="1" height="397" sizes="(max-width: 680px) 100vw, 680px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/08/0-1.jpeg?resize=680%2C397&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/08/0-1.jpeg?w=680&amp;ssl=1 680w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2018/08/0-1.jpeg?resize=300%2C175&amp;ssl=1 300w" width="680"/>
   </noscript>
  </p>
  <p class="graf graf--p graf-after--figure" id="277c">
   <strong class="markup--strong markup--p-strong">
    第三，技术发展为监督和间谍活动创造了新的机会，逃避了现有的监管和规范框架。
   </strong>
   情报部门和私营部门可以通过支持新市场的开发来利用这一灰色地带，例如商业间谍软件。虽然存在合法拦截的法律案件，但问责制正面临着严重的挑战，因为这些间谍工具被要求保密！这使得民间社会处于危险之中，正如围绕间​​谍软件提供商的诸多争议所显示的那样，如 FinFisher、Hacking Team 和 NSO 这些可恶的监视技术销售公司，其销售能力与领先的情报机构几乎一样，完全可以与几个专制政权相提并论。
  </p>
  <p class="graf graf--p graf-after--p" id="f396">
   <em class="markup--em markup--p-em">
    延伸阅读：《
   </em>
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%8F%AF%E8%A7%86%E5%8C%96%E6%95%B0%E6%8D%AE-%E5%85%A8%E7%90%83%E7%9B%91%E6%8E%A7%E8%A1%8C%E4%B8%9A-%E8%B0%81%E6%98%AF%E5%A4%A7%E5%93%A5-%E8%B0%81%E5%9C%A8%E4%BD%9C%E6%81%B6-de7592e0285e" href="https://medium.com/@iyouport/%E5%8F%AF%E8%A7%86%E5%8C%96%E6%95%B0%E6%8D%AE-%E5%85%A8%E7%90%83%E7%9B%91%E6%8E%A7%E8%A1%8C%E4%B8%9A-%E8%B0%81%E6%98%AF%E5%A4%A7%E5%93%A5-%E8%B0%81%E5%9C%A8%E4%BD%9C%E6%81%B6-de7592e0285e" rel="noopener noreferrer" target="_blank">

    》
   </em>
  </p>
  <p class="graf graf--p graf-after--p" id="bc31">
   <strong class="markup--strong markup--p-strong">
    第四，当大多数攻击实际上依赖于社会工程学技术时，将重点放在开发技术的解决方案上是错的。
   </strong>
   虽然像美国的漏洞权益流程（VEP）这样的举措是一种潜在的有前途的方法以规范技术漏洞的披露，但是，它无法解决越来越多地使用社交工程技术来利用人的行为心理弱点而发起的攻击 — — 例如好奇心，人类的下意识信任倾向和对社交的渴望、对缺乏关注的担忧。（
   <strong class="markup--strong markup--p-strong">
    后面我们会陆续详细介绍社交工程学骗局的实施，这个时代每个人都应该掌握一些这样的知识
   </strong>
   ）
  </p>
  <p class="graf graf--p graf-after--p" id="a83c">
   因此，单独调节技术手段就会忽略掉大多数威胁。在很多针对公民社会发起的攻击中，更多的努力被花费在有效的社会工程上，用于欺骗目标打开恶意附件或链接，而不是利用技术本身的漏洞，后者已经被黑市炒得很昂贵了。（您可以在这里看到一些介绍《
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%8D%B1%E9%99%A9-%E7%9B%91%E6%8E%A7%E6%8A%80%E6%9C%AF%E5%85%AC%E5%8F%B8%E6%AD%A3%E5%9C%A8%E6%8E%A8%E5%8A%A8%E5%AF%B9%E9%9B%B6%E6%97%A5%E6%BC%8F%E6%B4%9E%E7%9A%84%E5%88%A9%E7%94%A8-7db40bc15cf8" href="https://medium.com/@iyouport/%E5%8D%B1%E9%99%A9-%E7%9B%91%E6%8E%A7%E6%8A%80%E6%9C%AF%E5%85%AC%E5%8F%B8%E6%AD%A3%E5%9C%A8%E6%8E%A8%E5%8A%A8%E5%AF%B9%E9%9B%B6%E6%97%A5%E6%BC%8F%E6%B4%9E%E7%9A%84%E5%88%A9%E7%94%A8-7db40bc15cf8" rel="noopener noreferrer" target="_blank">
    危险：监控技术公司正在推动对零日漏洞的利用
   </a>
   》）
  </p>
  <p class="graf graf--p graf-after--p" id="9260">
   <strong class="markup--strong markup--p-strong">
    第五，很难量化间谍活动对目标公民社会组织的影响，特别是在长期范围内
   </strong>
   。攻击对公司和政府机构的影响可以通过其财务损失和泄露的敏感数据量来定量衡量，但影响公民社会组织的攻击很少导致财务损失，其导致的是逮捕和迫害目标个人及其家庭成员的人权，
   <span class="markup--quote markup--p-quote is-other" data-creator-ids="870a4b2e46a">
    以及由于异议人士的个人帐户被盗取，被用于在社交网络上传播恶意软件，从而导致的是对异议群体内部互信问题的更长期的影响。
   </span>
  </p>
  <p class="graf graf--p graf-after--p" id="8ce5">
   <strong class="markup--strong markup--p-strong">
    第六，没有一个通用的解决方案可以减轻公民社会组织面对的这种威胁
   </strong>
   。每个社区都会受到威胁，目标也不同。来自中国的威胁组织所针对的西藏侨民群体，可能与该政权和其他非国家行为组织针对叙利亚反对派成员发起的攻击，有着截然不同的威胁模式。
  </p>
  <p class="graf graf--p graf-after--p" id="5c13">
   任何这些挑战都没有简单的答案。尽管如此，权利组织、政策制定者和科技公司都可以采取一些措施来改善公民社会组织的网络安全。
  </p>
  <p class="graf graf--p graf-after--p" id="cc97">
   <strong class="markup--strong markup--p-strong">
    更重要的是，公民社会组织应该普及自我防御知识，加强应对危险的敏锐度，但目前似乎一些措施并不到位。
   </strong>
   去年一位公民社会的活动家拒绝了我发送的一封电子邮件，由于邮件附了一个很长的文档，他告诉我他不会打开任何附件和链接（甚至是来自熟人的，因为无法判断对方的邮箱是否被劫持了）。这种警惕性值得赞扬，但拒绝一切岂不是耽误工作了吗？实际上你完全可以
   <strong class="markup--strong markup--p-strong">
    使用基本技术把危险隔离出去
   </strong>
   ，避免受到伤害，
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E7%B3%BB%E7%BB%9F%E8%A2%AB%E6%94%BB%E5%87%BB-%E7%94%A8-tor-%E8%A2%AB%E9%92%93%E9%B1%BC-%E5%A5%BD%E5%A5%87%E5%AE%B3%E6%AD%BB%E7%8C%AB-%E6%80%8E%E4%B9%88%E5%8A%9E-%E6%8A%8A%E5%8D%B1%E9%99%A9%E9%9A%94%E7%A6%BB%E5%87%BA%E5%8E%BB-1e5ba25fe36b" href="https://medium.com/@iyouport/%E7%B3%BB%E7%BB%9F%E8%A2%AB%E6%94%BB%E5%87%BB-%E7%94%A8-tor-%E8%A2%AB%E9%92%93%E9%B1%BC-%E5%A5%BD%E5%A5%87%E5%AE%B3%E6%AD%BB%E7%8C%AB-%E6%80%8E%E4%B9%88%E5%8A%9E-%E6%8A%8A%E5%8D%B1%E9%99%A9%E9%9A%94%E7%A6%BB%E5%87%BA%E5%8E%BB-1e5ba25fe36b" rel="noopener noreferrer" target="_blank">
    这里是具体措施
   </a>
   的介绍。
  </p>

   主要发现
  </h3>
  <ul class="postList">
   <li class="graf graf--li graf-after--h3" id="cfe2">
    本报告分析了2018年1月至3月间针对西藏活动家、记者、西藏流亡议会(Tibetan Parliament in Exile)成员和西藏流亡政府(Central Tibetan Administration)的恶意软件攻击活动。
   </li>
   <li class="graf graf--li graf-after--li" id="6961">
    详细介绍 了对西藏 NGO 的入侵行动，并简要分析了入侵者在感染后的行为。
   </li>
   <li class="graf graf--li graf-after--li" id="758d">
    最近的这次攻击活动,以及公民实验室在 2016 年报道过的一次攻击, 都与一项更广泛的被称为“热带骑兵(Tropic Trooper)”的活动有关。这些关联的强度和意义仍在评估。
   </li>
   <li class="graf graf--li graf-after--li" id="7470">
    本报告研究了与调查封闭式间谍活动生态相关的挑战，准确描述了参与者及其造成的危害的重要性。
   </li>
  </ul>
  <h3 class="graf graf--h3 graf-after--li" id="8d12">
   介绍
  </h3>
  <p class="graf graf--p graf-after--h3" id="c360">
   2018年1月，一位西藏活动家收到了一封看似平凡的电子邮件，自称是一个非政府人权组织的节目更新。邮件的附件是一个 PPT 和一个文档。与很多流亡藏人一样，
   <strong class="markup--strong markup--p-strong">
    这位活动家已经对这种不请自来的带着附件的电子邮件抱有警惕，他没有打开附件，而是把附件交给了公民实验室的研究人员
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="45ab">
   他的怀疑是对的：附件是恶意的。
   <strong class="markup--strong markup--p-strong">
    如果点击了这个附件，它就将利用最近的漏洞用定制的恶意软件感染 Windows 计算机
   </strong>
   。这封电子邮件是2018年1月至3月期间针对西藏活动家、记者、
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Parliament_of_the_Central_Tibetan_Administration" href="https://en.wikipedia.org/wiki/Parliament_of_the_Central_Tibetan_Administration" rel="noreferrer nofollow noopener" target="_blank">
    西藏流亡议会
   </a>
   成员和
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Central_Tibetan_Administration" href="https://en.wikipedia.org/wiki/Central_Tibetan_Administration" rel="noreferrer nofollow noopener" target="_blank">
    西藏流亡政府
   </a>
   的恶意软件活动的开始。
  </p>
  <p class="graf graf--p graf-after--p" id="318c">
   公民实验室与目标群体密切合作，收集这些恶意邮件，并且与受感染的组织启动了事件响应。这种合作便于进一步了解攻击者使用的策略、技术和程序。
  </p>
  <p class="graf graf--p graf-after--p" id="4ad3">
   <strong class="markup--strong markup--p-strong">
    这一系列活动使用社会工程学技巧来欺骗目标打开利用了PowerPoint 的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://portal.msrc.microsoft.com/en-US/security-guidance/advisory/CVE-2017-11882" href="https://portal.msrc.microsoft.com/en-US/security-guidance/advisory/CVE-2017-11882" rel="noreferrer nofollow noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     CVE-2017–0199
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    漏洞和微软富文本文件(RTF)的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://portal.msrc.microsoft.com/en-US/security-guidance/advisory/CVE-2017-0199" href="https://portal.msrc.microsoft.com/en-US/security-guidance/advisory/CVE-2017-0199" rel="noreferrer nofollow noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     CVE-2017–11882
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    电子邮件附件。恶意软件包括了我们称之为 DMShell++ 的 PowerShell Payload，以及名为
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://blog.trendmicro.com/trendlabs-security-intelligence/tropic-trooper-new-strategy/" href="https://blog.trendmicro.com/trendlabs-security-intelligence/tropic-trooper-new-strategy/" rel="noreferrer nofollow noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     TSSL
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    的后门，还有我们称之为 DSNGInstaller 的善后工具
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="0ab9">
   我们称最近的这次活动为“重浮水面”活动，因为它与
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2016/11/parliament-keyboy/" href="https://citizenlab.ca/2016/11/parliament-keyboy/" rel="noreferrer nofollow noopener" target="_blank">
    2016年针对西藏流亡议会成员的活动
   </a>
   有关（我们称其为“议会活动”）。出现这种关联可能是因为它们是由相同的小组发起的，或者是在多个小组之间共享工具和基础设施。
  </p>
  <h3 class="graf graf--h3 graf-after--p" id="9642">
   流亡藏人：一个被高度瞄准的群体
  </h3>
  <p class="graf graf--p graf-after--h3" id="8ecf">
   数字间谍活动的威胁是流亡藏民需要面对的现实，他们
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/wp-content/uploads/2017/05/ghostnet.pdf" href="https://citizenlab.ca/wp-content/uploads/2017/05/ghostnet.pdf" rel="noreferrer nofollow noopener" target="_blank">
    十多年来
   </a>
   一直是恶意软件活动的目标。
  </p>
  <p class="graf graf--p graf-after--p" id="9952">
   从历史上看，这些攻击严重依赖于利用
   <a class="markup--anchor markup--p-anchor" data-href="https://targetedthreats.net/" href="https://targetedthreats.net/" rel="noreferrer nofollow noopener" target="_blank">
    已知漏洞和远程访问木马
   </a>
   (RAT)的恶意附件。这种策略可能是基于
   <strong class="markup--strong markup--p-strong">
    对于资源贫乏的民间团体的基本风险回报计算
   </strong>
   ：如果他们使用没打补丁的系统，那么使用简单的技术工具就足够了，为什么还要冒着暴露更复杂的技术工具的风险呢？
  </p>
  <p class="graf graf--p graf-after--p" id="03e3">
   相反，攻击者们似乎把
   <a class="markup--anchor markup--p-anchor" data-href="https://www.usenix.org/system/files/conference/usenixsecurity14/sec14-paper-hardy.pdf" href="https://www.usenix.org/system/files/conference/usenixsecurity14/sec14-paper-hardy.pdf" rel="noreferrer nofollow noopener" target="_blank">
    他们大部分的创新都集中
   </a>
   在了聪明的社会工程剂量上，并采用了“恰到好处”的方法。我们观察到的有限的技术创新可能是出于长期访问和持久性的现实需求，而不是为了更复杂的目的，例如模糊恶意软件作者的身份或者防止反编译。
  </p>
  <h3 class="graf graf--h3 graf-after--p" id="71c8">
   转变战术?
  </h3>
  <p class="graf graf--p graf-after--h3" id="368f">
   自 2016 年以来，针对藏族群体的恶意软件活动的报告数量出现了大幅下降。取而代之的是，我们观察到
   <strong class="markup--strong markup--p-strong">
    这些活动
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2018/01/spying-on-a-budget-inside-a-phishing-operation-with-targets-in-the-tibetan-community/" href="https://citizenlab.ca/2018/01/spying-on-a-budget-inside-a-phishing-operation-with-targets-in-the-tibetan-community/" rel="noreferrer nofollow noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     转向网络钓鱼
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    以获取在线账户的登陆凭证
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="c570">
   这一变化的一个值得注意的例外是
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2016/11/parliament-keyboy/" href="https://citizenlab.ca/2016/11/parliament-keyboy/" rel="noreferrer nofollow noopener" target="_blank">
    议会活动
   </a>
   ，它使用已知且已被修补的漏洞来传播名为 KeyBoy 的定制恶意软件。此次攻击活动是自议会活动以来我们首次观察到的针对藏人社群的有针对性的恶意软件活动。
  </p>
  <h3 class="graf graf--h3 graf-after--p" id="911b">
   熟悉的联系
  </h3>
  <p class="graf graf--p graf-after--h3" id="f107">
   重浮水面活动使用了与议会活动不同的漏洞和 Payloads，但是使用了相同的链接。这两个活动都使用了相似的钓鱼信息，都是针对西藏流亡议会成员定制的。重浮水面活动中的一个用来发送钓鱼邮件的地址(
   <code>
    tibetanparliarnent[@]
    <a class="markup--anchor markup--p-anchor" data-href="http://yahoo.com/" href="https://yahoo.com/" rel="noreferrer nofollow noopener" target="_blank">
     yahoo.com
    </a>
   </code>
   )在议会活动期间也被反复使用 。
  </p>
  <p class="graf graf--p graf-after--p" id="55a1">
   在常用工具与代码的相似性上，这两个活动还与一个更广泛的称为“热带骑兵(Tropic Trooper)”的行动相关联，这个行动至少始于2012年，在
   <a class="markup--anchor markup--p-anchor" data-href="https://blog.trendmicro.com/trendlabs-security-intelligence/operation-tropic-trooper-old-vulnerabilities-still-pack-a-punch/" href="https://blog.trendmicro.com/trendlabs-security-intelligence/operation-tropic-trooper-old-vulnerabilities-still-pack-a-punch/" rel="noreferrer nofollow noopener" target="_blank">
    2015年由趋势科技
   </a>
   首次报告。
   <strong class="markup--strong markup--p-strong">
    热带骑兵行动针对的是台湾和菲律宾的政府和公司，其标志是使用包括
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://blog.trendmicro.com/trendlabs-security-intelligence/operation-tropic-trooper-old-vulnerabilities-still-pack-a-punch/" href="https://blog.trendmicro.com/trendlabs-security-intelligence/operation-tropic-trooper-old-vulnerabilities-still-pack-a-punch/" rel="noreferrer nofollow noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Yahaoyah
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ,
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://blog.trendmicro.com/trendlabs-security-intelligence/operation-tropic-trooper-old-vulnerabilities-still-pack-a-punch/" href="https://blog.trendmicro.com/trendlabs-security-intelligence/operation-tropic-trooper-old-vulnerabilities-still-pack-a-punch/" rel="noreferrer nofollow noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Yahamam
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    , 和
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://blog.trendmicro.com/trendlabs-security-intelligence/tropic-trooper-new-strategy/" href="https://blog.trendmicro.com/trendlabs-security-intelligence/tropic-trooper-new-strategy/" rel="noreferrer nofollow noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     TSSL
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    在内的恶意软件
   </strong>
   。而重浮水面活动因为使用了 TSSL 而与热带骑兵联系到了一起。议会活动则因为 Keyboy 和 Yahaoyah 之间的代码相似性而被发现了关联。
   <a class="markup--anchor markup--p-anchor" data-href="https://blog.trendmicro.com/trendlabs-security-intelligence/operation-tropic-trooper-old-vulnerabilities-still-pack-a-punch/" href="https://blog.trendmicro.com/trendlabs-security-intelligence/operation-tropic-trooper-old-vulnerabilities-still-pack-a-punch/" rel="noreferrer nofollow noopener" target="_blank">
    趋势科技
   </a>
   指出，
   <strong class="markup--strong markup--p-strong">
    Yahoyah 与
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://blog.rapid7.com/2013/06/07/keyboy-targeted-attacks-against-vietnam-and-india/" href="https://blog.rapid7.com/2013/06/07/keyboy-targeted-attacks-against-vietnam-and-india/" rel="noreferrer nofollow noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     2013 年发现的 KeyBoy
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    使用了相同版本的配置文件编码算法。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="5a40">
   如果重浮水面活动与议会活动的背后是同一攻击者，那么他们似乎对自己的工具进行了有限和渐进的改进。然而，这些改进很小，而且不太可能花费太多成本。活动中使用的漏洞代码和 Powershell 代码都是公开发布的。漏洞代码在 Github 上就能找到，而 DMShell++(Powershell的Payload) 是用网上的代码改写的。
  </p>
  <h3 class="graf graf--h3 graf-after--p" id="3634">
   封闭的间谍生态：一项分析挑战
  </h3>
  <p class="graf graf--p graf-after--h3" id="cffb">
   这些类型的活动使用
   <em class="markup--em markup--p-em">
    封闭间谍生态
   </em>
   定制的恶意软件，其中涉及的各方（例如，编写恶意软件的开发人员、攻击者、和鼓励开展这些活动的情报人员）难以识别和完全分割。情报人员可能是主动地在管理工具开发与目标选择，也可能是被动地奖励那些获取了他们感兴趣的目标信息的攻击者。比起商品化的恶意软件工具包（例如用于
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2012/06/syrian-activists-targeted-with-blackshades-spy-software/" href="https://citizenlab.ca/2012/06/syrian-activists-targeted-with-blackshades-spy-software/" rel="noreferrer nofollow noopener" target="_blank">
    间谍活动
   </a>
   的网络犯罪工具）或者政府专有的恶意软件（例如来自
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2016/08/million-dollar-dissident-iphone-zero-day-nso-group-uae/" href="https://citizenlab.ca/2016/08/million-dollar-dissident-iphone-zero-day-nso-group-uae/" rel="noreferrer nofollow noopener" target="_blank">
    NSO集团
   </a>
   的产品），对于封闭的间谍生态的花费和投入是难以计算的 。
  </p>
  <p class="graf graf--p graf-after--p" id="9f21">
   <strong class="markup--strong markup--p-strong">
    封闭间谍生态中的“攻击者”经常能够被通过使用相同的工具和基础设施识别出来。这种归因分析能够把各种事件聚集成可识别的模式和指标
   </strong>
   。然而，小偷可以在不同时间使用不同的撬棍。例如，看似不同的活动和攻击者可以通过被 FireEye 称为“
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fireeye.com/content/dam/fireeye-www/global/en/current-threats/pdfs/rpt-malware-supply-chain.pdf" href="https://www.fireeye.com/content/dam/fireeye-www/global/en/current-threats/pdfs/rpt-malware-supply-chain.pdf" rel="noreferrer nofollow noopener" target="_blank">
    数字军需官
   </a>
   ”的东西联系在一起。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="7d1c">
   <strong class="markup--strong markup--p-strong">
    “数字军需官”是指在多个活动和小组之间共享的恶意软件开发和基础设施的资源。
   </strong>
   了解恶意软件活动中使用的工具和策略可以让人深入了解其技术能力，并且能让分析师跟踪一段时间内的活动，但仅凭这些并不能揭示攻击者收集的信息是如何被情报客户所使用的，也无法得知其对民间团体会造成怎样的伤害。
  </p>
  <p class="graf graf--p graf-after--p" id="0862">
   本报告分为一下几个部分：
  </p>
  <p class="graf graf--p graf-after--p" id="6e94">
   <strong class="markup--strong markup--p-strong">
    第一部分: 重浮水面活动
   </strong>
   —— 描述了2018年1月至3月间针对藏族群体的重浮水面活动。
  </p>
  <p class="graf graf--p graf-after--p" id="2b56">
   <strong class="markup--strong markup--p-strong">
    第二部分: 调查妥协 ——
   </strong>
   描述了一个西藏 NGO 的妥协，并分析了感染后攻击者的行为。
  </p>
  <p class="graf graf--p graf-after--p" id="2a6e">
   <strong class="markup--strong markup--p-strong">
    第三部分: 熟悉的联系
   </strong>
   —— 突出了议会活动与重浮水面活动之间的关系，以及与被称为“热带骑兵(Tropic Trooper)”的行动之间的关系.
  </p>
  <p class="graf graf--p graf-after--p" id="c3d3">
   <strong class="markup--strong markup--p-strong">
    第四部分: 封闭间谍生态和识别危害
   </strong>
   —— 讨论了分析封闭间谍生态的挑战，并将我们的调查置于针对流亡藏人的更广泛数字间谍活动趋势中。
  </p>
  <h3 class="graf graf--h3 graf-after--p" id="b69d">
   第一部分:重浮水面活动
  </h3>
  <p class="graf graf--p graf-after--h3" id="b5da">
   <em class="markup--em markup--p-em">

   </em>
  </p>
  <h3 class="graf graf--h3 graf-after--p" id="ac32">
   活动概述

   <p>
    第 4 部分：分析封闭生态系统的挑战
   </p>

   <p class="graf graf--p graf-after--h3" id="11fd">
    <em class="markup--em markup--p-em">
     本节讨论调查封闭式间谍活动的挑战，并将我们的调查置于更广泛的针对流亡藏人的数字间谍活动的趋势中。
    </em>
   </p>
   <p class="graf graf--p graf-after--p" id="7980">
    重浮水面活动使用了一系列新的和以前观察到了的工具，这些工具与以前用于热带骑兵活动的工具有相同的技术特征。 但是，仅凭这些联系并不能让我们最终确定这些活动是由
    <em class="markup--em markup--p-em">
     同一个
    </em>
    攻击者运作的。
    <strong class="markup--strong markup--p-strong">
     这种含糊不清的态度说明了在分析活动之间的联系以及理解封闭式间谍生态系统中不同参与者角色时所带来的一些分析上的挑战。
    </strong>
   </p>
   <p class="graf graf--p graf-after--p" id="b92f">
    研究人员会给攻击者分组命名以识别他们。 虽然名称是至关重要的，但正如
    <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@cyb3rops/the-newcomers-guide-to-cyber-threat-actor-naming-7428e18ee263" href="https://medium.com/@cyb3rops/the-newcomers-guide-to-cyber-threat-actor-naming-7428e18ee263" rel="noreferrer nofollow noopener" target="_blank">
     弗洛里安·罗斯
    </a>
    （Florian Roth）和其他人所指出的那样，选择这些名称的过程可能会导致同一组拥有多个名称以及可能对名称所指的混淆。 这些活动与热带骑兵背后的“攻击者”之间的区别显示了其中的一些挑战。
   </p>
   <p class="graf graf--p graf-after--p" id="81d0">
    关于热带骑兵的报告在描述它时使用了各种各样不同的，有时是模糊的方式（见表7）。 这种变化指出了使用一致命名所固有的一些麻烦：
    <strong class="markup--strong markup--p-strong">
     名字是指恶意软件活动，它们背后的“攻击者”，还是常见的工具集？
    </strong>
    它并不总是很清楚。
   </p>
   <p class="graf graf--p graf-after--figure" id="a42f">
    命名的复杂性部分源于主要活动中可能存在的多个操作角色。 这些角色可能包括恶意软件开发人员、攻击者以及情报人员。 这些角色之间的关系可以是简单的，也可以是多层的。 例如，开发人员可以兼作攻击者。 恶意软件开发人员可以与多个攻击者共享相互独立的工具。 情报人员可能是活跃的（ 即 直接参与攻击活动）也或被动的（ 即 从攻击者那里获得信息）。
   </p>
   <p class="graf graf--p graf-after--p" id="5a5a">
    不幸的是，在重浮水面活动中，我们看不到其组织角色的情况，这将不利于我们将其与热带骑兵之间建立更具决定性的关系。 同时，攻击者可以使用相同的工具来为多个情报人员处理多个任务。 当在活动间建立联系的主要手段是识别工具和基础设施时，这些复杂性就有可能带来挑战。
   </p>
   <p class="graf graf--p graf-after--p" id="ea3f">
    看看重浮水面活动中恶意软件和基础设施开发的时间表，可以说明这些挑战。 图8 显示，虽然基础设施是在发送第一个钓鱼邮件之前几个月部署的，但恶意软件的构建都是在活动开始前不久完成的。 基础设施部署和恶意软件构建之间的时差，再结合这次活动在代码上与热带骑兵相似的事实，表明恶意软件可能是在小组之间共享的资源。
   </p>

      <h3 class="graf graf--h3 graf-after--p" id="4a57">
       一种封闭间谍生态的观点 A View into Closed Espionage Ecosystems
      </h3>
      <p class="graf graf--p graf-after--h3" id="33ac">
       虽然开发者，攻击者和最终情报消费者之间的关系往往不明确，但美国司法部
       <strong class="markup--strong markup--p-strong">
        （DOJ）最近对中国境内的间谍团体提出的起诉书提供了这些团体如何互动的一瞥。
       </strong>
      </p>
      <p class="graf graf--p graf-after--p" id="3f82">
       2014年，美国司法部指控五名人民解放军军官经济间谍罪。 据称，这些军官是 APT1 黑客组织的一部分，
       <a class="markup--anchor markup--p-anchor" data-href="https://www.fireeye.com/blog/threat-research/2013/02/mandiant-exposes-apt1-chinas-cyber-espionage-units.html" href="https://www.fireeye.com/blog/threat-research/2013/02/mandiant-exposes-apt1-chinas-cyber-espionage-units.html" rel="noreferrer nofollow noopener" target="_blank">
        Mandiant
       </a>
       首先将其确定为人民解放军总参谋部第三局的第二局。 APT1 针对众多政府和 500 强公司，但公民实验室也发现了其针对
       <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.ca/2013/02/apt1s-glasses-watching-a-human-rights-organization/" href="https://citizenlab.ca/2013/02/apt1s-glasses-watching-a-human-rights-organization/" rel="noreferrer nofollow noopener" target="_blank">
        西藏活动家
       </a>
       和
       <a class="markup--anchor markup--p-anchor" data-href="https://targetedthreats.net/" href="https://targetedthreats.net/" rel="noreferrer nofollow noopener" target="_blank">
        大型国际非政府组织
       </a>
       的行动。
      </p>
      <p class="graf graf--p graf-after--p" id="2a1d">
       根据
       <a class="markup--anchor markup--p-anchor" data-href="https://www.justice.gov/usao-wdpa/pr/us-charges-five-chinese-military-hackers-cyber-espionage-against-us-corporations-and" href="https://www.justice.gov/usao-wdpa/pr/us-charges-five-chinese-military-hackers-cyber-espionage-against-us-corporations-and" rel="noreferrer nofollow noopener" target="_blank">
        起诉书
       </a>
       ，
       <strong class="markup--strong markup--p-strong">
        APT1 服务的情报消费者包括：中国政府、以及寻求竞争对手知识产权和信息的中国公司。
       </strong>
      </p>
      <p class="graf graf--p graf-after--p" id="f0e2">
       在 2014 年的另一起案件中，美国司法部
       <a class="markup--anchor markup--p-anchor" data-href="https://www.justice.gov/opa/pr/chinese-national-pleads-guilty-conspiring-hack-us-defense-contractors-systems-steal-sensitive" href="https://www.justice.gov/opa/pr/chinese-national-pleads-guilty-conspiring-hack-us-defense-contractors-systems-steal-sensitive" rel="noreferrer nofollow noopener" target="_blank">
        指控
       </a>
       一名名叫苏斌的中国公民参与长期共谋，破坏美国主要的国防承包商，并向中国的实体出售有关技术项目的被盗信息。
      </p>
      <p class="graf graf--p graf-after--p" id="de66">
       苏斌与两名名字未知的阴谋家合作进行了入侵。
       <a class="markup--anchor markup--p-anchor" data-href="https://www.theglobeandmail.com/news/national/article19704622.ece/BINARY/Su+Bin+1030+complaint.pdf" href="https://www.theglobeandmail.com/news/national/article19704622.ece/BINARY/Su+Bin+1030+complaint.pdf" rel="noreferrer nofollow noopener" target="_blank">
        起诉书
       </a>
       指出，共谋者位于中国，并与“中国的多个组织和实体”有关。 共谋者获得了 220 万元人民币（约合 995,400 美元）。 这些攻击者彼此分享了
       <a class="markup--anchor markup--p-anchor" data-href="https://www.theglobeandmail.com/news/national/article19704622.ece/BINARY/Su+Bin+1030+complaint.pdf" href="https://www.theglobeandmail.com/news/national/article19704622.ece/BINARY/Su+Bin+1030+complaint.pdf" rel="noreferrer nofollow noopener" target="_blank">
        一份报告
       </a>
       ，详细说明了针对其中一个目标的成功入侵行动。 该报告包括对“过去的成就”的描述，
       <strong class="markup--strong markup--p-strong">
        包括窃取“民主运动”（针对香港的民主活动家）和“西藏独立运动”的文件。
       </strong>
      </p>
      <p class="graf graf--p graf-after--p" id="728d">
       这些案例对开发商、攻击者和情报消费者之间的互动提供了罕见的一瞥，
       <strong class="markup--strong markup--p-strong">
        这表明情报消费者们为经济间谍活动提供数百万美元的计划同样可以用在民间组织上。
       </strong>
       虽然第一类行动可能导致知识产权和财产损失，但第二类可能会对目标个人或其家庭造成直接伤害。
      </p>
      <h3 class="graf graf--h3 graf-after--p" id="6ce4">
       解决分析挑战
      </h3>
      <p class="graf graf--p graf-after--h3" id="14d3">
       安全研究人员通常没有 DOJ 起诉书中引用的证据和透明度水平，并且必须依靠现有的技术指标来跟踪群体并假设他们在封闭的间谍生态系统中的动机和作用。
      </p>
      <p class="graf graf--p graf-after--p" id="1e1f">
       未来工作的一个可能领域是使用正式方法（ 即 计算机科学中开发的用于描述硬件和软件系统属性的数学技术）来关联技术指标和活动。 这些技术可以提供更加系统的方式来将小组联系在一起，并减轻模糊性。
      </p>
      <p class="graf graf--p graf-after--p" id="7b1d">
       但是正如我们所讨论的，攻击者和恶意软件开发人员的识别只是其中的一个难题。 了解数字间谍活动的最终危害需要与目标社区进行互动。
      </p>
      <h3 class="graf graf--h3 graf-after--p" id="cc16">
       显示危害：公民社会的视角
      </h3>
      <p class="graf graf--p graf-after--h3" id="2168">
       数字间谍已经成为流亡藏人的常见威胁。 数字安全意识和防御方法现在已成为社区的必需品。 根据这一经验，对于藏人来说，间谍活动的危害是显而易见的。 西藏数字安全培训师 Lobsang Gyatso Sither 提供了一个观点：
      </p>
      <blockquote class="graf graf--blockquote graf--startsWithDoubleQuote graf-after--p" id="5c3b">
       <p>
        “对于社区来说，摆脱‘我无所遁形’的思维方式非常重要，要思考我们之间的关系以及这些关系可能导致何种伤害。 流亡藏人通过各种组织相互联系，如果你受到了损害，你将成为最薄弱的环节，并允许间谍获得可用于锁定其他藏人的信息。 流亡藏人也经常与西藏境内的藏人接触，在那里伤害可能很严重 — — 包括逮捕，拘留和监禁。
       </p>
      </blockquote>
      <p class="graf graf--p graf-after--blockquote" id="8ee9">
       近年来，我们已经看到攻击者改用了网络钓鱼的策略，这使得重浮水面活动成为自 2016 年以来首次被发现的针对一个社群的恶意软件活动实例。
      </p>
      <p class="graf graf--p graf-after--p" id="d8c7">
       该活动采用熟悉的社会工程技巧与定制的恶意软件相结合。 为了应对数字间谍活动的持续威胁，西藏团体已经开展
       <a class="markup--anchor markup--p-anchor" data-href="http://cybersuperhero.net/" href="http://cybersuperhero.net/" rel="noreferrer nofollow noopener" target="_blank">
        基层工作
       </a>
       以增加数字安全教育，但改变行为和学会技能需要时间和耐心。
      </p>
      <p class="graf graf--p graf-after--p graf--trailing" id="1d4e">
       至少有一个组织受到重浮水面运动的影响，这表明熟悉的策略仍在使用，因为它们仍然有效。 然而，藏人并没有被这些威胁所阻挡，而是继续进行艰苦而必要的工作，以增强他们社区的能力，并防御数字间谍的攻击。
      </p>
     </div>
