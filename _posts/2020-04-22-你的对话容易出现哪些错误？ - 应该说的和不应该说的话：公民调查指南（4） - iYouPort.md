  <p class="graf graf--p">
   我们曾经一直以来着重于强调 *
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    开源情报
   </a>
   * 为公民独立调查人员所使用的能力，是的，它们都是可在线的操作 —— 不管您是一个人或团队合作，坐在屋里就可以办到的。
  </p>
  <p class="graf graf--p">
   但是，必须说清楚，⚠️对于真正的调查工作来说，仅仅坐在屋里是不够的。
  </p>
  <p class="graf graf--p">
   您需要与人接触，才能丰富对信息的理解。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    它的步骤
   </strong>
   大概是这样的（简化版）：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    你发现了一个主题
   </li>
   <li class="graf graf--li">
    你提出了有待验证的假设
   </li>
   <li class="graf graf--li">
    你搜寻公开资料来验证这一假设（这个部分是开源情报）
   </li>
   <li class="graf graf--li">
    最后，你通过积极地与人接触来丰富你的理解
   </li>
  </ul>
  <p class="graf graf--p">
   没错，如果没有最后一步，整个调查流程是严重不完整的。
  </p>
  <p class="graf graf--p">
   我们知道，互联网看起来无比地 “方便”，以至于越来越的调查者足不出户就能发表惊人的报告（就像我们在 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/osint/" href="https://www.iyouport.org/category/osint/" rel="noopener noreferrer" target="_blank">
    开源情报
   </a>
   ” 栏目中所演示的那样 —— 跨国搜寻）；但这仍不能代表真正的调查，相反，
   <strong class="markup--strong markup--p-strong">
    依赖于网络是必须被警告的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   因为这里有很多坑。包括技术本身的陷阱（比如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bd%a0%e6%ad%a3%e5%9c%a8%e5%a6%82%e4%bd%95%e8%a2%ab%e6%93%8d%e7%ba%b5%ef%bc%9a%e8%bf%99%e6%98%af%e6%af%8f%e4%b8%aa%e4%ba%ba%e9%83%bd%e9%9a%be%e4%bb%a5%e8%ba%b2%e9%81%bf%e7%9a%84%e6%97%a0%e5%bd%a2/" href="https://www.iyouport.org/%e4%bd%a0%e6%ad%a3%e5%9c%a8%e5%a6%82%e4%bd%95%e8%a2%ab%e6%93%8d%e7%ba%b5%ef%bc%9a%e8%bf%99%e6%98%af%e6%af%8f%e4%b8%aa%e4%ba%ba%e9%83%bd%e9%9a%be%e4%bb%a5%e8%ba%b2%e9%81%bf%e7%9a%84%e6%97%a0%e5%bd%a2/" rel="noopener noreferrer" target="_blank">
    算法专制
   </a>
   ，如果您避免得不够好的话），也包括您自己的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%be%9f%e8%b0%a3%e5%be%80%e5%be%80%e6%97%a0%e6%95%88%e9%99%a4%e9%9d%9e%e8%87%aa%e6%88%91%e6%89%b9%e9%a9%b3-%e8%af%86%e5%88%ab%e6%88%98%e7%95%a5%e6%80%a7%e6%ac%ba%e9%aa%97%e7%9a%84%e4%b8%bb/" href="https://www.iyouport.org/%e8%be%9f%e8%b0%a3%e5%be%80%e5%be%80%e6%97%a0%e6%95%88%e9%99%a4%e9%9d%9e%e8%87%aa%e6%88%91%e6%89%b9%e9%a9%b3-%e8%af%86%e5%88%ab%e6%88%98%e7%95%a5%e6%80%a7%e6%ac%ba%e9%aa%97%e7%9a%84%e4%b8%bb/" rel="noopener noreferrer" target="_blank">
    思维陷阱
   </a>
   （每个人都有这无需质疑）。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     不要错过

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%bb%95%e5%bc%80%e6%8e%a8%e7%89%b9%e7%ae%97%e6%b3%95%e5%99%aa%e9%9f%b3%e3%80%81%e4%bd%bf%e7%94%a8%e5%9f%ba%e6%9c%ac%e8%bf%90%e7%ae%97%e7%ac%a6%e6%8c%96%e6%8e%98%e6%8e%a8%e7%89%b9%e6%95%b0%e6%8d%ae/" href="https://www.iyouport.org/%e7%bb%95%e5%bc%80%e6%8e%a8%e7%89%b9%e7%ae%97%e6%b3%95%e5%99%aa%e9%9f%b3%e3%80%81%e4%bd%bf%e7%94%a8%e5%9f%ba%e6%9c%ac%e8%bf%90%e7%ae%97%e7%ac%a6%e6%8c%96%e6%8e%98%e6%8e%a8%e7%89%b9%e6%95%b0%e6%8d%ae/" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   尤其是：⚠️最激动人心的消息往往不在公开信息源中 — — 它在人的脑袋里。
  </p>
  <p class="graf graf--p">
   作为公民记者，您怎样才能找到拥有这些消息的人？怎样让他们将自己知道的东西告诉您？
  </p>
  <p class="graf graf--p">
   👉请不要低估这些技巧的价值。并不是所有人都具备这些技巧，而你的调查工作能将它们发展到一个很高的水平。
  </p>
  <p class="graf graf--p">
   也不要滥用这些技巧，不要忘记：作为一名公民记者，你可能会伤害一些人 — — 伤害他们的感情，影响他们的生计，甚至威胁他们的人身安全。务
   <strong class="markup--strong markup--p-strong">
    必不要仅仅因为他们选择了与你谈话就伤害到他们
   </strong>
   。
  </p>
  <p class="graf graf--p">
   💡本文将要考虑的是成为一名 “有价值的见证人” 的艺术 — — 可以让信息源安全并有效地对话的那种人。也就是本系列第一篇文章的标题《不再吃瓜：把你的围观变成利器》。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     绘制信息源地图
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   大多数调查者寻找采访对象时是通过阅读对事件的第一则报道，找出里面提到的人名，然后给他们打电话。
  </p>
  <p class="graf graf--p">
   ⚠️结果就是，这些少数几个信息源可能每天会接到几百个电话，当他们拿起电话的时候，能对第一百个打电话的人说出什么任何新东西吗？不能。所以为什么不去找一些其他人没有采访过的人呢？
  </p>
  <p class="graf graf--p">
   👉你利用公开信息源所做的工作能够给你提供一个名单，上面列出的是一些值得采访的更加有趣的人。
  </p>
  <p class="graf graf--p">
   比如，要调查一家公司，你可以先阅读针对该公司及其最强对手地位的金融分析报告。
  </p>
  <p class="graf graf--p">
   接着，和分析师谈谈，然后和它的竞争对手也谈谈。通过他们、以及报道该行业的媒体，找到谁离开了这家公司，跳槽或者退休都可以。（美国著名调查记者 Seymour Hersh 就是通过退休通告找到了美国中情局的许多信息）
  </p>
  <p class="graf graf--p">
   通过这些信息源，找到仍然在公司中并愿意接受采访的人。
  </p>
  <p class="graf graf--p">
   👉建议您尽快绘制一个简单的信息源地图，即与你的报道直接相关或可能直接相关的所有人的图示。
  </p>
  <p class="graf graf--p">
   这个地图看起来就好像一个有许多房子的村庄，房子里的人互相都认识，而这个村庄就是故事发生的地方。
  </p>
  <p class="graf graf--p">
   你可以做得简单些，也不必再进一步复杂化。（就算是一个只需要几分钟来准备的简单地图，也可以给你的大部分竞争带来优势）
  </p>
  <p class="graf graf--p">
   👉当你绘制一幅信息源地图的时候，用它来展现报道中人物的关系，这样一来，如果一个信息源的采访遇到障碍，你可以去找另一个能够越过此障碍的信息源。
  </p>
  <p class="graf graf--p">
   👉当你的地图上的一个部分接受了你的采访，其他部分接受你采访的可能性就会增加。
  </p>

  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     给信息源一个说话的理由
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   那些知道有意思的事实或故事的人可能有充足的理由不回答你的问题。
  </p>
  <p class="graf graf--p">
   通常来说，他们不知道你是否专业、负责、公正（这些怀疑都是合理的，因为不少职业记者就是不专业不公正不负责）。
  </p>
  <p class="graf graf--p">
   其次，你对信息的使用可能会危及采访对象的工作、人际关系，甚至是人身安全。
  </p>
  <p class="graf graf--p">
   你需要考虑所有这些。
  </p>
  <p class="graf graf--p">
   所以，当对方犹豫是否该和你说话的时候，请记住：你可能给他们带来最糟糕的后果。
  </p>
  <p class="graf graf--p">
   ⚠️人们拒绝接受采访并不是一件令人惊讶的事，事实上，如果大多数人在大多数情况下愿意接受采访，那才是真正令人惊讶的。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     人们为什么会接受采访？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   一般有两条理由，老编辑称之为
   <strong class="markup--strong markup--p-strong">
    “得意和痛苦”
   </strong>
   。
  </p>
  <p class="graf graf--p">
   你必须让你的信息源有机会满足其中的一条。
  </p>
  <p class="graf graf--p">
   ♦ 有东西令人们兴奋的时候，他们就会说话 — — 他们发现的一个足够炫目
   <br/>
   的东西时、他们已经取得或即将取得成功时、他们制定的一项拯救世界的计划时 …… 等等。
  </p>
  <p class="graf graf--p">
   讨论这些话题能让他们觉得快乐、感觉被重视，或二者皆有。
  </p>
  <p class="graf graf--p">
   ♦ 或者，就像医生们所知的那样，人们处于痛苦之中，亟需有人帮忙的时候，他们会说话。
  </p>
  <p class="graf graf--p">
   通常，
   <strong class="markup--strong markup--p-strong">
    痛苦比得意更强大
   </strong>
   ，这就是为什么在大多数调查中第一个接受采访的人往往是受害者 — — 那些遭遇冤屈的人，或那些价值观被自己所见到的东西深深冒犯的人。
  </p>
  <p class="graf graf--p">
   👉此外，还有一个让人愿意和你说话的原因，那就是：对方深深相信这样做是安全的。
  </p>
  <p class="graf graf--p">
   为了达到这一点，并将其一直保持下去，
   <strong class="markup--strong markup--p-strong">
    你必须和信息源建立一种关系
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在这种关系中，你们在一些事情上彼此信任，近乎可靠。你和你的信息源都可以给对方提供信息，并作出一些许诺。⚠️不管你的信息源有没有遵守他们的许诺，你必须遵守你自己的。
  </p>
  <p class="graf graf--p">
   这不仅是职业责任，还是品质问题。你必须是出于本能地可信赖，否则人们会觉得他们不能信任你。
  </p>
  <p class="graf graf--p">
   下面我们简单介绍一下步骤。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-10955 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-10.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-10.jpg?w=1280&amp;ssl=1 1280w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-10.jpg?resize=300%2C200&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-10.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-10.jpg?resize=768%2C512&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-10.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-10.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/11/4-10.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </noscript>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     初步接触：准备和邀请
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong>
     1．为会面做准备
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   与一个信息源最安全的沟通方式是面对面的会面（除非你的信息源有明确的威胁性，比如杀人狂 — 大多数时候你不会遇到这种）。
  </p>
  <p class="graf graf--p">
   你初步接触的目的是让会面得以实现。在第一个电话之前，你应该利用公开信息源
   <strong class="markup--strong markup--p-strong">
    研究一下
   </strong>
   这个人，以及相关议题。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    关于人 ——
   </strong>
  </p>
  <p class="graf graf--p">
   在开始之前您最少应该 Google 一下您的采访对象 —— 任何有关该信息源的新闻报道和其他文章都必须被查阅；如果数量太多，挑选一些重点出来。
  </p>
  <p class="graf graf--p">
   这里的目的是展现你对他的兴趣，以及对他（她）职业的了解。
  </p>
  <p class="graf graf--p">
   不要在采访中让信息源详述那些尽人皆知的职业历程，在你到达之前就应该了解这些。
  </p>
  <p class="graf graf--p">
   👉如果信息源给新闻媒体或专业出版物写过文章，把它们找来阅读。
   <strong class="markup--strong markup--p-strong">
    即使是守口如瓶或害羞的人也会在写作时展现他们的个性、价值观以及他们关心的话题。
   </strong>
  </p>
  <p class="graf graf--p">
   并且，这些材料能提供一些假设，在随后的采访中进行验证。
  </p>
  <p class="graf graf--p">
   你应该了解与议题相关的最新新闻和公开声明。你不需要成为一名专家，但你必须展现出自己对于信息源所在的世界使用的语言中关键术语的认识（如果不是理解的话）。比如你采访的人是民间的翻墙软件制造者，您至少需要知道 PPTP，L2TP / IPsec，OpenVPN 等等，都代表什么。
  </p>
  <p class="graf graf--p">
   接下来你可以请信息源向你解释这些术语。
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     2．进行联系
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   可以通过电话或信件联系 — — 但仅仅联系信息源的家，
   <strong class="markup--strong markup--p-strong">
    不要
   </strong>
   打电话到他（她）的工作场所，除非你绝对确定这是安全的。他（她）的老板可能在监听，电话可能被追踪（在这里看到这件事可以有多可怕《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%96%e4%bb%ac%e6%98%af%e5%a6%82%e4%bd%95%e6%8a%93%e6%8d%95%e8%af%b4%e7%9c%9f%e8%af%9d%e7%9a%84%e4%ba%ba%ef%bc%9f%e2%9a%a0%ef%b8%8f%e7%81%be%e9%9a%be%e5%92%8c%e6%95%99%e8%ae%ad/" href="https://www.iyouport.org/%e4%bb%96%e4%bb%ac%e6%98%af%e5%a6%82%e4%bd%95%e6%8a%93%e6%8d%95%e8%af%b4%e7%9c%9f%e8%af%9d%e7%9a%84%e4%ba%ba%ef%bc%9f%e2%9a%a0%ef%b8%8f%e7%81%be%e9%9a%be%e5%92%8c%e6%95%99%e8%ae%ad/" rel="noopener noreferrer" target="_blank">
    他们是如何抓捕说真话的人？⚠️灾难和教训
   </a>
   》）。
  </p>
  <p class="graf graf--p">
   ⚠️电子邮件也是这样，即使内容本身不敏感，因为老板很容易查到谁收到了一封来自记者的邮件。
  </p>
  <p class="graf graf--p">
   调查绝不是在纸上谈兵。我们曾经目睹一个调查团队对一个据说专制、偏执、腐败的官员进行调查，他们给他的秘书的办公室写信，请秘书帮忙。秘书拒绝了，但是，当官员发现这项调查的时候（被调查的人总是会发现的），你认为他会怎样对待那个可怜的女人？……
  </p>
  <p class="graf graf--p">
   👉在进行联系之前，想想怎样介绍你自己。你必须告诉信息源你是谁，你在做什么，你对自己任务的信心，以及你获得成功的能力。
  </p>
  <p class="graf graf--p">
   你不需要直接这么说，⚠️但你需要有一种感觉：在采写和发表这篇报道之后，这个世界将变成一个更好的地方。
  </p>
  <p class="graf graf--p">
   这很重要。
  </p>
  <p class="graf graf--p">
   ⚠️
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     以下是一些正确的和错误的例子 ——
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   ❌错误：“我想问你一些事，如果不是太麻烦的话……”。
  </p>
  <p class="graf graf--p">
   错在哪里？—— 你不是想问，而是要直接问。你不能给信息源这样的暗示：跟你说话意味着麻烦，你在提问时感到羞愧。
  </p>
  <p class="graf graf--p">
   ✅正确：“你好，我叫×××，我是××平台从事调查工作的人，我在做一篇关于××的报道。我相信这是一篇很重要的报道，我希望完整、准确地报道它，我们什么时候可以见面讨论这件事？”
  </p>
  <p class="graf graf--p">
   对在哪里 —— 你介绍了自己，并完整介绍了自己的目的，你让信息源有很好的理由和你谈话。
  </p>
  <p class="graf graf--p">
   你不是问 “我们是否可以见面”，而是问 “何时见面”。你不用“采访”这个词，这个词会让信息源把他（她）的名字和报纸标题以及将要到来的一连串麻烦联系起来。
  </p>
  <p class="graf graf--p">
   如果你不是为某一个特定媒体工作的话，你可以说你曾经在哪家媒体工作过。
  </p>
  <p class="graf graf--p">
   👉如果你是公民记者、没有在任何媒体工作过，你可以说你的报道将在哪里发表。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    请记得：你为谁工作并不重要，重要的是你如何工作
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ❌错误：“请帮助我，您是唯一可以帮助我的人！”
  </p>
  <p class="graf graf--p">
   错在哪里 —— 如果其他人都不愿意帮你，你又不能自救，我们为什么要帮你呢？
  </p>
  <p class="graf graf--p">
   ✅正确：“我知道您是这方面真正的专家，我非常欣赏您的洞察力。”
  </p>
  <p class="graf graf--p">
   对在哪里 —— 你在恭维你的信息源，但这种恭维是合理的，没有理由不去做。你还让他（她）知道你拥有其他的信息源，可能也是同样的专家。给对方一个竞争的暗示，大多数人会被激发兴趣。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    基本原则
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   永远假定你是一个有魅力的人，正在做一件重要的事，任何人都很高兴遇到你。如果这对你来说太困难了，也许您不适合做这项工作。
  </p>

  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     3．在哪里见面
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   如果找不到与信息源会面的地点，或是他（她）拒绝与你会面，或者没有理由地推迟会面时间，请考虑让自己出现在信息源无法走开的地方。
  </p>
  <p class="graf graf--p">
   如果他（她）在打官司，你去法庭；如果他（她）是教授，你去课堂。曾经有一个法国高官在几个月的时间里都在拒绝与记者见面，最后记者去了一个办公室，他每周都在那里与选民开会。
  </p>
  <p class="graf graf--p">
   记者在队列中等待，当轮到记者进入办公室的时候，记者说：“我们是队伍的末尾了，你还有 20 分钟，我们开始谈吧”。他笑了，说 “好的”。
  </p>
  <p class="graf graf--p">
   👉如果信息源愿意与你会面，去他（她）家或其他的他（她）认为舒服的地方。如果调查跟信息源的工作有关，而他（她）的单位知道这场采访，那么他（她）的办公室通常是最好的地点。
  </p>
  <p class="graf graf--p">
   办公室将展现出关于信息源的许多信息 — — 他（她）读什么书，他（她）的
   <br/>
   口味，他（她）将怎样应付打断采访的人，等等。（在 Connie Bruck 对华尔街的标志性调查中，揭示信息的时刻之一就发生在一个金融家没有理由地对自己的秘书尖叫的时候）
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     开始一段关系：目标和角色
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   ⚠️在新闻的世界里，记者与信息源的关系经常就像是一夜情之后就消失不见的讨厌伴侣。在灾难场景中更是如此，记者们大量无序地抵达，践踏视线中的一切，在评论当地的饮食、饮水和风俗有多坏之后，转身就走了。
  </p>
  <p class="graf graf--p">
   调查记者并没有努力成为理想情人 — — 严肃地说，和你的信息源睡觉、调情几乎总是一个糟糕的主意 — — 但他们肯定在寻找一种
   <strong class="markup--strong markup--p-strong">
    更稳定、长期的关系
   </strong>
   。
  </p>
  <p class="graf graf--p">
   因此，一段关系的开始是一个关键时刻，它很大程度上决定了接下来会发生什么。
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     始终保护信息源的匿名
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   在调查中，你能做的最重要的一件事是保护信息源的秘密，他们可能因为与你联系而陷入危险。
  </p>
  <p class="graf graf--p">
   当信息源请求匿名的时候，这种要求就更加强烈。
  </p>
  <p class="graf graf--p">
   承诺匿名意味着你必须努力使用 **一切方法** 避免留下信息源的踪迹。⚠️这包括当你的电子设备被警察或律师拿走的情况。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ae%89%e5%85%a8%e5%9c%b0%e7%95%99%e4%bd%8f%e8%af%81%e6%8d%ae%ef%bc%9a%e7%bb%99%e7%9c%9f%e7%9b%b8%e7%9a%84%e8%ae%b0%e5%bd%95%e8%80%85%e4%b8%80%e4%ba%9b%e6%8a%80%e6%9c%af%e6%8a%80/" href="https://www.iyouport.org/%e5%a6%82%e4%bd%95%e5%ae%89%e5%85%a8%e5%9c%b0%e7%95%99%e4%bd%8f%e8%af%81%e6%8d%ae%ef%bc%9a%e7%bb%99%e7%9c%9f%e7%9b%b8%e7%9a%84%e8%ae%b0%e5%bd%95%e8%80%85%e4%b8%80%e4%ba%9b%e6%8a%80%e6%9c%af%e6%8a%80/" rel="noopener noreferrer" target="_blank">
     如何安全地留住证据：给真相的记录者一些技术技巧建议
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bd%93%e4%bd%a0%e6%89%8b%e6%8f%a1%e7%9c%9f%e7%9b%b8%ef%bc%9a%e6%9c%80%e5%9f%ba%e6%9c%ac%e7%9a%84%e9%98%b2%e6%8a%a4%e7%9f%a5%e8%af%861/" href="https://www.iyouport.org/%e5%bd%93%e4%bd%a0%e6%89%8b%e6%8f%a1%e7%9c%9f%e7%9b%b8%ef%bc%9a%e6%9c%80%e5%9f%ba%e6%9c%ac%e7%9a%84%e9%98%b2%e6%8a%a4%e7%9f%a5%e8%af%861/" rel="noopener noreferrer" target="_blank">
     当你手握真相时，必需的基本防护方案
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   💡
   <strong class="markup--strong markup--p-strong">
    下列方法您可以使用
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   A / 不要打电话到信息源的工作场所，这些电话很容易被追踪。要做到高程度的安全，你们双方都必须使用预付费的手机。如果没有，使用一次性匿名的加密通信账户。
  </p>
  <p class="graf graf--p">
   B / 避免用电子邮件联系。那就像发送一张明信片。而PGP可能过于显眼，也许会带来不必要的关注。
  </p>
  <p class="graf graf--p">
   C / 和信息源在安全的地点见面，保证你们被认出的概率是最小的。
  </p>
  <p class="graf graf--p">
   D / 给信息源取一个化名，或编号（信息源 A、信息源 B）。不要在讨论或笔记中使用信息源的真名。
  </p>
  <p class="graf graf--p">
   E / 将与信息源相关的所有材料锁起来，理想的地点是那些不会被认为与你有关的地方。
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     设定你的目标
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   在第一次会面之前，明确自己想要达到什么样的目标。
  </p>
  <p class="graf graf--p">
   这一目标最少需要包括：
  </p>
  <p class="graf graf--p">
   你希望获得的有用处的东西。
  </p>
  <p class="graf graf--p">
   👉可以包括文档、秘密（confidences）、解释性的洞见或分析，以及需要进一步联系的信息源名单。
  </p>
  <p class="graf graf--p">
   ♦ 在一次会面中，你可以只寻求有限的东西。就如研究法国情报机
   <br/>
   关的调查专家 Philip Madelin 所说的那样，在一场典型的采访中，他只寻求确认或提取一条信息。
  </p>
  <p class="graf graf--p">
   ♦ 或者，你可以寻找最大数量的东西，取回每一个目力所及的文档。在这
   <br/>
   种情况下，
   <strong class="markup--strong markup--p-strong">
    确保你的信息源知道你为什么带走它们
   </strong>
   。
  </p>
  <p class="graf graf--p">
   ♦ 通常，在一次会面中寻找的最后一件东西是下一个采访对象的名字和联系方式。专业人士经常会说：“谁对我们所讨论话题的洞见让你钦佩呢？你知道我们怎样可以联系到他们吗？”
  </p>
  <p class="graf graf--p">
   你希望向信息源展示的关于你的计划的东西。
  </p>
  <p class="graf graf--p">
   👉建议你遵照英国外交的这三条规则：
  </p>
  <p class="graf graf--p">
   ♦ 不要撒谎。
  </p>
  <p class="graf graf--p">
   除非绝对必须，否则不要给出虚假的信息。
  </p>
  <p class="graf graf--p">
   记住：当一个谎言被发现，你将遭遇的后果可能是从房间里被撵走，也可能是被暴揍、被酷刑 —— 这是巴西一位暗访记者在与毒贩会面时摄像机被发现之后发生的事。
  </p>
  <p class="graf graf--p">
   ♦ 不要说出全部的真相。
  </p>
  <p class="graf graf--p">
   比如，当和极右人士谈话时，你可以说：“我们觉得您的媒体形象看起来不太准确，我们希望知道真实的情况。” 而不是说：“真实的情况可能更糟。”
  </p>
  <p class="graf graf--p">
   ♦ 如果你不能回答一个问题，就坦白告知，并告知自己什么时候能回答它。
   <br/>
   你希望了解的关于信息源的东西。
  </p>
  <p class="graf graf--p">
   我们在同什么样的人打交道？他们在回应什么引起他们兴奋的东西？他们于我们谈话的目的是什么？他们仅仅是想要或需要讲出他们的故事吗？还是利用我们达成另外的目的？
  </p>
  <p class="graf graf--p">
   英国情报机关用一个三角形的图表来评价瑞典国家电视台（SVT）的电视网络：
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-10957 jetpack-lazy-image" data-lazy-sizes="(max-width: 772px) 100vw, 772px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-34.png?resize=772%2C606&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-34.png?w=772&amp;ssl=1 772w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-34.png?resize=300%2C235&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-34.png?resize=768%2C603&amp;ssl=1 768w" data-recalc-dims="1" height="606" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/11/1-34.png?resize=772%2C606&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="772"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   👉当你使用这个图表的时候，记住下列事项：
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     关于动机 ——
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   合作的动机是什么？
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    复仇？
   </li>
   <li class="graf graf--li">
    正义？
   </li>
   <li class="graf graf--li">
    野心？
   </li>
   <li class="graf graf--li">
    对权力认可的欲望？
   </li>
  </ul>
  <p class="graf graf--p">
   <span style="color: #993366;">
    问自己：
   </span>
   此人经历过、听说过或看到过他（她）正在谈论的东西吗？或者他所提供的是二手信息吗？信息源有文件或其他信息源来证实自己的信息吗？信息与其他资料相符吗？信息源有什么掩盖的吗？信息源与你知道的其他人或组织有联系吗？信息源之前提供过正确的信息吗？
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    获取信息
   </span>
  </p>
  <p class="graf graf--p">
   信息源在获取信息的必要位置上吗？其他还有谁可以获取信息？可以在不暴露信息源身份的情况下使用信息吗？
  </p>
  <p class="graf graf--p">
   ⚠️具体是什么动机并不重要，重要的是它要可以理解，并使人信服。
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     关于信息质量 ——
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   在新闻报道中，信息的最高质量被认为是来自最高层的信息源。而调查者则认为，最高层的信息源对说真话的兴趣远不及下层。
  </p>
  <p class="graf graf--p">
   ⚠️从这个角度说，更高质量的信息更有可能来自组织较低层的、充满个人野心和反组织目标的人。
  </p>
  <p class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     关于获取信息 ——
    </strong>
   </span>
  </p>
  <p class="graf graf--p">
   如前所述，⚠️大多数调查中的
   <strong class="markup--strong markup--p-strong">
    理想信息源
   </strong>
   是组织中的中层人员，他们负责操作或计划。这些人能够获取重要文件，但对政策的形成或执行没有什么影响力。同样，他们在组织内是非常容易受伤害的。
  </p>
  <p class="graf graf--p">
   ⚠️因此，当这种信息源给你一个机密信息的时候，马上问他（她）：“还有谁知道这件事？” 向他们解释你不想引用直接可以导向单一信息源的信息 —— 这会暴露他的身份。
  </p>
  <p class="graf graf--p">
   如果你在会面中记下了受到限制的信息，在它旁边做一个记号以提醒自己不能引用它，并且要告诉信息源你这样做了。这是信誉。
  </p>
  <p class="graf graf--p">
   💡简单来说，让信息源看到你在考虑如何保护他（她），并且确保自己的确这么做了。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #00ccff;">
    <strong class="markup--strong markup--p-strong">
     选择你的角色
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在你的采访中，你可以扮演两种主要的角色。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    专家 ——
   </strong>
   知道大部分的答案，可以完全理解和重视一个同等的专业信息源提供的富有技术含量 的信息。
  </p>
  <p class="graf graf--p">
   对于专家来说，与信息源的谈话是高层次的，普通人难以跟上。
  </p>
  <p class="graf graf--p">
   我们曾经听信息源这样说：“和一个真正了解此事的人说话很棒，这意味着我可以尝试我的各种想法。”
  </p>
  <p class="graf graf--p">
   不过，如果你以专家的角色开始，请确保自己不会在后来的采访中暴露出自己的知识实际上没那么深厚，那样你会丢脸的。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    纯朴的人（或老实人）——
   </strong>
   他们和信息源的对话非常详细，因为他们所知甚少，并渴望被启发。
  </p>
  <p class="graf graf--p">
   这
   <strong class="markup--strong markup--p-strong">
    并不
   </strong>
   意味着纯朴的人是笨蛋，虽然他（她）有时可能希望被信息源低估。
  </p>
  <p class="graf graf--p">
   如果你看过 Columbo 工作的话， 你就理解了纯朴的人是什么样的。
  </p>
  <p class="graf graf--p">
   💡这可能是最强大的角色，因为它允许你说幼稚的话、提简单的问题，也可以有复杂的问题。很灵活。
  </p>
  <p class="graf graf--p">
   因为纯朴的人需要针对所有东西提问，他（她）
   <strong class="markup--strong markup--p-strong">
    避免了让信息源了解自己的目标及已经掌握的信息的危险。
   </strong>
  </p>
  <p class="graf graf--p">
   👉调查者常常以纯朴的人的角色开始采访，当对话进行时，逐渐显示自己的专家身份。
  </p>
  <p class="graf graf--p">
   👉如果你这么做，小心不要给信息源留下你对他撒谎的印象，除非你的目标就是打一场突袭， 然后永远不再见他。
  </p>
  <p class="graf graf--p">
   你可以在采访中用这两种角色中的一种，或者两种都用。
  </p>
  <p class="graf graf--p">
   👉关键是：确认自己在特定时刻扮演特定角色时的真实性。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在调查过程中与人交流是一种战术
   </strong>
   。每个人都存有自己的对话战术，许多人从来不改变战术，就像只会一句台词的蹩脚演员一样。
  </p>
  <p class="graf graf--p">
   您可不能这样。
  </p>
  <p class="graf graf--p">
   作为一名公民调查者，您需要多花时间和工作中包含提问的所有人相处，从中摸索最恰当的模式。
  </p>
  <p class="graf graf--p">
   💡最好的采访技巧反映了采访者的个性，所以当你准备自己的保留节目的时候，把它也考虑进去吧。
  </p>
  <p class="graf graf--p">
   下一篇文章中我们将详细
   <strong class="markup--strong markup--p-strong">
    推测一些计策
   </strong>
   给您，虽然不能说确保您人见人爱花见花开，但基本能帮助您套出想要的东西…… 下次见。⚪️
  </p>
