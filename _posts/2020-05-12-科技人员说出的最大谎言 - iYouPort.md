  <p class="graf graf--p">
   想象一下，您正在上在线商务课程，就是观看视频讲座然后在最后回答问题的那种。但这不是普通课程，
   <strong class="markup--strong markup--p-strong">
    你不只是在观看讲座 —— 他们也在看着你
   </strong>
   。
  </p>
  <p class="graf graf--p">
   每当监视系统确定您看起来无聊、分心或走神时，它都会做出记录。在每次课程之后，它只会考核您关于被记录下的那一刻所讲的内容。
  </p>
  <p class="graf graf--p">
   请注意：
   <strong class="markup--strong markup--p-strong">
    这并不是一个被假设出来的系统；这是由
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://nestor-ai.com/" href="https://nestor-ai.com/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     Nestor
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    公司部署的真实设备。而且，如果您不喜欢它，那就对了，因为其他人也不喜欢。
   </strong>
  </p>
  <p class="graf graf--p">
   当我问这个系统背后的人、法国发明家
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vice.com/en_us/article/j53ba3/facial-recognition-school-surveillance-v25n3" href="https://www.vice.com/en_us/article/j53ba3/facial-recognition-school-surveillance-v25n3" rel="noopener noreferrer" target="_blank">
    Marcel Saucet
   </a>
   ，关于这些课程的学生对被监视的感觉如何时，他承认他们不喜欢这样。
  </p>
  <p class="graf graf--p">
   他说，学生们感到这是一种侵犯性的监视，但他没有理会这些指责。“如今每个人都在这样做，”他告诉我，“这确实有点超前或者令人震惊，但是，我们不能违背自然进化的规律。”
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “进化”，作为报道技术和未来的记者，我不断听到这种说法 —— 技术人员试图将达尔文（Charles Darwin）在生物学上闻名的理论应用于自己的工作。
  </p>
  <p class="graf graf--p">
   有人告诉我，技术在进步，这一运动比任何个人发明家或首席执行官都要强大。他们说，他们 “只是跟随了潮流，被无法抗争的潮流席卷”。他们说，这使人们 不“可避免地” 采取了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e5%8f%8a%e5%af%b9%e6%8a%97%e6%96%b9%e6%b3%95/" href="https://www.iyouport.org/category/%e7%94%9f%e7%89%a9%e8%af%86%e5%88%ab%e5%8f%8a%e5%af%b9%e6%8a%97%e6%96%b9%e6%b3%95/" rel="noopener noreferrer" target="_blank">
    面部识别
   </a>
   （甚至部署
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%80%81%e5%a4%a7%e5%93%a5%e8%bf%9b%e5%85%a5%e5%8e%95%e6%89%80%ef%bc%9a%e5%ad%a6%e6%a0%a1%e6%ad%a3%e5%9c%a8%e4%bd%bf%e7%94%a8%e7%9b%91%e8%a7%86%e5%ba%94%e7%94%a8%e7%a8%8b%e5%ba%8f%e8%b7%9f%e8%b8%aa/" href="https://www.iyouport.org/%e8%80%81%e5%a4%a7%e5%93%a5%e8%bf%9b%e5%85%a5%e5%8e%95%e6%89%80%ef%bc%9a%e5%ad%a6%e6%a0%a1%e6%ad%a3%e5%9c%a8%e4%bd%bf%e7%94%a8%e7%9b%91%e8%a7%86%e5%ba%94%e7%94%a8%e7%a8%8b%e5%ba%8f%e8%b7%9f%e8%b8%aa/" rel="noopener noreferrer" target="_blank">
    在儿童身上
   </a>
   ）、能记录您悄悄话的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e6%99%ba%e8%83%bd%e9%97%b4%e8%b0%8d%e5%ae%b6%e5%b1%85-%e5%9f%8e%e5%b8%82/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e6%99%ba%e8%83%bd%e9%97%b4%e8%b0%8d%e5%ae%b6%e5%b1%85-%e5%9f%8e%e5%b8%82/" rel="noopener noreferrer" target="_blank">
    智能扬声器
   </a>
   、以及监视邻居的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%9a%e9%a9%ac%e9%80%8a%e5%b8%ae%e5%8a%a9%e8%ad%a6%e6%96%b9%e6%9e%84%e5%bb%ba%e4%ba%86%e4%b8%80%e4%b8%aa%e5%b9%bf%e6%b3%9b%e7%9a%84%e7%9b%91%e6%8e%a7%e7%bd%91%e7%bb%9c/" href="https://www.iyouport.org/%e4%ba%9a%e9%a9%ac%e9%80%8a%e5%b8%ae%e5%8a%a9%e8%ad%a6%e6%96%b9%e6%9e%84%e5%bb%ba%e4%ba%86%e4%b8%80%e4%b8%aa%e5%b9%bf%e6%b3%9b%e7%9a%84%e7%9b%91%e6%8e%a7%e7%bd%91%e7%bb%9c/" rel="noopener noreferrer" target="_blank">
    门铃
   </a>
   ……
  </p>
  <p class="graf graf--p">
   他们说，我们不能怪这些公司对隐私权、民主制度或对公共机构的信任的侵蚀，因为 “这一切迟早都会发生”。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “怎么可能把精灵落在瓶子里呢？” 他们说。他们甚至认为，人们购买这种东西是 “必须必然的”。公司只是在回应消费者的 “自然选择”。
  </p>
  <p class="graf graf--p">
   他们说，没有人应为此责怪。就像地心引力一样自然。
  </p>
  <p class="graf graf--p">
   也许没有人比发明家和未来主义者 Ray Kurzweil 在其2005年的著作《奇点即将来临》中更清楚地表达了这一信念：“技术的不断进步是我所谓的加速收益定律的隐含和必然结果，它描述了进化步伐的加快和指数级的增长。”
  </p>
  <p class="graf graf--p">
   ⚠️以上都是谎言。
  </p>
  <p class="graf graf--p">
   实际上，我们的世界是由做决策的人塑造的，技术公司也是如此。
  </p>
  <p class="graf graf--p">
   声称所谓的自然过程的结果，不仅误解了进化的原理，而且，还暗示了从生物武器到欺诈性初创公司（如
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vox.com/policy-and-politics/2019/8/31/20839873/james-mattis-book-theranos" href="https://www.vox.com/policy-and-politics/2019/8/31/20839873/james-mattis-book-theranos" rel="noopener noreferrer" target="_blank">
    Theranos
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/technology/2017/sep/01/juicero-silicon-valley-shutting-down" href="https://www.theguardian.com/technology/2017/sep/01/juicero-silicon-valley-shutting-down" rel="noopener noreferrer" target="_blank">
    Juicero
   </a>
   ）是 “必要且自然的”。
  </p>

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些“创新”的范围从危险到愚蠢，但它们却有着共同的线索：关于它们的一切都不是 “自然而然的”。
   </strong>
  </p>
  <p class="graf graf--p">
   没有自然的过程可以制造出“智能”发刷、“智能”人字拖或“智能”避孕套、或具有蓝牙功能的烤面包机、摄影公司的加密货币或连接互联网的空气清新剂。
  </p>

  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     进化是技术的可怕隐喻
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   技术人员渴望与进化平行的愿望是根本性的错误。
  </p>
  <p class="graf graf--p">
   进化是由随机突变（错误而不是计划）驱动的。尽管某些发明确实可能是不幸的结果，但技术公司对这些发明进行专利、生产和销售的决定绝非如此。
  </p>

   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%9d%83%e8%b0%8b%ef%bc%9a%e4%b8%ba%e6%b8%b8%e8%af%b4%e5%8d%8e%e7%9b%9b%e9%a1%bf%ef%bc%8c%e8%b0%b7%e6%ad%8c%e5%88%b6%e9%80%a0%e4%ba%86%e4%b8%80%e4%b8%aa%e5%86%a0%e5%86%95%e5%a0%82%e7%9a%87%e7%9a%84/" href="https://www.iyouport.org/%e6%9d%83%e8%b0%8b%ef%bc%9a%e4%b8%ba%e6%b8%b8%e8%af%b4%e5%8d%8e%e7%9b%9b%e9%a1%bf%ef%bc%8c%e8%b0%b7%e6%ad%8c%e5%88%b6%e9%80%a0%e4%ba%86%e4%b8%80%e4%b8%aa%e5%86%a0%e5%86%95%e5%a0%82%e7%9a%87%e7%9a%84/" rel="noopener noreferrer" target="_blank">
     权谋：为游说华盛顿，谷歌制造了一个冠冕堂皇的傀儡？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   在某些情况下，即使我们无法从字面意义上收回技术精灵，我们也可以人为干预以确保精灵遵循特定规则。
  </p>
  <p class="graf graf--p">
   关于公司在生物武器领域可以做什么和不能做什么的明确法律。
  </p>
  <p class="graf graf--p">
   就如FDA确保在出售药物之前对药物进行功效和安全性测试；USDA确保谨慎进行新的食品研究；我们不允许任何人在任何他们喜欢的地方进行压裂或钻探石油或建造核电站；我们不允许随意制造和销售汽车、飞机或枪支……
  </p>
  <p class="graf graf--p">
   因此，断言科技公司不可能以公众利益为重心而受制于或受到束缚的说法仅仅是认为它们与任何其他行业都根本性的不同。不，他们不是。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     这一想法之所以存在，部分原因是人们无法抵制“进步”的诱惑
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在许多方面，美国科技大亨将自己的选择归咎于某种无形的创新弧线，这仅仅是对创立美国的一种精神上的呼应：“不惜一切代价取得进步”。殖民美国的男人和女人们在很多大程度上只看到了前进，始终向前迈进，永远不要质疑、不要放慢脚步。
  </p>
  <p class="graf graf--p">
   本杰明·富兰克林（本人是发明家）
   <a class="markup--anchor markup--p-anchor" data-href="https://founders.archives.gov/documents/Franklin/01-31-02-0325" href="https://founders.archives.gov/documents/Franklin/01-31-02-0325" rel="noopener noreferrer" target="_blank">
    曾经写道
   </a>
   ：“如今，真正的科学飞速发展，使我有时感到后悔，因为我出生得太早了。很难想象一千年来，人类在物质上的力量可以承受的高度。”
  </p>
  <p class="graf graf--p">
   因此，放缓或甚至逆转，似乎与美国风格背道而驰。前Uber工程师 Anthony Levandowski
   <a class="markup--anchor markup--p-anchor" data-href="https://go.redirectingat.com/?id=66960X1516588&amp;xs=1&amp;url=https%3A%2F%2Fwww.newyorker.com%2Fmagazine%2F2018%2F10%2F22%2Fdid-uber-steal-googles-intellectual-property&amp;referrer=vox.com" href="https://go.redirectingat.com/?id=66960X1516588&amp;xs=1&amp;url=https%3A%2F%2Fwww.newyorker.com%2Fmagazine%2F2018%2F10%2F22%2Fdid-uber-steal-googles-intellectual-property&amp;referrer=vox.com" rel="noopener noreferrer" target="_blank">
    说
   </a>
   ：“已经发生的事并不重要。您无需知道历史就能在前人的基础上有所作为。在技​术上，最重要的是明天”。他只是在回应我们国家创始人的观点。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这些关于公司至高无上和技术进步的论点在美国人的心中是如此紧密地联系在一起，” 微软研究院研究行为和计算的研究员 Luke Stark 说，“这确实与整个美国的命运以及美洲定居者殖民主义的悠久历史息息相关。”
  </p>
  <p class="graf graf--p">
   其态度是，无论我们遇到什么阻碍，我们都必须前进，向前。回头根本不是一种选择。
  </p>
  <p class="graf graf--p">
   甚至那些害怕全能公司的人都认为，我们（消费者）必须加快步伐，而不是停顿一下。
  </p>
  <p class="graf graf--p">
   历史学家、Sapiens 的作者 Yuval Noah Harari 对《时代》杂志
   <a class="markup--anchor markup--p-anchor" data-href="https://time.com/4672373/yuval-noah-harari-homo-deus-interview/" href="https://time.com/4672373/yuval-noah-harari-homo-deus-interview/" rel="noopener noreferrer" target="_blank">
    说
   </a>
   ：“我想重点关注的不是停止技术进步的步伐。相反，我试图跑得更快 ……”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种以“进步”为名的无休止的惩罚性竞赛通常也推动了消费者的行为。尽管“美国梦”
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/business/2019/09/09/poor-middle-class-americans-are-much-less-likely-survive-into-their-seventies-than-wealthy-federal-report-says/" href="https://www.washingtonpost.com/business/2019/09/09/poor-middle-class-americans-are-much-less-likely-survive-into-their-seventies-than-wealthy-federal-report-says/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     越来越遥不可及
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，但即将到来的想法驱使人们购买这些产品。
   </strong>
  </p>
  <p class="graf graf--p">
   如果能拥有最新的应用，人们就认为生活会变得更轻松，也许将有更多的空闲时间，更多的美好时光。商业广告拼命地宣称通过互联网连接的灯泡有多迷人。
  </p>
  <p class="graf graf--p">
   只要有人购买了这些产品，科技公司就会将这作为绿灯，继续走他们的 “不可避免” 之路，
   <strong class="markup--strong markup--p-strong">
    迈向一个世界 —— 在这个世界上，亚马逊那种寡头确切地知道您在做什么、在想什么、在哪里做、甚至在您做之前就已经知道了。
   </strong>
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%b8%8a%e5%b8%9d%e4%b9%8b%e7%9c%bc%ef%bc%9a%e4%b8%80%e4%b8%aa%e6%b2%a1%e6%9c%89%e7%a7%98%e5%af%86%e7%9a%84%e4%b8%96%e7%95%8c%e6%84%8f%e5%91%b3%e7%9d%80%e4%bb%80%e4%b9%88%ef%bc%9f/" href="https://www.iyouport.org/%e4%b8%8a%e5%b8%9d%e4%b9%8b%e7%9c%bc%ef%bc%9a%e4%b8%80%e4%b8%aa%e6%b2%a1%e6%9c%89%e7%a7%98%e5%af%86%e7%9a%84%e4%b8%96%e7%95%8c%e6%84%8f%e5%91%b3%e7%9d%80%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     上帝之眼：一个没有秘密的世界意味着什么
    </a>
    ？》
   </li>
  </ul>
  <p class="graf graf--p">
   Stark 说：“这都是一个循环。” “有点奇怪。这就是使人们陷入困境的原因。他们认为他们应该能够拥有这一切。他们为自己做不到某些事而焦虑，而技术是应对这类疾病的一种措施。”
  </p>
  <p class="graf graf--p">
   这种循环出现的最有说服力的地方可能是在高科技婴儿设备市场：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.marketplace.org/2019/05/14/the-latest-smart-product-the-baby-diaper/" href="https://www.marketplace.org/2019/05/14/the-latest-smart-product-the-baby-diaper/" rel="noopener noreferrer" target="_blank">
    蓝牙尿布
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/technology/2018/12/20/nest-cam-baby-monitor-hacked-kidnap-threat-came-device-parents-say/" href="https://www.washingtonpost.com/technology/2018/12/20/nest-cam-baby-monitor-hacked-kidnap-threat-came-device-parents-say/" rel="noopener noreferrer" target="_blank">
    联网婴儿监视器
   </a>
   ，
   <a class="markup--anchor markup--p-anchor" data-href="http://nymag.com/intelligencer/2018/05/why-using-smart-wearable-baby-monitors-was-a-mistake.html" href="https://nymag.com/intelligencer/2018/05/why-using-smart-wearable-baby-monitors-was-a-mistake.html" rel="noopener noreferrer" target="_blank">
    智能脚链
   </a>
   可跟踪婴儿的一举一动、心跳和体温波动。
  </p>
  <p class="graf graf--p">
   在一个社会安全网不断侵蚀的国家，父母以关心和爱心的名义购买这些监视设备，并对健康有偏执的追求。
  </p>
  <p class="graf graf--p">
   正如 Stark 在有关对亲人进行监视的
   <a class="markup--anchor markup--p-anchor" data-href="https://static1.squarespace.com/static/59a34512c534a5fe6721d2b1/t/5cb363c124a69448722348dc/1555260354586/Media+Culture+%26+Society+2018+Stark.pdf" href="https://static1.squarespace.com/static/59a34512c534a5fe6721d2b1/t/5cb363c124a69448722348dc/1555260354586/Media+Culture+%26+Society+2018+Stark.pdf" rel="noopener noreferrer" target="_blank">
    研究论文中
   </a>
   所写的那样，“广告客户长期以来一直在提醒父母（尤其是母亲）他们的孩子正面临风险 —— 受到外部威胁、营养不良、社会排斥、自身的失败 —— 并宣称他们最可靠的途径就是全力以赴保护孩子，减轻他们对这些危险的恐惧。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他们被植入了这么一种价值观 —— 即使细节有多么的令人毛骨悚然，购买此设备也表明您在乎孩子，您是父母。
   </strong>
  </p>
  <p class="graf graf--p">
   而且，由于一些父母感到羞耻和轻率地购买了这些追踪设备，因此越来越多的父母涌入监视市场。技术人员将这些购买等同于 “自然选择”，但是对于大多数人来说，这其中几乎没有任何选择权。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    普通人无从选择
   </span>
  </h3>
  <p class="graf graf--p">
   通常，消费者根本没有太多选择权。经营小型企业的人都知道，几乎不可能离开 Facebook、Instagram、Yelp、Etsy 甚至亚马逊；雇主通常要求其员工使用某些固定的应用程序或系统，例如 Zoom、Slack 和 Google Docs ……
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   <strong class="markup--strong markup--p-strong">
    “只有超级特权的人现在有资格说，‘我不会给我的孩子这个，或者 ‘我不在社交媒体上，’”
   </strong>
   埃森哲的数据科学家 Rumman Chowdhury 说。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     技术曾经是为富人服务的。现在富人正在付钱以避免技术
    </strong>
    …《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%a9%b7%e4%ba%ba%e7%9a%84%e5%b1%8f%e5%b9%95/" href="https://www.iyouport.org/%e7%a9%b7%e4%ba%ba%e7%9a%84%e5%b1%8f%e5%b9%95/" rel="noopener noreferrer" target="_blank">
     穷人的屏幕
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   因此，当那些决定不是大多数消费者都擅长的决定时，我们就离开了那个声称受到欲望驱动的技术世界。
  </p>
  <p class="graf graf--p">
   日常用户对周围技术的看法与公司如何决定制造产品之间的鸿沟越来越大。然而，这些公司称，我们不能回去；我们无法阻止“技术的自然进化”。
  </p>
  <p class="graf graf--p">
   必须知道，“技术的自然进化”从来都是谎言，现在该去质疑 “进步” 的真正含义了。⚪️
  </p>
  <p class="graf graf--p">

   是一位作家和制作人，她的主题是探索科技与人类的关系。
  </p>
