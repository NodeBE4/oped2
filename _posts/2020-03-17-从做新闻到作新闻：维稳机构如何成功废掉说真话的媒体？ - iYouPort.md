  <p class="graf graf--p">
   自2013年6月开始发布关于美国国家安全局承包商爱德华·斯诺登泄露的美国政府机密文件的内容后，英国维稳部门就将枪口对准了《卫报》。
  </p>
  <p class="graf graf--p">
   众所周知斯诺登的重磅炸弹事件持续了数月之久，是有史以来最大的机密信息泄露事件，涉及美国国家安全局及同等的间谍机构英国政府通讯总部。机密文件透露了这两个机构实施的恐怖级大规模监视计划。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     在这里看到中文斯诺登文件完整版
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" href="https://www.iyouport.org/%e4%ba%92%e8%81%94%e7%bd%91%e9%9c%b8%e4%b8%bb%e7%9a%84%e6%97%b6%e9%97%b4%e7%ba%bf/" rel="noopener noreferrer" target="_blank">
     互联网霸主的时间线 — — 他们将互联网视为间谍、破坏和战争的主场
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   根据英国国防和安全媒体咨询委员会的会议记录，这些泄密消息引起了英国间谍机构和国防部的警觉。
  </p>
  <p class="graf graf--p">
   在这份会议记录中显示，2013年11月7日国防部说：“这一事件非常令人担忧，因为一开始，卫报就没有在发布第一批机密信息之前与[委员会]接触。”（
   <strong class="markup--strong markup--p-strong">
    这里的“接触”指的是主动接受审查
   </strong>
   ）
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    主动接受审查被认为是“规矩”《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%94%b5%e5%ad%90%e9%82%ae%e4%bb%b6%e8%a2%ab%e6%9b%9d%e5%85%89%ef%bc%9a%e6%8f%ad%e7%a4%ba%e8%ae%b0%e8%80%85%e4%b8%bb%e5%8a%a8%e6%8e%a5%e5%8f%97%e6%83%85%e6%8a%a5%e9%83%a8%e9%97%a8%e7%9a%84%e5%ae%a1/" href="https://www.iyouport.org/%e7%94%b5%e5%ad%90%e9%82%ae%e4%bb%b6%e8%a2%ab%e6%9b%9d%e5%85%89%ef%bc%9a%e6%8f%ad%e7%a4%ba%e8%ae%b0%e8%80%85%e4%b8%bb%e5%8a%a8%e6%8e%a5%e5%8f%97%e6%83%85%e6%8a%a5%e9%83%a8%e9%97%a8%e7%9a%84%e5%ae%a1/" rel="noopener noreferrer" target="_blank">
     电子邮件被曝光：揭示记者主动接受情报部门的审查，这究竟是为什么？
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%ba%bd%e7%ba%a6%e6%97%b6%e6%8a%a5%e6%89%bf%e8%ae%a4%ef%bc%8c%e5%9c%a8%e7%a8%bf%e4%bb%b6%e5%8f%91%e5%b8%83%e4%b9%8b%e5%89%8d%e5%ae%83%e4%bc%9a%e4%b8%bb%e5%8a%a8%e6%8e%a5%e5%8f%97%e6%94%bf/" href="https://www.iyouport.org/%e7%ba%bd%e7%ba%a6%e6%97%b6%e6%8a%a5%e6%89%bf%e8%ae%a4%ef%bc%8c%e5%9c%a8%e7%a8%bf%e4%bb%b6%e5%8f%91%e5%b8%83%e4%b9%8b%e5%89%8d%e5%ae%83%e4%bc%9a%e4%b8%bb%e5%8a%a8%e6%8e%a5%e5%8f%97%e6%94%bf/" rel="noopener noreferrer" target="_blank">
     ”纽约时报”承认，在稿件发布之前它会主动接受政府的审查
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   DSMA委员会由国防部运营，每六个月开会一次。还邀请了少数记者参加。其明确的目的就是审查，它们
   <a class="markup--anchor markup--p-anchor" data-href="https://dsma.uk/how-the-system-works/" href="https://dsma.uk/how-the-system-works/" rel="noopener noreferrer" target="_blank">
    称此为
   </a>
   “防止不经意的公开披露损害英国军事和情报行动的信息”。
   <strong class="markup--strong markup--p-strong">
    它可以向媒体发布“通知” —— 也就是禁令，以阻止记者发布某些信息。
   </strong>
  </p>
  <p class="graf graf--p">
   该委员会目前由国防部安全政策总干事多米尼克·威尔逊（Dominic Wilson）领导，他
   <a class="markup--anchor markup--p-anchor" data-href="https://www.gov.uk/government/people/dominic-wilson--2" href="https://www.gov.uk/government/people/dominic-wilson--2" rel="noopener noreferrer" target="_blank">
    此前曾担任
   </a>
   英国内阁办公室的情报总监。其秘书是 Geoffrey Dodds 准将，他
   <a class="markup--anchor markup--p-anchor" data-href="https://uk.linkedin.com/in/geoffrey-dodds-67883719" href="https://uk.linkedin.com/in/geoffrey-dodds-67883719" rel="noopener noreferrer" target="_blank">
    形容自己
   </a>
   是一位“有成就的、资深的前军事指挥官，具有丰富的作战层面领导经验”。
  </p>
  <p class="graf graf--p">
   该委员会将审查禁令
   <a class="markup--anchor markup--p-anchor" data-href="http://www.dnotice.org.uk/danotices/" href="http://www.dnotice.org.uk/danotices/" rel="noopener noreferrer" target="_blank">
    描述为
   </a>
   “自愿执行”，称 “媒体没有义务遵守发布的任何禁令”。如果这是真的，那么意味着在发布斯诺登文件之前，卫报应该无需咨询国防部。
  </p>
  <p class="graf graf--p">
   但是，该委员会的会议纪要中却说：“卫报有义务根据条款寻求……建议”。会议纪要补充说：“未能寻求建议是引起关注的关键原因，并且[该委员会]已经为解决问题做出了
   <em class="markup--em markup--p-em">
    相当大的努力
   </em>
   。”
  </p>
  <p class="graf graf--p">
   他们把真相的曝光视为“需要解决的问题”。
  </p>
  <h3 class="graf graf--p graf--startsWithDoubleQuote">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     “相当大的努力”
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   这些“相当大的努力”包括，该委员会于2013年6月7日（即《卫报》发布第一批斯诺登文件后的第二天）
   <strong class="markup--strong markup--p-strong">
    向所有英国主要媒体编辑部发出了禁令
   </strong>
   ，称：他们应避免发布会 “危及两国国家安全和危及英国人员的信息”。
  </p>
  <p class="graf graf--p">
   而且，
   <strong class="markup--strong markup--p-strong">
    该禁令本身还
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://order-order.com/2013/06/08/d-notice-june-7-2013/" href="https://order-order.com/2013/06/08/d-notice-june-7-2013/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     被标记为
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    “机密：不得在社交媒体上发布、广播或使用”。
   </strong>
  </p>
  <p class="graf graf--p">
   显然，该委员会不希望这份禁令被大众知道，甚至他们几乎成功了。只有右翼博客 Guido Fawkes 公开了这份禁令。
  </p>
  <p class="graf graf--p">
   根据委员会的会议记要，当时，“特别是情报机构，继续要求提供更多咨询意见（也就是审查禁令）”。
  </p>
  <p class="graf graf--p">
   情报部门清楚地知道，
   <strong class="markup--strong markup--p-strong">
    此类禁令绝非仅仅是向媒体提供建议的工具，而是一种威胁，威胁媒体不要发布斯诺登文件。
   </strong>
  </p>
  <p class="graf graf--p">
   一天晚上，就在第一个斯诺登文件的报道发布后，时任该委员会秘书的少将 Andrew Vallance 亲自打电话给《卫报》编辑 Alan Rusbridger。
  </p>
  <p class="graf graf--p">
   一位卫报
   <a class="markup--anchor markup--p-anchor" data-href="https://books.google.co.uk/books?id=ulDBAgAAQBAJ&amp;pg=PT116&amp;lpg=PT116&amp;dq=luke+harding+air+marshal+andrew+vallance&amp;source=bl&amp;ots=Aq42ENwAOM&amp;sig=ACfU3U0Osetygd61sr2TYmO1T_Ihxs0Wyw&amp;hl=en&amp;sa=X&amp;ved=2ahUKEwiqm_u20aPkAhUnRxUIHbRnD7gQ6AEwA3oECAoQAQ#v=onepage&amp;q=luke%2520harding%2520air%2520marshal%2520andrew%2520vallance&amp;f=false" href="https://books.google.co.uk/books?id=ulDBAgAAQBAJ&amp;pg=PT116&amp;lpg=PT116&amp;dq=luke+harding+air+marshal+andrew+vallance&amp;source=bl&amp;ots=Aq42ENwAOM&amp;sig=ACfU3U0Osetygd61sr2TYmO1T_Ihxs0Wyw&amp;hl=en&amp;sa=X&amp;ved=2ahUKEwiqm_u20aPkAhUnRxUIHbRnD7gQ6AEwA3oECAoQAQ#v=onepage&amp;q=luke%2520harding%2520air%2520marshal%2520andrew%2520vallance&amp;f=false" rel="noopener noreferrer" target="_blank">
    记者说
   </a>
   ，Vallance “明确表示了对《卫报》在发布文件之前未能事先咨询他的担忧”。
  </p>
  <p class="graf graf--p">
   在那年的晚些时候，英国首相戴维·卡梅伦（David Cameron）再次将该委员会的禁令系统
   <strong class="markup--strong markup--p-strong">
    用作对媒体的威胁
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他在给议员的一份
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2013/oct/28/david-cameron-nsa-threat-newspapers-guardian-snowden" href="https://www.theguardian.com/world/2013/oct/28/david-cameron-nsa-threat-newspapers-guardian-snowden" rel="noopener noreferrer" target="_blank">
    声明中说
   </a>
   ：“我不想使用禁令或通知书或其他更严厉的措施。我认为呼吁报纸的社会责任感要好得多。但是，如果他们没有表现出某种社会责任，政府就很难不采取行动了。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当权者的威胁发挥了作用。
   </strong>
  </p>
  <p class="graf graf--p">
   当时的 Press Gazette 杂志报道说：
   <strong class="markup--strong markup--p-strong">
    “《金融时报》和《泰晤士报》都没有提及斯诺登最初的揭露……而《电讯报》只发表了一篇简短的文章”。只有《独立报》“追随了实质性的揭露和指控”；“BBC 也选择了不理会斯诺登的故事”。
   </strong>
  </p>
  <p class="graf graf--p">
   但是，当时的《卫报》仍然不为所动。
  </p>

  <p class="graf graf--p">
   根据该委员会会议
   <a class="markup--anchor markup--p-anchor" data-href="https://dsma.uk/wp-content/uploads/2018/12/131107-DPBAC-Meeting-Minutes.pdf" href="https://dsma.uk/wp-content/uploads/2018/12/131107-DPBAC-Meeting-Minutes.pdf" rel="noopener noreferrer" target="_blank">
    纪要
   </a>
   ，《卫报》继续发表斯诺登文件这一事实，“无疑在某些方面引发了对该系统未来用途的质疑”。如果禁令系统无法阻止《卫报》发布令GCHQ最丢脸的机密，那还有什么用？
  </p>
  <p class="graf graf--p">
   对于他们来说，现在是时候控制《卫报》并确保这种抗旨的情况不再发生了。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     GCHQ和笔记本电脑
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   间谍部门在其 “相当大的努力” 之上继续加力，尽一切办法阻止媒体的曝光。
  </p>
  <p class="graf graf--p">
   2013年7月20日，英国间谍机构GCHQ官员在第一篇与斯诺登相关的文章发表六周后，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2013/aug/20/nsa-snowden-files-drives-destroyed-london" href="https://www.theguardian.com/world/2013/aug/20/nsa-snowden-files-drives-destroyed-london" rel="noopener noreferrer" target="_blank">
    直接闯进了
   </a>
   卫报在伦敦国王十字区的办公室。
  </p>
  <p class="graf graf--p">
   结果是，应政府和维稳部门的要求，
   <strong class="markup--strong markup--p-strong">
    《卫报》的副主编 Paul Johnson 和其他两个人
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/uk-news/2014/jan/31/footage-released-guardian-editors-snowden-hard-drives-gchq" href="https://www.theguardian.com/uk-news/2014/jan/31/footage-released-guardian-editors-snowden-hard-drives-gchq" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     花了三个小时的时间
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    销毁了所有存储斯诺登文件的笔记本电脑
   </strong>
   。
  </p>
  <p class="graf graf--p">
   据该报的一位记者说，《卫报》的工作人员
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/uk-news/2014/jan/31/footage-released-guardian-editors-snowden-hard-drives-gchq" href="https://www.theguardian.com/uk-news/2014/jan/31/footage-released-guardian-editors-snowden-hard-drives-gchq" rel="noopener noreferrer" target="_blank">
    带来了
   </a>
   “角向磨光机、带钻头的电动打磨机、还有口罩”；“间谍机构提供了一种高科技设备，即 “消磁器”，可以消磁并删除数据。”
  </p>
  <p class="graf graf--p">
   副主编 Johnson 说，销毁计算机是 “纯粹的象征性行为”，并补充说：“政府和 GCHQ 都知道这点，因为我们已经告知他们，这些文件已被带到美国与《纽约时报》共享。揭露报告会将继续进行。没有任何改变。”
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    真的没有任何改变吗？不，有些东西的确变了。
   </strong>
   正如那个发禁令的委员会在2013年11月的会议记录中概述的那样：“ [在7月底（由于计算机被销毁），卫报已开始寻求并接受本委员会的建议，不要公开某些高度敏感的细节，从那时起，与委员会的对话是合理的，而且正在改善 ……”
  </p>
  <p class="graf graf--p">
   Johnson 显然轻敌了。
   <strong class="markup--strong markup--p-strong">
    英国间谍部门所做的不只是“象征性行为”；它既是权力的表现，也是明显的威胁
   </strong>
   。
  </p>
  <p class="graf graf--p">
   当时，《卫报》是揭露美国和英国间谍机构大规模监视真相的举报人所依赖的唯一主要报纸，依靠其接收文件并保护举报人的身分；这对间谍机构构成了挑战。
  </p>
  <p class="graf graf--p">
   对《卫报》施加的越来越强悍的压制奏效了。该委员会
   <a class="markup--anchor markup--p-anchor" data-href="https://dsma.uk/wp-content/uploads/2018/12/140508-DPBAC-Meeting-Minutes.pdf" href="https://dsma.uk/wp-content/uploads/2018/12/140508-DPBAC-Meeting-Minutes.pdf" rel="noopener noreferrer" target="_blank">
    指出
   </a>
   ，在GCHQ监督了该编辑部的笔记本电脑捣毁之后，“与 Guardian 的互动……持续加强”。（请把“互动”理解为
   <strong class="markup--strong markup--p-strong">
    传唤/喝茶
   </strong>
   ）
  </p>
  <p class="graf graf--p">
   主编 Rusbridger 后来向民政事务专责委员会
   <a class="markup--anchor markup--p-anchor" data-href="https://books.google.co.uk/books?id=mjbDAwAAQBAJ&amp;pg=PA144&amp;lpg=PA144&amp;dq=alan+rusbridger+air+vice+marshal+vallance&amp;source=bl&amp;ots=ZwhId2fAs1&amp;sig=ACfU3U1lPIkHLsQBjDHa2yzQHrKYUZHmlw&amp;hl=en&amp;sa=X&amp;ved=2ahUKEwj_iOrp-qXkAhUVUhUIHXKFBN4Q6AEwBHoECAkQAQ#v=onepage&amp;q=vallance&amp;f=false" href="https://books.google.co.uk/books?id=mjbDAwAAQBAJ&amp;pg=PA144&amp;lpg=PA144&amp;dq=alan+rusbridger+air+vice+marshal+vallance&amp;source=bl&amp;ots=ZwhId2fAs1&amp;sig=ACfU3U1lPIkHLsQBjDHa2yzQHrKYUZHmlw&amp;hl=en&amp;sa=X&amp;ved=2ahUKEwj_iOrp-qXkAhUVUhUIHXKFBN4Q6AEwBHoECAkQAQ#v=onepage&amp;q=vallance&amp;f=false" rel="noopener noreferrer" target="_blank">
    作证
   </a>
   ，关于斯诺登事件后，D-Notice 委员会的 Vallance 与他本人“合作”（
   <strong class="markup--strong markup--p-strong">
    贴身审查
   </strong>
   ），并且 Vallance 甚至 “在《卫报》办公室里与所有记者交谈”。
  </p>
  <p class="graf graf--p">
   但是，
   <strong class="markup--strong markup--p-strong">
    这种胁迫和威胁攻势中最重要的部分是使《卫报》同意加入 D-Notice 委员会本身
   </strong>
   。该委员会的会议记要对此很明确，其中
   <a class="markup--anchor markup--p-anchor" data-href="https://dsma.uk/wp-content/uploads/2018/12/140508-DPBAC-Meeting-Minutes.pdf" href="https://dsma.uk/wp-content/uploads/2018/12/140508-DPBAC-Meeting-Minutes.pdf" rel="noopener noreferrer" target="_blank">
    指出
   </a>
   ：“[原文]这一过程最终结束于任命 Paul Johnson（卫报副主编）担任 DPBAC [即 D-Notice 委员会]成员”。（请理解为
   <strong class="markup--strong markup--p-strong">
    收编
   </strong>
   ）
  </p>
  <p class="graf graf--p">
   在2013年底或2014年初的某个时候，Johnson 被要求加入该 委员会。Johnson 是副主编，曾在英国情报人员的注视下捣毁了编辑部的计算机。他在
   <a class="markup--anchor markup--p-anchor" data-href="https://dsma.uk/wp-content/uploads/2018/12/140508-DPBAC-Meeting-Minutes.pdf" href="https://dsma.uk/wp-content/uploads/2018/12/140508-DPBAC-Meeting-Minutes.pdf" rel="noopener noreferrer" target="_blank">
    2014年5月
   </a>
   参加了该委员会第一次会议，并一直在那里待到
   <a class="markup--anchor markup--p-anchor" data-href="https://dsma.uk/wp-content/uploads/2018/12/181108-DSMA-Committee-Meeting-Minutes.pdf" href="https://dsma.uk/wp-content/uploads/2018/12/181108-DSMA-Committee-Meeting-Minutes.pdf" rel="noopener noreferrer" target="_blank">
    2018年10月
   </a>
   。
  </p>
  <p class="graf graf--p">
   简单说 ，他一直与那些阻止他的报道发表的审查人员坐在一起。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     一个新主编
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   卫报主编 Alan Rusbridger 在斯诺登文件的报道中承受了巨大的压力，但是他同意了 Johnson 加入 D-Notice 委员会，以此作为对维稳部门的战术手段。在他的整个任期内，《卫报》继续发表一些批评间谍机构的报道。
  </p>
  <p class="graf graf--p">
   但是，在2015年3月，《卫报》
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/media/2015/mar/20/guardian-appoints-katharine-viner-new-editor-in-chief" href="https://www.theguardian.com/media/2015/mar/20/guardian-appoints-katharine-viner-new-editor-in-chief" rel="noopener noreferrer" target="_blank">
    任命了
   </a>
   一位新主编 Katharine Viner 后，这种情况发生了变化。
  </p>
  <p class="graf graf--p">
   这个新主编 Katharine Viner 在处理间谍机构相关报道方面的经验显然比 Rusbridger 少得多。
  </p>
  <p class="graf graf--p">
   Viner
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/media/2015/mar/20/guardian-appoints-katharine-viner-new-editor-in-chief" href="https://www.theguardian.com/media/2015/mar/20/guardian-appoints-katharine-viner-new-editor-in-chief" rel="noopener noreferrer" target="_blank">
    最初来自
   </a>
   时尚和娱乐杂志《Cosmopolitan》，她在国家安全报道方面
   <strong class="markup--strong markup--p-strong">
    没有任何工作经验
   </strong>
   。据内部人士称，在斯诺登事件中，她的领导力要比美国的 Janine Gibson 差得多（Gibson 是 Rusbridger 继任者的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/media/2015/jun/16/janine-gibson-appointed-editor-in-chief-buzzfeed-uk" href="https://www.theguardian.com/media/2015/jun/16/janine-gibson-appointed-editor-in-chief-buzzfeed-uk" rel="noopener noreferrer" target="_blank">
    另一位候选人
   </a>
   ）。
  </p>
  <p class="graf graf--p">
   当时，Viner 是澳大利亚《卫报》（Guardian Australia）的主编，
   <strong class="markup--strong markup--p-strong">
    澳大利亚卫报是在斯诺登文件报道首次发表
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/Guardian_Australia" href="https://en.wikipedia.org/wiki/Guardian_Australia" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     前两周
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    成立的 —— 澳大利亚和新西兰组成了斯诺登揭露的“五眼”监视联盟的五分之二
   </strong>
   。
  </p>
  <p class="graf graf--p">
   这对那些间谍机构来说是个好机会。他们的诱导似乎从第二年就启动了。
  </p>
  <p class="graf graf--p">
   2016年11月，《卫报》与军情五处 MI5 负责人 Andrew Parker 合作发表了史无前例的 “独家新闻”。该文章指出，这是 “该间谍机构在107年历史上的第一次现任负责人接受报纸采访”。
  </p>
  <p class="graf graf--p">
   请注意，这篇所谓的 “独家” 是由副主编 Paul Johnson 共同撰写的，他以前从未写过有关间谍机构的文章，并且他当时仍在审查部门 D-Notice 委员会中任职。而且那篇文章中未提及这件事。
  </p>
  <p class="graf graf--p">
   Johnson 和他的合著者写道：“尽管斯诺登文件已经发表，但军情五处的负责人 Parker 是在和卫报、而不是其他报纸交谈”……（请感受
   <strong class="markup--strong markup--p-strong">
    狗粮的兴奋
   </strong>
   ）
  </p>
  <p class="graf graf--p">
   Parker 告诉两位记者：“我们认识到，在瞬息万变的世界中，我们也必须做出改变。我们有责任谈论我们的工作并进行解释。”
  </p>
  <p class="graf graf--p">
   在军情五处的表演性采访结束四个月后，2017年3月，《卫报》又发布了另一个前所未有的所谓 “独家报道”，这次是与英国军情六处的现任领导人 Alex Younger 的“合作”。（请感受
   <strong class="markup--strong markup--p-strong">
    收编的代价
   </strong>
   ）
  </p>
  <p class="graf graf--p">
   这份独家情报是由英国秘密情报局授予《卫报》的调查编辑 Nick Hopkins 的，他在14个月前被任命。
  </p>

  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “简而言之，我们必须吸引现代英国最棒的人，” Younger 在这篇报道中说， “英国各地的每个社区都应该感到自己的需求，无论其背景或地位如何。”

   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/uk-news/2019/aug/07/salisbury-attack-metropolitan-police-examine-role-vladimir-putin-russia" href="https://www.theguardian.com/uk-news/2019/aug/07/salisbury-attack-metropolitan-police-examine-role-vladimir-putin-russia" rel="noopener noreferrer" target="_blank">

  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这些间谍部门不断向《卫报》提供这些所谓的“独家”信息，很明显是牵制行动的一部分，从而令这家唯一的独立报纸变成间谍部门的喉舌

   </strong>
   。

   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">

  <p class="graf graf--p">

  <p class="graf graf--p">
   阿桑奇现在在贝尔马什（Belmarsh）最高安全级别的监狱里，即将被引渡到美国，面临永无止境的监禁。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     政治斗争中的玩偶
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在 Viner 的领导下，《卫报》的另一个主要重点是攻击英国工党领袖杰里米·科宾。
  </p>
  <p class="graf graf--p">
   其背景是 Corbyn 最近已成为间谍部门的目标。
  </p>
  <p class="graf graf--p">
   2015年，在他当选为工党领袖之后不久，《星期日泰晤士报》就警告说：“如果 Corbyn 当选首相，军队会直接挑战”。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     在这里看到间谍部门对工党领袖实施的惊人的攻击运动
    </strong>
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%a7%98%e5%af%86%e5%ae%a3%e4%bc%a0%e7%bd%91%e7%bb%9c/" href="https://www.iyouport.org/%e7%a7%98%e5%af%86%e5%ae%a3%e4%bc%a0%e7%bd%91%e7%bb%9c/" rel="noopener noreferrer" target="_blank">
     秘密宣传战网络
    </a>
    》
   </li>
   <li class="graf graf--li">
    在这里看到这场攻击运动之所以能成功的“基础设施”《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" href="https://www.iyouport.org/%e5%bf%83%e7%90%86%e6%93%8d%e7%ba%b5%e7%9a%84%e7%a7%98%e5%af%86%ef%bc%9a%e8%a1%8c%e4%b8%ba%e7%bb%8f%e6%b5%8e%e5%ad%a6pr%ef%bc%9fvideo/" rel="noopener noreferrer" target="_blank">
     心理操纵的秘密：”行为经济学”+PR=？
    </a>
    》有视频
   </li>
  </ul>
  <p class="graf graf--p">
   2017年5月20日，即2017年大选之前的两周多，《每日电讯报》声称 Corbyn 与爱尔兰共和军有联系。大都会警察特别处也在同一时期监视 Corbyn。
  </p>
  <figure aria-describedby="caption-attachment-10256" class="wp-caption aligncenter" id="attachment_10256" style="width: 1400px">
   <img alt="" class="size-full wp-image-10256 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpg?resize=1100%2C732&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpg?w=1400&amp;ssl=1 1400w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpg?resize=768%2C511&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpg?resize=1100%2C732&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="732" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/11/5.jpg?resize=1100%2C732&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

   </figcaption>
  </figure>
  <p class="graf graf--p">
   然后，在大选前夕，《电讯报》发布了一篇采访MI6前局长 Richard Dearlove 的文章，标题是：“Jeremy Corbyn 对这个国家构成威胁”……
  </p>
  <p class="graf graf--p">
   自2015年 Corbyn 当选以来，《卫报》自己就
   <strong class="markup--strong markup--p-strong">
    发表了大量批评 Corbyn 的新闻和评论文章，并且该报纸明显的敌对立场已
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="http://www.medialens.org/index.php/alerts/alert-archive/2019/910-the-campaign-to-stop-corbyn-smears-racism-censorship.html" href="http://www.medialens.org/index.php/alerts/alert-archive/2019/910-the-campaign-to-stop-corbyn-smears-racism-censorship.html" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     广为人知
    </strong>
   </a>
   。
  </p>
  <p class="graf graf--p">
   《卫报》和《观察家报》发表了数百篇抹黑工党“反犹主义”的文章，并且自去年年初以来，共发表了50多篇此类文章，标题显然都指向 Corbyn。
  </p>
  <p class="graf graf--p">
   尽管工党中确实存在反犹主义，但有证据表明反犹主义始终处于很低的水平。自2015年9月 Corbyn 成为工党领袖以来，已经对那 0.06％的工党成员进行了调查，以制止反犹言论或帖子。
  </p>
  <p class="graf graf--p">
   2016年，由工党委托进行的一项独立调查得出的
   <a class="markup--anchor markup--p-anchor" data-href="https://labour.org.uk/wp-content/uploads/2017/10/Chakrabarti-Inquiry-Report-30June16.pdf" href="https://labour.org.uk/wp-content/uploads/2017/10/Chakrabarti-Inquiry-Report-30June16.pdf" rel="noopener noreferrer" target="_blank">
    结论是
   </a>
   ，该党 “
   <strong class="markup--strong markup--p-strong">
    并没有
   </strong>
   反犹主义、伊斯兰恐惧症或其他形式的种族主义。相反，该党发起了每一项英国种族平等法律。”
  </p>
  <p class="graf graf--p">
   但是媒体的联手抹黑还是起效了。Corbyn 在任职的头两年中支持率大幅下降。自2016年1月以来，《卫报》已经发表了1,215篇抹黑工党的报道，平均每天大约一篇。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #ff6600;">
    <strong class="markup--strong markup--p-strong">
     可靠的出版物？
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   2019年5月20日，泰晤士报
   <a class="markup--anchor markup--p-anchor" data-href="https://www.thetimes.co.uk/article/torture-britain-breaks-law-in-ministry-of-defence-secret-policy-2rl5dn2kd" href="https://www.thetimes.co.uk/article/torture-britain-breaks-law-in-ministry-of-defence-secret-policy-2rl5dn2kd" rel="noopener noreferrer" target="_blank">
    报道了
   </a>
   Rendition Project 提出的信息自由请求，该组织是一群从事酷刑问题研究的学术专家，研究表明国防部一直在“制定一项秘密的酷刑政策，可能导致虐待被拘留者的行为”。
  </p>
  <p class="graf graf--p">
   传统上，这应该是卫报报道的东西，而不是鲁珀特·默多克（Rupert Murdoch）拥有的泰晤士报。然而，据一位民间社会人士透露，
   <strong class="markup--strong markup--p-strong">
    许多从事这一领域工作的民间团体不再信任《卫报》了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   一位前《卫报》记者也是这样说的：“重要的是，最近有关英国在酷刑和勾结中对嫌疑人和其他被拘留者进行审讯的独家消息已经传递给泰晤士报而不是卫报。
  </p>
  <p class="graf graf--p">
   泰晤士报以 “酷刑：英国在国防部的秘密政策中违反法律” 为标题，发表了独家新闻。但是，请注意，在这篇文章发表之前，国防部就向《卫报》提供了泰晤士报即将使用的相同文件，
   <strong class="markup--strong markup--p-strong">
    官方认为卫报可以通过侧面描述来减轻泄密的影响。
   </strong>
  </p>
  <p class="graf graf--p">
   《卫报》在《泰晤士报》的独家之前就
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/law/2019/may/20/rewritten-mod-guidance-could-leave-door-open-for-torture" href="https://www.theguardian.com/law/2019/may/20/rewritten-mod-guidance-could-leave-door-open-for-torture" rel="noopener noreferrer" target="_blank">
    发表了
   </a>
   他们自己的文章，其标题肯定让政府感到高兴：“国防部表示，修订后的酷刑指导方针不会降低标准”。但是，对文件的检查表明这显然是虚假信息。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    《卫报》从过去的短短六年间，就自然地从一个揭露者变成了间谍部门的帮凶，只为了扩展其业务
   </strong>
   。
  </p>
  <p class="graf graf--p">
   曾经相对独立的媒体平台已被间谍部门废掉了 —— 就是那些始终在担心自己的阴暗面被曝光的家伙们。
  </p>
  <p class="graf graf--p">
   这就引出了问题：下一个斯诺登可以投靠谁？⚪️
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dailymaverick.co.za/article/2019-09-11-how-the-uk-security-services-neutralised-the-countrys-leading-liberal-newspaper/" href="https://www.dailymaverick.co.za/article/2019-09-11-how-the-uk-security-services-neutralised-the-countrys-leading-liberal-newspaper/" rel="noopener noreferrer" target="_blank">
    How the UK Security Services neutralised the country’s leading liberal newspaper
   </a>
  </p>
