     <em>
      <strong>
       这是一个有趣的实战。起底一个法国的加密货币骗局，居然在其中挖出了疑似中国网络军队的成员。
      </strong>

  <p>
   正如您知道的，开源情报、道德黑客和社交工程，三部分是相通的，往往会结合使用，尤其是在一项深入追查的实践中。
  </p>
  <p class="graf graf--p">
   本案例就是这样一套实践笔记。
  </p>
  <ul>
   <li class="graf graf--p">
    <span style="color: #00ccff;">
     <em>
      <strong class="markup--strong markup--p-strong">
       尤其提醒注意几点
      </strong>
      ：1、社交网络情报的重要性和致命性 —— 那个中国网军就栽在这里；2、社交工程攻击模版的设计思路 —— 可以非常疯狂；3、追击和反追击同时需要注意的事项 —— 尤其是反追击的您，因为您的对手在此是非常专业的，其能力应该比本文更进一步。
     </em>
    </span>
   </li>
  </ul>
  <p class="graf graf--p">
   ⚠️这些方法是任何警察和间谍都惯于采用的，并无很高的技术含量，但它们的危害性非常大，如果你处在逃脱者的位置上，通过本文您应该大致能看出为什么我们要不断强调一系列基本的保护性措施。
  </p>
  <p>
   再次，这只是最基本的侦查，根据不同国家谍报机构的能力，实际上那些间谍会做得更加精彩和深入。
  </p>
  <p>
   不过一般人遇不到那种级别的特工。于是这些基本侦查手段是应该被大众熟悉的。
  </p>

  <p class="graf graf--p">
   今年是间谍挑战游戏的第三年。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “间谍挑战”是由三位研究人士于2016年创立的，这是一个基于OSINT，社交工程和物理攻防的竞赛：创建者是 MélikLemariey、Sylvain Hajri 和 Christophe Baland。
  </p>
  <p class="graf graf--p">
   本次比赛涉及一些由三人或四人组成的团队，根据任务自行组队。您也可以试着来玩。
  </p>
  <p class="graf graf--p">
   这个游戏分为三个阶段：OSINT、物理跟踪和物理入侵。本文只介绍前两部分。
  </p>
  <p class="graf graf--p">
   竞争者面临着一些挑战，他们需要努力赢过对手才能被选中进入下一阶段。
  </p>
  <p class="graf graf--p">
   在每个阶段结束时，参与者会发送所收集信息和所采取措施的详细报告。然后组织者使用这些报告来定义团队排名。
  </p>
  <h4 class="graf graf--h4">
   <strong class="markup--strong markup--h4-strong">
    —— 介绍 ——
   </strong>
  </h4>
  <p class="graf graf--p">
   根据第 502-NXH-857 号任务令，“Sociétéd’InvestigationSpécialisées” 组织的总部招募了33个由精英特工组成的小组，他们的调查目标是加密货币筹款活动 SupremIoT 公司。
  </p>
  <p class="graf graf--p">
   实际上，参与此次筹款活动的一些投资者已经对 SupremIoT 目标的合法性产生了怀疑，并要求尽可能多地获取有关该公司及其员工的信息。
  </p>
  <p class="graf graf--p">
   你的团队会为你提供有关此案例的部分基本档案，其余为机密内容需要你去挖掘。
  </p>
  <p>
   第一步先采集线索，第二步是为设计攻击作准备。
  </p>

   虽然 Ian 也喜欢美女，但除非特殊情况否则他不会结婚。另一方面，在查看了他经常光顾的地方之后，你可以通过谈论他可能感兴趣的主题，例如 漫画、动漫、国际象棋，
   <strong class="markup--strong markup--p-strong">
    很好地设计一套社交工程攻击的方案
   </strong>
   。
  </p>
  <p class="graf graf--p">

   3、Umberto
  </p>
  <p class="graf graf--p">
   Umberto 在开发加密货币方面看起来并不那么专业，正如上述视频逐帧分析中看到的那样，也要感谢他参加的那个声称在短短四个小时内就可以教授开发智能合约技术的培训。他最近才刚从建筑行业转行的经历符合这一假设。
  </p>
  <p class="graf graf--p">
   邀请他参加有足球比赛的活动可以成为接近成功的保证。知道他对国家的怀旧情绪，围绕葡萄牙社区组织的活动也会很有效。
  </p>
  <p class="graf graf--p">
   最后，挖掘经常为他提供额外服务的健身房，也能是收集信息的好方法。
  </p>
  <p class="graf graf--p">
   4、Benoît
  </p>
  <p class="graf graf--p">
   此人是标准诈骗者。除了承诺让他与世界上最伟大的人建立独家伙伴关系外，还能为他提供什么呢？
  </p>
  <p class="graf graf--p">
   然而，应该注意的是，如果他不太了解IT技术、并且对点击不那么谨慎的话，那么网络钓鱼攻击篡夺他的银行账户或他的首选交易平台账户可能是有效的。
  </p>
  <p class="graf graf--p">
   不要忘记 Benoît 似乎是一个
   <strong class="markup--strong markup--p-strong">
    自我表现欲狂热者
   </strong>
   的事实（参见相同照片的使用，地理定位错误等细节……）并且希望通过定期发布关于他的工作的信息来推动自己的形象，而他的合作者并没有透露很多关于工作的内容。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这种人也许会轻易说出一些他本应该保密的事
   </strong>
   ，他通过这种方法使自己看起来更有趣，特别是对女性而言，他对美女会说更多的话。这就为你的攻击模型奠定了一些基础。

   5、Katrien
  </p>
  <p class="graf graf--p">
   Katrien 被认为是千禧一代的标准社交网络成瘾者，并且很自恋，为自己的外表感到自豪。引申意义是
   <strong class="markup--strong markup--p-strong">
    她缺乏自信心
   </strong>
   。
  </p>
  <p class="graf graf--p">
   通过 Instagram 社交网络对她进行社交工程攻击似乎是最容易的，重点在于，你的袜子木偶需要拥有很多“粉丝”（去购买僵尸粉）并且你“很帅”（多发布不知名的帅哥照片，并声称那是你自己）。
  </p>
  <p class="graf graf--p">
   还应该记住，
   <strong class="markup--strong markup--p-strong">
    她是 SupremIoT 的实习生，她缺乏经验
   </strong>
   ，可能导致她无意中泄露组织的秘密。
  </p>
  <p class="graf graf--p">
   6、Pascal
  </p>
  <p class="graf graf--p">

   <strong class="markup--strong markup--p-strong">
    机会主义者的形象
   </strong>
   ，他对密码学知之甚少，但对自己的工作充满热情。
  </p>
  <p class="graf graf--p">
   而另一方面，他有点笨拙，他发布的广告与他的推特账号具有相同的昵称，说明他一点谨慎意识都没有。
  </p>
  <p class="graf graf--p">
   如果你想采取不那么积极的方法，可以发送独家邀请，请他参观鳄梨冰沙制造公司并承诺全程免费。

   一旦洗钱的那个人离开，渗透进内部的特工就想办法说服 Advei 跟他索取一些钱以偿还“不幸的客户”（只是编造的），就是为了让他打开那个箱子。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4084 jetpack-lazy-image" data-lazy-sizes="(max-width: 698px) 100vw, 698px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/64-钱.png?resize=698%2C588&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/64-钱.png?w=698&amp;ssl=1 698w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/64-钱.png?resize=300%2C253&amp;ssl=1 300w" data-recalc-dims="1" height="588" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/64-钱.png?resize=698%2C588&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="698"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4084" data-recalc-dims="1" height="588" sizes="(max-width: 698px) 100vw, 698px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/64-钱.png?resize=698%2C588&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/64-钱.png?w=698&amp;ssl=1 698w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/64-钱.png?resize=300%2C253&amp;ssl=1 300w" width="698"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    —— Mei-Shi 的案子 ——
   </strong>
  </p>
  <p class="graf graf--p">
   使用 KylinOS 操作系统销售她的 Raspberry Pi 的这位中国代理人 Mei-Shi 对总部非常感兴趣，但不仅仅如此……
  </p>
  <p class="graf graf--p">
   事实上，Mei-Shi 的老板要求她将电子设备送回去。于是跟踪者的目标就是不惜一切代价抢在被送回之前买到她的设备。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-4085 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/65-meishi.png?resize=1100%2C528&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/65-meishi.png?w=1104&amp;ssl=1 1104w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/65-meishi.png?resize=300%2C144&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/65-meishi.png?resize=768%2C369&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/65-meishi.png?resize=1024%2C492&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/65-meishi.png?resize=1100%2C528&amp;ssl=1 1100w" data-recalc-dims="1" height="528" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/65-meishi.png?resize=1100%2C528&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4085" data-recalc-dims="1" height="528" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/65-meishi.png?resize=1100%2C528&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/65-meishi.png?w=1104&amp;ssl=1 1104w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/65-meishi.png?resize=300%2C144&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/65-meishi.png?resize=768%2C369&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/65-meishi.png?resize=1024%2C492&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/65-meishi.png?resize=1100%2C528&amp;ssl=1 1100w" width="1100"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   间谍团队中的一人与 Mei-Shi 会面，以冒充他人的合法代理人。团队其他人协助更大范围的跟踪。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4086 jetpack-lazy-image" data-lazy-sizes="(max-width: 912px) 100vw, 912px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/66-会面地点.png?resize=912%2C684&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/66-会面地点.png?w=912&amp;ssl=1 912w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/66-会面地点.png?resize=300%2C225&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/66-会面地点.png?resize=768%2C576&amp;ssl=1 768w" data-recalc-dims="1" height="684" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/66-会面地点.png?resize=912%2C684&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="912"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4086" data-recalc-dims="1" height="684" sizes="(max-width: 912px) 100vw, 912px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/66-会面地点.png?resize=912%2C684&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/66-会面地点.png?w=912&amp;ssl=1 912w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/66-会面地点.png?resize=300%2C225&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/05/66-会面地点.png?resize=768%2C576&amp;ssl=1 768w" width="912"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   交易完成后，Mei-Shi 离开了，但渗透者被法国反间谍拦截。他们想了解更多有关 Raspberry Pi 的信息，特别是关于拥有此设备的人。
  </p>
  <p class="graf graf--p">
   渗透者被带到一个单独的房间，等待法国特工前来质疑他，并且被戴上了手铐。他身边有一个金属条和一个发夹，这样他就可以自己打开手铐。
  </p>
  <p class="graf graf--p">
   这里是考验基本操作方面的训练。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4087 jetpack-lazy-image" data-lazy-sizes="(max-width: 592px) 100vw, 592px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/67-手铐.png?resize=592%2C792&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/67-手铐.png?w=592&amp;ssl=1 592w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/67-手铐.png?resize=224%2C300&amp;ssl=1 224w" data-recalc-dims="1" height="792" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/67-手铐.png?resize=592%2C792&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="592"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4087" data-recalc-dims="1" height="792" sizes="(max-width: 592px) 100vw, 592px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/67-手铐.png?resize=592%2C792&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/67-手铐.png?w=592&amp;ssl=1 592w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/67-手铐.png?resize=224%2C300&amp;ssl=1 224w" width="592"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   关押他的房间受到射频识别(RFID)访问控制的保护。团队的其他特工负责对付前门的警卫，目的是复制到警卫的门禁卡。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-4088 jetpack-lazy-image" data-lazy-sizes="(max-width: 1092px) 100vw, 1092px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/68-复制门禁卡.png?resize=1092%2C714&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/68-复制门禁卡.png?w=1092&amp;ssl=1 1092w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/68-复制门禁卡.png?resize=300%2C196&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/68-复制门禁卡.png?resize=768%2C502&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/68-复制门禁卡.png?resize=1024%2C670&amp;ssl=1 1024w" data-recalc-dims="1" height="714" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/68-复制门禁卡.png?resize=1092%2C714&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1092"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-4088" data-recalc-dims="1" height="714" sizes="(max-width: 1092px) 100vw, 1092px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/68-复制门禁卡.png?resize=1092%2C714&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/68-复制门禁卡.png?w=1092&amp;ssl=1 1092w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/68-复制门禁卡.png?resize=300%2C196&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/68-复制门禁卡.png?resize=768%2C502&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/05/68-复制门禁卡.png?resize=1024%2C670&amp;ssl=1 1024w" width="1092"/>

   一旦被捕者逃脱，将立刻与在外面接应的团队成员汇合，游戏接近完成；现在他们要做的最后几件事是什么？
  </p>
  <p class="graf graf--p">
   好啦，希望通过这个游戏能帮您了解到深度调查取证的一些小技巧；并且，包括如何防御这些侦查手段。⚪️
