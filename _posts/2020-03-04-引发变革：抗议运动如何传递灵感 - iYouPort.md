  <p class="graf graf--p">
   每一位熟悉IYP内容的读者都知道，我们有多个栏目为活动家支援，从技术到策略，从技巧到安防；我们会尽可能分析全球的经验并将其中最成功的和最直接相关的可能性推荐给中国社会的积极人士。
  </p>
  <p class="graf graf--p">
   去年曾经一度我们的相关更新出现衰落，因为从流量上观察，上述内容被感兴趣的程度是最低的（幸运的是，防守技术的流量相对较高）；
  </p>
  <p class="graf graf--p">
   我们是网站，和其他所有网站一样，我们对流量的分析是为指导下一步的内容选取 —— 流量过低意味着“相关内容在本地受众范围内不讨喜”。
  </p>
  <p class="graf graf--p">
   虽然我们始终认为，
   <strong class="markup--strong markup--p-strong">
    缺乏相关技术技巧方面的培训，是绝大多数行动失败的主要原因
   </strong>
   —— 而非政治环境、社会环境问题，后者都是有可能突破的。
  </p>
  <p class="graf graf--p">
   如果信心不足，那不如先打稳根基；盲目行动的后果可能很惨重，会令整个社会在未来的许多年一蹶不振。
  </p>
  <p class="graf graf--p">
   信心来自于对行动的深刻理解，对策划的充分性、合理性、和适应性的详尽分析，对原理和创意的有效协调。
  </p>
  <p class="graf graf--p">
   这些都需要知识和智慧，而不仅仅是勇气。
  </p>
  <p class="graf graf--p">
   以下是 Rivera Sun 的 文章，充分体现了我们的观点。
  </p>
  <p class="graf graf--p">

  <p class="graf graf--p">
   变革并非凭空发生的。
  </p>
  <p class="graf graf--p">
   抵抗是一个连续的过程。
  </p>
  <p class="graf graf--p">
   非暴力运动是在许多其他斗争的努力中产生的。
  </p>
  <p class="graf graf--p">
   关于如何组织变革的知识是运动和几代激进主义者的全球遗产，其灵感来源长达数百年。（第一次有记录的罢工发生在公元前1170年，当时埃及金字塔的建造者拒绝工作直到付给他们薪水；从那以后罢工运动就一直在以同样的方式发生）
  </p>

   <li class="graf graf--li">
    《

     别把注意力集中在抵抗上：预设性干预战术
    </a>

   </li>

  <p class="graf graf--p">
   我们一直在直接和间接地互相学习。我们模仿创意策略、复制策略、从错误中学习。别人的勇气使我们变得更加大胆。
  </p>
  <p class="graf graf--p">
   我每周都会为非暴力新闻收集到30–50个有关非暴力反抗行动的故事，该新闻摘要显示了全球各地的人们如何做出改变。
  </p>
  <p class="graf graf--p">
   在新闻文章中，我经常注意到
   <strong class="markup--strong markup--p-strong">
    在全球运动之间传递知识共享和行动灵感
   </strong>
   的清晰示例。
  </p>
  <p class="graf graf--p">
   德国 Wunseidel 2014年的非自愿步行马拉松哄骗极右翼示威者捐款，结果这些人稀里糊涂地为反纳粹组织（他们的对手）捐赠了10,000欧元。
  </p>
  <p class="graf graf--p">
   这一非暴力行动方式也激发了俄勒冈州波特兰市的一项类似的行动，在极右翼的大规模集会中为移民权利组织筹集到了36,000美元。
  </p>
  <p class="graf graf--p">
   去年，香港示威者特意组织了一条28英里的人链，其灵感来自1989年的波罗的海之路 —— 该人链涉及220万人，横跨拉脱维亚、立陶宛和爱沙尼亚数百公里。他们甚至将其命名为“Hong Kong Way”。
  </p>
  <p class="graf graf--p">
   当移民救援船的船长卡罗拉·拉克特瓦斯（Carola Racketewas）因挽救生命而被捕时，第二艘船的船员亚历克斯（Alex）受到了启发，也违反了这种不人道的法律。
  </p>
  <p class="graf graf--p">
   尽管互联网帮助了这种模仿现象的传递，但是
   <strong class="markup--strong markup--p-strong">
    思想从一个行动复制到另一个行动的方式并不是新鲜的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   纵观历史，尽管步伐稍慢，但这种情况已经很多次发生。
  </p>
  <p class="graf graf--p">
   例如，“抵制”一词是在1880年创造的，当时爱尔兰的租户发起了一场社会抵制行动，反对查尔斯·博伊科特上尉在残酷驱逐中的作用。
   <strong class="markup--strong markup--p-strong">
    在六个星期之内，远至纽约市的报纸都在使用该术语。
   </strong>
  </p>
  <p class="graf graf--p">
   几年后，随着该词继续流行，猜猜英国的哪个学生正在阅读英国报纸关于爱尔兰和其他斗争的报道？—— 一个叫 Mohandas K. Gandhi 的年轻人。
  </p>
  <p class="graf graf--p">
   这不是甘地唯一的灵感，因为他在为印度脱离英国统治而进行的斗争中动员了大规模罢工、抵制和公民抗命。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    他既具有高度的创新能力，又是深思熟虑的思想家和战略家。他显然是从他那个时代的斗争中学到的
   </strong>
   。
  </p>
  <p class="graf graf--p">
   他从东方和西方的众多全球作家和思想家那里汲取了关于非暴力行动和哲学的想法。而他的独特行动之后在全球范围内产生影响。
  </p>
  <p class="graf graf--p">
   其中一些是自发的，但其中
   <strong class="markup--strong markup--p-strong">
    很多是通过直接连接发生的
   </strong>
   。例如，非裔美国人与甘地及其继任者进行了长期而有据可查的交流。关于非暴力斗争的信件和文章发表在非裔美国人的报纸和杂志上。
  </p>
  <p class="graf graf--p">
   1950年代初期，詹姆士·劳森（James Lawson）牧师在甘地被暗杀之后前往印度，以加深对非暴力抵抗的研究。
  </p>
  <p class="graf graf--p">
   回国后，他成为美国民权运动最重要的战略设计师之一。
  </p>
  <p class="graf graf--p">
   在后来的几年中，他参与了许多劳工正义运动和其他运动。在他的一生中，他还教导过无数组织者，并
   <strong class="markup--strong markup--p-strong">
    强调培训和学习对运动成功的重要性
   </strong>
   。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-9984 jetpack-lazy-image" data-lazy-sizes="(max-width: 1100px) 100vw, 1100px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-6.jpg?resize=1100%2C733&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-6.jpg?w=1280&amp;ssl=1 1280w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-6.jpg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-6.jpg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-6.jpg?resize=1024%2C682&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-6.jpg?resize=1100%2C733&amp;ssl=1 1100w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-6.jpg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="733" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/10/2-6.jpg?resize=1100%2C733&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>

  <p class="graf graf--p">
   运动不仅共享战术和策略，还共享艺术主题。
  </p>
  <p class="graf graf--p">
   当我用蒲公英作为抵抗的象征写下小说《蒲公英起义》后，许多读者写信给我讲述了他们的运动中对蒲公英的使用，就像挪威反对加入欧盟运动、美国70年代的新社会运动、最近的 BlackLivesMatter 运动，甚至全球气候正义运动。
  </p>
  <p class="graf graf--p">
   蒲公英，就像它的名字一样，它是一个可以不断随处弹出的符号。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    音乐、艺术、口号和图像，以无数种方式在各种运动之间循环
   </strong>
   。作为突出的一个例子，民权运动的标志性歌曲《We Shall Overcome》具有许多化身。
  </p>
  <p class="graf graf--p">
   第一个版本是由非裔美国人牧师约瑟夫·提德利（Joseph Tidley）于1900年撰写的，名为 “I’ll Overcome Some Day”。
  </p>
  <p class="graf graf--p">
   这个版本在整个十年的劳工运动中都是众所周知的。
  </p>
  <p class="graf graf--p">
   第二版《I Will Overcome》是1945年在南卡罗来纳州查尔斯顿的雪茄工人罢工中演唱的。皮特·西格（Pete Seeger）和齐尔菲亚·霍顿（Zilphia Horton）（Highlander Center 的音乐总监）将此版本收录在他们出版的民歌书中。
  </p>
  <p class="graf graf--p">
   这首歌在民权运动中被重新点燃。
  </p>
  <p class="graf graf--p">
   Guy Carawan 选择它为培训课程的闭幕歌曲，马丁·路德·金 和 Coretta Scott King 都参加了此次培训。
  </p>
  <p class="graf graf--p">
   从那里开始，他们和许多其他民谣歌手帮助在运动中普及了这首歌。
  </p>
  <p class="graf graf--p">
   然而，⚠️
   <strong class="markup--strong markup--p-strong">
    表面化地模仿运动存在着危险
   </strong>
   。
  </p>
  <p class="graf graf--p">
   对阿拉伯之春起义的评估之一就是这样 —— 后来的所有运动都失败了，因为后来者们仅仅从观看突尼斯和埃及的大规模示威的电视报道和互联网录像中学到的东西。
  </p>
  <p class="graf graf--p">
   ⚠️其他国家/地区的运动仅复制了大规模街头抗议活动，而未能看到并使用罢工、抵制和大规模的不合作运动的配合。
   <strong class="markup--strong markup--p-strong">
    表面化的模仿是不行的，您需要首先理解它的原理。
   </strong>
  </p>
  <p class="graf graf--p">
   ⚠️当随后其他国家的抗议者涌上街道时，警察和军人的残酷镇压迅速使这些运动销声匿迹了 —— 因为行动者尚未能制定出可以转向的其他策略，尤其是经济抵制。行动很快因镇压而失败。

  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">

   </strong>
   。
