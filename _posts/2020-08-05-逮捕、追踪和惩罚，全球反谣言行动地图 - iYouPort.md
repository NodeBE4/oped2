  <h3 class="graf graf--p">
   <span style="color: #993366;">
    <strong class="markup--strong markup--p-strong">
     澳大利亚 🇦🇺
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   行动：政府专案组和媒体素养运动
  </p>
  <p class="graf graf--p">
   重点：外国虚假宣传活动和媒体素养
  </p>
  <p class="graf graf--p">
   2018年6月，政府的四个部门成立了一个专案组，以针对即将举行的澳大利亚大选确定潜在的网络攻击和外国影响力运动。
  </p>
  <p class="graf graf--p">
   选举诚信保证工作队由民政事务部领导，是在情报界不断警告 “外国对澳大利亚的干涉” 的情况下
   <a class="markup--anchor markup--p-anchor" data-href="https://www.sbs.com.au/news/anti-meddling-task-force-set-up-ahead-of-australian-by-elections" href="https://www.sbs.com.au/news/anti-meddling-task-force-set-up-ahead-of-australian-by-elections" rel="noopener noreferrer" target="_blank">
    成立的
   </a>
   。
  </p>
  <p class="graf graf--p">
   2019年2月，ABC 的一项调查显示，澳大利亚选举委员会已通知 Twitter 和 Facebook，他们必须遵守其平台上关于非法广告的通知。AEC 用法院命令威胁社交媒体巨头。
  </p>
  <p class="graf graf--p">
   4月，他们还发起了
   <a class="markup--anchor markup--p-anchor" data-href="https://www.aec.gov.au/elections/electoral-advertising/stopandconsider.htm" href="https://www.aec.gov.au/elections/electoral-advertising/stopandconsider.htm" rel="noopener noreferrer" target="_blank">
    “Stop and Consider”
   </a>
   运动，鼓励选民关注有关选举消息的新闻来源。
  </p>
  <p class="graf graf--p">
   正如其他报告
   <a class="markup--anchor markup--p-anchor" data-href="https://www.afr.com/business/media-and-marketing/advertising/intelligence-officials-plan-to-repel-fake-news-in-australian-federal-election-20180906-h151y2" href="https://www.afr.com/business/media-and-marketing/advertising/intelligence-officials-plan-to-repel-fake-news-in-australian-federal-election-20180906-h151y2" rel="noopener noreferrer" target="_blank">
    所揭示的那样
   </a>
   ，这种遏制的努力
   <strong class="markup--strong markup--p-strong">
    并不
   </strong>
   直接针对虚假信息。准确说是，针对一切政府不喜欢的信息。

   行动：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/world/middle_east/bahrain-charges-lawyer-of-sharing-fake-news-for-his-tweets/2019/05/15/524e4a04-7727-11e9-a7bf-c8a43b84ee31_story.html?utm_term=.dcd10ba9d68a" href="https://www.washingtonpost.com/world/middle_east/bahrain-charges-lawyer-of-sharing-fake-news-for-his-tweets/2019/05/15/524e4a04-7727-11e9-a7bf-c8a43b84ee31_story.html?utm_term=.dcd10ba9d68a" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p">
   重点：媒体管制

  <p class="graf graf--p">
   2019年5月，一位巴林律师因在社交媒体上发布了被当局认为 “可能危害公共秩序的虚假新闻” 而被捕。
  </p>
  <p class="graf graf--p">
   美联社
   <a class="markup--anchor markup--p-anchor" data-href="https://www.washingtonpost.com/world/middle_east/bahrain-charges-lawyer-of-sharing-fake-news-for-his-tweets/2019/05/15/524e4a04-7727-11e9-a7bf-c8a43b84ee31_story.html?utm_term=.dcd10ba9d68a" href="https://www.washingtonpost.com/world/middle_east/bahrain-charges-lawyer-of-sharing-fake-news-for-his-tweets/2019/05/15/524e4a04-7727-11e9-a7bf-c8a43b84ee31_story.html?utm_term=.dcd10ba9d68a" rel="noopener noreferrer" target="_blank">
    报道说
   </a>
   ，检察官引用了该律师 **过去的** 推文，这些推文质疑政府保护公众的能力，这是他被捕的主要原因。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    追踪过去的而不仅仅是当前的发布内容，也是中国当局经常使用的方法
   </strong>

  <p class="graf graf--p">
   行动：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dhakatribune.com/bangladesh/2018/10/08/president-signs-digital-security-bill-into-law" href="https://www.dhakatribune.com/bangladesh/2018/10/08/president-signs-digital-security-bill-into-law" rel="noopener noreferrer" target="_blank">
    立法
   </a>
   管制和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.economist.com/asia/2018/10/06/bangladeshs-slide-towards-authoritarianism-is-accelerating?frsc=dg%7Ce" href="https://www.economist.com/asia/2018/10/06/bangladeshs-slide-towards-authoritarianism-is-accelerating?frsc=dg%7Ce" rel="noopener noreferrer" target="_blank">
    逮捕
   </a>
  </p>
  <p class="graf graf--p">
   重点：政治宣传和媒体管制
  </p>
  <p class="graf graf--p">
   2018年10月，孟加拉政府通过了一项
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dhakatribune.com/bangladesh/2018/10/08/president-signs-digital-security-bill-into-law" href="https://www.dhakatribune.com/bangladesh/2018/10/08/president-signs-digital-security-bill-into-law" rel="noopener noreferrer" target="_blank">
    法案
   </a>
   ，任何人如果散布有关1971年独立战争的 “宣传” 将被监禁。它还禁止发布所谓的 “令人反感和令人恐惧” 的内容。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    请注意措辞
   </strong>
   ：“反感和恐惧” 都是主观判断、是最有伸缩性的措辞，这样的词汇出现在法律条款中时您应该特别警惕，因为它给了执法者以非常广泛的裁定权力。对任何国家来说都是如此。
  </p>
  <p class="graf graf--p">
   经济学人
   <a class="markup--anchor markup--p-anchor" data-href="https://www.economist.com/asia/2018/10/06/bangladeshs-slide-towards-authoritarianism-is-accelerating?frsc=dg%7Ce" href="https://www.economist.com/asia/2018/10/06/bangladeshs-slide-towards-authoritarianism-is-accelerating?frsc=dg%7Ce" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   说，记者对此表示关注。 8月，一名摄影师发言以支持学生抗议活动，之后他因 “传播虚假信息” 而被捕。自由之家
   <a class="markup--anchor markup--p-anchor" data-href="https://freedomhouse.org/report/freedom-net/freedom-net-2018/rise-digital-authoritarianism" href="https://freedomhouse.org/report/freedom-net/freedom-net-2018/rise-digital-authoritarianism" rel="noopener noreferrer" target="_blank">
    称
   </a>
   ，这位摄影师因被指散布针对政府的虚假消息而面临长达
   <strong class="markup--strong markup--p-strong">
    七年
   </strong>
   的监禁，该法律在过去一年中已被用来拘留了
   <strong class="markup--strong markup--p-strong">
    数十名
   </strong>
   社交媒体用户。
  </p>

   <strong class="markup--strong markup--p-strong">
    任何人
   </strong>
   都可以对所谓的虚假信息提供者进行投诉。
  </p>
  <p class="graf graf--p">
   3月中旬，来自 Facebook 和 Google 等社交媒体公司的代表在吉隆坡
   <a class="markup--anchor markup--p-anchor" data-href="https://web.archive.org/web/20181205094120/http://www.straitstimes.com/asia/se-asia/malaysia-meets-social-media-giants-facebook-twitter-and-google-to-discuss-fake-news-law" href="https://web.archive.org/web/20181205094120/http://www.straitstimes.com/asia/se-asia/malaysia-meets-social-media-giants-facebook-twitter-and-google-to-discuss-fake-news-law" rel="noopener noreferrer" target="_blank">
    会见
   </a>
   了立法者，以征求他们对法律的看法。
  </p>
  <p class="graf graf--p">

    逮捕
   </a>

    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bb%89%e4%bb%b7%e9%80%a0%e5%81%87%e5%92%8c%e6%b7%b1%e5%ba%a6%e9%80%a0%e5%81%87%ef%bc%8c%e7%a9%b6%e7%ab%9f%e5%93%aa%e4%b8%aa%e6%9b%b4%e5%8f%af%e6%80%95%ef%bc%9f/" href="https://www.iyouport.org/%e5%bb%89%e4%bb%b7%e9%80%a0%e5%81%87%e5%92%8c%e6%b7%b1%e5%ba%a6%e9%80%a0%e5%81%87%ef%bc%8c%e7%a9%b6%e7%ab%9f%e5%93%aa%e4%b8%aa%e6%9b%b4%e5%8f%af%e6%80%95%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     廉价造假和深度造假，究竟哪个更可怕
    </a>
    ？》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e6%83%85%e6%8a%a5%e5%92%8c%e5%8f%96%e8%af%81%e5%ad%a6%e5%a6%82%e4%bd%95%e5%af%b9%e4%bb%98-%e6%b7%b1%e5%ba%a6%e9%80%a0%e5%81%87%ef%bc%9f%ef%bc%88%e6%9c%89%e5%be%88%e5%a4%9a%e8%a7%86/" href="https://www.iyouport.org/%e5%bc%80%e6%ba%90%e6%83%85%e6%8a%a5%e5%92%8c%e5%8f%96%e8%af%81%e5%ad%a6%e5%a6%82%e4%bd%95%e5%af%b9%e4%bb%98-%e6%b7%b1%e5%ba%a6%e9%80%a0%e5%81%87%ef%bc%9f%ef%bc%88%e6%9c%89%e5%be%88%e5%a4%9a%e8%a7%86/" rel="noopener noreferrer" target="_blank">
     开源情报和取证学如何对付 “深度造假”？（有很多视频）
    </a>
    》
   </li>
   <li class="graf graf--li">
    《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%a3%80%e6%b5%8b%e5%92%8c%e9%98%b2%e6%ad%a2%e6%b7%b1%e5%ba%a6%e9%80%a0%e5%81%87%e7%9a%84%e6%96%b0%e6%96%b9%e6%b3%95/" href="https://www.iyouport.org/%e6%a3%80%e6%b5%8b%e5%92%8c%e9%98%b2%e6%ad%a2%e6%b7%b1%e5%ba%a6%e9%80%a0%e5%81%87%e7%9a%84%e6%96%b0%e6%96%b9%e6%b3%95/" rel="noopener noreferrer" target="_blank">
     检测和防止深度造假的新方法
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   情报机构认为，俄罗斯在2016年美国总统大选期间干预社交媒体，导致联邦政府采取了几项零星行动。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    这篇分析很重要……《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/vpn%ef%bc%9f%e5%85%a5%e4%be%b5%ef%bc%9f%e6%af%94%e7%89%b9%e5%b8%81%ef%bc%9f%e4%bf%84%e7%bd%97%e6%96%af%e6%94%bb%e5%87%bb%e8%80%85%e7%a9%b6%e7%ab%9f%e6%98%af%e6%80%8e%e4%b9%88%e8%a2%ab%e5%8f%91/" href="https://www.iyouport.org/vpn%ef%bc%9f%e5%85%a5%e4%be%b5%ef%bc%9f%e6%af%94%e7%89%b9%e5%b8%81%ef%bc%9f%e4%bf%84%e7%bd%97%e6%96%af%e6%94%bb%e5%87%bb%e8%80%85%e7%a9%b6%e7%ab%9f%e6%98%af%e6%80%8e%e4%b9%88%e8%a2%ab%e5%8f%91/" rel="noopener noreferrer" target="_blank">
     VPN？入侵？比特币？俄罗斯攻击者究竟是怎么被发现的？
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   首先，国会于2017年10月
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theverge.com/2017/10/19/16502946/facebook-twitter-russia-honest-ads-act" href="https://www.theverge.com/2017/10/19/16502946/facebook-twitter-russia-honest-ads-act" rel="noopener noreferrer" target="_blank">
    宣布
   </a>
   了一项法案，该法案将要求 Facebook 和 Google 等在线平台保留广告副本，将其公开并密切关注谁在付款以及付款多少。本质上，该立法试图将现有的电视和广播广告法规强加于社交媒体公司。
  </p>
  <p class="graf graf--p">
   然后，在2017年11月，来自 Facebook、Twitter 和 Google 的代表向参议院司法委员会作证，说明了他们在选举期间传播虚假信息的作用。
  </p>
  <p class="graf graf--p">
   在那次会议期间，人们达成了广泛的共识，即俄罗斯操纵了他们的平台，但是在监控假账户和广告购买者时，这些平台显示出一种可控的外观。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     操纵选民认知的可不止俄罗斯
    </strong>
    …… 见《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" href="https://www.iyouport.org/%e7%9c%8b%e4%b8%8d%e8%a7%81%e7%9a%84%e7%8b%ac%e8%a3%81/" rel="noopener noreferrer" target="_blank">
     看不见的独裁
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   同时，加利福尼亚州政府于2018年9月通过了一项法律，以加强公立学校的媒体素养。它要求教育部列出有关如何评估可信赖媒体的教学材料和资源。
  </p>
  <p class="graf graf--p">
   这项法律的灵感来自斯坦福大学的一名学生，他发现大多数学生无法区分赞助内容和新闻报道，并且在至少24个州
   <a class="markup--anchor markup--p-anchor" data-href="https://medialiteracynow.org/your-state-legislation/washington-d-c/" href="https://medialiteracynow.org/your-state-legislation/washington-d-c/" rel="noopener noreferrer" target="_blank">
    进行
   </a>
   了许多尝试来提高媒体素养。
  </p>
  <p class="graf graf--p">
   其中一个州是华盛顿，立法者正在
   <a class="markup--anchor markup--p-anchor" data-href="https://medialiteracynow.org/your-state-legislation/washington-legislation/" href="https://medialiteracynow.org/your-state-legislation/washington-legislation/" rel="noopener noreferrer" target="_blank">
    辩论
   </a>
   一项媒体素养法案，该法案将为致力于将媒体素养纳入学校课程的组织建立一项赠款计划。2018年，马萨诸塞州立法者
   <a class="markup--anchor markup--p-anchor" data-href="https://medialiteracynow.org/mass-lawmakers-send-civic-education-mandate-with-emphasis-on-media-literacy-to-the-governor/" href="https://medialiteracynow.org/mass-lawmakers-send-civic-education-mandate-with-emphasis-on-media-literacy-to-the-governor/" rel="noopener noreferrer" target="_blank">
    通过
   </a>
   了一项法案，该法案要求进行公民教育，重点是媒体素养。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    <strong class="markup--strong markup--li-strong">
     关于媒体素养 — 也就是如何判断真伪 — 见

     <strong class="markup--strong markup--li-strong">
      列表-3 “斗智”
     </strong>
    </a>
   </li>

  <p class="graf graf--p">
   同样在加利福尼亚州，州长 Jerry Brown 否决了一项法案，该法案将创建一个咨询小组，旨在监视社交媒体上错误信息的传播并提出潜在的解决方案。Brown 称该小组 “不必要”。
  </p>
  <p class="graf graf--p">
   2018年9月中旬，两名民主党人和一名共和党代表致信国家情报局局长，要求情报界评估深度造假技术可能造成的国家安全威胁，并于2018年底向国会提交报告。
  </p>
  <p class="graf graf--p">
   据美国有线电视新闻网
   <a class="markup--anchor markup--p-anchor" data-href="https://www.cnn.com/2019/01/30/tech/new-york-attorney-general-social-media/" href="https://www.cnn.com/2019/01/30/tech/new-york-attorney-general-social-media/" rel="noopener noreferrer" target="_blank">
    报道
   </a>
   ，2019年1月，一家创建虚假社交媒体资料以赚取数百万美元收入的公司与纽约州律师达成和解。该和解是执法机构得出的第一个结论，即 销售造假的社交媒体内容制作技术是非法的。
  </p>

    越南 🇻🇳

   重点：虚假信息
  </p>
  <p class="graf graf--p">
   2019年1月，越南新法律生效，⚠️ 该法律要求互联网服务提供商披露用户数据，以便政府可以追踪特定帖子的来源。
