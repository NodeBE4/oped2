  <p class="graf graf--p">
   中国西北部的新疆正处于监视封锁状态。现在有一百万民族维吾尔人在集中营里。一举一动都在受到监视。
  </p>
  <p class="graf graf--p">
   但在距离3000英里外的伊斯坦布尔，一个维吾尔族妇女社区正在通过智能手机进行反击。
  </p>
  <h3 class="graf graf--p">
   <span style="font-family: impact, sans-serif;">
    <strong class="markup--strong markup--p-strong">
     住在信息真空之外
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   Nurjamal Atawula 记得她第一次拿到智能手机的样子。
  </p>
  <p class="graf graf--p">
   那是2011年，她住在中国西北部的新疆绿洲小镇和田。
  </p>
  <p class="graf graf--p">
   她喜欢为自己的孩子拍照，并在外出时与丈夫使用表情符号沟通。
  </p>
  <p class="graf graf--p">
   2013年，Atawula 下载了中国社交消息应用程序微信。
  </p>
  <p class="graf graf--p">
   不久之后，传闻在她的朋友网络中出现：政府可以通过手机跟踪你的位置。
  </p>
  <p class="graf graf--p">
   起初，她不相信他们。
  </p>
  <p class="graf graf--p">
   2016年初，警方开始对 Atawula 的家进行了例行检查。她的丈夫经常被叫到警察局。
  </p>
  <p class="graf graf--p">
   看见警察，Atawula 的孩子们开始害怕畏缩。
  </p>
  <p class="graf graf--p">
   这家人决定搬到土耳其以逃避压迫性的监视。
  </p>
  <p class="graf graf--p">
   Atawula 的丈夫决定在他住在新疆期间送她前往土耳其并等待孩子们的护照。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “我离开的那天，我的丈夫就被捕了，” Atawula 说。
  </p>
  <p class="graf graf--p">
   当她于2016年6月抵达土耳其时，她的手机就罢工了 —— 等她修好手机时发现，所有的朋友和亲戚都将她的微信账号删除了。
  </p>
  <p class="graf graf--p">
   她独自一人在伊斯坦布尔，她与新疆生活的数字联系已经结束。
  </p>
  <p class="graf graf--p">
   除了在2016年12月底与她的母亲打了11分半钟的 Skype 电话之外，与所有亲戚的沟通已经完全切断。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “有时我觉得我与家人在一起的日子只是我的梦想，我一生中一直是孤独的 —— 从我出生以来就是这样，”她说。
  </p>
  <p class="graf graf--p">
   30岁的 Atawula 现在独自生活在伊斯坦布尔的工人阶级社区 Zeytinburnu。
  </p>
  <p class="graf graf--p">
   这里是土耳其最大的维吾尔族人口的家园，维吾尔族是新疆的穆斯林少数民族，是中国古代丝绸之路贸易路线上广阔、资源丰富的沙漠和山脉之地。
  </p>
  <p class="graf graf--p">
   Atawula 是居住在土耳其的约34,000名维吾尔人之一。她无法联系任何亲戚 —— 不论是通过电话、微信、还是任何其他应用程序。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当我看到其他人与家人进行视频聊天时，我感到非常难过，”她说，“我想，为什么我们甚至不能听到自己孩子的声音？”
  </p>
  <p class="graf graf--p">
   对于新疆的维吾尔人来说，任何来自外国电话号码的联系，虽然没有规定非法，但是都会被视为可疑的极端主义行为，可能意味着即时逮捕。
  </p>
  <p class="graf graf--p">
   居住在土耳其的大多数维吾尔族人已被家人在社交媒体上删除。
  </p>
  <p class="graf graf--p">
   许多人不敢尝试联系，因为担心中国当局会惩罚他们的亲属。
  </p>
  <p class="graf graf--p">
   这只是习近平政府对中国维吾尔人进行严密控制的监控方式之一，它对生活在世界各地的维吾尔人产生了连锁反应。
  </p>
  <p class="graf graf--p">
   Aywula 居住的伊斯坦布尔郊区 Zeytinburnu 位于城市蜿蜒的高速公路后面，点缀着供应维吾尔美食的餐馆和咖啡馆：滑溜溜的面条、烤羊肉串和绿茶。
  </p>
  <p class="graf graf--p">
   维吾尔分离主义旗帜 — 土耳其国旗的浅蓝色版本 — 在这里是常见的景象。这是被中国禁止的，代表东突厥斯坦。
  </p>
  <p class="graf graf--p">
   新疆 — 中文意为“新边疆” — 于1949年被中国共产党控制。
  </p>

  <p class="graf graf--p">
   在二十世纪下半叶，维吾尔独立是一个笼罩在中国党议程上的威胁。
  </p>
  <p class="graf graf--p">
   政府试图通过鼓励中国占主导地位的汉族人群大规模迁移到新疆来消灭分裂主义并“同化”维吾尔人。
  </p>
  <p class="graf graf--p">
   九十年代，维吾尔族和中国警方之间爆发了冲突。在3月份发表的一份
   <a class="markup--anchor markup--p-anchor" data-href="http://global.chinadaily.com.cn/a/201903/19/WS5c9033f0a3106c65c34ef4ae.html" href="http://global.chinadaily.com.cn/a/201903/19/WS5c9033f0a3106c65c34ef4ae.html" rel="noopener noreferrer" target="_blank">
    白皮书
   </a>
   中，中国政府将那场冲突定义为分裂主义团体实施的“不人道、反社会和野蛮行为”。
  </p>
  <p class="graf graf--p">
   与此同时，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.amnesty.org/download/Documents/64000/asa170022007en.pdf" href="https://www.amnesty.org/download/Documents/64000/asa170022007en.pdf" rel="noopener noreferrer" target="_blank">
    国际特赦组织描述了
   </a>
   1997年在新疆古尔亚（伊犁）举行的抗议活动，这是一次和平示威最终变成了大屠杀，引用的是流亡的维吾尔活动家 Rebiya Kadeer。
  </p>
  <p class="graf graf--p">
   9/11之后，中国政府似乎通过模仿乔治布什的反恐战争来针对新疆的分裂主义团体。
  </p>
  <p class="graf graf--p">
   2009年，新疆首都乌鲁木齐的维吾尔族和汉族之间爆发了血腥的民族冲突。它被称为“中国的9/11”。
  </p>
  <p class="graf graf--p">
   警方将该市置于封锁状态，实施了互联网中断，并削减了手机服务。这是控制维吾尔族人口的新政策的开端 —— 数字化控制的开端。
  </p>

  <p class="graf graf--p">
   近年来，中国通过智能手机对“伊斯兰极端主义”进行了打击。
  </p>
  <p class="graf graf--p">
   2011年，中国IT巨头腾讯推出了一款名为 WeChat 的新应用程序 —— 在维吾尔语中被称为“Undidar”。它很快成为中国重要的沟通工具。
  </p>
  <p class="graf graf--p">
   微信的发起是“一个巨大的救济和自由的时刻”，维吾尔族学者 Aziz Isa 说，他曾在伦敦 SOAS 大学
   <a class="markup--anchor markup--p-anchor" data-href="http://www.soundislamchina.org/?page_id=1277" href="http://www.soundislamchina.org/?page_id=1277" rel="noopener noreferrer" target="_blank">
    研究维吾尔人使用微信的情况
   </a>
   。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “在维吾尔的生活中，我们从未有过以这种方式使用社交媒体的机会”，Isa 说，描述了维吾尔族人如何跨越阶级分歧公开讨论从政治到宗教到音乐的一切。
  </p>
  <p class="graf graf--p">
   到2013年，大约有一百万维吾尔人正在使用该应用程序。Harris 和 Isa 观察到伊斯兰教内容的稳步上升，“其中大多数是非政治性的，但其中一些是公开激进的和政治反对的。”
  </p>
  <p class="graf graf--p">
   Isa 记得他所看到了一些更民族主义的内容感到担忧，尽管他认为这样的内容占所有帖子的不到百分之一。他说，大多数维吾尔人当时“并不理解当局正在偷窥。”
  </p>
  <p class="graf graf--p">
   这种在微信上不受限制的沟通仅仅持续了大约一年。
  </p>
  <p class="graf graf--p">
   但是在2014年5月，中国政府邀请了一个工作组来消除即时通讯应用程序中的所谓“弊端”，特别是“导致暴力、恐怖主义和色情的谣言和信息”。
  </p>
  <p class="graf graf--p">
   微信与其竞争对手的应用程序一起被要求允许政府监控其用户的活动。

    <strong class="markup--strong markup--p-strong">
     中国数字古拉格的增长
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   2018年8月，一个联合国人权小组表示，相信有100万维吾尔人被关押在一个“保密的大规模拘禁营”中。
  </p>
  <p class="graf graf--p">
   起初，中国完全否认了拘留营的存在。但随后，在2018年10月，
   <a class="markup--anchor markup--p-anchor" data-href="http://www.xinhuanet.com/english/2018-10/16/c_137535821.htm" href="http://www.xinhuanet.com/english/2018-10/16/c_137535821.htm" rel="noopener noreferrer" target="_blank">
    政府宣布
   </a>
   启动了“职业教育和培训计划”，并通过了一项法律，将他们所谓的“培训中心”合法化。
  </p>
  <p class="graf graf--p">
   在2018年9月的一份报告中，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hrw.org/sites/default/files/report_pdf/china0918_web.pdf" href="https://www.hrw.org/sites/default/files/report_pdf/china0918_web.pdf" rel="noopener noreferrer" target="_blank">
    人权观察发现
   </a>
   新疆的侵犯人权行为具有文化大革命以来未见的范围和规模，营地的建立反映了北京要“以自己的形象改造新疆”的承诺。
  </p>
  <p class="graf graf--p">
   54岁的 Gulbahar Jalilova 是来自哈萨克斯坦的维吾尔族服装零售商，在乌鲁木齐的拘留营度过了一年零三个月。她现在住在伊斯坦布尔。
  </p>
  <p class="graf graf--p">
   根据乌鲁木齐市公安局在中国发出的逮捕令，她因涉嫌参与该地区的恐怖活动而被拘留。
  </p>
  <p class="graf graf--p">
   警察指控她通过她在乌鲁木齐的一名雇员洗钱，后者也被捕了。
  </p>
  <p class="graf graf--p">
   Jalilova 被带到了新疆首都众多临时拘留中心之一的 kanshousuo。在接下来的15个月里，她被转移到乌鲁木齐的三个不同的监狱和营地。
  </p>
  <p class="graf graf--p">
   她对拘留期间的生活印象深刻：一个10×20英尺的牢房，最多的时候容纳50人紧紧地坐在一起，完全伸不开腿。
  </p>
  <p class="graf graf--p">
   自2018年8月获释以来，Jalilova 一直在努力回忆，她用笔记本记录了与她一起被记录在案的所有女性的名字。
  </p>
  <p class="graf graf--p">
   她还记录了
   <strong class="markup--strong markup--p-strong">
    被捕的原因，其中包括下载 WhatsApp
   </strong>
   —— 一个在中国被封锁的应用程序 —— 存储了多位著名维吾尔族学者的作品，并在手机上被查到存在宗教内容。
  </p>
  <p class="graf graf--p">
   她记得牢房里四面都是摄像头，电视安装在门上方。 “北京的领导人可以看见你，”警卫告诉她。
  </p>
  <p class="graf graf--p">
   每月一次，Jalilova 描述卫兵如何将习近平的讲话发给囚犯，让他们写下检讨书。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “如果你写了一些不好的东西，他们就会惩罚你，”Jalilova 说， “你只能写’感谢党，我已经清除了这个或那个’，’一旦我被释放，我一定洗心革面重新做人……”。
  </p>
  <p class="graf graf--p">
   她于2018年8月获释并来到土耳其，在哈萨克斯坦不再安全，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.rfa.org/english/news/uyghur/deportation-06022011174917.html" href="https://www.rfa.org/english/news/uyghur/deportation-06022011174917.html" rel="noopener noreferrer" target="_blank">
    政府会将维吾尔人驱逐回新疆
   </a>
   。
  </p>

  <p class="graf graf--p">
   虽然没有关于拘留营的官方统计数据，但志愿者组织的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.shahit.biz/eng/#view" href="https://www.shahit.biz/eng/#view" rel="noopener noreferrer" target="_blank">

   </a>

  </p>
  <p class="graf graf--p">
   因此，近年来逃离新疆到土耳其的大多数人是女性。当地活动人士估计，土耳其65％的维吾尔族人口是女性，许多人与丈夫分开。
  </p>
  <p class="graf graf--p">
   在 Zeytinburnu，他们住在一个共享公寓网络中，通过在当地纺织业无证工作，赚取生活费。
  </p>
  <p class="graf graf--p">
   没有丈夫的妇女们在其他维吾尔人中被称为“寡妇”。他们的丈夫被困在新疆，她们不知道丈夫是活着、被监禁了还是已经死了。
  </p>
  <p class="graf graf--p">
   35岁的 Kalbinur Tursun 于2016年4月带着她的小儿子穆罕默德离开新疆，这是她当时唯一一个拿到了护照的孩子。她把她的其他孩子和丈夫留在了新疆。
  </p>
  <p class="graf graf--p">
   她怀上了第七个孩子，一个名叫 Marziya 的女儿，她担心她会被迫中止妊娠，因为她已经生育了比中国限制性计划生育政策标准更多的孩子。
  </p>
  <p class="graf graf--p">
   当 Tursun 第一次到达土耳其时，她每天都使用 WeChat 和她的丈夫视频通话。 Tursun 相信中国警察于2016年6月13日逮捕了她丈夫 —— 因为这是她最后一次与他交谈。
  </p>
  <p class="graf graf--p">
   然后，一位朋友告诉她，由于她决定离开中国，她的丈夫被判入狱十年。 “我很害怕我的孩子们恨我，”她说。

  <p class="graf graf--p">
   土耳其被视为是比其他穆斯林占多数的国家更安全的地方，包括巴基斯坦和沙特阿拉伯，其领导人最近都驳回了维吾尔人的避难申请。
  </p>
  <p class="graf graf--p">
   自20世纪50年代以来，维吾尔人一直从中国流入土耳其。他们没有获得工作许可证，许多人希望自己最终能够在欧洲或美国找到避难所。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    土耳其传统上一直担任维吾尔人的保护者，他们认为维吾尔族是突厥族，近年来，但是，由于与中国的贸易关系有所改善，总统雷杰普·塔伊普·埃尔多安一直不愿为维吾尔人说话。
   </strong>
  </p>
  <p class="graf graf--p">
   在2019年2月9日，
   <a class="markup--anchor markup--p-anchor" data-href="http://www.mfa.gov.tr/sc_-06_-uygur-turklerine-yonelik-agir-insan-haklari-ihlalleri-ve-abdurrehim-heyit-in-vefati-hk.en.mfa" href="http://www.mfa.gov.tr/sc_-06_-uygur-turklerine-yonelik-agir-insan-haklari-ihlalleri-ve-abdurrehim-heyit-in-vefati-hk.en.mfa" rel="noopener noreferrer" target="_blank">

  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这不再是一个秘密，超过一百万维吾尔族人遭到任意逮捕，在拘留营和监狱遭受酷刑和政治洗脑，”Aksoy 的声明中写道。
  </p>
  <p class="graf graf--p">
   两天后，中国外交部发言人华春英
   <a class="markup--anchor markup--p-anchor" data-href="https://www.fmprc.gov.cn/mfa_eng/xwfw_665399/s2510_665401/t1637013.shtml" href="https://www.fmprc.gov.cn/mfa_eng/xwfw_665399/s2510_665401/t1637013.shtml" rel="noopener noreferrer" target="_blank">

  </p>

  <p class="graf graf--p graf--startsWithDoubleQuote">

  <p class="graf graf--p">
   3月，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.reuters.com/article/us-china-xinjiang-exclusive/exclusive-china-to-invite-european-diplomats-to-xinjiang-in-new-diplomatic-push-idUSKCN1R10QQ" href="https://www.reuters.com/article/us-china-xinjiang-exclusive/exclusive-china-to-invite-european-diplomats-to-xinjiang-in-new-diplomatic-push-idUSKCN1R10QQ" rel="noopener noreferrer" target="_blank">
    路透社报道
   </a>
   中国将邀请欧洲外交官访问该地区。此前，新疆省长 Shohrat Zakir 发表声明称那个集中营是“寄宿学校”。
  </p>
  <p class="graf graf--p">
   今年春天，Harri 开始制作 #MeTooUyghur 的标签，鼓励世界各地的维吾尔人要求证明他们的家人还活着。
  </p>
  <p class="graf graf--p">
   来自国际维吾尔族侨民的大型 WhatsApp 新闻组也是社区团结的重要来源。
  </p>
  <p class="graf graf--p">
   2018年12月24日，Kalbinur Tursun 坐在她在 Zeytinburnu 管理的女装店，滚动着一个维吾尔 WhatsApp 群组。她是一位母亲，她的5个孩子被留在了新疆。
  </p>
  <p class="graf graf--p">
   她从早到晚不断查看这个 WhatsApp 群组。因为几百名维吾尔族成员发布了近乎不间断的视频和有关新疆危机的最新消息。
  </p>
  <p class="graf graf--p">
   她看到了一个孩子们做游戏的视频，Tursun 很惊讶。在图像左侧，她认出了自己六岁的女儿 Aisha。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “这就像一个奇迹”，她说， “我只会在梦中看到我的孩子，我从没想过会在现实生活中看到她。” 自从她上次听到女儿的声音已经过去两年了。
  </p>
  <p class="graf graf--p">
   该视频似乎来自和田省所谓的“小天使学校”之一，距离 Tursun 居住的喀什大约300英里，据报道
   <a class="markup--anchor markup--p-anchor" data-href="https://www.rfa.org/english/news/uyghur/separated-09132018132251.html" href="https://www.rfa.org/english/news/uyghur/separated-09132018132251.html" rel="noopener noreferrer" target="_blank">
    有近3000名维吾尔族儿童在这里被关押
   </a>
   。
  </p>
  <p class="graf graf--p">
   Tursun 想知道她的另外四个孩子是否可能被带到了更远的地方。该省共产党官员在接受自由亚洲电台采访时表示，孤儿院由警方巡逻，以“提供安全保障”。
  </p>
  <p class="graf graf--p">
   与全球维吾尔族侨民中的几乎所有人都不同，Nurjamal Atawula 在微信联系方式中断后设法找到了与家人联系的方法。她使用了最古老的方法之一：写信。
  </p>
  <p class="graf graf--p">
   2016年底，她听说 Zeytinburnu 的一位女士经常往返于土耳其和她留在新疆的父母居住的村子。她让那个女人给她的家人带一封信。女人同意了。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote">
   “当我写这封信时，我觉得自己生活在黑暗时代，” Atawula 说。她把信给了那个女人，同时还有她给孩子的小礼物和她为家人攒的钱。
  </p>
  <p class="graf graf--p">
   一个月后，她收到了答复。这位维吾尔族妇女，她称之为“姐姐”，将她哥哥的一封信偷偷带出了中国，藏在一包纸巾里。
  </p>
  <p class="graf graf--p">
   但是在第三次旅行捎信之后，那个女人就消失了。
  </p>
  <p class="graf graf--p">
   Atawula 不知道发生了什么事。她仍然继续写信给她的家人，但她的信件现在被保存在日记中，希望有一天她的孩子能够读这些信。
  </p>
  <p class="graf graf--p">
   自 Atawula 收到她哥哥的回信以来已经过去两年多了。她把那封信小心翼翼地折叠起来，仍然放在那堆纸巾里。在那段时间里，她只读过三次，好像读多了就会让那封信失去力量一样。
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    我漂亮的妹妹：
   </em>
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    你好吗？你离开乌鲁木齐后我们无法联系你，当我们收到你的来信时，我们非常高兴。我有一肚子的话……也许在我们团聚之后，我们可以互相说出来。你说你想念孩子们。愿真主给你耐心。妈妈、我和亲戚们都非常想念你。我们对你有非常多的希望。请坚强，不要担心孩子。
   </em>
  </p>
