  </p>
  <p class="graf graf--p">

   <strong class="markup--strong markup--p-strong">
    21世纪将是我们超越传统非暴力行动并完善其使用的时代。
   </strong>
  </p>
  <p class="graf graf--p">
   但是，可以肯定的是目前为止还啥都没实现。从许多角度来看，这个世界似乎已根深蒂固地陷入了反乌托邦。

   <strong class="markup--strong markup--p-strong">
    抵抗运动 —— 尤其是在越来越多狂暴于镇压的政权中发起的抵抗运动 —— 不仅是必要的，而且，必须依靠创造力和创新才能真正存在。
   </strong>
  </p>
  <p class="graf graf--p">
   实际上，我们可能还没能完善使用非暴力行动来增强公民的力量，但是我们已经超越了 Gene Sharp 的198种方法。
  </p>
  <p class="graf graf--p">
   最近一项新研究指出了300多种非暴力抵抗方法，代表了很多创新，特别是在技术和数字领域。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">

   </strong>
  </p>
  <p class="graf graf--p">

  </p>
  <p class="graf graf--p">

  </p>
