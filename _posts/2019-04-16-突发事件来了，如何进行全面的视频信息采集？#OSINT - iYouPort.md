              <p class="graf graf--p graf-after--figure" id="87f2">
               开源调查经常需要对特定事件进行深入研究，其中包括寻找有助于进行分析的所有可能的照片、视频和见证账户。尤其是突发事件出现时，很多用户会迅速上传照片和视频，并且，这种时候也是那些营销者推动假消息混淆其中的重点时段。
              </p>
              <p class="graf graf--p graf-after--p" id="070e">
               我们介绍过很多验证图像和视频信息真伪的工具。验证非常重要，但首先您必须找到它们在哪儿。用户通过不同渠道/平台传播信息，如何快速将相关事件的所有信息集中起来？
              </p>
              <p class="graf graf--p graf-after--p" id="db0e">
               本指南将提供有关如何在特定事件中收集尽可能多的视频的建议和一些提示，无论是来自自然灾害、群体事件、人为灾难、还是恐怖袭击的目击者。对于下面将呈现的大多数示例，将
               <strong class="markup--strong markup--p-strong">
                假设该事件是大型抗议或群体事件
               </strong>
               ，但同样的适用于其他事件。
              </p>
              <p class="graf graf--p graf-after--p" id="c6f2">
               下面开始。
              </p>
              <p class="graf graf--p graf-after--p" id="bda8">
               <strong class="markup--strong markup--p-strong">
                第一组搜索：直播
               </strong>
              </p>
              <p class="graf graf--p graf-after--p" id="5822">
               正如您所想象的那样，第一个显示事件的视频是实时发生的 — 即 直播。对于大型群体事件，会有大量参与者或媒体成员在事件发生时进行直播。
              </p>
              <p class="graf graf--p graf-after--p" id="d4f5">
               对此，以下是最流行的直播平台：
               <a class="markup--anchor markup--p-anchor" data-href="https://www.facebook.com/live/" href="https://www.facebook.com/live/" rel="noopener nofollow noreferrer" target="_blank">
                Facebook Live
               </a>
              </p>
              <p class="graf graf--p graf-after--p" id="79f3">
               作为此列表中最新的平台之一，Facebook Live 因其庞大的用户群和易用性而迅速普及。查找实时视频非常简单 — 访问
               <a class="markup--anchor markup--p-anchor" data-href="https://www.facebook.com/live" href="https://www.facebook.com/live" rel="noreferrer noopener nofollow" target="_blank">
                https://www.facebook.com/live
               </a>
               ，然后单击您感兴趣的位置中的蓝点。但是，这些视频仅在流式传输时才显示在地图上。
              </p>

              <p class="graf graf--p graf-after--figure" id="c1c8">
               要从 Facebook Live 中查找以前播放的视频，您有几个选择。例如，您可以访问您知道的播放流的个人或主页，并搜索他们的视频。例如，可以在
               <a class="markup--anchor markup--p-anchor" data-href="http://facebook.com/RebZtv/videos" href="https://facebook.com/RebZtv/videos" rel="noreferrer noopener nofollow" target="_blank">
                facebook.com/RebZtv/videos

              </p>
              <p class="graf graf--p graf-after--p" id="90b0">
               或者，您可以通过 Facebook 搜索功能广泛搜索实时视频。不幸的是，Facebook 的搜索功能不是很容易使用，但你可以通过仅过滤“直播”视频找到大多数结果。
              </p>

              <p class="graf graf--p graf-after--p" id="1998">
               虽然 YouTube Live 是一个更大的视频平台，但对于普通用户来说，YouTube Live 还不是一种非常受欢迎的直播服务，尽管它经常用于专业新闻媒体。

              <p class="graf graf--p graf-after--figure" id="08d0">
               <strong class="markup--strong markup--p-strong">
                Periscope
               </strong>
              </p>
              <p class="graf graf--p graf-after--p" id="c444">
               最受欢迎的直播平台之一是 Periscope，它于 2015 年被 Twitter 收购。与本指南中的其他平台一样，很难通过 Periscope 网站的界面搜索视频 — 事实上，几乎不值得使用 Periscope 网站上的搜索界面，通过 Google 或 Twitter 搜索要容易得多。例如，下面我们可以搜索 2017 年8月12日夏洛茨维尔活动当天的部分（但不是全部）Periscope 直播，关键字为“Charlottesville”。
              </p>

                但这不会给你全面的结果。几乎所有 Periscope 流都至少被转推过一次，无论是由他们的创建者转的（因为 Twitter 帐户经常链接到 Periscope 流）或有人在观看。
               </p>
               <p class="graf graf--p graf-after--p" id="572d">
                在 Twitter 中搜索类似的字符串与 Google 一起使用可能更有用，但您大概会有更多结果需要筛选。“ 最新 ”和“ 视频 ”选项将带来许多好的结果 — 通常人们会链接或提及某些流 — 但最近添加的“ 广播”“选项将给出最直接的结果。
               </p>

                从您正在寻找的事件中找到一个视频后，搜索 Periscope 用户的其他视频 — 您可能会在同一个地点和时间找到更多的视频流。

                 </strong>
                </a>
               </p>
               <p class="graf graf--p graf-after--p" id="1afe">
                <a class="markup--anchor markup--p-anchor" data-href="http://livestream.com/" href="https://livestream.com/" rel="noreferrer noopener nofollow" target="_blank">
                 Livestream.com
                </a>
                是此列表中最古老的服务之一，自社交媒体平台提供直播选项以来，其受欢迎程度逐渐降低，但它仍被许多团体和个人使用。
               </p>
               <p class="graf graf--p graf-after--p" id="0457">
                值得庆幸的是，它的搜索功能比 Periscope 和其他一些平台
                <strong class="markup--strong markup--p-strong">
                 更强大
                </strong>
                ; 然而，它们仍然不完美。搜索特定日期发布的视频很困难，搜索过滤器更接近 YouTube，“过去24小时”，“过去一周”等过滤器，而不是特定的日期范围。尽管如此，这个平台上的视频远远少于 YouTube 和 Periscope，从而减少了对有用和无关结果进行分类的麻烦。
               </p>

                   <img alt="" class="aligncenter wp-image-591 size-full" data-recalc-dims="1" height="746" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-77.jpeg?resize=1100%2C746&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-77.jpeg?w=1484&amp;ssl=1 1484w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-77.jpeg?resize=300%2C204&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-77.jpeg?resize=768%2C521&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-77.jpeg?resize=1024%2C695&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-77.jpeg?resize=1100%2C746&amp;ssl=1 1100w" width="1100"/>
                  </noscript>
                 </div>
                </div>
               </div>
              </div>
              <div>
              </div>
              <div>
               <p class="graf graf--p graf-after--figure" id="f071">
                除非您关注的事件发生在拥有多个 VK 和 OK 用户的位置，否则在这些平台上搜索新的视频片段可能不值得，因为这些网站上的大量剪辑都是来自其他来源的共享。
               </p>
               <p class="graf graf--p graf-after--p" id="2b57">
                <strong class="markup--strong markup--p-strong">
                 第三组搜索：难以找到 Snapchat 和 Instagram 内容
                </strong>
               </p>
               <p class="graf graf--p graf-after--p" id="c478">
                Snapchat 和 Instagram 是视频的两个非常常见的来源，但也是两个最难捕捉剪辑的平台。它们都没有直观的搜索界面可以让研究人员轻松地对视频进行分类和收集。
               </p>
               <p class="graf graf--p graf-after--p" id="d012">
                <strong class="markup--strong markup--p-strong">
                 Instagram
                </strong>
               </p>
               <p class="graf graf--p graf-after--p" id="48d1">
                在 Instagram 网站上搜索视频片段几乎没用，除非您很幸运或通过地理标记搜索到特定位置。只搜索关键字或主题标签，就无法仅对视频结果进行排序，也无法在特定日期范围或视频长度标准内进行搜索。
               </p>
               <img alt="" class="aligncenter wp-image-592 size-full jetpack-lazy-image" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-78.jpeg?resize=1100%2C1032&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" height="1032" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-78.jpeg?resize=1100%2C1032&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
               <noscript>
                <img alt="" class="aligncenter wp-image-592 size-full" data-recalc-dims="1" height="1032" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-78.jpeg?resize=1100%2C1032&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-78.jpeg?w=1590&amp;ssl=1 1590w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-78.jpeg?resize=300%2C282&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-78.jpeg?resize=768%2C721&amp;ssl=1 768w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-78.jpeg?resize=1024%2C961&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-78.jpeg?resize=1100%2C1032&amp;ssl=1 1100w" width="1100"/>
               </noscript>
               <div class="section-inner sectionLayout--insetColumn">
                <p class="graf graf--p graf-after--figure" id="689b">
                 您可以使用特定关键字或主题标签对最新视频进行排序，但对于城市名称等热门字词来说，这将为您带来远远超出您所需要的结果。通过地理标记搜索可能很有用，但会带来散射的结果 — 并非所有视频都有标签，甚至使用的不是您正在搜索的标签。例如，弗吉尼亚州夏洛茨维尔的主题标签会给你一些结果，而某个公园或街道名称的标签会给你不同的结果。除非您使用像 EchoSec 这样的付费服务，否则无法将结果集中在重点部分上。
                </p>
               </div>
               <div class="section-inner sectionLayout--outsetColumn">
                <div class="aspectRatioPlaceholder is-locked">
                 <img alt="" class="aligncenter wp-image-593 size-full jetpack-lazy-image" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-79.jpeg?resize=1100%2C991&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" height="991" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-79.jpeg?resize=1100%2C991&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
                 <noscript>
                  <img alt="" class="aligncenter wp-image-593 size-full" data-recalc-dims="1" height="991" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-79.jpeg?resize=1100%2C991&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-79.jpeg?w=1600&amp;ssl=1 1600w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-79.jpeg?resize=300%2C270&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-79.jpeg?resize=768%2C692&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-79.jpeg?resize=1024%2C922&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-79.jpeg?resize=1100%2C991&amp;ssl=1 1100w" width="1100"/>
                 </noscript>
                </div>
               </div>
              </div>
              <div>
              </div>
              <div>
               再一次，Google 通常比较好用。通过使用
               <strong class="markup--strong markup--p-strong">
                搜索运算符
               </strong>
               ，我们可以找到特定关键字和日期范围的视频结果，但结果非常有限。
              </div>
              <div>
              </div>
              <div>
               <img alt="" class="aligncenter wp-image-594 size-full jetpack-lazy-image" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-80.jpeg?resize=1100%2C1059&amp;is-pending-load=1#038;ssl=1" data-recalc-dims="1" height="1059" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-80.jpeg?resize=1100%2C1059&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1100"/>
               <noscript>
                <img alt="" class="aligncenter wp-image-594 size-full" data-recalc-dims="1" height="1059" sizes="(max-width: 1100px) 100vw, 1100px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-80.jpeg?resize=1100%2C1059&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-80.jpeg?w=1282&amp;ssl=1 1282w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-80.jpeg?resize=300%2C289&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-80.jpeg?resize=768%2C739&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-80.jpeg?resize=1024%2C986&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2019/04/0-80.jpeg?resize=1100%2C1059&amp;ssl=1 1100w" width="1100"/>
               </noscript>
              </div>
              <div>
              </div>
              <div>
               <p class="graf graf--p graf-after--figure" id="16e0">
                毫无疑问，抗议活动当天在 Instagram 上张贴了“夏洛茨维尔”这个词的视频超过五个，但这些是谷歌给出的唯一结果。不幸的是，在 Instagram 上搜索视频是一项艰巨的任务，你可能有更好的运气在 Twitter 等其他平台上搜索到 Instagram 视频，而不是在 Instagram 平台本身。
               </p>
               <p class="graf graf--p graf-after--p" id="08f3">
                <strong class="markup--strong markup--p-strong">
                 snapchat
                </strong>
               </p>
               <p class="graf graf--p graf-after--p" id="c485">
                除非您使用 Android 模拟器（即使您这样做成功也很有限），目前无法在笔记本电脑或台式电脑上搜索 Snapchat 视频 — 您只能在平板电脑或手机上使用 Snapchat 应用。在 Snapchat 应用程序中，只需搜索关键字或位置即可查找 Snapchat 故事。但与 Instagram 和 Facebook 一样，此内容可能受到用户隐私设置的限制。
               </p>
               <p class="graf graf--p graf-after--p" id="513b">
                <strong class="markup--strong markup--p-strong">
                 结论
                </strong>
               </p>
               <p class="graf graf--p graf-after--p" id="f59a">
                对于记者和研究人员来说，查找和验证视频的能力越来越重要。虽然最具爆炸性和有趣的视频通常会因各种社交媒体平台上的转发和分享而成为焦点，但通常您可以
                <strong class="markup--strong markup--p-strong">
                 在更加模糊的平台上找到不为人知的视频中被忽视的细节
                </strong>
                。它们也许很重要。
               </p>
               <p class="graf graf--p graf-after--p" id="7a55">
                如果您有更多好的想法，请留言交流。
               </p>
