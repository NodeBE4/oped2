  <h3 class="graf graf--h3 graf-after--p" id="9b05">
   — — 地图，卫星图像和街景 — —
  </h3>
  <blockquote class="graf graf--blockquote graf-after--h3" id="c827">
   <p>
    <em class="markup--em markup--blockquote-em">
     针对 人为或自然灾难、战争、救援、大型群体事件（尤其是造成冲突的时候）、杀戮、种族清洗运动、边境动态……等等状况的调查，拿到证据以便追究责任，这些工具非常有用。
    </em>
   </p>
  </blockquote>
  <p class="graf graf--p graf-after--blockquote" id="82df">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bing.com/maps/" href="https://www.bing.com/maps/" rel="noopener nofollow noreferrer" target="_blank">
    Bing Maps
   </a>
   ：Microsoft 卫星和地图服务。拥有比谷歌更新更快分辨率更高的图像；缺点是检查图像的日期比较难；
  </p>
  <p class="graf graf--p graf-after--p" id="477f">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.synnatschke.de/geo-tools/coordinate-converter.php" href="http://www.synnatschke.de/geo-tools/coordinate-converter.php" rel="noopener nofollow noreferrer" target="_blank">
    converting coordinates
   </a>
   ：这是一个转换地理坐标的工具；
  </p>
  <p class="graf graf--p graf-after--p" id="845a">
   <a class="markup--anchor markup--p-anchor" data-href="https://scihub.copernicus.eu/" href="https://scihub.copernicus.eu/" rel="noopener nofollow noreferrer" target="_blank">
    Copernicus
   </a>
   ：欧洲航天局的网站以及哥白尼六哨兵卫星的图像。比 Landsat 的分辨率更高。请参阅网站
   <a class="markup--anchor markup--p-anchor" data-href="https://gisgeography.com/how-to-download-sentinel-satellite-data/" href="https://gisgeography.com/how-to-download-sentinel-satellite-data/" rel="noopener nofollow noreferrer" target="_blank">
    GISGeography
   </a>
   上有关如何下载和免费图像的说明；
  </p>
  <p class="graf graf--p graf-after--p" id="f427">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.descarteslabs.com/" href="https://www.descarteslabs.com/" rel="noopener nofollow noreferrer" target="_blank">
    Descartes Labs
   </a>
   ：一种商业服务，每天从公共和商业图像提供商处收集数据，对记者的调查来说有很好的帮助；
  </p>
  <p class="graf graf--p graf-after--p" id="896a">
   <a class="markup--anchor markup--p-anchor" data-href="http://discover.digitalglobe.com/" href="http://discover.digitalglobe.com/" rel="noopener nofollow noreferrer" target="_blank">
    DigitalGlobe
   </a>
   ：卫星图像供应商。通过 catalogus 预览，非常容易使用的搜索工具；
  </p>
  <p class="graf graf--p graf-after--p" id="04f0">
   <a class="markup--anchor markup--p-anchor" data-href="http://data.mashedworld.com/dualmaps/map.htm?x=-87.981906&amp;y=43.029482&amp;z=16&amp;gm=2&amp;ve=4&amp;gc=0&amp;bz=1&amp;bd=0&amp;mw=1&amp;mi=0&amp;mg=1&amp;mv=0" href="http://data.mashedworld.com/dualmaps/map.htm?x=-87.981906&amp;y=43.029482&amp;z=16&amp;gm=2&amp;ve=4&amp;gc=0&amp;bz=1&amp;bd=0&amp;mw=1&amp;mi=0&amp;mg=1&amp;mv=0" rel="noopener nofollow noreferrer" target="_blank">
    DualMaps
   </a>
   ：将 Google 的路线图、鸟瞰图和街景视图结合在一个嵌入式工具中。
   <strong class="markup--strong markup--p-strong">
    探照灯一般
   </strong>
   ；
  </p>
  <p class="graf graf--p graf-after--p" id="90fc">
   <a class="markup--anchor markup--p-anchor" data-href="http://earthexplorer.usgs.gov/" href="http://earthexplorer.usgs.gov/" rel="noopener nofollow noreferrer" target="_blank">
    EarthExplorer
   </a>
   ：来自美国地质局。主要提供美国图像。提供对 Landsat 卫星数据以及 NASA 土地数据产品和服务的访问。 USGS 全球可视化查看器（GloVis）提供遥感数据。USGS 档案包含完整且维护良好的 NASA Landsat 数据集。
  </p>
  <p class="graf graf--p graf-after--p" id="d4f1">
   <a class="markup--anchor markup--p-anchor" data-href="https://eos.com/landviewer/" href="https://eos.com/landviewer/" rel="noopener nofollow noreferrer" target="_blank">
    EOS Landviewer
   </a>
   ：EOS Landviewer 提供最多 10 张图片的免费服务。更多图像和分析以折扣价提供给记者。联系方式：Artem Seredyuk
   <a class="markup--anchor markup--p-anchor" data-href="mailto:artem.seredyuk@eosda.com" href="/cdn-cgi/l/email-protection#4f2e3d3b2a22613c2a3d2a2b363a240f2a203c2b2e612c2022" rel="noreferrer noopener" target="_blank">
    <span class="__cf_email__" data-cfemail="31504345545c1f425443545548445a71545e4255501f525e5c">
     [email protected]
    </span>
   </a>
   。 EOS 正在开发一项暂时被称为 EOS Media 的服务，该服务将提供免费图像和主要自然灾害分析。
  </p>
  <p class="graf graf--p graf-after--p" id="0d95">
   <a class="markup--anchor markup--p-anchor" data-href="https://earth.esa.int/web/guest/eoli" href="https://earth.esa.int/web/guest/eoli" rel="noopener nofollow noreferrer" target="_blank">
    ESA Earth Online
   </a>
   ：整合欧洲航天局关于温度、农业和冰盖等主题的地球观测数据。
  </p>
  <p class="graf graf--p graf-after--p" id="be43">
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/musafir-py/find2places" href="https://github.com/musafir-py/find2places" rel="noopener nofollow noreferrer" target="_blank">
    find2places
   </a>
   ：允许在给定半径内以精确的距离查询两个特定位置的 Google Maps API。用于地理定位照片和视频。这是个脚本，用户界面还不那么好看。
   <strong class="markup--strong markup--p-strong">
    关联我们的文章《
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@iyouport/%E5%85%A8%E7%90%83%E6%90%9C%E7%BD%97-%E7%94%A8%E4%BA%8E%E4%BF%A1%E6%81%AF%E9%AA%8C%E8%AF%81%E7%9A%84%E5%9F%BA%E6%9C%AC%E5%9C%B0%E7%90%86%E5%AE%9A%E4%BD%8D%E5%B7%A5%E5%85%B7-osint-%E5%BC%80%E6%BA%90%E8%B0%83%E6%9F%A5%E6%8A%80%E6%9C%AF%E7%BB%A7%E7%BB%AD-867d45d8fe28" href="https://medium.com/@iyouport/%E5%85%A8%E7%90%83%E6%90%9C%E7%BD%97-%E7%94%A8%E4%BA%8E%E4%BF%A1%E6%81%AF%E9%AA%8C%E8%AF%81%E7%9A%84%E5%9F%BA%E6%9C%AC%E5%9C%B0%E7%90%86%E5%AE%9A%E4%BD%8D%E5%B7%A5%E5%85%B7-osint-%E5%BC%80%E6%BA%90%E8%B0%83%E6%9F%A5%E6%8A%80%E6%9C%AF%E7%BB%A7%E7%BB%AD-867d45d8fe28" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     用于信息验证的基本地理定位工具
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    》
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="62f8">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.geograph.org/" href="http://www.geograph.org/" rel="noopener nofollow noreferrer" target="_blank">
    Geograph
   </a>
   ：地理配准图像。
  </p>
  <p class="graf graf--p graf-after--p" id="802c">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.geonames.org/" href="http://www.geonames.org/" rel="noopener nofollow noreferrer" target="_blank">
    GeoNames
   </a>
   ：位置名称数据库。多语种的，以及各种不同的拼写。
  </p>
  <p class="graf graf--p graf-after--p" id="39e2">
   <a class="markup--anchor markup--p-anchor" data-href="https://search.descarteslabs.com/?layer=naip_v2_rgb_2014-2015#lat=39.2322531&amp;lng=-100.8544921&amp;skipTut=true&amp;zoom=5" href="https://search.descarteslabs.com/?layer=naip_v2_rgb_2014-2015#lat=39.2322531&amp;lng=-100.8544921&amp;skipTut=true&amp;zoom=5" rel="noopener nofollow noreferrer" target="_blank">
    GeoVisual Search
   </a>
   ：搜索引擎，允许用户直观地查询图像以获得类似的地理特征。这个笛卡尔实验室的平台建立在 Landsat、国家农业图像计划（NAIP）和 PlanetScope 的卫星图像基础上。
   <a class="markup--anchor markup--p-anchor" data-href="https://medium.com/@DescartesLabs/searching-the-world-wide-world-357828cf479a" href="https://medium.com/@DescartesLabs/searching-the-world-wide-world-357828cf479a" rel="noopener noreferrer" target="_blank">
    这里读到
   </a>
   关于如何使用它的说明。
  </p>
  <p class="graf graf--p graf-after--p" id="f94d">
   Google Earth Pro：添加 Bing 地图卫星图像图层。优点是有
   <strong class="markup--strong markup--p-strong">
    历史图像。
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="d768">
   <a class="markup--anchor markup--p-anchor" data-href="https://earthengine.google.com/" href="https://earthengine.google.com/" rel="noopener nofollow noreferrer" target="_blank">
    Google Earth Engine
   </a>
   ：开放式卫星图像和分析框架。实际上，从 NASA，NOAA，USGS 等收集的任何卫星图像都是可用的。可能略有些粗糙，不是高分辨率的商业图像。用 Javascript 学习曲线。
  </p>
  <p class="graf graf--p graf-after--p" id="4812">
   <a class="markup--anchor markup--p-anchor" data-href="http://maps.google.com/" href="https://maps.google.com/" rel="noopener nofollow noreferrer" target="_blank">
    Google Maps
   </a>
   ：StreetView、美国欧洲许多大城市都可见，以 3D 为蓝本。缺点是没有历史卫星图像，但许多地方都有
   <strong class="markup--strong markup--p-strong">
    历史街景图像
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="1064">
   <a class="markup--anchor markup--p-anchor" data-href="https://wego.here.com/" href="https://wego.here.com/" rel="noopener nofollow noreferrer" target="_blank">
    HERE WeGo
   </a>
   ：对于中东事件调查来说，它有比谷歌最近的卫星图像。
  </p>
  <p class="graf graf--p graf-after--p" id="9848">
   <a class="markup--anchor markup--p-anchor" data-href="http://mapchecking.com/" href="http://mapchecking.com/" rel="noopener nofollow noreferrer" target="_blank">
    Mapchecking
   </a>
   ：计算所选 Google 地图区域中的
   <strong class="markup--strong markup--p-strong">
    人数
   </strong>
   。单击地图以添加点并绘制区域。集会、群体事件、或者救灾……等等，都适用。
  </p>
  <p class="graf graf--p graf-after--p" id="0bf3">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.mapillary.com/" href="http://www.mapillary.com/" rel="noopener nofollow noreferrer" target="_blank">
    Mapillary
   </a>
   ：众包街道照片。是对 Google Streetview 的有用补充。不过对于中东地区来说不好用，在叙利亚、伊拉克等国家几乎找不到。
  </p>
  <p class="graf graf--p graf-after--p" id="3a3a">
   NASA EarthData：WorldView 允许从 NASA 可视化近
   <strong class="markup--strong markup--p-strong">
    实时
   </strong>
   的图像。优势很多，各种卫星和航拍图像; 广泛的搜索标准; 以及其他地图和可视化工具，例如用于火灾事件调查的 FIRMS，等等。可访问十几个 NASA 数据中心和相关的卫星数据产品。美国宇航局地球观测：大气、陆地、海洋、能源、环境等 50 多个数据集。
  </p>
  <p class="graf graf--p graf-after--p" id="376d">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.oldmapsonline.org/" href="http://www.oldmapsonline.org/" rel="noopener nofollow noreferrer" target="_blank">
    Old Maps Online
   </a>
   ：通过世界各地的众多数据库查找旧地图。易于使用，与 DigitalGlobe 目录类似的浏览。
  </p>
  <p class="graf graf--p graf-after--p" id="1722">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.openstreetcam.org/" href="http://www.openstreetcam.org/" rel="noopener nofollow noreferrer" target="_blank">
    OpenStreetCam
   </a>
   ：正如其名。
  </p>
  <p class="graf graf--p graf-after--p" id="c48f">
   <a class="markup--anchor markup--p-anchor" data-href="http://www.openstreetmap.org/" href="https://www.openstreetmap.org/" rel="noopener nofollow noreferrer" target="_blank">
    OpenStreetMap
   </a>
   ：正如其名。
  </p>
  <p class="graf graf--p graf-after--p" id="4cc2">
   <a class="markup--anchor markup--p-anchor" data-href="http://radiant.earth/" href="http://radiant.earth/" rel="noopener nofollow noreferrer" target="_blank">
    Radiant Earth
   </a>
   ：非营利组织，帮助全球发展社区发现、探索和分析
   <strong class="markup--strong markup--p-strong">
    卫星、无人机和航空图像档案
   </strong>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="a491">
   <a class="markup--anchor markup--p-anchor" data-href="https://resourcewatch.org/" href="https://resourcewatch.org/" rel="noopener nofollow noreferrer" target="_blank">
    Resource Watch
   </a>
   ：一个仍处于测试阶段的非营利平台，提供有关地球资源和公民状态的数百个数据集。由世界资源研究所和其他组织赞助。资源监视数据是免费的，用户可以下载数据。
  </p>
