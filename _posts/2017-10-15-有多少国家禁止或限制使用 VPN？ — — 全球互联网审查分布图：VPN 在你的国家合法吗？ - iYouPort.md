  <p class="graf graf--p graf-after--figure" id="e5c9">
   VPN 是否合法取决于你在哪个国家。然而很多管制和限制是不透明的，处于黑色地带。
  </p>
  <p class="graf graf--p graf-after--p" id="6cd4">
   以下的研究列举了 196 个国家的审查法律，及其对 VPN 的管制和限制状态。
  </p>
  <p class="graf graf--p graf-after--p" id="9acb">
   <strong class="markup--strong markup--p-strong">
    10 个禁止或限制 VPN 使用的国家/地区
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="cd7b">
   <a class="markup--anchor markup--p-anchor" data-href="http://1.xn--fiqs8s/" href="http://1.xn--fiqs8s/" rel="noreferrer noopener nofollow" target="_blank">
    1.
    <strong class="markup--strong markup--p-strong">
     中国
    </strong>
   </a>
   ：只允许使用被政府批准的 VPN
  </p>
  <p class="graf graf--p graf-after--p" id="1e0d">
   明确的文件是从 2018 年 3 月开始，事实上在至少三年前就有这一动议，虽然目前为止仍没有切实完成这一审查。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.techradar.com/news/china-will-block-all-non-approved-vpns-from-next-month" href="https://www.techradar.com/news/china-will-block-all-non-approved-vpns-from-next-month" rel="noopener nofollow noreferrer" target="_blank">
    官方给出的理由
   </a>
   是“限制非法跨境业务活动”。这意味着：VPN 服务提供商需要首先获得中国政府的批准（并且很可能违背 VPN 的目的）。
  </p>
  <p class="graf graf--p graf--startsWithDoubleQuote graf-after--p" id="6b64">
   “私自访问国际网络或通过”非法渠道“或未经授权提供相关服务的个人”可能会被
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vpnunlimitedapp.com/blog/penalties-for-using-vpn/" href="https://www.vpnunlimitedapp.com/blog/penalties-for-using-vpn/" rel="noopener nofollow noreferrer" target="_blank">
    罚款
   </a>
   高达15,000元人民币（约合2200美元）。
  </p>
  <p class="graf graf--p graf-after--p" id="0845">
   2、俄罗斯：只有政府批准的VPN才可以使用
  </p>
  <p class="graf graf--p graf-after--p" id="2b3e">
   开始自 2017年11月。
   <a class="markup--anchor markup--p-anchor" data-href="http://www.yugatech.com/news/vladimir-putin-signs-law-banning-vpn/" href="http://www.yugatech.com/news/vladimir-putin-signs-law-banning-vpn/" rel="noopener nofollow noreferrer" target="_blank">
    官方给出的理由
   </a>
   是，为了防止“极端主义材料”在互联网上的传播；旨在阻止“非法内容”的访问，并不打算对守法公民施加限制。
  </p>
  <p class="graf graf--p graf-after--p" id="6433">
   这意味着：如果您使用的 VPN 不受政府批准并被捕，则会受到特定
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vpnunlimitedapp.com/blog/penalties-for-using-vpn/" href="https://www.vpnunlimitedapp.com/blog/penalties-for-using-vpn/" rel="noopener nofollow noreferrer" target="_blank">
    处罚
   </a>
   ：用户 700,000卢布（12,000美元）为服务提供商提供 30 万卢布（5,100美元）。
  </p>
  <p class="graf graf--p graf-after--p" id="8275">
   3、伊朗：只允许政府批准的 VPN
  </p>
  <p class="graf graf--p graf-after--p" id="76ca">
   开始自 2013 年。据称是：起诉违反国家法律的用户，并在司法部门的监督下将罪犯带到国家法院。
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/VPN_blocking#Iran" href="https://en.wikipedia.org/wiki/VPN_blocking#Iran" rel="noopener nofollow noreferrer" target="_blank">
    据报道
   </a>
   ，政府批准的 VPN 使用导致了对私人数据的监视和审查。
  </p>
  <p class="graf graf--p graf-after--p" id="97cd">
   意味着：使用不符合伊朗法律的 VPN 可能会导致 91 天甚至长达1年的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.vpnunlimitedapp.com/blog/penalties-for-using-vpn/" href="https://www.vpnunlimitedapp.com/blog/penalties-for-using-vpn/" rel="noopener nofollow noreferrer" target="_blank">
    监禁
   </a>
   。
  </p>
  <p class="graf graf--p graf-after--p" id="8cf5">
   然而，尽管伊朗有大量的“非法” VPN 用户，但实际的逮捕非常罕见。为什么？因为，就像我们之前解释的那样，伊朗政府只是利用这部法律来起诉政治反对者，而不怎么在乎普通网民浏览 Facebook。
  </p>
  <p class="graf graf--p graf-after--p" id="a28f">
   4、阿联酋‘’；对个人进行限制
  </p>
  <p class="graf graf--p graf-after--p" id="3009">
   开始自 2012 年。其中一个重要
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theregister.co.uk/2016/07/28/vpn_users_in_uae_face_544k_fine/" href="https://www.theregister.co.uk/2016/07/28/vpn_users_in_uae_face_544k_fine/" rel="noopener nofollow noreferrer" target="_blank">
    原因
   </a>
   是人们使用 Skype，WhatsApp，Viber 或其他免费 VOIP 服务，而冷落了阿联酋的大型电信公司。
  </p>
  <p class="graf graf--p graf-after--p" id="932f">
   意味着：任何使用 VPN 伪装 IP 的用户一旦被发现将根据犯罪性质处以
   <a class="markup--anchor markup--p-anchor" data-href="https://www.tra.gov.ae/en/services-and-activities/internet-guidelines/details.aspx#pages-67183" href="https://www.tra.gov.ae/en/services-and-activities/internet-guidelines/details.aspx#pages-67183" rel="noopener nofollow noreferrer" target="_blank">
    监禁
   </a>
   或高达 40 万美元的罚款。使用 VPN 来使用 Skype 也可能导致惩罚。值得一提的是，银行、公司和机构可以自由使用VPN，被禁止的只是个人。
  </p>
  <p class="graf graf--p graf-after--p" id="6343">
   5、阿曼：只允许政府批准的 VPN 使用
  </p>
  <p class="graf graf--p graf-after--p" id="58fd">
   从 2010 年开始。很容易理解为什么 TRA 禁止使用VPN，因为在这个国家的主要用途是绕过 ISP 审查和禁止 VOIP 的使用。少数人也使用 VPN 服务伪造他们的 IP 位置，以便仅使用在某个地区提供的服务
  </p>
  <p class="graf graf--p graf-after--p" id="2e53">
   意味着：拟议的法律规定，个人违反者将被
   <a class="markup--anchor markup--p-anchor" data-href="https://thenextweb.com/me/2010/08/28/oman-to-ban-vpns-for-individuals/" href="https://thenextweb.com/me/2010/08/28/oman-to-ban-vpns-for-individuals/" rel="noopener nofollow noreferrer" target="_blank">
    罚款
   </a>
   500 阿曼里亚尔（1,300美元）没有适当许可证的公司罚款 1000 阿曼里亚尔（2,600美元）。“如果您使用政府批准的 VPN，您的所有日志将被保留。
  </p>
  <p class="graf graf--p graf-after--p" id="5ed2">
   6、土耳其：限制
  </p>
  <p class="graf graf--p graf-after--p" id="45b0">
   从 2016 年开始。
   <a class="markup--anchor markup--p-anchor" data-href="https://www.dailydot.com/layer8/turkey-block-tor-vpns-activists/" href="https://www.dailydot.com/layer8/turkey-block-tor-vpns-activists/" rel="noopener nofollow noreferrer" target="_blank">
    防止
   </a>
   人们通过 VPN 服务规避政府施加的审查制度。
  </p>
  <p class="graf graf--p graf-after--p" id="601b">
   意味着：使用 VPN 可能会导致一些问题，但
   <a class="markup--anchor markup--p-anchor" data-href="https://www.deepdotweb.com/2016/12/30/turkish-government-permanently-bans-tor-vpn-services/" href="https://www.deepdotweb.com/2016/12/30/turkish-government-permanently-bans-tor-vpn-services/" rel="noopener nofollow noreferrer" target="_blank">
    仍有
   </a>
   一些 VPN 能够绕过土耳其的网络封锁。
  </p>
  <p class="graf graf--p graf-after--p" id="68ed">
   7、伊拉克：全面禁止
  </p>
  <p class="graf graf--p graf-after--p" id="434c">
   从 2014 年开始。
   <a class="markup--anchor markup--p-anchor" data-href="http://observers.france24.com/en/20160323-social-media-iranian-authorities-break-internet-laws" href="http://observers.france24.com/en/20160323-social-media-iranian-authorities-break-internet-laws" rel="noopener nofollow noreferrer" target="_blank">
    官方
   </a>
   称，禁止 VPN 限制和阻止 ISIS 操纵社交媒体。然而以阻止 ISIS 为借口，伊拉克已经从禁止 VPN 到阻止社交媒体和聊天应用实施了一系列的严重的审查。尽管如此，并没有阻止政府官员和政府拥有的机构违反法律并使用这些工具。
  </p>
  <p class="graf graf--p graf-after--p" id="40bb">
   意味着：VPN 服务在伊拉克被禁止声称在线监控恐怖分子的活动。事实上对居住在该国的所有人都有影响。
  </p>
  <p class="graf graf--p graf-after--p" id="90be">
   8、土库曼斯坦：全面禁止
  </p>
  <p class="graf graf--p graf-after--p" id="da99">
   从 2015 年开始。是为审查外国媒体。这一切都与政治有关。土库曼斯坦只有一个 ISP，属于政府。这是一个全面审查的问题。
  </p>
  <p class="graf graf--p graf-after--p" id="29be">
   意味着：尝试使用代理服务器和 VPN 会被检测到，并被阻止; 使用者将受到行政
   <a class="markup--anchor markup--p-anchor" data-href="http://tbinternet.ohchr.org/Treaties/CCPR/Shared%20Documents/TKM/INT_CCPR_CSS_TKM_26541_E.pdf" href="http://tbinternet.ohchr.org/Treaties/CCPR/Shared%20Documents/TKM/INT_CCPR_CSS_TKM_26541_E.pdf" rel="noopener nofollow noreferrer" target="_blank">
    处罚
   </a>
   ，并被召集到国家安全部进行“预防性对话”，面临恐吓。“
  </p>
  <p class="graf graf--p graf-after--p" id="e73e">
   9、白俄罗斯：全面禁止
  </p>
  <p class="graf graf--p graf-after--p" id="d40d">
   2015 年开始。白俄罗斯政府将 Tor 和 VPN 看作是转发流量的服务，以绕过政治家可能采取的任何限制。白俄罗斯也为自己的公民
   <a class="markup--anchor markup--p-anchor" data-href="https://www.newsbtc.com/2016/12/18/belarus-fights-darknet-crime-actively-blocking-tor-vpn-access/" href="https://www.newsbtc.com/2016/12/18/belarus-fights-darknet-crime-actively-blocking-tor-vpn-access/" rel="noopener nofollow noreferrer" target="_blank">
    封锁
   </a>
   了大部分外国网站。尽管合法的Tor / VPN使用案例非常丰富，但政府认为封锁这项服务是最好的选择。
  </p>
  <p class="graf graf--p graf-after--p" id="0087">
   意味着：使用VPN绕过地理限制可能会导致罚款。
  </p>
  <p class="graf graf--p graf-after--p" id="8537">
   10、朝鲜：全面禁止
  </p>
  <p class="graf graf--p graf-after--p" id="4ce7">
   在这个问题上很难找到关于朝鲜的信息。作为游客，您可以使用 VPN 软件。但朝鲜禁止外交官使用互联网。
  </p>
  <p class="graf graf--p graf-after--p" id="0f46">
   <strong class="markup--strong markup--p-strong">
    其他国家的审查状态和 VPN 合法性
   </strong>
  </p>
  <p class="graf graf--p graf-after--p" id="c9dc">
   &gt; 阿富汗政府最近发布了
   <a class="markup--anchor markup--p-anchor" data-href="https://cpj.org/blog/2010/10/using-new-internet-filters-afghanistan-blocks-news.php" href="https://cpj.org/blog/2010/10/using-new-internet-filters-afghanistan-blocks-news.php" rel="noopener nofollow noreferrer" target="_blank">
    新闻
   </a>
   ，要求互联网服务提供商过滤被认为“令人反感”的某些类别的网站，并彻底禁止他们认为反感的所有网站。尽管如此，在阿富汗使用 VPN 仍然是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="e3b5">
   &gt; 阿尔巴尼亚宪法保护公民的言论自由和新闻自由，并
   <a class="markup--anchor markup--p-anchor" data-href="https://www.state.gov/j/drl/rls/hrrpt/2012humanrightsreport/index.htm?year=2012&amp;dlid=204254#wrapper" href="https://www.state.gov/j/drl/rls/hrrpt/2012humanrightsreport/index.htm?year=2012&amp;dlid=204254#wrapper" rel="noopener nofollow noreferrer" target="_blank">
    没有记录显示
   </a>
   政府试图侵犯这些权利以及公民使用互联网的权利。 VPN 在阿尔巴尼亚的使用是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="66e6">
   &gt; 阿尔及利亚的立法要求互联网服务提供商对在线内容进行监视，并且
   <a class="markup--anchor markup--p-anchor" data-href="https://opennet.net/blog/2010/01/algeria-joins-internet-censors-club" href="https://opennet.net/blog/2010/01/algeria-joins-internet-censors-club" rel="noopener nofollow noreferrer" target="_blank">
    有记录显示
   </a>
   阿尔及利亚政府阻止某些网站（包括政治反对派网站）的事例，以及阿尔及利亚政府
   <a class="markup--anchor markup--p-anchor" data-href="http://www.bbc.co.uk/news/world-africa-36570808" href="https://www.bbc.co.uk/news/world-africa-36570808" rel="noopener nofollow noreferrer" target="_blank">
    阻止所有社交媒体
   </a>
   “防止在全国考试中作弊”。但是，在阿尔及利亚，VPN 的使用仍然合法。
  </p>
  <p class="graf graf--p graf-after--p" id="f45e">
   &gt; 约有79％的安道尔人使用互联网，使其成为互联网普及率最高的国家之一。然而，在安道尔
   <a class="markup--anchor markup--p-anchor" data-href="https://www.state.gov/documents/organization/160178.pdf" href="https://www.state.gov/documents/organization/160178.pdf" rel="noopener nofollow noreferrer" target="_blank">
    没有文件记录
   </a>
   显示互联网审查，安道尔使用 VPN 是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="cbcf">
   &gt; 安哥拉试图
   <a class="markup--anchor markup--p-anchor" data-href="https://www.theguardian.com/world/2016/aug/19/angola-passes-laws-to-crack-down-on-press-and-social-media" href="https://www.theguardian.com/world/2016/aug/19/angola-passes-laws-to-crack-down-on-press-and-social-media" rel="noopener nofollow noreferrer" target="_blank">
    审查
   </a>
   媒体和互联网的访问，安哥拉总统甚至公开宣布要打击社交媒体的计划，但是并没有试图阻止该国的 VPN 使用。

   </a>
   塞舌尔公民进入互联网，特别是在试图镇压反对派活动家时。然而，VPN的使用目前在该国是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="b2f9">
   &gt; 目前还
   <a class="markup--anchor markup--p-anchor" data-href="https://www.state.gov/j/drl/rls/hrrpt/2016/af/265298.htm" href="https://www.state.gov/j/drl/rls/hrrpt/2016/af/265298.htm" rel="noopener nofollow noreferrer" target="_blank">
    没有
   </a>
   关于塞拉利昂政府试图干预公民互联网使用的文件报告，并且在该国使用VPN是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="43e9">
   &gt; 新加坡政治网站
   <a class="markup--anchor markup--p-anchor" data-href="https://freedomhouse.org/report/freedom-net/2015/singapore" href="https://freedomhouse.org/report/freedom-net/2015/singapore" rel="noopener nofollow noreferrer" target="_blank">
    已被关闭
   </a>
   ，政府批评者被定位，其他形式的审查也非常丰富。但是，在该国使用VPN仍然合法。
  </p>
  <p class="graf graf--p graf-after--p" id="2417">
   &gt; 斯洛伐克政府
   <a class="markup--anchor markup--p-anchor" data-href="http://pasos.org/wp-content/uploads/2015/03/Internet-freedom-Slovakia.pdf" href="http://pasos.org/wp-content/uploads/2015/03/Internet-freedom-Slovakia.pdf" rel="noopener nofollow noreferrer" target="_blank">
    很少被发现
   </a>
   干涉公民在该国使用互联网的权利，目前在该国使用VPN是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="e48f">
   &gt; 斯洛文尼亚有超过73％的人使用互联网，普及率相当高。目前还
   <a class="markup--anchor markup--p-anchor" data-href="https://www.state.gov/j/drl/rls/hrrpt/2016/eur/265474.htm" href="https://www.state.gov/j/drl/rls/hrrpt/2016/eur/265474.htm" rel="noopener nofollow noreferrer" target="_blank">
    没有
   </a>
   政府试图审查该国互联网审查的可信报告，目前使用VPN是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="2993">
   &gt; 尽管所罗门群岛这个大洋洲国家的互联网普及率很低，但
   <a class="markup--anchor markup--p-anchor" data-href="https://freedomhouse.org/report/freedom-press/2014/solomon-islands" href="https://freedomhouse.org/report/freedom-press/2014/solomon-islands" rel="noopener nofollow noreferrer" target="_blank">
    没有
   </a>
   关于政府试图干预公民互联网接入的文件记录。在这个国家，VPN的使用也是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="a49e">
   &gt; 索马里政府
   <a class="markup--anchor markup--p-anchor" data-href="https://citizenlab.org/2014/02/internet-filtering-failed-state-case-netsweeper-somalia/" href="https://citizenlab.org/2014/02/internet-filtering-failed-state-case-netsweeper-somalia/" rel="noopener nofollow noreferrer" target="_blank">
    经常阻止
   </a>
   批评者的网站，毫无疑问存在互联网审查制度。但是，VPN的使用尚未被取缔。
  </p>
  <p class="graf graf--p graf-after--p" id="a8c1">
   &gt; 当南非政府推出一项“在线监管政策”法案时，许多人称这是非洲
   <a class="markup--anchor markup--p-anchor" data-href="https://www.eff.org/deeplinks/2015/05/africas-worst-new-internet-censorship-law-could-be-coming-south-africa" href="https://www.eff.org/deeplinks/2015/05/africas-worst-new-internet-censorship-law-could-be-coming-south-africa" rel="noopener nofollow noreferrer" target="_blank">
    最严重的互联网审查法
   </a>
   。然而，就公民的互联网权利而言，该国一般是相对自由的。在该国使用VPN是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="c944">
   &gt; 韩国
   <a class="markup--anchor markup--p-anchor" data-href="https://freedomhouse.org/report/freedom-net/2015/south-korea" href="https://freedomhouse.org/report/freedom-net/2015/south-korea" rel="noopener nofollow noreferrer" target="_blank">
    监测和审查用户
   </a>
   的行为一再被曝光。不过，在这个国家使用VPNs是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="1ea8">
   &gt; 虽然南苏丹的互联网普及率较低，但政府
   <a class="markup--anchor markup--p-anchor" data-href="https://freedomhouse.org/report/freedom-press/2016/south-sudan" href="https://freedomhouse.org/report/freedom-press/2016/south-sudan" rel="noopener nofollow noreferrer" target="_blank">
    审查
   </a>
   了公民的互联网活动。在该国使用VPN是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="491f">
   &gt; 尽管西班牙在公民的互联网权利方面是一个普遍自由的国家，但最近在执政党提出
   <a class="markup--anchor markup--p-anchor" data-href="http://www.telegraph.co.uk/news/2016/11/10/no-joke-spanish-prime-ministers-party-suggests-ban-on-internet-m/" href="https://www.telegraph.co.uk/news/2016/11/10/no-joke-spanish-prime-ministers-party-suggests-ban-on-internet-m/" rel="noopener nofollow noreferrer" target="_blank">
    禁止社交媒体meme
   </a>
   时引发了轰动。在该国使用VPN是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="9df8">
   &gt; 有报道称斯里兰卡政府正在
   <a class="markup--anchor markup--p-anchor" data-href="http://srilankabrief.org/2016/10/sri-lanka-govt-restart-censoring-internet-a-tamil-web-site-blocked/" href="http://srilankabrief.org/2016/10/sri-lanka-govt-restart-censoring-internet-a-tamil-web-site-blocked/" rel="noopener nofollow noreferrer" target="_blank">
    审查其公民的互联网活动并阻止批评政府的网站
   </a>
   。不过在该国使用VPN是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="ef88">
   &gt; 在苏丹，互联网被关闭，网吧被袭击和网络用户被
   <a class="markup--anchor markup--p-anchor" data-href="https://freedomhouse.org/report/freedom-net/2016/sudan" href="https://freedomhouse.org/report/freedom-net/2016/sudan" rel="noopener nofollow noreferrer" target="_blank">
    逮捕
   </a>
   的情况并不少见。不过据称该国的公民在互联网自由方面正在“改善”。在该国使用VPN是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="75f1">
   &gt; 目前还
   <a class="markup--anchor markup--p-anchor" data-href="https://freedomhouse.org/report/freedom-world/2016/suriname" href="https://freedomhouse.org/report/freedom-world/2016/suriname" rel="noopener nofollow noreferrer" target="_blank">
    没有
   </a>
   关于苏里南政府试图检查公民互联网使用情况的可靠报告，并且在该国使用VPN是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="4124">
   &gt; 虽然有报道称斯威士兰政府正在积极
   <a class="markup--anchor markup--p-anchor" data-href="http://www.state.gov/j/drl/rls/hrrpt/humanrightsreport/index.htm?year=2012&amp;dlid=204174" href="http://www.state.gov/j/drl/rls/hrrpt/humanrightsreport/index.htm?year=2012&amp;dlid=204174" rel="noopener nofollow noreferrer" target="_blank">
    监测公民的互联网使用
   </a>
   情况，但还没有关于公民互联网活动受到审查的报道。目前在该国使用VPN是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="d71b">
   &gt; 91％以上的国民使用互联网，瑞典是全球互联网
   <a class="markup--anchor markup--p-anchor" data-href="https://freedomhouse.org/report/freedom-press/2016/sweden" href="https://freedomhouse.org/report/freedom-press/2016/sweden" rel="noopener nofollow noreferrer" target="_blank">
    普及率最高
   </a>
   的国家之一。目前还没有任何可信的政府干预公民互联网活动的报告，并且在该国使用VPN是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="c1c2">
   &gt; 尽管瑞士在互联网自由方面相对表现较好，但
   <a class="markup--anchor markup--p-anchor" data-href="https://www.ifex.org/switzerland/2016/10/03/yes_surveillance/" href="https://www.ifex.org/switzerland/2016/10/03/yes_surveillance/" rel="noopener nofollow noreferrer" target="_blank">
    65.5％的公民通过投票来支持授权政府权力监视
   </a>
   公民互联网活动的法案……不可理喻。在这个国家使用VPN是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="37fb">
   &gt; 叙利亚在
   <a class="markup--anchor markup--p-anchor" data-href="https://freedomhouse.org/report/freedom-net/2015/syria" href="https://freedomhouse.org/report/freedom-net/2015/syria" rel="noopener nofollow noreferrer" target="_blank">
    侵犯人权和网络审查
   </a>
   方面有着悠久的历史，但目前在该国使用VPN是合法的。
  </p>
  <p class="graf graf--p graf-after--p" id="1702">
   &gt; 台湾在几年前提出了
   <a class="markup--anchor markup--p-anchor" data-href="https://thenextweb.com/asia/2013/05/28/taiwan-proposes-china-style-block-on-overseas-internet-services-that-infringe-copyright/#.tnw_RNnodSWL" href="https://thenextweb.com/asia/2013/05/28/taiwan-proposes-china-style-block-on-overseas-internet-services-that-infringe-copyright/#.tnw_RNnodSWL" rel="noopener nofollow noreferrer" target="_blank">
    一项法案
   </a>
   ，赋予政府与中国审查互联网类似的权力，但到目前为止还没有关于政府审查互联网或禁止使用VPN的报道。
  </p>
  <p class="graf graf--p graf-after--p" id="3613">
   &gt; 2014年，美国
   <a class="markup--anchor markup--p-anchor" data-href="http://12mars.rsf.org/2014-en/2014/03/11/usa-nsa-symbolises-intelligence-services-abuses/" href="http://12mars.rsf.org/2014-en/2014/03/11/usa-nsa-symbolises-intelligence-services-abuses/" rel="noopener nofollow noreferrer" target="_blank">
    被加入了
   </a>
   无国界记者组织的
   <strong class="markup--strong markup--p-strong">
    “无国界敌人”
   </strong>
   名单，被指
   <strong class="markup--strong markup--p-strong">
    破坏对互联网和自身安全标准的信心
   </strong>
   。没错第一修正案保护公民的言论自由和新闻自由，但美国是美国，事实上被两码事了。虽然并没有积极地试图审查互联网，但是
   <strong class="markup--strong markup--p-strong">
    政府的秘密监视活动实在太令人震惊了
   </strong>
   。目前该国使用VPN是合法的。
  </p>
