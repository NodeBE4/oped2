    4 个附加组件，以提高您在 Thunderbird 上的隐私

    McCaffrey Knowledge Node , 技术防身/自我保护方法 , 给行动者的护身符和武器 2018年3月10日 2019年8月10日 0 Minutes

    Thunderbird 是 Mozilla 开发的流行免费电子邮件客户端。与 Firefox 类似，Thunderbird 提供了大量附加功能，可用于额外的和自定义功能。本文重点介绍四个附加组件，以改善您的隐私。

    Enigmail

    使用 GPG（GNU Privacy Guard）加密电子邮件是保持内容私密性的最佳方式。如果您不熟悉 GPG，请 在这里 查看我们提供过的最简易指南。

    Enigmail 是 OpenPGP 与 Thunderbird 一起使用的首选附加组件。实际上，Enigmail 与 Thunderbird 集成良好，可让您加密、解密、数字签名和验证电子邮件。

    Paranoia

    Paranoia 可让您访问有关传入电子邮件的重要信息。使用一个表情符显示电子邮件在到达收件箱之前经过的服务器之间的加密状态

    黄色，快乐的表情符号告诉您所有连接都已加密。蓝色，悲伤的表情意味着有一个连接未加密。最后，一个红色的，害怕的表情符号告诉你在多个连接上显示该消息未加密。

    根据这些提示及更多详细信息，您可以检查用于传递电子邮件的服务器是否安全。

    Sensitivity Header

    Sensitivity Header 是一个简单的附加组件，可让您选择发送电子邮件的隐私级别。使用选项菜单，您可以选择一个安全强度：正常、个人、私人和机密。

    添加此标头不会为电子邮件添加额外的安全性。但是，某些电子邮件客户端或邮件传输/用户代理（MTA / MUA）可以使用此标头根据不同的安全强度以不同方式处理邮件。

    请注意，这个附加组件被开发人员标记为实验性的。

    TorBirdy

    给所有关心隐私的人，TorBirdy 就是最好的附加组件。它将 Thunderbird 配置为使用 Tor 网络。这是非常棒的。

    请注意，要使用此附加组件，必须在系统上安装 Tor。
