    如何使树莓派看起来像Windows或macOS

    Gabriel Fornard Knowledge Node , 技术 2021年12月7日 2021年12月22日 1 Minute
    • 可帮助新手适应 Linux。

    用 Twister OS 给您的树莓派改头换面一下吧。它是一个Linux 发行版，具有内置的一键式主题 ，模仿 Windows 和 macOS 操作系统。有现代和复古的选择。例如，您可以选择 Windows 10-、Windows 7-、Windows XP- 或 Windows 95 风格的主题。

    令人难以置信的树莓派

    什么东西尺寸为3.5英寸乘2.2英寸乘0.7英寸（85毫米乘56毫米乘17毫米），销量超过3000万台？它是仅次于PC和Mac的第三大最畅销的普通计算机，也是有史以来最成功的英国计算机。这就是 树莓派单板计算机 。

    它是在2012年2月作为一种手段推出的。当时树莓派基金会的创始人埃本·厄普顿( Eben Upton )想生产一些东西，让英国的学童可以开始使用计算机和编码，就像1980年代的BBC微型计算机那样。

    树莓派的前两个型号被命名为 Model A和ModelB，事实上，Acorn 继续设计了ARM处理器，这是世界上使用最广泛的处理器，也是树莓派的核心。

    树莓派与BBC微型计算机的不同之处在于价格。1981年，一台BBC B型计算机的价格为399英镑（563美元）。2012年，树莓派B型机的价格是22英镑（31美元）。最便宜的树莓派售价仅为5美元（4英镑）。

    树莓派的低价和开放、便于修补的设计使它非常适合教育以外的数百种应用。业余爱好者和工业界都在使用它，它还推动了一个繁荣的售后附加产业。

    选择您的树莓派操作系统

    官方的树莓派操作系统是 Debian Linux 的一个定制版本，称为 Raspberry Pi OS 。但您并不局限于此。有20多个基于 Linux 的操作系统可供树莓派使用，也有很多非 Linux 操作系统。例如，您可以在树莓派上使用 Windows 10 IoT 、 Haiku 或 RISC OS Open 。

    但是，如果您想拥有一个看起来像 macOS 或 Windows 版本的东西呢？Windows 10 IoT 没有配备桌面图形用户界面，所以您需要找到另一个解决方案。 Twister OS 就是您正在寻找的解决方案。

    Twister OS 是 Raspberry Pi OS 的定制版本，有大量的主题选择 — — 包括图标 — — 提供各种版本的 Windows 和 macOS 的一般外观和感觉。Twister OS 包含一个小工具，可以让您非常容易地在各种主题之间来回切换。

    它在 Raspberry Pi 4 上运行效果最好，在 Raspberry Pi 3+ 上的反应也差不多。在 Raspberry Pi Model B+ 上测试的结果显示，它可以工作，但速度很慢。

    安装 Twister OS

    从 Twister OS 的下载页面 下载树莓派版本的 Twister OS。找到下载的文件并右键单击它。从上下文菜单中选择 “用存档管理器打开”。如果该选项没有出现，请选择 “用其他应用程序打开” ，并从那里选择存档管理器。

    在存档管理器中，右键单击存档中的单个文件，从菜单中选择 “提取”。

    您会被提示选择一个保存提取文件的位置。打开一个终端窗口，将目录切换到您保存提取文件的位置。

    本案例将需要把镜像保存到SD卡上。提取的镜像是10GB，所以您需要一张至少16GB的卡，但建议32GB。

    在您将SD卡连接到电脑之前，使用 lsblck 命令来识别您电脑中的硬盘驱动器。

    将您的SD卡连接到您的电脑。我们将再次使用 lsblck 。之前没有列出的设备就是您的SD卡。

    在用于本文的机器上，SD卡显示为设备SDC。记下您的SD卡的设备名，这对您来说至关重要。如果您把 Twister OS 镜像恢复到错误的设备上就麻烦了。

    将镜像刻录到SD卡上的命令是：

    sudo dd bs=4M if=TwisterOSv2-0-0.img of=/dev/sdc conv=fdatasync status=progress

    关于这个命令中的内容：

    sudo：您需要作为超级用户才能发送dd命令。
    dd：我们正在使用的命令。
    bs=4M：-bs（块大小）选项定义了从输入文件读取并复制到输出设备的每个块的大小。
    if=: if（输入文件）是 Twister OS 镜像文件的路径和名称。
    of=: -of（输出文件）是关键参数，是我们要写入镜像的设备。在本案例中，它是 /dev/sdc。确保您已经在您的计算机上正确地识别了正确的设备。
    conv=fdatasync：这可以确保在创建过程被标记为完成之前，写缓冲区被正确和完全地刷新。
    status=progress：这提供了一些视觉反馈，关于正在发生什么。

    系统将提示您输入密码，然后进程就开始了。

    该过程可能需要相当长的时间。本案例在测试机器上花了 10 多分钟。该过程完成后，将显示读取和写入的块总数。

    您可以解除对SD卡的挂载，并将其插入您的树莓派。

    启动 Twister OS

    一旦 Twister OS 启动，您会看到标准的 Twister OS 桌面。它使用 xfce桌面环境 。

    Twister Os 的默认桌面

    这是一个功能完美的 Linux 安装。打开终端，查看 “/etc/os-release” 文件，看到它是基于 Raspberry Pi Os 的（以前称为Raspbian），而 Raspbian 是由 Debian Linux Buster 版本衍生而来的。

    cat /etc/os-release

    “ThemeTwister” 图标位于桌面上。这是主题选择工具。

    双击该图标启动主题选择对话框。

    单击蓝色的 “下一步” 按钮以查看操作系统主题的选择。

    一些主题带有黑暗模式。您可以从以下方面选择；

    Twister OS浅色模式。
    Twister OS黑暗模式。
    Twister 95，一个Windows 95主题。
    Twister XP，一个Windows XP主题。
    Twister 7，一个Windows 7主题。
    Twister 10 light，一个Windows 10主题。
    Twister 10 dark, 一个黑暗模式的Windows 10主题。
    iTwister light, 一个macOs主题。
    iTwister dark, 一个黑暗模式的macOs主题。
    iTwister Sur light, 一个macOS Big Sur主题。
    iTwister Sur dark, 一个黑暗模式的macOS Big Sur主题。

    没有 Windows Vista 或 Windows 8 主题。

    应该指出，只要您点击主题缩略图下面的任何一个按钮，主题化过程就会开始。没有让您 “确定” 的警告。此外，改变主题将需要重新启动。一条信息会告诉您什么时候按 “Enter” 键重启您的树莓派。

    主题

    本案例尝试了 Windows 7、Windows 10 和 macOS Big Sur 主题。这里是Windows 7的桌面；

    它在唤起经典的 Windows 7 外观和感觉方面做得相当好。开始按钮带来了熟悉的系统菜单。当它们正在运行时，应用程序会出现在屏幕底部的任务栏中。

    Internet Explorer 图标打开了 Chromium 浏览器，Word和Excel图标打开了 LibreOffice Writer 和 Calc。Outlook 图标打开了 Evolution 邮件应用程序。“开始” 按钮上有一个微软色彩的 Twister OS 旋风标志。

    Windows 10 黑暗模式的表现同样令人信服。桌面拥有 Twister OS 徽标而不是 Windows 徽标，但外观和感觉以及对细节的关注强烈唤起了 Windows 10 体验。

    系统菜单中的 Office 菜单条目使用 Microsoft Office 徽标，但指向 LibreOffice 应用程序。

    任务栏中的图标采用当前 Microsoft Word、Excel 和 Outlook 图标的形式。

    Twister Sur 主题试图复制 macOS Big Sur 操作系统的外观及其桌面元素的位置。屏幕底部有程序坞，顶部有一个菜单栏。

    Dock 上的图标对于 Mac 用户来说非常熟悉。它们打开了等效的 Linux 桌面应用程序。

    最后

    Twister OS 是一个健全的基于 Debian 的 Linux，而 xfce桌面是一个轻量级的、可靠的图形用户界面。有很多主流的 Linux 发行版都是建立在这两个著名的 Linux 主干系统之上的。您的 Linux 用户体验不会因为使用 Twister OS 而有任何的降低。

    不同的主题可能会帮助新手适应 Linux，有一个熟悉的界面来工作。但大多数 Twister OS 用户也会喜欢这种轻松的感觉，并有机会告诉别人，是的，他们有一个Mac的树莓。⚫️
