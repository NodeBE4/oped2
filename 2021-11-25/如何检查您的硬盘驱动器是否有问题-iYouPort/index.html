    如何检查您的硬盘驱动器是否有问题

    Gabriel Fornard Knowledge Node , 技术防身/自我保护方法 2021年11月25日 2021年12月22日 1 Minute
    • 以便您可以及时转移数据。

    硬盘故障可能意味着珍贵的图片、重要的文件的灾难性损失。因此，例行检查您的硬盘是否有可能出问题可能是必要的。

    无论是在 Windows 还是 Mac 上，都有许多工具可以帮助您诊断硬盘并修复它。如果您怀疑硬盘接近失效，一定要尝试对您的所有数据进行备份，或者在运行下面的任何工具之前克隆您的硬盘。

    如何在 Windows 上检查您的硬盘是否出问题

    Windows 包括内置工具可以帮助检查您的硬盘，制造商也可能提供工具来帮助诊断。

    1、运行 Chkdsk

    Windows 的内置磁盘检查工具是诊断硬盘问题的最简单方法之一。

    以管理员身份打开命令提示符。

    在命令提示符出现后，输入 chkdsk 并点击回车。它可能需要几秒钟到几分钟来扫描您的驱动器。如果它发现了错误，它将纠正那些它可以纠正的错误。如果它发现了重大的错误，该操作将需要更长的时间来修复。它将在运行结束时向您提供一份它不能修复的错误报告。

    2、使用 SeaTools

    SeaTools 是 Seagate 专用的硬盘工具。它旨在寻找其他程序可能遗漏的希捷制造的硬盘的可能问题。

    有五个版本可供选择；

    • Seagate Bootable，可以加载到U盘上。
    • SeaTools Legacy Tools，可以访问旧版本的软件。
    • SeaTools SSD CLI，一个为固态硬盘设计的命令行工具。
    • SeaTools SSD GUI，一个为固态硬盘设计的图形仪表板。
    • SeaTools，该工具的最基本版本。

    每个版本的操作方式略有不同。启动 SeaTools，它会向您显示每个可用的驱动器。

    • 单击以选择驱动器。
    • 点击 “自我测试 “开始扫描。
    • 在扫描结束时，它将向您显示任何出现的问题。

    3、试试三星魔术师软件

    如果您有一个三星生产的硬盘，三星魔术师软件允许您做以下事：

    • 监测您的硬盘性能，密切关注其健康状况。
    • 自定义设置。
    • 查看您的硬盘在使用时的温度。
    • 运行基准来衡量速度，以及诊断性扫描来寻找潜在的问题。
    • 自定义超额配置功能进行常规维护以释放硬盘内的空间。

    4、学习阅读HDD S.M.A.R.T.属性

    每一块硬盘都有一个叫做S.M.A.R.T.的内置功能，即自我监测、分析和报告技术。它可以跟踪像以下这样的信息点：

    • 气流温度
    • 电源循环次数
    • 通电时间

    像HDD扫描这样的工具将信息提炼成不同的数据点列表。旁边的绿色标记意味着可以正常工作；如果出现黄色标记意味着您应该关注它，而红色标记则表明有问题。

    HDD扫描的强项之一是它能够轻松打印报告。如果您正在监控一个您怀疑出现恶化的硬盘，请保留报告的物理副本，以分析一段时间内的趋势。

    5、使用 Crystal Disk Info

    Crystal Disk Mark 是测量任何驱动器的读写速度的最佳工具之一。Crystal Disk Info 是该工具的诊断版本，可查看您的驱动器的健康状况，通知您异常情况，包括监测S.M.A.R.T.属性。

    Crystal Disk Info 的界面很简陋，但它使用起来很简单，而且完全免费下载。如果您正在寻找一个轻量级的免费工具来跟踪您的驱动器的健康状况，这个应用程序是值得考虑的。

    如何在Mac上检查您的硬盘是否出问题

    在Mac上没有那么多的第三方工具可用，但有一些强大的内置工具。

    1、使用Mac磁盘工具

    与 Windows 的检查磁盘工具一样，macOS也有一个内置的工具，叫做 Disk Utility，它可以完成从分区驱动器到监测驱动器健康的所有工作。访问它的最简单方法是下面的步骤：

    • 打开Spotlight（Command+空格键），搜索 Disk Utility
    • 从列表中选择一个驱动器，右键单击它，然后选择获取信息。这将显示一个关于您的Mac驱动器的相关信息列表。

    向下滚动到写有S.M.A.R.T.状态的地方。它将显示 “已验证”、“失败”，或者在某些情况下，“不支持”。如果它说 “已验证”，那就没问题；如果它说 “失败”，表明您的硬盘可能接近失败；如果它说 “不支持”，说明您的硬盘不支持S.M.A.R.T.。

    2、使用 Blackmagic 磁盘速度测试

    如果Mac磁盘工具不能为您提供所需的信息，第三方应用程序可以提供帮助。Blackmagic Disk Speed Test 就是这样一个工具，它可以测量您的Mac的读写速度。然后您可以将这些读数与您的机器的标准性能数字进行比较。

    如果您觉得您的硬盘没有做出应有的反应，可以测试一下上述这些程序之一。虽然您几乎无法修复一个出现故障的驱动器，但如果您知道它已经接近故障，就可以及时将数据转移到安全的地方。⚪️
