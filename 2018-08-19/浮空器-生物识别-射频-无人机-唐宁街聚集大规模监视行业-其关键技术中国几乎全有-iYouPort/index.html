    最近，英国政府接待了数百家监控技术公司，因为这些公司坚持认为 “基于监视技术的铺设是脱欧问题的终极解决方案” 。

    这是每年一次的“安全与警务”展示会，汇集了 来自世界各地政府机构的最先进的监控设备 。今年的活动是和脱欧谈判进程合拍的，关于爱尔兰 — 英国边境310英里区域维稳问题的具体建议。显然，唐宁街认为大规模监控是个“好主意”。

    但要知道， 简单地将技术投入复杂的政治中是解决不了根本性问题的。不幸的是如今很多政府都痴迷于技术控制社会，他们面对着形形色色的政治问题，却不反思自己，反而期待监视技术能带来绝对的“稳定”和经济效益。是的，他们眼里已经没有人权。

    Snowden 曝光美国大规模监控 有着无数的积极意义 ，我们已经深刻领悟，但任何事物都有两面，最伟大的曝光也一样，它携带着两个特殊的负效应：

    1、对社会的寒蝉效应。当人们不知道这些事的时候，他们反而有一种无知的勇气，虽然他们想不通自己是如何被抓捕的、如何在尚未行动的情况下被提前维稳，公民权如何被侵蚀。不要以为自我审查是奴性社会所特有的，比如中国，恐惧从来是人类的基础情绪之一，我们曾经介绍过美国社会在监视曝光后出现的 惊人的自我审查 ，与中国做了一个对比。你能在前面的链接中看到。

    2、给全球当权者的“参照”。而且并不限于威权政府，那些一直在想方设法控制社会以巩固权力施加暴力的纳粹遗少，甚至包括民主国家政府也一样，他们面对着诸多政治问题，面对着选民的控诉和逼问，于是他们往往最先想到的是 1984 的“捷径”。

    不消说大规模监控的确能给人一种可以针对性“解决”某些社会问题的 幻觉 ，比如校园枪支暴力，美国一些州正在推出人脸识别系统以遏制校园暴力事件。受到了惊吓的人们和孩子被伤害的家长对此表示支持，但更多人对此表示怀疑。很明显，这种声称为了保护个人而剥夺大众隐私的做法注定不是捍卫人权的性质。 并且更糟糕的是，没人能确保这一能力不会被滥用 ，因为它把肥肉扔在了狼的面前。当权者倾向于将民众视为失控的、无意识的生物，却很少去思考人们为什么选择那样的行为 — — 人们为什么会开枪，为什么不满。

    前者是可以被克服的，而后者需要在前者成功克服的基础上发挥民主的力量 。如果民主不是一个幻景的话。

    任何基于大规模监视的解决方案都需要物理基础设施，它必然是侵入性的、侵犯人权的和违宪的。不论其技术被描述为如何的高效，其性质依然是反民主的，与自由开放的承诺截然相悖。

    作为媒体，能做的最好的抵御就是揭露，这就是为什么 Snowden 的选择无比伟大，让更多人熟知真相，公民权利才有机会得到有效的发挥。

    以下是一系列高科技设备的组合，来自英国的采用。你能发现，它们中绝大部分都已经在中国实现。

    自动车牌识别（ANPR）

    至少在 2015 年就有披露，英国警方采用的这套自动车牌号码识别系统（简称 ANPR）已经成为全球规模最大的监控系统之一。并且，直到目前都还没有任何一套成熟的法律框架可以对其具体使用方式加以规范。

    三年前英国国内就已经设立了 8300 个 ANPR 摄像头，其每天收集到的数据会接受 2500 万到 3500 万次“读取”。这些记录随后会被发送至国家 ANPR 数据中心。这种监控 甚至不需要任何法定授权。

    这一技术的可怕在于它有 强大的追溯能力 ，基于一个规模巨大的数据库。可以在 2 秒内识别途经的所有车辆的历史资料。

    在此之后的两年，也就是 2017 年，马来西亚媒体报道，一名的士司机途经交警取缔行动路段时被自动车牌号码识别系统（ANPR）扫描，该系统揭发了他从 1998 年至今 — — 也就是 过去二十年来 — — 未曾缴付的交通罚单。

    要知道，这可不是交通罚单这么简单。这些数据深刻地揭示了一个人生命中的大部分图景。

    去年 11 月，中国，继推出全球首款深度学习网络硬盘录影机(NVR)之后，海康威视公司又宣布推出新系列的 IP 摄像机。所谓的「深眸」( DeepinView )系列 IP 摄像机据称能提供强大功能和情报。其中就包括自动车牌识别（ANPR）。

    射频识别 RFID

    这是一种目前已经非常常见的无线通信技术，可以通过无线电讯号识别特定目标并读写相关数据，而无需识别系统与特定目标之间建立机械或者光学接触。

    无线电的信号是通过调成无线电频率的电磁场，把数据从附着在物品上的标签上传送出去，以自动辨识与追踪该物品。某些标签在识别时从识别器发出的电磁场中就可以得到能量，并不需要电池；也有标签本身拥有电源，并可以主动发出无线电波（调成无线电频率的电磁场）。标签包含了电子储存的信息，数米之内都可以识别。

    注意，与条形码不同的是， 射频标签不需要处在识别器视线之内，也可以嵌入被追踪物体之内 。射频标签只要靠近或经过一个读取器就可以读取。无论是在手提箱里，纸箱里，盒子里等，射频标签都可以被读取。读取机可以一次读取上百个射频标签，而条形码只能一次一读。

    Security and Policing 是英国公司 TC SENS、RFID 技术制造商，专门做这种东西。

    无线射频识别系统在监控车辆方面相当高效。在大雾、大雨、沙尘等能见度较低的条件下，道路摄像头可能无法辨认车牌号，但 RFID 则毫无障碍 。

    就是今年六月的消息，⚠️ 中国正在建立一个利用电子识别系统跟踪汽车的 全国性项目 。无线射频识别 (RFID) 芯片将 安装在汽车挡风玻璃上 。专门的道路读取装置将扫描无线射频识别芯片上的信息，由此确定汽车的位置、牌号、型号和颜色。

    简单说就是，有了这种东西， 只要你被当局盯住了，没可能再使用机动车逃离 。

    中国官媒的报道说，在中国境内注册的所有汽车迟早都必须安装无线射频识别装置。事实上，在一些地区，无线射频识别系统已经通过先导测试。

    视觉监控

    英国采取的视觉监控不仅仅是传统的闭路摄像机。并且， 在无人机、浮空器或飞机上，先进持久性的空中监视系统正被用于执行全天候的老大哥任务，捕获详细的视频图像并将其传输到地面接收站进行分析。

    英国国防部已经从国防承包商和参展商空客公司购买了几颗“伪卫星” ，被称为“Zephyr S”。据称这东西能从世界上任何地方监控地面目标。可以在空中飞行高达 7 万英尺，是商用飞机高度的两倍，甚至可以在空中漂浮 45 天。它的动力来自太阳能，太阳能还为它的锂硫电池充电，然后在晚上作为它的能源。

    Search Systems Ltd 已经展示了他们这一广域监控系统 — — 由英国边防部队支持 — — 安装在一个浮空器上。

    其他公司包括 Sesanti ，其 LREO 先进远程成像系统能 提供热成像识别 ，并声称能够识别超过 30 公里区域内的车辆和超过 8 公里区域内的人，并且可以配备分析工具，例如通过其 AIS 识别船只信号。

    还有， Qinetiq 以前是英国政府的一部分，他们销售使用“先进的被动毫米波成像技术”的监视相机，以 检测人群，每小时可扫描1000人。SPO-NX 系统已经与英国内政部合作 。

    这些浮空器监视不是新玩意，早在 2015 年，中国官媒就报道称，中国电科自主研制出了浮空器技术，加入了美国和俄罗斯才拥有的监控技术水平。

    该报道说，2015 年初的一天，新疆某县的天空中出现了一个浮在空中的大型白色气球，这玩意被称为具备全球顶尖技术的“浮空器”。

    中国官媒还说， 这种监视设备能长期停留在空中作业，俯视全城，不仅能让地面上的所有目标一览无余，同时也节省了大量分散的监控工作，并且获得了当地公安部门的“高度评价“ 。

    生物识别技术

    生物识别技术的使用在日常生活中已经越来越普遍，并且政府和情报部门在这个市场中占据了绝对大的份额， 你知道这意味着什么 。

    使用“移动中”生物识别技术 — — 远距离获取生物识别数据 — — 是当权者的下一个“重大愿望”。要明确的是，“远距离”生物识别意味着在完全没有您本人参与的情况下、或者说在您完全不知情的情况下，进行的生物识别。

    因此，这是一个巨大的威胁，意味着大量的结果数据被各种公司和政府当局用于分析、预测和推断，以及将公民列入黑名单。

    在这场警务技术展览会上，意大利 武器制造商 莱昂纳多兜售他们的面部识别技术，据称使用人工智能进行分析，NEC 的人脸识别软件 NeoFace 现在已经被世界各国使用，这东西据称可以远距离捕捉面部图像。

    总部位于英国的 Tensor 和 Iproov 以及 Unisys，除了制作生物识别技术之外，还是 LineSight 技术的制造商，LineSight 是一种使用算法和机器学习来预测旅行者“意图”的分析平台 — — 就是 预测人们的行为，基于当权者的利益，将人们分类为“低风险和高风险”的概况 。

    在中国，面部识别已经是旧闻了，据称北京当局正在痴迷于虹膜。官方媒体的消息称，旷视科技（Face++）宣布获得了 4.6 亿美元 C 轮融资；商汤科技也于近日宣布获得 6.2 亿美元 C＋轮融资，他们的技术监视指纹、虹膜、手掌、静脉……

    虹膜是比人脸和指纹更难以伪造的生物特征。指纹有可能被模仿和伪造，人脸识别会存在双胞胎、父子母女难分辨的问题。而且，高超的整容技术、不同光照下的表情、姿态等也会影响识别效果。但，虹膜是终极的无法更改的东西。

    据称虹膜有 3 大特性：非接触性，中远距离就可以识别；大容量，速度快、实时性强；防伪性，通过瞳孔震颤可实现活体检测。

    铺设在地下的传感器

    上述的 Qinetiq 还展示了他们的名为 Optasense 的监视系统，这是 一套铺设在地下的光纤电缆，可作为声学、应力、压力和温度变化的传感器 。该传感器将能够探测一个区域内的人员或车辆，并且可以从一个位置点上监测长达 5000 公里的区域，这是英国 — 爱尔兰边界长度的十倍以上。

    Future Fiber Technologies 也在销售类似的光纤入侵检测和定位系统。

    你以为中国没有？那就错了。在中国，这东西被叫做“光纤振动传感技术”，就在本月的新闻中，这东西已经被铺设在南京大胜关长江大桥上。中国铁路上海局集团有限公司业务范围涵盖长三角地区，于是该公司计划先在这个区域铺设这种技术，然后再“辐射全国”！

    其实这种技术并不复杂，铺设在地下的光缆原本就有多条芯，只需占用其中一根即可。中国宣称目前可以实现单机双通道 80 公里的监测，如果每一米等效一个传感器，相当于一根光纤集成了 80000 个传感器。

    IMSI 捕获器

    你一定非常熟悉它了。这是一种高度侵入性技术，而且不新鲜。早在 2016 欧洲黑帽会议上，研究人员就展示了一种通过 WiFi 操作的新型 IMSI 捕捉器攻击方式，允许攻击者在 一秒内 捕获智能手机的 IMSI 号码，然后利用该 IMSI 号码来跟踪监视移动设备用户。

    IMSI 全称“ international mobile subscriber identity ”（国际移动用户识别码）是区别移动用户的标志，储存在 SIM 卡中，可用于区别移动用户的有效信息。不要将 IMSI 号码与 IMEI 号码混淆。 IMSI 绑定到用户，而 IMEI 绑定到设备。

    Wi-Fi 网络可以捕获附近智能手机的 IMSI 号码，无线跟踪并监控用户的根源在于智能手机（包括 Android 和 iOS 设备）连接 Wi-Fi 网络的方式。

    在大多数现代移动操作系统中有两种广泛实现的协议：可扩展认证协议（EAP）和认证和密钥协商（AKA）协议。这些协议允许智能手机通过自身设备的 IMSI 号码切换登录到已知的 Wi-Fi 网络，实现 WiFi 网络自动连接而无需所有者交互。攻击者可利用 WiFi 认证协议的这种机制伪装成 WiFi 网络，诱骗智能手机连接、捕获 IMSI 号码， 从而实现对用户的跟踪监视 。

    此外攻击者还可利用捕获的 IMSI 号码实现移动运营商提供的 WiFi 呼叫功能。智能手机 WiFi 呼叫功能允许用户通过使用加密的 IPSec 协议连接运营商的边缘分组数据网关（EPDG）。在这个过程中用于验证 WiFi 呼叫的互联网密钥交换（IKEv2）协议也是基于 IMSI 号码， 这可导致发生中间人（MITM）攻击，允许攻击者拦截通信流量 。

    众所周知，在美国，所谓的“手机信号塔转储”被执法部门使用（Verizon 在 2017 年上半年收到了近 9000 个法院命令 ），但这种权力此前并未在英国被公开宣布，而且关于如何使用的信息很少。Create Intelligence 是一家英国公司，生产用于数据分析的软件，据称其技术“部署在英国一些最大的执法机构中”。

    英国公司史密斯迈尔斯推出了“Airborne-NESIE”，能够对无人机或飞机上的移动设备进行地理定位和识别 ; 总部位于瑞士的 NeoSoft 公司试图向孟加拉出售 IMSI 捕手后，在国际隐私组织、WOZ 和瑞士的干预下 被叫停 ; 总部位于加拿大的 Exfo 展示了他们的 IMSI 捕获器能够拦截消息，呼叫和解密 ; 英国公司 DNA-Tracker，GOS Systems，Seven Technologies Group（前身为 Datong）Cellxion 和 Gamma International 都是如此。

    中国没有公开声称使用这种技术，然而它是如此的简单和具有普遍性，你能想象中国不会用吗？

    大规模监控

    大规模、无目标的监视使情报机构能够 使用互联网或电话监视任何人 ，利用这些数据进行详细的观察和推论， 不仅可以调查，还可以预测人们的行为 。

    而且不限于监视国内公民的数据，还能通过光缆以及国际服务器监视外国人的流量。在美国发生了什么现在所有人都知道了，同时，英国和爱尔兰当局也都参与了这种大规模监视。

    英国公司 GOS Systems 提供“大规模拦截”监控产品，英国最大的军火公司 BAE Systems 也是如此，据报道，该公司去年 曾向众多阿拉伯国家出口大规模互联网监控系统，这些国家的情报机构普遍存在侵犯人权的行为 。

    延伸阅读： 最新报告：是谁在提供先进的监控技术协助压迫性政府对人权的侵犯？

    英国公司 Gamma International 还为光纤网络中的国际网关部署了跨境的大规模互联网监控功能，“可以专注于特定国家或所有邻国”；总部位于荷兰的 2000 集团 在警务部门推出了一种新的大规模互联网监控产品 LIMA IP Mass Metadata Monitor，“旨在分析超大传输带宽的 IP 流量”并进行模式识别。

    美国公司 JSI 推出了一款“IP 记录器”，它使用 深度包检测 来收集互联网流量，并可以同时拦截“数千个 IP 目标”以 提取元数据和内容 。

    JSI 还推出了一种用于电话监控的“大规模情报分析”系统，允许“全国范围的数据收集”，旨在 收集“每天数以亿计的移动和固定电话记录” ，并保留它们进行深入分析：“这些大规模收集、并且长期被保留的数据，允许监控者对趋势做可视化分析，以预测人群在未来最可能出现的行为“。没错，就是《少数派报告》的现实版。

    我们不知道中国有没有铺设这样的技术，但俄罗斯已经被揭露了存在这种大规模监视系统。这是纯粹的间谍行为，对全体公民甚至邻国和外国人实施的间谍行为。如果您了解中国的相关信息，欢迎通过 安全渠道 与 独立媒体 联系。
