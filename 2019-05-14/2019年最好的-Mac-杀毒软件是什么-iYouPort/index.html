    2019年最好的 Mac 杀毒软件是什么？

    McCaffrey Knowledge Node , 技术防身/自我保护方法 2019年5月15日 2019年4月25日 1 Minute
    • Mac 被感染病毒的风险已经越来越高，这就是您需要采取保护行动的原因。那么，2019 年最好的 Mac 杀毒软件是什么？

    鉴于Macs“不会感染病毒”这种传说的存在，有些人可能会认为选择一款最好的 Mac 防病毒软件是个多余的任务。但尽管苹果机器的危险性不及 Windows，但它们仍然可能被感染病毒。

    而且风险在增加，这就是您需要采取保护行动的原因。本文提供一个列表，此列表中的每个产品都是合格的，因为它们专门用于保护 Mac 系统，并且在识别和删除病毒和恶意软件方面得到了很高的评价。

    macOS 操作系统基于 Unix 操作系统，它是沙箱。这意味着病毒很难造成任何伤害。把它们想象成锁在一个小盒子里，不管它们怎么努力都无法逃脱。

    这意味着您需要特定的 Mac 防病毒软件 — 无论是最强大的付费保护还是我们在本指南底部介绍的免费下载 — 这些软件都旨在满足 macOS 的独特需求。

    虽然针对 Mac 恶意软件的数量仍远低于针对 Windows 的数量，但它仍然比几年前专家预测的可能性要高出100倍。

    以下6个推荐实现100％检测而没有误报

    1. Bitdefender Antivirus for Mac

    Bitdefender Antivirus for Mac 是市场上最精确的Apple设备防病毒保护套件之一，它具有许多功能，包括多层勒索软件保护、广告软件删除、以及它还可以扫描 Windows PC 恶意软件。

    2. Intego Mac Premium Bundle X9

    Intego 已经有20年历史，它配备了一个防病毒引擎，可以应对最新的 Mac 威胁，以及Windows 和 Unix 恶意软件的威胁，它还具有用于管理网络权限的完整防火墙。

    3. Kaspersky Internet Security for Mac

    卡巴斯基 在防病毒保护领域备受好评，并凭借其 Internet Security for Mac 软件保持着良好的声誉，这带来了一系列让竞争对手深感不及的功能，包括防范勒索软件、家长控制和工具，确保您可以在线安全购物和处理银行存款。

    大量的功能当然是受欢迎的，但如果您正在寻找特定的工具，它可能看起来有点麻烦。

    4. Norton Security Deluxe

    Norton 是另一个在反病毒世界中广为人知的品牌， Norton Security Deluxe 软件通过优秀的恶意软件检测引擎帮助保护您的 Mac。它可以覆盖一系列设备，因此您不仅可以保护您的 Mac，甚至是您拥有的任何 Windows PC 或移动设备都可以。虽然它提供的安全性很好，但在使用时会对系统速度产生明显影响。

    如果你想让你的 Mac 感觉尽可能快，那么可以看看其他选择，但如果你想要强大，可靠的保护，那么无论如何，Norton Security Deluxe 绝对是值得投资的。

    接下来的四个推荐实现了99.5％和98.4％之间没有误报：

    5. Trend Micro Antivirus for Mac

    如果你正在运行最新版本的 macOS，macOS High Sierra（10.13），那么趋势 科技防毒软件Mac 版 是一个非常好的选择，因为它是为了兼容最近的操作系统而构建的。它带来了防病毒保护、电子邮件诈骗过滤、社交媒体隐私检查、以及新的勒索软件保护工具。自上一版本以来，扫描速度也得到了很多提升，可以更快地检查 Mac 上的病毒。

    6. ClamXAV

    ClamXAV 没那么最流行，但它是许多IT专业人士最喜欢的 Mac 防病毒软件 — 不仅仅因为它只为 Apple 电脑创建AV。

    这并不意味着它过于复杂 — 实际上，它具有此列表中所有 Mac 防病毒软件中最简单易用的界面之一。

    它特别擅长在电子邮件附件中捕获病毒，虽然它没有其他套件的某些互联网安全工具，但它仍然是保护 Mac 的绝佳选择。

    7. ESET Cyber Security for Mac

    ESET Cyber​​ Security for Mac 在独立测试实验中得分很高，它不是扫描 Mac 病毒的最快软件，但它深入到您的系统中以确保您的机器完全没有病毒，并且它具有非常有竞争力的价格。

    8. F-Secure SAFE

    F-Secure SAFE 作为整个家庭的防病毒服务，保护各种设备，包括PC、智能手机、平板电脑和 Mac，以及强大的家长控制和广告控制。

    AV-Test 发现 F-Secure 在2018年3月和4月期间 可以抵御100％的零日威胁 ，以及在过去四周内发现的所有普遍流行的恶意软件的威胁。但是截至2018年6月，MacOS 恶意软件的检测率为 96.4％。

    尽管如此，它确实提供了一些漂亮的功能，例如网络银行保护，它可以阻止特定网站和应用程序在您登录时与您的系统进行交互，以及针对勒索软件祸害的特定保护措施。

    简而言之，F-Secure 的用户友好用户界面和准确的保护能力使其成为一个合适的选择。

    想要免费的吗？

    无论您使用的是 Mac 还是 PC，争论都是一样的。免费的防病毒产品可有效抵御威胁，但缺乏某些高级功能。Mac 可能被认为比PC更安全，但您仍然可能不会为了节省一些钱而冒险。

    排名前5的最佳免费杀毒软件如下：

    1. Avast Free Mac Security

    Avast Free Mac Security 通过一个非常冗长的功能列表来保护您的 Mac。

    例如，核心防病毒引擎提供实时保护，以便在威胁出现时捕获它们。如果您怀疑有问题，可以运行按需完整系统扫描，或者对特定文件、文件夹或驱动器启动更有针对性的检查。甚至可以安排扫描在您不在时自动运行。

    Web Shield 会警告您恶意网站，阻止危险的下载和电子邮件附件，并通过消除侵入式广告跟踪来保护您的隐私。

    Avast 的无线网络扫描仪有一个不寻常的额外功能，它可以快速检查您的网络、路由器和连接的设备，提醒您任何漏洞。

    Mac 防病毒软件通常不会被独立测试实验室检查，但 Avast 的最新检查结果是好的，AV比较2018测试发现该软件包检测到100％的样本威胁。

    如果您还想要更多，升级到 Avast Security Pro 可以获得防勒索软件保护、和 Wi-Fi 入侵者的即时警报、或网络安全性的变化。一台Mac，一年大概49.99英镑（70美元）。

    2. Avira Free Antivirus for Mac

    Avira Antivirus for Mac 采用了更简单、更轻量级的方法。没有网络过滤，没有网络扫描或其他安全附加功能：它完全专注于保持您的系统免受恶意软件攻击。

    它的实时扫描程序会检查系统访问的任何文件，在它们造成任何伤害之前拦截威胁。调度程序可以设置应用程序以自动扫描您的系统，或者您可以随时检查整个系统。

    该应用程序不仅可以防御 Mac 威胁。它还使用 Avira 的 Windows 体验来搜查任何与PC相关的恶意软件，确保您不会共享任何可能危及您朋友的文件。

    主要测试实验室暂时没有对 Avira Free Antivirus 进行评估，但 AV Comparatives 上次 报告 发现Avira 阻止了100％的恶意软件，而且 Avira 在PC上也有强劲的记录，毫不怀疑这是一个值得信赖的应用。

    虽然这个列表都是关于免费的 Mac 防病毒软件，但 Avira 的商用 Avira Antivirus Pro 产品值得一提。它包括更多技术支持，并通过一个许可证（Mac，Windows和Android）保护多达三个设备，每年花费39.99英镑（42美元）。

    3. Bitdefender Virus Scanner for Mac

    Bitdefender Virus Scanner 是一款轻量级免费恶意软件猎手，可以追踪并删除与 Mac 和 Windows 相关的威胁。

    简单直接的界面使应用程序非常易于使用。状态显示让您及时了解系统的运行情况，四个按钮可以快速扫描关键位置，深入检查整个系统，或快速扫描正在运行的应用程序或您的文件夹。

    Bitdefender 每小时发布一次更新，Virus Scanner 始终在开始扫描之前获取最新版本。

    4. Malwarebytes for Mac

    Mac 防病毒软件不仅仅是检测最新的最致命的勒索软件或其他高风险威胁。大多数工具还可以删除广告软件和其他可能不需要的程序，这些程序可能不是很危险，但是会浪费系统资源并降低 Mac 速度。

    自2006年以来， Malwarebytes 一直在消除各种 Windows 上的威胁，其 Mac 版本同样简单、直接和有效。

    该应用程序是一个轻量级的安装，三个数字音乐文件的大小，你几乎没有注意到它的存在。虽然缺乏实时保护意味着 Malwarebytes 无法阻止攻击 — 只能删除现有的感染 — 但它也确保对您的系统资源没有实际影响，并且应用程序不太可能与其他防病毒工具冲突。

    性能是另一个亮点，Malwarebytes 声称系统在不到15秒的时间内被扫描。安装 Malwarebytes for Mac 还可以为您提供30天的 Premium 试用版，该版本能够检测并阻止出现的威胁。如果你对此不感兴趣，请忽略它；试用期满后，实时保护功能将消失，您只能进行按需扫描。

    升级到 Malwarebytes Premium 仅需29.99英镑（38美元），另外还可选择以49.99英镑（65美元）保护多达10台设备。

    5. Sophos Home for Mac

    免费的防病毒市场竞争非常激烈，个别公司很难脱颖而出，但英国供应商 Sophos 已经找到了一种方法：它在免费产品中为您提供了比许多竞争对手在付费版本中提供的功能更多的功能。

    按需扫描以清理受感染的系统？当然。实时保护首先防止攻击？没问题。浏览器过滤阻止访问危险站点？自然。家长控制、以定义您的孩子可以在线访问的内容？是的。

    你还可以用一个帐户覆盖最多三台 Mac 或 Windows 设备，并从中央网络控制台远程检查和管理它们的状态 — 在保护整个家庭时可能非常有用。

    AV Test 的2017年12月Mac检测率 测试表明 ，Sophos 提供的保护与 Avast，Bitdefender，Kaspersky 和其他大牌产品相同。

    Sophos Home Premium 确实有一些有价值的额外功能：高级勒索软件保护、监控网络摄像头和麦克风的访问、实时聊天和电子邮件支持，以及保护多达10个设备的选项。它功能强大，每年的价格为37.50英镑（48美元）。

    总之，选择什么仍然取决于你。也许问题不应该是反病毒软件是否物有所值，而是，你的资产是否比安全应用程序的成本更有价值。
