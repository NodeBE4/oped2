    如何强制 Firefox 永远不会保存任何密码

    McCaffrey Knowledge Node , 技术 , 技术防身/自我保护方法 2018年11月25日 2019年8月10日 0 Minutes
    • 不要让老大哥追踪到你。只需要一些小步骤

    浏览互联网时，您的隐私是最重要的问题之一。有许多眼睛能跟踪您的活动、位置等。通过跟踪您的浏览习惯，可以从中提取大量关于您的个人信息。这就是为什么要建议您在网上留下最少的痕迹 —— 我们称之为数据面包屑。

    • 如果您想要知道浏览历史记录有多么的危险，请阅读这篇文章，它是一个仅仅通过浏览记录就能挖掘一个人真实内心世界的跟踪演示《 ⚠️浏览历史记录曝光你的内心全景图：#OSINT 开源调查演示 》。请记住，这只是一个极小程度上的演示，已经足够恐怖，而真正的老大哥比这厉害一万倍。

    我需要防止这种事发生。

    作为 Web 浏览器，Firefox 是一个非常受欢迎的选择。它是开源的，免费的，非常强大的。

    每当您决定登录 Facebook、Twitter 或银行帐户时，Firefox 都会自动提示您保存密码。但是要知道，在浏览器上保存密码是一件很危险的事！浏览器密码保护功能并不强大，因为您每天都要访问数百个网站，所以永远不要让浏览器管理您的密码是至关重要的。

    我们应该迫使 Firefox 永远不要保存任何密码。
