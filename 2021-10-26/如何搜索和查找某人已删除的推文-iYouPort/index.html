    如何搜索和查找某人已删除的推文

    Gabriel Fornard Knowledge Node , OSINT 2021年10月26日 2021年12月21日 1 Minute
    • 不那么容易。但有4种方法可以试试。

    不管是名人的尴尬口误，还是异议人士想要避免被喝茶而进行的自我审查，其结果都是，某些推文很快就会被删除；或者有人开着第三方自动删推软件，定时删除一切。我们其实鼓励这种做法，尤其是在 社交媒体情报的威胁性越来越大 的情况下，但是，我们更需要提醒您一个常识：任何内容，只要它上了互联网，就很可能难以彻底删除了；尤其是社交媒体，尤其是那种您觉得危险的和尴尬的信息。

    删推是一种可能没什么用的措施，因为人们经常可以找到方法来搜索和查找到那些已经被删除的推文。

    本文将向您展示一些可以找到某人被删除的推文并保存它们的主要方法。

    1、通过谷歌搜索

    十有八九可以搜到，当您想找到某人被删除的推文时，而这个人又是一个著名人物，那谷歌可能是最好的方法。

    寻找被删除的名人推文的好处是，您可能不是第一个想这么做的人。从忠实的粉丝到媒体，有很多人都在盯着这些名人。他们的推文很有可能已经被其他人复制和截屏了，这就是为什么您可以谷歌搜到，大家都喜欢看名人出丑。

    例如， 有一个网站专门记录主要政客被删除的推文 ，包括他们一些更有争议的言论。

    寻找这些被保存的推文的最佳来源可能是Twitter本身。许多精明的用户对名人的推文进行截图，并在它们被删除时分享出来。这让许多知名人士被抓到把柄。

    2、检查谷歌的缓存

    众所周知您的浏览器会缓存网页，以帮助它们更快地加载。但是您知道谷歌本身也做了类似的事吗？在滚动浏览谷歌结果时，您可能已经注意到一些条目旁边有一个小箭头，点击这个箭头，您就可以查看该网站的缓存版本。

    这可以帮助您查看一个著名的 Twitter 账户的已经被删除的推文。如果该推文是最近删除的，有可能它仍然存在于缓存中。唯一的问题是，该账户需要足够有名，谷歌才会对其进行缓存。

    要查看某人的推文的缓存版本，首先搜索他们的推特ID。

    他们最近的推文将在搜索结果中显示出来。点击任何一个条目旁边的向下的小箭头，选择缓存。

    谷歌缓存版本的推文将打开。缓存的日期和时间将显示在顶部。由于新的缓存可以覆盖这些数据，建议您自己截图保存。

    3、使用 Wayback Machine

    如果您要找的被删除的推文时间太久远了，无法在谷歌的缓存中显示出来，也不知道有谁保存了它，那最好的办法就是使用时光机。这是一个将互联网上的公共网页存档以供后人查阅的网站。

    虽然还有一些其他的服务可以做同样的事，但 Wayback Machine 是迄今为止这些选项中最受欢迎和最可靠的。

    互联网档案馆持有所有公共网页的备份，这些备份是在定期的时间间隔内进行的。您可以回到过去，查看一个网站过去的样子。这包括许多知名人士的 Twitter 网页。

    • 对于 Wayback Machine 的使用方式 ，我们有过详细介绍，见《 如何快速找到你想要的东西：追踪数字证据的小技巧 》

    要搜索档案中被删除的推文，先打开 Wayback Machine 网站 。

    输入您要查看其历史记录的 Twitter 帐户的个人资料链接，然后单击 “浏览历史记录”。

    这将打开一个日历，显示目标 Twitter 帐户已被存档的所有日期。

    将鼠标悬停在日期上可查看当天可用的所有快照以及拍摄快照的时间。

    单击一个快照将其打开。这不仅仅是一个截图；整个网页将以其过去的状态加载。您可以正常与页面交互，并截取您想要的任何推文。

    4、搜索推特档案

    以上探讨的是搜索别人的被删除的推文的方法。但是，如果您正在寻找一种方法来查看您自己被删除的推文呢？

    这一次您不需要找得太辛苦了。推特本身保存着所有已经发布过的推文的档案，甚至包括被删除的推文。通常情况下，它不允许任何人访问这些数据，但任何用户都可以通过自己的Twitter历史进行搜索。

    要下载您的 Twitter 档案，打开 Twitter 主页，点击更多。在出现的菜单中选择设置和隐私。

    这将打开您的帐户设置。单击右侧窗格中的下载数据存档选项。

    您将被要求通过发送到您的电子邮件 ID 的代码来验证您的身份。完成后，您将看到请求存档按钮。

    Twitter 可能需要一整天的时间才能将存档通过电子邮件发送给您。解压缩 zip 文件并打开它后，将在您的 Web 浏览器中打开一个普通的 Twitter 窗口。唯一添加的是一个搜索框，可让您按日期查找过去的推文。

    最后

    Twitter 并没有让任何人都能很容易地搜索到被删除的推文。虽然推特存档可以帮助您访问您自己的推文历史，但找到别人被删除的推文要困难得多。

    如果这些推文属于知名人士，那么很有可能该推文被其他人在 Twitter 上截图并分享了。谷歌是在这种情况下的最容易使用的工具。

    如果这也不能给您带来任何结果，事情就会变得有点棘手。对于最近的推文，您应该在目标推特ID的缓存版本中寻找，因为谷歌经常将一个旧页面保存数天。如果您要找的推文的时间较长，则可以在 Wayback Machine 上搜索。⚪️
