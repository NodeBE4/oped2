    完美替身 – [video]

    McCaffrey Knowledge Node , 技术防身/自我保护方法 , 给行动者的护身符和武器 2019年9月25日 2019年9月23日 0 Minutes

    IYP 介绍过制作全套假身份的方法，详见《 少数派玩家 》。不过这里面没有照片。照片是非常重要的，每一个真人都要有一张脸才行，否则“真”不起来。尤其是如今人脸识别遍地都是，您不可能在网上搜别人的脸贴在自己的身份信息上，那相当于转嫁危险性。

    那怎么办呢？幸运的是，找到一张假脸已经非常容易。

    人工智能不仅可以制作逼真的人物假视频、可以描述一个真人现实生活中从未说过的话或从未做过的事，它还可以产生令人信服的假人脸，这些人脸从未在现实中存在过。

    你想要一张逼真的假脸吗？曾经做到这点需要一些超级昂贵且功能强大的硬件，但是，现在有一个网站，可以轻松实现这点：生成一个花哨的新面孔就像刷新浏览器一样简单： thispersondoesnotexist.com 。每次加载页面时，算法都会从头开始生成新的人脸。

    该网站使用被称为生成对抗网络（GAN）的机器学习实现。这些程序从大量的训练数据输入中“学习” — 比如真实的人脸，以产生新的例子。 Thispersondoesnotexist.com使用以前由 Nvidia 研究人员 在 GitHub 上发布的代码 。

    网站创建者和软件开发人员 Philip Wang 在一封电子邮件中说，“大多数人都不明白合成人脸图像有多么的重要。”在这里看到视频：

    GAN 首先由计算机科学家 Ian Goodfellow 及其同事于2014年推出。从那时起，该技术已经变得更有能力了，现在可以以惊人的高分辨率生成高度逼真的图像。

    这些技术已被用于制作人们跳舞的视频，并将夏季场景转变为冬季场境。令人沮丧的是，他们的简单可用性也导致了伪造的色情视频的大量产生，被称为“深度造假”。在这里看到相关详细分析《 当恶意者掌握技术：信息战的未来究竟有多可怕？ 》

    Wang 的网站运行在租用的服务器上，该服务器具有运行 Nvidia 软件的强大 GPU。 “我每隔两秒就会做出一个随意的人脸，并以可扩展的方式向世界展示”，他说。

    显然，GAN 是多功能的东西，但 Wang 说他想坚持用来它来制作面孔。 “脸部对我们的认知来说最为突出，因此我决定将这种特定的预训练模型放在上面，”他在 Facebook 上说。

    事实上同类的假脸生成服务正越来越多的出现。因为这个问题是紧迫的：在未来几年，我们与身份的关系将发生深刻的变化。

    Generated Photos 网站也能达到同样的目的，该网站使用人工智能创建了数千张面孔。

    所有这些面孔都是不存在的人。你可以随意使用这些面孔，而不会与任何活人“冲突”。

    通过开放的Google云端硬盘，该网站免费提供100,000张此类照片。

    该站点的目标最终是能够按需生成人脸。您可以在下面看到视频：

    好啦，现在你也可以制造出一整套完美的假身份！彻底切断在线和线下的身份联系。我们一起来创造一个新的“世界”。
